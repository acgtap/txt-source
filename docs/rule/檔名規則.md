# 檔名規則

1. 資料夾與檔名盡量避免可能變成超級長的狀況
2. 章節名為人名之類的則看情況選擇原名或譯名
3. 符號及數字盡量為全形（章節名的部分，章節序列的部分依照原來的或者看情況加上數字）
4. 非單字類英文也盡量全形（例如Ａ級）
5. 如果章節序列不存在或者章節序列非半形數字時（加上前導的排序用數字 例如 00010_）以　10 為間隔 方便日後可以隨時在中間插入新增章節，並且盡量以 10 開始，因為有時會出現追加了序章之類的狀況
6.
