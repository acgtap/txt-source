# 如何用码云(gitee.com)

首先先弄個 碼雲 帳號

https://gitee.com

## 介面說明

### 1
![ui001](img/ui001.jpg)

* Watch = 關注
* Star = 收藏
* [Fork](Pull%20Request.md) = 複製一份到你的空間內
* 

### 2
![ui002](img/ui002.jpg)

* 程式碼 = 檔案列表
* Issues = 留言/問題回饋
* [Pull Requests](Pull%20Request.md) = 申請合併的紀錄(包含 尚未接受 / 已合併 / 已取消)
* Wiki = 說明(如果有的話)
