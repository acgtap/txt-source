# 編輯按鈕無法按的相關問題

如果是透過修改紀錄查看檔案

![2018-07-06-01-52-59-12.png](img/2018-07-06-01-52-59-12.png)

這時候如果想要編輯會發現無法編輯的狀況  
因為目前的查看狀態為歷史紀錄裡面  
所以無法編輯

![2018-07-06-01-53-33-13.png](img/2018-07-06-01-53-33-13.png)

在左邊版本編號的地方 切換回 master

![2018-07-06-01-54-21-14.png](img/2018-07-06-01-54-21-14.png)

這時候就可以編輯了
