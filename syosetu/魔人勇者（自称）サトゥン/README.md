## preface


```
魔人界の中でも最強の一角と謳われる魔人サトゥン。
一つの夢を叶えるため、己の欲望のために彼は魔人界に別れを告げ、人間界へと転移する。

彼の夢――それは『リエンティの勇者』となり、人々に勇者としてちやほやされることであった。
これはおバカな魔人、自称勇者サトゥンと、そんな彼に振り回される苦労人な人々の成長物語。
```
