## preface


```
　ＶＲＲＰＧ『ソード・アンド・ソーサリス』をプレイしていた大迫聡は、そのゲーム内に封印されていた邪神を倒してしまい、呪詛を受けて死亡する。
　そんな彼が目覚めたのは、何処とも知れない森の中であった。
　しかも能力はゲームで使用したアバターその物。正にチートである。
　四十代ＤＴのおっさんは、この世界で平穏な生活を送れるのだろうか？
　そんな彼の異世界生活がはじまる。
　現在、貴族の家庭教師を終え、無職中。
```
