## preface


```
身懷無限魔力的金發蘿莉，幸福而又平凡的生活於小小村落之中。然而，這份平靜就這樣被打破。燃起的戰火，慘遭殺害的鄉親，被拐走的蘿莉。

日常崩壞，手無縛雞之力的蘿莉拿起了魔導槍。
為了復仇，為了一度失去的幸福日子，更是為了回到重要的親人身邊。
為了這一切，總是化身瘋狂殺戮的修羅也在所不辭。

魔彈蘿莉走入歷史的舞台。
```
