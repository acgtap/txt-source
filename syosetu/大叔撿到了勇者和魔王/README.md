## preface


```
冒険者ギルドを退職させられたおっさんは故郷に帰る事にする。
その道中、彼は二人の赤子を拾う。一人目は勇者の紋章を宿した人間の赤子。二人目は魔王の紋章を宿した魔族の赤子であった。
かくして、勇者と魔王を拾ったおっさんの物語が幕を開ける。
```
