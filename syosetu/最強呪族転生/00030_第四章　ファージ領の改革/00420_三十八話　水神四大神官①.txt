我在床上睡觉时，突然被巨大的爆炸声吓醒。
像前日一样，有什么东西进入了我的房间。
我慌张地紧握着杖，从床上飞快地跳落在地板上。
接着把杖尖指向传来声音的方向。
。
『咕哦哦哦！』
。
那明显不是人类或通常的魔兽的声音，而是十分可怕的叫声。
而且再加上断续的，仿佛在不断敲打着什么东西的碰撞声。
。
我设置的防身用欧特姆，反复地对放出青白色光芒的魔兽使出撞击。
魔兽发出尖叫，露出腹部拼命地乞求原谅。
欧特姆一边回转一边跃起，挖开了魔兽的腹部。
魔兽的身体大大地弹起，变成了无数的小小的光球，宛如在空气中融化般消失不见了。
。
「精灵兽……」
。
我呆呆的说道。
。
精灵兽有很多种定义，一般在精灵聚集成的生物个体当中也是危险度最低的。
没有复杂的魔法，也没有语言能力，只是模仿简单的动物的形状……等等，依存在不同的地方或时代，明确的判断基准也有些许不同。
也有人认为是下级恶魔的同类。
。
总之，认为是没有复杂的力量，没有语言能力，有着模仿动物姿态倾向的下级恶魔的总称就肯定没错了。
虽然也有稀少的难以判断的个体，不过也只是西瓜和草莓是蔬菜还是水果这种程度的问题罢了。
说实话怎样都行。
。
前几天在我感冒时袭击我的恐怕也是精灵兽吧。
难怪有打中的手感却没有发现目标。
大概是维持不了身体，分解成了精灵吧。
。
在敌人操纵哈梅伦的时候，我早就应该发现这一点。
说起来，在魔兽灾害之际好像也有因精灵兽造成损失的报告的样子。
对方是精灵使吧。
。
「精灵兽两只，和高位恶魔哈梅伦吗……」
。
要从精灵兽或恶魔中得到召唤纹并使役它们并不是一件简单的事情。
我也试过一次强行地获取召唤纹，但因为用力过度而差点把恶魔消灭了。
魔力的倾向、质量、波长都不吻合的话，召唤纹是不会那么容易就能获得的。
我现在饲养着的哈梅伦，明明在我提供魔力的时候那么开心，然而到现在还没有给我召唤纹的意思。
这些小家伙们就是这么的纤细。
正是因为这样，它们才会对给予召唤纹的人唯命是从。
。
「居然能驯服三只，看来是有相当的操纵恶魔的心得呢……好羡慕啊。能向他请教就好了」
。
只是那三只也已经被我搞定了。
。
不过，为什么要在这个时点向我袭击呢，我怎样都搞不懂他的企图。
早已失败过一次，对方也应该明白这件事情才对的。
从今以后也绝不退缩，大概就是这种意思的开战宣言吧。
。
想到这里时，从公馆的其他房间中传来了悲鸣。
不对，不只是这样，从领民们的房子那边也传出悲鸣，还能看见有精灵兽在远处奔跑。
。
怎么说也太惊人了。
他究竟使役着多少只精灵兽呢。
。
而且现在居然袭击整个村子，他到底想做出什么样的结果我也无从得知。
难道是放弃治疗，自暴自弃地转移到攻击模式？
但我不太认为他是会做出这种不经大脑的行动的对手……。
。
总之先去确保美亚和……屋内的人们的安全吧。
之后还有领地内的人们。
。
「运送吧」
。
我咏唱咒文，在屋内显现出十三个魔法阵，从中出现了相同数目的欧特姆。
和世界树制的不一样，转移用普通的木材制作的欧特姆需要花费庞大的魔力。
不过这些欧特姆都是在珐基领以内的东西，所以还在我魔力的容许范围之内。
虽然负担还是有点沉重。
。
「人形啊 舞动吧」
。
我再次展开十三个魔法阵从而操纵欧特姆。
十只欧特姆从被破坏的窗户里飞了出去，剩下的三支则留在公馆内。
很快，从公馆中响起了精灵兽的惨叫。
。
我通过走廊，进入了美亚的房间。
。
「喂，没事吧……」
。
美亚在床上用毛毯包裹着瑟瑟发抖的身体，两只欧特姆在她身旁咕噜咕噜地转圈。
。
「啊，亚贝尔……太好了……。美亚，还以为已经不行了……」
。
美亚用颤抖的声线说道，用身子被毛毯包裹的步履维艰的装束站了起来，摇摇晃晃地走到我身旁。
外面的墙壁开了一个大洞，似乎是被精灵兽撞开的。
。
看来美亚是被精灵兽的袭击给吓到了。
我总算是放下心来，把手搭在美亚的肩膀上。
。
「看到可怕的魔兽了吗？」
。
「不、不是，刚才在美亚的房间里……那个，有两只发出青白色光的狼闯进来了……」
。
两只？
明明我的房间里只进来了一只……。
。
「那两个家伙已经不在了吗？」
。
「……被那两只转来转去的欧特姆踩碎了」
。
我看向美亚指着的方向，那两只欧特姆似乎在跳着胜利的舞蹈。
。
「……是嘛，那真是太好了。其实那是**控的人偶，所以不用担心啦」
。
我用杖一指，欧特姆就从墙壁的洞中飞走，继续寻找其他的精灵兽。
……拉尔克府邸，一下子就变得通风了呢。
之后从楼下传出“咚、咚”的巨响，接着便回响起精灵兽临终的悲鸣。
。
我们和公馆里的人汇合了。
拉尔克和他的亲属三人，入住的佣人共三人……再加上我和美亚，集齐了合计九个人。
虽然也有受伤的人，但也不是关服性命的伤势。
。
「那些是亚贝尔少爷的木偶人形对吧。哎呀，真是帮大忙了。没想到领地内居然会混入这么多的魔兽……」
「要不是亚贝尔大人的木偶人形的话，实在是太惊险了。真是万分感谢……」
「那个人形，好像把王都的职人制作的一级乐器摔坏了……」
「太感谢您了！如果不是亚贝尔大人，我都不知道会变成怎样！」
。
拉尔克的亲属和佣人纷纷言谢。
只有拉尔克一人青着脸，表现得不知所措。
。
「玛、玛利亚斯不在这……」
。
听闻，公馆里的人们表情逐渐黯淡，视线朝下。
剩下的佣人成员中的确没有玛利亚斯的身影。
。
我终于知道敌方的意图了。
。
「请大家放心，她应该会没事。恐怕……是被抓去当人质了吧」
。
「人质？」
。
「是的，美亚的房间里也同时闯入了两只精灵兽。如果这就是用来对付我的手牌的话，那么优先袭击她们也是不奇怪的吧」
。
而派出精灵兽袭击我的原因，大概是想碰碰运气来一个出其不意吧。
或许是认为无论派多少只过来也没多大用处，所以只派出一只过来袭击我吧。
。
「恐怕……村子里有某种影响力的人也被它们带走了。去问问有没有人看见会掳走人类的精灵兽吧」
。
应该是会有故意让其他人看见逃跑方向的精灵兽的。
也许……这次袭击是给我的挑战书。
对方判断到这样下去领地会完全复苏，于是打算和我直接地对决吧。
。
命令精灵兽的话，领民之中的确会出现死者的哦，可能还包含着这种威胁的意思。
不管这样如果我的猜想是正确的话，大概是会出现下落不明的领民的。
林古斯也在其中的话，那就十有八九是他干的了。
。
但这个推论中存在一个漏洞。
明知道我打倒了纳尔加仑、哈梅伦和伊卡洛斯，为何还希望和我正面交锋。
。
难道说，他……还藏着再生纳尔加仑之上的绝招吗。
至今为止虽然有着各种各样的问题，但却比想象中的要顺利，或许是我太低估对手了呢。
我也得打起十二分精神来才行呢。
在引诱着我的前方，大概张罗着什么可怕的陷阱吧。
必须提高警惕。
。
「……亚贝尔，表情松弛了哦」
。
美亚用手作筒，偷偷地对我耳语。
我慌张地整顿表情向前一看，拉尔克在用不高兴的眼神盯着我。
不……不好意思。
