我在床上睡覺時，突然被巨大的爆炸聲嚇醒。
像前日一樣，有什麼東西進入了我的房間。
我慌張地緊握著杖，從床上飛快地跳落在地板上。
接著把杖尖指向傳來聲音的方向。
。
『咕哦哦哦！』
。
那明顯不是人類或通常的魔獸的聲音，而是十分可怕的叫聲。
而且再加上斷續的，仿彿在不斷敲打著什麼東西的碰撞聲。
。
我設置的防身用歐特姆，反覆地對放出青白色光芒的魔獸使出撞擊。
魔獸發出尖叫，露出腹部拼命地乞求原諒。
歐特姆一邊回轉一邊躍起，挖開了魔獸的腹部。
魔獸的身體大大地彈起，變成了無數的小小的光球，宛如在空氣中融化般消失不見了。
。
「精靈獸……」
。
我呆呆的說道。
。
精靈獸有很多種定義，一般在精靈聚集成的生物個體當中也是危險度最低的。
沒有複雜的魔法，也沒有語言能力，只是模仿簡單的動物的形狀……等等，依存在不同的地方或時代，明確的判斷基準也有些許不同。
也有人認為是下級惡魔的同類。
。
總之，認為是沒有複雜的力量，沒有語言能力，有著模仿動物姿態傾向的下級惡魔的總稱就肯定沒錯了。
雖然也有稀少的難以判斷的個體，不過也只是西瓜和草莓是蔬菜還是水果這種程度的問題罷了。
說實話怎樣都行。
。
前幾天在我感冒時襲擊我的恐怕也是精靈獸吧。
難怪有打中的手感卻沒有發現目標。
大概是維持不了身體，分解成了精靈吧。
。
在敵人操縱哈梅倫的時候，我早就應該發現這一點。
說起來，在魔獸災害之際好像也有因精靈獸造成損失的報告的樣子。
對方是精靈使吧。
。
「精靈獸兩只，和高位惡魔哈梅倫嗎……」
。
要從精靈獸或惡魔中得到召喚紋並使役它們並不是一件簡單的事情。
我也試過一次強行地獲取召喚紋，但因為用力過度而差點把惡魔消滅了。
魔力的傾向、質量、波長都不吻合的話，召喚紋是不會那麼容易就能獲得的。
我現在飼養著的哈梅倫，明明在我提供魔力的時候那麼開心，然而到現在還沒有給我召喚紋的意思。
這些小傢伙們就是這麼的纖細。
正是因為這樣，它們才會對給予召喚紋的人唯命是從。
。
「居然能馴服三只，看來是有相當的操縱惡魔的心得呢……好羨慕啊。能向他請教就好了」
。
只是那三只也已經被我搞定了。
。
不過，為什麼要在這個時點向我襲擊呢，我怎樣都搞不懂他的企圖。
早已失敗過一次，對方也應該明白這件事情才對的。
從今以後也絕不退縮，大概就是這種意思的開戰宣言吧。
。
想到這裡時，從公館的其他房間中傳來了悲鳴。
不對，不只是這樣，從領民們的房子那邊也傳出悲鳴，還能看見有精靈獸在遠處奔跑。
。
怎麼說也太驚人了。
他究竟使役著多少只精靈獸呢。
。
而且現在居然襲擊整個村子，他到底想做出什麼樣的結果我也無從得知。
難道是放棄治療，自暴自棄地轉移到攻擊模式？
但我不太認為他是會做出這種不經大腦的行動的對手……。
。
總之先去確保美亞和……屋內的人們的安全吧。
之後還有領地內的人們。
。
「হন運送吧」
。
我詠唱咒文，在屋內顯現出十三個魔法陣，從中出現了相同數目的歐特姆。
和世界樹制的不一樣，轉移用普通的木材製作的歐特姆需要花費龐大的魔力。
不過這些歐特姆都是在琺基領以內的東西，所以還在我魔力的容許範圍之內。
雖然負擔還是有點沉重。
。
「পুতুল　দখল人形啊 舞動吧」
。
我再次展開十三個魔法陣從而操縱歐特姆。
十只歐特姆從被破壞的窗戶裡飛了出去，剩下的三支則留在公館內。
很快，從公館中響起了精靈獸的慘叫。
。
我通過走廊，進入了美亞的房間。
。
「喂，沒事吧……」
。
美亞在床上用毛毯包裹著瑟瑟發抖的身體，兩只歐特姆在她身旁咕嚕咕嚕地轉圈。
。
「啊，亞貝爾……太好了……。美亞，還以為已經不行了……」
。
美亞用顫抖的聲線說道，用身子被毛毯包裹的步履維艱的裝束站了起來，搖搖晃晃地走到我身旁。
外面的牆壁開了一個大洞，似乎是被精靈獸撞開的。
。
看來美亞是被精靈獸的襲擊給嚇到了。
我總算是放下心來，把手搭在美亞的肩膀上。
。
「看到可怕的魔獸了嗎？」
。
「不、不是，剛才在美亞的房間裡……那個，有兩只發出青白色光的狼闖進來了……」
。
兩只？
明明我的房間裡只進來了一只……。
。
「那兩個傢伙已經不在了嗎？」
。
「……被那兩只轉來轉去的歐特姆踩碎了」
。
我看向美亞指著的方向，那兩只歐特姆似乎在跳著勝利的舞蹈。
。
「……是嘛，那真是太好了。其實那是**控的人偶，所以不用擔心啦」
。
我用杖一指，歐特姆就從牆壁的洞中飛走，繼續尋找其他的精靈獸。
……拉爾克府邸，一下子就變得通風了呢。
之後從樓下傳出“咚、咚”的巨響，接著便回響起精靈獸臨終的悲鳴。
。
我們和公館裡的人匯合了。
拉爾克和他的親屬三人，入住的傭人共三人……再加上我和美亞，集齊了合計九個人。
雖然也有受傷的人，但也不是關服性命的傷勢。
。
「那些是亞貝爾少爺的木偶人形對吧。哎呀，真是幫大忙了。沒想到領地內居然會混入這麼多的魔獸……」
「要不是亞貝爾大人的木偶人形的話，實在是太驚險了。真是萬分感謝……」
「那個人形，好像把王都的職人製作的一級樂器摔壞了……」
「太感謝您了！如果不是亞貝爾大人，我都不知道會變成怎樣！」
。
拉爾克的親屬和傭人紛紛言謝。
只有拉爾克一人青著臉，表現得不知所措。
。
「瑪、瑪利亞斯不在這……」
。
聽聞，公館裡的人們表情逐漸黯淡，視線朝下。
剩下的傭人成員中的確沒有瑪利亞斯的身影。
。
我終於知道敵方的意圖了。
。
「請大家放心，她應該會沒事。恐怕……是被抓去當人質了吧」
。
「人質？」
。
「是的，美亞的房間裡也同時闖入了兩只精靈獸。如果這就是用來對付我的手牌的話，那麼優先襲擊她們也是不奇怪的吧」
。
而派出精靈獸襲擊我的原因，大概是想碰碰運氣來一個出其不意吧。
或許是認為無論派多少只過來也沒多大用處，所以只派出一只過來襲擊我吧。
。
「恐怕……村子裡有某種影響力的人也被它們帶走了。去問問有沒有人看見會擄走人類的精靈獸吧」
。
應該是會有故意讓其他人看見逃跑方向的精靈獸的。
也許……這次襲擊是給我的挑戰書。
對方判斷到這樣下去領地會完全復甦，於是打算和我直接地對決吧。
。
命令精靈獸的話，領民之中的確會出現死者的哦，可能還包含著這種威脅的意思。
不管這樣如果我的猜想是正確的話，大概是會出現下落不明的領民的。
林古斯也在其中的話，那就十有八九是他幹的了。
。
但這個推論中存在一個漏洞。
明知道我打倒了納爾加侖、哈梅倫和伊卡洛斯，為何還希望和我正面交鋒。
。
難道說，他……還藏著再生納爾加侖之上的絕招嗎。
至今為止雖然有著各種各樣的問題，但卻比想像中的要順利，或許是我太低估對手了呢。
我也得打起十二分精神來才行呢。
在引誘著我的前方，大概張羅著什麼可怕的陷阱吧。
必須提高警惕。
。
「……亞貝爾，表情松弛了哦」
。
美亞用手作筒，偷偷地對我耳語。
我慌張地整頓表情向前一看，拉爾克在用不高興的眼神盯著我。
不……不好意思。
