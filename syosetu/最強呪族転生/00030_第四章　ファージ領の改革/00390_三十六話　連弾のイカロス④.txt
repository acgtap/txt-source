伊卡洛斯釋放的炎之魔彈向我襲來。
畢竟是突然的襲擊，似乎並沒有一擊必殺的意思，速度和威力也很小。
。
說實話，直接承受這個魔彈也是一種選擇，不過約定就是約定，姑且還是盡力去戰鬥吧。
我用一只手按著腦袋，用手指用力揉搓，拼命抑制住藥的副作用從而集中精神。
。
「শিখা　এই হাত炎啊 具象成球狀吧」
。
火焰在杖前亮起，但魔力分配怎樣不能順利地進行。
過多的魔力不管怎樣都讓火焰溢出，形成不了球狀。
。
「嘖！」
。
我更改魔法陣使其縮小，讓其在我面前爆發，順著沖擊往後退。
雖然後背著地摔在了地上，但總算是躲開伊卡洛斯的魔彈了。
。
「對、對待病人，這也太卑鄙了吧！」
「這根本不是決鬥！這樣的戰鬥別以為我們會認可啊！」
。
從觀眾們的一方開始出現不滿的聲音，並逐漸蔓延開來。
對我來說，剛才的突襲可以成為我的借口所以沒有多大的問題，但對於至今都徹底地籌集領民們支持的伊卡洛斯來說，剛才的行動的確是太顯眼了。
不僅以病人為對手，進行偷襲什麼的只會獲得不滿吧。
。
說起來，一開始伊卡洛斯看到我的狀況時，應該是沒有要戰鬥的意思的。
但在林古斯发話後，他決定了要繼續決鬥……難不成，其實是內心激動缺乏冷靜的，演技嗎？
。
做這種事只有一個好處。
這是為了對我造成過剩的傷害使我再起不能之後，從他人對自己的責難中逃離的布局。
伊卡洛斯認為，只要有我在自己的墮落就不可避免的話，打出這一手也不算奇怪。
。
實際上剛才在拉爾克的公館時，我也遭遇到某樣東西的襲擊。
無論是什麼都好，都肯定是跟伊卡洛斯有關系的吧。
在這種危急關頭襲擊我，之後還打算怎麼解釋。
。
「真是難看啊。看我用速攻魔術了解你」
。
在伊卡洛斯的左右，顯現出兩個魔法陣。
。
「শিখা　বরফএই হাত炎啊 冰啊 具象成球狀吧」
。
伊卡洛斯橫切般橫揮大杖，從兩個魔法陣中同時射出炎和冰的魔彈。
接著伊卡洛斯再次握好大杖，追在魔彈之後向我接近。
。
「伊、伊卡洛斯大人！不用那麼認真也……」
。
伊卡洛斯的其中一名部下大聲呼喊。
。
沒辦法了，雖然魔力的分配還不能順利地進行，用魔力填補吧。
畢竟也不知道何時會突然在我手頭上爆發。
既然做不到細微的調整，那就用粗暴的手段好了。
。
「শিখা　এই হাত炎啊 具象成球狀吧」
。
在我的杖前出現的火焰具象成了球形。
可能是控制得過於粗糙，火焰立刻就四分五裂了，但加在魔法陣上的魔力填補了這個缺陷。
又一次失去了平衡，因此又變大了一圈。宛如堆雪人一樣。
原本棒球大小的火球，最終變成了和我身高持平的魔彈。
。
「什！？」
。
伊卡洛斯發出驚嘆並停下腳步，向旁邊大跳一步。
因為魔彈是強行增大而處於接近破裂的狀態，增大被用於推進的魔力的量，勉強地把它發射出去了。
。
「去吧！」
。
把杖盡力一甩，巨大的魔彈切割著地面，往伊卡洛斯的方向飛去。
我釋放的魔彈輕鬆地吞沒了伊卡洛斯的兩個魔彈。
。
魔彈繼續削掉伊卡洛斯走過的地方，埋在地上停了下來，破裂並發出爆炸的聲音。
跳在空中的伊卡洛斯也受到其餘波，側腹摔在了地上，一身泥土地翻滾了十幾圈。
。
剛才還在喝倒彩的觀眾也變得鴉雀無聲。
。
「咳、咳呵！咳咳！」
。
在安靜的廣場上，回響起伊卡洛斯咳嗽的聲音。
在側腹著地時，身體內部似乎受到了損壞的伊卡洛斯，一邊咳嗽一邊摁住膝蓋，勉強地站了起身。
。
「……啊啊，意外地能行呢」
。
「伊卡洛斯大人啊！！」
。
在我自言自語的同時，伊卡洛斯的部下發出慘叫聲。
伊卡洛斯往部下們的方向看去。
。
「那、那個是打不贏的！果然是不行的！快投降吧！」
「再這樣下去會死的！」
。
伊卡洛斯的部下們臉色發青，左右搖頭。
伊卡洛斯確認了部下們那驚慌失措的臉色之後，回過頭看了看我魔彈削土前進的軌跡，和爆炸後挖開地面的痕跡後，他瞪大了眼睛。
然後，「哼哼、哼哼哼……」肩膀哆哆嗦嗦地笑了。
。
似乎是打算老實投降了。
看來伊卡洛斯並不是什麼需要我警惕的魔術師呢。
我緩緩地打算放下杖時，聽到了伊卡洛斯在喃喃自語地說著什麼。
。
「……決鬥是今天，真是太好了。平時的話，的確是沒有什麼勝算」
。
伊卡洛斯站了起來，重整體勢。
即使臉上流著粘汗，也嘴唇上揚地露出惡心的笑容。
似乎還想繼續戰鬥下去。
我重新架好放下的杖。
。
「真是出色的威力。哎呀，真是令人吃驚。論魔力的總量，你的確是比我高幾個階段」
。
伊卡洛斯保持笑容繼續說道。
不過，大概不是因為有餘力才笑得出來的吧。
從伊卡洛斯臉上溢出的粘汗順著臉頰，流到下顎並滴落在地上。
。
「如果是一流的魔術師的話……即使是在實戰中，也能同時展開三個魔法陣」
。
伊卡洛斯掀起大杖時，在他周圍顯現出四個魔法陣。
。
「但是我！只用魔彈的話，能夠同時展開四個魔法陣！」
。
伊卡洛斯一邊揮下杖，一邊吼叫。
。
「শিখা　এই হাত炎啊 具象成球狀吧」
。
從四個魔法陣中飛出的炎之魔彈，像描繪出半圓般地從不同的方向向我襲來。
……為了不被我的一发魔彈全部擊落，而打算分散攻擊嗎？
。
「再加上……這一发！」
。
伊卡洛斯向前一跳，再一次揮下杖。
第五发的炎之魔彈，筆直地沖向我的位置。
速度也好威力也好，在伊卡洛斯釋放的魔彈之中都是最高的。
。
「不、不愧是伊卡洛斯大人！在四重展開之後，居然還能打出追加的一发！放出這麼多魔彈的話，肯定也不會被一发抵消……」
。
伊卡洛斯的一名部下解說道。
我架著杖，在周圍展開十個魔法陣。
。
「……咦？」
。
狀況良好並且可以使用歐特姆的話，我有自信同時打出二十六发魔彈。
現在鬆懈制御的話也能打出三倍的數量，但打出那麼多也沒有意義，萬一走火的話就糟糕了。
十個就足夠了吧。
。
「শিখা　এই হাত炎啊 具象成球狀吧」
。
這一次我直接捨棄了細微的操作，從一開始就生成巨型的魔彈，然後一齊射出。
那十個魔彈一邊吞沒伊卡洛斯的魔彈，並扭轉方向向伊卡洛斯接近。
。
「哦、哦哦！？」
。
伊卡洛斯睜大雙眼，狼狽地往後退。
宛如幼兒不安時抱著雙親的手腕般，伊卡洛斯緊抱著大杖不放。
。
因為設定成在接近人類之前會撞在地上自爆，即使受到餘波也不會因此殞命的吧。
十個魔彈在伊卡洛斯周圍自爆了。
煙塵消散時，伊卡洛斯被燒成一塊黑炭。
。
「……已經，可以了吧？」
。
我向伊卡洛斯說話。
伊卡洛斯就像是不知道發生了什麼事一樣地目瞪口呆著，但和我相視時卻哄然大笑。
。
「哼、哼哼哼，哈哈哈哈哈哈哈！」
。
「…………」
。
我失聲啞然地看著伊卡洛斯的樣子時，伊卡洛斯停止了哄笑，用杖支撐著身體站了起來。
。
「怎、怎麼又笑了……」
。
「你怎麼可能懂啊我！我啊！終於成為了這個地方的王啊！在這個又小又無聊，垃 圾一樣的鄉下裡！拼命地向毫無關係的笨 蛋們示好！花上二十年以上的時間！終於要成為王啦！被眾人所畏懼的連彈的伊卡洛斯，就是我啊！知道為什麼嗎！」
。
伊卡洛斯翻著白眼，扯開嗓門叫喊。
我被他的迫力所壓倒，連話都說不出來地呆站著。
領民們似乎也是這種。
。
「就是你這樣的傢伙！我就是被你這樣的傢伙妨礙，前功盡棄，被迫淪落到這種邊境地區啊！我啊！其實是有機會侍奉更靠近王都的領地的大貴族啊！我也曾經有過啊！但是我拋棄了那些機會，選擇了在這種鄉下的地方工作啊！賭上了我的人生啊！」
。
伊卡洛斯精神失常，用拳頭打向地面。
伊卡洛斯的皮膚變得破破爛爛的，拳頭上泥和血混雜在一起。
。
「從二十到四十之間……我一直都在這個屎一樣領地上生活啊！然而，終於、終於有機會成為這個地方名副其實的王了，就是因為像你這樣的傢伙來到這裡！故意選擇這種邊境地方，就是為了不想讓你這樣的傢伙來這裡啊！事到如今才趕走我，那你讓我怎麼活！到底還想對我的人生做什麼！你以為我過去做了多少貢獻！你以為我是誰！我可是連彈的伊卡洛斯啊！我可不是在這種鬼地方腐爛的魔術師啊！沒有我這個領地……肯定是會更加荒廢的……！還給我……我才是王！我才是真正的王啊……」
。
伊卡洛斯一邊哭喊，一邊跪倒在地。
廣場被沉默所包圍。
在我放下杖打算後退的瞬間，伊卡洛斯的眼睛突然一閃。
伊卡洛斯猛然抬起上肢，把杖指向我周圍的地面。
。
「বায়ু　সাপ হাত土啊 化為毒蛇吧」
。
泥土開始變形並具象成蛇形。
雖然只不過是用泥土做成的土色皮膚的蛇，但只有牙的部分放出紫色的怪異光。
。
「咕！」
。
即便是現在的身體狀態，在這個距離使用魔術的話我也能在發動之前抵消掉，但一瞬間在腦內掠過了『不能使用魔彈以外的魔法』的規則，因此判斷遲了一步。
這樣的話，現在只能用不能穩定制御的魔術，射穿那只動來動去的小蛇了。
。
「哈哈哈哈！這樣，這樣你就死定了！這個領地是我的囊中之物啦！」
。
「বায়ু　তীর হাত風啊 具象成箭矢吧」
。
從我杖中出現的光與風纏繞，化成一只純白的箭矢並筆直地射出。
風之箭矢確實地貫穿了毒蛇的鐮刀狀脖子的顎下。
……連同在延長線上的，狂笑中的伊卡洛斯的腹部。
。
「啊」
。
這短短的一聲，也不知道是誰发出來的。
我也吃了一驚。
有可能是我自己发出來的聲音，也有可能是伊卡洛斯发出來的，又或者是某個觀眾的聲音。
總之，隨著這短短的一聲，決鬥也畫上了終止符。
這也是我和伊卡洛斯的權力鬥爭的結束的瞬間。
