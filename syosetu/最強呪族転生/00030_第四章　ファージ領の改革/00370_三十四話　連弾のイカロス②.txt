到了和伊卡洛斯決鬥之日，我在床上抱頭昏睡。
。
說實話，的確是有感冒的征兆。
從行使喚云魔術，被大雨淋濕時開始，就感覺到身體有些不適。
之後，又連日帶夜地開發歐特姆瓜。
雖然服用了亞貝爾藥水而肆意地驅使身體，但在藥效一過的那一瞬間，積聚的疲勞就一口氣壓垮了身體。
。
眼睛深處傳來一陣陣的疼痛。
就像是神經被損耗一樣的，陰濕的疼痛。
這份痛處還攜帶著熱量，一直在燃燒我的大腦。
。
並且還有某種物體即將從胃底湧上來的感覺。
形形式式的痛苦，從多個角度侵害著我的身體。
。
「……」
。
我睜開眼皮，向旁邊一看。
視線一瞬間變得朦朧，然後再慢慢地對焦。
美亞那擔心地注視著我的臉龐映入眼中。
。
「……亞貝爾，你醒了呢」
。
我在早上第一次起床之後，因為實在是感覺到不適，沒有吃早飯就直接回自己的房間睡覺了。
。
「沒、沒事吧？額頭……很燙啊」
。
「……好痛苦，要死了」
。
「請、請不要放棄！亞貝爾死掉的話……美亞、美亞也……」
。
美亞翻倒圓凳並站了起來，緊握住我的手。
。
「那個，決鬥呢？」
。
「現在不是說這個的時候！早上的湯好像還有剩……喝不下嗎」
。
我輕輕點頭。
即使現在吃下什麼，都會連同胃液一起在身體裡翻滾。
雖然對不起拉爾克先生，不過現在的確不是決鬥的時候了。
。
突然，立在美亞坐著的圓凳旁邊的弓出現在我的視野中。
。
「…………」
。
「啊，這、這個請不必在意！」
。
「啊，啊啊，這樣啊……」
。
伊卡洛斯提議採用維爾納瑟式決鬥的理由，我大概明白了。
詳細調查過後，決鬥規則本身是沒有什麼疑點的，因此完全搞不懂伊卡洛斯為何這麼執著的理由。
但在準備階段，必須進行一天的精神統一的規定有些許可疑。
伊卡洛斯瞄准的可能就是延遲的這一天。
。
恐怕在這期間，對我施下了讓病魔活性化的詛咒了吧。
不對，我完全察覺不到這件事。
大概是一開始就把已經活性化的病魔扔到我那貧弱的身體上了吧。
要不是這樣，感冒也不可能會這麼痛苦的。
我感覺比上次得感冒的時候，還要嚴重好幾倍的樣子。肯定是這樣。
這一定是伊卡洛斯的陷阱。
。
早知道就不應該輕易地接受敵人提出的規則。
我好恨當時的自己。
以為無論發生什麼事，手上有杖的話總會有辦法解決的，有可能是對自己的魔術過於自信了。
真正的惡意並不是那麼簡單的事情。
。
「拉爾克先生……怎麼說……」
。
無視拉爾克的忠告，以為自己穩操勝券的那個人是我。
一起努力到現在，結果卻是以這樣的結局收場，真的很抱歉。
。
「沒事的，不用擔心！領主先生也說亞貝爾的身體才是重要的……沒有必要勉強自己。好好休息吧」
。
「……拉爾克先生」
。
這時，我聽到了敲門的聲音。
。
「不好意思……那個，亞貝爾大人已經醒來了吧。可以進來嗎……」
。
是尤里斯的聲音。
美亞站了起身，挺直腰背並拿起弓，向門的方向走去。
。
「等一下美亞……」
。
「不可以！亞貝爾現在只需要靜養！明明早上都說了那麼多次了，為什麼你們還不明白！」
。
「可是！可是！明明已經走到這一步……今天不出場的話，伊卡洛斯又會重新壯大勢力的啊！如果是亞貝爾大人的力量的話，現在至少還有幾分勝機吧……」
。
忽然，門被稍微地推開了。
那空隙的旁邊的牆壁，被銳利的箭矢刺入著。
尤里斯嚇了一跳，慌忙地後退幾步。
。
「亞貝爾，亞貝爾都這麼嚴重了，伊卡洛斯伊卡洛斯什麼的……那種事情怎樣都無所謂把！領地和亞貝爾，哪一邊更重要呢！對領主來說，第三者的性命變成怎樣都無所謂嗎！下次不會打偏了！請離開這裡！」
。
「可可、可是……」
。
「快點關門！美亞說過下次不會打偏的！這不是在開玩笑的！」
。
美亞似乎完全地血沖上頭了，滿臉通紅的樣子。
壓著箭矢的手也在不斷顫抖。
。
「冷、冷靜一下美亞小姐！」
。
「STOP！STOP！總之先放下弓好嗎！咳咳！」
。
扯開嗓門時，剛好卡住喉嚨而咳出聲了。
接著，頭痛的強度也增強了一層。
。
「啊……庫……咳咳！咳咳！」
。
「亞貝爾，發出那麼大的聲音會……」
。
美亞回頭看向我時，眼睛睜大，瞬間停下了動作。
然後手也停止了顫抖，立刻射出了箭矢。
箭矢穿過了我的正上方，穿過窗戶並射中了某樣東西。
。
「嗚！」
。
傳出了高亢的、短短的悲鳴。
窗簾外面，似乎有什麼東西在潛伏著。
這裡是二樓。我不認為這是正常人會通過的地方。
考慮到現在的情況，我肯定那是瞄准了我的性命的某人。
。
我立即拿起床邊的杖，指向窗外。
。
「বাতাস　এইহাত風啊具象成球形吧」
。
狂風呼嘯並形成一個球體，緊隨箭矢向窗外的某人發起追擊。
窗戶的玻璃碎裂，在下一瞬間連牆壁也被瓦解，因為反作用力所以我也從床上掉了下來，在房間裡的東西也瞬間變得七零八落。
。
「果、果然是控制不住嗎……不過，應該是打中了」
。
「亞貝爾！振作一點！尤里斯小姐，可以借用其他房間的吧！」
。
美亞扶著我，眯起眼睛看向尤里斯。
。
「果然，還是能上的吧……」
。
尤里斯小聲地說著並走進房間，從牆上開著的大洞中觀察外面。
。
「……誰都不在。我的確也感覺到了窗外似乎有某種東西」
。
「誒？」
。
我抬頭看向房間裡的大洞。
剛才的確是有用風魔術擊中窗外的人的手感。
雖然威力不穩定，但被直擊的話也不可能會平安無事的。
。
「怎、怎麼會，剛剛明明……」
。
從走廊處傳來啪塔啪塔的接近我們的腳步聲。
往那一看，一名少女從門外探頭窺視著我們。
是女傭瑪利亞斯。
。
「沒、沒事吧！？那個、剛才發生了……」
。
「……啊，沒事，可能只是搞錯了」
。
「搞、搞錯了……？」
。
瑪利亞斯用無法理解的眼神，對比了我和房間的慘狀。
。
「……瑪利亞斯，你過來這裡也就是說，拉爾克大人那邊……」
。
尤里斯詢問道，但瑪利亞斯遺憾地搖了搖頭。
。
「領主大人提議了延期。可是，領民們反對的聲音實在是太大了……。雖然領主大人也很高興，終於可以趕走那個魔術師了……但是也到此為止了呢」
。
尤里斯失落地垂下肩膀。
然後慢慢地回頭面對我。
。
「那、那個……真的是沒辦法了嗎？」
。
很抱歉，身體已經沒辦法自由活動了。
意識也是岌岌可危。
魔術也幾乎是控制不住，大概也沒多少威力。
只是打破牆壁這種程度，在魔術師的對戰中是毫無意義的。
被其他魔彈壓制住就已經結束了。
大腦現在完全沒有集中力，同時展開連射或魔法陣的個數也受到限制。
說白了就是大不了。
。
「請不要讓我說這麼多次！受到亞貝爾恩情時明明是那麼地恭恭敬敬……情況變得稍微惡劣的時候，不管生死也要讓亞貝爾工作嗎！」
。
「嗚咕……」
。
「這也是沒辦法的事情尤里斯大人。我們並沒有訴說無理的請求的資格。即使失去了這次機會，下次也並不是沒有機會的……」
。
瑪利亞斯似乎也明白我不能動彈的事情，贊同美亞的意見並勸說著尤里斯。
尤里斯緊緊地咬著嘴唇，闔上雙眼似乎在迷惑著什麼，但很快就睜開了眼睛，宛如驅散心中的迷惑一樣搖了搖頭。
。
「我明白了……」
。
看來尤里斯終於是放棄了。
。
「……如果你出場決鬥的話，不問勝負，我都會向領主大人進言，建議領主大人通過之前亞貝爾大人的那些被駁回的生體魔術的使用許可的申請書」
。
……嗯？
。
「……但是那些申請，不可能通過……」
。
「領主大人應該也認為，即使亞貝爾大人勉強自己，也只會以失敗收場。瞄准這一點，就有可能通過那些申請了。我也會幫忙的，因此……」
。
……這是，機會嗎？
至少也可以為說服拉爾克鋪好了一條路。
安全性的話無須擔心，原本就是打算進中求穩的。
在此之上踏實地尋求妥協，再歪曲國法的解釋，行使與違法擦邊的魔術……。
。
「太煩人了尤里斯小姐！美亞，已經生氣了哦！」
。
「不、不行！只有那個魔術許可的申請，絕對不行！領主大人也絕對不會允許的，要是真的通過的話就很不妙了！」
。
我喝光了因魔術的反沖而翻倒在地的亞貝爾藥水，用力地敲了幾下自己的腦袋。
視線變得天旋地轉，頭痛也加深了一層。
吐意一口氣地湧了上來，不過總算是勉強地壓制住了。
。
三半規管總是在顫抖。
感覺好惡心。已經不只是惡心這個程度了。
腳跟搖搖晃晃，就像快跌倒一樣，美亞驚慌失色地扶著我。
。
「不、不可以！今天就好好休息吧？好嗎？明明連站都站不穩了……」
。
「我要去……決鬥……」
。
雖然我只能看到模糊的人影，我用沙啞的聲音對貌似是尤里斯的人影說道。
。
「……十分感謝。那麼，我們走吧」
。
尤里斯完全從別的方向回答我。
眯著眼睛一看，我發覺一直以來我看到的人影其實只是只歐特姆。
