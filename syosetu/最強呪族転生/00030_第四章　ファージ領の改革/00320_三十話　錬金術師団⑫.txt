「感谢大家在百忙之中抽空参与这次发表会。这次是由我，莉诺亚·利贝尔特带领的，鍊金术师团副队对开发的新品种农作物——欧特姆瓜的进行发表，为此而请领主大人準备好这个地方」
。
莉诺亚作为代表走在前面，扬声说道。
但对於莉诺亚的开场白，领民们的反应似乎不是很友好。
。
「副队……不是为了隔离鍊金术师团裡的累赘们才组成的队伍吗？」
。
前排的某个微胖的男人，向旁边的男人确认。
。
「毕竟不知道内情我也不好说，不过也见不到伊卡洛斯团长的身影……果然是有什麼内幕呢」
。
副队是为了疏离碍事的人而设立的队伍，这也许是真的。
不过真正的事实却是『伊卡洛斯认为是碍事的人』所组成的队伍。
把成功率极低的唤云魔术工作扔给莉诺亚一派的事情，就是最好的证据。
。
骚乱逐渐扩大，莉诺亚的开场白也不知道能传达到何种程度。
。
「——以上，我的开场白就结束了」
。
莉诺亚从中途就开始自暴自弃地，在开场白结束时流露出焦急而又厌烦的表情。
。
「喂伊卡洛斯大人在哪！喂！」
「那个作物真的是成果吧！要是骗我们在出结果之前还得花上一段时间的话，我可饶不了你哦！」
。
莉诺亚再行一礼，後退到了其他魔术师们的位置。
。
「莉诺亚小姐，您辛苦了」
「……那麼按照计划，後面就拜托你了」
。
接替精疲力尽的莉诺亚，我抱着欧特姆型花盆来到前面。
我把欧特姆放在地面上。
。
「现在你们看见的欧特姆瓜，只要调整环境的话，只是一个小时就可以结出果实了！虽然还有不少缺点，但撑过领地的危急是肯定没问题的」
。
说毕，方才还在喧哗的领民们的其中一部分回归平静。
然後又开始对旁边的人叽叽咕咕地说话。
。
大声诉说不满的人们也察觉到流向的改变，向周围的人询问發生了什麼事，寻求解释。
大概，是听不到我说的话吧。
。
「……你刚刚，说什麼来着？」
。
「一、一小时就可以结出果实」
。
「你确定不是一星期？如果是那种植物的话我倒是有听说过。不过似乎由於果实太小，完全没有实用性……」
。
所有人都顿时安静了下来，但那只是一瞬间的宁静而已。
马上就有人扯开嗓门，内容也变得过激。
。
「别、别开玩笑了！那种农作物怎麼可能有嘛！」
。
「所以，我就从零开始做起……」
。
「那个白毛，不是传闻中的魔术师吗！做出来什麼的，你不是前幾天才来到这裡的吗！」
。
「看啊，我的眼睛下面！形成黑眼圈得花多少时间……」
。
「别瞧不起人啊！」
。
不由自主地打算反驳，然而并没有什麼效果。
对方数量太多，根本对付不过来。
準备具体地回答的时候，下一句谩骂便扑了过来。
虽然在对待拉尔克的时候就已经知道，不过这裡的人们是真的没什麼礼貌。
毕竟长期被囚禁在不安稳的环境之中，说不定是积聚太多的压力了。
。
「林古斯先生说得对！突然出现的纳尔加仑讨伐和魔兽驱逐的好消息，都是为了抬举那个小鬼来抑制我们的不满而散播的谣言啊！包括这次的事情，肯定有什麼内情吧！」
。
林古斯……？
啊啊，就是那个传教者吗。
在大声吼叫的男人附近，我发现了在慌张地辩解的林古斯的身影。
。
「啊……不，wwo，我没有这麼说……」
「为什麼啊！事情如果真相大白的话，现在就表明立场不是更好吗！」
「冷、冷静一下！你在说什麼，我根本不懂！嘘—嘘—！」
。
林古斯露出谄媚的笑容，把食指抵在嘴唇上，拼命地说服他的手下。
流着冷汗，不断地眨眼示意。
。
虽然也知道他形迹可疑，果然就是他散播着领主的坏话。
拉尔克似乎也没有听漏刚才的对话，观望着林古斯的那一边。
对领民的心理安定和水的分发等等，因为对领地的安定有一定的作用所以才允许他的传教行为，但在背地裏贬低领主的话，就是个大问题了。
如果刚才的话是事实，只是驱逐出境也算是温柔了吧。
。
我取出欧特姆瓜的种子，撒在欧特姆型花盆裡。
。
这样下去会發生暴动的。
赶紧展示成果给他们看吧。
不管是不是说谎，要是亲眼看见欧特姆瓜快速生长的话也只能心服首肯了吧。
。
「হত্তয়া生长」
。
我挥下杖时，从欧特姆型花盆的中空的头部处，逐渐延伸出藤蔓。
藤蔓离开花盆，垂落在地上，长出新的叶子，并且用肉眼可见的速度不断膨胀。
。
这次，领民们终於一齐安静下来了。
某些人目瞪口呆，某些人保持着愤怒地举起右手乱挥的姿势。
是眼睛的错觉，还是在做梦呢。
似乎是不敢相信眼前的光景，接受不了事实的样子，领民们失声哑然了。
。
「诶……像这样，用魔力促进成长的话，不到一会就能结出果实了」
。
在被沉默支配的广场中，我的声音比想象中还要突出。
让我产生了我的声音突然变大的错觉。
。
在我说完的同时，欧特姆瓜的开始开花了。
。
「「「哦哦哦哦哦哦哦！！」」」
。
在领地裡回响起了近乎尖叫声的大大的欢呼声。
在欢呼声之中，欧特姆也在继续成长。
大大的花朵，不断地展开花瓣并开始吞食周围的杂草。。
。
「像这样子，连补充营养和驱逐害虫都能同时做到哦」。
。
「好厲害，这农作物是啥！」
「这还能叫做农作物吗！？」
「这玩意的出现可能颠覆世界的常识啊！」
。
形势一变，领民们的称讚如暴风雨般。
也不是说完美无缺，不过處於兴奋状态的领民们似乎没有在意有缺陷的地方。
事情朝着最好的方向发展呢。
。
「这个骚动是怎麼回事啊！！」
。
抵消掉欢呼声的怒吼，从领民们後面传来。
领民们被那吼声吓到，慌忙地让开了道路。
在那中央，一名壮年的男子毫不客气地通过。
是伊卡洛斯。
伊卡洛斯满脸通红，青筋暴露，看来是勃然大怒了。
。
一直以来，对自己百依百顺的拉尔克发起了明显的攻击，这似乎让他感到不愉快。
拉尔克为了束缚伊卡洛斯的行动，把不明所以的工作交给了他，看来是已经被他识破了。
他还听说拉尔克隐瞒自己暗中召集领民，察觉到拉尔克肯定是打算用某种办法来加害自己了吧。
伊卡洛斯之後，鍊金术师团的伊卡洛斯派的魔术师们也陆续跟随。
接着，笔直地走向我的方向。
可以啊，在这裡决一胜负的话我奉陪到底。
。
「শিখা燃烧吧」
。
我挥下杖，将欧特姆瓜烧毁了。
对於我的行动，领民们显现出莫大的动摇。
。
「亚贝尔……？」
。
美亚不安地对我表示关心。
。
「没事的。现在还不是让伊卡洛斯看到欧特姆瓜的时候」
。
为了让伊卡洛斯完全地落套，顺序是绝对不能乱的。
毕竟伊卡洛斯是个老奸巨猾的大叔，从正面进行攻击也只会被强词夺理。
那样的话就先等待伊卡洛斯的攻击，状况一变再扭转局势，用魔术把他打趴下是最简单的方法。
即便是伊卡洛斯，在这麼多的人面前把之前说过的话改口沓舌，也肯定会失去权威的吧。
。
「领主大人……这次的事情，能给我好好解释一下吗？」
。
伊卡洛斯说完，後面跟随着他的魔术师，朝拉尔克晃了晃手上的书类文件。
那估计是拉尔克用来糊弄伊卡洛斯的书类文件吧。
。
「这个集会，到底是怎麼回事。还真是有趣呢，这是在搞祭典什麼的吗？如果真是这样的话，用姑息的手段把长年为这片领地鞠躬尽瘁的我排除在外什麼的，不觉得太见外了吗？领主大人？」
。
伊卡洛斯刻意地环顾四周，然後才看着领主。
瞳孔大开，连鼻子都鼓了起来。
。
看来是完全生气了。
即使是身在远处，我也感受到了他的怒气。
贾伽说过伊卡洛斯经常用怒发冲冠的演技来打乱对话的步调，不过看样子这次是本色演出了。
。
「哦呀，哦呀哦呀……这不是亚贝尔吗、嗯嗯？我们又见面了呢」
。
伊卡洛斯弯下腰，歪着脖子，目眦尽裂地瞪着我，对我施加威压。
我也无言地回瞪着伊卡洛斯。
只要解决掉伊卡洛斯这个领地的毒瘤，向领地改革的成功就更进一步了。
似乎是长年在这裡肆意妄为的样子，但伊卡洛斯也差不多是时候从舞台上下台了。
