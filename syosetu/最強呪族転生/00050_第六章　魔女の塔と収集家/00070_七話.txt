和佩特罗道别后，我按照原定的计划，与梅娅和马车夫艾丽娅一起离开了帕尔加斯村。
目的地当然是领主拉尔克那儿……骗你的，其实是传说的炼金术师雅尔塔米亚沉眠的魔女之塔。
。
魔女之塔位于珐基领海岸线附近，是珐基领最最偏僻的地方。
在原本就是乡下的珐基领中，魔女之塔的位置处于离帕尔加斯村这个最边境的村落还要远的地方。
据闻魔女之塔中全是异态的魔兽和恶魔，而且四处都设置着恶毒的陷阱，仿佛是魔女对封印自己的王国的怨恨化为实体一样。
。
魔女之塔总共有八层，每上升一层，和魔女间的距离就会缩短一层，因此阶层中出现的魔兽也会变得更加狂暴。
一般的冒险者最多只能到达第三层，能通过第四层的人屈指可数，登上第六层的人似乎仍未出现。

『有人曾登上过第六层，只不过没能活着回来而已』虽然也存在这种说法，但这种说法更令人毛骨悚然。
。
原以为艾丽娅会拒绝这个委托，但在听完魔女之塔的概要后思考了数秒，她当场答复「……请交给我吧」。

我也隐隐约约地察觉到，这个人意外地富有挑战精神。
。
「客官，我想，我们到『那里』了」
。
从御者台的方向传来了艾丽娅的声音，于是我掀起帘布，探头眺望马车外的景色。
。
以海为背景，一座巨大的砖制圆塔拔地而起。
那不只是粗糙的土砖，而是混入了秘银等高价魔矿石而制成的特制砖。
说不定是用于封印雅尔塔米亚的结界的媒介。
外壁上刻印着数个术式，营造出令人敬而远之的氛围。
。
为了封印雅尔塔米亚，王国仿佛燃起了近乎病态的执念。
靠这种显而易见的大型陷阱，真亏他们能捕获雅尔塔米亚呢。
大概是捕获之后再在这里进行封印的吧。
。
「不过……那座塔一看就知道隐藏着不少的宝物。而且也有不少调查的价值呢」
。
我一边在手边捣鼓着拉彼德斯之剑的剑柄，一边满怀期待地眺望着魔女之塔。
梅娅也从我拉开的缝隙中探出头，发现魔女之塔后便由衷地发出感叹。
。
「話说回来……亚贝尔，这次想上到第几层塔呢？」
。
「嗯？啊啊……嘛，在不受伤的前提下，能上多少层是多少层，在适当的时候放弃，我是这么想的哦」
。
「……到最后都没受伤的話呢？」
。
「到最后……我想想……」
。
虽然我对塔的事情不太了解，但雅尔塔米亚恐怕就被封印在塔的第八层。
打开通往第八层的大门后，雅尔塔米亚的封印便会解开……也不是没有这种可能。
真是这样的話，那我至今为止累计的功绩都会成为一张白纸，而且必然会被王国指名通缉。
将来的我也许会以珐基领为据点，率领着泣不成声的拉尔克，开始与迪恩拉特王国的全面战争。
。
再怎么说，我也知道『可以做的事』和『不可以做的事』之间的区别。
就算有多么在意都好，一出现什么奇怪的机关就立刻折返。
即使能速战速决地向上进发，我也相信自己拥有着在第七层忍住好奇心并返回帕尔加斯村的自制力。
。
「别看我这样，其实我也有点到即止的打算哦」
。
「诶！？」
。
梅娅发出奇怪的叫声，发愣地张着嘴。
老实说，我也没想到她会这么吃惊。
。
的确，我是很在意雅尔塔米亚。
在百年前被封印的魔女到底拥有着怎样的力量，到底有多么强大，到底在研究着什么，问我在不在意的話我肯定是在意的。
可以的話真想和她促膝长谈一整晚，然后偷偷地把她带回来，但这不仅是对拉尔克的背叛，而且怨恨着王国的雅尔塔米亚也不知道会干出什么傻事。
。
而且莎姆才刚叮嘱过我，让我千万别把梅娅带到危险的地方。
先不说一般冒险者也能轻松通过的低阶层，但怎么想我都不可能把梅娅带去雅尔塔米亚本人那里吧。
。
在帕尔加斯村中，由于我的玩忽职守而让梅娅暴露在危险之中的情形时有发生。
玛利亚斯和好尼禄古力夫都是些狡猾而不能疏忽大意的人物。
……至于他们的魔术本领，嗯……嘛，也不一般。
虽然比一般的冒险者要出色些许，但借用水神利维伊之力后却只有那种程度的实力，不禁令人心生质疑。
。
总、总而言之，以后还有可能和利维伊教那边的人接触。
我只是在设想最坏的情况，防范于未然。
。
「你没事吧亚贝尔？是发烧了吗？还是说感冒了么？」
。
梅娅担心地把脸凑过来，用手贴着我的额头来测量温度。
。
「没、没事啊……怎么了？我说了什么奇怪的話吗？」
。
「看来没有发烧呢……」
。
梅娅放心地松了口气，把脑袋缩了回去。
接着她偷偷地瞄了一眼我手上的书。
。
「毕竟亚贝尔那么地热衷于有关魔女之塔的事，半途而废什么的根本不可能嘛……。平时的話，说不定会宣称要把整座塔带回家呢」
。
「……原来，你对我的印象是这样的吗」
。
我苦笑着翻开手上的书，低头阅读。
。
——————————
封印着被称为『灾厄之魔女』的炼金术师雅尔塔米亚的塔。
尽管外表上只是一座没有窗户的巨大圆柱，但内部的空间早已扭曲，各个阶层都有差异，深不可测。
明明身处于在塔内，探索者却会产生自己正在游历各地的错觉。
但无论是多么美丽的风景，都同样是拜雅尔塔米亚的魔力和执念所赐的幻想。
每上升一层，冒险者们也会慢慢地发觉，雅尔塔米亚的恶意正在悄然接近的事实吧。
(艾德纳·亚尔巴克著作『迪恩拉特王国迷宫排名前20』）

——————————
我从帕尔加斯村那里借走了几本有关雅尔塔米亚的资料。
这就是其中的一本。
。
如果这段内容是真实的話，雅尔塔米亚也许早已超越了人类，成为了与恶魔和精灵相近的存在。
扭曲魔力场，对空间造成影响并创造出特殊的魔物，这是恶魔的拿手好戏。
倘若她的确可以制造出能够在塔内具现各种景象的魔力场的扭曲的話，她确实是个不负大恶魔之名的怪物。
。
「…………」
。
「果然还是想上去的吧？不用太在意梅娅的哦？要是嫌梅娅碍事的話……梅娅和艾丽娅小姐一起在外面等就是了……」
。
也许是看穿了我表情上的依依不舍，梅娅向我提议。
从后半句开始，梅娅的声音便逐渐衰弱，显得有些寂寞。
她蜷缩起肩膀，失落地低着头。
。
「没、没有这样的事……」
。
艾丽娅从前面窥视这边的情况，用轻蔑的眼神盯着我，仿佛是在说『啊啊，又把梅娅弄哭了』的样子。
我立刻转移视角，正对着梅娅。
。
「没、没有这样的事，我也想和梅娅一起去啊！」
。
「但是，梅娅一点用处都没有……」
。
梅娅的眼角上泛起了一丝泪珠。
。
毕竟梅娅的最近的情绪相对稳定，所以我也没多作留意，但梅娅的心灵的确很脆弱，有时就像是突如其来的波浪一样突然发作。
。
「梅娅在我身边，我就有种轻松的感觉呢……也就是说，可以帮我吗，搬行李之类的……！」
。
「真的吗！梅娅派上用场了吗？」
。
梅娅终于破涕为笑。
这我就放心了。
看了眼艾丽娅那边，我发现艾丽娅并没有看着我，而是眯着眼睛注视着梅娅，就像是在问『这样就满足了吗？』
