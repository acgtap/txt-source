于迪恩拉特王国边境的某地，一名虎背熊腰的男人正独自一人地踱步着。
。
他披着庄严的青色外套，外套上刺绣着以金丝为素材的魔法阵。
在外套内也穿着以五颜六色的美丽布匹编织而成的羽织，散发着王者之风。
除此之外，他的身上也装饰着不少高价的魔金属制腕轮、指环和耳环。
隐藏在严实的外套深处的脸庞和手臂都被绷带包裹着，没有人能知晓他的真面目。
虽然是从远处就一目了然的奇怪而又显眼的打扮，但也许是在害怕着这个人，魔兽们不敢枉然接近。
。
然而有五名剑士装束的人物正打算靠近这个奇装异服的男人。他们五人都骑着马，全身上下的装备都一模一样。
他们正是迪恩拉特王国的第一王子，阿尔冯斯的部下。
。
青色外套的男人停下脚步，把视线落在他们身上。
那五人组在靠近青色外套的男人后从马上下来，站在男人的面前。
一名紫绀色长发的美男子走在前列，扯开嗓门宣言。
。
「我们是第一王子阿尔冯斯大人的亲卫队。我的名字是伦迪奥。看来阁下正是传说中的冒险者收集家。奉阿尔冯斯大人的命令，我们已经寻找阁下许多年了」
。
「…………」
。
号称收集家的青色外套的男人露出了平静的笑容。
。
——收集家。
亦即是被称为历代最强冒险者的人物。
经历了三百年以上的岁月，他仍在周游世界各地，并收集着各种各样的宝具。
。
「第一王子阿尔冯斯大人，为即将成为统治迪恩拉特王国之王的王太子。而阿尔冯斯大人现在直接指名了阁下。这是何等荣耀之事……」
。
「没兴趣。你觉得，现在的吾还会执着于地位吗？」
。
收集家终于肯开口了。
声音沙哑，并且令人反感。
。
「收集家，阁下有着形形色色的嫌疑。盗掘遗迹、破坏陵墓、谋杀、甚至于别国王宫盗窃财物……。阿尔冯斯大人也在担心阁下的安危」
。
「嚯？拒绝的話，就把吾杀了？在这里？就凭汝等？」
。
「并没有这个意思……倘若阿尔冯斯大人成为了阁下的后盾，即使被人如何质疑，也没有人胆敢谋害阁下的性命吧。这是场不错的交易吧？」
。
「嚯、嚯！原来如此，原来如此！」
。
收集家心情舒畅，快活地笑了笑。
。
「没兴趣！当今世上，还有谁能与吾匹敌！不过……再听汝等说一句也无妨。那个口出狂言，妄想雇佣我的笨蛋王子，到底为吾准备了什么？」
。
在伦迪奥身后待机的亲卫队骑士们对收集家的发言感到愤慨，纷纷拔剑。
阿尔冯斯王子身为迪恩拉特王国的下任国王候补，却被人污蔑成口出狂言之徒，这是何等的不敬。
尽管他们从一开始就对收集家的措辞感到反感，但这次他们终于忍无可忍了。
。
「**！不给你点颜色瞧瞧……！」
。
「住手。阿尔冯斯大人也明白，用一般的方法是行不通的。收集家阁下，阿尔冯斯大人当然准备好了与阁下相称的报酬」
。
伦迪奥劝解其他骑士后，向收集家回答。
。
「哼哈哈哈哈！好说！汝的意思是，现在就能拿出来？」
。
伦迪奥用眼神指示身后的骑士们。
亲卫队的其中一人提着巨大的箱子，放置在收集家的面前。
。
「唔姆唔姆」
。
收集家粗鲁地打开箱子，取出箱子里的宝剑端倪。
那是一柄刻入了某样术式的金碧辉煌的黄金细剑。并且在剑柄上嵌入了魔石。
。
「嚯」
。
收集家举起黄金之剑，以剑挡日。
。
「剑之名为，『黄金之女帝』。是对所有者的魔力产生反应，发挥其真正价值的魔剑。阿尔冯斯大人深知收集家阁下拥有多姿多彩的剑技和魔术，对在两个领域均得心应手的您来说，这应该是求之不得的宝物才对。请将此剑为阿尔冯斯大人而挥」
。
伦迪奥说完后，收集家往剑中注入魔力。
此时黄金之剑光彩耀目，照耀周围的闪光满满溢出。
包含伦迪奥在内的亲卫队员也忍受不住那炫目的光芒，连眼睛都睁不开。
。
(虽然听说过一些传闻……收集家……竟然拥有这么强大的力量。即使是让一流的魔术师注入魔力，『黄金之女帝』散发的光芒也没有现在的十分之一强烈……！阿尔冯斯大人虽然叮嘱过，倘若不顺从便灭口，但就凭我们五人似乎还不够。过早地将那把剑拱手相让，似乎是个坏的决定。这一束光……这把剑，现在究竟拥有怎样的威力呢……）
。
为了令交涉尽可能地按照自己的步调来发展，伦迪奥掩饰心中的焦虑，以微笑面对收集家。
。
「『黄金之女帝』为前两代的国王委托当时的宫廷锻冶师与炼金术师，不惜金钱和人力，为侍奉于国家的骑士所打造的宝剑。不仅拥有实用性，而且有着极高的历史价值。您满意了吗？当然，这只是冰山一角。阿尔冯斯大人为了收集家阁下，还准备了许多稀世的魔法道具哦」
。
阿尔冯斯认为，为了拉拢收集家，势必要用到那些可能值得收集家收藏的高价魔法道具或武器，于是便煞费苦心地准备了这把剑。
。
「唔姆」
。
虽然收集家陶醉般地眯着眼睛，但当他听到伦迪奥的发言后，他把黄金之剑换到右手，空出来的左手则往空无一物的地方轻轻一捏。
左手手中，突然出现一把比收集家的身高还要高许多的质朴大剑。
剑尖的一端的剑身，比剑柄的那一端更为宽阔，剑刃如新月般弯曲。
尽管迫力十足，然而从外观上看却欠缺了实用性。能在实战中运用自如什么的，令人难以想象。
。
(完全以无咏唱进行物质转移？大概是使用了某种魔法道具的力量吧……这种梦幻般的魔法道具，真的存在吗？可是，他到底要用那把巨剑做什么……）
。
收集家高举双手，以极快的速度让『黄金之女帝』和巨剑的剑身相撞。

『黄金之女帝』照耀出最大的光辉，火花四溅。
。
「呜咕！」
。
两把剑相撞的冲击，令伦迪奥他们无意识地弯下了腰，勉强地扎稳了脚步。
冲击过后，他们将用于抵挡沙尘的双手放下后，发现了断成两截的『黄金之女帝』的残骸。
收集家则不以为意地站在远处，踢飞了『黄金之女帝』的剑刃。
而手上的另一半也被他索然无味地扔在了地上。
。
「你……你……！」
。
「这把剑真是渣滓！不就是把只会发光的钝刀嘛！在听到是前两代的国王雇人打造的时候吾就有种不好的预感了！」
。
「竟然称『黄金之女帝』作钝刀……？你、你知道这把剑究竟拥有多少价值吗……！撤回你的发言收集家！」
。
「渣滓就是渣滓！这么想听吾就多说几次！渣滓渣滓渣滓！事实证明，这把钝刀不堪一击！吾怎么可能将这种没用的东西添加到收藏品里！这种玩意，就算吾我多少都好，都不够付一次跑腿费！不就是个一碰就碎的花瓶嘛！」
。
收集家粗鄙地放声大笑，将沙子踢到黄金之剑上。
。
「回去告诉你们的主人！用哥布林的骨头去养狗对汝来说才是最合适不过！」
。
「…………」
。
即使是伦迪奥，也忍受不住刚才的讥讽。
他眉头紧蹙，怒视着收集家。
收集家仍执拗地践踏着黄金之剑，扛起自己的大剑，向伦迪奥他们夸耀。
。
「話说回来，跟这根哥布林的骨头相比，吾之爱剑是那么地出色啊！看见了吗！哼哈哈哈哈！居然一刀就把它砍断了！如果能扛下三刀，吾也可以陪汝等再玩一玩，但居然一刀就断了！听好了！吾要让汝等知道，汝等爱不释手的这根哥布林的骨头，和吾之爱剑之间的差距有多大！这是前两代国王所打造的所以就有历史价值了，汝好像说过类似的話对吧？吾之爱剑是追溯到八百年前，于被称为剑之国的古代贝兰德拿王国的双胞胎王子诞生之际，因占术师的预言而倾注国力所打造的双剑之一！此剑剑名为『粉碎之右王』，乃将所触之物全部破坏的凶刃！这才是与剑之王相称的宝剑！」
。
收集家在赞扬自己的收藏品有多么优秀的同时，将阿尔冯斯所赠送的『黄金之女帝』贬低得一文不值。
亲卫队五人的杀气越来越重。
然而收集家并不在意，继续自己的演讲。
。
亲卫队的其中一人忍无可忍，向收集家反驳。
。
「像、像你这种没教养的野蛮人，怎么可能明白『黄金之女帝』的价值！而且，这把剑真正的价值，要靠持有者的魔力才能完全发挥！在手握『黄金之女帝』的时候，是你自己魔力太低了吧！就像半兽人的魔石一样呢！那块毫无品味的笨重金属块才最适合你……」
。
在亲卫队员说完之前，收集家将大剑砸向地面。
一声巨响，阻断了那名亲卫队员的話语。
。
「……蒙昧无知的凡夫俗子，不止打断了吾的演说，还胆敢贬低吾之爱剑！罪该万死！」
。
伴随着收集家的怒吼，大剑再次砸向了地面。
。
(已经不可能拉拢他入伙了……而且，阿尔冯斯大人准备的宝剑也被他折断，这不取下他的首级还怎么和阿尔冯斯大人交代！必须在他完全进入临战状态之前，先发制人！）
。
伦迪奥向其他队员们使了使眼色，然后把剑斩向收集家。
另外四人也迅速地分散，将收集家包围。
。
(最应该提防的……是那把，将黄金之女帝砍断的巨剑！尽管那把武器是如此巨大，但即使以我的双眼，也无法看清他的剑法。然而，就算拥有多么庞大的魔力，也逃不出人类的范畴！用包围战术围攻的話……）
。
以接近而来的伦迪奥为目标，收集家伸出了空着的右手。
。
「出来吧『毒大蛇 托尔克匹佟』！」
。
犹如从右手中发射一般，一条释放出灾祸之光的巨大锁链突然出现，迫近伦迪奥的位置。
。
「吓！？」
。
「哼哈哈！小心点哦！这可是连秘银都能腐蚀的死亡锁链哦！」
。
伦迪奥慌慌张张地打算用剑弹开锁链，但反而是剑的尖端粉碎，被打偏到别的方向。
而且，剑身从折断的部位开始迅速变色，眨眼之间就蔓延到剑柄那边。
。
「可、可恶！」
。
伦迪奥把剑扔向收集家并往地一蹬，横向平移。
然而锁链突然转弯，将伦迪奥扔出的剑弹飞，接着再次改变轨道追上了伦迪奥。
。
「……诶？」
。
「忘记跟汝等说了呢！『毒大蛇托尔克匹佟』里面封印着恶魔，是拥有意识的锁链！哼哈哈哈哈！」
。
毒锁打飞了伦迪奥，并顺势缠绕住他的身体。
。
「啊……啊……啊啊……」
。
不到一会，伦迪奥的身体便开始变色，逐渐被锁链吞噬。
紧接着，锁链将腐朽了的伦迪奥的身体撕成碎片。
。
「无聊！太无聊了！吾好不容易收集到的武器用在这群杂鱼身上，完全就是大材小用！给吾再反抗一下啊汝等凡夫俗子！」
。
「可……可恶！」
。
余下的四人一齐向收集家发起进攻。
收集家抬起手时，杀死伦迪奥的锁链便消失不见。但下一瞬间，刚才的毒锁以收集家为中心，犹如盘卷的毒蛇般再次出现。
在弹飞四人手中的剑的同时，用毒腐蚀他们的身体。
。
刹那间，他们的身体便染上了剧毒的颜色，在地上连滚带爬。
四人将胃里所有的东西一迸吐出，更有甚者，连内脏都吐了出来，一命呜呼。
。
「哈！真是群无聊的凡夫俗子！」
。
收集家将右手举过头顶。
一把弓出现在收集家手中。
。
「这就是诋毁吾之收藏品的下场！把吾之宝具的力量，深深地、深深地、深深地刻印在汝等的灵魂里，然后痛苦地死去吧！这就是汝等凡夫俗子唯一能做的，对吾之爱剑的赎罪！」
。
弓浮游于半空，离开了收集家的手边并移动到四人的上方，发射数十只纤细的光箭，将他们射成了马蜂窝。
。
收集家在尸体面前不愉快地冷笑了笑。
收集家最讨厌的，就是打断自己说話的人，以及诋毁自己的收藏品的人。
。
「时隔百年再访迪恩拉特王国，原以为会遇到些实力不俗的佼佼者，但人和事物不还是一成不变的渣滓嘛！只是孱弱的話还可以接受，但没想到会这么不愉快！要是魔女之塔也是这么没劲的話，那就是真的白跑一趟了！」
。
收集家眯起眼睛，眺望着坐落在远处的巨大圆筒形建筑物——魔女之塔。
。
「雅尔塔米亚这货，是个稍微难缠点的对手就好了呢。再这么下去，好不容易得到的武器也要生锈咯」
