幾經曲折才抵達的第四階層是森林型的迷宮。
長著五彩斑斕的樹葉的七彩樹漫山遍野。
然而對於已經爬了三個階層的我來說，這已經是迷宮探索的極限了。
。
這就是雅爾塔米亞的執念所創造出的惡毒而非人道的迷宮。
不可能那麼簡單就能攻略得掉。
也許，是我太小瞧這個迷宮了。
。
「…………」
。
我疲倦地依靠了背後的大樹。
美亞則挑起了將要滑落在地的我的肩膀。
。
「亞、亞貝爾！振作一點啊！」
。
「已……已經不行了。我好像，搞錯撤退的時機了」
。
「亞貝爾——！」
。
包圍著我和美亞的魔獸正虎視眈眈地往我們這邊靠近。
。
「咕嚕……咕嚕……咕嚕嚕」
。
龍首——附有翠綠色的堅固圓狀甲殻的魔獸。
其形猶如龍頭從龍身中分離，長出青蛙般的手腳，因此而得名龍首。
。
——————————
與那圓滾滾的可愛外表相反，龍首是種光是一頭就持有C級危險度的危險魔獸。
堅硬的體表，強韌的牙齒，以及擁有強大爆發力的雙腳。
近戰專精、甚至能吐出火焰的這種魔獸，通常會以大約三十頭為群體，成群結隊地行動，擁有著如噩夢般的特性。
一旦被龍首之群發現，隊伍全員的存活率便會變得非常渺茫。
無論是多麼熟練的冒險者，一旦被龍首之群包圍的話，也無法拿出正常的水平來戰鬥吧。
(引用：艾德納·阿爾巴塔著作『危險的魔獸』）

——————————
。
這魔獸我雖然曾經在書上看過，不過親眼一看的話還是挺滑稽，挺可愛的嘛。
在第四階層內行走的期間，不知不覺地就被這群傢伙給包圍了。
。
「不、不要過來！再過來，梅婭就要不客氣了！不想受傷的話就別靠近這裡！」
。
梅婭拉起弓，想借此牽制龍首之群。
然而一匹龍首仿彿在嘲笑梅婭的行動一樣，將長著大爪子的伸縮自如的手臂舉過頭頂，直沖向我們這邊。
梅婭松弦放矢。
但在龍首那靈巧的動作面前，梅婭的箭稍微地射偏了些許。
箭矢只是輕輕地擦過了龍首腹部的鱗片。
。
「啊，沒事……這群傢伙我會想辦法搞定的」
。
我打了個響指，我手上的世界樹歐特姆便沖了出去，用飛踢來回敬龍首。
。
「咕嚕嚕！？」
。
其他龍首瞬間勃然大怒，一齊向世界樹歐特姆發起猛攻。
接下來就是世界樹歐特姆的無雙時間了。
東一拳。西一踢。將龍首當籃球一樣扔來扔去。
即使被龍首噴出的火焰籠罩，世界樹歐特姆也能突破火焰並將龍首打飛，被打飛的龍首又撞飛了別的龍首，就像是在玩桌球一樣。
被打飛的龍首一下就砍倒了大樹，但並沒有就此停止，在地面上又滑行了數米的距離。
。
梅婭無言地欣賞了世界樹歐特姆的屠龍表演片刻，接著她如夢初醒般地回頭看向我。
。
「……亞貝爾這麼有精神，梅婭就放心了」
。
「沒，沒這回事……你看我，哪裡有精神了。腿、腿……快斷了……已經沒有回家的力氣了……」
。
我已經走不動了。
這裡的樓梯實在是太凶殘了。
。
「……相比起亞貝爾，龍首那邊的情況更慘哦。啊，快看，那頭龍首的尾巴都斷了」
。
「但我的腿都快變成木棒了」(原文俺は足が棒になりそうだ）
。
「那句話只是一種比喻而已哦」(足が棒になる，比喻腿酸，累得腿都直了）
。
就算體力耗盡，我也能正常使用魔術。
硬要說的話，就只是集中力有點散亂而已。
將歐特姆切換到自動模式，這種小事難不倒我。
。
「梅婭是真的很擔心亞貝爾的……」
。
梅婭鼓起臉頰，眯著眼睛盯著我看。
又、又鬧別扭了……？
可是，事實上我的腳也已經快不行了。
。
「抱、抱歉……不過，我……我真的走不動了……」
。
「……梅婭來幫你按摩好了」
。
「啊啊，每次都麻煩你了……」
。
還好，梅婭並沒有那麼生我的氣。
在我喘了口氣的時候，從梅婭身後的方向突然間又闖進了十多頭龍首。
是請來的增援嗎？或許是因為聽到打鬥聲才接近這邊的吧。
。
龍首之群中似乎混入了一只金黃色皮膚的龍首。
。
「……黃金龍首？」
。
黃金龍首是龍首當中的稀有品種。
比一般龍首更快，更堅硬，腦袋也更加聰明。
吐息的威力和射程也比一般的龍首高。
。
相應的，黃金龍首的皮膚不僅適用於防具的素材，而且據說能夠帶來好運，某些大商人還會將其剝制成標本用於裝飾，因此價格相當地高昂。
本來是百年不得一見的稀有魔獸……但這麼容易就讓我碰到一只，真不愧是雅爾塔米亞之塔。
果然是稀有魔獸的巢穴呢。
一想到上面的階層上滿地都是這種稀有魔獸，我就按捺不住慾望，決心不管發生什麼事都要爬上第七階層。
。
世界樹歐特姆向那十頭龍首的方向突擊。
在世界樹歐特姆打算踢飛正面的幾隻龍首的那一瞬間，龍首們忽然分成左右兩路，往兩個方向一哄而散。
見此，我慢了半拍才明白。
那十頭龍首並不是沖著我們來的，而是從什麼人的手中逃出來的。
。
世界樹歐特姆窮追右邊的龍首，在使出撞擊的同時，猶如追殺著龍首一樣，一塊小孩子那麼高的石塊高高地跳了起來。
。
石塊上有著如手腳般的粗略的分枝，全身上下都刻印著術式。
看來這並不是魔獸。
就是這傢伙在追殺著龍首吧。
。
石人偶佇立在我們面前，以怪異的動作迅速平移並來到梅婭面前，接著揮下手臂。
。
「嗚啊！」
。
梅婭被嚇得往後倒。
緊接著石人偶就被世界樹歐特姆狠狠地踩住了。
石人偶的雙腳扎進了地下，只有腦袋外露在地面上，腦袋上似乎有個表示著『一號』的記號混雜在術式之中。
。
眼看梅婭就要摔在地上。
我以靠著樹坐著的姿勢立刻跳了起來，慌張地接住梅婭，成為了梅婭的安全氣墊。
。
「疼、好疼……」
。
「亞、亞貝爾，對不起……」
。
一般來說這應該是男方帥氣地接住女方然後被女方道謝的場景，而淪落到被人道歉的境地的我實在是太丟人了。
我重整體勢，走近只露了個腦袋在地上的石人偶。
。
「這、這東西到底是什麼啊？」
。
「……格雷姆，似乎是亞種的格雷姆呢。老實說，我也不太清楚」
。
我慎重地靠近石人偶。
這東西能單方面地獵殺危險度C級的魔獸之群。
不深入考慮的話，危險度應該也有B級吧。
。
「……明明發現了稀奇古怪的玩意，亞貝爾卻不怎麼高興呢」
。
「等等，這種魔法陣的組合方式……和澤舒姆遺跡裡的，還有艾貝爾海德所使用的很相似。我猜這並不是雅爾塔米亞的產物。因此，和雅爾塔米亞無關，這裡還存在著想攻擊我們的魔術師」
。
正當我警戒著周圍的情況的時候，我察覺到了精靈們異常的變化。
不知位於何處的某人，在以這一帶附近為對象發動了魔術。
不出我所料，魔法陣忽然在我們的面前出現，從魔法陣中能窺探出某人的身影。
。
如波浪般的金色卷髮。
雙眸清秀而細長，右眼為祖母綠般的碧綠色，左眼卻為鐵銹般的深紅色，是傳說中的異色眼。
身上披著被魔石和寶石點綴的琉璃色的法袍。
是個異常勻稱的美男子。
。
再加上又長又尖的耳朵。
雖然從剛才的術式就能聯想得到，果然是精靈種呢。
外表上看似乎有二十五到二十六歲左右，但倘若是精靈種的話就得另當別論了。
年齡推測為外表的五倍到十倍。
。
托艾貝爾海德的福，我也算是見過精靈種，然而眼前的精靈種，魔力的性質和氣場都明顯和艾貝爾海德不同。
就像是完全不一樣的生物一樣。
只是被盯著就讓人情不自禁地挺直了後背。
。
精靈種雖然擺出了一副十分嚴肅的表情，但在看到我和梅婭之後便放下戒心，露出了溫和的笑容。
我也不由自主地陪著他笑了起來。
。
一時間還以為他會做出什麼事，看來對方並沒有惡意呢。
這個地方已經明文規定禁止入內。
想不到這裡居然還有其他人存在。先前的攻擊就當做是事故吧。
我都不知道，除了我和梅婭之外還有其他來訪者。
。
接著，精靈種看了眼被埋在地下的石人偶，然後優雅地捂著嘴笑了笑。
。
「還以為被什麼魔獸給絆倒了，什麼嘛，原來只是兩名通常種嘛」
。
「……嗯？」
。
我和梅婭都不是通常種。
我是瑪連種，梅婭則是多姆種。
可能在精靈種看來，精靈族以外的人類種都是通常種吧。
。
就算我再怎麼笨，也能從剛才的那句話中推斷，這位精靈種並不友好。
。
「哎呀，真不爽。感覺真不爽呢。區區通常種，竟然敢將我的石從者打倒」
。
精靈種這麼對我說。
他的臉上雖然掛著笑容，但仔細一看的話，那並不是該用於觀察人類的笑容。
反而像是發現了奇珍異獸般的笑容。
這廝明顯是在小看我們。
