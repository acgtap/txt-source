
和尼禄古力夫分别之後，我们造访了海尔·珐基村长的公馆。
。
由於海尔村长在病倒之前，坚持将利维伊教徒的无偿治疗称为「这只是改宗的工作罢了」，因此在村内被孤立，教徒们讨厌他的话周围的人当然也孤立着他。
而现在的状况是，只有海尔的女兒芙露露独自一人照看瘫痪着的他。
佣人们可能认为，如果被牵连进去，连自己也接受不到利维伊教徒的恩典的话就糟糕了，於是也纷纷离开了吧。
。
「亚贝尔大人和美亚大人……感谢你们的到来。只是，你们两人现在是村子裡的名人，我认为不要经常来这个地方比较好」
。
在客厅中，我们隔着桌子和芙露露面对面交谈。
虽然珐基家遗传的真红秀髮并没有褪色，但也许是在孤立无援的状态下照看病人而积聚了疲倦，脸色比上一次见面时还差。
被当成村子裡的累赘一事也可能是造成精神负担的原因吧。
。
据尼禄古力夫所言，这次的病魔骚动实际上为利维伊教徒的自导自演，在结果上海尔的判断是正确的。
但从村人的角度来看的话，这只是由於偏见，而拒绝了对處於水深火热之中的自己伸出援手的利维伊教的行为罢了。
。
「没问题的啦。尼禄古力夫先生也拜托我们去帮助海尔村长。村长的那一份药的材料等会儿也会悄悄地给我们送过来的」
。
「能、能分给我们药吗？可是……这件事要是败露出去的话，也许会影响到尼禄古力夫大人的地位吧……」
。
芙露露语无伦次地说着。
。
「尼禄古力夫先生说过内部的事情就交给他好了，没问题的。况且人命关天，不需要这麼客气的啦」
。
尼禄古力夫说交给我来办时给人的安心感太强了。
毕竟那个人拥有那麼高的领袖魅力，只是一些小事应该不会影响他的立场的吧。
即使一部分的教徒对他有意见，最後也只会以被认为是对否定了利维伊教的人也伸出援手的圣人的结局收场吧。
魔术的手腕相当之高，做事也很果断。
。
只是，有一个脱线的地方……那就是，从不怀疑自己身边的人的性格。
从某个意义上来讲这是个优点，对於提升领袖魅力似乎有相当大的作用。
。
况且人无完人。
原本的话，副官应该是为了填补缺点而存在的……而那作为副官的克洛伊却是幕後黑手的候补，实在是令人痛心疾首。
就譬如埋伏在拉尔克身边的玛利亚斯，看来利维伊教徒的确很擅长混入在重要人员的旁边。
。
「非常感谢……请您，救救我父亲」
。
芙露露低头言谢。
长长的髮梢垂落到桌上。
。
「啊啊，之後就交……」「叩叩」
。
在说完之前，我便听到了敲门的声音。
似乎不是正门。这是……从裡面的方向发出来的吧？
。
「真是稀奇呢。今天，居然会有两批人造访这裡……。该不会是尼禄古力夫大人吧？」
。
「请稍等片刻」
。
芙露露的离席被我伸手制止。
大概是莉芙蛙的材料吧。
。
「要让美亚去观察情况吗？」
。
「不，还是让我去吧。毕竟制作的方法也还不清楚呢」
。
我离开客厅，往公馆的後门走去。
卡姆拉拿着一个大箱子等待着我。
似乎因为是熟人所以才被选中。
。
「原来是卡姆拉先生啊」
。
「啊、啊啊……尼禄古力夫大人委托我，将这个交付给亚贝尔」
。
「然後呢，制作的方法要怎样做？」
。
「……果然，这裡面是莉芙格拉斯的材料吗。毕竟是尼禄古力夫大人，考虑到时间和地点的话我也隐约察觉到了哦。不过，我只是被委托将这个箱子交付给你。箱子裡的东西我就当做什麼都不知道，送货这件事在离开的时候我也会忘得一乾二淨。不必担心，尼禄古力夫準备得很周到。肯定是已经帮你处理好後事了」
。
卡姆拉按着脑袋，叹了口气。
。
「卡姆拉先生也挺辛苦的样子呢……」
。
「那个人是个十足的老好人。嘛，正因为如此他才会被仰慕。逗留太久被人怀疑就不好了，我就先告退了……」
。
放下箱子後，卡姆拉匆匆忙忙地离开了。
将箱子搬入公馆後，我打开了盖子。
除了瓶装的药品之外，裡面还有一张空白的羊皮纸。
一开始我以为是其中一种材料，但这又有点讲不通。
。
我拿起羊皮纸，对其注入魔力。
纸上逐渐浮现出文字。
看来是用亚雷文字(魔术师用於魔术书的专门语言）将莉芙格拉斯的秘方缀写在羊皮纸上。
似乎是将文字隐藏起来的构造。
。
尼禄古力夫说过，莉芙格拉斯的制作方法是利维伊教的机密。
也许在转交给他人之手的场合，只能以非口头的形式转交。
。
「虽然在和材料配成一套的时点上，就觉得有些怪怪的……」
。
也许是因为国家的差异，亚雷文字的书写习惯和模式虽有些大同小异，但只要基础牢固的话解读也并没有想象中那麼难。
。
读着读着，羊皮纸从一角开始逐渐瓦解。
看来是使用被注入的魔力，让羊皮纸自毁的设计呢。
。
我追加魔力并顺着魔力轻轻地进行解析，然後发现这裡居然附加了相当複杂的魔术。
为了防止他人妨碍自毁的术式也施加了好幾个。
。
「真是慎重呢……就没想过在我读完之前就销毁的可能性么」
。
这也是他相信着我的地方吧。
不过，被这种方式试探，我也只会萌生出徒劳的对抗意识。
。
我拿起锡杖，在羊皮纸上轻轻地划了个十字。
。
「维持现状」
。
咏唱了咒文後，羊皮纸便停止了瓦解。
哼哼，我又略胜一筹了呢。
当然，我丝毫没有故意为难尼禄古力夫的意思，仅仅是作为一名魔术师想挑战一下这种複杂的防卫术式罢了。
不只是我，若是其他魔术师的话不管是谁都肯定想尝试挑战一次。
。
当然，看完之後我必然会烧掉这张羊皮纸的。
。
我请求美亚和芙露露的协助，开始了莉芙格拉斯的制作工作。
。
用药水(基本形）将高等曼德拉草在锅中高温煮熟，从而抽取高等曼德拉草的有效成分。
这时，为了不让有效成分气化逃逸，必须在锅中张开简易结界，将挥发的有效成分锁在裡面。
。
淡绿色的药水咕嘟咕嘟地开始沸腾，高等曼德拉草的紫色素逐渐扩散到全体。
曼德拉草特有的强烈药臭味充盈了整个房间。
药水煮干了的话，必须立即补充药水。
因为高等曼德拉草很坚硬，不在高温环境下根本抽取不了有效成分。
。
同时，用魔术将莉芙蛙的毒素取除，在另一口锅中加入些许蔚蓝史莱姆的消化液并轻轻加热，浸入莉芙蛙。
蔚蓝史莱姆的消化液逐渐将莉芙蛙溶解。
。
最後将两种材料放进同一口锅裡，将箱子裡的三种药品按照各自的时机掺和的话就大功告成了。
这些药品似乎都是把药草熬干後的产物，因为和莉芙蛙和高等曼德拉草不同，不会因为时间的经过而劣化，所以预先就準备好乐。
。
「从这边开始，分别要经过五小时、十小时、十四小时後才能掺入……大概到明天早上就能完成了呢。制作时必须用结界和魔法阵进行细微的状态控制……在这裡住一晚可以吗？」
。
「这、这倒是没什麼问题……从现在开始，到早上……一直都要同时维持着结界和魔法阵吗？」
。
芙露露担心地向我询问。
。
「没问题。我已经习惯熬夜了」
。
「请、请其他魔术师帮忙比较好吧？」
。
「嗯~……可是也不能再劳烦尼禄古力夫先生了啊……」
。
毕竟我是夜猫子嘛，完全没有问题。
以前在集落裡的时候，我还彻夜进行了大约五天的研究。
事到如今，熬一夜什麼的，简直是小事一桩。
。
「美亚也来帮忙！让美亚来吧！」
。
「……嗯~毕竟是高精密度的调整所以……」
。
「这样啊……」
。
美亚沮丧地耷拉着肩膀。
抱歉，心领了。
。
「啊啊，对了。忘记烧掉那个配方……嗯？咦？美亚，问下……我刚才拿着的，用亚雷文字缀写的羊皮纸去哪了？」
。
「诶？咦、咦？美亚只碰过一次哦？」
。
「然後呢……」
。
「还、还给亚贝尔了……应该，大概吧……。诶，美、美亚做错了什麼事吗！？怎、怎麼办啊！对、对不起……美亚真是……」
。
美亚突然变得惊慌失措。
。
……不过这麼一说的话，在美亚把羊皮纸还给我之後的一段时间之内还是有记忆的。
然後……烧掉了吗？咦，烧掉了吗？烧掉了吧？
以防万一，在把箱子拿出去烧掉的时候，跟箱子一起销毁了……吧？
。
也许是因为对制作莉芙格拉斯感到过於兴奋的缘故，所以便疏忽了对羊皮纸的管理。
那张纸留下一丁点都很危险。
。
「啊，大概……已经烧掉了吧……嗯，抱歉，应该是烧掉了」
。
「对、对美亚心有顾忌才这麼说的吧？」
。
「绝对！绝对是烧掉了！说起来，在那个时候放进箱子裡了。嗯，肯定没搞错！」
。
我就不应该提起这件事。
尼禄古力夫也应该预想到了这种情况，肯定施加了自动销毁的程序了吧。
