
第二天的晌午，我被引領到南教堂前的廣場。
戴著手銬、被五花大綁地釘在木板上。
真是渾身都不舒服。
。
廣場上已經人山人海。
教徒和村民們，各自顯露出憤怒、困惑或是不安的表情。
但最不可思議的是，即使怎麼尋找也找不到美亞的身影。
。
「…………」
。
是不是克洛伊對美亞干了什麼呢，一想到這裡我就開始坐立不安。
我就不應該讓美亞一個人呆著。
或許美亞現在也被囚禁在某個地方。
只能等這場鬧劇結束之後，怒斥克洛伊一番再讓她全盤托出才行。
。
順帶一提，尼祿古力夫也不在場。
也許是為了掌握確鑿的證據，打算直至形勢安定為止都隱藏起來吧。
。
終於，克洛伊來到我的跟前，準備開始進行惡魔裁決。
。
「那麼，從現在開始執行對亞貝爾·貝雷克的惡魔裁決。亞貝爾·貝雷克盜取了對於治療村人而必不可少的莉芙格拉斯，盜取了作為我教機密的莉芙格拉斯的制作方法，不僅向教堂縱火企圖銷毀莉芙格拉斯，並且存在著襲擊大神官大人並使他負傷的嫌疑」
。
在克洛伊陳述我的罪狀時，處於前排的臉相凶惡的利維伊教徒就開始對我破口大罵。
。
「肅靜！肅靜！此次的惡魔裁決是使用聖火的神聖儀式！這就等同於在利維伊大人的面前，請各位深刻反省！」
。
克洛伊一吼，教徒們就瞬間安靜下來。
隔了數秒，克洛伊再次開始發言。
。
「並且，由於在此期間存在著亞貝爾通過無詠唱而行使魔法，即是行使魔術的目擊情報，為了甄別真偽而特意準備了這次裁決。聖火！」
。
克洛伊說畢，一名抱著大鐵棒的男人隨即從觀眾的那一側往這邊靠近。
鐵棒的前端為喇叭的形狀，燃燒著青色的火焰。
鐵棒上雕刻著精緻的術式，散發出莊嚴的氣氛。
火把……不對，這個應該叫火炬。
。
這火焰的搖晃……怎麼看都只不過是用魔法將光反射，從而改變顏色的表演用的普通火焰。
這種程度的話我也能做出來。
看來這並不是什麼聖火，而是為了其他工序才準備的吧。
。
「克洛伊大人，請接過聖火」
。
「誒？」
。
我忍不住發出聲音。
。
「……那、那不是普通的火嗎？你確定不是什麼裝飾嗎？」
。
在一瞬間的寂靜之後，前排的教徒又開始對我破口大罵。
。
「事到如今你還想狡辯嗎！你這不肯死心的惡魔！」
「人在做天在看！這怎麼可能只是普通的火焰！想侮辱克洛伊大人也別太過分了！」
。
克洛伊用大杖敲了敲地面。
。
「肅靜！」
。
克洛伊再次出演勸說眾人的餘興，就當沒事發生一樣收場了。
原來如此，還以為做的有模有樣，原來是私底下已經串通好了呢。
回想起來，他們就是用這種卑鄙的手段把我們耍的團團轉呢。
這些傢伙，應該改名叫利維伊劇團比較合適吧。
。
看來是打算用這火焰來燒死我呢。
嘛，要是尼祿古力夫在此時颯爽登場，中止惡魔裁決並回收偽聖火火炬的話就是大團圓結局了。
雖然讓尼祿古力夫失去意識，並且讓我心甘情願地接受惡魔裁決的確是值得稱讚，但也因此被尼祿古力夫產生懷疑，在惡魔裁決中暴露出真身什麼的，真是聰明反被聰明誤呢。
。
尼祿古力夫也差不多該現身了吧。
再拖延下去也沒有意義。
。
克洛伊將大杖放在腳邊，從男人手中接過了燃燒著偽聖火的火炬。
。
「司掌水曜之神，吾等創造主利維伊大人。以您的心眼，為吾等識破邪惡之物吧！」
。
克洛伊在我面前高舉火炬，念念有詞。
。
尼祿古力夫還沒來嗎。
即使拼命地往遠處眺望，也完全沒有發現尼祿古力夫的踪影。
說起來卡姆拉和索菲亞也不在這裡。
。
糟、糟糕。
尼祿古力夫不來的話，單憑力氣我是絕對脫不了身的。
雖說事先弄壞阿洛古雅石的手銬也是個好辦法，但結果是在逃出去之後，就很難再將利維伊教徒從帕爾加斯村中趕出去了。
估計是尼祿古力夫的身上又有什麼麻煩了吧。
。
「啊……」
。
忽然發現從遠處氣喘吁吁地跑過來的美亞。
。
「趕、趕上了！火下留人！火下留人！」
。
美亞一邊撥開人群一邊前進，勉強來到觀眾面前。
當然，在觀眾之間也掀起了一陣喧囂。
美亞大概是走了很遠的路才來到這裡，滿臉通紅地。
。
「亞貝爾……找到了。找到那個了哦！是美亞發現的！啊、可、可是……雖然說起來有點尷尬，那件事可能不是美亞的錯哦……」
。
「那、那個……？」
。
聽她這麼說我也想不出是啥。
在我歪頭納悶的時候，克洛伊眯起眼盯著美亞。
。
「你把神聖的惡魔裁決當作是什麼！還不快速速退下！」
。
美亞從懷裡取出一張皺巴巴的羊皮紙，擺在克洛伊的面前。
。
「瞧瞧這個！」
。
「什、什、什麼……」
。
克洛伊睜大眼睛並後退了幾步。
她表情僵硬，聲音顫抖。
。
我對這張紙的確有印象。
。
「這是亞貝爾經由卡姆拉先生，從尼祿古力夫先生處得到的莉芙格拉斯的制作方法！尼祿古力夫先生為了拯救海爾村長，將這張紙托付給了亞貝爾！只要有了這張紙，亞貝爾為什麼擁有莉芙格拉斯之事，還有被尼祿古力夫先生傳授制作方法之事都可以證明了！而且也能證實亞貝爾不存在襲擊尼祿古力夫的理由！」
。
那是莉芙格拉斯的配方。
在興趣盎然地阻止羊皮紙的自毀後，我隨手一扔然後羊皮紙就消失不見了。
。
「干、干得好！幹得漂亮啊美亞！」
。
「誒嘿嘿……美亞，很久沒有被亞貝爾表揚了呢」
。
原本是因為莉芙格拉斯的配方可能會連累尼祿古力夫，所以我才打算把它處理掉，但到現在就不一樣了。
比起尼祿古力夫在教會中的失去地位的風險，現在在這裡讓我退場的損失反而更大，尼祿古力夫肯定也是這麼認為的。
之後肯定會一笑了之，付之東流的吧。
但是這裡必須渡過難關，令克洛伊垮台。
。
「喂、喂，這是怎麼回事！」
「那些紙又能證明什麼！」
。
「淨、淨耍小聰明！這一沓紙我沒有印象！你在說什麼我也完全聽不懂！」
。
克洛伊想一把抓住美亞。
雖然美亞慌忙地抽身，但克洛伊比她要快一剎。
她漂亮地抓住了配方的一角，然後用力一扯。
。
「啊啊！」
。
配方被撕成碎片，四分五裂的紙片隨風飄逸。
教徒們開始爭先恐後地拾取。
。
「這、這不是我們教團的上層在隱蔽性極高的聯絡中使用的，高級紙嗎……」
「……這亞雷文字的特徵，不是只在利維拉斯國才有的嗎？」
「不，我對魔術一竅不通所以看不出來……」
「為什麼克洛伊大人說沒有印象……？單靠片段的內容也能馬上明白，這明顯就是利維伊教的私物……」
。
克洛伊的臉染成了超越蒼白色的純白色。
看上去就像瑪連族人一樣。
她蜷縮著肩膀，兩腿脫力地盤坐在地上。
。
「那麼，能解釋清楚的話就請你解釋一下吧！」
。
「唏、唏……」
。
……我一追問，克洛伊便突然過度呼吸症發作。
眼角處還摻雜著淚水。
畢竟克洛伊看起來還是個孩子，在大人們面前把孩子弄哭什麼的，有種說不出來的罪惡感。
這邊可是要被燒死的，所以也顧不了這麼多了。
。
「……趕、趕緊放我出來吧！」
。
「肅靜！」
。
雖然並非是克洛伊那般響亮的聲音，但那聲低響卻在廣場中縈繞。
觀眾們紛紛左右讓道，讓尼祿古力夫通過。
。
「尼祿古力夫大人！」
「您終於醒了嗎！」
。
只是登場就傳出各種各樣的歡呼聲。真是有人氣啊。
這個人真的需要去掙取派閥鬥爭的人氣嗎。
。
「尼祿古力夫先生！來的也太遲了吧……雖然這邊也總算是想辦法解決了……」
。
我吐了口氣，對著尼祿古力夫會心一笑。
尼祿古力夫拾起碎紙，往地面狠狠一摔后又跺了兩腳。
。
「誒……？」
。
「老朽傳授給亞貝爾閣下……嗎？唔姆，不記得了，嘛，這並沒有關係。使用聖火的話就能夠瞬間分辨出黑白。有什麼話，之後再說。你也是這麼想的吧，克洛伊？」
。
尼祿古力夫溫柔地眯著眼睛，向克洛伊開口。
。
「尼、尼祿古力夫，先生？」
。
被我呼喚名字時，尼祿古力夫朝我笑了笑，接著徑直地走向吵鬧的觀眾席。
。
「肅靜。這可是在利維伊大人的面前。即使不那麼躁動，一切都已經在利維伊大人的心眼面前揭曉。要追究克洛伊的話，稍後再辦吧。惡魔裁決是神聖之事，可不能讓利維伊大人白白地等待」
。
尼祿古力夫的一番話讓周圍的一片人萬籟俱寂。
。
這時，我終於發現。
從最初的相遇……不，從相遇之前、在最初被面相凶惡的利維伊教徒包圍，直至現在的這個瞬間，我都一直在這個人的手掌心上跳舞。
。
「那麼，我宣布惡魔裁決重新開始」
