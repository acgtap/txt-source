
第二天的晌午，我被引领到南教堂前的广场。
戴着手铐、被五花大绑地钉在木板上。
真是浑身都不舒服。
。
广场上已经人山人海。
教徒和村民们，各自显露出愤怒、困惑或是不安的表情。
但最不可思议的是，即使怎麼寻找也找不到美亚的身影。
。
「…………」
。
是不是克洛伊对美亚干了什麼呢，一想到这裡我就开始坐立不安。
我就不应该让美亚一个人呆着。
或许美亚现在也被囚禁在某个地方。
只能等这场闹剧结束之後，怒斥克洛伊一番再让她全盘托出才行。
。
顺带一提，尼禄古力夫也不在场。
也许是为了掌握确凿的证据，打算直至形势安定为止都隐藏起来吧。
。
终於，克洛伊来到我的跟前，準备开始进行恶魔裁决。
。
「那麼，从现在开始执行对亚贝尔·贝雷克的恶魔裁决。亚贝尔·贝雷克盗取了对於治疗村人而必不可少的莉芙格拉斯，盗取了作为我教机密的莉芙格拉斯的制作方法，不仅向教堂纵火企图销毁莉芙格拉斯，并且存在着袭击大神官大人并使他负伤的嫌疑」
。
在克洛伊陈述我的罪状时，處於前排的脸相凶恶的利维伊教徒就开始对我破口大骂。
。
「肃静！肃静！此次的恶魔裁决是使用圣火的神圣仪式！这就等同於在利维伊大人的面前，请各位深刻反省！」
。
克洛伊一吼，教徒们就瞬间安静下来。
隔了数秒，克洛伊再次开始發言。
。
「并且，由於在此期间存在着亚贝尔通过无咏唱而行使魔法，即是行使魔术的目击情报，为了甄别真伪而特意準备了这次裁决。圣火！」
。
克洛伊说毕，一名抱着大铁棒的男人随即从观众的那一侧往这边靠近。
铁棒的前端为喇叭的形状，燃烧着青色的火焰。
铁棒上雕刻着精緻的术式，散發出庄严的气氛。
火把……不对，这个应该叫火炬。
。
这火焰的摇晃……怎麼看都只不过是用魔法将光反射，从而改变颜色的表演用的普通火焰。
这种程度的话我也能做出来。
看来这并不是什麼圣火，而是为了其他工序才準备的吧。
。
「克洛伊大人，请接过圣火」
。
「诶？」
。
我忍不住發出声音。
。
「……那、那不是普通的火吗？你确定不是什麼装饰吗？」
。
在一瞬间的寂静之後，前排的教徒又开始对我破口大骂。
。
「事到如今你还想狡辩吗！你这不肯死心的恶魔！」
「人在做天在看！这怎麼可能只是普通的火焰！想侮辱克洛伊大人也别太过分了！」
。
克洛伊用大杖敲了敲地面。
。
「肃静！」
。
克洛伊再次出演劝说众人的餘兴，就当没事發生一样收场了。
原来如此，还以为做的有模有样，原来是私底下已经串通好了呢。
回想起来，他们就是用这种卑鄙的手段把我们耍的团团转呢。
这些傢伙，应该改名叫利维伊剧团比较合适吧。
。
看来是打算用这火焰来烧死我呢。
嘛，要是尼禄古力夫在此时飒爽登场，中止恶魔裁决并回收伪圣火火炬的话就是大团圆结局了。
虽然让尼禄古力夫失去意识，并且让我心甘情愿地接受恶魔裁决的确是值得称讚，但也因此被尼禄古力夫产生怀疑，在恶魔裁决中暴露出真身什麼的，真是聪明反被聪明误呢。
。
尼禄古力夫也差不多该现身了吧。
再拖延下去也没有意义。
。
克洛伊将大杖放在脚边，从男人手中接过了燃烧着伪圣火的火炬。
。
「司掌水曜之神，吾等创造主利维伊大人。以您的心眼，为吾等识破邪恶之物吧！」
。
克洛伊在我面前高举火炬，念念有词。
。
尼禄古力夫还没来吗。
即使拼命地往远处眺望，也完全没有發现尼禄古力夫的踪影。
说起来卡姆拉和索菲亚也不在这裡。
。
糟、糟糕。
尼禄古力夫不来的话，单凭力气我是绝对脱不了身的。
虽说事先弄坏阿洛古雅石的手铐也是个好办法，但结果是在逃出去之後，就很难再将利维伊教徒从帕尔加斯村中赶出去了。
估计是尼禄古力夫的身上又有什麼麻烦了吧。
。
「啊……」
。
忽然發现从远处气喘吁吁地跑过来的美亚。
。
「赶、赶上了！火下留人！火下留人！」
。
美亚一边拨开人群一边前进，勉强来到观众面前。
当然，在观众之间也掀起了一阵喧嚣。
美亚大概是走了很远的路才来到这裡，满脸通红地。
。
「亚贝尔……找到了。找到那个了哦！是美亚發现的！啊、可、可是……虽然说起来有点尴尬，那件事可能不是美亚的错哦……」
。
「那、那个……？」
。
听她这麼说我也想不出是啥。
在我歪头纳闷的时候，克洛伊眯起眼盯着美亚。
。
「你把神圣的恶魔裁决当作是什麼！还不快速速退下！」
。
美亚从怀裡取出一张皱巴巴的羊皮纸，摆在克洛伊的面前。
。
「瞧瞧这个！」
。
「什、什、什麼……」
。
克洛伊睁大眼睛并後退了幾步。
她表情僵硬，声音颤抖。
。
我对这张纸的确有印象。
。
「这是亚贝尔经由卡姆拉先生，从尼禄古力夫先生处得到的莉芙格拉斯的制作方法！尼禄古力夫先生为了拯救海尔村长，将这张纸托付给了亚贝尔！只要有了这张纸，亚贝尔为什麼拥有莉芙格拉斯之事，还有被尼禄古力夫先生传授制作方法之事都可以证明了！而且也能证实亚贝尔不存在袭击尼禄古力夫的理由！」
。
那是莉芙格拉斯的配方。
在兴趣盎然地阻止羊皮纸的自毁後，我随手一扔然後羊皮纸就消失不见了。
。
「干、干得好！幹得漂亮啊美亚！」
。
「诶嘿嘿……美亚，很久没有被亚贝尔表扬了呢」
。
原本是因为莉芙格拉斯的配方可能会连累尼禄古力夫，所以我才打算把它处理掉，但到现在就不一样了。
比起尼禄古力夫在教会中的失去地位的风险，现在在这裡让我退场的损失反而更大，尼禄古力夫肯定也是这麼认为的。
之後肯定会一笑了之，付之东流的吧。
但是这裡必须渡过难关，令克洛伊垮台。
。
「喂、喂，这是怎麼回事！」
「那些纸又能证明什麼！」
。
「净、净耍小聪明！这一沓纸我没有印象！你在说什麼我也完全听不懂！」
。
克洛伊想一把抓住美亚。
虽然美亚慌忙地抽身，但克洛伊比她要快一刹。
她漂亮地抓住了配方的一角，然後用力一扯。
。
「啊啊！」
。
配方被撕成碎片，四分五裂的纸片随风飘逸。
教徒们开始争先恐後地拾取。
。
「这、这不是我们教团的上层在隐蔽性极高的联络中使用的，高级纸吗……」
「……这亚雷文字的特徵，不是只在利维拉斯国才有的吗？」
「不，我对魔术一窍不通所以看不出来……」
「为什麼克洛伊大人说没有印象……？单靠片段的内容也能马上明白，这明显就是利维伊教的私物……」
。
克洛伊的脸染成了超越苍白色的纯白色。
看上去就像玛连族人一样。
她蜷缩着肩膀，两腿脱力地盘坐在地上。
。
「那麼，能解释清楚的话就请你解释一下吧！」
。
「唏、唏……」
。
……我一追问，克洛伊便突然过度呼吸症發作。
眼角处还掺杂着泪水。
毕竟克洛伊看起来还是个孩子，在大人们面前把孩子弄哭什麼的，有种说不出来的罪恶感。
这边可是要被烧死的，所以也顾不了这麼多了。
。
「……赶、赶紧放我出来吧！」
。
「肃静！」
。
虽然并非是克洛伊那般响亮的声音，但那声低响却在广场中萦绕。
观众们纷纷左右让道，让尼禄古力夫通过。
。
「尼禄古力夫大人！」
「您终於醒了吗！」
。
只是登场就传出各种各样的欢呼声。真是有人气啊。
这个人真的需要去挣取派阀鬥争的人气吗。
。
「尼禄古力夫先生！来的也太迟了吧……虽然这边也总算是想办法解决了……」
。
我吐了口气，对着尼禄古力夫会心一笑。
尼禄古力夫拾起碎纸，往地面狠狠一摔后又跺了两脚。
。
「诶……？」
。
「老朽传授给亚贝尔阁下……吗？唔姆，不记得了，嘛，这并没有关係。使用圣火的话就能够瞬间分辨出黑白。有什麼话，之後再说。你也是这麼想的吧，克洛伊？」
。
尼禄古力夫温柔地眯着眼睛，向克洛伊开口。
。
「尼、尼禄古力夫，先生？」
。
被我呼唤名字时，尼禄古力夫朝我笑了笑，接着径直地走向吵闹的观众席。
。
「肃静。这可是在利维伊大人的面前。即使不那麼躁动，一切都已经在利维伊大人的心眼面前揭晓。要追究克洛伊的话，稍後再办吧。恶魔裁决是神圣之事，可不能让利维伊大人白白地等待」
。
尼禄古力夫的一番话让周围的一片人万籁俱寂。
。
这时，我终於發现。
从最初的相遇……不，从相遇之前、在最初被面相凶恶的利维伊教徒包围，直至现在的这个瞬间，我都一直在这个人的手掌心上跳舞。
。
「那麼，我宣布恶魔裁决重新开始」
