1
我被莉诺亚叫去锻造屋，说是委托她的东西做好了。
不用说，是用青魔像和泽舒姆遗迹的魔矿石合成的魔金属为材质的那把剑的事情。
我稍有浮躁地走向锻造屋。
。
「那个……亚贝尔委托的是剑吧？」
。
路上，跟我一起过来的美亚开口问道。
。
「嗯？啊啊，因为在之前的骚动中没有对偷袭做好准备呢」
。
无论怎样提高魔术的性质、压缩魔法阵的展开的咏唱，都只会落后一步。
若是被弓箭从远处狙击，如果没注意到的话一切都结束了。
。
在欧特姆中编入对应的动作用来保护自己的话，虽然也可以将就一下，不过常时将欧特姆放在身边就有点不顺眼了。
总之现在就先用手头有的东西做一个试试，于是便设计出能够轻松携带的剑了。
实话说，只是想试着做一下的心情才是最主要的。
。
做一个编入防御术式的长袍可能也不错，但要做的话我想做的更正式一点。
毕竟现在材料不够。
。
「之前……是说和玛利亚斯小 姐战斗的事情吧。亚贝尔遇上危险了吗？」
。
美亚似乎有点吃惊地睁大了眼睛。
。
「不是不是。是在和伊卡洛斯大叔决斗的前一天，玛利亚斯的精灵兽潜入过我的房间了吧？那时候，如果不是美亚用弓箭打中的话可能就危险了」
。
「美亚派上用场了吗！」
。
美亚提高音量地说道，高兴地手舞足蹈。
然后像是想到了什么一样停下动作，小声问道。
。
「……那个，玛利亚斯本人呢？」
。
「……嘛，感觉一般般吧」
。
「一般般……」
。
大概和以前战斗过的山矮人的大叔骑士同一等级吧。
召唤出大量的恶魔似乎是最大的优势，要是分散战斗的话在大规模的战斗中或许能大显身手。
多多少少也感觉到她原本就不是倾向于在表面战斗的类型。
。
她的确有发出指令的领导力，也有能召唤出与状况相应的恶魔的对应能力，但没有能让恶魔保护好作为本体的玛利亚斯的力量的话，不应该先考虑防御这方面吗。
。
隔了一阵子沉默之后，美亚露出略有所思的表情，向我提出疑问。
。
「……那啥，亚贝尔要挥动那把剑吗？」
。
「嗯？」
。
「不、不是……那个……那种东西，看上去有点重……」
。
「啊啊，我把它设计成能用魔力操控所以没问题的」
。
与其说是剑，不如说是高性能的魔法道具比较准确。
我将它设定为成能够到处乱飞攻击敌人，并且还能准确地保护我的机制。
。
「原来是这样……那美亚也就放心了」
。
美亚放下心来地笑了。
。
……总觉得刚才我的自尊心受到了伤害。
我挽起手臂，用手摸了摸手上的肌肉。
咦……其实不也是有一点的吗？
。
自从我走出集落之后，我的步行量也是在逐日增加的。
虽然每天都被肌肉酸痛所困扰，但这就是身体在逐渐强化的证明。
。
「嘛、嘛……其实，用手挥剑我也是可以做到的……」
。
我喃喃细语地反驳道。
。
在锻造屋中，憔悴的店主和莉诺亚在等待着我。
。
「……这是跟你说的一样，用那种金属做成的剑『柄』」
。
我从莉诺亚处接过了染成铅色的剑柄。
。
在剑柄的正反面上，有三个模仿眼珠的图案纵向地并排在一起。
虽然是有点恶趣味，不过这是能够轻松固定青金魔石而想出来的设计。
。
「咦……刀刃的部分呢？」
。
美亚吃惊地用手指指着剑柄上的眼珠问道。
。
「青魔像拥有自己创造出石像的身体，并凭依到石像上的性质。因此利用它性质灌输魔力的话，就可以生成出相应的刀刃了」
。
这样的话就只需要柄的部分，也方便携带。
而且因为是精灵体和魔力传导性极高的金属块，即便是用转移魔术也可以花费较少的魔力转移到手上。
。
「在生成刀刃的期间，这把剑会遵从我的命令漂浮起来，自动地攻击敌方。因为也会作出防御，所以在逼不得已的时候一直不收刀就好了」
。
这次交给莉诺亚的金属块，是以泽舒姆遗迹……在那个空中要塞中的部件的碎片为基础做成的金属，在调整作为青魔像残骸的精灵体之后再强行让其凭依上去的金属。
运用精灵的技术，能够悬浮在空中并自由地飞行。
在被输入魔力的期间会根据术式而作出我所控制的对应动作，几乎是自动地对敌人进行攻击或防御。
泽舒姆好厉害，谢谢你啊艾贝尔海德先生。
。
「十分感谢你们，莉诺亚小 姐，还有店主先生」
。
「哈哈哈，我也学到了不少东西呢。没想到，世界上还有这种素材。如果还想做什么奇怪的东西的话，绝对要叫上我哦！」
。
虽然店主看上去精神满面地挂着笑容回答，不过眼皮底下的黑眼圈很重。
……下次还是别让他们这么赶好了。
。
我向莉诺亚和锻造屋的店主道谢之后，马上就尝试对剑柄注入了魔力。
光之粒子聚集在剑柄的前端，不久后便形成了细长的刀身。
。
就连刀身的形状都能变成喜欢的形状，这也是它强大的功能之一。
无论是大长刀还是镰刀，都能自由自在地变换。
。
剑从手中离开时在空中一转，在我面前倾斜刀身，宛如准备好进攻一样静止在我面前。
。
嗯，虽然有点简便，从今天起你的名字就叫青魔剑吧。
。
「好想砍些东西试一试刀啊……」
。
总是有种坐立不安的感觉。
不蹦出个纳尔加仑什么的吗。
。
「……团长喜欢的话就再好不过。炼金术师团那边的话，我后天再去吧……」
。
莉诺亚似乎也积聚了不少疲劳。
看来是打算休息一天。
。
「我明白了。那我差不多该走了，毕竟午后还有对炼金术师团的成员的指导工作……」
。
当我打算离开锻造屋，转身迎向出口时，锻冶屋的大门被粗暴地推开了。
不知发生了什么大事，脸色发青的尤里斯走了进来。
。
「亚贝尔大人！拉尔克大人的紧急呼叫……」
。
「这次又出现了纳(什）尼(么）加(事）仑！？」
。
「……诶？那个，也不是那样……」
。
就像是被挫了锐气一样，尤里斯突然露出困惑的表情。
。
真是个好机会。
这么快就能测试青魔剑的性能……好想快点试刀啊。
希望是又大又顽强，值得一斩的对手呢。
