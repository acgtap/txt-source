「亚贝尔团长……这个你看做的怎样？」
。
炼金术师团一员的贾伽，诚惶诚恐地把手中的欧特姆递给我看。
在忙于制作欧特姆的其他团员们对贾伽的话有所反应，停下手上的工作并往我这边看。
要制作怎样的欧特姆才达到合格线呢？他们大概是在探查我的基准吧。
。
我接过贾伽手中的欧特姆，拿在手上来回翻转。
。
「嗯、嗯，感觉不错啊。比之前好很多了」
。
贾伽放心地松了口气。
其他人听到我说的话似乎也放下心来，继续开始手头的作业。
。
其实我想说的话有一大堆……嘛，也不可能让他们一直雕刻欧特姆。
能做到最低限度的欧特姆就足够了。
。
「然後沟纹太粗糙和表面太过凹凸不平……还有部位连接的比例太差的部分更正过来就好了。还有……怎么说呢，没有嵌入灵魂什么的……」
。
说完，周围的团员们手中的木雕用小刀便掉落在地上。
。
「这、这不就是让我们重做吗……那个，我们还要雕刻这玩意多久……」
。
「总之雕完这周就好了，所以你们放心吧。我可是从四岁开始就一直雕刻到现在的哦」
。
「那、那休息呢……」
。
「诶……不过，不加快一下进度的话就赶不上预期的计划了……要是那样的话也对不起拉尔克先生了……」
。
提高炼金术师团的魔术技术是我计划的领地改革的基础。
只要熬过这裡的话後面就大体上畅通无阻，反过来说在这裡卡住的话起步就晚了。
。
话说回来……这个似乎比想象中要花时间呢。
但假如不让他们对欧特姆雕刻更加熟练的话，那就真的是无从谈起了。
要说我是哪边的话，我大概是感觉派吧，所以也不怎么擅长教别人东西。
目前让我直接指导也没多大意义。
。
「拉、拉尔克大人也不是那么急的吧」
。
「…………嘛，老实说，是我自己想要快一点而已」
。
「刚才说出真心话了吧！？」
。
「我问拉尔克先生是不是快一点比较好，他也说好啊……」
。
「这么问肯定也是这么回答啊！」
。
……该说是意识不同，还是说意向相差太远了呢。
想起以前指导西碧的时候也用姬泽引诱他，强行向他灌输技术，结果他魔术的手腕也没见有多少提高。
我，大概是不适合做老师吧。
只有姬泽露在我讲授的知识中汲取我的经验并持续锻炼，所以才那么上手。
。
「说起来，莉诺亚副团长呢？怎么没看见她？」
「我也没看见她……」
「难道是在精灵兽骚动时受伤了吗？」
。
贾伽这么一说，其他团员也纳闷了。
。
「……那个，我让她去加工我炼成的魔金属了。她看过我的制作过程之後就哗啦啦地说出一堆改善的地方，要不是我让她重头开始做的话，又会浪费我的时间……啊啊不过！我已经叫她快一点完成的了，在那之後应该也会来这裡露脸的啦……」
。
「…………」
。
私兵团全员用严肃的表情盯着我看。
怎么看都像是在表达『冷静一想的话这个人把炼金术师团当做私有物的程度不是在伊卡洛斯之上吗？』的眼神。
。
「说、说的也是呢……也、也不用这么着急对吧……大家看上去也累了，那就休息一会吧！好吗？」
。
我这么一说，团员们就地放下木雕用的小刀，一个又一个地倒下了。
。
「你、你们怎么了？」
。
「精神快支撑不住了……有种奇怪的头痛……」
。
团员中的一人保持着仰倒的姿势，微微点头说道。
。
「毕竟在魔术的使用中，集中力是重要的因素之一，这么快就支撑不住就说明这裡是弱点了呢」
。
「这么快……？从早上开始都快过去半天了……？」
。
「但是你们现在感觉到的疲倦，就是成长的证据，照这个状态继续下去的话……」
。
「这个状态、继续下去……？」
。
……或许我应该改变一下计划。
。
「团长……俺今天已经可以回去了吗？大概……这已经是我的极限了……」
。
又有其他团员开口说道。
。
「那个……总之从现在开始给你们放个长假，以後你们还想继续的话我会很高兴的……」
。
「亚贝尔亚贝尔！拉尔克先生开始整理之前讨论过的那些美术品了哦！一起去看看吧！」
。
美亚从拉尔克的公馆跑过来大声呼唤我。
为了复兴珐基领和其他领地的交易，拉尔克打算将上一代领主因兴趣而收集的美术品卖掉用作复兴。
本来在前几天就应该去典当的了，但由于玛利亚丝引起的骚动而不得不推迟，大概是安排到今天了吧。
虽然我对这些事没什么兴趣，不过美亚说想去看所以才约好了陪她一起去的。
。
「啊……那、就是这么回事……我去去就回……」
。
空气中仍然蔓延着躁动不安的气息，我打算暂时地离开这裡。
。
「不过还有力气的人，在我离开的时候也随便地雕刻一下的话就省事了呢……」
。
「那个……美亚在想，今天就这么解散不也挺好的嘛」
。
看着死尸累累的团员们，美亚小声嘀咕道。
。
「……果然是这样吗？那么今天就解散吧，你们好好休息准备明天，还有自行复习……」
。
说完，倒地的团员们纷纷抬起上半身。
。
「太棒啦啊啊啊！」
「十分感谢，十分感谢美亚小 姐！那个，下次也过来露个脸的话就帮大忙了……」
「就算我们怎么说，他都听不进去啊……」
。
团员们开始向美亚低头谄媚。
其中还有人开始流泪了。
。
「请、请抬起头来吧！亚、亚贝尔，你对他们做了什么！？」
。
……我、我倒是没有严格到这种地步的打算啊……果然还是得重新修改计划呢。
总之，去参观美术品的时候顺便和拉尔克商讨一下吧。
