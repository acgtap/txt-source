在亚贝尔一行去往国境边缘的山内寻觅莉芙·法格之时，利维伊教的大神官尼禄古力夫，正和他的副官克洛伊在教堂的某个房间中谈话。
这个房间被施加了结界，有着防止被外界用魔术获取内部的声音和视野的效果。
克洛伊困惑地发话。
。
「尼禄古力夫大人……我说，为什麼要对外人透露莉芙·法格的情报呢？那明明是在利维伊教之中也只有一小部分人知道的事情……」
。
「呵呵呵……伟大的利维伊大人是宽大为怀的。为了拯救人命而打破规定什麼的，利维伊大人是不会在意的吧」
。
尼禄古力夫笑着回答。
然而克洛伊却是愁眉苦脸的样子。
。
「我并不想听表面上的解释。尼禄古力夫大人，究竟在考虑些什麼？那个叫亚贝尔的男孩，不可能只是普通的旅行者您应该是知道的。说不定是拉尔克男爵，迪恩拉特王国派过来的间谍吧？为什麼要对他这麼亲切？差不多是时候让我听听您真正的目的了吧……」
。
尼禄古力夫用手扶着下巴，用眼睛巡视房间的四处，再次确认周围是否有局外人的气息。
。
「还以为你在老朽离开的期间有所成长……你对这些事还不够成熟呢。嘛……从现在开始慢慢学习就好。毕竟能够继承老朽的，也就只有克洛伊你一个人啊」
。
尼禄古力夫轻轻地敲了敲克洛伊的脑袋两下。
克洛伊似乎感到害羞，不耐烦地撇着嘴。
。
「亚贝尔阁下毫无疑问是拉尔克男爵派来的侦察兵。和玛利亚斯阁下失去了联繫，玛利亚斯阁下借用恶魔的力量而形成的魔兽包围网也遭到中断，拥有魔术心得之人的突然来访……应该是没错了。在珐基领内，優先选择来到这个偏僻的地方也是个粗糙得不行的谎话。可是，作为王都的尖兵，不仅缺乏慎重而且漏洞也很多……。在交谈时，还有在隐瞒情报时也表露出显而易见的感到抱歉的表情」
。
「都、都知道得这麼清楚了，为什麼……！」
。
「冷静下来，克洛伊。亚贝尔阁下的确是有相应的魔术手腕，但看上去并不习惯那种场面……。这种能用的人才并不多见。老朽早已做好和亚贝尔接触的布局了」
。
尼禄古力夫保持着温柔的笑容，耸肩笑了笑。
。
尼禄古力夫异常地擅长得到他人的信赖。
被教徒们纠葛时的帮助，对自己不利的情报的公开，对对方过剩的赞许。
全部都是为了获得亚贝尔信赖的陷阱。
说出和教堂的司祭相遇的过去也是其中的一环。
将以前有关係的克洛伊優先作为辅佐，他的确有这个打算。
要说什麼话对方才会放下戒心，尼禄古力夫是最熟悉不过，并且拥有相应的演技和周全的思考能力。
。
若是平时的话，最多只会在唆使教徒们找茬之後自己再伸出援手……但考虑到之前的接触和假说，他判断到亚贝尔有着利用的价值，即使多花点功夫也值得笼络。
。
「那，那样的话，难道说我妨碍到尼禄古力夫大人了吗……。试探得太过深入……大概让他产生警觉了吧」
。
「那也在老朽考虑的範畴之内。放心就好。若是从珐基领来到这裡的话，首先肯定不会对我们抱有良好的印象。不如说就是因为克洛伊你扮演了讨人厌的角色，老朽才能顺利地推动话题。克洛伊已经做的不错了」
。
「是、是这样吗……那可真是太好了……」
。
克洛伊放鬆地喘了口气。
。
「把讨人厌的角色交给你来演，真是抱歉呐」
。
尼禄古力夫关心地对克洛伊表示慰劳，又把手伸向她的头上。
。
「请不要这样！……真是的。我承认因为血的缘故，身体的成长是有点迟缓，但内在已经是成熟的大人了。请不要把我当作孩子了」
。
克洛伊扭头躲过尼禄古力夫的手掌。
尼禄古力夫“呵呵”地笑了笑，然後吸了口气，回到刚才的紧张的表情。
。
「老朽採用这种手段，也是希望克洛伊能尽早学会老朽做事的手法啊」
。
尼禄古力夫被利维伊选为大神官之前担任着教堂的司祭，而克洛伊也是那时候养育的孤儿……这些话都是真实的。
虽然克洛伊在尼禄古力夫离开後继续担任教堂的司祭，但似乎是最近才有机会，以自己的意向担任他的副官。
。
因此，克洛伊对於这次必须要运用到肮髒手段的任务还不够经验，对尼禄古力夫独特的手段也不够熟悉。
因罪恶感和紧张而过於敏感，村人们多少会对他们抱有疑心。
亚贝尔也会用疑惑的目光看待他们。
。
在这点上，能够让村人们完全地接受，就连无论是否侦查而来的亚贝尔的眼睛都能成功欺骗的尼禄古力夫的对人技术是压倒性的。
尼禄古力夫并不是乱说谎，而是在谎言中穿插着真实从而获得信任，擅长将不利的事实巧妙地加以掩饰。
。
即使统一口径瞒天过海，也总有一天会出现破绽。
然而无意义地隐瞒身世的行为更令人心生怀疑。
归功於多年从事利维伊教司祭工作而积累的经验，他把真实与虚假的百分比调整得十分巧妙。
尼禄古力夫的部下能够坦然地说出尼禄古力夫过去的事情也是这个原因。
。
「虽然我很感激尼禄古力夫大人对我的厚爱……可是，对亚贝尔透露莉芙·法格的秘密什麼的，我果然还是不能赞同……。如果亚贝尔入手了莉芙·法格的话，说不定他就真的制造出莉芙格拉斯了。那样的话，處於我们控制之中的帕尔加斯村的平衡就会被打破」
。
「有什麼问题？将我们保管、收藏着的古旧莉芙格拉斯稀释使用就好。帕尔加斯村的平衡不会被打破的。倒不如说，我们能够利用这件事让村人们强烈地意识到莉芙格拉斯的重要性。而且，亚贝尔阁下看起来是很容易随波逐流的类型呢。把他吹捧成为村子的英雄的话，很快就会依存於这个村子了吧」
。
「原……原来如此……。的确，莉芙格拉斯的分量，无论是多是少都可以蒙混过关」
。
「而且，入手了莉芙法格怎麼说都不亏。即使没有入手，在亚贝尔被其他教徒狠狠地斥责之後，再由老朽慢慢地安慰就好。在这个被隔离的村子受到的轻蔑和赞许，会简单地扰乱人类正常的判断能力。这就等於已经完全掌握亚贝尔阁下一样。没有什麼值得担心吧」
。
「您已经思考到这个地步了吗……。可、可是，今後，不也有莉芙·法格的秘密从他身上泄露到迪恩拉特王国的危险吗？」
。
「呵呵呵……不必担心。要是拉拢失败，还试图逃出我的手掌心的话，杀掉也无妨。为此我才让那三个人跟着。若是他们的话，无论發生什麼事都不会让亚贝尔逃掉的吧」
。
与亚贝尔同行的利维伊教三人组，是尼禄古力夫部下中的精英。
他们在利维伊教会的活动中，和尼禄古力夫相处的时间比克洛伊还要久远。
绝不是随便挑选和毛遂自荐的结果。
无论是战鬥或是侦查，都是超一流的水准。
。
实话实说，村子裡的利维伊教徒都是尼禄古力夫忠实的部下。
为了充数而随意带来的教徒，一个都没有。
粗暴的教徒，也只是尼禄古力夫命令的演技而已。
。
他对那三人下了命令，如果亚贝尔是不安定因素的话，便格杀勿论。
即使亚贝尔的魔术有多麼优秀，被三人同时偷袭也绝对没有胜算。
。
在关键时刻下不了杀手的人是不存在的。
即便是發生了什麼事故，他们都拥有能够冷静处理的能力。
到了那个时候，估计亚贝尔连發生了什麼事都不知道就被他们给做掉了吧，尼禄古力夫这麼认为。
。
「…………」
。
杀掉也无妨，尼禄古力夫这麼说着的时候，克洛伊不由自主地低下头。
。
「你还不习惯这些事情吗」
。
「尼禄古力夫大人……有点变了呢」
。
克洛伊看上去似乎有些寂寞，小声嘀咕。
。
「…………做大事要不拘小节。在利维伊大人降临的现在，企图消灭利维伊大人的大邪神库多尔也终於开始苏醒了吧。必须在这之前征服迪恩兰特王国，加强在国内某处沉眠着的大邪神库多尔的封印。要是那傢伙醒来的话，这片土地又会再次涂炭生灵吧。为此……不得不扼杀自己的心。老朽不要求你铁石心肠，但不要表现在外。即使玷污这双手，也要让邪恶的存在在这个世界上消失」
。
尼禄古力夫一边流泪，一边握紧了拳头。
。
「尼禄古力夫大人……」
。
在克洛伊打算为尼禄古力夫擦拭眼泪而掏出手帕时，尼禄古力夫在思考别的事情。
。
