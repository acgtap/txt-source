「這本書就這麼有趣嗎？那樣的話，把它也搬到樓上去吧……」
。
美亞對緊緊地抱著魔導書的我說道。
。
「不用，也不是說想要帶走……我只是想讀一讀這個部分而已……」
。
受到『高位精靈圖譜』重擊的我總算是復活過來，繼續尋找倉庫內還有沒有亮眼的書物。
因為『高位精靈圖譜』太重，拿都拿不起來，所以讓傭人把書搬到我的房間裡了。
。
「但是你一直都在讀書……直接搬上去不就好了麼……」
。
「怎麼說好呢，搬上去之後又感覺不是我想讀的書，不先確認一下內容總感覺有點不放心呢」。
「明明還有其他看上去很有趣的壺和畫啊……。吶，過去看看那邊好嗎？吶？不，美亞其實……也並不是特別感興趣的……」
。
……用這種說法懇求的話更是讓人萌生出罪惡感，有種不去看不行的不可思議的感覺。
美亞在鍊金術師團的活動中過來邀請我時的高興的表情，在我腦海裡一掠而過。
。
「我知道了」
。
「太好啦！走吧，亞貝爾……！」
。
「那個……再等我五分鐘吧」
。
「啊……好吧……」
。
把魔術書放回到原來的位置之後，我和美亞在倉庫裡漫步。
倉庫裡的東西已經大致上整理好了，現在是選出賣去其他街道的美術品並用布包裹的階段。
。
遺憾的是我完全沒有關於壺和繪畫的知識，所以不是很清楚。
至少也應該在下面寫上作者和畫的名字吧。
不過，值得高興的是魔獸和惡魔的繪畫很多。
庫多爾神和水神利維伊之類的繪畫也有。
。
在利維伊的繪畫旁邊，有一幅被一群矮人跪拜的巨大的四頭鹿的繪畫。
四個腦袋分別朝著左右的斜上下的四個方向，下顎對著正中央，姿勢十分奇怪。
矮人就像是沙粒一樣，是幾乎看不見的尺寸。
。
這大概就是地神伽爾迦吧。
雖然我從未見過刻畫伽爾迦樣子的繪畫所以也不是很確信，不過伽爾迦創造的被稱為岩巨兵的石像就站在它的腳邊。
。
順帶一提……在伽爾迦旁邊的繪畫，是拉爾克祖父的肖像畫。
把自己的肖像畫和庫多爾、利維伊、伽爾迦並排在一起，真是個豪快的人呢。
不對，安排位置的不一定是他本人。
。
在我觀摩拉爾克祖父的肖像畫時，美亞拉了拉我的衣袖。
。
「怎麼了？」
。
「……總覺得，只有那幅畫有點不一樣？」
。
跟隨美亞的視線，我髮型了一幅街道被燒毀的繪畫。
……對建築物和人的描畫與其說是正確，不如說是生動。
這幅畫的構圖本身，就像是某個人的視野一樣。
看上去就覺得心情不太好。
。
「哦？小姐，看中那幅畫了嗎。那幅畫並不是隨隨便便就可以賣到其他地方的東西，老朽對拉爾克大人這麼提議了呢」
。
大概是已經完成了任務，被請來做鑒定工作的約瑟夫來到了我的身旁。
。
「是有名的畫家的作品嗎？」
。
「名字是格倫·格里希，約百年前的作家的簽名都寫在背面哦。一般人都很少知道他的作品，老朽也不是知道地很詳細。只是作品數量很多，因為在鏡框的背面有特徵的簽名所以在某些繪畫愛好者中很出名。雖然是這麼說，這對這個畫家並沒有多少附加的價值」
。
「那麼為什麼您會留意這幅畫呢？」
。
「實際上格倫只會畫自己親眼所見的風景畫，這是個挺有名的故事呢」
。
聽完，我對燃燒的街道的繪畫另眼相看了。
這似乎是在百年前，格倫親眼目睹的慘狀。
。
「……原來是這樣，這的確不是可以隨便賣出去的東西呢」
。
「這幅畫不可以隨便賣出去的原因，不只是這樣而已哦」
。
約瑟夫走近繪畫，用手指著在高層建築物上的男人。
。
那是一個長著綠白色長卷髮的貴族風的中年男人。
身上穿著刺滿刺綉的真紅的上衣。
或許是因為上衣太大，看上去只是衣服膨脹而已。
他俯視著街道，放聲大笑。
。
「……知道自己已經沒救，所以就發瘋了嗎？」
。
約瑟夫靜靜地搖頭。
。
「這個男人其實和某個有名的人物很相似。酋雷姆·奧諾雷亞伯爵有聽說過嗎？」
。
「……酋雷姆……啊啊，有聽說過」
。
酋雷姆伯爵就跟都市傳說差不多。
住在羅馬努街道時也聽說過好幾次。
據說是在大約六百年前的迪恩拉特王國的伯爵，在某一天卻行踪不明。
但從那開始的百年之后，不僅在迪恩拉特王國內，在世界各地都開始有人目擊到和酋雷姆伯爵相似的人物。
有時身處在歷史上殘留的大事件的現場，一個人在遠處大笑。
。
……但在我看來，只是剛好有類似的人物在場，大概是為了炒熱話題和引人注目才吵嚷著‘我也看到了’‘我也看到了’的吧。
六百歲還沒有衰老什麼的，即使是高等精靈也不太可能。
。
「雖然我是第一次看見酋雷姆伯爵的樣子……就這麼吻合嗎？」
。
「是的。……不過，格倫的確是被卷入宗教團體的恐怖襲擊而身亡的，但在老朽看來那只不過是掩蓋真相的小把戲罷了」
。
約瑟夫那故意表現得很嚴肅的表情放鬆下來，就像是開玩笑一樣地笑了。
美亞聽著約瑟夫的話，瑟瑟發抖地緊握著我的手，但聽完結局後減輕了握力，放心地嘆了口氣。
。
「請不要嚇唬人家啊……」
。
「吼吼吼，所以我才說不要隨便賣去其他地方啊。如果這明顯是惡作劇的作品的話，還有可能會讓拉爾克大人蒙羞呢」
。
不管怎麼說，拉爾克的祖父真是喜歡收集一些奇怪的東西呢
我又看了一眼在畫中大笑的男人，也嘆了一口氣。
的確，畫的就像是在嚇唬看畫的人一樣。
在畫面中，只有男人的周圍表現得很露骨。
。
「拉爾克先生的工作也差不多該結束了，他說想和你商討一下關於領地今後的事情呢」
。
我離開那幅燃燒的街道的繪畫，往拉爾克的方向移動。
只是在中途感覺到一陣惡寒，我回頭一看。
。
對在燃燒的街道中央的男人，我總有種似曾相識的感覺。
然而我不以為然，很快又轉身離開了。
