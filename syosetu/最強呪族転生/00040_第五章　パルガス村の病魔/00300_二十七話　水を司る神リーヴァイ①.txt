其他教徒们也不知如何是好，只是狼狈不堪地在原地徘徊。

『不必警觉。亚贝尔·贝雷克，汝有意愿成为余的手下，填补四大神官的空缺吗？汝的魔术手腕，余一直都有在观察……余十分满意』

「什麼！万万不可啊，利维伊大人！」

尼禄古力夫从旁插 入。

。
『在不久的将来，余会让汝和萨特利亚交换，成为教皇。如何？顺从於余的话，余会把利维拉斯国的一切为汝準备。之前發生的事既往不咎，和余一同将这个世界收入囊中吧』

利、利维拉斯国的教皇？
看来利维伊是在劝诱我的样子。
虽然明白它话中的含义，但大脑却跟不上理解。
。
『余以为，只将神枪夺回，还不足以对「它」竖起反旗。万万没想到，会在此处得到两只棋子』

「对它，竖起反旗？」

利维伊不是處於顶端吗，难道在它上面还存在着什麼人物？
。
『同盟的决裂实为必然。库库……虽说余已经察觉到，「它」也并没有把矛头指向余的餘力。嘛，这些话在汝向余宣誓忠诚、余判断汝值得信赖之後再详谈。那麼，让余听听汝的回答……』
。
「শিখা火焰啊　এই হাত具象成球状」

我咏唱起咒文。
我可不想和利维伊教的人同流合污，而且要是拒绝的话，对方很可能会立即对我进行攻击。
那就只好先下手为强了。
。
在锡杖的一端生成炎之魔球，永无止境地注入魔力，并用结界将火球压缩，强行使其保持球状，进而源源不断地增大能量。
老规矩，用亚贝尔玉伺候。
。
『这就是汝的回答吗，亚贝尔·贝雷克。真是愚不可及。……然也。那麼，回收其中一只棋子作为战利品也好。神枪啊，到余的身边来』
。
利维伊的手指一弯，它的手上便出现苍蓝色的巨枪。
。
巨枪的颜色单一，枪柄与枪尖似乎都是以统一的苍蓝色矿石所制成。
大量的术式就像是花纹一样刻满在枪上，没有任何空餘的地方。
。
「利、利维伊之枪！」
。
利维伊之枪是世界上最有名的武器。
被认为是利维伊的一部分，不存在正确的名称。
。
利维伊拿出来的，恐怕是真货。
我曾经在书上看过，利维伊所持的枪，有着无论在世界的何处、只要动一动手就能回到持有者手中的能力。
。
书中记载着，利维伊之枪拥有着即使打偏，也可以通过改写命运来精确地命中目标的力量……。
虽然我对这作弊能力感到半信半疑，但在面对着这份威压的现在，实在是难以否定。
毕竟利维伊之枪很有名气，虽然相关的情报已经在第一时间掌握了，但是否有应对的方法就完全是另一回事了。
关於利维伊使用神枪，在神话时代大闹特闹的描述也残留至今。
在信仰着其他神灵的地方打出一个大洞，以神枪之力让其他神灵敬而远之，贯穿了库多尔的身体之类的传说不计其数。
。
『尼禄古力夫，看好了。难得的机会……就让汝看看余真正的力量吧。毕竟以汝们人类的肉身，根本發挥不了这个宝具真正的价值』
。
利维伊的手腕聚集苍色的魔力，利维伊之枪与此共鸣，被魔力包裹，發出连本体都无法让人以肉眼观察的光芒。
。
「万、万万不可！这、这麼做的话，连老朽也会被波及……！」
。
『在玛利亚斯之後，要是连汝也失去的话，四大神官便只剩下作为装饰的萨特利亚和那个愚蠢的人了。毕竟还有要回收的棋子……放心，余会适当地调整範围。但……汝可别随意走动』
。
被光芒缠绕的神枪的准心，径直地瞄准了我的方向。
。
『永别了，亚贝尔·贝雷克。毁灭吧，然後光荣地死在余之手下吧』
。
利维伊攥紧了手中的神枪。
。
「可、可恶！」
。
我把杖一挥，将完成了的业火、亚贝尔玉投向利维伊。
利维伊迟了一步，也投出利维伊之枪。
。
亚贝尔玉和利维伊之枪两者在空中相撞。
从对撞的枪尖上释放出强烈的极光，全世界都被其染成白色。
下一刹那，轰鸣和爆炸的气浪向我们袭来。
我趴在地上，勉强地承受住暴风。
。
视觉和听觉仍未恢復。
不过我还活着，也就是说亚贝尔玉扛过了神枪的一击。
我擦拭脸庞，观察上空。
朦胧地看见形似神枪的物体从半空坠落。
神枪虽然保留着形状，但威力似乎已经完全地被我抵消掉了。
。
「……这，真的是神话最强的武器吗」
。
原以为至少能偏移一下枪的轨道，没想到这麼不堪一击。
说不定是假货吧。
不管怎麼说，在靠近利维伊那一边發生碰撞真是太幸运了。
要是再靠近这边一点的话，就不只是暂时失明这麼简单了，也许还会被风暴吹飞到不知道什麼地方。
。
虽说如此，刚才的一击估计也对利维伊造成不了多少伤害吧。
那支枪拥有三个有名的特性。
我再度抡起世界树之杖。
。
「হন传送吧」
。
八重魔法阵……将處於地面的我和从空中坠落的神枪包围，展开了巨大的立方体。
欧特姆从各个魔法阵的中央出现。
。
「বাধাসজ্জা结界展开」
。
犹如将神枪团团围住一样，光之屏障在欧特姆与欧特姆之间张开，立方之结界就此完成。
这样就能妨碍由外向内的魔术干涉了。
。
『万万没想到，居然能抵消神枪之一击！看来是有些小瞧汝了，但从现在开始……将余枪之本领牢牢地刻印在脑海裡，陷入绝望吧。神枪啊，回到余之手中！』
。
然而神枪并没有回应利维伊的话语，只是一面地往附近的教堂坠落。
但尼禄古力夫还站在屋顶上。
。
「哦、哦，哦哦哦哦哦哦哦哦！」
。
尼禄古力夫从教堂顶上跳了下来。
下一瞬间，教堂伴随着一声巨响而坍塌。
虽然尼禄古力夫的身影被烟尘所掩盖，不知道情况如何，但从三层建筑的屋顶上跳下来肯定不只是摔伤这麼简单。
。
『回到余手中！回到余手中！为、为何……为何回不来了！回到余之手中！回不来了吗！』
。
在遥远的上空，利维伊正在拼命地来回翻弄着手指。
。
『为、为何！？是那张薄弱结界的缘故吗！？神枪之力难道并非優先于任何魔法干涉的吗！回到余手中！』
。
看来它是想立刻放出第二击。
用结界封印住干涉真是太好了。
……倒不如说，这个神大人难道没有神枪就什麼都干不了了吗。
。
我靠近已经倒塌的教堂，用杖指着瓦砾。
。
「বা土啊　বাহু হাত具象成手腕吧」
。
崩塌的教堂的残骸被推开，巨大的土之手腕腾空而起。
我操纵土臂，让其握住利维伊之枪。
。
『回到余手中！快回来啊啊啊啊！』
。
「既然你这麼想要，我就还给你好了！」
。
我把枪投向着伸出利维伊手臂的空间裂缝。
神枪笔直地飞行，穿过利维伊的手臂，就这麼贯穿浓厚的乌云，在天空打出一个大洞。
。
『……汝、汝，刚才做了……』
。
利维伊的手臂似乎是不受神枪影响的样子呢。
我只看见神枪穿透了利维伊手臂的一幕。
大概，没打中吧？
。
这麼想着的时候，在利维伊的手臂上出现了一个逐渐扩大的大十字风洞，利维伊的手臂也开始逐渐瓦解。
。
『啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊啊！！？？』

扩大後的风洞完美地将利维伊的手臂切断，与本体分离的手臂从裂缝处坠落。
落在地面上的满身疮痍的手臂，仍在精神奕奕地左右挣扎。
就像是在地上活蹦乱跳的鱼一样。
。
『亚贝尔·贝雷克！余记住汝的名字了！汝是永远的灾厄！将来会是永恒的不幸！余的信徒绝不会饶恕汝的暴行！』
。
不不不……别诅咒我啊，我就在你面前，有本事你现在出来啊。
。
利维伊的手臂仍残留在地上，但空间裂缝已经开始关闭。与此同时，云上的空洞也犹如旋涡一样逐渐扩张，天空再次放晴。
。
「神枪……真是浪费呢」
。
或许是便宜货，但肯定也算得上是高价的魔法道具吧。
毕竟也有承受亚贝尔玉的强度，以後可能就没机会再获得了。
没有缠上魔力，只是像标枪一样扔出去就被贯穿什麼的，利维伊本身似乎也并不是那麼强的样子。
。
虽说利维伊并没有其他能够出其不意的攻击手段，但一时冲动把枪扔出去的行为也值得反省。
完全丢失还好，但只要利维伊还存活，神枪便又会回到它的手上。
因此应该把利维伊之枪封入结界之内。
。
「机会难得，利维伊的手臂也得回收一下……嗯？」
。
我把视线移向利维伊的手臂上。
被一分为二的利维伊的手臂和刚才一样，仍在痛苦地来回挣扎。
就像是蜥蜴的尾巴一样。
。
不过，我在意的不是这件事。
在分为左右两半的利维伊的手背上，分别浮现出半边的象征着枪的纹章。
。
某些地方仍存在着疑问。
神枪到底是如何判别所有者的呢。
因为是神话所以也并没有深入地思考，但假如那个纹章也和恶魔的召唤纹一样，拥有着同等的效果的话，那就可以理解了。
。
在持续了短暂的寂静之後，村子裡的人纷纷發出欢呼声。
重新环顾四周，有村民正在手舞足蹈地欢呼，也有经过这幾分钟後、一口气老了幾岁利维伊教徒。
。
「快、快来人啊！这样下去老朽要死了……谁、谁来治疗一下……！虽然老朽自己做了应急处理，但老朽被瓦砾压着不能动弹！老、老朽还不想死……！」
。
忽然听见尼禄古力夫求救的声音。
。
「…………」
。
不管怎麼说他也是重要的证人。
可不能让他死掉。
。
在我打算靠近尼禄古力夫的时候，我發现教徒和村民们正往尼禄古力夫的方向走去。
我停下了脚步。
。
尼禄古力夫的言行虚伪到何种程度，我也不得而知。
但即便如此，尼禄古力夫也必然在教徒和村民的心中留有一定地位。
他们肯定不会那麼轻易地忍痛割爱吧。
。
「等、稍等一下！误会！老朽也被骗了！是真的！老朽、老朽……！老朽明白！现在正是把话说清楚的时候！所以，请高抬贵手吧！」
。
「咦……？」
。
村民和教徒露出愤怒的神色，各自拿起石头和瓦砾，慢慢地把动弹不得的尼禄古力夫团团围住。
我慌张地跑向尼禄古力夫的所在之处。
