
三天过後，我在击败利维伊的广场上接受海尔村长的表彰。
。
「亚贝尔阁下，衷心感谢您拯救了帕尔加斯村。我代表全村人向您表示谢意」
。
海尔村长深深地弯下腰，亲手将表彰状颁发给我。
从四面八方响起了村人们的掌声。
我瞄了一眼不知为何自豪地昂首挺胸的梅娅，也向海尔村长微微鞠躬。
。
「不、不客气……」
。
暴晒在一群陌生人的视线中的我实在是静不下心来。
老实说，我是真的很想去个安静的地方专心雕刻欧特姆。
。
打败利维伊之後的事宜进展得很顺利。
被村民们用木棒围殴的尼禄古力夫也回收成功。
说实话这货就算死了也跟我没太大关係，然而不知是幸运还是不幸，明明从三楼摔下来还被围殴，竟然也只是断了八根骨头。
。
教徒们似乎也对尼禄古力夫和利维伊的醜態感到万分震惊，幾乎所有人都放弃了抵抗，在村民们的帮助下轻鬆地逮捕了他们。
根據村民们从教徒那裡供出来的真相，教徒们虽然从一开始就清楚，这是一场以侵略为目的的自导自演，但尼禄古力夫主导的、将利维伊作证的恶魔裁决用於作恶一事，以及尼禄古力夫出卖克洛伊一事好像对他们造成了相当大的冲击。
。
总而言之，我火速和领主拉尔克取得联繫，向王都报告後等待对方的指示。
他们也许会跟玛利亚斯那一派人一样，被护送到其他领主的管辖地吧。
。
多亏了莉芙格拉斯的效果，海尔村长在事件发生的翌日就已经取回了意识。
接着他取代了霸占村子的利维伊教徒，正在对村子进行整顿。
说起来作为我到访这裡的缘由的、拉尔克偶然入手的纸片，似乎是以信鸽为媒介的书信交流被利维伊教徒妨碍，是海尔村长拼上性命传递的其中一个碎片。
海尔村长愤慨地表示，自己在寄出大量的书信时就已经被盯上，也许就是在这个时候中了病魔的诅咒。
。
同一天，我威胁尼禄古力夫，让他召唤出作为病魔元凶的恶魔并立即消灭，成功地将病魔赶出帕尔加斯村。
虽然还有的人身体不太舒服，但顶多只是感冒的程度而已。
静养幾天就会恢復的吧。
。
表彰完结过後，我回到原·利维伊教的教堂裡，累得把下巴撑在桌子上，
。
「啊……累死老子了。让欧特姆替我上台不也可以吗」
。
「……怎麼说也太失礼了吧」
。
毕竟机会难得，我将变成一片废墟的利维伊教堂作为住所，和梅娅、艾丽娅(车夫）一起住了下来。
顺带一提村裡头的方针是，将建造而成的部分教堂作为其他设施而循环利用，剩下的似乎都被拆了。
海尔村长曰『看着就觉得不爽』。
。
「哈啊」
。
坐在椅子上的艾丽娅叹了口气，眼神就像是死掉一样。
艾丽娅从教团中被释放後，和其他村民们一起去践踏了利维伊教团的披风。
毕竟她也算是一个相对来说很温和的人，所以有点意外。
看来是积累了不少压力呢。
。
「艾、艾丽娅小姐，你没事真是太好了……哈哈哈……」
。
「…………」
。
艾丽娅眯着眼，用锋利的目光注视着我。
。
「讨、讨厌啦艾丽娅小姐。您、您最近，真是越来越会开玩笑了呢。哈哈哈哈……」
。
看上去似乎不是故意发怒的样子，即使我含糊其辞，艾丽娅的眼神也是坚定不移。
看、看样子是真的在生气呢。
说不定是对我们外出狩猎莉芙蛙后，把她遗留在教团之中的事情耿耿于怀吧。
。
顺带一提这个教堂裡也藏着战利品。
从利维伊教的行李中发现了大量的阿洛古雅石。
原本的话，这些都必须作为扣押金而送去王都，但我请求海尔村长将一部分石头作为战利品私吞了。
要是还有什麼其他有趣的东西的话也尽量转让给我，我向海尔村长嘱咐。
。
「这些……真的可以拿走吗？」
。
梅娅偷瞄了一眼堆积如山的阿洛古雅石，坐立不安地询问我。
。
「反正也得到了村长的允许，而且我也付出了相当的努力……就、就这麼点，无所谓吧……」
。
万一被责备了，只要事後低头道个歉就没事了吧。
在迪恩拉特王国内，阿洛古雅石可不是能够轻易入手的东西。
把阿洛古雅石拱手让出去什麼的，实在是捨不得啊。
就算不是我，只要是魔术师的话无论是谁都会必然这麼做的吧，大概。
。
除此之外还有别的战利品。
往右手的手背注入魔力的话，手背上便会浮现出象征着戟的纹章。
这是刻印在利维伊手臂上的利维伊之枪的召唤纹。
我将利维伊的手臂保管好，花上一天将召唤纹连同内藏的魔力一起转移到自己手背上。
。
虽然调查过後才知道，利维伊之枪其实是将精灵封闭在用鍊金术生成的特殊金属中的魔法道具，可以视为单只恶魔。
简而言之，这就跟我所制作的拉碧丝之剑一样。
。
实际上，发动召唤纹之後真的可以把利维伊之枪移动到手边。
但由於体积过大，失手召唤出来也只会毁掉附近的建筑物。
。
可以说利维伊之枪是这次最大的战果。
虽然还不知道是不是真货，但肯定是相当高级的魔法道具没错了。
在解析的过程中，我甚至不由自主地发出了赞叹。
要是私吞阿洛古雅石被责难的话便诚恳地道歉并上交回去，但只有利维伊之枪我打死也不想上供给国家。
。
利维伊的手臂虽然也庄重地保管在教堂地下的结界裡，但这手臂却突然发生异变，结果还是坏掉了。
因为对异变很感兴趣所以前去视察了下情况，但说不定在下一刻就突然暴走、将附近触手可及的物体全部破坏。
将其石化之後，再慢慢研究是最安全的选择。
。
如果前幾天遇到的利维伊是真傢伙的话，那麼这只手臂就是本应在神话时代死亡的神明的其中一部分了呢。
肯定是价值连城的。
。
然而对我来说最震惊的事情是，利维伊自身也是精灵。
虽然从古至今就有学者对外主张，四大创造神其实是为了永生而赋予了过大力量的恶魔，但当中被教团盯上并被處刑的人也不在少数。
虽然这也只是众多假说中的其中一种，但如果真的是创造神的话，就不可能由从生命的碎片中诞生的精灵所组成，神话的大部分内容都只是四大创造神为了将自身粉墨成支配者而创作的作品。
毕竟是创造神所创造之物，即使用与创造神的身体同样之物构成灵魂也不稀奇。
。
只看这次的话，那个自称利维伊的傢伙也许只是一只纯粹的恶魔而已……不管怎麼说，想活得久一点的话，还是假装不知道比较好。
虽然教徒们称之为真物，然而在村人们之间传出一种理解，那就是利维伊教的一部分过激派被恶魔的演技给骗了。
他们也準备向王都这麼报告。
。
「……说起来，『那个』到底在哪呢？」
。
「诶，『那个』……？啊啊！亚贝尔是在说，记述着莉芙格拉斯的制作方法的那张纸吗！是梅娅发现的哦！」
。
「还以为铁定被烧掉了……真亏你能找到呢」
。
「嘿嘿嘿……。其实，是黏在厨具棚裡面的锅下面了」
。
梅娅腼腆地笑了笑，有些难以启齿地，不好意思说出口。
。
「啊……原来在那裡啊……」
。
为了寻找适合于制作莉芙格拉斯的道具，我们曾把海尔村长家裡的所有锅都拿了出来作比较。
也许就是在那时夹了一张在锅下面，然後就这麼一直贴着。
。
完、完全没有保管好嘛……。
虽然至少在记忆中是打算好好保管的，实际上却是处理地相当随便。
也许是由於沉迷于莉芙格拉斯导致意识模糊，没办法嘛。
不过也因此让尼禄古力夫原形毕露，从某种意义上来说我得感谢一下这张配方。
结果allright。
。
正当和梅娅、艾丽娅她们交谈时，我听到了门铃响起的声音。
。
「……嗯？是海尔村长吗」
。
「梅娅去看看」
。
「啊，我也去……」
。
结果我和梅娅两人来到玄关，把门打开。
是我们都认识的扎着马尾辫的少女。
。
「太—好—了！梅娅酱没事吧？」
。
少女猛扑向梅娅，就这麼抱着她。
。
「抱歉啊！我都不知道，我介绍的地方居然出了这麼大的事情！去跟领主见了一面，听他说亚贝尔酱会想办法的我才松了口气，但他居然又信口雌黄地说什麼把你们送去了危险的地方！我都快担心死你们了……」
。
她是我们在雅西姆街道认识的女冒险者莎姆。
不仅告诉了我们玛连族来到了罗马努街道的情报，还将珐基领介绍给我们。
听闻珐基领发生了大事，於是便慌慌张张地赶来这裡打听情报。
。
「太、太夸张了吧莎姆小姐。没事的，毕竟有亚贝尔在嘛……」
。
梅娅说完後，莎姆便将视线落在我的身上。
眯起了像猫咪一样的圆滚滚的大眼睛。
。
「不可以哦，亚贝尔酱。虽然你对魔术是很有自信，但最好别把梅娅酱带到这麼危险的地方哦。……毕竟介绍珐基领给你们的是我，不会勉强你们的」
。
「好、好的……」
。
的确，梅娅总会奉陪我的好奇心，跟我一起前往危险的地方。
特别是这次，只要走错一步就无法挽回了。
实际上，我粗心大意的行动也有可能会导致梅娅被当成人质。
。
「下次会注意了……」
。
正当我垂下头时，梅娅生气地撇了撇嘴。
。
「梅娅是喜欢才和亚贝尔在一起的，没有任何问题！而且梅娅更讨厌留守！虽然梅娅很感激莎姆小姐，但我们之间的事情请你不要插嘴！」
。
「梅娅……！」
。
「……梅娅酱不可以太宠溺亚贝尔酱哦。这孩子，不好好拴住的话肯定又会跑去别的地方了。到那时候，要是一头扎进了危险的地方就真的会死哦。恶魔的话，真正危险的傢伙可是深不见底的呢」
。
「……梅娅，会努力的」
。
「我也会尽力注意的……」
。
「唔姆！乖孩子」
。
莎姆抱着胳膊，用鼻子哼了一声。
。
「不过，确认到梅娅酱和亚贝尔酱没事就好。也打过招呼了！这麼有精神我也就放心了。我还有事，还得回雅西姆街道一趟……」
。
「真的只是来看一眼吗？」
。
「嗯！无论如何都很在意梅娅酱和亚贝尔酱的情况」
。
虽然看上去有些轻浮，但责任感似乎是相当强的样子呢。
也许是认为自己的推荐不太恰当。
。
「总感觉不好意思呢。旅费也花了不少吧。我也替你分担……」
。
「哼哼！毕竟机会难得，我还顺便接下了确认没有从珐基领那裡归来的恋人的平安与否的任务，这方面你不用担心哦」
。
真是机灵……。
不得不提前回去的原因就是这个吧。
。
「那对恋人怎样了？」
。
梅娅担心地询问时，莎姆便捂着嘴笑了。
。
「很好很好！噗噗！毕竟他们也已经有孩子了。日子过得很幸福哦」
。
「太好了！是在拉尔克所在的地方吗？听说有小宝宝哦，小宝宝哦！亚贝尔，待会去看看吧！」
。
…………。
虽然大部分时候被牵连的都是我，不过梅娅看上去这麼高兴，我也不多说什麼了。
