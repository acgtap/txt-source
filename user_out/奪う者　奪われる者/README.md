## preface


```
佐藤 优，12岁
日复一日地过着被继父虐待的生活
有一天为了还债而盯上保险金，进而将优杀了
本应是死了才对,但回过神却发现已身处异世界
这是少年在异世界成长的故事。

佐藤　優（サトウ　ユウ）１２歳　
義父に日々、虐待される毎日、ある日
借金返済の為に保険金を掛けられ殺される。
死んだはずなのに気付くとそこは異世界。
これは異世界で少年が成長していく物語です。
```
