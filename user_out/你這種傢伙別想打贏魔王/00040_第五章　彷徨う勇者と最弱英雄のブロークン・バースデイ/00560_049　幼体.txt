那么讓它反轉的話，就會開始反向旋轉，發出非正而負的能量，消失不見的吧。
芙拉姆特意停下腳步，向著自己飛來的看不見的力量揮出劍──

Reversal

「反轉！」

使用了反轉的魔法。
這樣一來發揮了那么大威力的魯克的力量，輕易的就煙消云散了。

「… … 什么？」

他雖然對此目瞪口呆，但還是不吸取教訓地再次出拳。
咚嗡嗡嗡嗡！
划開天空逼近的比剛才凝聚了更多力量的一擊

Revarsal

「清除。」

這次又被芙拉姆空手給清除了。
簡而言之──讓物體回轉的魯克的能力和使它反轉的芙拉姆的能力，相性是最差的。
魯克用五股力量發來的攻擊，芙拉姆用一股力量就能消除。
也就是說，不管他的狀態值有多高，只要他是依賴「回轉」的能力戰斗，就沒有足以擊敗他芙拉姆的能力。
盡管如此，芙拉姆應該除了釋放靈氣以外也沒其他能夠攻擊這個高度的手段了吧──魯克這樣預想。
但是那最后的希望，也被她輕易的粉碎了。
芙拉姆反轉重力，踢向地面一躍而起。
她把刀刃放平，劍舉到腰部以下，以這種姿勢瞬間向空中的飄浮著的魯克逼近。

「不可能，竟然飛起來了！？」

驚慌失措想要射出螺旋，但為時已晚。

「哈啊啊啊啊啊啊！」

以砍掉頭部為目標將噬魂揮了過去。
為了用空氣的回轉抵消這個攻擊，魯克把右手伸向前方。
嚓咻！
但是寄宿著反轉魔力的刀刃，無視他纏繞這的力量把手臂砍掉了。

「嘎，啊啊啊啊！」

失去了手肘以前的部分，如灼燒的感覺進入了魯克的大腦。
傷口很快扭曲，出血也止住了，但是劇痛帶來的目眩還沒有平息。
解除重力反轉的芙拉姆在落下的過程中扭動身體發出氣劍斬。
魯克可能是因為被傷口分心反應慢了。
攻擊即將到來前才試著讓身體傾斜來躲避，但還是差一點，沒能躲過肩膀表面和耳朵都被砍掉的結果。

「咕唔唔… …！」

魯克在空中搖搖晃晃左右搖擺，終于從那個高度掉了下來。
由于受傷，能力控制似乎出現了紊亂。
安全著陸的芙拉姆，瞄准掉下來的少年跑過去。
看到這個，盧克特意解除能力加快落下速度，放棄勝負試著逃跑。
但是──芙拉姆的向他攻擊過去的速度明顯更快。
接近過去，奪走少年的性命，為此芙拉姆兩手用力握住了劍柄。
魯克意識到會受致命傷，把手伸進口袋，決定使用Mother讓自己拿著的那個。
但是──

「嗚，嗚哇啊啊啊啊啊啊啊啊啊啊！」

在分出勝負的那一瞬間，從教會里傳來了與斯勞相似的男性的尖叫聲。
集中精力的芙拉姆不去在意的揮出劍。
但是不知是不是無意識的受到影響，沒有足夠的接近。
那一刀在左上方留下了一道深深的傷口，但是沒能砍下他的腦袋。
芙拉姆還想進一步追擊的樣子。

「這樣好么，那個小少爺會被第三代吃掉的哦？」

魯克輕笑著說到。
第三代──那是魯克他們下一代的螺旋的赤子的意思。
雖然有虛張聲勢的可能性，但是斯勞正遭遇危機也是事實。
在芙拉姆猶豫不決的時候，盧克利用回轉進行高速移動瞬間離去。
雖然也不是追不上，但不能對斯勞棄而不顧。

「啊真是夠了，又從我面前逃走了！」

芙拉姆懊悔地跺著腳，向教會內部走去。


◇◇◇

──時間稍微回溯。
安全逃進禮拜堂的斯勞坐到最里面的長椅上，背靠著把整個身體都搭了上去。
雖說是一個神聖的地方，沒有人的教會還是有點兒讓人不舒服。
也許也是因為他之前看到的臉像內脏一樣的怪物──魯克的錯。
斯勞呆呆地望著天花板、祭壇和起源像，感覺得那雙眼睛像看過來了一樣，呼著氣同時閉上了眼睛。
從外面傳來芙拉姆和魯克戰斗的聲音。

「芙拉姆小姐，沒關系吧… …」

被比自己年少的的少女保護，作為男人真是不能更丟臉了。
本想著自學練成的風屬性魔法終于到了有機會派上用場的時候，但現實並不是那么的美好。

「但是那個家伙… … 瞄准的是我吧，為什么是我啊？明明只是被普通人的母親生下來養大的，做著平淡無奇的工會工作人員而已。」

沒有任何能聯想到的部分。
說到他的人生，特別的事情，就只有完全不知道父親的長相這件事了吧。
然后就這樣──沒錯，一味地過著平淡無奇的普通生活。
而且今后只要有合適的收入，過著普通的生活就足夠了。

激烈的戰斗聲不斷地持續著。
閉上眼睛的話，視覺的部分就會被分配給其他感官，可以更清晰地聽到聲音。
這樣做了之后，不是從外面，而是從里面──從教會的深處，發現混雜的另一個聲音。

「… … 哼… … 啊… … 啊哇… … 啊… …」

斯勞直起身，盯著禮拜堂后面的門。

「有誰在么？」

如果說還有修女在的話，為什么不來看外面的情況。
如果展開了那種程度的戰斗的話，應該已經注意到異常了。
接近房門的斯勞，緊張的把手伸向把手。
從這里往前，是普通人不能進入的區域。
如果用商店比喻，這就是寫著職員專用的門前。
但現在是緊急時刻，一定就算擅自進入也沒人會在意。

「打擾了。」

即便如此，斯勞還是小聲說道，戰戰兢兢地，壓著腳步聲前進。

「好孩子… … 我… … 喲。嗯呼呼… … 呼呼… …」
「笑聲，么？」

雖然只是一般的女性在笑著，但是考慮到狀況不妙，就更加的令人毛骨悚然。
顫抖著身體的斯勞，更加走近發出聲音的房間，把耳朵貼在門上。

「我的孩子… … 只屬于我的… … 可愛啊，為什么會這么可愛啊… …」

看來房間里有一個疼愛著嬰兒的母親。
原來如此，如果有一個剛出生不久的孩子的話，那么留在教堂里也沒什么奇怪的。
斷定了聲音的真實身份后松了一口氣的斯勞，離開了那個地方回到了禮拜堂。

「哎呀… … 怎么… … 了？和媽媽… … 嗯… … 么？」

咚、咚。
斯勞走在走廊上的時候，感覺好像有什么巨大的東西在移動，連地板都在顫動。

「… … 不行，還沒… … 哎呀，那樣… … 啊… … 的話，殺… … 別哦。」

砰，砰，砰。
那聲音一點點接近了。
然后，想著暫且停下來了──咣！剛才把耳朵貼在上面的門從里面炸開了。
木制的那個門以巨大的力量撞在對面的牆壁上，變得粉碎。
被強烈的聲音嚇得渾身發抖的斯勞回過頭來，發現那里是──

「啊啊啊啊啊啊啊啊啊啊啊啊──！」

房間里只露出了一張臉，是一張嬰兒的臉。
只不過那個大小，僅臉部寬度就能超過一米。
從半開的口中發出粗大的聲音，透明的粘液大量灑出滴落。

「嗚… … 嗚啊，啊… …」

他的眼睛凝視著什么力量都沒有，什么都不知道，普通的平民斯勞。

「嗚，嗚哇啊啊啊啊啊啊啊啊啊！」

他不可能承受得住這種恐懼──斯勞直不起腰癱在地上，發出了讓喉嚨破壊程度的尖叫。
