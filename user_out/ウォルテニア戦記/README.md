## preface

```
高中生御子柴亮真被召喚到烽火連天的異世界。他發現召喚自己過來的奧德梅亞帝國手段卑劣，在來到異世界的人失去利用價值後便會棄之不顧，於是發揮他擅長的武術，逃出奧德梅亞成為冒險者。某天，亮真拯救了一對被盜賊挾持的雙胞胎姊妹。擁有「法術」之力的她們獲救之後，宣誓服從亮真──以此為契機，亮真開始邁向成為霸王之路。
```
