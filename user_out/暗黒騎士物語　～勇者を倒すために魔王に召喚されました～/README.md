## preface

```
译名：暗黒騎士物語  ~为了打败勇者，被魔王召唤了~

这是主人公作为暗黑骑士，为了阻止勇者保护魔王而被召唤的故事。

突然被召唤到异世界的黑树，以及召唤了黑树的魔王莫德斯。然后，现在魔王所支配的国家正处在眼看就要被勇者毁灭的时候。
拥有凶恶外表的魔王莫德斯，向黑树地下了头。

「拜托您救救我吧！！救世主大人！！」  

于是黑树顺势成为了暗黑骑士去帮助魔王。
```
