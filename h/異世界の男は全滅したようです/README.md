## preface


```
俺こと向坂博牟は大学に入り表面上はアニメやゲームが好きな善良な学生を演じながら、裏では喧嘩は日常茶飯事、恐喝やレイプなどいろいろな悪事に手を染めていた。それでも俺はどこか退屈さを感じながらも楽しく過ごしていた。
しかしあるとき、友人とラーメンを食べていると俺は突然森の中にいた。
訳が分からないが、どこか非日常に憧れていた俺は好き勝手に生きていくことに決めた。
```
