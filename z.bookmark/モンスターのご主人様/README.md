```
　　落入異世界並遭同學背叛的高中生真島孝弘，死前偶然獨特的自身異能馴服怪物，成為怪物的主人。

　　在無政府狀態的異界裡飽嚐背叛與紛爭，讓他從此不再信任人類，發誓不再與人打交道。

　　然而在怪物充斥的異世界森林裡，主角又該如何和擁有自我意志的怪物眷族相依為命，一同生存下去？
```
