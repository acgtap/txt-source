杰克（視角）◆
#
接近祭壇的菲爾————
#
————不經意間，就從懷中掏出了刀來。
#
連思考疑問的時間都沒有，刀就被刺入了睡著了的女性的胸口。
由于結界的影響。
無法造成傷害。

　但是。

#
——靈力耗盡了。
在鳥籠中的〈オロバス〉的虛像隨即黯晦消沉
#
「在干什么呢———」

沒能夠介入。
菲爾再一次，將刀刺入了女性的胸口。

#
這次是——

　鮮血、飛舞。
#
　令人吃驚的量。
　令人吃驚的氣勢。
鮮紅鮮紅的液體，像噴水一樣從女性的胸口噴出。

　#
　就在那旁邊的菲爾像淋浴一樣沐浴著飛濺的血沫。

#
手也、腳也、臉也、衣服也、全部被染成紅色。
#
　做什么。
　為什么？

　#
　那樣、做了的話。
結界再也——

#
「———啊啊」

#
恍鐺、鐺。

　將用完的小刀丟到一旁后，菲爾用手背擦拭濺在臉上的血。

#
「……擦不掉呢。算了那樣也好，反正是要扔掉的身體。就算是那樣，也有不得不開心的事情呢。礙事的結界消失了，終于可以開始【掃除】了呢！」

#
那是，菲爾的聲音。
菲爾的臉。
#
然而。
只有笑容不是菲爾的。
是別的誰。
#
我所知道的，別的誰。
在這個世界上最恐怖的、再也不想看到第二次的笑容。
#
菲爾的眼睛看著我。
菲爾的臉朝向我。
菲爾的嘴唇打開——
菲爾的聲音說道。
