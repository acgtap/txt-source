
　罗伊在空中创造的光球正维持着效果，照亮地下城那黯淡的通道。
　由於雷斯迪凯普并没有除此之外的光源，所以也就只能看到光球所照到的狭窄範围。
。
　视野能见度大约是有7亚鲁多(约10米)。
　如果在这视野能见度之外被狙击的话，罗伊姑且不论，我一下子就完蛋了。
。
　虽说视野裡没有魔物存在，可也不容有半点大意。
　在道路上提高警惕走了一会儿，持有自动探知技能的罗伊大声说道。
。
「敌人接近！5秒後接触。给我敌方战力分析和指示！」
「我明白了！」
。
　从现在开始的战鬥流程和指示就会由我来决定。
。
　队伍的领导，指挥官，部队长。
　以一句话来说，我即将要担任这些职务。
。
　可我仍然是新手胆小的魔导师。
　如果造成无法挽回的巨大失败该怎麼办，这一不安让我心跳鼓动加快。
。
　可是，既然罗伊说过我或许拥有这样的才能，那麼就只能拿出自信干到底。
。
　於是魔物真的在五秒後刚刚好现身。
　与魔物接触後应当要做的事情，首先是把握敌人战力的构成。
。
　敌人前卫　巨魔×3、妖精骑士×3
　敌人中卫　妖精弓箭手×2
　敌人後卫　魔术师×2
。
　迅速确认完敌人队伍构成之後，我说出我分析的敌人战力。
。
「敌方战力偏向前卫！由於敌方前卫数量过多，所以集中我方战力逐个击破敌人」
「明白了。具体要怎麼做！」
。
　看到逐渐接近过来的魔物，我心情不由得焦急着我不快点发出指示的话。
　冷静下来，在这地下城的第一层战鬥已经不是第一次了。
。
　如果失败了，还有罗伊会来补救我。
　我深呼吸一次，迅速思考着应当建立的战术。
。
　思考吧。不管怎麼想，最令人惊异的就是中卫的妖精弓箭手。
　妖精弓箭手这种魔物的特征是在战场稍微後方的位置射出弓箭。
。
　妖精那施加上风之魔法加护的弓箭攻击，拥有惊人的命中率。
　如果被射击到要害，拥有一击就使人不能战鬥的危险性。
。
　虽说这是想要立刻就打倒的魔物，可那边被前卫的巨魔与妖精骑士稳稳抵挡着，形成肉盾。
　应该是为了主力战力不会被简单地被击溃吧。
。
　妖精弓箭手被巨魔它们安全地护卫着，发射出必杀的一击。而不让这边受到此致命伤是比什麼都还要重要的事情。
。
　妖精弓箭手在性质上是以弓箭来攻击，所以一旦这边成功接近它。它在近战的战鬥中是非常弱小的。
。
　所以首先是把最具威胁性、最脆弱的要点破坏掉。
。
　我张开声音喊道。
。
「以敌人中卫的妖精弓箭手作为第一目标进攻。
　由我来用雷射弹组成弹幕来吸引敌人前卫。
　趁这一空隙，罗伊就突破中央。
　到达敌人中央後，尽可能迅速地解决掉妖精弓箭手。
　就这样冲到後面也把魔术师解决掉。
　然後我在前，罗伊你从後面夹击在战场中央被孤立的敌方前卫，就这样决出胜负！」
。
「了解」
。
　罗伊微微点头，拔出剑以猛烈的速度奔跑着。
　虽然我自己也觉得我建立的战术并不是满是漏洞，看来那句所谓的「我来培育鲁库」是相当认真的样子。
。
　他是在信赖着我。
　只见罗伊毫不怀疑我的战术，往敌方中央突击而去。
。
　对此，我快速发射全部的雷射弹来掩护他。
。
　雷属性子弹犹如下雨一般往魔物倾注而下，虽然还达不到致命伤，可巨魔和妖精骑士都微微地产生伤害硬直状态。
　罗伊并没有放过这一瞬间的空隙。
。
　他以仿佛如同疾风一般的速度横跨战场，瞬间冲到敌人中央处。
　一转眼就摆脱敌方前卫的他，与妖精弓箭手对峙着。
。
　被接近到眼前的妖精弓箭手无法发挥弓这一种武器的最大长处，就已被罗伊秒杀。
　可魔物也不愚蠢，它们开始以包围歼灭阵型袭击而去。
。
　巨魔举起棍棒，而妖精骑士是以铁剑砍下。
　罗伊不得不暂时专心躲避敌人的攻击，并不能打倒第二只妖精弓箭手。
。
　然而，这种时候才是身为中卫魔导师的我出场。
　在稍微远离一点的地方发射猛烈的雷属性子弹，攻击巨魔它们的後背。
。
　这是为了牵制它们的行动，使行动受到限制。
。
　巨魔这种拥有坚硬皮肤与刚毛而具有优秀防御力的魔物，以雷射弹根本就无法造成多大的伤害。
　但是，即便只是被蚊子叮咬的程度，接二连三地受到雷射弹攻击也会感到厌烦。
。
　巨魔和妖精骑士看到我，发出烦躁般的咆哮。
。
「嘎啊啊啊啊啊！！」
。
　是决定先要攻击弱小的我吗。
　巨魔它们无视罗伊给予的攻击，愤怒到气喘吁吁地朝我这边冲来。
。
　可我也不是会老老实实地被幹掉的白痴。
　张开雷属性子弹的弹幕，边利用在道路上的大岩石和拐弯处等等的障碍物，边与魔物保持一定的距离适当地後退。
。
　巨魔高高举起地棍棒也被火焰球给击落。
　而魔法伤害对於妖精骑士比较有效果，所以用雷射弹攻击来阻止其步伐。
。
　逐渐後退边展开弹幕，尽量争取时间。
　如此一来，罗伊就能更加从容不迫。
。
　他已经把第二只妖精弓箭手砍死了。
。
　拖延战术。
 在我方主力救援之前，尽量使敌方焦急并引诱它们。这是我临场思考而出的战术。
。
　这对於战鬥力缺乏的我来说，不管是拖延敌人行动，还是妨碍敌人都是最能起到作用。
。
　在和罗伊组成的两人队伍裡面，我的职责并不是主力攻击手。
　只要交给最高火力的罗伊来做，不管有多少敌人都会被打倒，所以我只要辅助他或者妨碍魔物就可以了。
。
　只见罗伊也好像干掉了後卫的魔术师，回过身往这边过来了。
。
　这样一来
　我→巨魔・妖精骑士←罗伊
　就会形成这样的一列排序，我在前罗伊在後攻击巨魔它们这样的一个夹击作战就成立了。
。
　连突破雷弹幕都甚至陷入苦战的魔物们，在被天才剑士・罗伊从後面攻击的情况下，惊慌失措着。
　前有烦人的雷弹幕倾注而去，後有罗伊所释放的如同闪光的一击袭来。
。
　漂亮地使得魔物的队列崩溃了。
　不久之後，魔物全部被打倒消灭掉，只留下魔石而已。
。
　呼……。
　初次指挥的战鬥总算是胜利了。
。
　如今心臟还仍在啪嗒啪嗒地跳动。
。
　没有什麼地方做错了吗，有什麼地方出现致命性的失误了吗，不安到毫无办法。
　看样子，我所想出来的幼稚拙劣的作战姑且是以成功收场了。
。
「辛苦了」
「您也辛苦了」
。
　我跟平时一样在战鬥记录上书写着，而罗伊在捡魔。
。
「鲁库，这些给你」
。
　罗伊边说着边把魔石交给我，魔石比平时给我的份量还要多了。
。
「今天很多呢」
「嘛，这也就说期待你今後的表现吧」
「结果，只不过是依靠罗伊的输出作战……」
。
　罗伊一走，我甚至连能不能与魔物正面战鬥都感到蹊跷。
。
「因为我们现在还只是在第一层战鬥而已，所以魔物的战力也不是那麼惊人。即便陷入困境，以我一个人也能简单地挽救回来。
　但是，一旦在更深处的阶层战鬥的话，魔物也会越来越强。也更会採用狡猾恶心的作战狡猾。
　以我的力量是绝对抵抗不过的。这是我的亲身经历。
　所以你的力量是必要的」
。
　听到罗伊这番话，我默默地点下头。
。
「虽然你现在或许还是刚起步的冒险者。可从今以後，你所思考的战术会越来越熟练，如果能摆脱危机的话，对我来说就是帮大忙了」
「这个地下城……雷斯迪凯普究竟有多少阶层？」
。
「我所确认到的是25层为止。甚至连我也只是去过一次25层。而且还是不知道是幸运所眷顾还是不幸，触发陷阱一口气掉下五层直接让我临近濒死才终於到达那裡」
「那裡就是最终层吗？」
。
「不，恐怕不是。虽说到了25层就是死路，也不是不能说是最终层。可是在25层最後的房间裡有个转移魔法阵。
　我已经临近濒死连走路都很困难的状态，所以还没有调查过走上去会有怎样的效果。
　调查那玩意也是我的目标之一」
。
「25层的转移魔法阵……。到底隐藏了怎麼样的陷阱啊」
「虽说只是我的猜想……恐怕我要寻找的女人，就是消失在那前方」
。
「原来如此……。那麼我得想出更好的战术，成为更优秀的魔导师。等到能帮助罗伊的时候，我也能为罗伊寻找那位消失在雷斯迪凯普的重要的人添一臂之力吧」
。
「嘛啊，姑且是会这样」
。
　罗伊害羞似的搔搔脸颊。
。
「别太过於努力啊。你只要在你自己道路上前进。变强这种事，不用那麼焦急也可以哦」
「我明白了。我会脚踏实地一步一步地在道路上前进」
。
「这样就好」
。
　我和罗伊互相点头，又再专心攻略地下城了
