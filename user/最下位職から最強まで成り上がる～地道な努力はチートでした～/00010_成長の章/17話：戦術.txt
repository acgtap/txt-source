
　羅伊在空中創造的光球正維持著效果，照亮地下城那黯淡的通道。
　由於雷斯迪凱普並沒有除此之外的光源，所以也就只能看到光球所照到的狹窄範圍。
。
　視野能見度大約是有7亞魯多(約10米)。
　如果在這視野能見度之外被狙擊的話，羅伊姑且不論，我一下子就完蛋了。
。
　雖說視野裡沒有魔物存在，可也不容有半點大意。
　在道路上提高警惕走了一會兒，持有自動探知技能的羅伊大聲說道。
。
「敵人接近！5秒後接觸。給我敵方戰力分析和指示！」
「我明白了！」
。
　從現在開始的戰鬥流程和指示就會由我來決定。
。
　隊伍的領導，指揮官，部隊長。
　以一句話來說，我即將要擔任這些職務。
。
　可我仍然是新手膽小的魔導師。
　如果造成無法挽回的巨大失敗該怎麼辦，這一不安讓我心跳鼓動加快。
。
　可是，既然羅伊說過我或許擁有這樣的才能，那麼就只能拿出自信干到底。
。
　於是魔物真的在五秒後剛剛好現身。
　與魔物接觸後應當要做的事情，首先是把握敵人戰力的構成。
。
　敵人前衛　巨魔×3、妖精騎士×3
　敵人中衛　妖精弓箭手×2
　敵人後衛　魔術師×2
。
　迅速確認完敵人隊伍構成之後，我說出我分析的敵人戰力。
。
「敵方戰力偏向前衛！由於敵方前衛數量過多，所以集中我方戰力逐個擊破敵人」
「明白了。具體要怎麼做！」
。
　看到逐漸接近過來的魔物，我心情不由得焦急著我不快點發出指示的話。
　冷靜下來，在這地下城的第一層戰鬥已經不是第一次了。
。
　如果失敗了，還有羅伊會來補救我。
　我深呼吸一次，迅速思考著應當建立的戰術。
。
　思考吧。不管怎麼想，最令人驚異的就是中衛的妖精弓箭手。
　妖精弓箭手這種魔物的特徵是在戰場稍微後方的位置射出弓箭。
。
　妖精那施加上風之魔法加護的弓箭攻擊，擁有驚人的命中率。
　如果被射擊到要害，擁有一擊就使人不能戰鬥的危險性。
。
　雖說這是想要立刻就打倒的魔物，可那邊被前衛的巨魔與妖精騎士穩穩抵擋著，形成肉盾。
　應該是為了主力戰力不會被簡單地被擊潰吧。
。
　妖精弓箭手被巨魔它們安全地護衛著，發射出必殺的一擊。而不讓這邊受到此致命傷是比什麼都還要重要的事情。
。
　妖精弓箭手在性質上是以弓箭來攻擊，所以一旦這邊成功接近它。它在近戰的戰鬥中是非常弱小的。
。
　所以首先是把最具威脅性、最脆弱的要點破壞掉。
。
　我張開聲音喊道。
。
「以敵人中衛的妖精弓箭手作為第一目標進攻。
　由我來用雷射彈組成彈幕來吸引敵人前衛。
　趁這一空隙，羅伊就突破中央。
　到達敵人中央後，盡可能迅速地解決掉妖精弓箭手。
　就這樣沖到後面也把魔術師解決掉。
　然後我在前，羅伊你從後面夾擊在戰場中央被孤立的敵方前衛，就這樣決出勝負！」
。
「了解」
。
　羅伊微微點頭，拔出劍以猛烈的速度奔跑著。
　雖然我自己也覺得我建立的戰術並不是滿是漏洞，看來那句所謂的「我來培育魯庫」是相當認真的樣子。
。
　他是在信賴著我。
　只見羅伊毫不懷疑我的戰術，往敵方中央突擊而去。
。
　對此，我快速發射全部的雷射彈來掩護他。
。
　雷屬性子彈猶如下雨一般往魔物傾注而下，雖然還達不到致命傷，可巨魔和妖精騎士都微微地產生傷害硬直狀態。
　羅伊並沒有放過這一瞬間的空隙。
。
　他以仿彿如同疾風一般的速度橫跨戰場，瞬間沖到敵人中央處。
　一轉眼就擺脫敵方前衛的他，與妖精弓箭手對峙著。
。
　被接近到眼前的妖精弓箭手無法發揮弓這一種武器的最大長處，就已被羅伊秒殺。
　可魔物也不愚蠢，它們開始以包圍殲滅陣型襲擊而去。
。
　巨魔舉起棍棒，而妖精騎士是以鐵劍砍下。
　羅伊不得不暫時專心躲避敵人的攻擊，並不能打倒第二只妖精弓箭手。
。
　然而，這種時候才是身為中衛魔導師的我出場。
　在稍微遠離一點的地方發射猛烈的雷屬性子彈，攻擊巨魔它們的後背。
。
　這是為了牽制它們的行動，使行動受到限制。
。
　巨魔這種擁有堅硬皮膚與剛毛而具有優秀防御力的魔物，以雷射彈根本就無法造成多大的傷害。
　但是，即便只是被蚊子叮咬的程度，接二連三地受到雷射彈攻擊也會感到厭煩。
。
　巨魔和妖精騎士看到我，發出煩躁般的咆哮。
。
「嘎啊啊啊啊啊！！」
。
　是決定先要攻擊弱小的我嗎。
　巨魔它們無視羅伊給予的攻擊，憤怒到氣喘吁吁地朝我這邊沖來。
。
　可我也不是會老老實實地被幹掉的白痴。
　張開雷屬性子彈的彈幕，邊利用在道路上的大岩石和拐彎處等等的障礙物，邊與魔物保持一定的距離適當地後退。
。
　巨魔高高舉起地棍棒也被火焰球給擊落。
　而魔法傷害對於妖精騎士比較有效果，所以用雷射彈攻擊來阻止其步伐。
。
　逐漸後退邊展開彈幕，盡量爭取時間。
　如此一來，羅伊就能更加從容不迫。
。
　他已經把第二只妖精弓箭手砍死了。
。
　拖延戰術。
 在我方主力救援之前，盡量使敵方焦急並引誘它們。這是我臨場思考而出的戰術。
。
　這對於戰鬥力缺乏的我來說，不管是拖延敵人行動，還是妨礙敵人都是最能起到作用。
。
　在和羅伊組成的兩人隊伍裡面，我的職責並不是主力攻擊手。
　只要交給最高火力的羅伊來做，不管有多少敵人都會被打倒，所以我只要輔助他或者妨礙魔物就可以了。
。
　只見羅伊也好像幹掉了後衛的魔術師，回過身往這邊過來了。
。
　這樣一來
　我→巨魔・妖精騎士←羅伊
　就會形成這樣的一列排序，我在前羅伊在後攻擊巨魔它們這樣的一個夾擊作戰就成立了。
。
　連突破雷彈幕都甚至陷入苦戰的魔物們，在被天才劍士・羅伊從後面攻擊的情況下，驚慌失措著。
　前有煩人的雷彈幕傾注而去，後有羅伊所釋放的如同閃光的一擊襲來。
。
　漂亮地使得魔物的隊列崩潰了。
　不久之後，魔物全部被打倒消滅掉，只留下魔石而已。
。
　呼……。
　初次指揮的戰鬥總算是勝利了。
。
　如今心臟還仍在啪嗒啪嗒地跳動。
。
　沒有什麼地方做錯了嗎，有什麼地方出現致命性的失誤了嗎，不安到毫無辦法。
　看樣子，我所想出來的幼稚拙劣的作戰姑且是以成功收場了。
。
「辛苦了」
「您也辛苦了」
。
　我跟平時一樣在戰鬥記錄上書寫著，而羅伊在撿魔。
。
「魯庫，這些給你」
。
　羅伊邊說著邊把魔石交給我，魔石比平時給我的份量還要多了。
。
「今天很多呢」
「嘛，這也就說期待你今後的表現吧」
「結果，只不過是依靠羅伊的輸出作戰……」
。
　羅伊一走，我甚至連能不能與魔物正面戰鬥都感到蹊蹺。
。
「因為我們現在還只是在第一層戰鬥而已，所以魔物的戰力也不是那麼驚人。即便陷入困境，以我一個人也能簡單地挽救回來。
　但是，一旦在更深處的階層戰鬥的話，魔物也會越來越強。也更會採用狡猾惡心的作戰狡猾。
　以我的力量是絕對抵抗不過的。這是我的親身經歷。
　所以你的力量是必要的」
。
　聽到羅伊這番話，我默默地點下頭。
。
「雖然你現在或許還是剛起步的冒險者。可從今以後，你所思考的戰術會越來越熟練，如果能擺脫危機的話，對我來說就是幫大忙了」
「這個地下城……雷斯迪凱普究竟有多少階層？」
。
「我所確認到的是25層為止。甚至連我也只是去過一次25層。而且還是不知道是幸運所眷顧還是不幸，觸發陷阱一口氣掉下五層直接讓我臨近瀕死才終於到達那裡」
「那裡就是最終層嗎？」
。
「不，恐怕不是。雖說到了25層就是死路，也不是不能說是最終層。可是在25層最後的房間裡有個轉移魔法陣。
　我已經臨近瀕死連走路都很困難的狀態，所以還沒有調查過走上去會有怎樣的效果。
　調查那玩意也是我的目標之一」
。
「25層的轉移魔法陣……。到底隱藏了怎麼樣的陷阱啊」
「雖說只是我的猜想……恐怕我要尋找的女人，就是消失在那前方」
。
「原來如此……。那麼我得想出更好的戰術，成為更優秀的魔導師。等到能幫助羅伊的時候，我也能為羅伊尋找那位消失在雷斯迪凱普的重要的人添一臂之力吧」
。
「嘛啊，姑且是會這樣」
。
　羅伊害羞似的搔搔臉頰。
。
「別太過於努力啊。你只要在你自己道路上前進。變強這種事，不用那麼焦急也可以哦」
「我明白了。我會腳踏實地一步一步地在道路上前進」
。
「這樣就好」
。
　我和羅伊互相點頭，又再專心攻略地下城了
