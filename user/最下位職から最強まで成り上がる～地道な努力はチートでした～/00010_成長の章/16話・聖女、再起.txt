
　从我(莉莉)眼里溢出的泪水，都不知道已经浸湿了多少个枕头。
　从鲁库被处刑那一天开始，我一直把自己关在一个人会显得寂寞般的宽敞宅邸里，直至现在。
。
　我已经连续一星期以上私自放弃骑士团的任务了。
　团长和副团长也应该相当不高兴吧，事实上，他们也为了让我接受任务而三番两次来我家见我。
。
　可我就像是个小孩子一样，只是在床上躲进被单里，缩成一团一个劲的在哭泣而已。
　对于甚至连外面都不出来的我，他们不久后都惊呆着、愤怒地回去了。
。
只留下『我们骑士团全员会等待你的回归』这样的一句社交辞令。
。
　陷入到这种境地，我才深切体会到自己还是未成熟的天真女人。
。
　被最重要的人所背叛、逝世的痛苦，其实就算不是我，在这世界上也有许多地方都有很多人经历过吧。
。
　对，这不是什么特别的事情。
　不管是谁都会发生的极其理所当然的事情。
。
　大家即便有过这样的经历，即使抱着一份撕心裂肺的悲伤，也还是坚强地活下去。
。
　心中都怀着一份痛苦面对着日常的工作，正确坦率地度过日常生活，一直等待着心中的伤痛随着时间经过而会在某一天痊愈之时。
。
　可我并不是觉得我有那么坚强。
　仅仅只是回想起鲁库的事情，心中就仿佛撕裂开来。
。
　鲁库……。
　明明你说过要一直在一起的，为什么丢下我自己走掉了呢。
。
　在那一天，在那法庭上。
　鲁库说出口的那一番话，我至今都还不敢相信。
。
　应该是有什么搞错了吧。
　那并不是鲁库，而是恶魔还是什么拟态成他说出来的一番话不是吗，等等这些。
。
　大脑拒绝接受这一事实。
。
　鲁库不是会说这种话的人。
　他无论何时都会很温柔。
。
　在稍微远离一点的地方注视着我。
　好像很害羞似的这么说。
。
　莉莉很善良呢。是个好孩子呢。
。
　仅仅被这么一说，心里就温暖起来。
　明明高兴到几乎要跳起来。
　明明我最喜欢温柔的你。
。
　明明是这样，为什么？
　为什么啊，鲁库。
　为什么要做出背叛我的心意的事情。
。
　我有什么不好的地方吗。
　对我有什么不满吗。
。
　明明只要你说给我听，我就会改掉。
　如果是为了你，我明明很乐意改变自己。
。
「为什么……为什么啊，鲁库……」
。
　他临终的一番光景深深地铭刻在脑海中，无数次、无数次地重复回放。
　而那每一次都会伤到我那柔弱的心灵，把我的心给碎尸万段。
。
　已经什么都不想做。
　骑士团啊贵族什么的怎么样都好。
　甚至连饭都不想吃。
。
　都想就这样死掉算了……。

。
　从那以后又过了一段时间吧。
　我的一天就是在床上使劲哭泣，哭完就睡。
　不管是窗户还是窗帘都是紧闭着，所以甚至连现在是早上还是晚上都不知道。
。
　对于一直哭泣感到疲惫，我久违地在床上起身，来到宅邸的客厅。
　简朴的沙发和放置一张餐桌的客厅。
。
　啊啊……。
　我与他在这里，到底是有多么的平凡、幸福地聊着天呢。
。
　仅仅只是看到客厅，就仿佛感觉到鲁库的痕迹一般，不禁流出眼泪。
　用手拿着他以前使用过的毛毯，用鼻子深深一吸。
。
　他所使用过而残留的味道让我心跳加速。
　回想到这个行为，觉得自己好愚蠢。
。
「我啊，是个笨蛋女人……。明明被那么明确的拒绝和背叛，还喜欢鲁库什么的……」
。
　深切感觉到自己没救了。
　无所事事地在客厅的沙发上，脑袋空空地发着一会呆之后，宅邸的门铃响起来了。
。
　事到如今，还有谁有事情找我呢。
　应该又是骑士团的团长为了让我复归而来劝说的吧。
。
　虽然是打算装作不在家无视掉的，可过于固执也感觉太幼稚了。所以我以哭肿的模样直接走向玄关。
。
「好的。请问是哪一位」
。
　在我一星期久违地出门之后，在那里站着的是没看见过的大叔。
　啊，不对，好像在哪里看见过。
。
　对了。是鲁库在最后被处刑的那一天，担任他的死刑执行者的士兵。
　大叔难为情似的嘎吱嘎吱地绕绕头，开口说道。
。
「虽然我觉得这样做是违反那家伙的意思、违背鲁库」
「……？」
。
　唐突的话题让我歪着头。
　是什么啊。
。
　新花样的搭讪吗。
　如果是这样请务必回去。
。
　这边可没有心思去管这种事情。
。
「可是啊，就这样下去鲁库那家伙也安息不了吧，而且听说你也一直在宅邸里闭门不出，果然我还是觉得好好说明比较好」
「鲁库？鲁库怎么了！？」
。
　我感兴趣地追问大叔所说的。
。
「那家伙，是被陷害的啊」
「诶――？」
。
「那次裁决，打从一开始就决定鲁库是有罪的。即便小姐你多么拼命地去辩护，上层都已经谈好了。鲁库的罪名已经是不可动摇」
「诶、诶……？为、为什么。那么根本就没有召开裁决的意义不是吗」

 「小姐你能想到这种事情，大概是因为你至今都是度过纯粹的幸福生活吧。可是，维鲁利亚王国就不是这样。这个王国正腐败着。为了自己的利益而不正当地贬低别人，为了让自己出人头地而把底下的人当做用完就丢的道具，这样的人可是有一大把」
。
「虽然我……完全不明白你说的意思……。可是，那也就是说，鲁库从一开始就已经决定是死刑了吗！？」
「就是这么回事」
。
　大叔恭恭敬敬地点着头。
。
「所以，那家伙才会为了不给你添半点麻烦而在大众之下的法庭上说些背叛你的话。为了守护小姐你那灿烂夺目的未来」
。
　我呆然地听着大叔说的话。
　为了、守护我的未来……？
。
「这、这是怎么一回事！？为什么鲁库为了守护我的未来而不得不说出那番话来呢！？」
。
「因为小姐你是被才能眷顾的人，所以应该不知道加入骑士团的重要性，也不知道维鲁利亚全国民是多么的憧憬骑士吧。
　以我们庶民的角度来看，骑士团的入团资格是极其渴望得到手的。的确工作或许是很辛苦，可一接受骑士授勋就能成为下级贵族。
　如此一来，家族的生活就安宁无事了。至少绝对不会失去工作而迷失于街头」
。
「骑士到底有多大的恩惠什么的根本无所谓！回答我的问题！为什么鲁库要说出那番话来！？」
。
「所以说是为了守护小姐你那恩惠生活、还有那灿烂辉煌的未来，鲁库才特意跟小姐你绝交。因为他觉得如果把已经确认是犯罪者的鲁库本身留在身边，就会让小姐你那成功的未来受到玷污吧」
。
「诶……诶……！？」
。
　至今一直抱着的忧郁般的感情，还有寂寞的思念都一下子灰飞烟灭。
。
　鲁库，为了我才特意说些惹人厌的话……？
　为了守护我作为骑士和下级贵族的未来，而自己一个人当坏人……？
。
「鲁库不想扯小姐你的后腿才自己当坏人，只让自己受到惩罚而死去。这都是为了不给小姐你添麻烦。
　请小姐你务必明白他这份思念。在这样看着小姐你孤独地腐朽而去的身姿，对于知道事情缘由的人来说，实在是过于悲哀。那家伙也会安息不了吧……」
。
　滂沱的泪水从我双眼里滴落而下。
。
　鲁库……。
　鲁库……为了我……自己一个人背负所有的罪死去了。
。
　为了不给我添麻烦。
　为了守护我的将来。
。
「啊……啊啊啊……」
。
　制止不住身体的颤抖。
　全身仿佛被想为他献身的思念而鸣动着一般。
。
　鲁库并没有讨厌我。
　也没有背叛我。
。
　岂止如此，还把献身般的爱情倾注于我。
　爱情这个词的真正意义，我感觉在这一天才明白了。
。
　至今为止，我到底知道鲁库的什么。
　即使一直在一起度过那么长的时光。
。
　我也没能理解到他一丁点的想法。
　只是看到事物的表面而已。
。
　啊啊，笨蛋的女人。
　我啊，真的是个笨蛋的女人。

「所以，我希望小姐你在今后的人生坚强地活下去。我认为，那是小姐你的义务与责任。我想要说的就只有这些，抱歉了啊」
。
　大叔对着呆然的我说完，就离开而去。
　我当场倒了下来。
。
　明明以为差不多已经哭尽的泪水，又再一次流下来了。
。
　然而那绝不是什么不好的感情。
　如今流下的泪水，是渗透着美丽的纯粹透明的尊敬之情。
。
「呜啊啊啊啊啊！鲁库！为什么你要一个人把这些事情都埋藏在心里！？为什么，为什么你没有对我说过半句话啊！？明明只要你告诉我，或许还能做点什么！」
。
　为什么。
　为什么，你会那么温柔呢。
　鲁库。
。
　他所遗留下来的残酷般的温柔，使我发狂。
　对于至今太过于愚蠢的自己，心中仿佛燃烧起来。
。
「啊啊啊啊啊！」
。
　从我眼泪流出来的泪水，把玄关的石板路浸湿了。
。
「鲁库……鲁库，对不起了……我是笨蛋呢……」
。
　不久之后，我的眼瞳里重新取回光辉。
　紧紧地紧握拳头，让自己的意志重新焕发。
。
　他那抱着遗憾而死的思念。
　他持续祈求的夙愿。
　给与我的慈爱的精神。
。
　绝对不可以在这里就此结束。
　绝不可能把这些当做没发生过而活下去。
。
　想要变强。
　想要坚强地活着。
。
　改变吧。
。
　舍弃掉既幼稚又任性还为所欲为的自己。
　成为像鲁库这样既善良而又美丽的人。
。
　在这里，我立下神圣的誓言。
。
　已经不会再让谁牺牲。
　为了不再有人拥有像鲁库这般悲哀的思念。
　我要改变这个腐败的王国。
。
　不管是没受到恩惠的人，还是有悲伤回忆的人，又或者是困境中的人。
　把这些人们。
　把映入眼中的一切，都拯救。
。
　我相信，这就是对于把未来托付于我而逝世的鲁库的赎罪。
。
「鲁库……。不管你在何时何地，都要看着我哦。
　我、会努力的。把你持续到最后的那份思念当做我永远的宝物，我会成为与你相称的女人给你看」
。
　我在这一天。
。
　这一瞬间。
。
　在这不久之后，我改变成为被维鲁利亚国民称呼为『救国的圣女』的女人。
