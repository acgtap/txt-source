
　我和罗伊在旅馆度过一夜，身体也充分休息过后又再次潜入地下城。
　对我而言，为了赚取生活费是必须要潜入地下城的，可我纯属想变得更强。
。
　如果我拥有更强大的力量，就可以减少像莉莉这样拥有悲伤回忆的人。
。
　如果我变得更强，不管有多少个贵族、有多大的权利，我或许还能守护好莉莉也说不定。
　或许，何时又能再遇见莉莉的时候，有可能在这一次就能成为她的力量。
。
　心中怀抱着这一份浅淡的期待，我一心一意地追求着强大。
。
「罗伊」
「怎么了」
。
　迅速穿过雷斯迪凯普的第一阶层，走在通往第二阶层的螺旋形楼梯时，我如此说道。
。
「一直以来在魔物袭击过来的时候，不管是我们还是魔物都是有视野战斗不是吗」
「是啊」
　所谓有视野战斗，也就是双方都能以肉眼确认到对方身影的距离上战斗。
　雷斯迪凯普的有效视野范围是半径7亚鲁多(约10米)。
。
　本来的话，有效视野范围应该是更广泛才对，可由于这里是被黑暗包围的雷斯迪凯普。
　所以有效视野范围比起地面上的战斗还要狭隘。
。
「这一点，我感觉稍微可以有点改变」
。
　我把想到的新作战说给罗伊听。
。
「可以改变？怎么说」
。
「是的。正因为第二阶层的敌人比起我还要略高一筹，所以我想要在最初的战斗中掌握主导权。
　在最初的流程上，如果不先把敌人引诱到这边，就会像最近的战斗那样，全部的对应都是由我们后手」
。
　听到我所说，罗伊点着头。

「嗯。的确在剑术上也有句相似的，取得先手等同于胜了七成啊。
　然而即使想要掌控主导权也不是那么简单哦」
。
「正因为如此，我才想要从有效视野外进行先制攻击」
「有效视野外的先制攻击？」
。
　罗伊浮现出疑问号重复一遍我的问题，我说「对」，然后点着头进行说道。
。
「罗伊使用魔物探知技能，总是在战斗开始前5秒左右就查探到不是吗」
「是啊」
。
「在这5秒的时间里，我们至今大多数都是在做好准备等待敌人哦」
「嘛啊实际上，近战职业的我在敌人没有进入有效视野前，什么都做不了啊」
。
「罗伊这样当然是没问题，可我是魔导师。
　从中距离到远距离都能进行攻击的魔法职业」
。
　听到我说的，罗伊说着「原来如此」并拍了下手。
。
「换句话说……鲁库打算在敌人进入有效范围之前，用魔法给与到某种程度上的打击。这么说可以吧？」
「对，我想通过魔法进行先制攻击。即使不能打倒魔物，只要让前卫和后卫的队列崩溃，大概会让战斗形势有利起来」
。
　罗伊听到我的提案笑着说。
。
「有趣。试试看吧」
「好的！」
。
　就这样，我对罗伊说出我敲定出来的新作战，我们都决定试试看先制攻击。
　如此这般走到第二阶层，罗伊大声地发出紧张的声音。
。
「敌袭！5秒后接触。鲁库，试试看刚才说过的方法吧！」
「了解！」
。
　罗伊维持的光球正照亮着前方7亚鲁多(约10米)的地方。
　在那更前方，明明什么都看不见，可却有魔物袭来的压迫感。
。
　我来到罗伊的身边，用魔法进行先制攻击。
　使用的魔法是雷射弹。
。
　雷属性的子弹撕裂漆黑飞翔而去。
。
　在黑暗的前方，有雷射弹打中的感触。
　魔物还发出低沉的悲鸣声。
。
　而我并没有因此停止，一个劲地持续释放雷射弹。
　不久之后，只见魔物在黑暗的彼端狂怒的样子。
。
「进入有视野战斗！停止先制攻击，回到后卫！」
「哦」
。
　我迅速后退到罗伊后面。
　现身的魔物队列混乱，前卫与后卫的队列乱七八糟。
。
　分析魔物的战力构成。
。
　敌方前卫　中级巨魔×3　中级猪头人×2
　敌方中卫 巫妖×2
　敌方后卫　高级魔法师×3
。
　魔物原本应该是这样的队列，可由于我的先制攻击而让队列混乱到高级魔法师都出现在前列。
(颯:你不懂，法爷都有一颗近战的心)
。
　而且魔物突然被魔法从看不见的距离上攻击，而正动摇着的样子。
。
「很好，先制攻击有效果了！鲁库，从哪一个开始击溃」
「最具威胁的是高级魔法师！我和你一起一瞬间打倒吧」
「了解」
。
　战斗的基本是集中我方战力于一点，分割敌军进行逐个击破。
　在因先制攻击而列队混乱的时候，再由这边结合目标逐个击破。
。
　罗伊以令人觉得是神速般的猛冲，缩短与高级魔法师的距离。
　而我用新学会的魔法、雷鸣长矛对准高级魔法师。
。
　还用沙之锁拘束住高级魔法师×3的双手双脚。
　高级魔法师被封住行动，法杖也掉落而下，无法咏唱魔法。
。
「nice掩护」
。
　罗伊简短地夸奖我，迅速地砍倒高级魔法师×3。
　化为魔物残骸的同时，变成了魔石。
。
　上一次那般苦战的高级魔法师×3，一瞬间就成了亡者。
。
「接下来，边后退射击，边分割敌方战力！」
「了解了」
。
　我随着罗伊边后退，边释放雷射弹打击因高级魔法师被打倒而狂怒起来的中级巨魔们。
。
　于是敌人队列又再次产生空隙。

 中级巨魔与中级猪头人的防御力和魔法耐性非常高，雷射弹是几乎给与不了多少伤害。
　然而，中卫的魔法职业・巫妖并不是如此。
。
　即使是雷射弹的威力也会感到胆怯，而又被引发麻痹效果而当场产生伤害硬直状态。
。
　以结果来说，就是直直前进的中级巨魔们与巫妖拉开距离。
　在充分地分割敌方战力的时候，我停止后退射击，转为攻势。
。
「队列已经混乱，开始反击！」
「了解」
。
　罗伊用剑一闪就砍倒在战场上被孤立的中级巨魔们。
。
　恐怕罗伊如果只是独自一人打倒魔物的话，根本就不需要这般麻烦的作战吧。
　但是，为了至少可以提高生存率，也为了今后的阶层也能实用。现在就该彻底采用令敌人厌恶的战术。
。
　因为我认为这会是跟陷入困境有很大关联的战斗判断。
。
　在罗伊打倒中级巨魔的时候，我也用新魔法雷鸣长矛打倒一匹中级猪头人。
　攻击速度是A级别的雷之矛，贯穿猪形魔物的巨大身体，一击毙命。
。
「雷鸣长矛……这个能使用……！」
。
　在拥有坚固防御力的第二阶层前卫面前，十分实用。
　这段时间就把雷鸣长矛作为我的主要魔法，一个劲地提高等级来增加总魔力量吧。
。
　如此一来，即便还不到能无限地使用雷鸣长矛，可也应该能够连续释放吧。
。
　在这场战斗中，我的确有感觉到手感。
　残留的魔物只有巫妖×2二姨。
。
　巫妖原本是人类魔女魔物化的魔物，属于在中卫释放各种魔法进行攻击。
　然而，由于败给雷射弹的先制攻击与后退射击给予的伤害，巫妖的行动变得迟钝起来。
。
　我们最强的前卫可没弱到会放过这一空隙。
。
「哈！」
。
　罗伊以猛烈速度缩短与巫妖之间的距离，集中气势，猛地用剑一砍。
。
「グゥゥゥゥ！」
。
　巫妖发出低沉悲鸣声，而罗伊就这样打倒两只巫妖。
　战斗结束。
。
　魔物全部化为魔石，而我还是老样子的战斗记录时间。
。
　今天的收获是……先制攻击变得习惯了。
　我希望这一点在接下来的战斗中也能继续下去。
　这个战术究竟能做到什么程度，就让我边试行错误边改善吧。
。
　当我记录完毕，捡完魔石的罗伊往这边走过来。
。
「这一次作战非常好。接下来也拜托你这么做吧」
「是，了解」
。
　我收下几个中级魔法，边回了个笑容。
