
　在冒险者公会用魔石换到金钱後，我步行于西里卢卡的街道上。
　由于不是大城镇，所以一般主要的商业设施都会在大路上，而巷子裡居民的住宅就会是挤在一起的感觉。
。
　我应该要采取的行动是，购入装备、取得魔法、吃饭、这三个选择项。总之先吃饭吧。
　自从掉落在雷斯迪凯普以来，我就只吃到鱼啊、肉食系魔物的肉啊、植物系魔物的叶子而已。
。
　虽然这些都让擅长求生能力的罗伊给安全的调理了，所以对身体并没有毒。可果然人类是不吃到美味的饭菜就不行的生物。
。
　我一边参考着看板上描绘的菜单和在摊子上人们吃着的饭菜，一边适当地琢磨着要进入的饭馆。
　在维鲁利亚王国裡，基本上是面包加汤还有主食的肉和蔬菜，这种真要说是平淡的饮食生活。
。
　可看样子，在艾杰巴拉是以大量添加香辛料和调味料的调理方法为主流。从西里卢卡的大路上的摊子飘来刺激食欲的气味。
。
一瞥，在稍微拉开点距离偷窥到的摊子上，是用猪肉片、胡萝卜、洋葱还有黑胡椒来调味做出来的炒饭。

　刺激性的香料气味瘙痒着我的鼻孔。
。

「很好吃的样子……」
「哦？男孩，要吃吗？一盘5枚铜币哦！」

。
　店长耳朵灵光地听到我不禁说漏的話语。
。

「那么，就让我吃一顿」
「来吧来吧！就给你特别多盛一些吧！」
「因为我肚子饿扁了，所以这样做就太高兴了」

。
　店长大叔哈哈大笑着。
　要是在这种真正的买卖交谈中，我成为商人肯定会盲目听信商谈对象说的話而造成巨大损失吧
。
　曾经有过被莉莉这么说过的记忆。

『鲁库太过于老实地听取别人说的話了！以後会上当受骗啊！』

。
　嗯~，就算这么说，这一点也是我的性格啊……。
。
　如果是以魔物为对象的冒险者就不怎么有像这样人与人之间的交涉技巧，所以我还是作为一个冒险者度过一生吧。
　人也是有所谓的适才适用。
。
　我坐在摊子的椅子上等待料理做好。
　大叔使用铁锅的那熟练手法真不愧是老手，在那样激烈的动作中真亏他没有洒落出来。
。
　不久之後，一盘装满许多用黒胡椒、猪肉、蔬菜和米炒出来的炒饭就端出来了。
。

「好，久等了！」

。
　我用银色的勺子舀一勺炒饭放进嘴裡。
　咀嚼一口，黒胡椒那刺激感在整个嘴裡扩散而开。
。

「好好吃……！」
「哈哈哈。男孩你的吃相真意外呢！」

。
　店长边为下一位客人炒着饭的同时，斜眼看着我豪爽地笑了。
　我就这样继续享受黑胡椒的炒饭，在吃完之後支付给店长定价的金钱。
。
　虽然店长稍微有些惊讶，可立刻又满脸笑容地对我说「下一次再过来吧！」。

　我也道谢完离开了摊子。

。

　肚子也舒舒服服地鼓起着，接下来是装备和魔法强化。
　在雷斯迪凯普的战斗中学到的，有个『要学会迅速释放的魔法』的课题内容。
。
　以现状来看，火焰球只要经过细微的调整就可以顺利地灵活使用。但以现在的状况来看，应该会出现不能通用的情况吧。
。
　到不能顺利使用的时候再来改善，那就已经太迟了。
　即便能顺利地使用，只要大概感觉是稍微有点欠缺啊、可以再稍微改善啊，就可以立刻改善。
。
　就算改善後会导致一时的坏结果，那也是与长期的成长是紧密相连。
。
　这也是从罗伊那裡现学现卖的話。
。
　然後我，首先为了学会新魔法而前往圣教会。
　在这大陆上，大多数国家普遍都有圣教会，几乎所有的城镇都有圣教会的支部。
。
　就跟我一开始接受祝福之仪的时候一样，当想要学习新魔法和新技能就要捐献金钱给圣教会，接受名为成长之仪的仪式。
。
　基本上提高1级，就会出现2～3个能学会的新魔法・新技能，然後从裡面选择想要学会的魔法和技能，差不多是这种感觉吧。
。
　当然，根据魔法和技能的熟练度，能学会的项目也会发生变化。
。
　所以真正想要变强的人，就会勤勉地支付给教会金钱，让他们给自己举行成长之仪。
　不管怎样，为了变强，钱就要有。
　圣教会应该大赚一笔了吧。
。
　当然，不拜托圣教会，而是自己本能的学会新技能和新魔法的人。虽说不多，可也不是完全没有的样子。
　罗伊大概就是后者的类型吧。
。
　我完全是凡人类型的人。由于是前者，所以我决定去教会学习新魔法。
　走进跟维鲁利亚王国的圣教会完全相同构造的教会裡。跟神父交流完之後，他露出笑容。
。

「没问题哦。最初的祝福之仪已经接受完毕了吧」
「是的。天职是低位魔道士，等级是……7」

。
　用鉴定技能确认一下，我的等级从5上升到7了。
。

「魔法职业吗。以这个等级，一次成长之仪要一枚银币和五枚铜币」
「喔……勉强足够」

。
　魔石的收入是两枚银币和两枚铜币。
　再减去刚才黑胡椒炒饭的五枚铜币。
。
　剩下的就是一枚银币和七枚铜币。
　把这些花了就不能买到好装备，恐怕连旅馆都住不上。嘛，这一次就没办法了。
。
　我从钱包裡取出一枚银币和五枚铜币交出去，神父拘谨地笑着说「这或许是神的意向呢」。

成长之仪迅速地准备着。
　并不是像接受祝福之仪的时候那种周密的仪式，而是能够以私服的状态接受仪式。
。
　跟着神父的祝词复述一遍之後，神父拿着的水晶上浮现出文字来。
。

「这个水晶上已经浮现出可以取得的技能与魔法。请过目，决定好想要学会的魔法」

　在那裡面浮现出文字。

「噢噢……有许多可以学会的选择项呢」
「是的。你想要学会什么样的魔法呢」

　神父亲切地跟我商量。
　圣教会在经营祝福之仪与成长之仪的性质上，还经常受到许多冒险者的咨询商量。

　所以像我这样的新人冒险者也会教我许多事情。
。

「我想要学会攻击速度比较快的魔法。所以，各系統的子弹系魔法就可以了吧」
「是呢。如果是与您要求一致的魔法，那么雷射弹就可以了吧。可是仅从可以取得的魔法来看，你火系统的熟练度很高呢」

。
　的确。
　火系統在这一时点上已经能够学中级魔法了。
。

「专注于独个系统来提高熟练度，这样学习技能比较好吗？」
「基本上我是这样推荐的。可是呢，在与火属性抵抗力高的魔物战斗的时候，只依赖独个系统魔法就会陷入苦战吧。在这种时候如果有队伍来拟补这一弱点，那就另当别论」

　是吗，也就是说还有这样的缺点吗……。
　也不可能一直让罗伊来保护我，我还是想要某种程度上可以让我单独行动的能力。
。
　我烦恼过後的结果，还是决定取得与目的最吻合的攻击速度A的『雷射弹』。

「那么，雷射弹就拜托了」
「明白了」

。

　这样一来，我手头上的手段就增加一个了。
　我有种想要快点在实战中尝试一下的冲动。

。
　我顺利地学会雷射弹并对自己使用鉴定技能确认完毕之後，又前往装备店。

。

「嘛啊，总之就只是去看一下吧。会贩卖什么样的装备呢，事先看一下也不错」

。
　而且我在这城镇裡也没有其他要做的事情。
。
　走回刚才过来的大路上，进入一家专卖各个职业装备的店裡。
　只要不是维鲁利亚王都和艾杰巴拉皇都，基本上是不会流行魔导师用的装备专卖店。
。
　西里卢卡的装备店是由一个宛如感到人生非常枯燥无味的姐姐在经营着，即便我走进店裡
。

「欢迎光临……」

。
　也是非常懒散的态度。

。

　从右手开始是放置着剑士和近战职业用的剑・斧・枪，还有像是盾牌的防具。以绕了店裡一圈的感觉来看，职业都分类出来的样子。
。
　我想要的魔导师用的法杖和斗篷都在中央处，可种类并不丰富。

。
　覆盖全身的斗篷――黑色底色外加描绘出红色火焰花纹的斗篷，非常地帅气实在很想要，可看到商品说明之後

　好贵。

。

。

。
　现在就算没有装备效果也可以，总之我想要有魔导师的感觉。
　我这么想着，尽量以有点脏兮兮、破破烂烂，看起来很便宜的长袍为中心看看价格，可最低价位基本上都是7枚铜币到8枚。
。
「嗯~……这样的装备也没有装备效果，甚至就连明显是旧货的长袍，上面边缘有点破烂而且还皱皱巴巴的都还要6枚铜币么。手头只有两枚铜币，买不了啊~……。事到如今，还真心疼花在炒饭上的5枚铜币啊。可是那个也很好吃啊，连吃饭都不能花钱的話，心都凉了……」

。

「嗯？想要长袍吗？」

。

。

「啊，是的。想要跟魔物战斗的时候使用的长袍来着」

「是的……」

。
　我垂头丧气地说完之後，店长姐姐笑着说。
。

「是吗是吗。也就是说你是刚出道的贫困新人冒险者吗」

。

「那么就给你杀必死吧。如果是没有装备效果的长袍，就以2枚铜币买给你吧」

。

　对不起，我还以为是对人生感到枯燥无味的姐姐。

「作为替代，在出名赚大钱之後就要买下那边的黑炎龙长袍哦～～」

。

「非常感谢！此恩必报！」

「在这个国家裡，还是不要以定价的价格买东西，或者在外面吃饭比较好哦。因为基本上大家砍价都已经是常识了，所以不管在哪裡价格都会定得很高」

。

「嘿~。那么在一开始的时候会比较容易上当受骗呢。要小心哦。在这个国家，不会交涉可是会吃大亏的哦」

「嘛，会适当地习惯的啦，总有办法的。诀窍是，就算交涉对象一副讨厌的表情也只不过对方的演技而已，所以不用在意」
