
　雷斯迪凯普的第二阶层。
　在光球照耀的黑暗之中，我和罗伊探索着、一直在持续猎杀魔物。
。
「敌人接近！5秒後遭遇」
。
　随着罗伊这句战鬥前的言语，我也开始熟悉的先制攻击。
雷性子弹犹如被漆黑的彼端所吸收一般发射出去，响起魔物的低沉悲鸣。
。
　不久之後，在黑暗对面的魔物们慢慢现身了。
　敌我距离，约7亚鲁多(10米)。
。
　在战鬥开始之前，给与最後一击・雷鸣长矛伺候它们。
　巨雷命中魔物前卫的中级猪头人，它发出死亡的悲鸣声後消灭了。
　转换为魔石。
。
「好。接下来由我来支撑」
「拜托你了！」
。
　我立马後退到後卫，返回队列。
　如果不维持这个队列，我就会被敌人前卫一击毙命。
。
　敌方战力构成是
。
　前卫　中级猪头人×2　妖精武士×2
　中卫　巫妖×2
　後卫　高级魔法师×3
。
　在这些魔物之中，最具威胁性的就是高级魔法师。
。
　上一次战鬥中，由於雷鸣长矛过於优秀，导致战鬥方式变为拼命火力压制的死脑筋类型。这是该反省的一点。
　首先是切勿急躁，用妨害魔法使得决定性目标无力化。
。
「从哪一个开始，鲁库！」
「我要用狂沙拘束限制住高级魔法师的行动。先击溃高级魔法师吧！」
「了解」
。
　通过简短的对话，我就把意思传达给罗伊。
由我的先制攻击使得敌方队形混乱，让高级魔法师出现在前卫早已经是惯例。
。
　後卫职业的各个防御力和近距离战鬥能力都很低。
　出现在前卫的魔法职业等於是在说请打准我打吧。
。
　我咏唱魔法，对三只高级魔法师施加狂沙拘束，限制它们的双手双脚。
　妨碍魔法咏唱。
。
　罗伊看看我的妨碍魔法是否有正常运作之後，一瞬间拉近与高级魔法师之间的距离，用那银色的剑一闪而过。
　一记砍击把高级魔法师的身体砍断。
。
「グゥゥゥゥ！」
。
　高级魔法师无法抵抗，就这样被罗伊一刀两断。
　三只高级魔法师一瞬间就成为尸体。
。
　在罗伊对付高级魔法师的时候，中级猪头人与妖精武士意图包围攻击他，可我并没有让它们称心如意。
　我从後面快速发射雷性子弹，牵制、威吓它们。
。
　当看到中级猪头人它们目标转变为我这边，我就迅速後退。
　边後退边使用雷射弹拖延时间，拖延战术。
。
　後撤射击也成为我们战鬥中常用形式的战术。
。
　利用後撤射击吸引敌人前卫的注意力，接着在罗伊解决掉中卫的巫妖的时候，我们就转化为攻。
。
　战况是，我　→　中级猪头人・妖精武士　←　罗伊
。
　形成这样的构图，夹击、夹攻就完成了。
　既然战局已然发展至此，等同於胜利在眼前滚滚而来。
。
　我对准前方投射雷之枪，罗伊从它们背後挥下剑。
　受到前後夹攻，中级猪头人们无计可施。被打倒消灭了。
。
　魔物的尸体转变为魔石。
　战鬥结束。
。
「辛苦了」
「您辛苦了」
。
　罗伊在捡取魔石，而我在记录战鬥经过。
。
　果然我还是不要以输出魔法打倒敌人为優先事项比较好。
　彻底妨碍敌人、阻扰敌人行动、确保战鬥优势，逐个解决掉。
。
　这边还有个超优秀的主力攻击手的罗伊。
　完全不需要我作为输出活跃。
。
　如此一来，不管是学会用於先制攻击的範围魔法，还是学会新的妨碍魔法都应该比较好。
　首先是把等级提高到在第二阶层也能够毫无压力战胜的程度吗。
。
　真想在这个阶层升两级啊。
。
「罗伊」
「什麼事」
。
「我打算在第二阶层再提升两级」
「在那之後就回去城镇可以吧？」
。
「好的。只要不大意，即便没有範围魔法也能够在第二阶层战鬥」
「知道了」
。
　我们商量完毕，又一直在提升等级。


　◇　◆


　我们一直在雷斯迪凯普的第二阶层持续猎杀着魔物，顺利提升两级之後。我和罗伊回到西里卢卡城裡来了。
。
　跟平时一样决定好旅馆，在城镇门口分别，分别行动。
　西里卢卡城的大街上，朝气蓬勃的人潮来来往往，是座活力缤纷的城镇。
。
　当为了贩卖魔石而走在商业公会的路上的时候，我经常关顾的摊子老板的大叔对我打招呼。
。
「哟，鲁库。探索地下城辛苦了。好久不见了啊」
「大叔。2天……不见了吧」
。
「是罗德里格斯。这一次也平安无事来回一趟雷斯迪凯普了吧？」
「嘛，算是吧」
。
　我缅甸地笑着，罗德里格斯豪爽地笑着欢迎我。
。
「总之先吃饭吧。肚子饿扁了吧？」
「可以吗。今天是什麼菜？」
。
　我坐摊子的椅子上。
　是因为早过中午的缘故吗，除了我以外并无其他客人。
。
「是白兔奶汁烤菜」
「呜哇啊，好怀念！是莉莉喜欢的吧……」
。
「莉莉？鲁库的那个吗？」
「虽然不是这一回事就是了」
。
　突然想起她的事情，我苦笑着。
。
　如果她在王国还是精神饱满就好了。
　不过以莉莉那正直的性格，真怕她会与拥有不正当勾当的上层对立。
。
　我和罗德里格斯闲聊一会儿之後，白兔奶汁烤菜就做好了。

 奶汁烤菜是在罗罗纳村的时候，莉莉的得意料理。
。
　用白酱汁炖煮兔肉一阵子，接着是把胡萝卜和花椰菜切成一块一块。
　然後把这些都装进容器裡，在这些上面塗满大量奶酪。接着放进炉灶裡煎烤。
。
　白酱汁与奶酪结合在一起，绝妙的味道。
　她的奶汁烤菜实在美味……。
。
　我用叉子插进放在眼前的奶汁烤菜，粘稠粘稠的。
　迅速含一口。
。
　奶酪那浓厚味道和独制白酱汁的美味在口中扩散而开。
　虽然我挺喜欢莉莉做的奶汁烤菜，可这也有这的美味。
。
「好好吃……」
「对吧？」
。
　罗德里格斯也喜滋滋地频繁点着头。
。
「罗德里格斯的店又是炒饭又是肉丸子，还能做奶汁烤菜。不管什麼都做得出来呢」
「那当然，不然每天都是一样的菜色，客人也会吃腻的吧。每天的菜色都是根據那一天买入的食材来决定」
。
「嘿~。虽然对於吃饭的人来说，有各种各样的菜色的确开心，可制作人就不得了呢」
「嘛。就算开了家店，每天都还是在学习喔」
。
　果然，厨师的世界也很严酷的样子。
　不管是哪一个领域，如果没有日日精进的话，是无法成为一流人士吧。
。
　我把烤好的奶汁烤菜吃完，填饱肚子之後。笑着道谢交给大叔三枚铜币。
。
「喂。鲁库，两枚铜币就可以了吧」
「不。今天的非常美味。而且……还想起以前的事情来了」
。
　当看到我表情上微微渗透的哀愁，罗德里格斯默默点下头表情。
。
「要是能见面就好了啊。跟那位叫莉莉的小姐」
「是呢。只要活下去，总有一天绝对会的」
。
　我离开了摊子。

。

　然後步行在大街道上，在商业公会把魔石卖掉。
　由於这一次中级魔石有三颗，所以总共卖出5枚银币。
。
　总资产是6枚银币。
　这样一来，口袋也不会是空空如也。
。
　而且也能够购买新的装备。
　嘛，首先是去圣教会接受成长之仪。
。
　卖完魔石後前往圣教会，我决定拜托神父为我举行成长之仪。
　支付1枚银币，接受仪式。
。
　在教会那神圣气氛中，响起神父的祝词，仪式结束。
。
「这一次能学会的魔法是这些吧」
。
　我眺望在水晶映射的可以学会的魔法一览表。
。

　噢，能够取得新的防御魔法了。
　为了先制攻击，第一个就确定是学会範围魔法。毕竟也没有其他应该学会的妨碍魔法，这一次或许学会防御魔法比较好。
。
範围魔法有熔岩巨涌和荒雷巨蟒吗。
要选哪一个比较好呢
。
　就情况而言，雷系统是我主用的魔法，所以範围魔法还是想避免学会荒雷巨蟒而提升太多雷系统熟练度。
　变强也不是说有时间限制，现在就慢慢提高各种系统的熟练度吧。
。
　如此这般，应当学会的範围魔法就是火系统的熔岩巨涌。
。
「总之第一个已经决定是熔岩巨涌，拜托您了」
「我知道了」
。
　神之祝福的光芒漫天飞舞，我接受天啟而学会了新魔法。
。
「然後是关於第二个要学的魔法……神父，水系統能够熟练使用的时候，有什麼好处吗？」
「水系統是治疗和防御优秀的系统，所以想要在这方面发展的话，就应该要学会吧」
。
　原来如此。
　既然我要以全能力战斗的魔导师为目标，情况紧急的时候有治疗魔法也没有损失吧。
。
　而且瀑布墙是防御魔法，像最近跟骷髅奇美拉战鬥的时候，那种炎之吐息也能够抵抗了。
　真要说起来，我老是输出和妨碍敌人而已，防御方面就很少接触。
。
　现在就加强防御能力吧。
。
「那麼，第二个就拜托你给我瀑布墙」
「嗯」
。
　就这样，我学会了範围魔法熔岩巨涌和防御魔法瀑布墙。
