
　雷斯迪凱普的第二階層。
　在光球照耀的黑暗之中，我和羅伊探索著、一直在持續獵殺魔物。
。
「敵人接近！5秒後遭遇」
。
　隨著羅伊這句戰鬥前的言語，我也開始熟悉的先制攻擊。
雷性子彈猶如被漆黑的彼端所吸收一般發射出去，響起魔物的低沉悲鳴。
。
　不久之後，在黑暗對面的魔物們慢慢現身了。
　敵我距離，約7亞魯多(10米)。
。
　在戰鬥開始之前，給與最後一擊・雷鳴長矛伺候它們。
　巨雷命中魔物前衛的中級豬頭人，它發出死亡的悲鳴聲後消滅了。
　轉換為魔石。
。
「好。接下來由我來支撐」
「拜托你了！」
。
　我立馬後退到後衛，返回隊列。
　如果不維持這個隊列，我就會被敵人前衛一擊斃命。
。
　敵方戰力構成是
。
　前衛　中級豬頭人×2　妖精武士×2
　中衛　巫妖×2
　後衛　高級魔法師×3
。
　在這些魔物之中，最具威脅性的就是高級魔法師。
。
　上一次戰鬥中，由於雷鳴長矛過於優秀，導致戰鬥方式變為拼命火力壓制的死腦筋類型。這是該反省的一點。
　首先是切勿急躁，用妨害魔法使得決定性目標無力化。
。
「從哪一個開始，魯庫！」
「我要用狂沙拘束限制住高級魔法師的行動。先擊潰高級魔法師吧！」
「了解」
。
　通過簡短的對話，我就把意思傳達給羅伊。
由我的先制攻擊使得敵方隊形混亂，讓高級魔法師出現在前衛早已經是慣例。
。
　後衛職業的各個防御力和近距離戰鬥能力都很低。
　出現在前衛的魔法職業等於是在說請打准我打吧。
。
　我詠唱魔法，對三只高級魔法師施加狂沙拘束，限制它們的雙手雙腳。
　妨礙魔法詠唱。
。
　羅伊看看我的妨礙魔法是否有正常運作之後，一瞬間拉近與高級魔法師之間的距離，用那銀色的劍一閃而過。
　一記砍擊把高級魔法師的身體砍斷。
。
「グゥゥゥゥ！」
。
　高級魔法師無法抵抗，就這樣被羅伊一刀兩斷。
　三只高級魔法師一瞬間就成為屍體。
。
　在羅伊對付高級魔法師的時候，中級豬頭人與妖精武士意圖包圍攻擊他，可我並沒有讓它們稱心如意。
　我從後面快速發射雷性子彈，牽制、威嚇它們。
。
　當看到中級豬頭人它們目標轉變為我這邊，我就迅速後退。
　邊後退邊使用雷射彈拖延時間，拖延戰術。
。
　後撤射擊也成為我們戰鬥中常用形式的戰術。
。
　利用後撤射擊吸引敵人前衛的注意力，接著在羅伊解決掉中衛的巫妖的時候，我們就轉化為攻。
。
　戰況是，我　→　中級豬頭人・妖精武士　←　羅伊
。
　形成這樣的構圖，夾擊、夾攻就完成了。
　既然戰局已然發展至此，等同於勝利在眼前滾滾而來。
。
　我對准前方投射雷之槍，羅伊從它們背後揮下劍。
　受到前後夾攻，中級豬頭人們無計可施。被打倒消滅了。
。
　魔物的屍體轉變為魔石。
　戰鬥結束。
。
「辛苦了」
「您辛苦了」
。
　羅伊在撿取魔石，而我在記錄戰鬥經過。
。
　果然我還是不要以輸出魔法打倒敵人為優先事項比較好。
　徹底妨礙敵人、阻擾敵人行動、確保戰鬥優勢，逐個解決掉。
。
　這邊還有個超優秀的主力攻擊手的羅伊。
　完全不需要我作為輸出活躍。
。
　如此一來，不管是學會用於先制攻擊的範圍魔法，還是學會新的妨礙魔法都應該比較好。
　首先是把等級提高到在第二階層也能夠毫無壓力戰勝的程度嗎。
。
　真想在這個階層升兩級啊。
。
「羅伊」
「什麼事」
。
「我打算在第二階層再提升兩級」
「在那之後就回去城鎮可以吧？」
。
「好的。只要不大意，即便沒有範圍魔法也能夠在第二階層戰鬥」
「知道了」
。
　我們商量完畢，又一直在提升等級。


　◇　◆


　我們一直在雷斯迪凱普的第二階層持續獵殺著魔物，順利提升兩級之後。我和羅伊回到西里盧卡城裡來了。
。
　跟平時一樣決定好旅館，在城鎮門口分別，分別行動。
　西里盧卡城的大街上，朝氣蓬勃的人潮來來往往，是座活力繽紛的城鎮。
。
　當為了販賣魔石而走在商業公會的路上的時候，我經常光顧的攤子老板的大叔對我打招呼。
。
「喲，魯庫。探索地下城辛苦了。好久不見了啊」
「大叔。2天……不見了吧」
。
「是羅德里格斯。這一次也平安無事來回一趟雷斯迪凱普了吧？」
「嘛，算是吧」
。
　我緬甸地笑著，羅德里格斯豪爽地笑著歡迎我。
。
「總之先吃飯吧。肚子餓扁了吧？」
「可以嗎。今天是什麼菜？」
。
　我坐攤子的椅子上。
　是因為早過中午的緣故嗎，除了我以外並無其他客人。
。
「是白兔奶汁烤菜」
「嗚哇啊，好懷念！是莉莉喜歡的吧……」
。
「莉莉？魯庫的那個嗎？」
「雖然不是這一回事就是了」
。
　突然想起她的事情，我苦笑著。
。
　如果她在王國還是精神飽滿就好了。
　不過以莉莉那正直的性格，真怕她會與擁有不正當勾當的上層對立。
。
　我和羅德里格斯閑聊一會兒之後，白兔奶汁烤菜就做好了。

 奶汁烤菜是在羅羅納村的時候，莉莉的得意料理。
。
　用白醬汁燉煮兔肉一陣子，接著是把胡蘿蔔和花椰菜切成一塊一塊。
　然後把這些都裝進容器裡，在這些上面塗滿大量奶酪。接著放進爐灶裡煎烤。
。
　白醬汁與奶酪結合在一起，絕妙的味道。
　她的奶汁烤菜實在美味……。
。
　我用叉子插進放在眼前的奶汁烤菜，粘稠粘稠的。
　迅速含一口。
。
　奶酪那濃厚味道和獨制白醬汁的美味在口中擴散而開。
　雖然我挺喜歡莉莉做的奶汁烤菜，可這也有這的美味。
。
「好好吃……」
「對吧？」
。
　羅德里格斯也喜滋滋地頻繁點著頭。
。
「羅德里格斯的店又是炒飯又是肉丸子，還能做奶汁烤菜。不管什麼都做得出來呢」
「那當然，不然每天都是一樣的菜色，客人也會吃膩的吧。每天的菜色都是根據那一天買入的食材來決定」
。
「嘿~。雖然對於吃飯的人來說，有各種各樣的菜色的確開心，可制作人就不得了呢」
「嘛。就算開了家店，每天都還是在學習喔」
。
　果然，廚師的世界也很嚴酷的樣子。
　不管是哪一個領域，如果沒有日日精進的話，是無法成為一流人士吧。
。
　我把烤好的奶汁烤菜吃完，填飽肚子之後。笑著道謝交給大叔三枚銅幣。
。
「喂。魯庫，兩枚銅幣就可以了吧」
「不。今天的非常美味。而且……還想起以前的事情來了」
。
　當看到我表情上微微滲透的哀愁，羅德里格斯默默點下頭表情。
。
「要是能見面就好了啊。跟那位叫莉莉的小姐」
「是呢。只要活下去，總有一天絕對會的」
。
　我離開了攤子。

。

　然後步行在大街道上，在商業公會把魔石賣掉。
　由於這一次中級魔石有三顆，所以總共賣出5枚銀幣。
。
　總資產是6枚銀幣。
　這樣一來，口袋也不會是空空如也。
。
　而且也能夠購買新的裝備。
　嘛，首先是去聖教會接受成長之儀。
。
　賣完魔石後前往聖教會，我決定拜托神父為我舉行成長之儀。
　支付1枚銀幣，接受儀式。
。
　在教會那神聖氣氛中，響起神父的祝詞，儀式結束。
。
「這一次能學會的魔法是這些吧」
。
　我眺望在水晶映射的可以學會的魔法一覽表。
。

【魯庫　可以取得的魔法・技能一覽表】
。
＜可以取得的新魔法＞

。

。

。
　噢，能夠取得新的防御魔法了。
　為了先制攻擊，第一個就確定是學會範圍魔法。畢竟也沒有其他應該學會的妨礙魔法，這一次或許學會防御魔法比較好。
。
範圍魔法有熔岩巨湧和荒雷巨蟒嗎。
要選哪一個比較好呢
。
　就情況而言，雷系統是我主用的魔法，所以範圍魔法還是想避免學會荒雷巨蟒而提升太多雷系統熟練度。
　變強也不是說有時間限制，現在就慢慢提高各種系統的熟練度吧。
。
　如此這般，應當學會的範圍魔法就是火系統的熔岩巨湧。
。
「總之第一個已經決定是熔岩巨湧，拜托您了」
「我知道了」
。
　神之祝福的光芒漫天飛舞，我接受天啟而學會了新魔法。
。
「然後是關於第二個要學的魔法……神父，水系統能夠熟練使用的時候，有什麼好處嗎？」
「水系統是治療和防御優秀的系統，所以想要在這方面發展的話，就應該要學會吧」
。
　原來如此。
　既然我要以全能力戰鬥的魔導師為目標，情況緊急的時候有治療魔法也沒有損失吧。
。
　而且瀑布牆是防御魔法，像最近跟骷髏奇美拉戰鬥的時候，那種炎之吐息也能夠抵抗了。
　真要說起來，我老是輸出和妨礙敵人而已，防御方面就很少接觸。
。
　現在就加強防御能力吧。
。
「那麼，第二個就拜托你給我瀑布牆」
「嗯」
。
　就這樣，我學會了範圍魔法熔岩巨湧和防御魔法瀑布牆。
