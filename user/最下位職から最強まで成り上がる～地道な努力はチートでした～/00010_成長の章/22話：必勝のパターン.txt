
　我和羅伊正探索著被光球所照亮的第二階層。
　地下城到處都設置了陷阱，而每當我接近這些陷阱，我手中的法杖就會通過發出紅色光芒來告訴我。
。
　賦予陷阱探知魔法效果的法杖。
　當法杖會發出紅色光芒，也就代表前方繼續前進就會有陷阱。
。
　每當法杖發出紅色光，我就走到道路邊緣處，迂回走過。
　即使萬能藥可以治療毒與麻痹狀態，能夠不觸发陷阱的情況下進行探索才是謝天謝地。
。
　於是就這樣平淡無奇地繼續探索著，直到魔物襲擊而來。
。
「敵人接近，5秒後交戰了！」
「是。我進行先制攻擊」
「拜托了」
。
　跟羅伊簡短地交談完，確認戰術後。我往黑暗的彼端釋放雷射彈。
　在漆黑的黑暗對面，魔物發出低沉悲鳴聲。
。
　一群魔物仿佛要分開雷屬性彈幕似的現身了。
。
「好，接下來交給我負責！退到後衛去，魯庫」
「請等一下，還沒好！」
。
　敵我距離，約7亞魯多(約10米)。
　這就是迷宮裡有效視野戰鬥的最大距離。
。
　反過來說，也就代表在進入近距離戰鬥之前還有4亞魯多(約5米)的緩期。
　在這緩期時間裡也不是沒有可以使用的攻擊手段。
。
　我停止釋放雷射彈，在遭遇敵人的同時釋放出最後一擊、中級魔法雷鳴長矛。
。
　雷之矛迸裂而出。
　魔法命中那群魔物中最前面的中級豬頭人，魔物隨著發出「噫呀呀呀啊！」這一悲鳴聲就消失了。

　如同我所想要瞄准的對象。
。
「我要退到後衛。前衛就拜托你負責了！」
「了解」
。
　我立刻隱藏在羅伊背後。
　在上一次戰鬥中我就知道雷鳴長矛足夠當做主要輸出魔法來使用，所以這一次的先制攻擊就決定是以雷射彈的彈幕射擊和即將要遭遇魔物的時候釋放的雷鳴長矛。
。
　而這個方案都漂亮的命中敵人，在戰鬥開幕的時點上，就已經成功大幅度削弱敵人戰力了。
　由於我在開幕前就用雷鳴長槍打倒一頭中級豬頭人，所以敵人殘余戰力是
。

　前衛　樹妖×2　中級豬頭人×1
　中衛　骷髏奇美拉×2
　後衛　巫妖×2

。
　在那這段時間裡，驚異的就會是釋放魔法攻擊的巫妖和以炎之息攻擊而來的骷髏奇美拉。
　前衛的樹妖和豬頭人都是只會用單純的物理攻擊攻擊過來，所以只要交給羅伊就萬無一失。
。
　只見羅伊和樹妖已經進入交戰狀態，羅伊邊鑽過樹妖來回揮舞的粗大巨腕，邊反擊回去。
。
　我也不做好自己應該做的事情的話。
　迅速判斷敵人中衛到後衛的戰況。
。
　骷髏奇美拉一擊進入吐息攻擊的準備階段，巫妖由於雷射彈的傷害而行動少許有點遲鈍。
　既然如此，這裡應該打擊的是骷髏奇美拉。
。
　以我的主要魔法，攻擊速度是A級別的雷鳴長矛。
　對著骷髏奇美拉釋放出雷之矛。
。
　雷鳴長矛打中了大幅度仰著頭、進入吐息攻擊的準備動作的骷髏奇美拉。
　骷髏奇美拉1發出死亡的悲鳴後消失了。
。
　然而，在這一空隙中，骷髏奇美拉2的吐息攻擊已經準備完畢的樣子。骷髏奇美拉2噴出炎之吐息。
。
「咕……！」
。
　連用狂沙拘束來封印行動的時間都沒有，猛烈燃燒的炎之吐息就发射而來。
　我立刻判斷往後方跳躍。
。
　烈焰在我和羅伊之間的地面上高高燃起。
　在戰場中央的火柱仿佛分割我們兩人一般，劇烈燃燒著。
。
　視野被火焰所遮擋而變得狹隘。
　可並沒有受到傷害。
。
　然而卻被炎之壁限制了行動。
　雖說可以釋放雷鳴長矛連同炎之壁一起攻擊，可連是否命中都不清楚。

 然而就這樣等待炎之壁消失，也就代表我和羅伊會被區分開來。　
　在戰力被分割的情況下還進行戰鬥，簡直是愚蠢至極。
。
　羅伊即使一個人也能從容不迫吧，可如果在戰力分割的時候受到背後攻擊，我瞬間就會被幹掉吧。
。
　我為了消滅火柱，釋放至今從來沒用過的水瀑球往火柱发射出去。
　水球與劇烈燃燒的火焰相撞，火焰邊釋放出水蒸氣邊熄滅了。
。
「魯庫！沒事吧！」
。
　在滅掉的火焰前方，羅伊很擔心似的往這邊回過頭。
　羅伊似乎已經全滅敵人前衛的樣子，殘余的魔物就只有中衛的骷髏奇美拉與後衛的巫妖。

「是的，沒問題。對不起，戰力被分割了」
「今後也不得不練習能夠應對敵人的魔法攻擊的技術了啊」
。
　正是如此。
　魔物也不可能一直會老老實實的隨著我方隨心所欲地去做。
。
　在用先制攻擊給與到魔物打擊之後，接下來的展開也要有利于戰鬥。
　得把這樣的戰術與魔法能力練得更為精通。
。
　我重振姿勢，往後衛的巫妖投射雷鳴長矛。
　兩支切開大氣的雷之矛。
。
　雷鳴長矛命中巫妖，而巫妖發出悲鳴聲消失了。
　與此同時，羅伊也打倒奇美拉，我們戰鬥也就等於是勝利了。
。
　拿出戰鬥記錄日記，回顧一下這次戰鬥的反省點並記上去。
。
「對了……即便在一開始給與打擊而建立起戰況優勢，可如果在之後的展開中還是被分割戰力那就不行了。如果不學到一種能在魔法戰中起到優勢地位、能給予敵人後衛致命性傷害的魔法的話……」
。
　我一個人嘟嘟嚷嚷開著反省會的時候，撿完魔石的羅伊走過來。
。
「魯庫。你也應該是時候學習範圍魔法了吧？」
「範圍魔法嗎」
。
「沒錯。魯庫你的魔法基本是單體攻擊魔法吧？以現狀來說，大多數情況是以先制攻擊的雷射彈使其敵人隊形崩潰，在即將要遭遇時釋放的雷鳴長矛，緊接著就是以狂沙拘束纏住敵人並釋放雷鳴長矛就沒了。
　這樣一來，以如今的情況就只能是逐個打倒魔物。所以這一次，才會被兩頭骷髏奇美拉的吐息給分割開來」
。
「我正在深刻反省……」
。
　我垂頭喪氣著。
。
「啊啊，不是。抱歉，我也不是在責備你。正因為如此，如果在遭遇之前是使用一擊強力的範圍魔法，以你的魔法不就可以讓敵人中衛到後衛都完全崩潰嗎。不就能輕鬆的讓戰況有利起來，我想說的只有這些」
。
「的確是這樣呢……」
。
　由於這次在遭遇前是使用了雷鳴長矛，所以也就只是打倒一頭中級豬頭人而已，可如果這時候是用還沒學會的範圍魔法、熔岩巨湧和荒雷巨蟒，應該能夠給與到魔物相當致命的傷害吧。
(颯:最近重溫一下碧之軌跡，发現漢化組是翻譯成熔岩爆發)
。
　既能讓隊列崩潰，也能一口氣讓2～3只魔物死亡。
　如此一來，中衛～後衛就會是滅亡狀態。在戰鬥開始的時點上，主導權一下子就已經在我們這邊。
。
「不錯呢，範圍魔法。或許跟我的先制攻擊相性合得來也說不定」
「對吧？如果再加上你的範圍輸出魔法，我覺得應該非常實用。這就將會是我們的必勝形式哦」
。
　聽到羅伊所說，我“嗯嗯”地點著頭。
。
「不錯的主意，通過釋放範圍魔法的先制攻擊。立刻去學會吧」
「今天就到此為止，先回一趟城裡取得新魔法怎麼樣？」
。
「啊，請稍微等一下。最近已經學會兩個魔法了，所以不知道現在還能不能學習……讓我用鑒定魔法確認一下是否已經到達能夠學會新魔法的等級」
。
　我如此拒絕羅伊，對自己使用鑒定魔法。
　我如今的等級是9。
。
　從等級5到7取得了雷射彈。
　然後提升到8、9取得了雷鳴長矛和狂沙拘束。
。
　要學會下一個魔法，最低也要到達等級10，不然根本不可能。
。
「好像不再提升一級就不能學習魔法。再稍微在第二階層提高等級吧」
「我明白了。那麼，繼續探索狩獵魔物吧」
。
「對不起，讓你陪著我一起」
「別在意。培育你就是我的工作」
。
　一直以來都謝謝了，羅伊。
