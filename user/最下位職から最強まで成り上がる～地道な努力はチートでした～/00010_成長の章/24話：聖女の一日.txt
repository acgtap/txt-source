
　早上好！我是莉莉。
　我的一天很简单。
。
　在天亮之前就从床上起来，简单的整理好仪容仪表。
　往睡觉的时候压到的翘头髮沾点水，用木制梳子梳顺。
。
　纵使有多麼急急忙忙，至少也一定要洗把脸，把头髮打理好。
　最後剩餘的是我作为女孩子的自尊心。
。
　当这些搞定之後，就会来到一楼的客厅吃着最近雇佣的佣人为我制作的早餐。
。
　从厨房那裡传来温暖人心的汤汁香味。
　啊啊，仅仅是闻到气味，似乎很好吃的样子。
。
「塞西莉，谢谢你平时都这麼早」
「啊，大小姐。早安」
。
　尽管之前鲁库还会为我做早餐，可如今这麼宽敞的豪宅裡，除了我以外谁也不在。
。
　由於太过寂寥，而且骑士团的工作和家务双双应付也很辛苦，於是就雇佣了个女孩子塞西莉作为女仆。
。
「今天的菜色是什麼？」
「黑麦麵包、蔬菜汤、莴苣还有奶酪色拉」
(颯:你要我这个每天起床路上买根油条吃边急急忙忙上班的社畜怎麼想)
。
「好像很好吃的样子。塞西莉真擅长做料理呢」
「没有，没这回事……」
。
　缅甸的塞西莉真可爱。
。
　吃完塞西莉做好的早餐，我就穿戴上骑士甲胄。
　穿好白银铠甲，把骑士剑在腰上挂好。準备就完毕了。
。
「那麼，我今天也要出去了哦」
「好的。路上请小心，大小姐」
「谢谢」
。
　对迎送我的塞西莉道谢，我走出家门。
　黎明时分，我走在王都的路上，左顾右盼地观察人们。
。
　一大早就要把牛奶和水果送往贵族宅邸的人，从大清早开始就在準备的肉店大叔。平常无事可干而早起的贵族妇人们也正在散步。
。
　当我走在这条有这般形形色色的人群街道上，就被人搭话了。
。
「早上好，莉莉酱！」
「啊，麵包店的阿姨。早上好」
。
「前段时间真是谢谢你了呢。我家老公在商业都市行商回来的路上被魔物袭击的时候，是你保护他们的吧？」
「没事。保护市民就是我们骑士团的职责所」
。
　前段日子，我们骑士团在王都外面进行讨伐魔物任务的时候，由於商人一行人被魔物所袭击。我违背团长的停止命令，以个人名义前往救助这件事。如今被阿姨感谢了。
。
　袭击行商人士的魔物相当强大，团长考虑到骑士团全体的安全，硬是下达对商人见死不救的判断。
　可我无法放任不管，一意孤行去保护商人们。
。
　当然，违法命令的我受到严酷斥责，还有让那些为我掩护的其他团员陷入生死危险的责任。
　即便如此，对我说过的那句“谢谢你救了我”，让我感到温暖开心。
。
「莉莉酱是个好孩子呢。要是骑士团和王国上层都像莉莉酱这样的孩子的话……」
「哈哈。可我现在还只是个不行的新人骑士哦」
。
　不过，只是看到麵包店阿姨的笑容，就会让我想到『啊啊，有努力过真是太好了』。
。
「虽然会被周围人说三道四，不过我会支持帮助过我老公的你哦。加油吧！」
「好的！」
。
　结束与麵包店阿姨的寒暄，我就直接以王都外面走去。
　当抵达通往城外的城门，卫兵对我敬礼。
。
「莉莉大人。早上好！请问您今天也是自主锻炼吗？」
「早上好。嗯，对。能稍微让我通过城门吧」
。
「当然没问题。请一路小心！」
「谢谢你」
。
　通过城门，在那外面是一片广阔草原连绵不断。
　我瞭望遥远的水平线上的太阳升起，边开始做屈伸运动。
。
　当我做了一会儿準备运动，有个人打着哈欠来到我身边。
。
「呒哈啊……。莉莉每天真能坚持呢」
。
　同样是身穿白银骑士甲胄的同事女孩子・贝亚托莉杰。
　贝亚托莉杰是北方国家出身，以这个国家的发音显得稀奇，所以简称为莉杰来称呼。
。
「莉杰。早上好」
「早上好。哈啊……虽然从莉莉复归之後，我就陪着你做早晨的自主锻炼呢，可我已经坚持两星期要吃不消了……」
。
　贝亚托莉杰和我幾乎是同一期的女孩子，不管是在我宅在家裡前还是复归之後，都还是以温柔的态度对待我。
。
「既然想睡觉的话，去睡不就好了莉杰。训练我也会尽我最大努力去做」
「嗯~，虽然是这样啊。可是感觉会被抛之脑後，我不要」
。
　贝亚托莉杰睡眼惺忪地撅着嘴巴说出不满，我苦笑着。
。
「毕竟是同期呢，我和莉杰。不怎麼想拉开差距吧」
「对对。同样是女孩子、同样是骑士团所属，所谓的同期可是很宝贵的哦」
。
　实际上，所属於骑士团的15岁左右的女孩子是非常稀少的。
　整个骑士团是由40名团员左右所构成，而那幾乎都是20岁以上的男性。
。
　虽说职业的技能补正使得能够拟补某种程度上的力量差・体力差。可这原本就是男性優先的职业，如果以未成年的女孩子成为骑士的情况来看，如同濒临灭绝。
。
　所以自然而然的与同期、同年、同性的莉杰关係变好。
。
「那麼，开始做吧」
「呜……」
。
　莉杰打从心底发出倦怠声，点着头。
。
　在我复归骑士团以来，除了早上团员进行的团体训练，还给自己追加新的自主训练课程。
。
　无论如何，首先我的力量是决定性的不足。

 不管是与魔物战鬥的力量，还是权利之类的，我的力量实在不足以从这些巨大事物中保护好重要的东西。
。
　所以至少想要激励自己提高战鬥能力而开始自主训练。
　於是，早上一起床就穿上骑士铠甲围绕王都周围跑步，跟莉杰进行模拟战、以双手双脚登上城墙等等诸如此类的训练。
。
　我在心中对陪我一起的莉杰道谢着，边在太阳照耀下的草原上奔跑。
。
「说起来啊，莉莉」
「嗯？」
。
　我们是打算锻炼到即使在慢跑中对话也不会呼吸急促的程度。
。
「有听说过艾杰巴拉王国的那个传闻吗？」
「传闻是什麼传闻？」
。
「距离艾杰巴拉最近的城镇・西里卢卡城……换句话说，也就是我国维鲁利亚王国的国境附近的城镇来着」
「嗯」
。
　我有规律的哈、哈、哈调整呼吸，边跑着边点头。
。
「听说在那座城镇裡有个两人组的男冒险者每天都潜入地下城呢」
「嘿~」
。
　我并没有多大兴趣，就随声附和了。
。
「嘿是什麼啊……」
「不，因为啦。实际上也就只能这麼说吧。虽然会觉得好厲害啊，居然能在雷斯迪凯普猎杀魔物啊什麼的」
。
「对。这裡就是问题所在哦，莉莉小姐」
「问题是什麼啊，莉杰小姐」
。
　双方都莫名其妙地使用敬语，我歪着头。
　在慢跑的途中，莉杰用手指指着我的脸。
。
「普通的冒险者会认为能在雷斯迪凯普猎杀魔物吗？出现在地面上的魔物会成群袭击人类，可也不到雷斯迪凯普那般坚固无比的队伍构成啊。你觉得特意在魔物成群结队袭击而来的雷斯迪凯普猎杀有什麼利益吗？」
「不觉得」
。
「那麼，你不觉得那个两人组到底是什麼来头吗？」
「不觉……觉得！」
。
　听到我的回答，莉杰满意似的频繁点头。
。
「对吧。是这样对吧。於是优秀的名侦探莉杰大人通过私人渠道试着收集情报了哦」
「莉杰真的是喜欢查探这样的传闻和内幕呢」
。
「那边的！别说我性格恶劣！」
「我没说过，我没说过」
。
　我苦笑着摇摇头否定。
。
「真是的……明明还想给你带来个超级好消息……」
「然後呢，是什麼啊。那个谜一样的两人组真面目」
。
「给我听好了。一个是曾经被称之为王国最强骑士、那个传说中的罗伊・埃梅拉路多」
「那个传说中、是说我也只是知道传闻的程度」
。
「嗯嘛啊，虽然我也一样……。嘛，先不管这件事」
「嗯」
。
　我随声附和，等待话题的接续。
。
「那个传说中的原骑士培育的伙伴，看样子好像是新人魔导师哦」
「嗯~……」
。
　从听到这段话开始，我心中产生出感兴趣的萌芽。
　新人魔导师？
。
「是跟我们同个年龄的黑髮男孩子，不管对谁都是用敬语，谦虚谨慎、礼仪端正。再加上传闻说是从维鲁利亚王国来的男孩子。呐。不觉得很熟悉吗？」
。
「不会吧」
。
　不会吧，不会吧。
。
　连续三次吐露出惊讶之声。

「对。剑神罗伊如今包养的伙伴的名字是鲁库。听说是魔导师鲁库哦，莉莉」
。
　我过於惊讶而顿时停下脚步。
　颤抖地吐了口气，径直地看着贝亚托莉杰那端正脸孔。
。
　她仿佛像是在说「怎麼样」，回了我个微笑。
。
「莉杰……！」
「我偶尔也会派的上用场吧？」
。
　就连莉杰那玩笑般的眨眼都让我觉得甚至是女神的微笑。
　我犹如晴天霹雳一般呆然着，那停止运转的大脑也被唤来一丝生机。
。
　黑髮，敬语，从维鲁利亚来到艾杰巴拉的男子。
　名字是……鲁库。
。
　还活着啊……。
　泪水、悄然地在脸颊上流下。
。
　意识到的时候，我已经奋力抱紧她了。
。
「莉杰，莉杰，莉杰！」
「好痛痛痛，有点痛啊莉莉」
。
　我丝毫不在意铠甲，只是全力抱住莉杰。
。
「莉杰，谢谢你！」
「啊哈哈……不客气。虽然最初是因我个人兴趣才调查的呢。能帮上忙真是太好了」
。
「还活着！鲁库真的还活着！」
「嘛，就是这麼一回事吧」
。
　喜悦充斥着全身。
　太过於欢天喜地，以至于我无法压抑住泪水。
。
「不敢相信……居然还会有这种事……！啊啊，太好了……太好了鲁库……！」
。
　如果能够再次相会，我一定要说一件事。
。
　谢谢你，最喜欢你了。
。
　我期望着把我满身爱情传达给他。
　我已经不想在心意还没传达出去之前，就失去最珍贵的人了。
。
「嗯。然後呢，你要怎麼办，莉莉」
「什、什麼怎麼办？」
。
　意料之外的询问使我退缩起来。
。
「虽然我觉得莉莉你即使把在维鲁利亚培育的一切都弃之不顾，前往他的身边也是极有意义的事情」
「诶……这是什麼意思……？」
。
　我没办法立刻理解贝亚托莉杰的话。
。
「所以说啊，他是你喜欢的男人吧？为了你而奋不顾身、你最喜欢最喜欢喜欢到无法自拔的他对吧？无论是放弃骑士团还是贵族，或是在这个王国应当守护的一切，我觉得作为推动你前往鲁库的身边的理由应该十分足够才对」
。
「那、是……」
。
　如果能够做到，我也想这麼做。
　纵使把这个王国我所拥有的一切珍贵事物都凝聚而起，也比拟不了他的存在。
。
　但是。

 脑海裡回忆起刚才麵包店阿姨说过的感谢话语。
。
　莉莉酱，谢谢你了。
。
　仅仅是单纯的一句话，却让我十分开心。
　让我觉得即使在这个王国裡生活也不错。
。
　并不只是麵包店的阿姨。
　在中途保护的行商人们、日夜保护王都市民的卫兵们、在王都之外的农村裡与魔物造成的损失战鬥，而死命耕作农物的农民们。
。
　大家的期待都由圣十字骑士团的莉莉这一存在所背负。
。
　所以，我领悟到了。
　啊啊，如果是鲁库的话，大概是不会背叛大家的期待，而前往一个女人的身边吧。
。
「办不到……」
「诶？」
。
「办不到。正因为我在王国有应当要守护的事物，所以我这裡有不得不做的事情」
。
　我如此说道，贝亚托莉杰目瞪口呆。
　把在这个国家应该要做的事情都完全解决掉。
　我打算在把那些悲哀之人都拯救完毕之後，再前往他的身边。
。
　到时候，鲁库肯定会笑着迎接我。
　如果是他，就会理解我相信的道路。
。
「真的好吗？国家不同，随时都有可能发生战争。或许还会不能自由地再次相见了哦」
「即使如此，即使如此。我也做不到把他为我创造的立场抛弃掉，也无法背叛王国的人民对我的期待」
。
「是吗。真是了不起的事情，不过那可是修罗之路哦，小姐」
「我还有你在，所以没问题哦」
。
　我和贝亚托莉杰相互之间对视而笑。
