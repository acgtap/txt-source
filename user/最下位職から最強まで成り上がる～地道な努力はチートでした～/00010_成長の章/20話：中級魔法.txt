。
　好好吃。
　果然美味的飯菜能讓人感到幸福。
。
　我心滿意足地前往聖教會。
。

　把以大理石製作的聖教會大門打開，進入其中。跟平時一樣被神聖的氣氛所圍繞。
。
　太陽光穿透彩色玻璃，以光輝耀眼的七色彩傾注而下。

。
「您好，神父」
「哦呀，是魯庫君啊。歡迎來到聖教會。有什麼事嗎？」
。
「想拜託你成長之儀」
「了解」
。

　雖然成長之儀的手續費跟上回一樣是1枚銀幣和5枚銅幣。即便如此，我還是下定決心試著交涉看看，結果把金額砍價到1枚銀幣。
。
　嗯~既然交渉能夠砍價，那就沒有不老實的理由了吧……。
　我想要以這個狀態繼續主動提出交涉。
。
　神父是沒有其他的工作嗎，就這樣在祭壇面前實行成長之儀。
。
　我等著神父說完祝詞與禱告。
　只有這些是超麻煩的，完全開心不起來。
。
　即便如此，一想到這是即將要成長的時機，心就砰砰直跳。
。
「好了，神也把祝福獻給于魯庫君了。開始學習魔法・技能吧」
。
　來了來了。
　能學會新魔法的瞬間，才是最開心的時刻。
。
　我窺視著水晶裡面可以取得的魔法與技能的一覽表。

。

「噢~！可以取得的魔法增加許多了呢」
「看來是這樣呢。你想要學習哪一個？」

　我們兩人一同看著水晶，邊商談關於今後取得的魔法。
。
「哪一個比較好啊……。如今在雷斯迪凱普的第二階層戰鬥，實際感覺到魔法能力的壓倒性不足」
「現在魯庫你是以使用什麼魔法為主？」
。
　我邊看著水晶，邊回答他的提問。
。
「用雷射彈阻止其步伐，再用火焰球打上去。這就是如今我的王道戰法吧」
「的確這樣做，或許在雷斯迪凱普是很棘手吧……。那麼我推薦你學習速度還是維持原樣，可威力能夠加強的雷鳴長矛吧」
。
　的確。
　雷鳴長矛完全是雷射彈的上位魔法。
。
　攻擊速度還是A，可威力能夠提升到C。
　相反，魔力消費A可就不是鬧著玩的。
。
「可這個魔力消耗太猛烈了吧。感覺連續地使用，很快就會陷入魔力缺乏的狀態」
「即便如此，魯庫也是著實地在提高等級，總魔力量也應該增加了才對哦。我覺得你一開始可以隨身攜帶魔力藥水，當情況緊急的時候還是有備無患比較好」
。
「嗯~……」
。
　仔細考慮神父說的話。
　在速度不降低的情況下，能夠增強威力我是很開心，而且先取得雷鳴長矛也沒有什麼損失麼。
。
「我知道了，總之第一個魔法就決定是雷鳴長矛了」
「了解」
。
　神父對神獻上禱告，然後閃閃發光的光輝聚集在我的身體上。
。
「取得完畢」
「非常感謝。第二個是……」
。
　繼續看著水晶深入思考。
　對了。
　為了應付第二階層的陷阱，陷阱探知魔法是必須的。
。
「我想要能夠探知陷阱的魔法來著，這種魔法要怎麼樣才能學會呢？」
「探知魔法是上位系統的無系統魔法呢。只要提高土系統和風系統的熟練度，早晚會出現的吧」
。
　我把手這下巴上思考著。
　是嗎。
　只要完全提高一個種類，也會出現這樣偏倚的一面嗎。
。
「感覺不是一朝一夕就能得到的呢」
「是呢。踏踏實實地提高熟練度等待無系統的出現，又或許急需的話，還可以用施加探知技能的裝備來補足這一辦法」
。
「裝備！還有這一手嗎！」
。
　真是說到盲點了。
　如今才開始提高風與土，都不知道何時才能夠取得。
。
　如果能用裝備來補充，那我還是選擇裝備好了。

 「魯庫君的裝備是在莎蕾的店裡買的吧？」
「是在稍微有些懶散、沒幹勁的店長姐姐的店裡買的這件長袍」
「是莎蕾吧，不會錯的」
。
　神父微微一笑。
。
「別看她那樣，莎蕾的店還是具備許多優秀的裝備，所以試著去跟她商量一下也不錯哦」
「我明白了。那麼陷阱探知就用裝備來補足，第二個魔法就試試看妨害魔法的狂沙拘束吧」
。
　從名字可以想像得到是用土沙封印住敵人行動的魔法，感覺如果有這個魔法，戰術也就能更加多樣化。
　真要說的話，感覺我比起用輸出魔法壓制到底，不如掩護羅伊主動妨礙敵人，邊進行戰鬥還比較合我性子。
。
　而且一旦土系統也提高熟練度，到以後想要陷阱探知的魔法的時候也能立刻學會，這樣就可以了吧。
　而且還想學會其他的上位系統的無系統魔法。
。
　我打算邊學會各種系統的魔法，邊以綜合實力戰鬥的魔導師為目標。
。
「那麼第二個魔法就拜託你給我狂沙拘束」
「了解。給予魯庫君神之祝福」
。

　學會新魔法雷鳴長槍和狂沙拘束之後，我對神父道謝完就離開了聖教會。

。

　我推開木制門扉，進入店裡。
。
「歡迎光臨……」
「還是老樣子沒幹勁的打招呼呢」
。
　看到懶洋洋地以手托腮的店長莎蕾，我苦笑著。
　莎蕾店裡的商品都整理得漂漂亮亮。
。
「噢噢，這不是從維魯利亞而來的少年嗎」
「我是魯庫。又過來買裝備了」
。
「魯庫君嗎。我叫莎蕾哦。要成為我店裡的常客了吧。歡迎歡迎～♪」
「其實，我有要尋找的裝備來著」

。

「在雷斯迪凱普第二階層遭遇到陷阱，所以我想要防備這個。請問有附加上陷阱探知效果的裝備嗎？」

「陷阱探知効果的裝備、嗎……有是有，可不管哪一個都很貴哦？」

。
「如果要賺到能夠雙手佩戴最多10個可以抵抗毒・麻痺・結凍等等的狀態，還要有陷阱探知的效果的話，金額就相當高了。

。
「為什麼長袍和法杖會便宜呢？」
「因為長袍會附加上最大防御力・魔力最大值倍率提升的魔法效果，而法杖是攻擊速度提升和魔法攻擊力提高等等的超優秀魔法賦予效果」
。
「那很厲害呢」
「是呢。以魔導師的最終裝備來考慮的話，果然還是想要在法杖和長袍上賦予賦予最大防御力、攻撃速度、魔法攻撃力、魔法熟練度的提升効果呢。
　所以大家都不怎麼在法杖和長袍上賦予抵抗屬性和探知魔法。
　因為剛才說過的賦予魔法太過於優秀了呢」
。
「是嗎……。雙手能佩戴10個戒指的話，大家都是從這些戒指上獲取毒和冰結抵抗效果、陷阱探知的吧」
。
「就是這麼一回事。也就是說，總之在法杖和長袍上賦予陷阱探知也可以的話，2枚銀幣還是可以通融的」
。
　應該要怎麼做。

。
　當在第二階層能夠提升等級之後，就能賺到更多的錢。

「嗯嗯。如果因為這次收益還想要得到新裝備，或者有困擾的事情的話，不管是什麼都可以來拜託我哦」

。
「不用客氣」
