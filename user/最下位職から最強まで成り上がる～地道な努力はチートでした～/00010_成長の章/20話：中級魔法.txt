
　在大叔的摊子上狂吃米饭和添加甜醋的肉丸子，我的心情已经是幸福美满。
。
　好好吃。
　果然美味的饭菜能让人感到幸福。
。
　我心满意足地前往圣教会。
。
　走在西里卢卡城的街道上，抵达雕刻着庄严肃穆般纹样的建筑物。
　把以大理石制作的圣教会大门打开，进入其中。跟平时一样被神圣的气氛所围绕。
。
　太阳光穿透彩色玻璃，以光辉耀眼的七色彩倾注而下。
　我笔直地走到中央深处的祭坛，对神父搭话说。
。
「您好，神父」
「哦呀，是鲁库君啊。欢迎来到圣教会。有什麼事吗？」
。
「想拜托你成长之仪」
「了解」
。
　神父笑着点头。
　虽然成长之仪的手续费跟上回一样是1枚银币和5枚铜币。即便如此，我还是下定决心试着交涉看看，结果把金额砍价到1枚银币。
。
　嗯~既然交渉能够砍价，那就没有不老实的理由了吧……。
　我想要以这个状态继续主动提出交涉。
。
　神父是没有其他的工作吗，就这样在祭坛面前实行成长之仪。
。
　我等着神父说完祝词与祷告。
　只有这些是超麻烦的，完全开心不起来。
。
　即便如此，一想到这是即将要成长的时机，心就砰砰直跳。
。
「好了，神也把祝福献给于鲁库君了。开始学习魔法・技能吧」
。
　来了来了。
　能学会新魔法的瞬间，才是最开心的时刻。
。
　我窥视着水晶裡面可以取得的魔法与技能的一览表。

。

。

「噢~！可以取得的魔法增加许多了呢」
「看来是这样呢。你想要学习哪一个？」
。
　我们两人一同看着水晶，边商谈关於今後取得的魔法。
。
「哪一个比较好啊……。如今在雷斯迪凯普的第二阶层战鬥，实际感觉到魔法能力的压倒性不足」
「现在鲁库你是以使用什麼魔法为主？」
。
　我边看着水晶，边回答他的提问。
。
「用雷射弹阻止其步伐，再用火焰球打上去。这就是如今我的王道战法吧」
「的确这样做，或许在雷斯迪凯普是很棘手吧……。那麼我推荐你学习速度还是维持原样，可威力能够加强的雷鸣长矛吧」
。
　的确。
　雷鸣长矛完全是雷射弹的上位魔法。
。
　攻击速度还是A，可威力能够提升到C。
　相反，魔力消费A可就不是闹着玩的。
。
「可这个魔力消耗太猛烈了吧。感觉连续地使用，很快就会陷入魔力缺乏的状态」
「即便如此，鲁库也是着实地在提高等级，总魔力量也应该增加了才对哦。我觉得你一开始可以随身携带魔力药水，当情况紧急的时候还是有备无患比较好」
。
「嗯~……」
。
　仔细考虑神父说的话。
　在速度不降低的情况下，能够增强威力我是很开心，而且先取得雷鸣长矛也没有什麼损失么。
。
「我知道了，总之第一个魔法就决定是雷鸣长矛了」
「了解」
。
　神父对神献上祷告，然後闪闪发光的光辉聚集在我的身体上。
。
「取得完毕」
「非常感谢。第二个是……」
。
　继续看着水晶深入思考。
　对了。
　为了应付第二阶层的陷阱，陷阱探知魔法是必须的。
。
「我想要能够探知陷阱的魔法来着，这种魔法要怎麼样才能学会呢？」
「探知魔法是上位系统的无系统魔法呢。只要提高土系统和风系统的熟练度，早晚会出现的吧」
。
　我把手这下巴上思考着。
　是吗。
　只要完全提高一个种类，也会出现这样偏倚的一面吗。
。
「感觉不是一朝一夕就能得到的呢」
「是呢。踏踏实实地提高熟练度等待无系统的出现，又或许急需的话，还可以用施加探知技能的装备来补足这一办法」
。
「装备！还有这一手吗！」
。
　真是说到盲点了。
　如今才开始提高风与土，都不知道何时才能够取得。
。
　如果能用装备来补充，那我还是选择装备好了。

 「鲁库君的装备是在莎蕾的店裡买的吧？」
「是在稍微有些懒散、没幹劲的店长姐姐的店裡买的这件长袍」
「是莎蕾吧，不会错的」
。
　神父微微一笑。
。
「别看她那样，莎蕾的店还是具备许多优秀的装备，所以试着去跟她商量一下也不错哦」
「我明白了。那麼陷阱探知就用装备来补足，第二个魔法就试试看妨害魔法的狂沙拘束吧」
。
　从名字可以想象得到是用土沙封印住敌人行动的魔法，感觉如果有这个魔法，战术也就能更加多样化。
　真要说的话，感觉我比起用输出魔法压制到底，不如掩护罗伊主动妨碍敌人，边进行战鬥还比较合我性子。
。
　而且一旦土系统也提高熟练度，到以後想要陷阱探知的魔法的时候也能立刻学会，这样就可以了吧。
　而且还想学会其他的上位系統的无系統魔法。
。
　我打算边学会各种系统的魔法，边以综合实力战鬥的魔导师为目标。
。
「那麼第二个魔法就拜托你给我狂沙拘束」
「了解。给予鲁库君神之祝福」
。
　神圣的光芒降临在我的身体上。
　学会新魔法雷鸣长枪和狂沙拘束之後，我对神父道谢完就离开了圣教会。

。

　在那之後来到了莎蕾的装备店。
　我推开木制门扉，进入店裡。
。
「欢迎光临……」
「还是老样子没幹劲的打招呼呢」
。
　看到懒洋洋地以手托腮的店长莎蕾，我苦笑着。
　莎蕾店裡的商品都整理得漂漂亮亮。
。
「噢噢，这不是从维鲁利亚而来的少年吗」
「我是鲁库。又过来买装备了」
。
「鲁库君吗。我叫莎蕾哦。要成为我店裡的常客了吧。欢迎欢迎～♪」
「其实，我有要寻找的装备来着」
。
　我畏畏缩缩地提出话题。
。
「嗯？可以哟，不管是什麼都说说看吧」
「在雷斯迪凯普第二阶层遭遇到陷阱，所以我想要防备这个。请问有附加上陷阱探知效果的装备吗？」
。
「陷阱探知効果的装备、吗……有是有，可不管哪一个都很贵哦？」
「诶。果、果然是这样吗。请问多少钱？」
。
　在商业公会得到4枚银币的收入，吃饭花2枚铜币，在成长之仪上用了1枚银币。
　所以剩下的资金是2枚银币和18枚铜币。
。
「还有付与陷阱探知魔法效果的部位呢」
「部位？」
。
「对对。鞋子啊铠甲啊，剑什麼的或者护手」
「啊啊，原来如此……。因为装备可以穿戴在各个部位上，所以不得不考虑到哪个部位要追加什麼魔法效果呢」
。
　就是这麼回事，莎蕾点着头。
。
「如果要赚到能够双手佩戴最多10个可以抵抗毒・麻痹・结冻等等的状态，还要有陷阱探知的效果的话，金额就相当高了。
　反过来铠甲什麼的，啊啊，以你情况的来是长袍吧？　如果附加上陷阱探知的长袍法杖也可以的话，就会很便宜哦」
。
「为什麼长袍和法杖会便宜呢？」
「因为长袍会附加上最大防御力・魔力最大值倍率提升的魔法效果，而法杖是攻击速度提升和魔法攻击力提高等等的超优秀魔法赋予效果」
。
「那很厲害呢」
「是呢。以魔导师的最终装备来考虑的话，果然还是想要在法杖和长袍上赋予赋予最大防御力、攻撃速度、魔法攻撃力、魔法熟练度的提升効果呢。
　所以大家都不怎麼在法杖和长袍上赋予抵抗属性和探知魔法。
　因为刚才说过的赋予魔法太过於优秀了呢」
。
「是吗……。双手能佩戴10个戒指的话，大家都是从这些戒指上获取毒和冰结抵抗效果、陷阱探知的吧」
。
「就是这麼一回事。也就是说，总之在法杖和长袍上赋予陷阱探知也可以的话，2枚银币还是可以通融的」
。
　应该要怎麼做。
　装备和魔法赋予效果当然是越多越好，可如今比起一下子就以最终装备为目标，还不如踏踏实实地收集抵抗异常状态和探知之类的能起到用处的装备，然後再慢慢地改善装备就好了。
。
　当在第二阶层能够提升等级之後，就能赚到更多的钱。
。
「我知道了。那麼我想要有陷阱探知的法杖」
「噢。感谢惠顾～♪」
。
　莎蕾好像因商品卖出去而感到开心似的，把有陷阱探知效果的法杖从架子裡面拿出来。
。
「生财之道生财之道。真美味呢」
。
　好像在说着什麼阴沉的事情。
。
「啊，可是请给我打折。就拜托你以1枚银币和18枚铜币卖给我」
「噢噢……少年也终於学会交涉了吗」
。
「多亏了莎蕾的栽培哦」
「这个混蛋。嘛，算了。像鲁库这样有希望的新人定期的购买我家的装备的话，对我来说绝对是赚到了呢」
。
「是的。请让我永远关顾这裡」
「嗯嗯。如果因为这次收益还想要得到新装备，或者有困扰的事情的话，不管是什麼都可以来拜托我哦」
。
「谢谢你 莎蕾」
。
　我露出笑容道谢着。
。
「不用客气」
。
　就这样，我得到新魔法与新装备，打造出攻略雷斯迪凯普第二阶层的基础了。
