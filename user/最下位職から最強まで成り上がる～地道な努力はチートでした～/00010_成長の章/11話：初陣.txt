
「首先，我有想要確認的事情」

。
　在魔物無限湧出的巢穴、雷斯迪凱普的空白地帶，羅伊對我這麼說。
。

「雖然魯庫說過自己是低位魔導師，可具體能使用什麼樣的魔法？」
「各個火・水・雷系統的初級魔法吧」

。

「那麼，一上來就跟難易度比較高的魔物戰鬥也太棘手了。雷斯迪凱普跟地下城一樣是復層構造，基本上越是往裡面前進，魔物也就會越強大。在低層的魔物大多是弱小的。

　當然，由於魔物也是會逐漸從下往上前進，不久後全部都會進攻人類世界。所以，也有在低層遭遇到強死人的魔物這一例外。
　總之由我來輔助，在第一階層磨練技能和戰鬥經驗吧」

。

「是的，請您多多指教！」
「不錯的回答」

。
　羅伊聽到我的回答，笑著點頭。
　在那之後，我們從有河川流動的空白地帶沿著谷底走了一會兒。
。
　緊接著，有個像是通往地下的樓梯。
。

「那就是雷斯迪凱普真正的入口。別大意了，一旦疏忽，一瞬間就失去性命」
「我明白了」

。
　我點頭肯定羅伊說的話，跟著他的背後進入迷宮裡面。
　雷斯迪凱普的裡面涼颼颼的，沒有光源，理所當然會很黑暗。
。

「如果沒有學會光系魔法，就會在這裡陷入瓶頸。因為無法確保視野啊」

。
　羅伊如此說道的同時，發動名為閃光球(Flash Ball)的魔法。
　手上的光球浮空而起，照亮迷宮內部。
。

「所以基本上在攻略地下城的時候，以前衛火力到後衛支援這種固定編成來組成隊伍」
「至今攻略雷斯迪凱普的維魯利亞王國的討伐隊都全部失敗，有過這樣的一件事……？」

。

「對。就我所知，雷斯迪凱普有二十層以上。人類攻略成功的也只是到第六層」
「魔物有那麼強嗎？」

。

「在強大的以上，雷斯迪凱普的魔物還基本跟人類一樣，會為了擬補相互之間的弱點而組成一隊。使用一旦命中就會施加抵抗不可・回復不可的異常狀態的技能，還有一發動就絕對命中的攻擊等等，瞬間秒殺的魔物實在太多了」
「那麼，比如說――」

。
　在我想要如此詢問的時候，羅伊用手蓋過我的聲音，簡短地大叫。
。

「敵襲！準備好，絕對不要從我背後出來！」
「好、好的！」

。
　在宛如迷宮般的地下城深處，魔物們源源不絕地爬出來。
。
　四只微藍色的透明黏膠形魔物，兩只贅肉鼓鼓的巨大笨重豬形魔物，還有三只披著破爛的紅色斗篷的骨頭魔物。
。

「敵人戰力確認！史萊姆4，豬頭人2，骷髏魔法師3(Bone Wizard)。以上的構成」

。
　羅伊準確地看穿敵人的構成，拔出腰上的銀劍往魔物突進而去。
　敵人前衛是四只史萊姆結黨形成一座牆壁。
。
　趁著羅伊砍倒史萊姆的那一空隙，豬頭人繞到側面用棍棒對著羅伊狠打下去。
。

「呼！」

。
　羅伊側步回避掉豬頭人的一擊。
　然而，魔物們攻擊並沒有就此結束。緊接著，在後衛的骷髏魔法師釋放了魔法。
。
　雷槍與冰槍接二連三地襲擊羅伊，可羅伊以巧妙地步伐躲閃而過。
　這太過於高明的回避行動，讓我不禁看呆了。
。
　雖然我在王城的時候有看過莉莉在騎士團訓練的情況，可他是以這幾倍優雅的步伐。
　比起騎士團團員還要強的他，到底是什麼來頭？
。

「魯庫，我來吸引魔物的仇恨度。你一有從容就在後面用魔法幹掉它們！不管什麼都要有經驗」
「我明白了！」

。
　初次實戰，讓我的手緊張到顫抖。
。
　大口地吐了口氣，我發動著魔法。
　在這個世界裡，一旦學習到的魔法就會完全被記憶在腦海裡。因此，可以如同吃飯時使用勺子那般簡單的來操作。
。
　火系統的初級魔法・火焰球對準骷髏魔法師釋放出去。
　三個火焰球於虛空中出現，火花四散的同時，已飛到敵人的後衛。
。
　在那之中，雖然有一發射偏了，可其他兩發命中了。
　骷髏魔法師被火焰灼燒，發出痛苦的絕叫。
。
　羅伊也間不容髮地追加攻擊，打倒兩只骷髏魔法師。
。

「好！以初次實戰來說，資質不錯哦，魯庫！」

。
　被羅伊誇獎，讓我不由得高興起來。可我被要求的是工作並不是主力輸出。
　而是掩護羅伊。
。
　一看它們，由於已經打倒三只史萊姆和一頭豬頭人，所以剩下的魔物是一只史萊姆・一頭豬頭人・一個骷髏魔法師。
　考慮到我如今魔法的命中精度，恐怕我直接掩護羅伊會導致到誤射友方。
。
　既然如此，那就徹底妨礙骷髏魔法師，讓它不能為它們的前衛做出掩護攻擊，又或者是打倒它。
。
　由於羅伊偶爾會對敵人後衛進行攻擊，所以骷髏魔法師的注意力都被吸引住，對這邊的意識很薄弱。
。
　如今的話，或許可以辦到。
。
　我繼續發動火焰球，創造出三顆火球。
　就跟從小時候開始每天使用勺子和叉子就會越來越順手一樣。魔法也是越使用，使用者使用的魔法也會越熟練。
。
　也不是說有明確性的技能・魔法熟練度的數值，只是持續使用同一個魔法的話，好像某一天就會學到上位的魔法。
。
　也就是所謂的才能發光吧。
　就跟麵包師和皮革製作專家日常進行的修行・訓練是一樣的感覺。

魔法也不是例外，持續地使用好像也很重要。
。
　我再一次使用火焰球，就可以創造出比起剛才還要大一點點的火球出來。
　在我眼前浮空而起的火球，緩慢地描繪出橢圓形飛往骷髏魔法師。
。

「嘎啊啊啊啊啊！」

。
　這一次三發火球都命中了骷髏魔法師。
　紅色的破爛斗篷劇烈燃燒著，只留下死亡的悲鳴就消滅了。
。

「做到了……！」

。
　打從出生以來，終於做到打倒魔物了。
　我感動到顫抖的同時，看向羅伊。前衛的戰鬥也好像結束的樣子。
。
　羅伊把全部的魔物都砍倒了。
　魔物全部躺在地下城的地面上，不久後化為青白色光輝消失殆盡。
。

「戰鬥結束。辛苦了」
「辛、辛苦了」

。
　羅伊對我說出慰勞的話語後，把魔物的掉落物撿了起來。
。

「請問這是在撿什麼？」
「魔力的結晶體啊。也就是所謂的魔石」

。
　魔石已經成為人類生活中不可欠缺的物品。
　比如說，得到水魔石就不用特意去井口打水就能了事，有火魔石就可以省去生火的麻煩。
。
　在有魔法的世界裡，不能使用魔法的人也能享受到魔法恩惠的道具。
　那就是魔石。
。

「魔石是這樣得到的啊」
「因為魔物本身就是製造魔力的存在。所以像這樣打倒之後就會有核心的魔石掉落出來啦」

。

「失去魔力而消失的話，好像很難取得魔物的素材呢」
「只能在完全打倒魔物之前剝下素材。因為完全打倒之後，就會失去魔力而只剩下魔石而已，所以要在瀕死的狀態下剝取材料」

。

「總感覺、很麻煩呢……」
「啊啊，像剛才低等級的魔物素材基本上都賣不出去也不被需要，所以放任不管了。反過來說，高級別的魔物大多都有製作優秀魔道具的時候，用得上的素材。高級別的隊伍裡還有專門剝取素材的職業呢」

。

「嘿~」

。
　所謂的恍然大悟的感覺吧。
。

「嘛，大概就是這樣。給，這是魯庫的份」

。
　羅伊把從骷髏魔法師身上得到的魔石交給我。
　僅僅是看著閃閃發光的紅色結晶，都覺得美麗。
。

「非常感謝。我只是在安全的後方使用魔法而已，讓我拿走真的可以嗎？」
「沒關係。這是你的功勞。紅色魔石是火魔石。拿到公會去，一個差不多可以賣到3枚銅幣哦」

。
　得到三個就是9枚銅幣。
　一枚銀幣的價值是20銅幣，所以一場戰鬥基本上能賺到半枚銀幣的金錢。
。
　有10枚銅幣，即便隨便找家飲食店吃到飽也沒問題。
　在我還在騎士團事務所工作的時候，工資是一個月20枚銀幣，所以冒險者可以說是賺錢效率不錯的好工作。
。

「打倒魔物還挺賺錢的呢」
「因為相對的，會與死亡為鄰啊」

。
　羅伊笑著說。
。

「魯庫在戰鬥中比我想像的還要能行動的樣子，所以暫時就妨礙敵人後衛的步伐，可以的話就拜託你殲滅掉。以這個狀態在第一層習慣戰鬥，提高戰鬥技術吧」
「是！」

。

「還有，這個給你」

。
　羅伊從掛在腰部的皮包裡，拿出一本書。
　表面是用動物的皮革製作，裡面是可以寫日記般的內容。
。

「日記、嗎……？」
「啊啊。從今以後進行戰鬥，在這日記本上把各場戰鬥中的反省點和順利幹掉的事情、失敗的事情、想要改正的事情記上去，就算每一場戰鬥只寫一個也好」

。

「我明白了」

。
　老實說，雖然我不知道到底會有什麼用，可因為是羅伊說的事情，所以我老實地點著頭。
。

「魯庫，為你著想我還是清楚的說吧。你根本不是什麼天才」

。
　雖說這句話我從很早以前就已經做好覺悟了，可被像羅伊這種擁有神技級戰鬥技術的人一說，果然還是有些失望。
。

「可是，也不是說沒有追得上天才的方法」
「而其中一個方法，就是這本日記嗎」

。
　我一說完，羅伊微笑著。
。

「對。記錄每一場戰鬥，定期地回顧戰鬥內容。

　『啊啊，今天沒能做到這件事啊。明明這裡想要改正過了的，可仍然還沒改正啊。在明天習慣這種戰鬥方式習慣之後，下一次就試一下看看吧』。

　把這種一次一次戰鬥中的思考過程、反復嘗試的過程都記在本上。時而回顧一次，學習改正的內容。
　或許你現在還不能理解我說的話。可是，差不多經過三個月，絕對會出現效果」

。

「我就當做是被騙了，試一下看看吧」
「老實這一點，是魯庫你最好的長處」

。
　羅伊猶如向日葵綻放一般，露出了笑容。
　就這樣，我跟著羅伊在地下城裡持續戰鬥，收取著戰鬥記錄。
