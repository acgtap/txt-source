
　我和羅伊在雷斯迪凱普的地下城裡專心致志地打倒魔物，賺取魔石和積累戰鬥經驗。
　基本上第一階層的構造和魔物特性都已經把握了。
。
　如今在第一階層對我們構成威脅的魔物有妖精弓箭手的狙擊和豬頭人的致命打擊，一般在戰鬥中是不存在讓我們落敗的魔物。
。
　或許是因為我等級提高而變強了。
　我已經得能夠流暢地進行指揮，毫無危險性的無傷完全勝利也變多了起來。
。
　羅伊斟酌著這個時機，開口說道。
。
「魯庫，差不多是時候挑戰第二階層試試看了吧」
「階層越是往下走，魔物就會越來越強對吧。如今的我沒問題嗎？」
。
「我覺得能行。雖然罕見的會出現10～20階層級別的魔物，嘛，這樣還能遇到就只能說是運氣實在太糟糕了而已」
「如果遇到第20階層的魔物什麼的，感覺我會被簡單的幹掉呢……」
。
「放心吧，由我來保護你。如果真的走投無路了，那只要什麼都不管直接逃走就好了」
「因為性命是無可代替的呢」
「就是這樣」
。
　我跟著如此笑著說道的羅伊，決定下去前往第二階層。

。
　在光球照亮了一片黑暗的狀態下，我們正走在通往第二階層的樓梯上。

。
　第二階層的構造與第一階層的構造並沒有什麼不同，我們邊走著邊警戒魔物。
。
「從第二階層開始就會有陷阱，所以要小心哦」
「好的。會有什麼樣的陷阱呢？」
。
「第二階層的陷阱也不是那麼厲害。毒啊、麻痺啊、隨機被轉移到這階層的某處，這種程度而已。只要跟著我就一定沒事吧」
。
「如果有學會陷阱探知魔法就好了呢」
「有的話的確是很方便吧。從第五階層以後，致命性的陷阱就多了。不過我有自動探知技能所以並不需要，可由於效果範圍幾乎是個人領域。所以如果你踩到陷阱，抱歉了我也很難救助你」
。
「我知道了。這也是今後的課題呢」
「對」
。
　我們邊聊著這些，邊在第二階層前進著。
　過了一會兒，羅伊突然停下腳步，迅速地往背後回過頭。
。
「怎麼了嗎」
「…………」
。
　我們是以前衛與後衛排成縱一列在地下城裡探索的，所以我還以為回過頭是有事找我，可看樣子是搞錯了。
　他是一動不動地盯著我更後面的地方注視。
。
「羅伊？」
「啊啊，沒事……。感覺好像被誰看著」
。
「在雷斯迪凱普、嗎？有其他的人，這就稍微可能性有點低了」
「雖說是這樣……莫名的感覺到一種黏在身上的視線」
。
「以防萬一還是確認一下」
。
　我把壓制住威力的火焰球投向黑暗的盡頭。
　于虛空之中飛越的火球一瞬間照亮了黑暗，接著撞到牆壁引發爆炸。
。
　在火球燃燒的通道上，連一只老鼠都不存在。
。
「誰也不在呢」
「或許是我的錯覺也說不定。抱歉了，浪費了時間」
「不會，沒關係的」
。
　我們重振精神往前方前進。
　在我們走了一會兒之後，當我要踏出腳步時就被羅伊拉住身體並大聲說道。
。
「那裡有陷阱！提高警惕！」
「誒」
。
　雖然上半身被羅伊拉住了，可踏出的腳步卻沒有因此停下來，腳踩在地面上發出『咔嚓！』的聲響。

。
「羅伊，這是什麼！？」
「是轉移魔法陣。抓住我的手，不管發生什麼都要提高警惕！」
。
羅伊如此說道，並抓著我的手往他的手腕上緊抓住。
　恐怕是為了轉移陷阱在發動的時候，不管轉移到哪裡都能一起跟著去吧。
。
　我以緊張的神色一動不動的，可不管經過多久時間，陷阱都沒有發動的樣子。
。
　相反的，能聽到從地下城遠處傳來的噢噢噢的凶猛聲音。
。
「陷阱……沒發動吧？」
「看起來好像是這樣……。或許是運氣好也說不定」
。
「果然必須要得到探知魔法了呢。這種事時常發生的話，羅伊姑且不論，我有幾條命都不夠用」
「姑且先回一趟城裡吧？」
。
「我本來也想這麼做……可既然都已經來到第二階層了，不跟魔物戰鬥也太可惜了。所以起碼要戰鬥一次再說」
「我知道了。絕對不要離開我的身邊哦」
「好的」
。
　得到能應對陷阱的能力。
　在第二階層發現的今後的課題第一點就是這個。
。
　有過陷阱沒發動那件事之後，我十分謹慎地只踩著羅伊走過的地方。
　我如同描繪出他的足跡一般行走著，不久之後羅伊大聲說道。
。
「敵襲！幾秒後要接觸了哦！」
「我知道了！我沒有關於第二階層魔物的情報，所以請給我一些情報收集和分析的時間！　就跟最初做過的那樣採取分擔戰術」
「了解」
。
　不久之後，魔物從道路的深處接二連三地出現。
　把握敵方的戰力構成。

。
　敵方前衛・樹妖(Treant)×2、骷髏奇美拉(Bone Chimera)×2、黑暗騎士(Dark Knight)×3
　敵方中衛・巫妖(Lich)×2
　敵方後衛・高級魔法師(High Mage)×2
(颯:第一層的魔物我就想吐槽不弱的樣子，第二層看名字就慫了)
。
「好多……！」

 我不禁咒罵起魔物，邊看著它們出現的樣子。
。
「直到你能冷靜的分析敵方戰力之前，敵人前衛到中衛都交給我吧」
「對不起，拜托你了！」
。
　羅伊就跟最初做過的一樣，完全控制住樹妖和黑暗騎士的敵對心，把攻擊都全部由自己一個人承受下來。
。
　簡直是神技。
　如果沒有他，我已經在雷斯迪凱普的裡面成為屍體了吧。
。
　我邊對敵人後衛的高級魔法師釋放雷射彈，邊看著羅伊的戰鬥來收集情報。
。
　樹妖是硬木的魔物，是以來回揮動樹幹那般粗的巨大手腕給與打擊的類型。
　而黑暗騎士則是身穿漆黑甲胃，揮著劍的魔物。
。
　如果是這兩種魔物，那麼就跟第一階層的妖精騎士和巨魔的威脅性沒有什麼不同。
　那麼，我判斷沒有必要去警戒。
。
　然而，一旦變成同是後衛的魔法戰，就會產生不利了。
　高級魔法師和我一樣是魔法職業。
。
　它身穿破破爛爛的赤紅色斗篷，肉體猶如不死族腐爛的身體一般。
　還會散發出惡臭，所以是不怎麼想近距離戰鬥的類型。
。
　而這樣我要跟兩只高級魔法師戰鬥，那我就會陷入劣勢。
。
　那是因為純粹的魔法能力的差距。
　高級魔法師主要是使用中級魔法。
。
　就因為它會使用火系統中級魔法的烈火長矛和水系統中級魔法的極冰長矛，所以純屬被威力差距給壓制住了。
　我如今擁有的火焰球還只是半吊子，雷射彈也只能給與到戳痛人程度的傷害。
。
　不管是正面挑起勝負，還是突然襲擊都會有不利。
。
　可也不是說以手頭上的魔法就不能戰鬥了。
　如同我平時的王道戰法那般，用雷射彈牽制住敵人的行動，一旦出現破綻就迅速用主要的魔法火焰球打上去。
。
　雖說管用是管用，可是對上兩只比我還要強大的對手來做，就很棘手了。
　如果不在切換戰法的速度和發動時機上想辦法的話，以自己的實力就會被壓制住。
。
　果然，不管再怎麼訓練作戰和戰術，加強自己的魔法能力才是當務之急吧。
　在這方面上，我想要取得牢靠的火力魔法和有效的妨礙魔法。
。
　在這場戰鬥結束之後，就回到城裡接受成長之儀吧。
　探知魔法也是如此，如果不學會新魔法，在第二階層根本就敵不過。
。
　在這意義上，挑戰第二階層就有價值了。
　只要生命不會被奪取，甚至就連失敗也是有學習的意義。
。
　不對，這場戰鬥也沒打算輸。
　如果就這樣什麼都不做直接對高級魔法師發出敗北宣言也挺受打擊的，所以盡可能的反擊回去。
。
　在高級魔法師釋放火焰球的同一時間，我以大幅度的橫向步伐從魔法的命中預測點上偏離開來。
　魔法一旦發動，要修正其軌道是極其困難的。所以，只要在命中預測點上稍微移動下身體就能夠回避掉。
。
　火焰球打中地下城那什麼都沒有的地面上，燃燒起來。
　而當躲過對方魔法之後，就會是這邊大好機會。
。
　只要魔物使用的魔法原本攻擊速度並不是那麼高，那麼在連續的釋放之間就會產生微微的空隙。
　我把那稱之為魔法發動硬直。
。
　我連喘口氣的時間都沒有，又用雷射彈擊中引發了魔法發動硬直狀態的高級魔法師。
　攻擊速度比較高的雷屬性子彈猶如吸進去一般，打中了高級魔法師。
。
「嘎啊啊啊啊！」
。
　高級魔法師1發出低沉的悲鳴，痛苦著。
　高級魔法師2對我釋放雷鳴長矛。
。
「咕……在有基本的魔法能力差距下，同時以兩只為對手太棘手了！」
。
　這一次我又急忙躲開攻擊。
　雷鳴長矛又再切開虛空，徒然地在什麼都沒有的空間之中引發雷擊。
。
「哈啊……哈啊……！」
。
　依靠不擅長的身體能力來躲避，與高自己一等的對手為敵，都讓體力被削弱。
。
　而且不僅僅只是魔物，就連人類的我也是存在細微的魔法發動後的硬直時間。
　如果說魔物是幾秒鐘，那我就是零點幾秒的領域，所以人類的發動硬直就等於是沒有一樣。
。
　即便如此，看准時機交互地使用雷射彈和火焰球之後，大腦就稍微有點混亂而處理不來。
。
　所以在魔法戰鬥上，以兩只可以互相補足發動硬直時間的魔物為對手，實在是非常不利。
。
　而用魔法擊滅對手的魔法是盡量想要避免的事情。
　因為我想要通過身體能力躲過魔法，利用敵人的硬直時間來用魔法打中它。
。
　我從急忙躲避的狀態起身之後，用火焰球對著高級魔法師1釋放出去。
　高級魔法師1漂亮地被雷射彈的麻痺效果給引發異常狀態，如今仍然還不能從行動不能狀態中恢復過來。
。
　緊接著發射的火焰球，滿意地命中了。
　高級魔法師被卷入火焰的漩渦路，不久後化為魔石消滅了。
。
「呼……！」
。
　我深深地吐了口氣。
　形勢變成1對1的話，即使有多少的魔法能力差距也能想點辦法吧。
。
　另一個高級魔法師2如同要為同胞報仇似的，一副拼命的樣子連續使用極冰長矛。
。
　往這邊飛過來的冰之矛。
　以我的火焰球是來不及迎擊的。
。
　雖然用雷射彈擊滅也可以，可我想避免術後硬直狀態。
　所以，我又是在地下城的地面上狼狽地翻滾回避著。
。
　在我移動過的地面都被冰之矛給刺成粉碎，散落而開。

 當我看穿這一空隙，立馬用雷射彈使其引發傷害硬直狀態，緊接著是決定性的火焰球。用平時這樣一種固定連擊，總算是把高級魔法師2打倒了。
。
「哈哈……！」
。
　贏了。
　總算是贏了。
。
　即便是高我一等的對手，我也還是勉強得到了勝利。
。
　可是以如今的我，果然在第二階層是不怎麼敵得過。
　我想要沒有發動硬直的技能，還有在中級魔法的對決之中擁有優勢地位的魔法啊。
。
　在這個時候突然想到羅伊情況怎麼樣了，而看著他的樣子。
　雖然從中途開始就沒有從容去觀看那邊的戰鬥，可羅伊還是老樣子以神技般的回避躲開敵人攻擊，毫髮無傷地持續屠殺著敵人前衛和中衛。
。
　魔物也只是剩下一只的狀況。
　明明我打倒兩只敵人後衛都要瘋狂到死，而羅伊對上敵人前衛中衛共9只魔物還能完勝，肯定是神還是什麼吧。
。
　當我正在調整呼吸的時候，羅伊已經砍倒最後的魔物，結束了戰鬥。
　在這個時候，感覺聽到了遠處有什麼崩潰的聲音。
。
「辛苦了」
「噢。你那邊看起來是苦戰啊」
。
　完全被看到了嗎。
　總覺得好羞恥。
。
「痛切地感覺到魔法能力的差距了呢。僅擁有速度的雷射彈也過於無力了。話說是這麼說，火焰球不管是威力還是速度都是半吊子，比敵人使用的魔法還要劣等。果然，這就是下級魔法的極限了。我不學點中級魔法的話」
。
「就跟你的一樣，在第二階層戰鬥一次也算是有意義了。那麼，照預定回去城裡吧。魔石也積累了許多，應該能接受成長之儀吧」
。
「好的。我也想這麼說」
。
　我點頭同意羅伊說的話，然後我們返回雷斯迪凱普。

。

　在歸去的中途察覺到異變的是羅伊。
。
「魯庫。這裡是我們剛才下來第二階層的時候走過的通道吧」
「是呢。怎麼了？」
「看那個吧」
。
　羅伊讓光球高升，照亮道路的前方。
　緊接著，道路被大量的沙土給埋沒殆盡。
。
「這到底是……！」
「還記得嗎，魯庫。這裡就是你踩到陷阱，可什麼都沒發生的地方」
。

。
「那麼，那個陷阱是延遲性的土石流什麼的嗎」
「或許吧，物理性的陷阱卻是延遲性的，到底有什麼意義啊」
「真是謎呢……」
。
　傷腦筋。
　到底這座土石山是為了什麼？
　是這地下城的魔物還是誰嗎，是在主張這裡不希望被通過嗎。
。
　我歪著頭思考的時候，察覺到從埋沒的石土縫隙裡有什麼像是書物的東西。

。
「魯庫，什麼啊這是」
「看樣子應該是某種資料……那個、上面寫著『諾亞方舟計劃』呢」

「你說諾亞方舟計劃？」
「這是什麼，羅伊你知道嗎」
。
　看到睜大眼睛的羅伊，我歪著頭。
。
「…………。我以前還在王國的時候，我那名叫優梅莉亞的戀人，感覺好像有說過這個詞。
　諾亞……方舟計劃……。
　對了，記得和優梅莉亞相遇的時候她說過『為了執行諾亞方舟計劃才來到維魯利亞王國』」
(颯:這裡既可以是執行計劃也可以是完成計劃)
。
「是什麼呢，諾亞方舟計劃。是某種大規模計劃……國家級的企劃團隊什麼的嗎。

。
「稍微讓我看一下」
。
　羅伊從我手中拿過羊皮紙。
。
「這不是那傢伙的字跡。只看這個標題根本就不知道是什麼意思，感覺不像是什麼特別的線索。魯庫不需要的話就扔掉了哦」
「好的。我也不需要」
。
　如此斷言之後，羅伊把羊皮紙揉成一團燒掉了。
。
　諾亞方舟計劃。
　那究竟是什麼呢。
