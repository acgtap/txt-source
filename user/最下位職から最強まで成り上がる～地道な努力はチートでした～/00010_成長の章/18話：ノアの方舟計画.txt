
　我和罗伊在雷斯迪凯普的地下城裡专心致志地打倒魔物，赚取魔石和积累战斗经验。
　基本上第一阶层的构造和魔物特性都已经把握了。
。
　如今在第一阶层对我们构成威胁的魔物有妖精弓箭手的狙击和猪头人的致命打击，一般在战斗中是不存在让我们落败的魔物。
。
　或许是因为我等级提高而变强了。
　我已经得能够流畅地进行指挥，毫无危险性的无伤完全胜利也变多了起来。
。
　罗伊斟酌着这个时机，开口说道。
。
「鲁库，差不多是时候挑战第二阶层试试看了吧」
「阶层越是往下走，魔物就会越来越强对吧。如今的我没问题吗？」
。
「我觉得能行。虽然罕见的会出现10～20阶层级别的魔物，嘛，这样还能遇到就只能说是运气实在太糟糕了而已」
「如果遇到第20阶层的魔物什么的，感觉我会被简单的干掉呢……」
。
「放心吧，由我来保护你。如果真的走投无路了，那只要什么都不管直接逃走就好了」
「因为性命是无可代替的呢」
「就是这样」
。
　我跟着如此笑着说道的罗伊，决定下去前往第二阶层。

。
　在光球照亮了一片黑暗的状态下，我们正走在通往第二阶层的楼梯上。
　走完螺旋状的楼梯，我们抵达了第二阶层。
。
　第二阶层的构造与第一阶层的构造并没有什么不同，我们边走着边警戒魔物。
。
「从第二阶层开始就会有陷阱，所以要小心哦」
「好的。会有什么样的陷阱呢？」
。
「第二阶层的陷阱也不是那么厉害。毒啊、麻痹啊、随机被转移到这阶层的某处，这种程度而已。只要跟着我就一定没事吧」
。
「如果有学会陷阱探知魔法就好了呢」
「有的话的确是很方便吧。从第五阶层以後，致命性的陷阱就多了。不过我有自动探知技能所以并不需要，可由于效果范围几乎是个人领域。所以如果你踩到陷阱，抱歉了我也很难救助你」
。
「我知道了。这也是今後的课题呢」
「对」
。
　我们边聊着这些，边在第二阶层前进着。
　过了一会儿，罗伊突然停下脚步，迅速地往背後回过头。
。
「怎么了吗」
「…………」
。
　我们是以前卫与后卫排成纵一列在地下城裡探索的，所以我还以为回过头是有事找我，可看样子是搞错了。
　他是一动不动地盯着我更後面的地方注视。
。
「罗伊？」
「啊啊，没事……。感觉好像被谁看着」
。
「在雷斯迪凯普、吗？有其他的人，这就稍微可能性有点低了」
「虽说是这样……莫名的感觉到一种黏在身上的视线」
。
「以防万一还是确认一下」
。
　我把压制住威力的火焰球投向黑暗的尽头。
　于虚空之中飞越的火球一瞬间照亮了黑暗，接着撞到墙壁引发爆炸。
。
　在火球燃烧的通道上，连一只老鼠都不存在。
。
「谁也不在呢」
「或许是我的错觉也说不定。抱歉了，浪费了时间」
「不会，没关系的」
。
　我们重振精神往前方前进。
　在我们走了一会儿之後，当我要踏出脚步时就被罗伊拉住身体并大声说道。
。
「那裡有陷阱！提高警惕！」
「诶」
。
　虽然上半身被罗伊拉住了，可踏出的脚步却没有因此停下来，脚踩在地面上发出『咔嚓！』的声响。
　魔法阵以我右脚踩着的地面为中心浮现出来。
。
「罗伊，这是什么！？」
「是转移魔法阵。抓住我的手，不管发生什么都要提高警惕！」
。
罗伊如此说道，并抓着我的手往他的手腕上紧抓住。
　恐怕是为了转移陷阱在发动的时候，不管转移到哪裡都能一起跟着去吧。
。
　我以紧张的神色一动不动的，可不管经过多久时间，陷阱都没有发动的样子。
。
　相反的，能听到从地下城远处传来的噢噢噢的凶猛声音。
。
「陷阱……没发动吧？」
「看起来好像是这样……。或许是运气好也说不定」
。
「果然必须要得到探知魔法了呢。这种事时常发生的话，罗伊姑且不论，我有几条命都不够用」
「姑且先回一趟城裡吧？」
。
「我本来也想这么做……可既然都已经来到第二阶层了，不跟魔物战斗也太可惜了。所以起码要战斗一次再说」
「我知道了。绝对不要离开我的身边哦」
「好的」
。
　得到能应对陷阱的能力。
　在第二阶层发现的今後的课题第一点就是这个。
。
　有过陷阱没发动那件事之後，我十分谨慎地只踩着罗伊走过的地方。
　我如同描绘出他的足迹一般行走着，不久之後罗伊大声说道。
。
「敌袭！几秒後要接触了哦！」
「我知道了！我没有关于第二阶层魔物的情报，所以请给我一些情报收集和分析的时间！　就跟最初做过的那样采取分担战术」
「了解」
。
　不久之後，魔物从道路的深处接二连三地出现。
　把握敌方的战力构成。

。
　敌方前卫・树妖(Treant)×2、骷髅奇美拉(Bone Chimera)×2、黑暗骑士(Dark Knight)×3
　敌方中卫・巫妖(Lich)×2
　敌方后卫・高级魔法师(High Mage)×2
(颯:第一层的魔物我就想吐槽不弱的样子，第二层看名字就怂了)
。
「好多……！」

 我不禁咒骂起魔物，边看着它们出现的样子。
。
「直到你能冷静的分析敌方战力之前，敌人前卫到中卫都交给我吧」
「对不起，拜托你了！」
。
　罗伊就跟最初做过的一样，完全控制住树妖和黑暗骑士的敌对心，把攻击都全部由自己一个人承受下来。
。
　简直是神技。
　如果没有他，我已经在雷斯迪凯普的裡面成为尸体了吧。
。
　我边对敌人后卫的高级魔法师释放雷射弹，边看着罗伊的战斗来收集情报。
。
　树妖是硬木的魔物，是以来回挥动树干那般粗的巨大手腕给与打击的类型。
　而黑暗骑士则是身穿漆黑甲胃，挥着剑的魔物。
。
　如果是这两种魔物，那么就跟第一阶层的妖精骑士和巨魔的威胁性没有什么不同。
　那么，我判断没有必要去警戒。
。
　然而，一旦变成同是后卫的魔法战，就会产生不利了。
　高级魔法师和我一样是魔法职业。
。
　它身穿破破烂烂的赤红色斗篷，肉体犹如不死族腐烂的身体一般。
　还会散发出恶臭，所以是不怎么想近距离战斗的类型。
。
　而这样我要跟两只高级魔法师战斗，那我就会陷入劣势。
。
　那是因为纯粹的魔法能力的差距。
　高级魔法师主要是使用中级魔法。
。
　就因为它会使用火系統中級魔法的烈火长矛和水系統中級魔法的极冰长矛，所以纯属被威力差距给压制住了。
　我如今拥有的火焰球还只是半吊子，雷射弹也只能给与到戳痛人程度的伤害。
。
　不管是正面挑起胜负，还是突然袭击都会有不利。
。
　可也不是说以手头上的魔法就不能战斗了。
　如同我平时的王道战法那般，用雷射弹牵制住敌人的行动，一旦出现破绽就迅速用主要的魔法火焰球打上去。
。
　虽说管用是管用，可是对上两只比我还要强大的对手来做，就很棘手了。
　如果不在切换战法的速度和发动时机上想办法的话，以自己的实力就会被压制住。
。
　果然，不管再怎么训练作战和战术，加强自己的魔法能力才是当务之急吧。
　在这方面上，我想要取得牢靠的火力魔法和有效的妨碍魔法。
。
　在这场战斗结束之後，就回到城裡接受成长之仪吧。
　探知魔法也是如此，如果不学会新魔法，在第二阶层根本就敌不过。
。
　在这意义上，挑战第二阶层就有价值了。
　只要生命不会被夺取，甚至就连失败也是有学习的意义。
。
　不对，这场战斗也没打算输。
　如果就这样什么都不做直接对高级魔法师发出败北宣言也挺受打击的，所以尽可能的反击回去。
。
　在高级魔法师释放火焰球的同一时间，我以大幅度的横向步伐从魔法的命中预测点上偏离开来。
　魔法一旦发动，要修正其轨道是极其困难的。所以，只要在命中预测点上稍微移动下身体就能够回避掉。
。
　火焰球打中地下城那什么都没有的地面上，燃烧起来。
　而当躲过对方魔法之後，就会是这边大好机会。
。
　只要魔物使用的魔法原本攻击速度并不是那么高，那么在连续的释放之间就会产生微微的空隙。
　我把那称之为魔法发动硬直。
。
　我连喘口气的时间都没有，又用雷射弹击中引发了魔法发动硬直状态的高级魔法师。
　攻击速度比较高的雷属性子弹犹如吸进去一般，打中了高级魔法师。
。
「嘎啊啊啊啊！」
。
　高级魔法师1发出低沉的悲鸣，痛苦着。
　高级魔法师2对我释放雷鸣长矛。
。
「咕……在有基本的魔法能力差距下，同时以两只为对手太棘手了！」
。
　这一次我又急忙躲开攻击。
　雷鸣长矛又再切开虚空，徒然地在什么都没有的空间之中引发雷击。
。
「哈啊……哈啊……！」
。
　依靠不擅长的身体能力来躲避，与高自己一等的对手为敌，都让体力被削弱。
。
　而且不仅仅只是魔物，就连人类的我也是存在细微的魔法发动後的硬直时间。
　如果说魔物是几秒钟，那我就是零点几秒的领域，所以人类的发动硬直就等于是没有一样。
。
　即便如此，看准时机交互地使用雷射弹和火焰球之後，大脑就稍微有点混乱而处理不来。
。
　所以在魔法战斗上，以两只可以互相补足发动硬直时间的魔物为对手，实在是非常不利。
。
　而用魔法击灭对手的魔法是尽量想要避免的事情。
　因为我想要通过身体能力躲过魔法，利用敌人的硬直时间来用魔法打中它。
。
　我从急忙躲避的状态起身之後，用火焰球对着高级魔法师1释放出去。
　高级魔法师1漂亮地被雷射弹的麻痹效果给引发异常状态，如今仍然还不能从行动不能状态中恢复过来。
。
　紧接着发射的火焰球，满意地命中了。
　高级魔法师被卷入火焰的漩涡路，不久後化为魔石消灭了。
。
「呼……！」
。
　我深深地吐了口气。
　形势变成1对1的话，即使有多少的魔法能力差距也能想点办法吧。
。
　另一个高级魔法师2如同要为同胞报仇似的，一副拼命的样子连续使用极冰长矛。
。
　往这边飞过来的冰之矛。
　以我的火焰球是来不及迎击的。
。
　虽然用雷射弹击灭也可以，可我想避免术後硬直状态。
　所以，我又是在地下城的地面上狼狈地翻滚回避着。
。
　在我移动过的地面都被冰之矛给刺成粉碎，散落而开。

 当我看穿这一空隙，立马用雷射弹使其引发伤害硬直状态，紧接着是决定性的火焰球。用平时这样一种固定连击，总算是把高级魔法师2打倒了。
。
「哈哈……！」
。
　赢了。
　总算是赢了。
。
　即便是高我一等的对手，我也还是勉强得到了胜利。
。
　可是以如今的我，果然在第二阶层是不怎么敌得过。
　我想要没有发动硬直的技能，还有在中级魔法的对决之中拥有优势地位的魔法啊。
。
　在这个时候突然想到罗伊情况怎么样了，而看着他的样子。
　虽然从中途开始就没有从容去观看那边的战斗，可罗伊还是老样子以神技般的回避躲开敌人攻击，毫发无伤地持续屠杀着敌人前卫和中卫。
。
　魔物也只是剩下一只的状况。
　明明我打倒两只敌人后卫都要疯狂到死，而罗伊对上敌人前卫中卫共9只魔物还能完胜，肯定是神还是什么吧。
。
　当我正在调整呼吸的时候，罗伊已经砍倒最後的魔物，结束了战斗。
　在这个时候，感觉听到了远处有什么崩溃的声音。
。
「辛苦了」
「噢。你那边看起来是苦战啊」
。
　完全被看到了吗。
　总觉得好羞耻。
。
「痛切地感觉到魔法能力的差距了呢。仅拥有速度的雷射弹也过于无力了。话说是这么说，火焰球不管是威力还是速度都是半吊子，比敌人使用的魔法还要劣等。果然，这就是下级魔法的极限了。我不学点中级魔法的话」
。
「就跟你的一样，在第二阶层战斗一次也算是有意义了。那么，照预定回去城裡吧。魔石也积累了许多，应该能接受成长之仪吧」
。
「好的。我也想这么说」
。
　我点头同意罗伊说的话，然後我们返回雷斯迪凯普。

。

　在归去的中途察觉到异变的是罗伊。
。
「鲁库。这裡是我们刚才下来第二阶层的时候走过的通道吧」
「是呢。怎么了？」
「看那个吧」
。
　罗伊让光球高升，照亮道路的前方。
　紧接着，道路被大量的沙土给埋没殆尽。
。
「这到底是……！」
「还记得吗，鲁库。这裡就是你踩到陷阱，可什么都没发生的地方」
。
　被这么一说，感觉是这样。
。
「那么，那个陷阱是延迟性的土石流什么的吗」
「或许吧，物理性的陷阱却是延迟性的，到底有什么意义啊」
「真是谜呢……」
。
　伤脑筋。
　到底这座土石山是为了什么？
　是这地下城的魔物还是谁吗，是在主张这裡不希望被通过吗。
。
　我歪着头思考的时候，察觉到从埋没的石土缝隙裡有什么像是书物的东西。
　我为了不再触发陷阱而慎重地接近，把那张羊皮纸从石土里拉出来。
。
「鲁库，什么啊这是」
「看样子应该是某种资料……那个、上面写着『诺亚方舟计划』呢」
。
「你说诺亚方舟计划？」
「这是什么，罗伊你知道吗」
。
　看到睁大眼睛的罗伊，我歪着头。
。
「…………。我以前还在王国的时候，我那名叫优梅莉亚的恋人，感觉好像有说过这个词。
　诺亚……方舟计划……。
　对了，记得和优梅莉亚相遇的时候她说过『为了执行诺亚方舟计划才来到维鲁利亚王国』」
(颯:这裡既可以是执行计划也可以是完成计划)
。
「是什么呢，诺亚方舟计划。是某种大规模计划……国家级的企划团队什么的吗。
　总而言之，这张羊皮纸也是破破烂烂的，除了标题以外都阅读不了。因为或许是罗伊你的恋人的线索，要不要由你来保管？」
。
「稍微让我看一下」
。
　罗伊从我手中拿过羊皮纸。
。
「这不是那家伙的字迹。只看这个标题根本就不知道是什么意思，感觉不像是什么特别的线索。鲁库不需要的话就扔掉了哦」
「好的。我也不需要」
。
　如此断言之後，罗伊把羊皮纸揉成一团烧掉了。
。
　诺亚方舟计划。
　那究竟是什么呢。
