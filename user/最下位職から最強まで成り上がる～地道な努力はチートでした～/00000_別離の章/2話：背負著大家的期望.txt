
　在那天早上。
　我和莉莉被亚露玛长老呼唤，因而前往她的家。
。

「长老。您叫我们吗？」
「对，你们来了么，鲁库、莉莉。坐在那边吧」

。

「是」
「失礼了」

。
　我们进去长老家里，被亚露玛催促着移动到铺有稻米的客厅。
。

「请问究竟有什么事，这么一早就呼唤我们……」

。
　莉莉以为是做错了什么事，用不安似的声音询问着。

「啊啊，也不是说要斥责你们。不用这么戒备」

。
　如此说完之后，我也好莉莉也好，肩膀都放松下来。
。

「可是，既然不是如此，那为什么在这个时间」
「你们两人虚岁都15了，没错吧？」

。

「是的。我还有莉莉都已经15岁了」

。
　长老听到我的回答“嗯”地点着头。
。

「根据村里重镇一群人互相谈论的结果，决定让你们两个人接受祝福之仪」
「「祝福之仪！？」」

　祝福之仪是只在王都或者大都市等等的地方，15岁以上到20岁未满的小孩才可以接受的魔法仪式。
　接受祝福之仪的小孩，将可以被神明赋予能使用各种技能魔法的天职。
。
　接不接受这个仪式会让今后的人生大为不同。
。
　但是，据说如果不是贵族或者大商人家庭出身的小孩是难以接受仪式的。
　那是因为不得不捐献一笔大金额给执行此仪式的圣教会。
。
　即便扣除这一点，在这个世界上能够使用魔法与技能也是非常有益的。
。

「到底是经过怎么样的对話才要让我们接受祝福之仪……」
「没什么。最近丰收了呢。所以金钱也就富裕了啊」

。
　不管是我还是莉莉都明白这是在说谎。
　这座村子很贫穷，并没有前途。
。
　应该没有从容到让我们接受祝福之仪才对。
　为什么即使说这样的谎也要让我们接受祝福之仪，我大脑的一偶思考着这问题。
。
　在旁边的莉莉一副冷静的表情说着。
。

「长老……让我们接受祝福之仪也实在太糟蹋了。请务必重新考虑一下」
「闭嘴吧，莉莉」

。
　这句伴随着威压感的話语，让莉莉吓得身体颤抖了一下。
。

「就当做我跟你没有血缘关系吧，我觉得村里所有人都是我的小孩。不管是你还是鲁库也都是我可爱的孙子啊」
「长老……。可是，既然是这样那更是如此。只有我们拥有幸福的回忆，那实在是」

。
　隔着地炉坐在对面的长老，轻轻地靠近过来抚摸着莉莉的脸颊。
。

「莉莉，鲁库。你们是温柔善良的孩子。那份性格肯定在不远的将来为人民带来幸福。正因为如此，前往王都。这是长老的命令。去王都接受祝福之仪」

。

「长老……」

。
　莉莉好像从亚露玛的那般言语中察觉到了什么。
　低着头，身体颤抖着，默默地点着头。
。
　长老面向我这边继续说着。
。

「鲁库，你是个慎重、认真的孩子。正因为认真善良，才会对所有在眼前有困难的人伸出援手」
「怎么可……不对，是的」

。

「但是，这有时候会是一步坏棋。真正重要的事情，真正应当守护的是什么。仔细考虑一下得出答案来吧」
「我明白了」

。
　长老对我的回答感到满足似的点着头，然后面向莉莉说道。
。

「莉莉你是聪明伶俐的女孩子。即便不用我说，也非常清楚应该做什么事情吧。和鲁库结合两个人的力量，不管遇到什么困难都不落荒而逃，而是勇于对抗。如果是你们两人，肯定能跨越难关」

。
　莉莉一副惊讶的表情抬起头，暂时的默默流下眼泪。
。

「好的……好的……绝对……」

。
　长老把我们抱在怀里，温柔地这么说。
。

「鲁库，莉莉。我一直把你们当作是自己的孩子。你们是聪明的小孩。总有一天一定会成为对他人有帮助人。听好了？前往王都接受祝福之仪，成为出色的人物吧。这就是村里大家的期望」

。
　虽然我不怎么理解长老说的事情，可莉莉无数次地点着头。
　从她眼里滴落的泪水，把地面上的干草沾湿了。

。
　长老就这样结束掉話题，我和莉莉立刻就准备出发前往王都。
　由于不能使用私人马车之类的高级物品，所以决定先步行到附近的大城镇，然后在那里坐上驿站马车前往王都。
。

「話说回来，长老他们也商量得太快了吧。今天早上才刚说完，中午前就要出发什么的……」

。
　莉莉半分期待着祝福之仪，半分不满地撅起嘴巴。
　我们立即准备好行李，出村之后为了前往有马车出现的城镇而步行于街道上。
。
　离开村子的时候，长老还有爷爷奶奶他们那非常亲切的态度。
　小孩子他们的哭脸。
　无数次反复地说着「别忘记我们的事情哦」这一句話。
。
　这简直就像是永远不会再相见的离别一般。
。

「总有种讨厌的预感」
「你说什么，什么事？」

。
　耳朵灵光的莉莉反问过来。
。

「那是我不知道的某种什么啊」
「什么啊这回答。……可是，我觉得你总会明白的哦」

。

「莉莉知道吗？」
「女人的直觉呐」

。

「那还真是不能小看啊」
「对吧？」

。
　我和莉莉互相对视笑了出来。
。

「話说回来，祝福之仪么。会获得什么天职呢？」

。
　莉莉面向前方说道。

「莉莉不就是那个？因为擅长狩猎兔子和蛇，所以是女狂战士什么的」
「看样子你是想死吧，鲁库君」

。
　虽然莉莉满脸笑容，可是反而让人觉得这个笑容好可怕。
。

「开玩笑的……」
「下次再说的話，我可是会生气的哦」

。

「对不起，我太无神经了」
「很好」

。
　莉莉好像对我的道歉感到满足，哼着歌走在前方。
　然而，在莉莉那幸福的表情中，渗透出无法完全隐藏住的悲伤。

　◇　◆

　在街道上走了一会儿，终于抵达附近比较大的城镇。
　城镇萨迪库。
　在这里就有前往王都维鲁利亚的驿站马车。
。
　驿站马车的价格是两人10枚银币。
　由于基本上可以乘坐6～7人，所以车主就可以赚到将近30枚银币。

　当然，不可能只是让人类乘坐上去。
　基本上是让像我们这样贫困的旅行者坐在商人的货物堆积成山的空隙之中。
。
　所以据说驿站马车根据这个处理方法就能够赚大钱。
　当然，初期投资就需要准备高价的马匹和马车就是了。
。

「莉莉。在驿站马车到来之前还有点时间，要去哪里吃顿饭吗？」

。
　从太阳的角度和每一刻钟的钟声来逆算，就可以计算出在驿站马车到来之前，时间还很从容。
。

「啊~，也是呢。就这么办吧」
「嗯。想吃什么？」

。

「什么都可以哟。我们金钱也没那么多对吧？」
「那也是。路摊的串烧可以吧」

。

「嗯」

。
　带上笑着点着头的莉莉，我们走近路摊。
　用鼻子闻一闻，有股美味的香气。
　用大量胡椒与酱汁烤好的香料味鸡肉串烧。
。

「呜哇~，好像很好吃的样子」
「不好意思，叔叔。请给我两串鸡肉串烧」

。

「噢，谢谢惠顾，小哥。还带着可爱的女朋友呢~！」
「不不，只是单纯的青梅竹马」
「是这样吗？」

。
　我一说完，莉莉露出一抹不满的表情。
。

「好。一串3枚铜币。可爱的小姐就杀必死算作2枚铜币吧！」
「哇，谢谢。帅气的哥哥！」

。

「哈哈哈。口齿伶俐呢，小姐。虽然我已经30岁以上了呐」
「帅气的男人不管几岁都是有魅力的哦」

。

「哟是，特别杀必死。用一枚铜币拿走吧！」
「谢谢~，最喜欢哥哥了！」

。
　这就是所谓女人的谄媚么。
。
　我以半呆然的心情看着莉莉与露摊的叔叔之间的对話。
　莉莉买到非常便宜的串烧，吃串烧吃到脸颊都鼓鼓的。
。
　由于前方稍微一点的地方有木制的长椅，所以我们决定坐在那边吃串烧。
。

「还真亏你说得出那种显而易见的客套話呢，莉莉」

。
　顺便一提，路摊的叔叔，头都秃顶了。
。

「因为啦，说了也没有人会感到困扰不是吗」
「困扰的人……或许有也说不定哦……」

。
　不由得就吞吞吐吐了。
　莉莉看到我这个样子后满脸笑嘻嘻的～～如此可以形容的微笑。
。

「呐，呐。具体的来说？具体谁会感到困扰呢？说出来看看嘛，来来。好，请说」
「…………罗罗纳村的亚修雷特什么的」

。
　亚修雷特是对莉莉求婚被拒绝的其中一个男子。
。

「啊，逃避了。鲁库真没出息」
「我又没有……」

。
　我一副哑然的表情咬住鸡肉。
　程度适当的香料味在嘴里扩散开来，非常美味。
。

「这个要3枚铜币很便宜啊」
「虽然我只要一枚铜币而已」
「是是是」

。
　差不多是这种感觉，我和莉莉就这样度过平稳的时间。

　◇　◆

　听到这份悲痛的消息是在驿站马车出发之后，过了两天左右的时候。
。
　驿站马车没有被魔物袭击过，每天都往王都的方向前进着。
　在第二天夕阳西下，向设立在街道中的修道院借宿的时候，其他旅行者的对話传到耳朵里。
。

「听说卡斯特纳地区的村子都好像几乎全灭了」
「啊啊，好像是雷斯迪凯普的大肆进攻吧？」

。

「这已经是不得不让王都圣十字骑士团出动了吧」
「可是即便反复打倒魔物，凯普还是白热化啊。到底什么时候才能把魔物从这个世上驱逐出去啊」

。
　若无其事的听闻这段对話的我，不由得逼近过去。
。

「卡斯特纳地区！？请问卡斯特纳地区的村子怎么了！？」

。
　卡斯特纳地区是我们罗罗纳村的某个地方的名字。

「什、什么啊小哥。有亲属在卡斯特纳地区吗？」
「我们出身的村子就在卡斯特纳地区！」

。
　我说完之后，旅行者他们一副尴尬的表情。
。

「啊~……那当然会，节哀顺变。传闻说那附近的村子都因雷斯迪凯普的大肆进攻而几乎全灭呢」
「怎……怎……怎么可能……」

。
　愕然到脸色突变。
　脚下摇晃不定，连正经地站着都甚至难以做到。
。
　难道说，长老他们知道有大肆进攻才让我们……？
。
　我立刻去找住在修道院女子宿舍的莉莉。
　把女子宿舍的监督管叫出来，把这件事告诉给莉莉之后。她一副毅然的表情点着头。
。

「我……隐隐约约感觉到了。叫我们去王都的长老。感觉有点奇怪」
「那么，莉莉是……知道了还去王都……？」

「嗯。对不起，没有跟鲁库说」
「为什么！？　为什么没有对我说啊！？我们留下来的話，或许能做点什么也说不定！」

。

「……大概，长老他们知道会这样才命令我们前往王都不是吗？　即便我们留在村子里也只是稍微延迟一点村子灭亡的时间吧」
「就算是这样，不是也可以跟大家一起战斗吗！」

。

「那么！战死的話，要怎么办啊！」

。
　莉莉悲痛的叫声刺痛着我的心。
。

「那不就什么都没有留下了！大家、大家都是知道的才让我们去王都对吧！？　因为，如果是十五岁的我们就可以接受祝福之仪！所以，长老才会说『成为一个出色的人物』，不是还给我们留下最后的期待吗！？　给我明白把我们送出去的大家的想法啊，鲁库！」

。
　莉莉那洁白的手掌，拍响了我的脸颊。
　响起“啪”的清脆声，比起疼痛感，我更因莉莉打我这件事而受到打击。
。
　这也就说，如今的我是多么白痴……。
。

「对、对不起……火气上涌了……」
「不会……我也是、突然打你实在对不起了呢……」

。
　我暂时无言地和莉莉站在修道院女子宿舍的入口前面。
。

「我也，仍没有完全想开。所以今天对不起了。该睡觉了」
「…………我知道了，晚安，莉莉」
「晚安」

。
　我就这样呆然的站了一会儿。
。
　罗罗纳村……毁灭了……。
　那么，我们到底要回到哪里才好呢？
。
　绝望支配了我的思考。
