
　听到罗罗纳村已经毁灭的这一消息，至今我的情绪仍然没有整理完毕。
　即便如此，太阳还是在运转。
　无论是谁，明天都会平等地来访于所有生物。
。
　我就这样心不在焉的迎接下一个早晨，乘坐在摇晃不定的马车上，往王都前进。
。
　明明今天还是莉莉遭遇到痛苦的事情的翌日，可是平时那副的开朗表情并没有消失。
　她以与平时毫无变化的表情，与往常那般毫无改变的温柔，只是说了一句話。
。

「因为大家，最喜欢鲁库了呢」

。
　仅仅只是这般的言语，却渗透着我的心灵。
　莉莉那句犹如贴近我心灵般的言语，总令我感到救赎。
。
　罗罗纳村的毁灭，让我们变为无依无靠的天涯孤独之身。
　正因为如此，只有莉莉是无论如何都要保护好。
。
　虽然我们还是没有正经地理解到社会严酷性的天真小孩，可是跟她在一起的話，一定不管到哪里都能够活下去。
　我会守护莉莉。即使有任何改变。
。
　于是我们乘坐的驿站马车还是持续着那令人不愉快的摇动，终于在六天后抵达王都。
。

「那么，两人十枚银币」
「好 」

。
　我从挂在腰上的钱包里取出银币交给好好送我们到王都的马车。
。

「谢谢惠顾。有缘再会的时候再来拜托我吧」
「非常感谢」

。
　我急忙低下头行礼，定睛注视着他离开远去。
。

「鲁库。我们也不快点走的話」
「啊啊……也对呢」

。
　莉莉即便听说村子毁灭了，也还是一副刚强的样子。
　应该没事吧。
。
　不对，不可能是这样。
　肯定是抱着一份悲伤，可还是笑着度过。
。
　真是个坚强的孩子。
　这份态度，作为一个人来说实属美丽。
。
　对啊。身为男人的我不能老是磨磨唧唧的。
　我闭上双眼，双手用力地拍打脸颊。
。
　啪，啪！
。

「怎、怎么了鲁库。这么突然」
「不能在这样磨磨唧唧地消沉下去！亚露玛他们，肯定是把村子的未来托福给我们」

。
　我一说完，莉莉宛如花朵绽放般微笑着。
。

「对。对啊，鲁库。只要我们活下去，罗罗纳村的血缘就会延续下去」
「总有一天回到那座村子，我和你再复兴罗罗纳村吧」
「嗯」

。
　莉莉露出笑容。
　如同天使的笑容一般，清澈无比的笑着。

　◇　◆

　王都果然是人很多。
　明明萨迪库城镇就已经够人眼花缭乱，像罗罗纳村这种边境之村根本就无法比拟。
。
　步行于石板路之上的人们活力充沛。
　左右两边的尖拱式建筑并排而立，路摊和售货摊的数量也非比寻常。
。
　店主招揽顾客的声音比比皆是，小孩子欢笑着跑来跑去。
。

「鲁库。要去哪里比较好来着？」
「首先……不去卖掉民艺品的話」

。
　首先决定前往商业公会，让他们买下村里的民艺品。
　由于不清楚商业公会的所在位置，所以向附近看起来很闲的人询问之后。才知道在大路上直走就到达了。
。
　在最大的道路上行走着，就有一块金币图案的看板高挂着。
。

「那就是商业公会吗」
「好像是呢」

。
　我和莉莉下定决心走进那建筑物里面。
　商业公会里面相当混乱，买卖的交涉声四处交错。
。
　近似于怒声的声音也相当多，令人觉得在这如此吵闹的公会里应该是听不到不是大声喧闹的。
。

「请问光临本公会有什么事情吗？」

。
　注意到我和莉莉在入口呆站着，像是公会职员的人物过来搭話了。
。

「啊……那个，想要把村里制作的民艺品卖出去来着」
「请往这边走」

。
　如此这般就被带往到柜台处，紧张的我和莉莉与公会接待员的姐姐面对面。
。

「虽然您希望贩卖商品，可是请问可否乐意让我看一下实物呢」
「当然」

。
　我从背着的麻袋里拿出民艺品。
　用小刀把木头削开制作而成的木偶，还有用只能在村子附近才能获取的特殊染料染成的织物。
。

「嘿~……是卡斯特纳地区的民艺品呢」
「嗯，我们正是那里出身的」

。

「那件事应该要说遗憾好呢，还是该说不幸中的万幸好呢……」
「大肆进攻那件事吗」

。
　莉莉说完之后，接待员姐姐好似寂寞般笑了。
。

「嗯。卡斯特纳地区的民艺品从以前开始就因那精致而拥有许多人气呢。也经常高价卖出去。而这一次大肆进攻是那里吧？应该说是悲剧的主人公比较好么，如今市场上卡斯特纳地区的民艺品价值上涨了現在。大概是以几倍的高价交易着」

。

「是这样啊……」

。
　莉莉的表情很黯淡。
　牺牲亲人赚到的钱，应该不可能老实地感到高兴。
。

「嗯。对于本公会来说，作为对卡斯特纳地区的各位小小的回报，这些民艺品会以高价买下。失礼了，请问有所属于其他地区的公会之类的吗？」
「没有吧」

。
　我回答了这问题。
。

「足够了。那么由于要详细鉴定一下，请你们在等候所稍等一下」
「我知道了」

。
　结果，罗罗纳村的民艺品全部以差不多3枚金币卖出去了。
　因为平常以一枚金币成交都算好了，所以即便是因为市场价格提高了，也肯定是相当让价了。
。

　在那之后，我和莉莉去旅馆。
　为了翌日接受祝福之仪而住宿一夜恢复疲劳。

在进入旅馆之后，店主以为我们是情侣而训问着我们说「同一间房间也可以哦？」，可是莉莉迅速拒绝了。
　有点受伤了。
。
　旅馆的级别可以说是马马虎虎，简陋的床上铺上一层稻草外加床单。
　然后是桌子还配些椅子，早晚的伙食是有偿的。
　利用从魔物身上获得的水魔石的澡堂，使用费用是一次一枚银币。
。
　好贵。
。
　于是我只好让店主帮我打桶水，用棉锦布浸湿来擦拭身体而已。
　話说回来，莉莉就好像理所当然般使用着澡堂，可我也没蠢到对这件事说三道四。
。
　把装备脱下，保养好用惯的铁剑之后，我就倒在房间的床上。
　就这样仰望着房间的天花板一会儿，沉浸于思虑之中。
。
　一做完，由于不久后莉莉就来到房间里说「反正难得，去王都散散步吧」，于是我决定跟着一起去。
。
　傍晚时分的王都散步。
。

「哎呀，王都真是活力满满呢」
「因为王都都没活力的話，这个国家就完蛋了呢」
「那也是」

。
　对于我的俏皮話，莉莉毫无顾虑地笑着。
　两人暂时地说着笑，逛逛路摊、饰品店。说着「那个不错」「这个好可爱」之类的无意义对話。
。
　虽然是打算陪着莉莉买东西的，可是过了一会儿就想到，莉莉是在担心我。
　因为一个人在房间里的話，无论如何都会想起罗罗纳村的事情。
　所以才会像如今带我出来吧。
。

「莉莉……谢谢了」
「嗯？什么事？」
「为了我不会一个人感到寂寞，才会来邀请我去买东西的吧」

。
　如此说完，莉莉好像困扰般笑着。
。

「也不是，其实真的是想试着在王都游乐一番的啦。虽然没有钱什么都买不了就是了」
「一点点的話也可以的吧。毕竟我也有带钱出来」

。

「不行不行。我们的金钱可没有滥用的从容」
「为了接受祝福之仪，金钱是必要的呢」

。

「也是呢。如果能获得不错的天职和技能就好了呢」
「莉莉肯定能获得非常棒的天职哦」

。

「鲁库也是哦。…………呐，鲁库」

。
　莉莉抬头看着太阳在水平线彼方沉浸的那块紫色天空，嘀咕地说道。
。

「我们，会一直在一起的吧？」

。
　莉莉以渗透着哀愁般的声音说道。
。

「这是当然。接受祝福之仪，获得天职。两个人当上冒险者活下去吧。然后赚到一大笔钱，再来复兴罗罗纳村」

。

「约定了呐。绝对要遵守约定哦」
「嗯」

。
　我点头之后，莉莉就绽开微笑。
。

「别让我，独自一个人呐……」

。
　那仿佛，如同天使的愿望一般。

　◇　◆

　翌日清晨。
　我和莉莉前往圣教会，决定要接受祝福之仪。
。
　大圣堂在王都之中也是格外显眼，所以连迷路都没有就到达目的地了。
。
　村里大家的期望，就在装有金币的袋子里。穿过大圣堂的庄严门扉。
　太阳光从被制作为天花板的采光窗照射而入，穿过彩色玻璃的五颜六色的光芒倾注而下。
。
　管风琴的声音与信徒们献上祷告的身影。
。
　静謐的气氛。
　甚至连发出低声细语都令人犹豫不定。
。

「鲁库……进去里面也可以吗」
「嗯~，可是，也只能去了吧」

。
　我对着莉莉说的話点着头，笔直地往大圣堂的里面直走。
　能感觉到左右两边坐席上，献上祷告的信徒他们的视线。
。
　在中央深处的祭坛上，神官正向神低声祈祷着。
。
　安静的接近。
　虽然神官精神集中到声音都可以忍受。可不久后，睁开眼睛回过头声。
。

「请问来圣教会有何事呢」
「我们想要接受祝福之仪」

。
　虽然突然回过头来让我吓了一跳，可我还是回答了。
。

「哦？。期望神的祝福么。可是从我眼前所见来看，你们好像并没有那么的富裕？」
「背负村里大家的期待才终于来到这里」

。
　打开金币袋，神官只是深深地点头。
。

「原来如此。也就是被周围人关爱的孩子吗。名字是？」

。

「罗罗纳村的鲁库」
「我是莉莉」

。

「是吗……卡斯特纳地区的……」

　仅仅如此，神官好像察觉到了什么。
。

「很好，我们也仅仅是慈善事业就已经运转不佳，报酬就收下了。可是，会尽可能的如你所愿那般妥善处理的」
「非常感谢」

。

「准备好祝福之仪吧。往这边过来」

。
　神官把信徒交由给修道女，带着我和莉莉走过礼拜堂里面的门扉。
　穿过太阳光照射而入的圆柱式走廊，来到神官和女官他们日常使用的房间。
。

「女性的莉莉殿下就往这边。鲁库殿下是这间房间」

。
　我和莉莉离别在各自的房间，用作为祈祷而制作的混杂着盐的水――也就是所谓的圣水清洁身体，换上全白的修道服。
　跟着神官复述献给神明的祈祷話语，当做好贡献身体的准备之后，我和莉莉一起来到教会尤其深处的房间。

那是间神秘的房间。
　大圣堂从那性质之中，所到之处都会装设采取太阳光的采光窗。
。
　然而在这最深处的房间里，完全没有任何的采光窗。在这完全封闭的房间里，就只有蜡台的火焰在燃烧罢了。
　圣教会崇拜的神、沃鲁班神的神像被摇曳的灯火照耀着。
。
　那是纯碎的幻想般的光景。
。

「祝福之仪就在这边举行。心理准备好了吗」

。

「是的」
「……没问题」

。
　会获得怎么样的天职呢。
　会是能回应村里大家期待的职业吗。
。
　应该很不安吧。
　由于莉莉的声音都在颤抖，所以我悄悄地握住她的手。
。
　然后，莉莉露出好像放心似的笑容，微微地吐舌头。
。
　谢谢。
　应该是在如此说道吧。
。

「那么现在即将开始赋予神的祝福。汝，鲁库哟。汝能否发誓其一生奉献于神明所有呢」
「我发誓」

。
　莉莉也同样叙说一样的誓言，接着神官把某种像是水晶般的东西交给我们手中。
。

「在祈祷的过程中，请一直抱住这个。只要这个直到祈祷最后都没有损坏的話，就代表神已经接纳你。可如果在中途就破碎了，那就遗憾了」

。

「好的。我明白了」
「我知道了」

。
　在那之后的一小时都是在复述祈祷文，一心一意地进行对神献上祈祷的作业。
　听说教会的仪式很无聊，看来还真是这样子。
。
　在途中，有几次想要打哈欠，可还是忍住了。
　过了大约一小时的时候，神官的祈祷文终于结束了。
。

「那么，从现在开始，神的祝福就赋予给你们吧。还真亏你们水晶没有损坏，努力到这地步呢。把那个举起来吧」
「是的」

。
　把水晶举到胸前。
　沉甸甸的透明色结晶正充满着光辉。
。

「此时此刻起，你们将成为神之使徒日。请成为一名清廉、清正、为了人民的人吧。恭喜，赋予天赋与技能于你们吧」

。
　神官如此说完，充满光辉的水晶即刻破裂。祭坛也满载神圣的光芒。
　总觉得身体里有种不可思议的力量涌起的感觉。
。
　祝福之仪的结束，我和莉莉都被赋予了天职与技能。
