
　-DIVE　TO　GAME-

。
　存在着大帝国的世界。
　那是一部分人类觉醒新能力的世界。
　那即是技能，亦或是特殊能力。
。
　存在着大帝国的世界。
　那是将一切都以数值化来表现的世界。
。
　存在着大帝国的世界。
　那是并不是“此处”的世界。
。
　――错误的未来。

。
　向全世界發动侵略的“帝国”以极快的进击速度将世界约六成的版快纳入手中。这是由於他们的技术优秀到无人能及。
　再来也是因为国内诞生出许多觉醒新力量的“能力者”。

。
　出类拔萃的科学力量，以及能力者――

。
　这两方面使得帝国疯狂，使得帝国行动而起。
　结果在世界人类歷史上最残忍、最邪恶、且最灿烂辉煌，名为“大帝国”的国家就此诞生。
。
　然而，大帝国的进击在得到世界六成的时点上便停止了。
　因为他们的版图实在是过於辽阔。
　当然，成为属国的众多国家不可能会坐以待毙，世界卷起一阵名为恐怖袭击以及武装政变的龙卷风。
。
　而且围绕着诸多的特权，内部频繁發生对立与决裂。如同着陆在陆地上的鲸鱼，大帝国因自身的庞大而无法自由行动。
　往底处看，就是一个每天频繁發生恐怖袭击的地狱世界。往高处看，那就是一群重複着加入与离合的愚蠢高官。

。
　就在这种时候。
　一个男人向大帝国的上层部提出“一个提案”。

。
　那是愉快无比的遊戏。
　那是无比幸福的遊戏。
。
　既能给身为大帝国人名的“神民”笑容，甚至还可给予属国的“国民”救济。

。
　无论何事都喜欢华丽进行的大帝国，不再实行“樸素”的镇压与虐杀――

　而是藉由这“遊戏”向反逆的属国降下铁锤，并且是以彻底肃清内部的方针进行。没错，送往这遊戏裡的参加者并不只是国民，就连犯下失败的中枢部人士，甚至垮台的高官都包含在内的提案。
。

　直至昨天的胜者，一旦失败便会送往断头台（遊戏）。

　无论地位高低，这都是毫无任何情面可言的极彩色地狱绘图。
　而计划出这提案并提出的男人，他的名字就叫做九内伯斗。

。
　于後来――被称之为“大帝国魔王”的男人。

。

　■□■□

。

　――圣光国　国境附近

。
　（我睡着了吗……总觉得好像梦到了怀念的往事啊……）

。

　在马车舒适的摇晃中熟睡的魔王醒了过来。
　昨晚一夜未眠应该就是原因吧。何况这辆大型马车的休息设计就是躺着的形式。
　这辆马车可说是兼备冒险者特有的帐篷。魔王挺起上半身，注意到膝盖上有点重这件事。

。

「喂，为什麼这傢伙会睡在我的膝盖上啊……」

。

「就算你问我，我也不知道啦」

。
　雪风在魔王的膝盖上熟睡着，發出安静的熟睡声。
　她的睡脸有如梦幻般的雪，犹如绘画中出现的美少女。在魔王的感觉中，她散發出远比TV里出现的偶像更加楚楚可怜的氛围。
　一旦知道雪风的性别，即便是魔王也会脸色巨变吧。

。

（不行，脑袋还是晕晕的啊……）

。
　或许是睡眠不足从中作祟吧，魔王一副困倦的表情向马车外面望去。
　外面是景色糟糕的乾枯大地与乾燥的空气。他起床时经常会看着数字清醒大脑，不过不巧的是手表放在村子裡了。因为考虑到前往迷宫，他担心会弄坏。
。

「你是叫做蜜康吧，现在是幾年几月啊……？」

。

「……哈？」

。

「好了啦快回答我」

。

「什麼嘛……现在是“圣历”2000年七月十日啊！？」

。

「……2000年吗，哈哈。」

。
　魔王像是对什麼感到怀念而發出乾笑声。有什麼可笑的，有什麼觉得怀念的，在蜜康看来是完全摸不着头脑。
　原本她的表情就已经很不开心，现在更是愈發险恶。

。

（真是不可思议啊。连年月、时间、语言都是相同……）

。
　平假名，片假名，汉字，以及英语。
　就连运用这幾点的方式也都跟现代日本极为相似。就好像原地不封地複制出来一样。对於魔王而言，那是极为方便的事情。
　然而，浮现於魔王脑海中的却是截然不同的感想。
。

（怎麼说呢……真是“粗糙的工作”啊……）

。
　无论这种思考方向是否正确，又或是错误。在这个男人看来就是最终的“答案”。因为他想说的是，这只不过是複制现代日本的情报，再以此黏贴的简单工作而已。

。

「记得好像是说过现在是“战争时期”吧？」

。

「………」

。

「你没听见吗？冷冻橘子」

。

「谁是冷冻啊！你啊，从刚才开始也问得太多了啦！」

。
　蜜康摇晃着赤红色的头髮，生气地回復道。连眼瞳都是红色的她一旦大叫，配合那纤细的体型，有种令人联想到杜伯曼犬的野生性质的氛围。
　而且肌肤也是褐色，那件裸露出大片肩膀的轻便衣服，以及短裙中得以窥见的大腿都会让人感觉到健康性的色气。

。

「老是问我也烦死了，所以我一次性回答你啦！北方诸国从四月到九月是战争时期！十月到三月是休战时期！你明白了吗！？」

。

「抱歉，你说得太快我都不知道你在说什麼。你再说一遍」

。
「呜姆姆……真是麻烦……！北方诸国～～」

。

「抱歉，因为雪风打呼噜，我没有听见。你再说一遍」

。

「她根本是睡得安安静静的吧！」

。

　蜜康倒竖着赤红头髮，魔王大笑着。
　因为这男人喜欢捉弄像蜜康这样的女人。当然，这对於蜜康而言只不过是麻烦事而已吧。

。

「归根究底，你啊真的是“人类”吗？以那种方式打倒伽尼巴鲁……根本无法想象是人类哦？」

。

「别说这些失礼的話，我可是人类」

。

「我先说好，所谓的人类呢――呜哇哇！」

。
　蜜康兴奋地站起来，然而马车或许是碾过小石头之类的而轻微摇晃起来。
　平衡失调的蜜康倒了下来，她身上穿着的红色内裤也映入魔王的眼裡。在短暂的时间裡――马车内一直持续着沉默。

。

「…………了吧」

。

「嗯？」

。

「你、刚刚看到我的……内裤了吧……」

。

「我眼瞳裡映照出来的――从古至今只有蓝色的天空而已」

。
　魔王以自信满满的态度说出意义不明的事情。
　而听到这番話的蜜康，身体开始抖动起来。

。

「别给我说这些意义不明的狡辩！你不是一瞬间变成“战士的眼神”吗！」

。

「嘛，“晚霞”也包含在天空之中」

。

「你、你这不是连颜色都清楚地看见了吗！」

。

「我可是在说关於天空的事情……倒是你从刚才开始究竟在说些什麼呢？」

。

　魔王呀嘞呀嘞地摇着头。
　这男人擅长的是即便受到别人的谴责，也会反之其道。刚才的“红”保存於脑中文件夹的同时，表面上依然是若无其事的表情。
　正所谓是――难以饶恕的邪恶存在。
。

「为什麼我的周围总是有这种人啦ーーー！」

。
　蜜康抱着头大叫，而雪风也因此醒了过来。
　她那睡醒的身姿可谓是富有雪之妖精的风情。

。

「……叔叔大人的膝盖，非常硬邦――啊姆」

。

「在说些莫名其妙的事情之前，先给我舔舔这个」

。
　魔王也学到教训了吧，在雪风开口之前便放进糖果。
　虽然在遊戏只是回復5点气力的辣鸡道具，不过由於数量能够以10为单位进行捡取，因此在紧急时刻之外也派的上用场。
　根據情况，这也会提供草莓味和可乐味等无谓的口味。
5
。

「这就是大人所谓的智慧」

。
　魔王自豪地笑着，把另一个糖果向蜜康丢过去。
　这是橘子味，虽然魔王并没有特意选择。

。

「……你拿出来的东西什麼的，根本无法信任」

。

「这是女性喜欢的甜品啊」

。

「这是在用砂糖诱惑我？不过我可不是那麼廉价的女人」

。
　蜜康开始在包裡找东西，从裡面取出一个魔法道具。
　那是一个双手各拿着红与白旗帜的人偶。魔王望向那尊人偶，以哀叹的声音说道。

。

「你……还没有从人偶过家家毕业吗？」

。

「笨蛋……这个呢，可是《总之使用君》哦！连这都不知道吗，你的一般常识是怎麼学的啊」

。

「虽然我觉得这个名字才是非常识……」

。

「哈啊啊？明明就简单易懂不是吗」

。
　蜜康把手掌上的糖果伸向人偶。
　不久後，人偶举起白旗發出一声严肃的「OK」。

　见状，魔王从座位上倒了下来。

。

「什麼啊，那个非常识的玩意！」

。

「哼，非常识的玩意是你吧。这个呢，是可以告诉我食物和饮料裡是否有危险物质的贵重品」

。
　实际上这个魔法道具非常有名。
　就跟圣职者使用的《天使的汤勺》具有同样的效果。对於那些队伍裡没有圣职者的冒险者而言，是梦寐以求的物品。

「这也是在迷宫什麼的得到的东西吗？」

。

「不是，这是在都市国家买到的哦。虽然价格很贵，不过有值得我买下的价值……好～～～甜！」

。
　解开包装纸，把糖果放进口中的蜜康不禁大叫道。
　若说到这世界上的甜味，果然最具代表性的就是砂糖吧。倘若还要列举其他的話就是水果，不过像糖果这种甜点根本不可能存在。
　初次体验到的甜味令蜜康不禁绽放微笑。

。

「这是橘子的味道，是我的味道……」

。

「……我的味道，蜜康是变态」

。

「你给我闭嘴！」

　在两人吵嚷着的时候，魔王别开眼光想着其他的事情。
　那就是关於名为都市国家的地区。由於现代地球并没有这样的国家，因此他完全无法想象是怎麼样吧。

。
「关於这个都市国家，能否详细地说给我听」
。
「我才不要再对你说明！」
。
「……叔叔大人，我来告诉你。告诉你之後我还想舔舔」

。
　蜜康身体转向一边，而雪风则是说些莫名其妙的事情，然而魔王对此并没有吐槽，而是继续询问。
　无知的时刻早已经结束了――他应该是这麼想的吧。
。
　即便从雪风听来都市国家的概要，魔王还是无法理解。每一个都市就是国家，而这些都市形成一个集体，就形成一个庞大的国家。
。
　每一座都市拥有自治权，尽管法律不尽相同，然而都是以同一个国家的名义自称。
　当然，这只是从北方诸国的战争中守护自身而诞生的一种自卫・同盟性质的国家，不过在魔王看来就是不存在于身边的国家、体系。

。

（要比喻的話，就是东京国、埼玉国、神奈川国之类的结合起来，再自称是“关东王国”这麼一回事吗？一个县无法应付的问题，由多个县来处理，结局就不一样了吧）

。
　魔王脑海裡浮现出的当然是日本地图。
　他把这些信息都作为暂定的概要，放到脑海的角落裡。这终究只是暂定。
　他基本上是相信自己亲眼所见的、自己亲耳所闻的、自己亲自嗅到的事物，再以此作为判断的关键点。
　乍看之下，这些会令人联想到他是沉着冷静的人物，但却有点不同。
。
　从中得以看出的事实是，他是一个时常以“自我中心”为根據的人物。
　无论何事都由自己下判断，重视自己的判断。
　根據场合，这种类型有时候会轻易地陷入自以为是的状态。
。
　然而不知是幸运亦或是不幸，这个男人是――魔王。
　作为立於顶点的“独裁者”来说，这是难得的一种资质。时常贯彻自己的判断，贯彻自己的意志。
　歷史上，这种人物一旦掌握权利，将会产生善恶两面这种相差甚远的结果与损害。
。

「迟早有一天去一趟都市国家看看吧」

。

「……叔叔大人，到时候希望你也可以带上我」

。

「我可不要去！绝对不去！」

。
　喧哗的马车之旅仍在进行，不久後穿过亚侯之城，驶过国境城塞。
　原本在这种时期要渡过国境，若不是情节重要的事情根本就无法做到。然而魔王提交出从玛妲穆而来的书信，却轻易地做到这种事情。
　正所谓是――已掌握的“人脉的力量”。
