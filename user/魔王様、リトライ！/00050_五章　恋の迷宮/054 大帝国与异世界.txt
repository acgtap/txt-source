
　-DIVE　TO　GAME-

。
　存在著大帝國的世界。
　那是一部分人類覺醒新能力的世界。
　那即是技能，亦或是特殊能力。
。
　存在著大帝國的世界。
　那是將一切都以數值化來表現的世界。
。
　存在著大帝國的世界。
　那是並不是“此處”的世界。
。
　――錯誤的未來。

。
　向全世界發動侵略的“帝國”以極快的進擊速度將世界約六成的版快納入手中。這是由於他們的技術優秀到無人能及。
　再來也是因為國內誕生出許多覺醒新力量的“能力者”。

。
　出類拔萃的科學力量，以及能力者――

。
　這兩方面使得帝國瘋狂，使得帝國行動而起。
　結果在世界人類歷史上最殘忍、最邪惡、且最燦爛輝煌，名為“大帝國”的國家就此誕生。
。
　然而，大帝國的進擊在得到世界六成的時點上便停止了。
　因為他們的版圖實在是過於遼闊。
　當然，成為屬國的眾多國家不可能會坐以待斃，世界卷起一陣名為恐怖襲擊以及武裝政變的龍卷風。
。
　而且圍繞著諸多的特權，內部頻繁發生對立與決裂。如同著陸在陸地上的鯨魚，大帝國因自身的龐大而無法自由行動。
　往底處看，就是一個每天頻繁發生恐怖襲擊的地獄世界。往高處看，那就是一群重複著加入與離合的愚蠢高官。

。
　就在這種時候。
　一個男人向大帝國的上層部提出“一個提案”。

。
　那是愉快無比的遊戲。
　那是無比幸福的遊戲。
。
　既能給身為大帝國人名的“神民”笑容，甚至還可給予屬國的“國民”救濟。

。
　無論何事都喜歡華麗進行的大帝國，不再實行“樸素”的鎮壓與虐殺――

　而是藉由這“遊戲”向反逆的屬國降下鐵錘，並且是以徹底肅清內部的方針進行。沒錯，送往這遊戲裡的參加者並不只是國民，就連犯下失敗的中樞部人士，甚至垮台的高官都包含在內的提案。
。

　直至昨天的勝者，一旦失敗便會送往斷頭台（遊戲）。

　無論地位高低，這都是毫無任何情面可言的極彩色地獄繪圖。
　而計劃出這提案並提出的男人，他的名字就叫做九內伯斗。

。
　于後來――被稱之為“大帝國魔王”的男人。

。

　■□■□

。

　――聖光國　國境附近

。
　（我睡著了嗎……總覺得好像夢到了懷念的往事啊……）

。

　在馬車舒適的搖晃中熟睡的魔王醒了過來。
　昨晚一夜未眠應該就是原因吧。何況這輛大型馬車的休息設計就是躺著的形式。
　這輛馬車可說是兼備冒險者特有的帳篷。魔王挺起上半身，注意到膝蓋上有點重這件事。

。

「喂，為什麼這傢伙會睡在我的膝蓋上啊……」

。

「就算你問我，我也不知道啦」

。
　雪風在魔王的膝蓋上熟睡著，發出安靜的熟睡聲。
　她的睡臉有如夢幻般的雪，猶如繪畫中出現的美少女。在魔王的感覺中，她散發出遠比TV里出現的偶像更加楚楚可憐的氛圍。
　一旦知道雪風的性別，即便是魔王也會臉色巨變吧。

。

（不行，腦袋還是暈暈的啊……）

。
　或許是睡眠不足從中作祟吧，魔王一副困倦的表情向馬車外面望去。
　外面是景色糟糕的乾枯大地與乾燥的空氣。他起床時經常會看著數字清醒大腦，不過不巧的是手表放在村子裡了。因為考慮到前往迷宮，他擔心會弄壞。
。

「你是叫做蜜康吧，現在是幾年幾月啊……？」

。

「……哈？」

。

「好了啦快回答我」

。

「什麼嘛……現在是“聖歷”2000年七月十日啊！？」

。

「……2000年嗎，哈哈。」

。
　魔王像是對什麼感到懷念而發出乾笑聲。有什麼可笑的，有什麼覺得懷念的，在蜜康看來是完全摸不著頭腦。
　原本她的表情就已經很不開心，現在更是愈發險惡。

。

（真是不可思議啊。連年月、時間、語言都是相同……）

。
　平假名，片假名，漢字，以及英語。
　就連運用這幾點的方式也都跟現代日本極為相似。就好像原地不封地複製出來一樣。對於魔王而言，那是極為方便的事情。
　然而，浮現於魔王腦海中的卻是截然不同的感想。
。

（怎麼說呢……真是“粗糙的工作”啊……）

。
　無論這種思考方向是否正確，又或是錯誤。在這個男人看來就是最終的“答案”。因為他想說的是，這只不過是複製現代日本的情報，再以此黏貼的簡單工作而已。

。

「記得好像是說過現在是“戰爭時期”吧？」

。

「………」

。

「你沒聽見嗎？冷凍橘子」

。

「誰是冷凍啊！你啊，從剛才開始也問得太多了啦！」

。
　蜜康搖晃著赤紅色的頭髮，生氣地回復道。連眼瞳都是紅色的她一旦大叫，配合那纖細的體型，有種令人聯想到杜伯曼犬的野生性質的氛圍。
　而且肌膚也是褐色，那件裸露出大片肩膀的輕便衣服，以及短裙中得以窺見的大腿都會讓人感覺到健康性的色氣。

。

「老是問我也煩死了，所以我一次性回答你啦！北方諸國從四月到九月是戰爭時期！十月到三月是休戰時期！你明白了嗎！？」

。

「抱歉，你說得太快我都不知道你在說什麼。你再說一遍」

。
「嗚姆姆……真是麻煩……！北方諸國～～」

。

「抱歉，因為雪風打呼嚕，我沒有聽見。你再說一遍」

。

「她根本是睡得安安靜靜的吧！」

。

　蜜康倒豎著赤紅頭髮，魔王大笑著。
　因為這男人喜歡捉弄像蜜康這樣的女人。當然，這對於蜜康而言只不過是麻煩事而已吧。

。

「歸根究底，你啊真的是“人類”嗎？以那種方式打倒伽尼巴魯……根本無法想像是人類哦？」

。

「別說這些失禮的話，我可是人類」

。

「我先說好，所謂的人類呢――嗚哇哇！」

。
　蜜康興奮地站起來，然而馬車或許是碾過小石頭之類的而輕微搖晃起來。
　平衡失調的蜜康倒了下來，她身上穿著的紅色內褲也映入魔王的眼裡。在短暫的時間裡――馬車內一直持續著沉默。

。

「…………了吧」

。

「嗯？」

。

「你、剛剛看到我的……內褲了吧……」

。

「我眼瞳裡映照出來的――從古至今只有藍色的天空而已」

。
　魔王以自信滿滿的態度說出意義不明的事情。
　而聽到這番話的蜜康，身體開始抖動起來。

。

「別給我說這些意義不明的狡辯！你不是一瞬間變成“戰士的眼神”嗎！」

。

「嘛，“晚霞”也包含在天空之中」

。

「你、你這不是連顏色都清楚地看見了嗎！」

。

「我可是在說關於天空的事情……倒是你從剛才開始究竟在說些什麼呢？」

。

　魔王呀嘞呀嘞地搖著頭。
　這男人擅長的是即便受到別人的譴責，也會反之其道。剛才的“紅”保存於腦中文件夾的同時，表面上依然是若無其事的表情。
　正所謂是――難以饒恕的邪惡存在。
。

「為什麼我的周圍總是有這種人啦ーーー！」

。
　蜜康抱著頭大叫，而雪風也因此醒了過來。
　她那睡醒的身姿可謂是富有雪之妖精的風情。

。

「……叔叔大人的膝蓋，非常硬邦――啊姆」

。

「在說些莫名其妙的事情之前，先給我舔舔這個」

。
　魔王也學到教訓了吧，在雪風開口之前便放進糖果。
　雖然在遊戲只是回復5點氣力的辣雞道具，不過由於數量能夠以10為單位進行撿取，因此在緊急時刻之外也派的上用場。
　根據情況，這也會提供草莓味和可樂味等無謂的口味。
5
。

「這就是大人所謂的智慧」

。
　魔王自豪地笑著，把另一個糖果向蜜康丟過去。
　這是橘子味，雖然魔王並沒有特意選擇。

。

「……你拿出來的東西什麼的，根本無法信任」

。

「這是女性喜歡的甜品啊」

。

「這是在用砂糖誘惑我？不過我可不是那麼廉價的女人」

。
　蜜康開始在包裡找東西，從裡面取出一個魔法道具。
　那是一個雙手各拿著紅與白旗幟的人偶。魔王望向那尊人偶，以哀嘆的聲音說道。

。

「你……還沒有從人偶過家家畢業嗎？」

。

「笨蛋……這個呢，可是《總之使用君》哦！連這都不知道嗎，你的一般常識是怎麼學的啊」

。

「雖然我覺得這個名字才是非常識……」

。

「哈啊啊？明明就簡單易懂不是嗎」

。
　蜜康把手掌上的糖果伸向人偶。
　不久後，人偶舉起白旗發出一聲嚴肅的「OK」。

　見狀，魔王從座位上倒了下來。

。

「什麼啊，那個非常識的玩意！」

。

「哼，非常識的玩意是你吧。這個呢，是可以告訴我食物和飲料裡是否有危險物質的貴重品」

。
　實際上這個魔法道具非常有名。
　就跟聖職者使用的《天使的湯勺》具有同樣的效果。對於那些隊伍裡沒有聖職者的冒險者而言，是夢寐以求的物品。

「這也是在迷宮什麼的得到的東西嗎？」

。

「不是，這是在都市國家買到的哦。雖然價格很貴，不過有值得我買下的價值……好～～～甜！」

。
　解開包裝紙，把糖果放進口中的蜜康不禁大叫道。
　若說到這世界上的甜味，果然最具代表性的就是砂糖吧。倘若還要列舉其他的話就是水果，不過像糖果這種甜點根本不可能存在。
　初次體驗到的甜味令蜜康不禁綻放微笑。

。

「這是橘子的味道，是我的味道……」

。

「……我的味道，蜜康是變態」

。

「你給我閉嘴！」

　在兩人吵嚷著的時候，魔王別開眼光想著其他的事情。
　那就是關於名為都市國家的地區。由於現代地球並沒有這樣的國家，因此他完全無法想像是怎麼樣吧。

。
「關於這個都市國家，能否詳細地說給我聽」
。
「我才不要再對你說明！」
。
「……叔叔大人，我來告訴你。告訴你之後我還想舔舔」

。
　蜜康身體轉向一邊，而雪風則是說些莫名其妙的事情，然而魔王對此並沒有吐槽，而是繼續詢問。
　無知的時刻早已經結束了――他應該是這麼想的吧。
。
　即便從雪風聽來都市國家的概要，魔王還是無法理解。每一個都市就是國家，而這些都市形成一個集體，就形成一個龐大的國家。
。
　每一座都市擁有自治權，盡管法律不盡相同，然而都是以同一個國家的名義自稱。
　當然，這只是從北方諸國的戰爭中守護自身而誕生的一種自衛・同盟性質的國家，不過在魔王看來就是不存在于身邊的國家、體系。

。

（要比喻的話，就是東京國、埼玉國、神奈川國之類的結合起來，再自稱是“關東王國”這麼一回事嗎？一個縣無法應付的問題，由多個縣來處理，結局就不一樣了吧）

。
　魔王腦海裡浮現出的當然是日本地圖。
　他把這些信息都作為暫定的概要，放到腦海的角落裡。這終究只是暫定。
　他基本上是相信自己親眼所見的、自己親耳所聞的、自己親自嗅到的事物，再以此作為判斷的關鍵點。
　乍看之下，這些會令人聯想到他是沉著冷靜的人物，但卻有點不同。
。
　從中得以看出的事實是，他是一個時常以“自我中心”為根據的人物。
　無論何事都由自己下判斷，重視自己的判斷。
　根據場合，這種類型有時候會輕易地陷入自以為是的狀態。
。
　然而不知是幸運亦或是不幸，這個男人是――魔王。
　作為立於頂點的“獨裁者”來說，這是難得的一種資質。時常貫徹自己的判斷，貫徹自己的意志。
　歷史上，這種人物一旦掌握權利，將會產生善惡兩面這種相差甚遠的結果與損害。
。

「遲早有一天去一趟都市國家看看吧」

。

「……叔叔大人，到時候希望你也可以帶上我」

。

「我可不要去！絕對不去！」

。
　喧嘩的馬車之旅仍在進行，不久後穿過亞侯之城，駛過國境城塞。
　原本在這種時期要渡過國境，若不是情節重要的事情根本就無法做到。然而魔王提交出從瑪妲穆而來的書信，卻輕易地做到這種事情。
　正所謂是――已掌握的“人脈的力量”。
