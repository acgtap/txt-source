
　长发男子正背着年幼的小孩子行走着。
　如果只听到这番话，应该会觉得这是一对父子，不过男人的外貌实属异常，拥有这世界上较为罕见的黑髮，而且其长度宛若女性一般。
　何况全身上下都是一片黑的衣物，越看越像是死神或者恶魔。

。
「你好像说的是愿望祠堂吧？是那种地方吗，投点香钱然後祭拜的？」
。
「我也不知道详细情形，不过我听说智天使大人就是藉由这间祠堂的力量封印住恶魔王」
。

　听到这句话，魔王小声地嘀咕道「这是啥神鬼片啊」。
　他还以为是神社或者其他设施的同类，不过似乎是千错万错。
　即便原本在话题中就已经说过智天使、恶魔什么的，不过应该还是不会想到这些吧。
。
　其内在可谓是大野晶的男人对于宗教毫无兴趣，近似于无神论者。
　若换作是九内伯斗，如果被询问神究竟乃为何物――他应该会当机立断地回答是“自己”吧。
。
　无论怎么想这种人都不适合在幻想世界中生存，如果有异端审问团，肯定会是先被执以火刑的目标。
。

「我总觉得有种去了也没意义的感觉……」

「没、没有这回事！传说中听说会实现来访者的愿望」
。

　恶如此说着，又往抱住我脖子的手上加强力气。
　好像是畏惧着魔王的奔跑速度，为了不被甩落而用力抱紧。不过从态度上看，似乎不仅仅是因为如此。

。
「跟、跟别人、像这样接触什么的，那个、是第一次来着」

。
　恶的脸紧贴在背上，开口说道，听到这番话的魔王则是蹙起半边眉头。

。
（对于这家伙的性别，我到现在依旧是不明不白……）

。
「魔、魔王大人要是前往祠堂的话，肯定是想许愿得到征服世界的力量吧！」
。
「这种力量我才不需要！」

。
　虽然魔王作为成年人的态度正逐渐奔溃，不过他们终于抵达目的地。这裡是杳无人迹的森林之中更加人影稀疏的场所。
　那便是巨大的山墙上凿开的――洞窟。



　■□■□



（喂喂，这气味不就是……）
。

　一接近洞窟，一股异样的臭味便扑鼻而来，魔王不悦地蹙起眉头。

。
「恶，你在这裡等着。裡面好像很危险」
。
「好、好的！」
。

　气味的根源显而易见。洞窟深处尽是人类的尸体。
　既有似乎被巨大的尖爪切伤的，也有身体只剩一部分的焦炭。从这些尸体裡流出的大量血液与内脏溅洒而出的尿粪相互结合，产生出异样般的臭味。

。
（这是在搞什么黑魔术的仪式吗）
。

　一座石像坐镇于此，仿佛在俯瞰着这些尸体。
　只见石像上索绕着邪气，这邪气足以令人觉得光凭浓度就可以杀死人。愿望祠堂这一名字与这幅光景实在相差甚远。
。
　当魔王往前迈出步伐的瞬间，石像的眼睛散发出赤红之光。
　魔王见状，把右手伸向小刀。不过石像纹丝不动，仿佛在用那双眼睛仔细端详着入侵者的全身。
　毫无行动迹象的石像开口说道，嘟嚷着什么。
。

「原来如此――这的确是“魔王”」

「……嗯？」

「虽然吾已经实现了诸多愿望，不过这恐怕将会是生平最後一次了吧」

「等、等一下……你知道些什么吗？难道说，是你把我叫过来的？」
。

　石像对于魔王的质问保持沉默，不久後开口说道。其回答对于魔王而言是极为重要的事情，然而魔像却若无其事地说了出来。
。

「并非是吾――亦或是说这群人类吧。要让“魔王降临”呢」
。
「这些家伙吗……！？刚才的怪物也是这些家伙叫出来的？」
。
「它是以一己之力复活的――拜此所赐，吾之力也已经消失殆尽」
。
「消失殆尽……那么，在力量消失之前先让我回到原本的世界」

。
　石像的回答实在过于简洁――「无法办到」。
　过于残酷的断定语气，令魔王不禁挠破脑袋。
。

「为什么办不到啊。想要香钱吗？应该不会是像这群家伙一样，要我献上祭品吧」
。
「这裡的死人都是古雷欧鲁干的好事。而且吾无法实现“违背愿望”的愿望――」

。
　简而言之，令魔王降临的愿望已经实现了――所以能够令这个愿望作废的愿望都无法实现吧。某种意义上真是循规蹈矩。

。
「虽说如此，汝亦即是最终的来访者――就把这个赋予给汝」
。

　从石像身上出现了不详的戒指，并强制性戴在魔王的手指上。看到这枚戒指的恶心设计，魔王拼命想要摘下来，无奈戒指纹丝不动。

。
「别给我开玩笑了，你这家伙！随身携带这枚戒指是要搞什么惩罚游戏吗！」
。
「吾祈祷着――汝之愿实现之时――」
。
「你这个混蛋邪神……给我等一下！」
。
「吾本应乃为清廉之身――只因常年累月中，人类的邪恶之愿才化为此身」
。

　石像宛如说完遗言一般，身体便粉身碎骨，遂而化为粉末，从台座上流落而尽。而看着这幅景象的魔王却无能为力，只能呆然地看着石像崩毁。
。

「哈啊……就算得到这枚戒指，又要怎么办才好啊」
。

　魔王看见右手指上戴着的戒指，不禁垂头丧气。
　无论用上多少力气，做什么事都完全摘不下戒指。戒指样式也是，完全就是负面道具。

「管理者权限――《道具鉴定》」
。

　魔王发着牢骚进行道具鉴定，结果更是垂头丧气。
　如果这是拥有“诅咒属性”的道具，还可以使用解咒道具摘下戒指，不过期待似乎落空了。

。
「为什么是以普通的道具对待啊……而且连鉴定都要使用SP么……」

。
　魔王不停地叹气。
　这下子根本就不是愿望祠堂，而是叹息祠堂。
　当出去洞窟外面，恶就满脸笑容地搭话说「怎么样？」，不过魔王依旧板着一张脸。
。

「果然是征服世界吧？还是酒池肉林之类的呢？」
。

　魔王听到这句话，一声不吭地封住恶的脖子。
　接着，缓慢地用拳头轻轻的往恶头上招呼。
。

　铿锵！铿锵！
。
「好痛好痛！请住手吧，魔王大人！」



　■□■□



（接下来要怎么办呢……要去其他城市收集情报吗？）
。

　“我”把长发往背後一甩，沉浸在思考之中。
　不知道的事情，不清楚的事情实在太多。
　如果还像这样对事情不明所以，迟早会陷入意想不到的圈套中。是时候从这偏僻的森林脱身了吧。

。
（结果也没从那些尸体中找到什么呢……）
。

　那副光景仅仅只是回想起来，就有股想吐的冲动。
　虽说游戏中经常出现大量尸体，不过跟在现实中看见尸体完全就是两码事。如果说那副光景是刚才的恶魔造成的，那么消灭掉它果然是正确的选择。
　只要那只恶魔还活着，就只会产生人类的尸体而已。
。

「恶，这附近有什么大城市吗？」
。
「有的……可是在这之前，能不能去一趟村裡？　虽然我的行李很少，不过我想带走……」
。
「嗯？你还打算跟着来？」

「不、不行吗……那个，我明明就已经作为祭品贡献出去了，又不可能再回去过日子……呜呜……」
。

　听到恶这番话，我不禁想抱着头，不过重新想一想，这或许也不错。
　不管怎么说，我对于这个世界实在不甚了解。如果身边有位这个世界的居民，也算是一剂强心剂吧。
　而且，从迄今听到的遭遇来看，恶即便继续呆在那村子裡也只会遭受到严酷的对待吧。
。

「我知道了，那就先去你们的村子吧。近吗？」
。
「非常感谢！以魔王大人的速度很快就到了！」
。

　就这样，魔王又再次背着小孩子奔跑起来。
　虽然魔王嘟嚷着「又不是带子雄狼……」，不过恶的表情松弛着，总觉得似乎很高兴的样子。
(颯:《带子雄狼》（日文：子連れ狼）是日本的一部以江户时代为背景的时代剧漫画，描述遭柳生一族陷害的高强剑客拜一刀，为了避难与伺机报仇，带着幼子大五郎浪迹天涯的故事。by百度百科)


　■□■□



（这种时候，在抵达村子之前还是先打听一些事情吧……）
。

　路途中，我向恶提出各种疑问与询问。
　事到如今，已经知道日语能够沟通，而记在本子上的日语、数字以及罗马音，恶都能顺利地读出来。
　是因为这世界原本就是能用日语沟通的吗，还是说有某种过滤器在运作呢，这方面根本无从得知。
。

「嘛，既然日语可以相通那是再好不过……这种岁数还要学习异世界语言什么的，老子才不干」

「……？这字体非常漂亮，也容易看懂哦？」
。

　对，写在本子上的文字漂亮到我自己都怕。
　这恐怕不是我的字迹，而是九内的吧。真不愧是设定上的高官。
　自己的字迹比较丑这一点，莫名感觉到败北感。
。

「不过祭品还真是古时代的习俗呢……这附近也有这种习俗吗？」
。
「由于恶魔王在几年前复活，这一带都遭受波及，因此附近的村庄也陆续有了献上祭品的习俗……」
。
「虽然我不怎么清楚……不过你曾经说过的“国家”就没有做过什么措施吗？无论是讨伐恶魔又或者是什么不也可以」
。
「因为这一带是远离神都的土地……」

。
　换句话说，从“都会”的角度来看，这裡是怎样都好的土地？
　在日本也有边境的集落，离岛之类的。偶尔会在新闻上看到这些信息，差不多是这种感觉吧。
。

「那个、魔王大人……究竟是从什么地方过来的呢？」
。

　我一瞬间犹豫着要怎么回答。
　毕竟就算说出日本也不可能知道，况且大帝国之类的更是意义不明。因为那只不过是以我的妄想制作的游戏世界，根本不存在于现实之中。
。

「嘛、嘛……该说是从很远的地方而来吧」

。
　我暧昧不清地说着，加快步伐。毕竟实话实说谁也不会相信我，而且若被人认为是一位好疯子，那也是个悲剧。
。

「啊、魔王大人，那片栅栏对面就是我们村子！」
。
「那裡吗……」
。

　虽然我对自己逐渐习惯被人称呼“魔王”这件事感到可怕，不过栅栏对面的光景才更加可怕。眼前阔展的村庄与其说是无人居住，不如说是鬼怪故事裡出现的寂静小村。
