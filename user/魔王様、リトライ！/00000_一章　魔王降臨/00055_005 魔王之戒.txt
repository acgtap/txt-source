
　明明现在仍是白天，可却没有从那村子裡感觉到任何“人类的活力”。
　脑袋裡浮现出深山内部裡封闭式村庄的印象。

。
「这村子给人的感觉就像是无法相信外地人呢」
。
「是、是的……我们村子是有排斥外来者的倾向」

。
　还真的是这样啊……虽然我只是说出我的偏见而已。
　我还想着如果有机会就分点食物或者旅费呢，不过期待落空了。
。

「我的家在这边。这裡或许会让您留下不好的回忆，务必请宽容大量……稍等一下，我马上就回来」
。

　恶从我背上下来，拖着右脚般走了过去。
　看到恶的背影，心中不禁感到悲哀。或许恶都无法正经地看过医生说不定。
　我边跟在恶的身後，边谨慎地观察村子裡面。该说果真如此吗，完全看不出是近代文明的感觉。

。
（房屋是以裸露的木头以及凝固的土块做成……屋顶也是铺上一层层稻草）
。

　在日本可谓是理所当然的空调、暖气、以及电线等都完全看不到一丝痕迹。
　我再次认清这裡是与日本截然不同的世界。
　当我还在端详着周遭的事物时，远方到处都看得到村民的身影。
　是因为游戏的影响吗，我立即隐藏身姿――由于在游戏被其他玩家发现就会遭受到残酷的对待，所以这反射动作几乎已经算是职业病。
。

「应战态势变更――――《隐秘态势》」
。

　在这句话脱口而出的瞬间，我的身体仿佛与周围的景色同化一般，轮廓变得模糊不清。
　这在游戏中拥有急剧降低被对象发现的效果，不过却有大幅度降低攻击力与防御力的缺点。
。
　当看见自己的身体消失了，我才安心地摸了摸胸口。
　虽说心裡还是有点紧张，不过从周围的反应来看，似乎并没有看到我。
。
　连这些系统都跟游戏如出一辙，莫名感到有些可怕。
　这世界――是否能够把游戏中存在的都全部具体化呢？
　那么，尝试一下其他事情吧。
。

「啊咧……魔王大人？」

。
　果然似乎就连恶也看不到我。
　在这堪称绝妙的时机上，我有件务必想要尝试的事情。
。
　在游戏中几乎很少使用的《通讯》变成什么样了呢？这功能主要是在玩家之间传达信息。
　在游戏最初运营的时候，当然不存在什么智能手机。那时候的通讯可谓是风靡一时，不过在近年来早已经是过时的机能。
。

《通信“恶”――听得见吗？》

《……诶，大脑裡响起了魔王大人不详的声音！？》
。
《不详这句话是多余的。我的事情就不用在意了，反正就在附近》

《好、好的》
。

　我顺利地确认完通讯机能的运作状态。
　果然，其他的事情也尝试看看吧。如果我孤零零地呆在那片森林裡，肯定还会是不明不白的状态之中吧。
　或许我该向恶表示感谢也说不定呢……。
。

（不过我神经也太大条了吧……照一般情况来说，现在我就算是哭天喊地也不奇怪）
。

　难道说连我的意识都向着身体的持有者“九内本人”靠拢了？

。
（别想了，别想了……这是什么恐怖片啊）

。
　自己想到的――是非常可怕的想象。
　我连忙摇摇头，把无意义的幻想抛之脑后。
。

「喂，垃圾――为什么你会在这裡！」

。
　我听见不悦的声音，便朝着声音源头的方向一看。只见几位像是村民的人，手指着恶怒吼道。
　不用想也知道，这就是那帮欺凌恶的人吧。不对，或许是整座村庄也说不定。
。

「垃圾，你……应该不会是逃跑了吧！」
。
「别给我开玩笑！要是那只恶魔来到村裡，你打算怎么办啊！」
。
「你到底知不知道作为祭品的意义啊！？」

。
　听到村民们说的这些话，我头都大了。
　这些家伙到底知不知道自己在跟小孩子说些什么？
　虽说如此，并不是这世界上居民的我，也难以插手这世界的事情(？)。
。
　在这个世界上，向恶魔献上祭品或许是理所当然的事情。现代中也经常出现外国独特的习俗或者「在日本是难以想象的事情，不过在某些国家却是常识」等情形。
。

（话是这么说、呢……）

。
　果然，大人斥责小孩子并不是令人愉快的事情。
　如果保持隐秘态势的状态下直接带走的话，应该可以平安了事吧？

。

『若是感到不悦，“肃清”即可――――』

。

　在大脑裡回响的声音，令我背脊发寒。
　我立马明白这究竟是“谁”。接着，右手指产生一股――难以忍受的苦痛。
　我不禁蹲下来按住右手。
　疼痛过于激烈，我完全无法站起来……！
。

『处分已审判的无法者――此“权限”便在“吾”身』

。
　别开玩笑……那是仅限于游戏中发生的事情。
　对于现实中生存的人类进行肃清、处分，这种事情我怎么可能做得出来。

。
『汝说了莫名其妙的话。制作这样的权限，这样的游戏，这样的系统，这么疯狂的国家――不全都“汝”吗』

。
　听到这番话，我不禁陷入沉默。

。
『汝才是――“诸恶的根源”。我若是魔王，那汝亦即是为全世界招来破灭的――』

。
　我紧握着戒指，以仿佛会握碎戒指的力度强行让声音消失。
这番话只不过是我的幻听罢了――我如此相信着，深深地闭上眼睛。发生这么多事情，肯定是我累了吧。

。
《恶，别管这些家伙，去把行李拿回来就好了》

。
　等了一会儿，并没有答复我。
　我睁开眼睛，恶那仰空倒在地面上的身影便映入眼帘。村民举起手，似乎在用什么肮脏的话语怒骂着。
　我无法再继续袖手旁观，解除了隐秘态势。
。
　瞬间――村民们吵闹不已。这也是没办法的事情，毕竟突然有人凭空出现。不过他们那副狼狈的样子真使人发笑。

。
「恶，拿回行李就赶紧过来」

「我、我……知道了」
。

　我看着恶拐着脚往房屋裡走进去之後，慢慢点上香烟。这时候村民们依然没有停止吵闹，人数还反而还越来越多。
　原本我应该是要向他们打听许多事情的，不过我没兴趣跟这些家伙聊天。
。

「你、你是恶魔吗……？是那位大人的手下吗……？」
。
「别对我们村子出手！我们已经献上祭品才对！」
。
「跟约好的不一样！说到恶魔应该都是会遵守约定的吧！」
。

　我边吐出烟气，边思考着村民说的话。
　跟恶魔约定、契约还真是引人趣味的事情。不过我倒是比较担心“对方”是否有打算要遵守约定的意思。
　倒不如说，这应该是对方先提出契约，再借此把对象局限在封闭的环境之中吧？
。

「让您久等了，魔王大人！」

「居、居然是魔王！？」

「魔魔魔王！」
。
（喂喂，别说些多余的事情啊……！）

。
　恶的无心之言，让村民们的喧吵声更上一层楼。
　虽说如此，我也无心解开这帮家伙对我的误会。把小孩子叫做垃圾，当做祭品贡献，再加上整座村庄都在施虐。
　我有必要为这些家伙努力吗――？不，没有。
。

「我、我先去通报领主大人吧！」
。

　说完，撂倒恶的男子跑了起来。
　其表情上浮现出令人生厌的笑容。
　或许是想到什么事情吧，男人先是回家一趟，出来的时候就已经拿着个背包。应该是为了得到奖赏而准备放进去的吧。

。
　我――无意间蹙起眉头。

。
　戒指仿佛呼应着我的感情，释放出妖艳的光芒――只不过这一次我并没有打算阻止。
　右手伸进外套裡面，毫无取巧地往男人家裡投掷一把小刀。
　小刀准确无误地刺进房屋上，迸发出漆黑火焰。用木头制作而成的房屋一瞬间起火了――包裹在袅袅升起的黑烟之中。
。

「我、我的……我的家啊啊啊啊啊啊！」

『――哈哈哈！无论何时，一看见火仿佛心灵都受到洗礼！』

。
　突然，我仰天说出这番话来。
　随即慌慌张张地背上恶――离开了这裡。如果放任自己委身于这枚戒指，似乎会酿成大祸。

。
（这不就像是个纵火魔一样吗！）
。

　我擦拭冷汗的同时，全速逃跑着。景色宛若飞奔一般地流逝，仿佛只有自己是在快进中。
。

「魔、魔王大人――没问题吗，那种事！」
。
「你、你别误会喔，我只是想取暖而已。就算说是我的温柔也不为过。嗯，我有这种感觉！」

。
　踏出的步伐已然无法停止。
　随意胡说的事情也已然无法停止。
。

「不、不过……！稍微有一点点、神清气爽了！」

。
　恶如此说完，便露出笑容。
　或许这是我和恶相遇以来，最初看到的“笑容”。

。
「对、对吧！人类的温柔果然是要传播出去的吧！」

。
　随意的言语，令我自己都忍不住喷出来。
　注意到的时候，两人早已经爆笑不停。
　抬头望天，明亮的太阳已经沉浸，夜幕即将要降临。如果是这具不知疲倦的身体，我感觉无论是天涯海角我都能奔腾而至。
。

「魔王大人，我们要去哪裡！？」
。

　在宛如吹拂着身体的风中，恶大叫道。
　迄今为止，恶的表情上总有某种难以掩饰的阴影，不过如今却绽放出与年龄相应的光彩。我――也不禁忘记年龄，大声回答道。
。


「我们将前往都会――那个叫“神都”的噢！」



。

　如此这般，化身成为小小纵火魔的魔王大人，与明亮的“恶”的旅行，就此开始。
　两人引发的无数次骚动，把圣光国搞得是天翻地覆……


。
　――――不过，那是在不远的将来所发生的事情。

。

「Status」
攻击与防御的“1”，为人类拥有的平均数值。
这数值无论是小孩子亦或是大人都不会产生变化。
换言之，若数值为“2”，那就是人类平均数值的两倍。
而“3”就是三倍的数值，如果是4或者5……
。
若硬要用现代的观念来衡量，可以说是近似于超一流运动员或是奥运会选手。
在这世界上1与2有相当大的差距，每提升一数值都要面临巨大的墙壁。或许可以称之为“才能之壁”。

。
「气力」
气力拥有形形色色的用途，魔法也是以此为基准而释放的。
基本上在指数完全耗尽之前都可以全力行动。
若是超一流的战士，都具备50点属性。
这代表两小时之内能够全力行动的指数。
。
九内的气力是600。
他在游戏上是二十四小时――全时段都能够全力行动。
既是强大无敌的“怪物”，也是一位“魔王”
