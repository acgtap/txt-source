
（真是不得了的美女啊……）

。
　“我”走在聖城的走廊中，回想起剛才的會談。
　由於是頗具超凡脫俗的美女，為了冷靜下來不禁狂吸著煙。
　最近在糾結場合吸煙問題，所以連煙灰缸也隨身攜帶著。她應該也沒那麼生氣吧。

。
（話又說回來，還真是相當正經的聖女來著……）

。
　或許是因為至今以來都過於不尋常吧，甚至對於名叫安潔露.霍瓦伊特的聖女抱有一絲好感。
　我一直認為這個國家的命名品味最糟糕了，現在終於有種遇見名字與人格一致的人物。
。
　最大的成果應該是把我的意向傳達給她這件事吧。
　見識我的實際行動，而不是單靠民眾的傳聞。
　在鎮壓完恐怖集團惡魔信奉者之後，又在露娜的村莊裡建造「醫院」和打造「溫泉」。
。
　無論由誰來看都是與魔王相差甚遠，完全就是善良的人類。
　當看到這些之後，就應該可以讓她知道這邊真正的樣子了吧。畢竟已經向她傳達我跟露娜關係很好的事情，也給與我書庫的利用許可。
　也可以說是最先的一步。
。

「昨天的龍人你看見了嗎！？」
「看到了看到了！真是帥得不行！」
「連古依大人都迷得神魂顛倒了吧！」

。
　聽見街道上傳來的對話，我差點摔了一跤。
　沒錯，上次的騷動中那個暴走族完全被推崇為「龍人」這種奇妙的東西。
　話是這麼說，我也得到預想以上的成果。
。
　面對持有巨大力量的“自稱魔王的男人”，這個國家會跳出幾個想要討伐魔王的人物也並非不可能。
　這種時候就由零來承擔這樣的職責，盡管不可能發生這種事，不過卻能讓他們以為既然有零在，一旦發生了什麼事應該會做點什麼而“放鬆警惕”吧。
　趁這段期間，這邊就穩當地發展事業。

。
（話說昨天……沒感覺到“違和感”啊）

。
　幾乎可以說是「一體化」了。
　在施展孤注一擲的時候，甚至有種幾乎是自己在揮出拳頭的感覺。
　這件事令我重新回想起我所製作的人物角色。

。
（遊戲的前半期，我就是九內伯斗本人）

。
　不對，名為九內伯斗的人物角色根本不存在于哪處，而是使用著我的本名「大野晶」。
　隨著時間的經過，我察覺到把自己本名暴露在網上很危險，於是在中途就決定改名，九內伯斗便就此誕生了。

。
（在進入遊戲中期開始之後，就一直是零來著）

。
　這真的可以說是作為零戰鬥的時間不僅有一年兩年，而是將近十年都在一起。
　雖說是當初作為NETA角色製作而成的人物，不過能這麼不知厭倦地沉溺其中，我應該是比起我自己想象中還要更加中意這角色吧。

。
（不對，已經不是這種程度了嗎……）

。
　若將近10年間都在使用這個人物角色，即便說是「自己本身」也不為過。
　雖然我不怎麼認同自己是那種暴走族……。
　一邊是魔王，一邊是暴走族。
　兩邊如果都說是不正經，也的確是如此……。

。
（我又不是像九內伯斗這種淡然施加暴虐的類型。話是這麼說，也不像零那樣找碴的硬派型暴走族 ……）

。
　如果是遊戲裡那倒還好，可在現實世界中對於「零所做的事情」會感到羞恥，我個人是拒絕的……。
　或許自己如果能夠接受零的話，真有可能會成為「我自己本身」不是嗎。
。
　這是一個既令人懼怕，又有某種愉快，以及像是要打從心底歡笑般的奇妙想法。

。
（……別想了別想了！比起這件事來，現在應該考慮今後的事情吧）

。
　已經得到出入書庫的許可這件事。
　首先要解決的是盡情調查資料這邊吧。而且SP也充分積累了許多。
　還必須得考慮召喚親信。上次剩餘的跟這次戰鬥所得的SP加起來，總共是有1249點SP。

。
（咕呵呵……啊哈哈哈！）

。
　我不禁想要放聲大笑。
　這樣一來即便召喚親信，建造醫院打造溫泉都還會有剩餘的SP。
　所謂的痛快指的就是這種事吧。僅僅是想象到底要召喚誰都情不自禁地興奮不已。
。

「說起來，好像是有新的項目《解放》了啊……」

。
　我不禁脫口而出，開始思考起來。
　新解放的指令是名為《全移動》的項目。
　在遊戲中由於有必要在廣闊的會場中來回行動，因此只要使用這項指令就可以瞬間進行移動。
　但是，卻有消耗30氣力的缺點。
。

（如果照這種狀況下去，商店和徽章交換也恢復的話就好了……）
。

　商店――顧名思義，即是用金錢購買道具的系統。
　這方面跟所持金有所關聯，因此不用在意。
。
　如果能夠順利賺到錢，遊戲裡的商店也恢復了，那麼或許就有可能在裡面購買道具。
　如此一來，即便不用消耗SP一個一個地製作道具也可以解決問題吧。
。
　徽章交換正如其名，通過收集遊戲會場中掉落的徽章而能夠換取各式各樣的高性能道具。
　這是一種尋寶遊戲，為了照顧不善於戰鬥的人也能享受過程，裡面還含有玩樂要素。
　雖說是玩樂，只要收集許多徽章，也能夠換取完全不能開玩笑的多種類道具……。

。
「魔王大人，跟聖女大人的談話怎麼樣了？」
。
一走出走廊，惡便滿臉笑容地跑過來。
　可愛到這邊都情不自禁露出笑容。
　我不禁以往常的習慣單手抱起她。惡也很開心似地雙手抱住我的脖子。

。
（越來越像父親與女兒了啊……這樣子……）

。
　我一邊眺望著照射大地的太陽，一邊想著這些無意義的事情――



　■□■□

。

　――聖城　古依的房間

。
　上次的戰鬥平安結束，古依在房間裡修養身體。
　不僅使用回復魔法，而且還毫不憐惜地使用昂貴的藥水，所以她的身體已經恢復到將近戰鬥前的狀態。
　雖說如此，由於有必要觀望情況，因此需要在這裡安靜地度過一段時間。
。

「哈啊啊……零大人……」
。

　雖說她呼吸紊亂，臉面泛紅，不過其實也不是發燒了。
　不如說她精神到在床上抱著枕頭翻來覆去。身體的傷口幾乎已經痊癒，不過別的地方倒是患上重症了。

。
　――完全患上了戀愛之病。

。
　若知道平時她的為人，一旦看見這幅模樣，心臟勢必會為之凍結。
　不對，在這之前就已經在物理上慘遭殺害也說不定……。

。
「真是超具魅力啊……真帥氣……」

。
　古依的腦海裡浮現出零的英姿，沉浸在宛若沉醉的心情中。
　那條雄偉的猛龍，以及那道仿彿撕裂黑暗的銀色閃光。
　銳利而又極為端正的臉孔。
　全身的一切肉體歷經鍛鍊，仿彿是為了戰鬥而存在。構成零的所有一切都是她理想中的期望。

。
　――――天喲，盡管拭目以待吧！
。

「啊啊啊啊啊啊啊……！我已經不行了……！」

。
　終於連耳朵也回響起“龍之咆哮”，古依在床上苦悶著。
　她抱住枕頭打著滾，接著從床上滾落下來，不過還是繼續滾到房間的角落，直到與牆壁激烈相撞才終於停止下來。

。
　我有要完成的“使命”――

。
　這是零臨走前說過的話。
　古依不禁對於這件事深入思考。究竟是什麼使命呢。
　那種實力的“龍”所懷抱的使命究竟是多麼重大的事情。

。
（難道是奈落……？）

。
　古依腦海中立即浮現出“那個”。
　回想起來，每當零出現的時候，奈落也絕對已經出現了。
　上一次奈落在不知不覺之間就消失了，這一次也是同樣。簡直就像是懼怕著龍一樣。

。
（那些洞口深處……也已經是空空如也了啊……）

。
　在神都打開的三個洞口當然已經派遣以騎士團為中心的調查隊進去裡面。
　以人類之手想要挖掘那些洞口，可謂是相當的執念……不對，該說是妄念。
　然而調查隊所調查的結果顯示洞口深處已經空空如也，什麼痕跡都沒有。當然，就連奈落的影子也是。
。
　如今動員所有使用「土」的魔法師進行修復工程，以極高的效率填補洞口。
　這次還慎重地鋪設幾層強硬的泥土與岩石。

。
（那個該死的奈落……啊，可是……）
。

　盡管古依想對可恨的奈落咂嘴，不過回想起多虧奈落才與零邂逅一事。
　她憎恨奈落，想要親手摧毀它，想要踐踏它，想要蹂躪它。
。
　然而只要它出現，或許又能再次相會也說不定。
　那條勇猛的――銀色之龍。

。
「啊啊啊啊啊啊啊……！我到底要怎麼辦啦……！」

。
　古依煩躁地苦惱著，又再次滾起來。
　看來與霍瓦伊特同樣，她的苦惱要持續一段很長的時間。
