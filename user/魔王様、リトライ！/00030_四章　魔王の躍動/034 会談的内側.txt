
（真是不得了的美女啊……）

。
　“我”走在圣城的走廊中，回想起刚才的会谈。
　由于是颇具超凡脱俗的美女，为了冷静下来不禁狂吸着烟。
　最近在纠结场合吸烟问题，所以连烟灰缸也随身携带着。她应该也没那么生气吧。

。
（话又说回来，还真是相当正经的圣女来着……）

。
　或许是因为至今以来都过于不寻常吧，甚至对于名叫安洁露.霍瓦伊特的圣女抱有一丝好感。
　我一直认为这个国家的命名品味最糟糕了，现在终于有种遇见名字与人格一致的人物。
。
　最大的成果应该是把我的意向传达给她这件事吧。
　见识我的实际行动，而不是单靠民众的传闻。
　在镇压完恐怖集团恶魔信奉者之後，又在露娜的村庄裡建造「医院」和打造「温泉」。
。
　无论由谁来看都是与魔王相差甚远，完全就是善良的人类。
　当看到这些之後，就应该可以让她知道这边真正的样子了吧。毕竟已经向她传达我跟露娜关系很好的事情，也给与我书库的利用许可。
　也可以说是最先的一步。
。

「昨天的龙人你看见了吗！？」
「看到了看到了！真是帅得不行！」
「连古依大人都迷得神魂颠倒了吧！」

。
　听见街道上传来的对话，我差点摔了一跤。
　没错，上次的骚动中那个暴走族完全被推崇为「龙人」这种奇妙的东西。
　话是这么说，我也得到预想以上的成果。
。
　面对持有巨大力量的“自称魔王的男人”，这个国家会跳出几个想要讨伐魔王的人物也并非不可能。
　这种时候就由零来承担这样的职责，尽管不可能发生这种事，不过却能让他们以为既然有零在，一旦发生了什么事应该会做点什么而“放松警惕”吧。
　趁这段期间，这边就稳当地发展事业。

。
（话说昨天……没感觉到“违和感”啊）

。
　几乎可以说是「一体化」了。
　在施展孤注一掷的时候，甚至有种几乎是自己在挥出拳头的感觉。
　这件事令我重新回想起我所制作的人物角色。

。
（游戏的前半期，我就是九内伯斗本人）

。
　不对，名为九内伯斗的人物角色根本不存在于哪处，而是使用着我的本名「大野晶」。
　随着时间的经过，我察觉到把自己本名暴露在网上很危险，于是在中途就决定改名，九内伯斗便就此诞生了。

。
（在进入游戏中期开始之後，就一直是零来着）

。
　这真的可以说是作为零战斗的时间不仅有一年两年，而是将近十年都在一起。
　虽说是当初作为NETA角色制作而成的人物，不过能这么不知厌倦地沉溺其中，我应该是比起我自己想象中还要更加中意这角色吧。

。
（不对，已经不是这种程度了吗……）

。
　若将近10年间都在使用这个人物角色，即便说是「自己本身」也不为过。
　虽然我不怎么认同自己是那种暴走族……。
　一边是魔王，一边是暴走族。
　两边如果都说是不正经，也的确是如此……。

。
（我又不是像九内伯斗这种淡然施加暴虐的类型。话是这么说，也不像零那样找碴的硬派型暴走族 ……）

。
　如果是游戏裡那倒还好，可在现实世界中对于「零所做的事情」会感到羞耻，我个人是拒绝的……。
　或许自己如果能够接受零的话，真有可能会成为「我自己本身」不是吗。
。
　这是一个既令人惧怕，又有某种愉快，以及像是要打从心底欢笑般的奇妙想法。

。
（……别想了别想了！比起这件事来，现在应该考虑今後的事情吧）

。
　已经得到出入书库的许可这件事。
　首先要解决的是尽情调查资料这边吧。而且SP也充分积累了许多。
　还必须得考虑召唤亲信。上次剩余的跟这次战斗所得的SP加起来，总共是有1249点SP。

。
（咕呵呵……啊哈哈哈！）

。
　我不禁想要放声大笑。
　这样一来即便召唤亲信，建造医院打造温泉都还会有剩余的SP。
　所谓的痛快指的就是这种事吧。仅仅是想象到底要召唤谁都情不自禁地兴奋不已。
。

「说起来，好像是有新的项目《解放》了啊……」

。
　我不禁脱口而出，开始思考起来。
　新解放的指令是名为《全移动》的项目。
　在游戏中由于有必要在广阔的会场中来回行动，因此只要使用这项指令就可以瞬间进行移动。
　但是，却有消耗30气力的缺点。
。

（如果照这种状况下去，商店和徽章交换也恢复的话就好了……）
。

　商店――顾名思义，即是用金钱购买道具的系统。
　这方面跟所持金有所关联，因此不用在意。
。
　如果能够顺利赚到钱，游戏裡的商店也恢复了，那么或许就有可能在裡面购买道具。
　如此一来，即便不用消耗SP一个一个地制作道具也可以解决问题吧。
。
　徽章交换正如其名，通过收集游戏会场中掉落的徽章而能够换取各式各样的高性能道具。
　这是一种寻宝游戏，为了照顾不善于战斗的人也能享受过程，裡面还含有玩乐要素。
　虽说是玩乐，只要收集许多徽章，也能够换取完全不能开玩笑的多种类道具……。

。
「魔王大人，跟圣女大人的谈话怎么样了？」
。
一走出走廊，恶便满脸笑容地跑过来。
　可爱到这边都情不自禁露出笑容。
　我不禁以往常的习惯单手抱起她。恶也很开心似地双手抱住我的脖子。

。
（越来越像父亲与女儿了啊……这样子……）

。
　我一边眺望着照射大地的太阳，一边想着这些无意义的事情――



　■□■□

。

　――圣城　古依的房间

。
　上次的战斗平安结束，古依在房间裡修养身体。
　不仅使用回复魔法，而且还毫不怜惜地使用昂贵的药水，所以她的身体已经恢复到将近战斗前的状态。
　虽说如此，由于有必要观望情况，因此需要在这裡安静地度过一段时间。
。

「哈啊啊……零大人……」
。

　虽说她呼吸紊乱，脸面泛红，不过其实也不是发烧了。
　不如说她精神到在床上抱着枕头翻来覆去。身体的伤口几乎已经痊愈，不过别的地方倒是患上重症了。

。
　――完全患上了恋爱之病。

。
　若知道平时她的为人，一旦看见这幅模样，心脏势必会为之冻结。
　不对，在这之前就已经在物理上惨遭杀害也说不定……。

。
「真是超具魅力啊……真帅气……」

。
　古依的脑海裡浮现出零的英姿，沉浸在宛若沉醉的心情中。
　那条雄伟的猛龙，以及那道仿佛撕裂黑暗的银色闪光。
　锐利而又极为端正的脸孔。
　全身的一切肉体历经锻炼，仿佛是为了战斗而存在。构成零的所有一切都是她理想中的期望。

。
　――――天哟，尽管拭目以待吧！
。

「啊啊啊啊啊啊啊……！我已经不行了……！」

。
　终于连耳朵也回响起“龙之咆哮”，古依在床上苦闷着。
　她抱住枕头打着滚，接着从床上滚落下来，不过还是继续滚到房间的角落，直到与墙壁激烈相撞才终于停止下来。

。
　我有要完成的“使命”――

。
　这是零临走前说过的话。
　古依不禁对于这件事深入思考。究竟是什么使命呢。
　那种实力的“龙”所怀抱的使命究竟是多么重大的事情。

。
（难道是奈落……？）

。
　古依脑海中立即浮现出“那个”。
　回想起来，每当零出现的时候，奈落也绝对已经出现了。
　上一次奈落在不知不觉之间就消失了，这一次也是同样。简直就像是惧怕着龙一样。

。
（那些洞口深处……也已经是空空如也了啊……）

。
　在神都打开的三个洞口当然已经派遣以骑士团为中心的调查队进去裡面。
　以人类之手想要挖掘那些洞口，可谓是相当的执念……不对，该说是妄念。
　然而调查队所调查的结果显示洞口深处已经空空如也，什么痕迹都没有。当然，就连奈落的影子也是。
。
　如今动员所有使用「土」的魔法师进行修复工程，以极高的效率填补洞口。
　这次还慎重地铺设几层强硬的泥土与岩石。

。
（那个该死的奈落……啊，可是……）
。

　尽管古依想对可恨的奈落咂嘴，不过回想起多亏奈落才与零邂逅一事。
　她憎恨奈落，想要亲手摧毁它，想要践踏它，想要蹂躏它。
。
　然而只要它出现，或许又能再次相会也说不定。
　那条勇猛的――银色之龙。

。
「啊啊啊啊啊啊啊……！我到底要怎么办啦……！」

。
　古依烦躁地苦恼着，又再次滚起来。
　看来与霍瓦伊特同样，她的苦恼要持续一段很长的时间。
