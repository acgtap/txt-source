
　――圣光国　某处

。
　那裡是一个无法令人想象是在地下的广阔空间。
　这裡是恶魔信奉者的根據地，并且还是幹部级的成员集中的地方。现在正针对於前些天的袭击展开商议。

。
「沃金格，你连奈落都携带而出，可却是这幅惨样吗？」
。
「虽说你精通指挥，不过看来还是没有摆脱胆怯怕事的缺点」

。
　交谈的对话中，大多数是在辱骂沃金格。
　然而他并没有提出反论，一直在忍耐着。因为无论是谁身在当场，都会确信那已经是必败无疑。
。

「那个龙人拥有不同寻常的力量。仅此而已」

。
　面对这句仿佛无话可说的断然说辞，其他人更加喧哗。
　然而，位于深处椅子上坐着的人物在举起手的瞬间――房间随即便鸦雀无声。
　恶魔信奉者的领导者——乌托邦沉稳地开口说道。

。
「比起那个男人，我更在意“魔王出现”这条传闻」
。

　谁也无法对这句话做出回答。
　魔王降临是他们的夙愿，然而奔赴愿意祠堂之者却没有任何人回来过。所以视为计划失败也是极为妥当。
。
　目前神都已经流传起「自称魔王的存在」的人物画，可那无论怎麼看都只不过是男性的人类。虽然长相有幾分可怕，不过还是无法称得上魔族。

。
「也有这条传闻呢，恶魔王苏醒了――不过遭到魔王杀害」
。

　乌托邦接着叙述的字句令所有人面露苦涩的表情，自己极其奢望降临的魔王把恶魔王杀死，那是多麼荒唐无稽的事情。
。
　尽管谣言说已经遭到魔王杀害，可就连恶魔王的復活都甚至只不过是传闻罢了，只是听在耳裡着实令他们感到不是滋味。
。

「无论如何，奈落需要更多的“祭品”与“力量”」

。
　那便意味着更大的混乱以及招致更多的伤亡。
　所有人听闻至此，都深深地点头。对於他们而言，他们认为这个国家的病魔已经达到无法痊愈的地步。
　既然如此，便唯有把全部化成灰烬再来重新建立，他们便应该是如此考虑的吧。

。
「――毁灭神都，无论好坏都一同投放于奈落裡面吧」

。
　从乌托邦口裡说出的话语便是破灭的宣言。
　幹部们连忙从席位上站起，各自为了做好应做的準备而离开这房间。还留在房间裡的只剩乌托邦一人。

。
「古雷欧鲁这愚蠢之者――究竟栽在了什麼“小石”身上了呢」

。
　在谁也不在的房间裡，乌托邦独自嘀咕着这句话。


。

。
　――神都　酒馆「诺玛诺玛」
。

　若说起神都聚集的冒险者最多的酒馆，那应该便是这裡吧。
　无关冒险者阶级高低，冒险者从早期以来便有时常关顾的倾向。
　多数人在仍未熬出头的时候开始就已经受到店家的关照，所以应该都怀有强烈的留恋之情吧。
。
　在这层意义而言，受到这家店关照的人为数不少，闻名遐迩之後也有许多人继续关顾这家店。
　原本便是小型酒馆的这家店，在经歷过数次的扩展，如今已然发展到号称神都首屈一指的名店。
　店家与顾客共同在成长，可堪称是稀有的实例。

。
「魔王、那就是魔王，不会错的！」

。
　蜜康向着店家老板娘吐露出抱怨之类的难以形容的话。其手上还拿着一杯啤酒，貌似喝得酩酊大醉。
　看来似乎是无法忘怀前些日子遇见的魔王。

。
「魔王……那傢伙是个好男人吗？」
。

　听完这句话，老板娘也回了句无比豪爽的话。
　她的名字伊夜。
　体态姣好，拥有五十多岁的妈妈风采。由於十分擅长照顾别人，因此仰慕她的冒险者也有不少。

。
「……是位时髦的叔叔大人。真想让他填补我的心与屁股上的空洞」
。
「啊哈哈！小雪风还是老样子呐。男人啊，只要抓住蛋蛋就是这边的东西啦」
。

　雪风边喝着牛奶，边说出令人无法恭维的言论。然而伊夜对此也豪爽地大笑道。
　两人没喝醉就已经是这副样子的话，一旦喝醉究竟会变成什麼样呢，真是令人惧怕。
　其中可谓是常识人的蜜康，无法忍受般红着脸大叫道。

。
「雪风，那可是魔王哦？你啊，是明知道还要这麼说吗！？说穿了，你根本就是男人吧！」
。
「……蜜康无知。伪娘也是能够好好怀孕的」
。
「你那杯牛奶是加了酒精吗！？」

。
　身在酒馆的男人们，听到这段对话表情都扭曲了。
　当听闻自身深爱的偶像「雪风大人」的心已经被其他人所夺走，应该无法再保持冷静了吧。

。
「那个男人……无、法、原、谅」
「什麼魔王嘛！绝对是夸大妄想的傢伙」
「在下反而倒是想被雪风殿下搞得怀孕是也」
「白痴！梦话给我到睡觉的时候再说！」

。
　他们作为雪风粉丝俱乐部的一员，跨越队伍之间的隔阂组成了同盟。制作各式各样的周边商品，时而也会做出一些本子。
　甚至还会有人身穿印有她样貌的斗篷。
　无论身處於哪个世界，粉丝所採取的行动似乎都是一个样。
。

「说到底呀，那可是有将近三百匹沙狼哦……？　一瞬间就全部烧死什麼的怎麼可能做得到啊！那是叫做灾难的存在啦！」
。
「……蜜康，你没有忘记自己是被叔叔大人拯救这件事吧？」
。
。
「呜……虽、虽然、是这样……」
。
「……报恩应当用身体偿还」
。
「这种事你就给我自己去做吧！」
。

　诺玛诺玛的喧哗在今天也依旧是持续不断。
　然而，其中却有一位男子把剑抱在膝盖上，没有与任何一个人进行交谈，一直不停地在喝着度数不低的葡萄酒。
　他是这个国家的著名剑士，是位誉为“剑闪之亚路贝鲁托”之称的男人。
　以一剑便把一切都斩断的剑豪。

。
「哈哈，自称魔王的男人吗……出现了一位提高名声的人物啊」
。

　把传闻中的存在砍死――究竟会得到多少功名成就与财富呢。
　他仅仅是想象着，便面露无畏的笑容。

。

。

　――神都　酒馆「亚露缇米丝」
。
　这裡并不是冒险者的酒馆，而是贵族时常关顾的加上“超”字高级店。
　店裡弥漫着可谓是静谧的氛围，是与喧闹相差甚远的气氛。
　在店裡的一个角落处，两位女性正安静地用着餐。
。
　在周围尽是贵族的环境中，身着冒险者打扮的她们是种轻浮的存在，可没有人会对此提出意见。
　因为她们便是称之为“Star Player”的人物，位居S阶级的世界级冒险者。
　她们聊天的话题并不是魔王――而是“龙人”。

。
「欧露康，你觉得真的会是龙人吗？」

。
　女人吃了一口料理，并开口说道。
　她身着的衣物充满着神圣的力量，宛若纯粹的修道士打扮。甚至就连下级恶魔都难以靠近。
　然而，比什麼都值得一提的是她的胸部。
　衣服仿佛都要撑破般高高顶起，令旁人目睹都会觉得痛苦似地。
。
　女人名叫米库。
　毫无疑问是Star Player其中一人，其美丽的蓝髮与巨大的胸部无不诱惑着男人的眼光。
。

「不会――我虽然想这麼说，可在这世上也不是百分百不存在」

。
　被称呼为欧露康的女人吃了一口沙拉，回答道。
　她并没有往沙拉上添加什麼调味料，似乎就直接吃进嘴裡。
　其身材娇小得有如小孩子，由於全身覆盖着一件黑色长袍，因此不是很清楚她的容貌。
　不过，以娇小玲珑的小嘴巴吃着沙拉那副脸颊鼓鼓的身姿却有幾分可爱之处。
。
　她是极为稀奇的混血儿――“魔人”。
　由於根據国家的不同而会成为该讨伐的对象，因此她隐藏真面目，一直变更着活动据点度过日子。
。
　圣光国当然也不认同魔人的存在，她们只是旅途中顺路来到这裡罢了。
　如今纵使找遍全世界也找不到知道她是魔人的人物。因为知情者――全部都已经消失得无影无踪。

。
「不过，真是有梦想的事情呢。憎恨邪恶的龙人什麼的」
。
「虽说如果真的存在的话，兽人国不可能会放任不管」
。
「你想说他是野生的龙人？」
。
「无聊透顶。如果真的存在，那还请务必让我见识一下――」
。

　明明同样是混血儿，龙人受人尊敬，魔人却遭受忌讳。
　从欧露康的角度来看，龙人是令她感到毫无道理可言，也是令她感到滑稽可笑的存在。
。

「我从传闻中听说魔王现身了。我的“暗”也在如此述说着」
。
「你应该是修道士吧……什麼暗啊」
。
「这只右眼的疼痛……不会错的。令世界陷入混沌的魔王已经降临，我们要讨伐成为黑暗的魔王」
。
「你到底在说什麼啊」
。

　遗憾的事情是米库患上了中二病。
　无论是哪支队伍，有一方是奇特人物或许已经算是既定规律也说不定。

。

　不知这些冒险者的思绪，魔王正逐渐接近神都。
　与此同时，恶魔信奉者也开始在水面之下展开行动。
　在不远的将来，于神都所引发的骚乱已然无法避免――
