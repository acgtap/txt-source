
　――聖光國　聖城

。
　聖光國是由三位聖女作為頂端的國家。
　其下聖堂教會與聖堂騎士團並列同一立場，一致抵御外敵，團結守國。
。
　教會是上至大貴族下至孤兒，若擁有魔法的素質無論是誰都一律接受，是個領域廣闊的組織。
。
　騎士團亦是如此，若擁有武術才能，無論來者何人都一律接收的組織。
　當然，這些組織都有無比嚴酷的考驗，必須是經過考驗的合格之者。可即便如此，對於任何人都敞開大門這點依舊毫無改變。
　這種形式即便放眼觀望周邊諸國――也可說是極為稀奇的事例。
。
　然而貴族近來的興起卻是極為顯著，全都陸續加入聖堂騎士團。
　時而委婉地以地位為由、時而以名譽為由、時而又是以金錢為由，展現出一切的事物，猶如在對待親手養育的小孩一般進行籠絡。

。
　這是因為――聖堂騎士團的意見在聖女的接替中占據重要立場。
。

　反觀教會，那邊的人難以進行收買。
　那邊不僅僅要有魔法的才能，還得是能夠配得上侍奉天使大人的清廉之身，主要是出自於這種考量，因此他們的廉潔要求極為嚴格。
　即便是貴族也無法收買他們，進而加入教會。
。
　故而他們把目標放在聖堂騎士團身上，為了產生令自身為之所用的聖女，花費漫長歲月緩慢逐漸地侵蝕著聖域。
　於是如今――替換出可謂理想般的聖女已經成功。
。

　露娜.愛蕾崗特　――　雖擁有魔法才能，卻對政治毫無興趣。
　琪菈.古依　――　在戰鬥相關事務上可謂理想般的人物，然而也對政治與金錢不感興趣。

。
　國家頂端人士對於政治以及權益都不抱有興趣，那也的確是很理想吧。
　通俗點來說，便是為所欲為。
　在當今這種狀況之中――
　于聖城的圓桌之上，代表國家的兩位貴族以及第三位聖女正召開著會議。

。
「哎呀，露娜大人似乎也萌生作為聖女大人的自覺了……」

。
　男人晃動著肥胖的身軀，肥大的臉頰也抖動著。
　這男人名為――多納.多納。
。
　統領多數貴族，是聖光國最富裕的大貴族。鐘愛金錢與年輕女子，典型的貴族典範。
。
　反之他在對待貴族方面上擁有先天性才能，其統率力不容小看。由於抱有多數領地內的礦山，而魔石價格也日復一年地提高，因此實則上是掌握國家經濟的男人。

。
「……可喜可賀」

。
　坐在多納的面前，簡短地嘀咕一句的男人則是 瑪夏魯(Marshal)・安茨。
　具有無法想象得到是貴族的緊綳堅硬體格，他還身穿鎧甲。
　歷經戰場的戰士——應該要這麼形容吧。他的年齡雖早已超過六十歲，可一頭白髮卻往后腦梳成一縷，其眼神也是銳利無比。
　他是統領武斗派的貴族，另一位國家代表。
。

「可是我很擔心呢……那孩子會不會是受到那個自稱魔王的男人誆騙之類的」

。
　身坐圓桌的上席，粉色頭髮搖擺不定的便是最後的聖女。
　安潔露.霍瓦伊特。
　無論是其頭髮、眼瞳、亦或是嘴唇都全是淡粉色，甚至連身上散發的氣息也深感端莊神聖――其美麗無法令人想象是這世上的人。
。
　多納.多納看見她便吞了口口沫。
　顯而易見，他正考慮著齷蹉的事情。他意圖總有一天會讓這位聖女成為自己的所有物。
。

「請務必放心。您若在意那個男人圖謀不軌，那就交由我這邊來處決吧……」

。
　多納.多納那雙粘稠般的視線看向聖女的胸口處。
　應該是在腦海中幻想著揉摸那一對圓潤碩大的胸部吧。
　連在圓桌之下的雙手都在不停地動彈著。

。
　她們交談的話題――是聖女露娜。
。

　她似乎說要親自大展身手，治理至今遭受教會管理者棄之不顧的土地。這該視為某種變化呢，又或是小小的成長呢，霍瓦伊特(純白)仍在煩惱著。
。

。
　兩位貴族無視沉默不語的霍瓦伊特，互相探討起來。
。

「……既然是露娜大人所信任的人物，那就不是我們該干涉的事情」
。
「哼，城裡不早就連通緝令都發布了嗎！」
。
「……多納，具體而言那個男人有招致過什麼損害嗎。單憑世俗傳聞你就想處決露娜大人所信用的對象？」
。
「我這邊可是有從邊境的比利茨而來的一份燒盡整座村莊的報告！首先，連露娜大人的護衛們不也被拋飛了嗎！」

。
　兩人毫不隱藏敵意，互相瞪視著對方。
　雖說原本兩人對待國家的心情便不盡相同，不過在這之前以作為人而言就已經水火不容。

。
「……然而我手上得到的情報顯示，卻只有一戶農家遭到燒毀？多半是比利茨想要把事情鬧大吧」
。
「安茨！你這人是想要說你懷疑“貴族”說的話嗎！」
。
「……並不是取決於身份，而是是否有值得相信對象的情報罷了。至少對於我來說，比起比利茨的闡述，我更相信自己部下的報告」
。

　兩人互相瞪視個不停。
　實際掌握著經濟與軍事頂端的貴族正處於分裂的狀態，也因此聖光國如今依然未能從長年的停懈之中脫離開來。
。
。

「那孩子難得會自发性提出這方面的事情……我認為暫時先觀望一下情況」
。
「既然聖女大人您這麼說的話……」
。
「……遵旨」

。
　多納面露不滿地點著頭，而安茨則是靜靜地閉上眼睛。
　當兩位貴族離開此地之後，仿佛取而代之般古依粗暴地打開門扉，發出巨響地坐在椅子上。
。
　猶如在述說這裡是我的固定位置一樣，她高高地抬起腳擱在圓桌之上。行為舉止毫無任何禮儀之處。

。
「前些天真是不得了呢……你平安沒事實在太好了」

。
　即便霍瓦伊特向她搭話說，古依也只是望著上面。
　不，根本不知道是否真的有傳進耳朵裡。

。
「…………迷上了」
。
「誒？」
。
「迷上了啦，對那個男人――現在回想起來也是帥到不行」
。
「等、等等……稍等一下，你在說什麼事情？」

。
　霍瓦伊特跟古依已經來往已久，所以很清楚古依離奇的言行舉動。可從她口中說出的話卻令她都要懷疑自己的耳朵。

。

「難道說，你是指傳聞中的龍人……？古依，我說你啊。按照常識來思考根本就不存在龍人什麼的不是嗎……」
。
「那是因為大姐你沒看見過所以才能說出這種滿不在乎的事情啦！那一拳！那身影！那張英姿凜然的容貌！以及那帥氣的台詞！」
。
「你、你冷靜一下……」
。
「我得快點找到零大人……哈啊啊，再把臉埋進那件衣服上啊啊啊……」
。
「為什麼、為什麼兩個妹妹都變奇怪了啦……！？」

。
　安潔露霍瓦伊特――唯一的一個正經聖女。
　而且，當然也是位辛苦的人。
　這份辛苦――勢必會持續一段漫長的時間。
