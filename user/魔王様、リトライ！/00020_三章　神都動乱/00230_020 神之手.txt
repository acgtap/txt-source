　连自己创造的人物角色都无法相信是要闹哪样。
。

「那麼，小恶……放鬆身体」
。
「好、好的……」

「那、那个……这、这只手……是……」

。
「没事的，小恶……就尖头部分而已，就尖头部分而已啦」

「呜……气喘吁吁的，总觉得好可怕！」

　唔喔喔喔！这傢伙到底在说些什麼！

　根本不是你所喜欢小正太啊！

「小恶，治疗结束之後要不要试一下男装？跟姐姐一起玩吧」

「我、我不想……！」
。
「悠，别玩了，快点做」
。
「这还真是失礼了」

　再继续放任不管也不知道她会说些什麼鬼。

「那麼，长官。开始了――」

「嗯，那就拜托你了」

。
　悠的右手——变化为各种各样的器官触碰着恶的脚。
　“那东西”仿佛像是在触诊般运作着，在疑似注射器的物体上满溢出液体，她把这液体注射在恶的脚内。

「等、等一下……魔王！真的没问题吗！？」
。
「别担心。你就相信我和我的部下吧」
。
「我、我可从来没有相信过你哦……」

。
　虽然我以冠冕堂皇的态度回答道，可其实内向是宛若祈祷般的心情。
　应该是医疗工具的一种吧，悠的手指连续不断地发生变化，依次接触着恶的右脚。
　而每一次都会令人觉得恶右脚的血色增加了。仿佛就像是没有血液流通的部分开始恢復正常一样。
。

　（拜托你了喔，悠……）
。

「好了，这样就结束。伤痕也帮你完美消除掉了。毕竟是女孩子的脚」

。
　……诶，已经治好了吗！？
　悠的手指头变化成像是刷子一样的东西，每当抚摸着伤口表面，伤痕便逐渐消失不见。即便含蓄地说也是惊人的光景。
　我安抚住激烈动摇的情绪，故作理所当然的举止。

。
「嗯……结束了吗」
。
「是的，长官。没有问题了」
。
「恶，站得起来吗？虽说已经治好伤口了，可也得稍微走一下让身体习惯起来」
。
「好、好的」

。
　恶站了起来，走了幾步。
　动作虽说有些僵硬，不过似乎已经不会再拖着脚走路了。

。
「走、走起来了……！我、我的脚！」
。
「那就好。那麼，稍微到外面走一下试试看吧」
。
「那、那个！悠大人，非常感谢您……！关、关於这件事，该怎麼说才好呢、那个……」
。
「好了啦，要道谢的话就向长官道谢」

。
　悠笑着如此说道，可我其实又没做过什麼事……。
　总之，让她在外面走一走稍微练习一下吧。虽说我幾乎没有康復训练的相关知识，不过牵着她的手一起走路这种事我还是办得到的。

。
「我稍微出去一下。你们就呆在这裡」

。
　我牵着恶的手走出据点。
　外面的太阳一如既往地在迎接我们，像今天这样值得贺喜的日子，只有万里晴空的天气才配得上――

。
「魔、魔王大人！我真的在走路……这样好厲害啊！」
。
「我早就说过了吧――大帝国没有不可能」

。
　我只字不提刚才内心其实在祈祷的这件事，很伟大似的回答道。
　不如说，我经常持续着演技也逐渐疲惫不堪了。在亲信面前虽说还不能放鬆警惕，不过跟恶两人单独在一起的时候，即便不用装模作样也可以吧。

。
「真的就好像在做梦一样……我能够普通地走路什麼的」
。
「不是在做梦啦。从今往後都能普通的走路、普通的生活」
。
「魔、魔王大人……能不能捏一下我的脸？」
。
「又是这种古典、的――」

。
　我笑着说出口的言语，在途中就噤口不言。
　因为从恶的眼睛裡――流出了眼泪。
　看到她这副样子，我莫名地无法冷静而移开了目光。不行了，这种氛围我从以前开始就很不擅长应付啊……。

。
「嘛，就是那个……怎麼说，实在是太好了」

。
　什麼鬼，这句台词……！
　就不能再稍微说些感觉不错的话吗。
　明明平时就能够随意地胡扯几句，关键时刻却是这样子，真想把自己的嘴巴给抓一顿。
。

「……魔王大人。可以再稍微走一下吗？」

恶如此说着，并握着我的手。虽然这风景实在不适合散步，不过其实我也不讨厌在这个异世界散步。
　乾涸的大地处处布满龟裂，照射着大地的太阳也无情地释放热量。
　可是，为什麼呢……我感觉比起现实中的生活，还是这边的生活比较有趣。

。
（或许是因为有这孩子在吧……）
。

　起初我只是在哀叹我为什麼会来到这种异世界，如果能够治好恶的脚伤，那我自己来到这个世界或许也算有所价值了。
　一想到我是为此来到这裡，这种不讲理的事情我也能够勉强接受。

。
「已经不能再让魔王大人背着我了呢――」
。
「没事，要是你累了我还是会背着你的啦」

。
　我莫名觉得恶的表情有股成熟的气息，不禁点上了香烟。
　虽说女孩子成长很快，可真的是这样吗。
　不巧我只是单身狗，所以关於这方面的事情也不是很清楚。当我想着这种事情的时候，感觉到牵着的手上力量加强了。
。

「魔王大人，请一直要呆在我身边哦」

「…………。这种话，你还是为了将来会喜欢的男人说出来比较好」
。

　一直。
　代表要在这世界上一直活下去。
　全数捨弃现实中的一切。
　明明今後还要调查炽天使的事情，我实在无法说出这种不负责任的话来。

。
「因为我已经“决定”了――」
。
「我说你啊……」
。

　13岁的小孩子到底在说些什麼啊……。
　想起初中的时候，也有一些人喜欢上身材高挑的老师，就跟那个差不多？

「那麼，是时候该回去了吧」

。
　结果，牵着的手――在回到据点之前都没有分开过。

「长官还真是相当关心那个孩子呢……」

（真想知道呢……关於这世界上人类的多方面事情）

「不过我也向你道谢。虽然我不会对那傢伙说出这种话，可对於你还是很感谢的」
