
　在亞露緹米絲的某一桌，有點引人注目。
　店裡雖流淌著由各式各樣的樂器所奏響的優雅音色，供顧客享受悅耳的聲音。可他們的視線卻集中在這一桌身上。
。
　聖女與自稱魔王的男人，以及――瑪妲穆.巴塔福萊依。
　若提到貴族，夫人掌握夫君的把柄是常有的事。而統領這些恐怖夫人們的大BOSS便是巴塔福萊依。
　她的影響力無法估量，程度達到她所討厭的人都會遭到社交界的驅除。周圍毫無關聯的顧客們也懷著一份緊張的心情觀望著事態進展。

。
「話又說回來――瑪妲穆女士真是頗為美麗。一想到您在美麗上付出的艱辛，本人都欽佩地想低下頭」
。
「哎呀，這是在追求我這種老太婆嗎」
。

　雖然魔王與巴塔福萊依的談笑已然開始，可眼神裡卻毫無笑意。
　互相打探著對方的模樣，等同於是白刃相交的樣子。

。
「吶，以“魔王大人”來看，這座都市感覺如何――？」
。
「是座極為美妙的都市。非常的直率，非常簡單易懂」
。
「……從他國而來的旅客大多也是驚訝無比呢。差距過於巨大，仿彿是天國與地獄這類說法也是毫不稀奇」
。
「原來如此，這種說法倒也不失妙處。不過，有一種方法可以解決此問題」

。
　魔王說到這裡便緘口不言。
　解決的對策，魔王語畢便悠然自得地點火抽了支煙，他沒有繼續說下去，而是悠閑地享受著抽煙。
　緊接著反倒是瑪妲穆這邊焦躁一般，從餐桌上探出身詢問道。

。
「還請務必讓我聽解一下“魔王大人”的睿智」
。
「也沒什麼，很簡單的事情――只要將“全部的地域”都化成天國，什麼問題都會沒有。極為單純的事情喔」
。
「全部……這、這還真是規模相當龐大的事情呢」
。
「――――假如我能立於國家頂端，幾年就可以實現吧」
。

　面對魔王這番話，瑪妲穆無言以對。
　這已經不是大放厥詞的問題，而是幾乎要懷疑頭腦是否正常。
　至今以來能令瑪妲穆喜歡的人物，大致上來說是以會敘說花言巧語者居多，能說大話到這種地步的人物應該還是初次遇見吧。
。
　可實際上這男人不可能有將全世界化為天國的計策。
　如果這男人做得到這種事，那他早在現實世界裡當上了政治家才對。

。
（嗚哈哈。畢竟單純的言論，無論說什麼都不用負責任）
。

　這是一種將錯就錯，令人懷著「不同尋常的人物」這種印象而做出的賭博般的演技。
　然而就瑪妲穆角度來看，坐在一旁的美女就好像在無言地肯定“這句話”。
　仿彿在述說剛才的發言並不是在說謊一樣的態度。

。
「不過在探討國家這種無聊的事物之前――我更想為瑪妲穆您提供別的“天國”喔」
。
「真是深感興趣呢……究竟會是怎樣的“天國”呢？」
。

　魔王用張漂亮的紙包起餐桌下製作完成的物品，並遞交給瑪妲穆。
　那便是深得女性好評的《香皂》。
。
　然而，交到她手上的香皂卻讓瑪妲穆的臉上流露出難以言喻的複雜神色。她還以為這位說出不同尋常的事情來的男人會交給自己什麼東西，沒想到是香皂。
　雖說香皂在這國家是以1枚～5枚銀幣才能買到的高價商品，不過對於瑪妲穆而言並不是什麼稀奇之物，也不是什麼值得感謝之物。
。

「啊，那香皂很厲害哦！」
。
「啊啦，是小露娜也推薦的東西嗎……？」
。
「洗身體的污垢都像是施加了“魔法”一樣，肌膚也變得滑溜溜的！」
。
「魔法……滑溜溜呢……」
。
（Nice助攻，露娜！）

。
　意想不到的掩護令魔王在心中擺出勝利的姿勢。
　實際上從露娜口中說出“魔法”一詞所具有的份量不同。
　對於瑪妲穆而言，露娜仍只是個小孩子，僅僅是天真無邪之處令人感到憐愛罷了。
。
　可只有她的魔法才能不同――是貨真價實的“真貨”。
　露娜對於自己的魔法引以為傲，這也是她全部的自尊心，瑪妲穆對此心知肚明。對於她而言，從露娜口中說出“魔法”的意義程度那就不一樣了。
。

「確實，在這裡的小姐們肌膚也似乎很漂亮呢」
。

　瑪妲穆環視著餐桌上女性陣容，微微浮現出嫉妒神情。
　年紀尚輕，其肌膚也毫無一絲皺紋。
　在此之上，聚集的――又都是美少女與美女。

。
「請務必在今晩嘗試一下。我認為您就能明白具體的效果」
。
「也是……那我就向魔王大人感謝這份送給我的禮物吧」
。

　瑪妲穆不禁往手上的香皂投以熱烈的眼光。
　對於女性而言，當年齡步入幾十來歲皮膚便是在意的事情。無論擁有多少財富，只有老化這方面是令人無計可施，“美”這一項目可說是鐵壁般的瑪妲穆唯一的弱點。
。

「不不不，就這種程度就受到您的感謝我也很困擾。我最近打算在拉比村打造溫泉――肌膚、全身都如同字面意思上獲得“新生”喔」
。

　那是一句――與統領魔界的王相稱的“魔性的說辭”吧。
。

「新生什麼的，您還真是說了相當誇張的事情呢……」
。
「我對於自己提供的東西都不會敘說任何謊言――因為從我口中闡述的事情，都將全部成為“現實”」
。
現在完全是暢所欲言。
　事已至此，現場早已是這男人的獨場秀了吧。

。
「那邊不僅是提供普通的熱水，碳酸溫泉、熱水浴、岩石浴、岩盤浴、冷水浴、通電浴、藥草浴和鹽性濕蒸桑拿以及蒸汽桑拿等項目均能享受得到。不單單驅除疲勞，對於肩痛、腰痛還有怕冷體質都具有非常不錯的效果。當然，享受完之後肌膚也會恢復年輕的樣子吧」
(颯:不愧是洗澡大國，查資料查了不止半小時，順便一提，除了上面這些還有其他N種桑拿)
。
「是、是這樣嗎……」

。
　瑪妲穆並沒有正確地理解到魔王所說的事情。
　然而，從中「這是不得了的東西」這件事倒是充分地傳達給了她。僅僅是聽他所說，自己如今都想去見識一下。
。
　事實上當這個男人把“這東西”建設完畢時，這些都成為了“現實”。
　原本就是提高體力回復速度的設施，可溫泉上所記載的效果也依據現實產生出“結果”了吧。
。

「真令人期待呢……不，這要到什麼時候才準備呢……」
。
「在這裡結束調查之後，立即就可」
。
「請、請務必讓我前去拜訪。小露娜，你要對卡其福萊依保密哦？」
(颯:名字為英文的炸牡蠣)
。
（還卡其福萊依！你們都是油炸食品嗎！）

。
　魔王在心中吐槽，不過卻絲毫沒有在表情上展露出來。可或許是在忍耐笑意吧，拿著香煙的手微微顫動著。

。
「知道了。可是瑪妲穆還是老樣子跟妹妹的關係很差呢」
。
「就跟小露娜一樣哦？所謂的姐妹還真是相當合不來的關係」
。
（同是油炸食品就給我好好相處啊……）

。
　魔王邊壓抑住想要大笑的衝動，邊滿臉笑容地與瑪妲穆握手。雖說遭遇時還想著會怎麼樣，不過平安無事地結束了。
　在那之後，一行人熱鬧地享受用餐。然而響起一陣尖銳的悲鳴聲，店內也騷然而起。
。
　從外面傳來的回響是震動與怒號。
　若傾耳聆聽，叫喊的只是同一件事情。

。

「「是、是惡魔信奉者的襲擊啊啊啊啊啊！」」

。

　――魔王的表情改變了。
　那是宛如小孩子找到玩具般的表情，又是難掩喜悅逐漸湧現的樣子。

。
「小丑似乎正好來參加晚餐會的餘興了呢――――」

。
　那模樣令瑪妲穆吞了口沫。
　因為身在此處的，是與至今以來的魔王有所不同――“名副其實的魔王”。
