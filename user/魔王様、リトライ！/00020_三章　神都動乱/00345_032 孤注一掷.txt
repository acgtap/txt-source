
「啊……咕…………」

。
　歐魯依特如今大腦正處於混亂至極。
　強韌的身軀也因嚴重的痛楚而響起慘叫。
。
　為何？
。
　平凡人類的普通拳頭為何會產生如此激烈的疼痛。仿佛就像是“無視”几重防御魔法一樣，攻擊貫穿身軀造成嚴重的傷害。
　並且還是足以令靈魂慘叫的痛苦。如果是聖元素使用者那倒另當別論，可對手的拳頭並沒有附加一絲魔力。

。
「你這傢伙還要給我“睡”到什麼時候――――」

。
　零毫不猶豫地用腳踹了歐魯依特的側腹。
　集讚美與憧憬於一身，被譽為暗公爵的上級魔族猶如足球慘遭踢飛――撞上聖城的結界。
　這一瞬間，歐魯依特的全身冒出火來。
。

「嘎啊啊啊啊啊啊啊！」

。
　巨響和火花傳遍於周遭，周圍也充斥著一股燒焦的味道。
　這也是理所當然――上級惡魔的身軀觸碰到智天使所鋪設的結界根本不可能會平安無事。
。
　無論他是否有意無意，都會宛如“導電爆炸死鬥”。
(颯:似乎是拳擊一種比賽方式)
　對於龍人甚至不把上級惡魔放在眼裏的實力，周圍人的熱情更上一層樓。然而零見狀，表情卻是愈發凶惡，遂而咂嘴。

。
「你這傢伙，居然更比我讓觀眾沸騰起來……“銀龍正合我意”嗎，**！？」

。
　這是暴走族特有的理論。
　正因為會從使得觀眾沸騰這種誇耀自身的行為中產生生理性的快感，因此一旦別人更加引人注目便完全惱羞成怒。
　要歐魯依特來說，應該就是「別開玩笑了！」這樣吧。

。
「你這個……**雜碎啊啊啊啊！《暗黒獄光線》」

。
　歐魯依特卯足全力釋放出暗魔法，漆黑光線直接命中零的身體。然而，本應貫穿萬物的光線在消逝後，站在原地上的依舊是毫髮無傷的暴走族。
　即便是暗公爵的魔力也無法對發動《狂亂麗舞》的零造成一絲傷害。
。
「你這傢伙……在跟我的“單挑”中還有從容“玩樂”？」

。
　零蹙起眉頭，毫無道理的怒氣愈加火大。
　或許是感到被愚弄了吧，零因憤怒而顫抖著身體。從他的角度來看，心情上就像是被小孩子用「激光槍」之類的玩具射擊他。
　在歐魯依特看來，這完全只能說是“惡夢”。

。
「少……少開玩笑了！高貴的我，居然會在這種地方！」

。
　歐魯依特展開身後的翅膀，翱翔於天際之上。
　隨即靠近在空中面無表情地俯瞰這場戰鬥的魔人――“托瓏”，用手刀毫不猶豫地刺穿。

。
「……啊……」

。
　歐魯依特抬起少女的身體，把身體裡流出的鮮血一飲而盡。
　這是他所持有的技能《吸血》。
　若對象只不過是人類，那就幾乎無法期望會有多少效果，可如果是魔人的血，勉強可以期待當場的生命值恢復。
。

「骯髒的血……臭得讓人受不了」

。
　他把托瓏的身體猶如像是丟髒物一樣往下一砸，與地面相撞。由於腹部遭受到破裂般的傷害，她的生命已經即將消逝。
　在少女逐漸模糊的視野中，茫然地想著。
。
　為什麼自己會受到這種對待。
　她懂事時父母早已不在，從魔族領的徹底性差別對待與日常性施暴的生活中努力過活。
　喝泥水，也吃豬腐爛的屍體，有時還會進食難以忍受的毒草。
。
　即便逃亡到人類的領土，迎來的也是迫害與討伐。
　無論在哪裡都絲毫沒有她的生存之地。盡管勉強被烏托邦所收留，可等待她的依然是不變的鄙視與宛如對待垃圾一般的待遇。
。
　我到底是為了什麼而誕生的呢――浮現在少女腦海中的是極其空虛的事情，放聲痛哭的悲痛情感不禁油然而生。

。
「居然做出這麼殘酷的事情……喂，小鬼頭，你還活著嗎？」

。
　少女的視野裡映照出“銀色的龍”。
　這就是她所看見的臨終光景――
。

　――本應如此。

。
「喂，張開口。沒聽見嗎，小鬼」

。
　龍硬是撬開少女的嘴巴，把某種東西放進嘴裡。
　隨著感覺到不可思議的口感以及略微甜味的瞬間，本被刺穿的腹部以不尋常的速度痊癒了。
　如果說這是“龍之血”，少女也會信服吧。

。
「誒……」
。

　少女感覺到身體的疼痛消失了，血也止住了。
　這已經不是堵住傷口之類的水平。
。
　而是宛如“時間倒轉”般極為荒唐的光景。龍放進她嘴裡的是遊戲回復道具――卡路里冥土。
　雖說名稱詼諧，不過回復效果卻是優秀的100點生命值。
。
　在這世界上一般藥草之類的回復效果是在1～3之間，甚至就連冒險者所使用的昂貴藥水，其回復效果也幾乎是不滿10點生命值。
　所以當場即時性地回復100點生命值完全是不可能發生的事情。
。
　從他人的角度來看，應該會以為是傳說中無論任何疾病和傷口都可治好，甚至得以不老不死的“龍之血”吧。
　就連歐魯依特也戰慄地觀望這幅荒唐的光景――――
。

「你小子是“新種的龍”嗎……居然有這種事……！」

。
　歐魯依特叫喊著什麼，終於下定決心。
　若不把這條龍「當場」殺死，一旦事態嚴重到他來到魔族領的話，根本無從想象究竟會產生怎樣的大亂。
他以雙手刺進自己身體，猶如在打開門扉一樣撕裂自己的身軀――

。
　另一方面，零確認到少女的傷口癒合之後，放心似地點點頭。
　可少女還是面無表情地散發出陰郁的氛圍。
。

「表情真陰暗呢，小鬼頭。既然已經獲救，笑一笑怎麼樣」
。
「笑一笑……我早就忘了該怎麼笑了……」

。
　少女悲痛欲絕地說道。
　零像是感到困惑般皺著臉面，不過察覺到身後蠢蠢欲動的氣息，隨即便抓住少女的後頸毫無造作地往角落裡丟出去。
　雖說舉止粗暴，不過卻是出自於不讓“小貓”卷入戰鬥的關心。
。
　零回過頭來，歐魯依特已經撕裂自己的身體――“表里翻轉”。
　從中出現的是蝙蝠。而且是幾百只、幾千只、無以計數的蝙蝠形成一個輪廓。
。
　毛骨悚然的飛翔聲與一雙赤紅發亮的眼睛。
　這就是歐魯依特原本的姿態。一旦化為這種姿態，身體能力會有飛躍性的提高，不過由於聖元素抵抗力過於低弱，因此很少轉化為這種姿態。

。
「令人忌諱的“龍”！就讓我來為你這份採取“中立”立場的傲慢進行制裁吧！」

。
　即便承受著歐魯依特露骨的殺意，零的注意力依然是不是放在眼前的怪物，而是身後的少女。
　縱使是零也知道這位少女度過艱苦的人生走過來的。
　因為他也絕不是在優異的環境過活。
。

「笑吧，小鬼。無論是多麼痛苦的時候，像個笨蛋一樣大笑的傢伙都會是最強的啦」

。
　背向少女的零並不知道她對此會露出什麼樣的表情。
　不過只有倒吸一口氣的跡象倒是傳達過來了。

。
「看往別處真是從容啊，龍……你就與這份傲慢一同在此毀滅吧！」

。
　歐魯依特的身體宛如爆發般進行突擊。
　速度極為飛快，以至於響起大氣遭受切割般的呻吟。
　而迎敵的零則是全身爆發出可說是形象顏色的銀色之炎。
。
　龍大叫著。
　以卯足全力的音量大聲地――！

。

『――――天喲，盡管刮目以待吧！《孤注一擲》』

。

　巨大的暗與銀色正面沖撞――相互交錯。
　在宛如世界停止不動的靜謐中，巨大的暗浮現出數道龜裂，最終全身邊發出尖叫聲，邊化為黑色的煙霧消失殆盡。
。
　零施展的技能――是對象殺人數每增加一人，便追加10點傷害的開掛技能、巔峰攻擊。
　並且還是設定為最高峰的上限500點傷害。
　惡魔虐殺無以計數的人類，根本無法抵擋這一招。
。
　零暫時保持著揮出拳頭的姿勢，不久後拳頭便朝著天際舉起，對天空豎起食指。

。
　――――這是堂堂正正的“NO1”宣言。

。
　這是對聚集而來的民眾做出的宣誓。
　見狀，原本鴉雀無聲的群眾都舉起手來，發出吼叫聲。由幾百人到幾千，最終到幾萬人都同樣舉起拳頭發出熱烈的歡呼聲。
　聲音猶如海嘯一般傳達到其他地區，不久後整座神都都被狂熱的漩渦所包圍。

。

「怎麼樣，小鬼頭……想起“該怎麼笑”了吧？」

。

　零難得露出像是不良少年的笑容，笑著對角落的少女說道。
　少女一時之間驚訝得睜大眼睛，不久後便流著眼淚大大地點著頭。或許是已經忍耐不住了吧，她跑往零的身邊，跳著抱住了他。

。
「啊哈哈哈！這不是變得很精神了嘛，小鬼頭」
。
「……我才不是小鬼頭。是托瓏」
。
「托瓏嗎？怎麼說，這名字真是令人想睡覺……」

。
　零如此嘀咕時，背上被某種柔軟的東西所覆蓋。
　是古依紅著臉抱著他。從他人的角度來看，實在是招人喜歡的模樣。
　即便是對小孩子也泰然自若的零也不由得狼狽不堪。他是古風的暴走族，也是已經滅絕的“硬派”男人。
。

「你、你……你在別人面前……！」
。
「我已經不想再分開了――」

。
　古依面露恍惚深情緊貼著他，而胸部的柔軟度卻讓零紅著臉。
　因為在零眼前的古依是完美無缺的美少女，所以從他的角度來看這根本開不了什麼玩笑。

。
「放、放開我！你聽好了，男人啊，在別人面前跟女人是――」
。
「零大人……這背上的龍非常的雄偉……這個胸膛也……」

。
　古依的手指在堅挺的胸膛上滑過，零便驚訝得嚇一跳。
　直到剛才那副英姿颯爽的模樣不知消失到何處。其樣子令數萬的群眾發出洪亮的笑聲，包圍神都的黑暗――已經完全消滅乾淨了。


。

　如此這般，流傳於聖光國之間殲滅邪惡的銀色龍人降臨，與魔王降臨這兩道相反的傳聞終於傳遍整個國家。
　不久後這些傳聞不只是停留於國內，甚至還傳到周邊諸國。
　由一位男子所產生的混亂與巨大誤會，終於波及到整座大陸，事態已經发展到全世界都卷入騷動之中……

。
　這是在不遠的將來所發生的事情――――

。
