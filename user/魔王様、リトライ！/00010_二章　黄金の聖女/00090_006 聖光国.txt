　――聖光國　偏僻地區的領主館

。
　統治附近幾座貧寒鄉村的領主——比利茨.蘭克因久違的喜訊而面露笑容。
　復甦的“惡魔王”居然已經死了。
　原本就無法期待這塊地區會有豐腴的稅收，何況遭受到那只怪物的摧殘，這已經是任誰也無力回天了吧。
。

（時來運轉……）
。

　然後是緊接而來的「魔王降臨」。當比利茨聽聞這消息，他就感覺到命運女神在對著自己微笑。
　無論再怎樣派人快馬疾馳前往神都，傳達惡魔王復活的通報，都皆是音信全無。可如果是魔王降臨――神都也無法再繼續視而不見了吧。
。

（盡可能搞大騷動，把神都也卷進去）
。

　比利茨本人並沒有相信魔王的存在性。
　他覺得這只不過是愚民引起的騷動而已，不過他卻害怕著自己會在這偏僻地區壽終正寢。
　如果發生某種動亂，或許就有可能從滿是封閉感的該死偏僻地區中脫穎而出。

。
（先把一座村莊給完全燒掉吧？）
。

　報告上只說是一間農家起火而已。
　聽到這些，比利茨不禁啞然失笑。
　僅憑這些信息就能明白魔王本身的存在性實在令人質疑。以比利茨的看法，這應該是走投無路的掉隊傭兵放的火。

。
（不過……惡魔王為什麼死掉了？）

。
　它是往昔由智天使大人所封印的存在。
　它並不是人類竭盡全力就能令其為之所動的生物。
　它是類似於地震與台風的“災害”。

。
（是因為不完全性質的復活嗎。果然傳說往往都是可信可無的東西吧）
。

　比利茨打從出生開始就已經是貴族，是位不知勞苦為何物的男人。
　如果台風已經消失，他並不會想到追究台風消失的原因。只不過會以此作為基石，費盡心機地從悲催般的處境中逃脫而已。


（好，就說是這位魔王把惡魔王消滅掉的！）

。
　麻煩的事情就由麻煩的人物來處理，比利茨還以為是個好主意而自鳴得意。
　不過他並不知道。
　他隨意捏造的事實――是不折不扣的“現實”。

　――聖光國　某處
。

　身著修道服的女孩子，發出與那清純的衣物毫不相稱的尖銳聲。
　她如今正乘坐在豪華的馬車上前往偏僻地區，不過似乎是對於馬車的搖晃感到不甚滿意。

。
「你啊，對我的信仰心是不是不夠充分啊！」

。
　聲音使得握住馬車繮繩的車夫縮起了腦袋。
　乘坐在馬車上的是貴人中的貴人――“聖女大人”的其中一位。
　車夫搞不好還真的有可能會遭受火刑。特別是在三位聖女大人之中，位居年齡最小的她是極其任性妄為，無論對於誰而言都是棘手的人物。
。
　不過其外觀――卻擁有與被稱之為聖女相互符合的容貌。
　波浪形的粉色頭髮，宛若櫻花般給人有如楚楚可憐的印象，連眼眸也是淡淡的粉紅色。
　雖說身軀被修道服所包裹，不過雙手雙腳極為纖細，富有魅力。
　話雖如此，由於如今還是個小孩，所以只有胸部那裡是一馬平川。

。
「我呢，如今可是要去討伐傳承中受到世人歌謠的魔王！。在這之前屁股要是磨出痦子，你怎麼負責！？」
。
「非、非常抱歉……因為這附近的道路也沒有修過……」
。
「你……是想受到御政道的批判？」
。
「豈、豈有的事！」
。

　的確，這附近的地面還說不上是道路。
　在神都都是鋪上石板，時而運用魔法鋪好道路。不過貧乏的地域幾乎都是飽受日曬，如果下起雨來，光是排水就已經是一大難題。
。

「不、不過，其他聖女大人沒有勞駕於此？」

「你這句話是什麼意思……想說我一個人應付不了？」
。
「不敢不敢！如果是露娜大人，一個人就綽綽有餘了吧！」
。
「哼，這是當然的吧。我又不可能一直都輸給姐姐大人！」
。

　露娜.愛蕾崗特――16歲。
　常說名字代表內在，不過這位聖女看來是與愛蕾崗特一詞相差甚遠。
(颯:愛蕾崗特=elegant，既高雅、優雅之意)
　馬車周圍還跟著多達25名的護衛隊，不過她並沒有對他們的力量抱有期待――她打算憑一己之力討伐魔王，將功績都占為己有。
。
　事實上，她擁有魔法相關的天賦。
　在與“魔王”戰鬥之際――應該會成為強大的利刃吧。

　――聖光國　前往神都中途
。

　在周遭的火藥味變濃之前――
　既沒有人影也毫無燈光的道路上，“魔王”和小孩子正喧嘩著。
　以一對組合的打扮來說就像是父子，不過似乎有所不同。魔王那邊面紅耳赤，微妙喝了點酒。
　貌似是從過往馬車的親切馬夫上得到了幾瓶酒。

。
「吾之睿智的一角喲，顯現吧――《野營套裝》」

。
　魔王把手伸進漆黑的空間之中，取出一件巨大的物品。
　這是大帝國制的道具。效果正如其名，裡面含有許多野營相關的物品，在遊戲中是必需品之一。
。

「好厲害，魔王大人！還有其他的嗎！？」

。
　反觀，小孩子臉上也微微泛起紅暈。
　不過並沒有喝酒，應該是氣氛的緣故吧。這世界等同於是毫無約束，即便是小孩子也可以喝酒，不過被魔王制止了。
　魔王這種地方莫名有股小市民的感覺。

順便一提，這塊地區的酒較為廉價――反而是水比較昂貴。
。
「吾之“漆黑”毫無止境――《防衛套裝》」
。
　這位魔王得意忘形起來。
　擺出莫名其妙的姿勢，再次取出道具。
　這也是大帝國制的道具。內含許多保護其他玩家的物品，這也同樣是遊戲中的必需品。
　這些道具在獨立狀態下並沒有效果――不過“合成”的時候就能發揮出真正的價值。
。
「道具合成――――《堡壘設置》」
。
　兩個道具仿彿遵從著魔王的命令一般，合二為一。於是，“據點”瞬間就製作完畢。
　“據點”在遊戲中擁有大幅度減少其他玩家造成的傷害，也是能夠安心回復體力的優秀道具。
　如果再往據點添加道具，便能擁有更多的功能。
。
「這是魔法嗎！？太厲害了吧，魔王大人！」
。
　小孩子――不，惡猶如感動至極一般，抱住了魔王。魔王也心情愉快地大笑著。
。
「這可不是魔法喔？惡，就讓我告訴你一件重要的事情吧」
。
　語畢，魔王深吸一口氣。
　然後指著惡，仿彿像是在宣告重大的事情一般，開口說道。
。
「聽好了――沒有什麼是大帝國做不到的！」
。
　魔王撥起長髮，朝著天際高舉拳頭。
　怎麼看都是愚蠢至極的姿勢――不，單純只是個醉鬼吧。
　惡應該什麼都不明白吧，不過還是啪啪地拍著手。
。
「這東西……好巨大。而且硬邦邦的，好堅硬……」
。
　惡說著莫名其妙的事情，不過已經醉醺醺的魔王並沒有注意到。豈止如此，魔王心情越發愉快，還開始自吹自捧。
。
「這玩意連火箭砲之類的《砲擊》都能夠承受，在遊戲中如果沒有這玩意，根本就沒辦法安心睡覺。通過《強化資材》或者《防火壁》強化據點，還可以使據點達到中規模、大規模，防御力也會相應地提高喔」
。
「炮雞、嗎……？魔王大人說的話好難懂……」
。
「嘛，也就是說這玩意很堅固。惡，今天就在這據點裡面睡覺吧。我可是玻璃心，野營什麼的容我敬謝不敏」
。
「好的，家務就請交給我吧！」
。
　我行我素的兩人哼著歌走進據點裡面。
　兩人如今無從得知彌漫於周遭的可疑氛圍。

「據點」
藉由野營套裝與防衛套裝合成製作的道具。
擁有降低其他玩家造成的傷害，保護自己免受砲擊的功能。
砲擊擁有減少SP，以及剝奪防具耐久力的效果。
對此若毫無對策，將招來致命性的事態。
。
正因為是據點，內部具備床鋪、鐵桶澡盆以及簡易的廚房，能夠維持最低限度的生活。
如果據點升級到中規模或者大規模，其內部裝潢也將會豪華無比。
。
到底是以什麼材料構建而成的呢，其防御力連RPG-7或者RPG-29等反坦克火箭筒的攻擊都能抵御。
在這世界上要攻破這座據點，是比登天還要難。
由於遊戲中特有的設計，無論是設置據點，亦或是折疊變小隨時攜帶，都是個人的自由。
