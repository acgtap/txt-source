
　――有名的服飾店 「時尚檢查」

。
　店主對於走進店裡的兩人投以尖銳的視線，只用一瞬間就從頭到尾地檢視一遍。
　黑色的頭髮，從未看見過的衣服――完全就是外國的人民。
　雖說如此，這座城鎮對於外國人也是極為寬容。正因為是以交易為主流的城鎮，所以這種事也是理所當然。不過即便是從氣質上講，這座城鎮也算是朝氣蓬勃。
。
　不過，讓店長蹙眉的原因――是跟在他身後的小孩子。
　身穿與乞丐僅有一步之遙的簡陋衣服，與前面的男子相比，難以抹滅一種不平衡感。

。
（奴隷嗎……？）

。
　即便再怎麼說是外國人，帶著奴隷還是有些不妥。
　原本應該是要讓奴隷待在城鎮外面，或者交給別人照顧，不然就是別搞得太引人注目才是最低限度的規矩。
　這位客人的裝束非常不錯，難道不知道這方面的潛規則嗎？
。

「歡迎光臨，請問今天光臨本店有什麼需求嗎？」

。
　男人面露和藹可親的營業式微笑，搭話說。
　只能在對話途中暗中試探，委婉地勸誡了吧。萬一連這邊都遭受波及那可受不了。
。

「抱歉，準備一些適合這孩子的衣服吧」
。
「怎、不……是這邊這位客人，沒有搞錯吧……？」
。
「嗯？就是這樣」

。
　為奴隷買衣服？
　難道是要享受這種play嗎。買一些好衣服穿上去，然後再盡情撕破……不，我沒必要想這麼多吧。
　生意就是生意，既然會付錢也沒必要刨根問底。剩下的只要確認這小孩子是不是奴隷就好了。

。
「我不擅長幫人選衣服。就交給你們這些專家來做吧」
。

　語畢，男人從懷裡拿出皮袋。
　從外觀上來看，似乎挺沉甸甸的。不禁吞了一口水。
　或許這位客人會弄丟一枚金幣給我吧？這種猥瑣的預感在心中一掠而過。

。
「這是最大的吧……就拿這個」
。

　男人抬頭仰望著若無其事拿出來的東西。
　僅僅是拿出來，都能讓店裡明亮到耀眼的――「大金幣」。

。
「什……這、這位客人……您、您要用這個……？」
。
「嗯……？不夠嗎？」
。
「豈有的事！現在，立刻，馬上為您準備！」

。
　把全部員工叫來跟隨著這位稀奇地觀望各處的“高貴大小姐”。以防萬一，還指示他們把今天休息的員工也叫過來，同時還要準備飲料。
　如今或許是人生中頭腦運轉最快的一次也說不定。
。

「還有，我想要吸煙來著，給我個煙灰缸」
。
「請抖落在這手上！請務必把煙灰抖落到這雙手上！」
。
「好怕！我說的不是這個意思，是普通的煙灰缸，還有哪裡有椅子坐」
。
「請坐在我背上！請務必坐在我的背上！」
。
「好可怕！什麼鬼，這家店！？」
。

　在兩人無限循環這種事的期間，惡在店員的陪伴下，被許多從未看見過的衣服所包圍。

。

　■□■□

。

　店內洋溢著前所未有的活力。
　連休息的員工也出動了，五位店員慌慌張張地來回跑動。
　他們糾纏著惡，把這邊那邊各式各樣的衣服、首飾、帽子等都拿過來，引起不合時宜的大騷動。
。
　這也難怪，畢竟魔王拿出了――「大金幣」。
　大金幣是除了智天使遺留的為數不多「拉姆達聖幣」之外，價值最高的貨幣。除非是代表國家的大商會人士，否則不可能見識到真貨。
。
　店長发憤圖強，提出搬來的衣服若是令惡喜歡就會有一枚銀幣的額外補貼。於是店員們也完全點燃鬥志了。
　魔王並不知背後發生的事情，他望向窗外的景色，看到大街上生氣勃勃的風景之後，便鬆弛臉頰。
。
　看樣子他並不討厭熱鬧的光景。
　似乎有種深切感覺到自己來到異世界的實感。等了一會兒，後面的簾幕打開了，惡的身姿也隨之出現。
。

「那、那個……適、合我嗎……？」
。
「哦、哦……」
。

　身著純白禮服的惡，令魔王暗中倒吸一口氣。
　在奇妙的地方顯得粗枝大葉的魔王，連惡的性別都不知道就隨便進了一家店。
　於是，出來的是一位――嬌小玲瓏的公主大人。

。
「請問如何，這位客人……如果可以，也買點其他東西」
。
「也、也也也是啊……合身的東西都全部買下來吧」
。

　已經深陷動搖的魔王把煙灰抖落這地面上，不過店長反而是更動搖的一邊。

。
「您、您說的是全全全部嗎……這位客人……」

。
　惡聽到這句話想要說些什麼，不過店長的聲音比她來得還要快。

。
「大家！快點！給大小姐準備最好的衣服！」
。
「「是！」」
。

　如此這般，結果把無以計數的衣服、鞋子等都連續不斷地買下來。這是由於魔王無論看見什麼都會說「合適」的緣故。
　也確實，惡無論穿上什麼都很適合。
　不僅僅是可愛的衣服，就連穿上男式的衣服也顯得合身，因此店長的表情可謂是滿臉賺到。
。
　買下的商品連續不斷地送往旅館，連旅館的老板都驚訝得目瞪口呆。
　不過再怎麼說也是住在最高級旅館的“大富豪”，他覺得這也是理所當然般點著頭，為大客戶的到來而情不自禁地擺出勝利姿勢。
。
。
　於是魔王與惡轉眼間就形成傳聞，什麼「他國的王族」「北方的大富豪」等等，在這附近愈發出名。
　若讓魔王聽聞應該會歡聲大笑，若讓惡知道應該會嚇得暈過去吧。


。
　■□■□

。

　當兩人結束購物，便來到旅館古古雷隔壁的西餐廳。
　魔王並不是身穿以往那一套衣服，而是白色襯衫配上黑色的晚禮服。
　胸口的衣袋裡也裝進袋巾，完全就像是貴族。惡也身穿首次嘗試的純白禮服，成為一位奪走人們目光，楚楚可憐的少女。
　店裡的客人以不會令人感到厭惡的程度望向兩人，聊起話題。

。
「這些衣服很少見呢，北方流行的嗎？」
「我有在都市國家郡那裡看過類似的衣服喔」
。
「那兩個人是父女嗎？」
「不，不會吧。氣氛不一樣」
。
「據聞是小國的王族」
「我聽說是他們非常了不起的富豪呢。總之應該是在觀摩買賣吧？」
。
「據說他為了那位少女的衣服就用上了大金幣？」
「看來是十分溺愛那位少女啊……」
。

　在此起彼落的隨意談論中，兩人來到餐桌上坐下，打開菜單。
　魔王適當地點完菜就交給男服務員，並吩咐他拿瓶最好的葡萄酒過來。其行為舉止沒有絲毫顧慮。
　他這副樣子更加深周圍人對他的誤會，不過本人並沒有察覺到這種氣氛，實在是我行我素。
。

「剛進來就要最好的葡萄酒嗎」
「說不定不是商人，而是礦山的老板吧」

。
　魔王不顧這些閑話，緩緩為香煙點上火。
　雖說這世界也有雪茄煙，不過那是展示體面的交易品，也是一支價值不菲的高級品。
　他所抽的煙草是從未看見的白色，僅僅如此就已經引人注目。
。
　他在眾目睽睽之下喝了一口葡萄酒，說了一句「不合口味」。令人疑惑他平時到底喝的什麼呢。
　而且不知道他是想到什麼，又重新叫來一瓶啤酒，喝完之後還面露笑容。
。

「嗯，立麥(上班族)果然還是這個吧。第一杯還是這個最好」
。

　雖然沒聽說過立麥這個詞，不過應該是地名吧。客人們在腦海中隨意想著。
　畢竟北方有將近十個國家正處於群雄割據的狀態，東北還有都市國家郡。
　從聖光國的角度來看，其他國家如今還是野蠻的――亂世時期。



　■□■□

。

（裝腔作勢也沒有意義吧……）

。
　“我”喝了口啤酒轉換口味之後，把搬上餐桌的肉切出一小塊，放進口中。這塊肉相當不錯，跟中午吃的肉截然不同。
　其實我是想吃喝遊戲中經常使用的大帝國制食物，不過我得盡可能避免消費SP。

。
「真的就像是、做夢一樣……」
。
「……嗯？」
。

　往前一看，惡並沒有動手吃著擺在餐桌上的料理。
　她身穿純白的禮服，頭戴宛如王冠般的頭飾，看起來仿佛就像是某處的公主大人。

。
「非常感謝。對我這種人，付出這麼多……」
。
「別在意。比起這件事來，趁飯菜還沒涼之前吃掉吧」
。
「我會――在意」
。

　往惡那邊看過去，她那強而有力的眼神不禁讓我停下雙手。
　左右不同的眼瞳――赤紅與碧色之光，氣勢不禁被壓倒。僅僅是盯著看就會感覺到某種神秘感，我情不自禁地就移開視線。
。

「為什麼對我這麼好？」
。
「……我沒打算對你這麼好」

。
　實際上我也沒有對她這麼好的打算。
　如果要說我做過的事情，無非是虐殺襲擊過來的怪物，在逃跑的時候放了點火，對聖女做出體罰行為並搶奪金錢罷了。
。
　重新想一想，自己還真是過分。
　一想到自己的所作所為，都不禁要把啤酒噴出來。雖說是正當防御，不過即便被人說鬼畜也是無可厚非。

。
「我不知道怎麼做才能償還這份恩情」
。
「其實你也沒必要回報我。歸根究底，我本來就沒打算施恩於你」

。
　說完，我連續不斷地把肉放進口中。
　冷肉跟溫啤酒是我在這個世上最討厭的東西。
　特別是溫啤酒。我最喜愛的酒居然會成為這世上最難喝的飲料，甚至讓我覺得人生真是變化無常。

。
「魔王大人，無論是什麼吩咐都可以。只要是我力所能及的事情我都會做」
。
「嗯？剛才你好像是說什麼事都會做吧？」
。

　話音剛落，一道巨響從店裡門口傳來。
　豈止如此，大門都被踢飛了。
。

「――我找到你了！魔王！」

。
　前些天遭受到體罰的聖女，如今正滿臉通紅地站在那邊。
　總覺得她氣喘吁吁，正興奮著。
　因此，我的回答就只有一句。
。

「又是你嗎。我下單的時候可不記得叫了份跟踪狂來著――」
。

　第二次聖女大戰就此開幕――
