
　城鎮的街道上搬運著受傷人士，急忙地進行修復工作。
　然而平時都會帶起頭大聲吶喊的古依，卻坐在椅子上發著呆。
　完全沒有指揮修復工作的意向。
。
　雖說她這幅樣子不成體統，不過福吉看到她卻露出溫柔的微笑。因為這個國家的男人無論是誰都無法令她感到滿意。
　他只擔心敬愛的大姐頭照這種情況下去會變成一位剩女。

。
（話說回來，真是個“勇猛的男人”……）

。
　在惡魔信奉者的集團面前也絲毫沒有退縮之意，在福吉眼中還看見“奈落”甚至都仿彿在避開那個男人。
　奈落――被冠以此名的存在並不是代指那種液體。而是在指存在於大陸中央的巨大洞穴。
。
　北方諸國正處於群雄割據的戰爭狀態，從中產生的屍體都開始將投放於這個洞穴之後，這名稱便由此而來。
　埋葬屍體需要花費時間、占據土地、費用也不可輕視。
。
　於是各國為了節省功夫，便決定把所有的屍體都全部投放進傳聞中連通地底的大洞穴。
　正因為是黑暗便往黑暗丟，這也是世人常言道的字句。
　據聞從這洞穴之中時而會有毛骨悚然的液體爬出來，進而襲擊人類。
。
　其中更有稱之為百年一次的機率才會出現的“黑色液體”，這液體會奪走神聖的力量，是所屬於天使之者的天敵。

。
（至少那個男人不是所屬於天使大人的麾下）

。
　福吉雖這麼思忖，不過也正確無誤。
　若他是持有神聖的力量，又或是擁有所屬於天使力量的人類，那必然會被奈落奪走力量吧。
　不如說他豈止是天使――只不過是單純的“暴走族”。
　那就將會是擾亂治安與秩序的人物。
。

（無論如何，這男人非得招聘到我國不可……）
。

　雖然福吉思考著這些事情……
　不過那男人與魔王是――表里一體。

。

。
「然後呢，那個龍人一往地面揍下去，人都飛上天了哦！就噗哇的！」
。
「嘿~，真厲害呢！」

。
　自那以後魔王等人便離開了亞侯之城，不過露娜以仿彿親眼目睹昨天戰鬥般，自滿地敘述道。
　事實上她也失去意識了才對，只不過是惡單純地對此深信不疑。
　當然，魔王聽聞這些只是露出扭曲的嘴臉。

。
（什麼龍人啊……那純粹只是暴走族吧！）

。
　魔王胸中的羞恥心令他有股有洞就想鑽的心情。明明身體就無法動彈半分，僅有曖昧的意識也應該無力回天吧。
。

「魔王，你或許也會被那個龍人揍飛呢！真不錯呢！」
。
「好的好的……」
。

　魔王適當地應付露娜說的話。
　這是不可能發生的事情。
　因為兩個人無法同時存在。

。
「話又說回來，露娜，你帶過來的騎士怎麼了？」
。
「嗯？因為不需要我就讓他們回去了」
。
「你啊……」
。

　可謂是國家頂端人士的存在單獨行動著。
　從魔王的觀念來看，這是不可能發生的事情。昨天不就發生了襲擊和暗殺事件嗎。

。
「要是發生什麼事你打算怎麼辦啊」
。
「……不、不是還有你在嗎」
。
「哈？？什麼時候我變成你的保鏢了」
。
「吵死了！明明呆在我身邊就很高興！這個變態！」

。
　聽到這句話，魔王額頭上浮現出憤怒符號，不過坐在兩人之間的惡卻發出笑聲。

。
「熱鬧起來真高興呢！我一直想要度過這樣的旅行！」
。
「真的假的……」
。
「你看，惡不也這麼說嗎。你就為了我幹活啦！」

。
　露娜很偉大似地用指著魔王，魔王嘆了口氣。
　他暫時沉默了一會，或許是忍住怒氣吧，開始製作著道具。

。
「畢竟也有馬車這件事，跟著我們到王都也可以吧……不過你這打扮也太引人注目了。要讓你換套衣服喔」
。
「換、換衣服……你、你是打算要讓我穿上下流的衣服吧！」
。
「白痴嗎你。把這衣服穿上」
。
「什麼啊、這……話說你到底是從哪裡拿出來的呀」
。

　魔王拿出來的是《西服》。
　即便走在街道上，這件衣服應該也不會有那麼大的違和感吧。其他還有水手服、女僕服、旗袍等多數防具，不過這些東西當某天拿出來之時就免不了要遭受名為變態的誹謗。
。
　即便令她穿上這套衣服，以這世界上的普通衣服而言也著實擔心防御力。
　似乎要下定消耗SP的苦澀決定了。
。

「防御力10雖說不高，不過如今還是優先隱藏聖女的身份」

「10……？你啊，偶爾也會說些奇怪的事情呢」

。
　魔王對此並沒有多做回答，繼續製作著道具。
　那便是――《御手洗團子》。
　在遊戲中是具有恢復50氣力的消費道具，不過在這世界上是堪稱神藥級別的物品。

。
「惡，你馬車坐不慣也該累了吧？可以吃這個休息一下。這是我國家的點心」

「哇哇哇……非常可愛的點心呢！而且有股甜甜的香味……魔王大人，非常感謝！」
。
「好、好像很好吃的樣子呢……我也要吃」
。
「畢竟還有六串，隨便怎麼吃都可以啦。我稍微出去外面抽條煙，這段時間你就換好衣服吧」

。
　魔王來到馬車的外面，天空是一片晴朗。
在蔚藍的天空之下，他以嫻熟的手法開始抽著煙。
。
　（再不從哪裡賺錢SP的話……）
。
　現在剩餘的SP只有60而已，實在不多。
　雖這麼說，其實寵溺著惡與露娜也應該是原因之一。
。
（如果跟遊戲一樣，只要戰鬥就能得到SP……）
。
　魔王腦海裡最先想到的，就是幻想世界中必有的冒險者。
　實際上這世界上也存在冒險者，既然可以完成各式各樣的委託，也就可以退治魔物，也有像是探險者的人。
　若前往北方，由於戰爭不停的緣故，大多數無法過活的人都會成為傭兵。
。
（真正被認定為魔王我也受不了，不然應該干份正經的工作？）
。
　根性是上班族的“大野晶”浮現出這種思考方式。
　另一方面，他也悲傷地嘀咕著「來到異世界還要工作嗎」什麼的，實在令他心中複雜。
。
「聖女大人，這點心好甜！」
。
「惡還真是小孩子呢。獨當一面的淑女對於點心……好美味！」
。
「總覺得連疲勞都全部消失不見了！」
。
「那個變態……給我們撒下誘餌，意圖做些下流的事情呢！真不甘心，不過我要吃掉了！」
。
　魔王聽見馬車內傳來的騷動聲，嘀咕道「這是什麼小短劇啊」，不過那超群的聽力又捕捉到其他的聲音。
　那是道粗野的人聲，以及武器聲。凝神一看，女性兩人組被疑似狼的生物所追趕。
。
（這真是求之不得的機會啊……）
。
　他自然而然地嘴角上揚。那正是與魔王相襯的表情。
