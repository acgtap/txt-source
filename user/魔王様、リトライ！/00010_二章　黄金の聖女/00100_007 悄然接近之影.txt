
　――圣光国　前往神都途中
。

（这就是洗澡……！）

。
　恶从走进据点开始，就不停地感到震惊。
　最先感觉到与自己家裡根本性的不同――就是“坚固性”。譬如说，纵使熊和猪冲进来，据点是不是依然稳如泰山呢？
　即便对象换做是凶恶的魔物，感觉也不可能冲破这裡。
。
　据点之中还配备有床铺和厨房，终于有种自己完全就是井底之蛙的感觉。
　在家裡顶多也就是用简陋的毛布铺在稻草上，然後睡在上面而已。
　与其相比，这裡简直就像是天国。

（值得一提的，就是这个铁桶浴――！）

　听魔王大人所说，这似乎是叫做「铁桶澡盆」。
　令人吃惊的事情是，这裡面放进了“热水”！奢侈到无法相信的地步。即使我已经泡在裡面，身体也止不住颤抖。
。

（不过，魔王大人为什么要道歉……）

。
　正确的来讲，应该是「抱歉，现在这座据点就已经是极限了」来着？
　老实说，我完全听不懂这是什么意思。
　如果是想说这已经是“极限”的话，那是理所当然。因为我根本就想象不出还有什么比这更加奢侈的事情。
。

（魔王大人究竟是来自哪裡呢？虽然我也想过是魔界之类的地方……）

。
　有点在意魔王大人喝醉的时候说过的“大帝国”。
　那裡有可能是魔王大人的故乡也说不定。
　当我想着这些事情，就听到魔王大人的声音从门对面传过来。

。
「抱歉，地方太小了。不过你给我等着――迟早我要设置“温泉旅馆”」

。
　魔王大人仅仅说完这些，便发出脚步声离开了。
　魔王大人说的话非常艰涩难懂……如果我更加聪明，或许就能知道是什么意思，不过没有的东西强求也无济于事。

。
（话说回来，什么是文圈驴广？）

。
　即使恶浮现出单纯的疑问，铁桶浴依然让恶感到十分满足――
　这是非常和谐的光景。


。
　另一边――
。
　魔王侧身躺在床上，眺望着窗户外面的夜空。
。

（这个世界也有月亮啊……）
。

　像这样呆一会儿，应该会逐渐醒酒吧。
　来到这个世界已经过了相当长的时间，不过并没有发生回到原本世界的自然迹象。

。
（如果权限全部都恢复了，真的就能回去了吗……？）
。

　至少管理者权限是游戏中的相关系统，所以当然不存在「回到原本的世界」的指令。
　如果我是由那座像是邪神的石像召唤过来的，那就代表我只能寻找类似的人物，拜托它送我回去吗？
。

（最大的问题应该是，就算我回到原本的世界，时间段是否会是一样吧？）
。

　如果能回到那一天那一刻，那倒没什么问题。
　可如果在这裡度过的时间跟原本世界的时间是同样在运作的话，那将是致命性。
。
　假如说我一个月之後回到原本世界，本应作为失踪案件处理的我，肯定会引起巨大骚动。「我去了趟异世界，嘿嘿」这句话也未必行得通。
　到时候，我应该会被送进精神病院，长期被关在付有铁栏的病房裡。
。

（现在应该先以恢复全部权限为目标吗？）

。
　不过为此需要什么，我仍然不清楚。
　只要单纯积累SP就好了吗，还是说赚钱就可以了，又或者是收集道具呢。
　无论如何，继续漫无目的下去也不会有任何改变。那还不如想着可以旅行了，抱着期待过活还比较有建设性。

。
（就算回去了，也是日复一日的工作么……）
。
「魔王大人~！铁桶浴实在太幸福了呢！」

。
　听到这句话，我喷了出来。
　又不是战争时期，就算你对我说这种洗澡很幸福，我也不知道该露出什么样的表情。
　虽说因为某些因果关系而一起旅行……不过管理机能要是恢复了，做些让恶大吃一惊的事情或许也蛮不错的。
　毕竟恶直到现在，似乎都没有度过幸福的日子。

。
（温泉旅館、吗……既然要做，干脆也把“不夜城”复活吧？）

　不夜城――是游戏中以“九内伯斗”为首的全部亲信所坐镇的最终领域。

。
　铁壁般的最终防御据点不仅仅只有亲信在保卫着，还有两千多名部队。
　如果让恶见识到集结了大帝国全部科学力量的不夜城――究竟会露出什么样的反应呢？
。
　不夜城度过了十年以上的漫长岁月，无数次击退了玩家，也因此是个沾满鲜血的地方。不过只要有不夜城，就能够完全确保人身安全。
　因为不夜城在过去从未陷落过。
　何况其他亲信如果也来到这个世界，行动范围也会大幅扩大吧。

。
「魔王大人，今天我也用肥皂把身体洗干净了」
。
「嗯……的确，有股香味呢」
。
「真的吗？诶嘿嘿……」

。
　恶很高兴似的笑着，接着钻进被窝裡。
　因为恶的头发是金髮，就像是毛色不错的小猫，有种令人怀念的心情。
。

「你啊，打算跟我一起睡吗」
。
「不行吗？」
。
「你好像是把我这种大叔当做是父亲一样，不过我有件非常重要的事情要告诉你――听好了，真正的我其实是“哥哥”。我再说一遍，我还不是大叔」

「等一下魔王大人，我听不懂您在说什么」
。

　如此这般，魔王做着无谓的挣扎，就这样度过一晚。

　几天後――圣光国　山中

。
　山中有将近四十名“山贼”在蠢蠢欲动着。
　他们听闻圣女乘坐的马车将会通过这条山麓。所以山贼们不会放过这绝妙的机会，做好袭击的准备。
。
　这一带有被称之为土龙――遭受人们忌讳的一帮人。
。
　他们不会选择猎物。
　无论对象是多么厉害的强者，亦或是多么柔弱的弱者，他们都会一视同仁地进行袭击。
　时而也因此受到惨痛的反击，不过其总人数比起减少的数量，反倒是增加的人数比较多。这也着实证明，这国家是多么混乱不堪。
。
　首领率领着这个不惧生死的集团，坐在树桩上傲然盯着山麓。
　他从十岁起就干着抢劫的勾当，如今已然是驰名于附近一带的山贼首领。虽说年龄高达51岁，不过体格依然健壮如牛，与年轻时期别无二致。
　正当他缓缓倾斜着酒瓶之时，便看到山麓裡的人影。
。
　――不是那些猎物。
。
　他立即得出判断，不过又想到其他的事情。
　两个人马车也不坐，就在这种堪称荒野的偏僻地区上走着。
。
　――陷阱吗。
。
　如果我们盯上这份人数不多的美味诱饵，背後却有本队进攻过来――
　回过神来已经被团团包围，只能连滚带爬地逃之夭夭。这种事也是常有的。
　这时候，原本无法掌握圣女目的的他，终于也明白过来了。圣女是打算引诱我们出现，再借此机会完全歼灭。
。

「……呲，还真是被小看了」
。
「首领，那两个人要怎么办？」
。
「杀掉――在背後的本队赶来之前撤退」
。

　正因为首领在这条路上生存已久，所以才要迅速地下定决断。
　在紧急关头根本无暇感到迷茫，因为如果不做出行动，立即就会丧失性命。已经有许多同伴就因为关键时刻不知所措，导致失去生命。
　手下们静悄悄地行动起来，把弓箭搭在弓弦。
　全员同时释放弓箭――那是对于杀人丝毫不感到踌躇的连串动作。

。

（真是粗劣的攻击……）

。
　魔王看见飞过来的弓箭，叹了口气。

　――什么力量都没有。

　如果是在游戏中，属性技能《雨箭连击》就能轻松地向我发射出几十支既死级的弓箭吧。从魔王的角度来看，这只不过是朝着过往的路人随意射出弓箭的难看攻击。

（要用身体硬抗下来吗――？）
。

　魔王下定决心，决定要扛下来。
　他想要尝试一下，这究竟会对自己造成多大的伤害。自从与那只恶魔遭遇之後，由于没有任何敌对者，也因此没办法正经地进行实验。
　这是一次绝妙的机会。
。
　以防万一，魔王已经取消了自动迎击和自动反击的效果。
　既然不知道这究竟是基于什么缘由触发效果，那么也就没办法加以信任。
。
　搞不好会对于毫无问题的行为进行可谓是致命性的反击。如果反击的对象是怪物那倒还好，可如果是人类，麻烦就大了。

「恶，退下」
。
「……诶？」
。

　飞过来的弓箭，在魔王眼裡仿佛就像是停止不动一般。
　这也是当然――因为他在游戏中曾经就躲过突击步枪发射的子弹，甚至就连散弹枪发射出来的散弹都躲开了。
　飞过来的弓箭终于迫近到魔王眼前，不过在刺进身体之前就微微响起电子声，弓箭也无力地掉落在地面上。

「哼，不满LV30吗……真是垃圾」

。
　说到底，即便有等级达到上限的玩家，也无法跟魔王进行真正的战斗吧――他就像是RPG游戏的最终BOSS一般，如果不使用各类道具都无法给与一定程度上的伤害，而且其体力也实属异于常人。

「明明打算要杀人，却还要居高临下吗――？」

　听到魔王这番话，山贼们终于现身了。
　它们的表情上无不是惊讶与迷惑不解。
。

「你这家伙，到底做了什么……魔法师吗！？」

「别给我太得意忘形喔。魔法使一旦用完气力，就只不过是条靶子！」

「嚯，靶子呢……还真是听到一件不错的事情」

「如果你们之中有哪位是魔法师的话，请务必来一发让我见识一下」
。

　听到这句话，山贼们轰然吵闹着。
　这已经不是被人小看的级别――而是某种以昆虫为对象进行各种实验般的对待方式。

「喂，大叔。你还挺能吹的嘛」

「谁是大叔啊――！」

「魔、魔王大人！太危险了……我们逃跑吧！」
