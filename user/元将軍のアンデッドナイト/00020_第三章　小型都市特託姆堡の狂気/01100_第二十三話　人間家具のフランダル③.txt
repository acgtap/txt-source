「雖、雖然不知道你是誰，不過還是謝了。閣下到底是……」
。
被弗蘭達爾的人間家具『機』踢倒在地的亞爾安特對突然出現的救命恩人——蘭貝爾表示感謝。
亞爾安特剛才已經深深地體會到弗蘭達爾的人間家具的頑強生命力。
即使被強烈的毒物侵蝕，即使腦袋被削去一半，這些怪物也能夠在短時間內復活。
而能夠將這種怪物游刃有餘地斬殺掉的蘭貝爾無疑是個非同尋常的劍士，亞爾安特如此確信。
。
「你……似乎不是一般的貨色呢。竟然，這麼輕鬆地就將我的『機』一刀兩斷」
。
弗蘭達爾坐在『椅子』上，眺望著將人間家具的尖兵『機』斬殺的大鎧之男。
大鎧之男——蘭貝爾仍手執大劍，同樣觀察著弗蘭達爾。
。
「嘛，不用這麼生氣。你打倒的『機』，是作為天才藝術家的我所創造的數一數二的銘品。而你玷污了我的藝術，應當千刀萬剮……平時的話我會這麼說，但這次我就寬恕你好了」
。
弗蘭達爾伸手制止蘭貝爾，閉著眼搖了搖頭。
那副動作就像是在說，自己沒有戰鬥的意思一樣。
然而，在下一瞬間，弗蘭達爾便開始仰天大笑，連牙齦都一覽無餘。
。
「你真是太棒了！我感覺到了！氣質、榮譽、威嚴，樣樣俱全！而且那魁梧而優美的身體，我了如指掌！你啊，是為了成為我的家具才誕生到這個世界上來的啊！你理應成為我的收藏品！不，不對！你希望成為家具！沒錯，一定是這樣！」
。
弗蘭達爾睜開雙眼，敞開雙手並開始了熱情的演說。
他的眼中只有瘋狂。
。
「怎樣？你也希望成為我的收藏品之一吧，吶！？吶！？」
。
弗蘭達爾唾沫四濺地大聲叫喊著。
被弗蘭達爾的瘋狂嚇壞了的亞爾安特刷白了臉，僵硬得一動不動。
亞爾安特和其他『死之天使』的成員一樣，雖然將弗蘭達爾當成不能理解的存在，但在現在他終於認識到，弗蘭達爾到底是有多麼地瘋狂。
將人類改造並稱之為『家具』的弗蘭達爾，他那異常行為的基准點，現在正顯露在他臉上。
他堅信人體改造是藝術，亦是自己理應所做的工作。
。
在亞爾安特被弗蘭達爾的氣勢壓倒的時候，蘭貝爾仍從容不迫地聳立著。
。
「看來我們之間無法交流，不過打從一開始，我就沒打算跟你們多聊，動手吧」
。
蘭貝爾踏出了一步。
只因這個動作，弗蘭達爾那瘋狂的表情就產生了動搖。
縱使直視著弗蘭達爾那邪惡的本質，蘭貝爾仍若無其事地向前進。
。
弗蘭達爾用手扶著額頭。
。
「我竟然會被壓制？我對高尚藝術的渴望，竟然敗給了他的威壓？不可能……這絕對不可能！」
。
弗蘭達爾的雙眼中，再次染上瘋狂的顏色。
。
「去吧、『棚』！給我抓住他！」
。
臉龐被綳帶包裹著的無手的臃腫女人，往蘭貝爾的方向衝刺。
蘭貝爾也手握著大劍，一往直前。
打算在將『棚』斬殺後順勢擊殺弗蘭達爾。
。
(比先前的尖兵還要遲鈍……。也許肉會比較結實，不過這種程度的話也沒問題）
。
蘭貝爾為了將身形龐大的『棚』一擊必殺，在手臂上積聚了力量。
。
「的確是完全成熟的劍士的動作……！遠遠地超越了我所認知的人體的極限。但是，決定勝負的不只是力量和速度！」
。
「噢咕哦哦哦哦哦哦！」
。
在蘭貝爾的射程的兩步以外，『棚』的身體上突然張開無數的孔，發射了大量如矢尖般細小的物體。
尖銳的金屬暴風襲擊了蘭貝爾。

『棚』的身體千瘡百孔，瞄準著面前的蘭貝爾的性命。
。
「哈哈哈！因為裡面放了很多東西所以才叫做『棚』啊，嚇了一跳吧？本來是打算盡量不傷到你的身體的，但安然無恙地把你抓住什麼的，怎麼想都是不可能的吧！」
。
蘭貝爾稍往後移並神速地揮舞大劍。
那速度令人目不暇接。
大劍之刃將迫近而至的彎曲金屬片一一打亂。
零碎的金屬片紛紛散落在地。
。
「這、這到底發生了什麼……？」
。
弗蘭達爾無法接受眼前的景象，驚訝地自言自語。
。
發射出最後一塊金屬片的『棚』也只能呆呆地站在原地。
蘭貝爾再次舉起大劍。
目的是將『棚』和『機』一樣，從頭到尾一刀切斷。
。
突然，從『棚』那肥大的身軀中深處了兩把巨大的刀刃。
一把從胸膛中，一把從下腹中。
兩把一組的刀刃削去血肉，猶如斷頭台一樣向蘭貝爾發起襲擊。
。
(發動了『斷頭鉗』嗎。啊啊，明明是個好素材，這樣子的話連家具都做不成了呢……）
。
弗蘭達爾望著從『棚』中伸出的刀刃，遺憾地嘆了口氣。

『棚』最後的手段——『斷頭鉗』，是在『棚』被逼入絕境時，自削血肉，自動發動的武器。
。
它的強大之處是，以遠超人類反應的極限的速度迅速斬擊，令對手確實地死去。
無論是多麼強大的人類，也只會在剎那之間殞命，連殺死自己的凶刃也無法看清。
因此為必殺之一擊。
絕不會偏離，死神之鐮的一揮。
。
兩人的身影交錯。
而後，漆黑的血肉四濺。
。
「噢噢噢噢噢哦哦哦」
。
『棚』那肥大的上半身發出奇怪的叫聲並飛到空中。

『斷頭鉗』的巨刃的碎片散落一地，對弗蘭達爾來說慘不忍睹。
。
「……哈？……咦？」
。
蘭貝爾才剛剛回避了必死的一擊並反過來殺死了『棚』，然而他卻不以為意，保持著同樣的速度沖向弗蘭達爾的所在之處。
