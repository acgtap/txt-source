身穿鎧甲的大漢，蘭貝爾正平淡地進入拉戈爾子爵公館中。
他的足跡，形成了一條血路。
這當然不是蘭貝爾自己的血。
而是斬向非法入侵者的拉戈爾子爵的私兵的血跡。
。
將棘手的照看孩子工作強推給克羅伊茨的蘭貝爾，正如他所宣誓的那樣闖進了拉戈爾子爵公館。
三名私兵與蘭貝爾對峙，阻擋在前。
然而，私兵們的劍尖正在戰慄。
他們切身地體會到，蘭貝爾將一人又一人的私兵斬殺，並不斷往這邊接近的事情。
單靠他們三人是絕對阻止不了這名狂劍士的。
。
起初這三名私兵也嘲笑了這名無謀的入侵者。
原本就膽小的拉戈爾子爵自從和『吹笛之惡魔』接觸之後，考到到被卷入鬥爭的可能性而擴大了私兵團，同時也增強了公館的警備。
而居然有人單槍匹馬地闖入這個地方，無非是自 殺的行為。
。
然而，警備的士兵，都已經成為了蘭貝爾的刀下亡魂。
。

「這、這個怪物是怎麼回事！」

。

「你們兩個想辦法吸引他的注意力！趁這個機會，由我來殺了他！」

。

「了、了解，索多姆(ソドム）隊長！」

。
站在後方的私兵大喊時，盡管對蘭貝爾感到畏懼，前方的兩人也頷首以回應。
。

「拉戈爾在哪？快說。拉戈爾是個賣 國 賊。即便如此，你們也要對他表示忠誠的話，那就別怪我不客氣了」

。
蘭貝爾平淡地宣言。
。

「真是可笑！我們已經回不了頭啊！」

。
站在陣型後方的名為索多姆的男人向蘭貝爾大吼。
前方的兩人兵分兩路，對蘭貝爾放出斬擊。
。
(無論你有多強……從三個方向同時攻擊的話，也必然會出現空擋。我眼神很好，是絕對不會放跑這個空擋的……！）

。
比兩人稍遲，索多姆從正面撲向蘭貝爾並伺機而動。
他睜大雙眼，準備追上蘭貝爾的動作。
。
(靠這雙眼睛，我才爬上拉戈爾子爵大人的私兵隊長這個位置。你的確很強。但今天就讓我告訴你，弱者也有弱者的戰鬥方式！即使你斬殺其中一人，我也會趁這個空擋將你殺死！）

。
索多姆全神貫注盯著的蘭貝爾的大劍突然消失。
。
(嗯，在哪……）

。
下一瞬間，在索多姆的胸部上出現一道閃光。
隨後胸骨折斷，骨肉一同斷裂。
。
(為、什麼……）

。
在索多姆正要失去意識之際，他恰好看見自己的兩名部下分別被縱向和斜向切斷、鮮血從切斷面中噴湧而出的一幕。
。
。
在勢如破竹的蘭貝爾走出大廳之時，他迎來了一大排整齊的士兵。
數量將近四十。
站在私兵中央、身型與蘭貝爾不相上下的扛著巨斧的壯年大漢，用鼻子輕輕吭了一聲。
。

「真是意外。聽說有人闖進了鬧事我才出來看看，沒想到居然真的只有一個人呢」

。
斧之大漢，是拉戈爾子爵所有的私兵團副團長、『戰斧的奧爾頓(オルドーン）』。

他以千錘百鍊的大斧來戰鬥。
然而他的戰鬥風格卻與他憨直的相貌和武器不相符，反而是技巧派。
將自傲的力量完全活用，在對手逼入絕境的同時抓住機會，打出對手預料之外的一手，從而確鑿地將對手擊倒，這就是奧爾頓的基本戰術。
。

「鎧之劍士啊，你是打算幫助領民嗎？然而，單槍匹馬地闖入拉戈爾大人的公館什麼的，你是不是搞錯了什麼？」

。
奧爾頓一說，周圍的私兵開始接連不斷地對蘭貝爾發出嘲諷。
。

「雖說手腕是不錯，但腦袋卻不好使呢」
「難不成你真的以為能正面突破我們私兵團嗎」
「我們的同僚真是受你照顧了呢。這必須得百倍奉還啊。別以為能輕鬆地死去哦？」

。
蘭貝爾瞥了眼周圍的私兵。
接著往自己的背後觀望。
從背後出現了悄悄跟隨在蘭貝爾身後的私兵們的身影。
。

「……在館內，並沒有追上來，而是和我保持一段距離。雖然也有過疑問……看來，在這裡將我包圍，同時以增加人數來活用數量的優勢，並且封鎖逃跑路線才是你們的目的呢」

。

「正是如此。真是遺憾呢，鎧之劍士。單憑力量是不可能生存下去的哦」

。
奧爾頓對蘭貝爾說。
這個用於對付入侵者的包圍網也是他想出來的計策。
雖然奧爾頓的實力也是一流，但除了在戰鬥中戰略之外，他也是個十分擅長謀略和戰術的策士。
。

「小的們，給我一起上。想提高士氣也無所謂。不過你們可別抱著遊玩的心態。不然死的就是這邊了吶」

。
奧爾頓向部下們下達命令。
。

「是、是、奧爾頓副團長……」

。

「不要給他喘息的機會。雖然也想通過拷問、將他的真面目和飼主供出來，但這個男人實在是太危險了。要確確實實地殺了他」

。
奧爾頓平淡地再次命令。
奧爾頓不僅兼備力量和聰慧，並且有著絕不驕傲的性格。
以這個數量布下天羅地網，無論是怎樣的敵人都必定能扼殺。
然而即便如此，奧爾頓也不會手下留情。絕不給敵人留下一絲希望。
。
因奧爾頓發出的命令，部下們一齊向蘭貝爾發起了進攻。
奧爾頓預見入侵者慘死的未來，用鼻子哼了一聲後轉身離去。
。

「殺了他之後，無論如何都要查清楚他的身份。聽到了嗎？而且還要好好懲罰他的親人……」

。
背後響起了轟鳴。
地板開裂，過於猛烈的沖擊力，將地板掀起。
同時數名私兵也被輕易地彈到了空中
。
私兵們霎時間傻了眼，僵住了身體。
然而他們很快認識到，必須將眼前的威脅盡快鏟除，於是以視死如歸的覺悟再次發起第二波攻勢。
蘭貝爾的大劍痛快淋漓地接連揮舞。
大劍的斬擊將地板劈開，將鐵鎧連同私兵一同切斷。
每一斬每一切，都確實是奪命死神的一擊。
被斬殺的私兵彈飛到後方，血肉模糊，氣絕身亡。
。
在第二波全滅的同時，第三波的私兵們已經連一步也邁不出來了。
蘭貝爾悠然地穿過一動不動的奧爾頓和私兵，向公館的深處繼續前進。
