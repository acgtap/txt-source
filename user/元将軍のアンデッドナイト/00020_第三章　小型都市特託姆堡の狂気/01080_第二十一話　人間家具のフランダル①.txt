——吾，乃『吹笛之惡魔』，八賢者之一『真理的紡織者』。
。
吾不存在于世間。
吾早已脫離肉身。
。
只是重複著研究，只是紡織著真理的存在。
因而不老，亦因而不死，並因而不滅。
。
誰也無法識破吾之正體。
這個都市中的研究記錄，都會成為吾的東西。
有了這些的話，把這個都市廢棄掉也無所謂。
畢竟這裡是個可以隨時捨棄的地方。
。
往昔，師傅以為吾是不成材的**，將吾逐出了師門。
。
但他錯了。
吾升華了師傅的魔術，並獲得了紡織真理的時間。
這是多數魔術師所渴望、未能實現的黃粱一夢。
即使在師傅的弟子當中，也只有吾一人成功。
吾，已經超越了師傅。
。
接下來，既然拉戈爾子爵的叛變已經被王國發現，那麼這個地方也不能久留了。
不過，吾對這片領地仍有留戀。
特尼雷爾稱吾為研究狂人，然而他並不知道，吾也十分享受剎那間的快樂。
況且，吾還不得不對那些乘虛而入並肆意妄為的鼠輩們降下制裁的鐵錘。
。
◆
在特托姆堡的地上，赤髮的青年——亞爾安特正推著輪椅移動。
在輪椅上坐著的，是作為實驗體的少女艾麗澤。
在街上帶孩子們散步是他日常的工作，不過今天的狀況和平時不太一樣。
。
「得趁這個機會逃跑！至少也得離開這裡！」
。
「……不可能的，大哥哥。帶著我這個累贅，怎麼可能逃得掉呢。拋下我，趕緊逃走吧」
。
艾麗澤用顫抖的聲音說著，用手指著前方。
。
亞爾安特趁著騷亂將被囚禁的孩子們釋放並逃出了地下設施。
然而，帶著行動不便的艾麗澤逃跑需要花費相當的時間。
畢竟其他孩子也沒有理由去等她，所以就先走一程了。
。
「……大哥哥，去引領那些先走的孩子們吧？好嗎？大哥哥的魔術本領也不比那些壞蛋們差，不是嗎？」
。
艾麗澤用顫抖的聲音說著。
在這裡拋下艾麗澤的話，要是被『死之天使』的人發現，也不知道會遭受怎樣的對待。
艾麗澤並不是那麼勇敢的女孩。

『死之天使』或許會為了泄憤而殺死他們，而且他們能不能趁這次騷亂逃出生天也是個問題。
。
即便如此，如果連被自己拖累的亞爾安特也被他們抓住的話，倒不如只犧牲自己，這是她拼命思考得出的結論。

『死之天使』的副團長特尼雷爾異常地寵溺亞爾安特。
。
艾麗澤清楚，特尼雷爾並不是那種會畏懼王國兵團的人物。
她曾聽說過，特尼雷爾一邊用鼻子哼著歌，一邊則將活著的孩子解剖。
一想到特尼雷爾對亞爾安特的執著，便能想象得出他為了帶回亞爾安特而窮追不捨的景象。
。
「而且，大哥哥……為什麼是現在呢？大哥哥的話，趁著壞蛋們移動到其他據點的時候逃走，不是更能……啊……」
。
說到這裡，艾麗澤看見亞爾安特那難堪的表情後才恍然大悟。
到了那時，艾麗澤到最後的最後也逃不出去。
會帶著行走不便的她一起逃走的人，根本不存在。
。
亞爾安特也曾期待過，『死之天使』會將孩子們置之不顧，逃之夭夭。
若是如此，就像艾麗澤所說的那樣，在等待騷亂平息之後，趁著轉移據點時再逃走也不遲。
。
然而特尼雷爾在接受拉戈爾子爵的訃告後，便對在場的『死之天使』魔術師們這麼說。
。
『本來還有大量實驗體還沒到確認實驗成果的時間……嘛，這也是沒辦法的吧？把他們全部解體，確認實驗情況吧。剛好需要些臨死前受過驚嚇的屍體呢。畢竟也不想被小瞧，沒下過手的實驗體也殺掉好了』
。
特尼雷爾並不打算放跑任何一個人。
在那一刻，亞爾安特也決定要採取行動了。
即使是勝算多麼渺茫的賭局，也只能賭一把。
。
「不會的！王國兵團已經入侵了特托姆堡！王國出手的速度，比我們想象中還要快得多呢！只要到達他們的所在之處，細說原委之後他們肯定會保護我們的！你看，不是挺簡單的嗎！」
。
「…………會這麼順利嗎」
。
「當然了！」
。
實際上，所謂的王國兵團只不過是來領地偵查的一個小部隊而已，並沒有能與魔術師集團抗衡的戰力。
至於部隊隊長克羅伊茨，為何他會平白無故地闖進敵人的根據地特托姆堡呢，亞爾安特百思不得其解。
。
「可是，可是……」
。
回頭看向亞爾安特的艾麗澤的表情瞬間凝結。
僵持數秒之後，艾麗澤才回過神來擦了擦眼睛。
。
「艾麗澤……？」
。
「大哥哥，那是什麼……？」
。
亞爾安特回首時，發現後方有三只未知的生物在靠近他們。
那是單憑三言兩語也無法形容的異形。
。
它們並沒有雙手，身上纏繞著一圈又一圈的綳帶。
雙眼也被綳帶蒙蔽，取而代之的是如同孩子涂鴉般的一只眼睛的圖案。
頭頂上還插著好幾根試管。
。
其中一只異形忽然離奇地折疊起身體，變成椅子的形狀，而一名蒙著麻袋的男人則抱著胳膊並坐在異形椅子之上。
麻袋上也畫著如同孩子涂鴉般的兩只眼睛的圖案。
而且他的衣著和亞爾安特相同，身上披著『死之天使』的法袍。
。
「來~玩~耍~吧，亞爾安特醬，還有艾麗澤醬。特尼雷爾在找你們哦。畢竟狀況危急，離家出走也得適可而止哦，趕快回到我們身邊吧~」
。
「芙、芙蘭達爾！」
。
在『死之天使』的鍊金術師中，有一名被王國高價懸賞的犯罪者，那就是芙蘭達爾·蘭克帕。

『人間家具的芙蘭達爾』，這個外號在王都之中無人不知無人不曉。
。
在治安良好的王都中，曾有一年出現了將近百名的失踪者。
而他們所有人都在作為王都盛有名氣的白魔術師——芙蘭達爾所住的公館地下室中，以異樣的姿態被人發現。
其中也包含不少有名的冒險者。
。
更為悲慘的是，他們都仍存活著。
當中也有人在被保護後，由於身體的構造被改造成過於奇特的樣子而無法繼續維持生命，在數小時後因衰弱而死亡。
。
最後異端審問會決定為他們執行安樂死。
因過激的判斷而一直被譴責的異端審問會這次並沒有遭受世人的非議。
。
而芙蘭達爾現在則潛伏在『吹笛之惡魔』，所屬於『死之天使』。
。
「上頭說要把逃走的孩子們趕盡殺絕，所以不要恨我哦？抱歉吶艾麗澤醬！其實我也不想去做殺人這種勾當的！在王都引起騷亂的時候，我也沒有殺死一個人哦，對吧？殺死他們的是別的蠢 貨哦。那些蠢 貨，竟敢把我苦心孤詣做好的藝術品，當做垃 圾！」
。
「干！偏偏是那個芙蘭達爾！」
。
注意力集中在身後的亞爾安特的手上忽然傳來劇痛。
看准這一瞬間，艾麗澤左右搖晃身體，奮力將輪椅推倒。
少女的身體摔了在地上。
。
「艾、艾麗澤醬！為什麼要這麼做……」
。
「大哥哥，快逃……吧。其他孩子們肯定也在害怕。這樣下去，大家都會被殺掉的……但是，大哥哥肯定能拯救他們的吧」
。
即使三番四次傳達自己甘願犧牲的意志，亞爾安特也不肯聽從，為了讓頑固的他存活下去，艾麗澤故意編造了理由。
亞爾安特咬緊牙關，然後拔出了別在腰間的錫杖並沖向芙蘭達爾的方向。
。
「大哥哥！」
。
艾麗澤就像是責難一樣地呵斥。
亞爾安特則以痙攣的笑容向艾麗澤回眸。
。
「沒事的……沒事的，畢竟我是天才啊。所以，我才會被綁架到這種地方來啊。所以，包在我身上……」
。
「喂喂，這可真好笑啊。你這富家子弟只不過是被親戚推薦來這裡而已，別太得意忘形了！特尼雷爾疼愛你，只是因為你的長相符合他的喜好而已哦？嗯嗯？難道說你誤會了嗎？在過去，那些大貴族為了見我一面而長途跋涉，還把最重要的女兒帶過來了呢，我才是真正的天才」
。
芙蘭達爾故意地聳了聳肩，仿佛在嘲笑一般。
。
「不過他那寶貝女兒在治好之後，就成為了我其中一件藝術品了呢」
