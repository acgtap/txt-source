于都市巴莱拉，讨伐『吹笛之恶魔』的八贤者之一曼奇并斩断过去的因缘——盖洛夫之书的兰贝尔，辗转各个村落和城市，继续旅行，追迹『吹笛之恶魔』的情报。
当然，兰贝尔并没有饶恕打算将雷吉欧斯王国化为乌有的『吹笛之恶魔』的打算。
只希望能够尽早发掘出他们的根据地，斩草除根。
。
兰贝尔骑着无头的黑马梦魇，在原野上疾驰。
下一个目的地是在旅途中，在投宿的村子里听闻到了奇怪传闻的拉戈尔子爵领所属的都市库兰。
。
都市库兰是拉戈尔子爵所有的最大的都市，拉戈尔子爵本人也在那里设立了公馆。
然而拉戈尔子爵领很贫穷，都市库兰也不太起眼。
。
都市库兰有不少流浪者和孤儿，过半的都市都已经成为了贫民窟。
都市全境都已为无法无天的地区，而拉戈尔子爵的私兵则用武力将其镇压，这就是拉戈尔子爵领的现状。
。
据说在都市库兰附近的森林深处，发现了不少溶解的、已经凝固的人类尸体。
尸体上被刻入了魔术式，因此兰贝尔认为，这可能是以死操术为首的操纵禁忌魔术的『吹笛之恶魔』潜伏的地点。
。
(如果那里是他们的根据地的話，那就好说了呢……嗯？）

。
于正在思考的兰贝尔的视野中，出现了马车，以及咬着马车不放的骑兵的身影。
骑兵为一群半裸而粗鲁的男人，手握棍棒和长枪作为武器。
数量为五人。
。

「恰哈哈！马车是跑不过我们的！」
「欧拉，不想死就拼命地跑啊！恰哈哈哈！」

。
男人们不断对马车施以谩骂。
他们故意不追上去，就像是在玩耍一样。
。
(盗贼团吗……和传闻中一样，拉戈尔子爵领的治安似乎有点不好呢……）

。
兰贝尔将大剑从剑鞘中拔出，准备突击。
。

「梦魇，给我加快速度。要大闹一场了」

。
梦魇扬起无头的脖子。
从不知何处发出了毛骨悚然的嘶叫声。
。

「什么声音……？喂，好像有什么东西追上来了！」
「想妨碍俺们狩猎吗，好大的胆子……喂，那是什么怪物」

。
跨着无头的大马的全身铠甲的男人，这个场面足以令人闻风丧胆。
。

「是妖怪啊啊啊啊啊！」
「马车什么的别管了！那家伙很危险啊！」

。
一齐逃走的盗贼们，仍被兰贝尔紧追不舍。
毕竟梦魇生前也是有名的骏马。
即使承载着超重量级的兰贝尔，一般的马也无法挣脱它的速度。
。
兰贝尔在眨眼之间就追上盗贼们，用剑腹将落在最后面的男人打落。
。

「呜咕！？」

。
只用一击就将脖子、腹部、胸部和腰骨打断，以奇怪的动作从马上摔了下来。
男人在落地时就已经断气了。
。

「达、达尔塞……哦咕！？」(反过来就是塞尔达了）

。
扭头往后看的男人的脑袋，被兰贝尔的大剑打飞。
再次挥舞的大剑刺入第三个男人的后背，贯穿了他的心脏。
。

「即使是一角兔之群，也至少拥有分散逃跑的知性呢」

。
对于兰贝尔的話语，最后的两人面面相觑，无言地颔首，然后兵分两路。
很快兰贝尔追上了其中的一方，用袈裟斩将男人连同马一同撕裂。
这时，剩下的一人已经逃到了相当远的地方。
。

「太、太好了！太好了，总算是，活下来了……」

。
气喘吁吁地，盗贼说着生前最后的一番話。
然而，他下一刻便听见大地被刨起的巨响，脸上被绝望笼罩。
。

「噫……唏噫噫噫噫噫！绕、饶命……」

。
男人的上半身被斩飞。
五名盗贼就这么绝息了。
。
(试试向马车的主人询问拉戈尔子爵的事情吧。……不过，梦魇还在就谈不上話了。要是被认为是饲养着不死族的話问题就大了。而且，梦魇的身姿，对这个的时代的人们来说似乎是太恐怖了呢）

。
相比较而言，与其说是梦魇的过错，不如说是兰贝尔想要骑上梦魇的想法太过强烈才是最大的原因，然而并没有人能指责出这一点。
。
兰贝尔从梦魇背上跳下。
梦魇转身离去，在阴影中逐渐消失。
。
虽然马车的马之前还处于兴奋的状态，但在兰贝尔解决掉盗贼后又走了一段路，终于冷静下来，停下了脚步。
兰贝尔往停下的马车接近。
。

「喂，我有些事情想问你……」

。
回应兰贝尔的話语，一名中年男人从马车中出现。
。

「啊，刚才真是危险啊……是您赶走了刚才那些强盗吧。感激不尽……要是刚才那样下去的話，我迟早都会被他们杀掉的。虽然我是来这里行商的，但没想到居然会遭这种罪……还以为来错地方了呢」

。
男人这么说着，用手擦拭汗水。
。

「我叫刚扎斯。有什么事尽管问，别看我这样，我可是见得多了，哪个地方我没去过的」

。
说到这里，刚扎斯皱眉并环顾四周。
。

「呐……您刚才还是骑着马的吧？马去哪了？」

。

「比起这个，快回答我的问题，我想知道这附近的领主、拉戈尔子爵的事情」

。
兰贝尔强行岔开了話题。
。

「啊、好的……」

。
刚扎斯虽然对兰贝尔的样子感到不信任，但被兰贝尔岔开話题的豪迈和他魁梧的身材以及气场给压倒，不再追问下去，只好不断地点头。
