继瓦砾之後，在废墟中传出了一人份的落地的声音。
兰贝尔降下大剑，回过头来看着拉戈尔子爵。
。

「死、死了吗？露、露露可死了吗……？」

。
拉戈尔那憔悴的脸庞上重新燃起了希望的灯火，向兰贝尔询问。
。
兰贝尔俯视被自己破坏的墙壁下方的瓦砾之山，平静地回顾拉戈尔子爵。
。
拉戈尔子爵知晓『吹笛之恶魔』，坚信欧鲍克伯爵“今後掌握国家主权的必然是他们”的主张并与『吹笛之恶魔』发生接触，然而拉戈尔子爵已经对他们的所作所为感到厌烦。
并且露露可在拉戈尔子爵领中逐渐扩大『吹笛之恶魔』的影响力，成为了领地内实质上的统治者。
虽然这并不是拉戈尔子爵心目中的互惠互利的体制，但由於对他们感到畏惧，结果只能不得不接受他们日渐过分的要求。
。
况且自己也是叛国之身，不可能去依靠王家。
事到如今也无法收手，拉戈尔子爵陷入了进退两难的狼狈处境。
。
ここで窓口であり，作为『吹笛之恶魔』幹部的八贤者露露可的死亡，对拉戈尔子爵来说是好是坏，很难断定。
拉戈尔子爵认为，将『吹笛之恶魔』赶走从而取回被他们夺走的实权，并且对王家隐瞒自己所做的恶行是最優先的事项。
。
虽然拉戈尔子爵认为就这麼把『吹笛之恶魔』和自己的关係弄得含糊不清就好，但为此也存在着不少的问题。

『吹笛之恶魔』的报復也是个问题，但首先必须将兰贝尔的嘴巴封住。
。
拉戈尔子爵毫不迷茫地，向兰贝尔使出蹭破头皮的土下座。
周围的私兵们目瞪口呆，打算扶起拉戈尔子爵。
。

「剑士阁下……其实、其实，老朽、老朽的领地，长年以来都被『吹笛之恶魔』所威胁和支配……！若不是剑士大人出手相救，老朽也不知道自己会有什麼下场……」

。
拉戈尔子爵向兰贝尔倾诉，尽管被泪水和鼻涕弄髒了脸。
周围的私兵们哑口无言了。
。
原本招待『吹笛之恶魔』的就是拉戈尔子爵自己，失去控制也只是他们自作自受罢了。
只展示出对自己有利的一面、而将不合逻辑的部分通过凭空杜撰来弥补、坚称自己才是受害者，这一连串可悲的动作他都可以毫无抵触地做到，可以说是达到影帝的领域了。
而目睹了这一切的私兵们也认为世上不可能存在这麼恬不知耻的人，产生了实际上自己的记忆出了错，拉戈尔子爵所说的話才是正确的的错觉。
。
虽然作为领主的拉戈尔子爵无德无能，但在转嫁责任的决断力和对於明哲保身的执着上却有着一流的水准。
。

「拜托您了，剑士阁下……！如您这般的剑技，老朽从未见过！仍然、『吹笛之恶魔』的人仍然潜伏在这片领地！请替老朽解决掉他们！」

。
兰贝尔将头盔深处的眼孔，往热切恳求的拉戈尔子爵的方向对焦。
。

「不用你说，我也会做。我会将他们赶尽杀绝。但是，你的性命我也要取走」

。

「你、你说什麼！为何！老朽、老朽，明明都这麼低声下气地求饶了！」

。

「有遗言的話赶紧说。把情报都吐出来，赶紧的」

。

「你、你……！」

。
拉戈尔子爵狼狈不堪。
即使多少有些可疑的地方，拉戈尔子爵也没想到自己会被正面宣布處刑。
。

「那、那样也好！杀了他！谁来，把他杀了！快点！」

。
拉戈尔子爵拼死地呼喊。
兰贝尔不慌不忙地持剑并面向拉戈尔子爵。
拉戈尔子爵当场摔倒，即便如此，他仍用双手爬行，妄想和兰贝尔拉开距离。
。

「噫！不是老朽的错！这不是老朽的错！因为，除了这麼做之外，还有什麼办法！？老朽没有做错！」

。
一名私兵惊恐地靠近拉戈尔子爵。
。

「拉戈尔子爵大人……您，额头上的魔法阵是……？」

。

「噫！？」

。
拉戈尔子爵害怕地用手触摸自己的额头。
。

「……真是糟糕透顶。居然会是贝尔菲斯王国的许德部落的秘术。估计是待你背叛之时才发动的术式吧」

。
——八国统一战争时代，贝尔菲斯王国从古时开始就已经在国内各地出现权力鬥争，背叛与同族杀害时有发生，国民也自然而然地选择独善其身。
其中，贝尔菲斯王国内的以虫为媒介、操使咒术的许德族，完成了对人类精神的细微变化作出反应并杀死宿主的秘术，因此被王家所器重。
。
然而，这种咒术也存在不少缺陷，错误发动的情况也时有发生。
这麼便利的东西，从一开始就不可能做出来。
并且突然有人利用这个缺陷，有意图地让咒术发动并声称此人企图谋反，将不顺眼的人连同一族一齐诛灭，从那之後便一发不可收拾。
贝尔菲斯王国虽为强大的魔术国家，而早早地败给其他王国的最大的原因，也许就是这个咒术的缘故。
。
据称在贝尔菲斯王国灭亡後，将这个咒术视为危险的战胜国的将军，将许德一族全部處刑，烧毁了多数的资料，从这个世界上完全消失了。
虽说不完美，但许德部落的秘术实在是过於便利。
正因为太便利所以才危险。
将军认为，万一本国的王家也打算对家臣使用这项秘术的話，到那时本国便会重蹈贝尔菲斯王国的覆辙。
。

「那个咒术绝不能遗留到後世。拉戈尔，你有什麼线索吗？你身上的某处，应该被某人施加了魔术才对。并非刚才的小丑，到底是什麼人？什麼都可以，快把情报吐露出来。说完之後，我再送你一程」

。
兰贝尔向拉戈尔子爵接近，将持有大剑的那只手伸长。
拉戈尔子爵睁大双眼。
。

「不、不要啊！老朽还不想死！拜托你，快把这个拿掉！快把这个拿掉！快点！啊，不要、不要啊！停下、快停下，谁来停止这玩意！」

。
拉戈尔子爵拼命地挠着额头。
皮肉被抓破，额头被挠出血。然而，魔法阵已经开始发光。
。

「不、不要啊啊啊！」

。

「中止的方法是有的」

。

「真、真的吗？」

。
兰贝尔颔首。
。

「砍下你的头颅即可」

。

「不要啊啊啊啊啊！」

。
拉戈尔子爵一边号啕，一边在地上哭闹。
私兵们也不知所措，只能眼睁睁地看着。
。

「呜哇啊啊啊！不要啊！呜哇啊啊啊！老朽做了什麼！老朽到底做了什麼啊啊啊！」

。

「快说！」

。
兰贝尔大喊，但拉戈尔子爵并没有回应。
他突然停下哭闹，仰躺在地上。
紧接着，拉戈尔子爵的身体开始一颤一颤地痉挛，从嘴裡涌出白泡。
。

「哦哦、哦哦哦哦哦哦哦哦哦哦哦哦哦哦哦」

。
拉戈尔子爵翻着白眼，额头开裂，从中钻出一条附有甲壳的多足昆虫。
。

「拉、拉戈尔子爵大人！？」

。
巨虫顺着拉戈尔子爵的身体爬了下来，往附近的呆若木鸡的私兵接近。
。

「噫！别、别过来啊啊啊！」

。
兰贝尔跳到不知所措的私兵的面前，将巨虫瞬杀。
尽管被切成两半，巨虫依然在不停地蠕动，最终停止了活动。
。

「……这也是他们实验中的一环吧」

。
兰贝尔说着，把视线落在拉戈尔子爵的尸体上。
额头上开着一个大洞，露出被蚕食过的大脑。
从裂开的额头中流出血液腐败般的、绿色而浑浊的液体。
从七孔中也流出同样的体液。
。
接着兰贝尔回顾墙壁後的空洞，往下观察。
瓦砾的位置有些许变动，并增添了不少血的污迹。
看来露露可是挣脱了瓦砾并逃了出去。
。
(……让她落到下面，真是失策了呢。虽然相比於下去确认她的生死，将只有拉戈尔子爵才知道的情报榨乾才是最優先的事项，但两边都没能兼顾呢。不仅放跑了敌人，连情报的来源也死了……这是何等的失态啊，兰贝尔。固执于性命而不择手段的魔术师有多麼地难缠，你又不是不知道的。你尾巴翘上天了吗？）

。
兰贝尔一边反省，一边将大剑收回剑鞘。
与八国统一战争相较，现代的魔术师大多数都能轻鬆地杀死，但如果说自己完全没有松懈的話，兰贝尔也无法否认。
。
在那个时代，本以为已经打倒的魔术师却用意想不到的方式存活下来，酿成大量士兵白白战死的局面，这是在战争中十分常见的场面。
在兰贝尔看来，那个时代的魔术师早已不是人类。
。
