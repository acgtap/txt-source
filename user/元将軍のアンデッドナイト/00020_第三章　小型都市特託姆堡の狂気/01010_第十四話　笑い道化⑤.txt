繼瓦礫之後，在廢墟中傳出了一人份的落地的聲音。
蘭貝爾降下大劍，回過頭來看著拉戈爾子爵。
。

「死、死了嗎？露、露露可死了嗎……？」

。
拉戈爾那憔悴的臉龐上重新燃起了希望的燈火，向蘭貝爾詢問。
。
蘭貝爾俯視被自己破壞的牆壁下方的瓦礫之山，平靜地回顧拉戈爾子爵。
。
拉戈爾子爵知曉『吹笛之惡魔』，堅信歐鮑克伯爵“今後掌握國家主權的必然是他們”的主張並與『吹笛之惡魔』發生接觸，然而拉戈爾子爵已經對他們的所作所為感到厭煩。
並且露露可在拉戈爾子爵領中逐漸擴大『吹笛之惡魔』的影響力，成為了領地內實質上的統治者。
雖然這並不是拉戈爾子爵心目中的互惠互利的體制，但由於對他們感到畏懼，結果只能不得不接受他們日漸過分的要求。
。
況且自己也是叛國之身，不可能去依靠王家。
事到如今也無法收手，拉戈爾子爵陷入了進退兩難的狼狽處境。
。
ここで窓口であり，作為『吹笛之惡魔』幹部的八賢者露露可的死亡，對拉戈爾子爵來說是好是壞，很難斷定。
拉戈爾子爵認為，將『吹笛之惡魔』趕走從而取回被他們奪走的實權，並且對王家隱瞞自己所做的惡行是最優先的事項。
。
雖然拉戈爾子爵認為就這麼把『吹笛之惡魔』和自己的關係弄得含糊不清就好，但為此也存在著不少的問題。

『吹笛之惡魔』的報復也是個問題，但首先必須將蘭貝爾的嘴巴封住。
。
拉戈爾子爵毫不迷茫地，向蘭貝爾使出蹭破頭皮的土下座。
周圍的私兵們目瞪口呆，打算扶起拉戈爾子爵。
。

「劍士閣下……其實、其實，老朽、老朽的領地，長年以來都被『吹笛之惡魔』所威脅和支配……！若不是劍士大人出手相救，老朽也不知道自己會有什麼下場……」

。
拉戈爾子爵向蘭貝爾傾訴，盡管被淚水和鼻涕弄髒了臉。
周圍的私兵們啞口無言了。
。
原本招待『吹笛之惡魔』的就是拉戈爾子爵自己，失去控制也只是他們自作自受罷了。
只展示出對自己有利的一面、而將不合邏輯的部分通過憑空杜撰來彌補、堅稱自己才是受害者，這一連串可悲的動作他都可以毫無抵觸地做到，可以說是達到影帝的領域了。
而目睹了這一切的私兵們也認為世上不可能存在這麼恬不知恥的人，產生了實際上自己的記憶出了錯，拉戈爾子爵所說的話才是正確的的錯覺。
。
雖然作為領主的拉戈爾子爵無德無能，但在轉嫁責任的決斷力和對於明哲保身的執著上卻有著一流的水準。
。

「拜托您了，劍士閣下……！如您這般的劍技，老朽從未見過！仍然、『吹笛之惡魔』的人仍然潛伏在這片領地！請替老朽解決掉他們！」

。
蘭貝爾將頭盔深處的眼孔，往熱切懇求的拉戈爾子爵的方向對焦。
。

「不用你說，我也會做。我會將他們趕盡殺絕。但是，你的性命我也要取走」

。

「你、你說什麼！為何！老朽、老朽，明明都這麼低聲下氣地求饒了！」

。

「有遺言的話趕緊說。把情報都吐出來，趕緊的」

。

「你、你……！」

。
拉戈爾子爵狼狽不堪。
即使多少有些可疑的地方，拉戈爾子爵也沒想到自己會被正面宣布處刑。
。

「那、那樣也好！殺了他！誰來，把他殺了！快點！」

。
拉戈爾子爵拼死地呼喊。
蘭貝爾不慌不忙地持劍並面向拉戈爾子爵。
拉戈爾子爵當場摔倒，即便如此，他仍用雙手爬行，妄想和蘭貝爾拉開距離。
。

「噫！不是老朽的錯！這不是老朽的錯！因為，除了這麼做之外，還有什麼辦法！？老朽沒有做錯！」

。
一名私兵驚恐地靠近拉戈爾子爵。
。

「拉戈爾子爵大人……您，額頭上的魔法陣是……？」

。

「噫！？」

。
拉戈爾子爵害怕地用手觸摸自己的額頭。
。

「……真是糟糕透頂。居然會是貝爾菲斯王國的許德部落的秘術。估計是待你背叛之時才發動的術式吧」

。
——八國統一戰爭時代，貝爾菲斯王國從古時開始就已經在國內各地出現權力鬥爭，背叛與同族殺害時有发生，國民也自然而然地選擇獨善其身。
其中，貝爾菲斯王國內的以虫為媒介、操使咒術的許德族，完成了對人類精神的細微變化作出反應並殺死宿主的秘術，因此被王家所器重。
。
然而，這種咒術也存在不少缺陷，錯誤發動的情況也時有发生。
這麼便利的東西，從一開始就不可能做出來。
並且突然有人利用這個缺陷，有意圖地讓咒術發動並聲稱此人企圖謀反，將不順眼的人連同一族一齊誅滅，從那之後便一發不可收拾。
貝爾菲斯王國雖為強大的魔術國家，而早早地敗給其他王國的最大的原因，也許就是這個咒術的緣故。
。
據稱在貝爾菲斯王國滅亡後，將這個咒術視為危險的戰勝國的將軍，將許德一族全部處刑，燒毀了多數的資料，從這個世界上完全消失了。
雖說不完美，但許德部落的秘術實在是過於便利。
正因為太便利所以才危險。
將軍認為，萬一本國的王家也打算對家臣使用這項秘術的話，到那時本國便會重蹈貝爾菲斯王國的覆轍。
。

「那個咒術絕不能遺留到後世。拉戈爾，你有什麼線索嗎？你身上的某處，應該被某人施加了魔術才對。並非剛才的小丑，到底是什麼人？什麼都可以，快把情報吐露出來。說完之後，我再送你一程」

。
蘭貝爾向拉戈爾子爵接近，將持有大劍的那只手伸長。
拉戈爾子爵睜大雙眼。
。

「不、不要啊！老朽還不想死！拜托你，快把這個拿掉！快把這個拿掉！快點！啊，不要、不要啊！停下、快停下，誰來停止這玩意！」

。
拉戈爾子爵拼命地撓著額頭。
皮肉被抓破，額頭被撓出血。然而，魔法陣已經開始發光。
。

「不、不要啊啊啊！」

。

「中止的方法是有的」

。

「真、真的嗎？」

。
蘭貝爾頷首。
。

「砍下你的頭顱即可」

。

「不要啊啊啊啊啊！」

。
拉戈爾子爵一邊號啕，一邊在地上哭鬧。
私兵們也不知所措，只能眼睜睜地看著。
。

「嗚哇啊啊啊！不要啊！嗚哇啊啊啊！老朽做了什麼！老朽到底做了什麼啊啊啊！」

。

「快說！」

。
蘭貝爾大喊，但拉戈爾子爵並沒有回應。
他突然停下哭鬧，仰躺在地上。
緊接著，拉戈爾子爵的身體開始一顫一顫地痙攣，從嘴裡湧出白泡。
。

「哦哦、哦哦哦哦哦哦哦哦哦哦哦哦哦哦哦」

。
拉戈爾子爵翻著白眼，額頭開裂，從中鑽出一條附有甲殻的多足昆虫。
。

「拉、拉戈爾子爵大人！？」

。
巨虫順著拉戈爾子爵的身體爬了下來，往附近的呆若木雞的私兵接近。
。

「噫！別、別過來啊啊啊！」

。
蘭貝爾跳到不知所措的私兵的面前，將巨虫瞬殺。
盡管被切成兩半，巨虫依然在不停地蠕動，最終停止了活動。
。

「……這也是他們實驗中的一環吧」

。
蘭貝爾說著，把視線落在拉戈爾子爵的屍體上。
額頭上開著一個大洞，露出被蠶食過的大腦。
從裂開的額頭中流出血液腐敗般的、綠色而渾濁的液體。
從七孔中也流出同樣的體液。
。
接著蘭貝爾回顧牆壁後的空洞，往下觀察。
瓦礫的位置有些許變動，並增添了不少血的污跡。
看來露露可是掙脫了瓦礫並逃了出去。
。
(……讓她落到下面，真是失策了呢。雖然相比於下去確認她的生死，將只有拉戈爾子爵才知道的情報榨乾才是最優先的事項，但兩邊都沒能兼顧呢。不僅放跑了敵人，連情報的來源也死了……這是何等的失態啊，蘭貝爾。固執於性命而不擇手段的魔術師有多麼地難纏，你又不是不知道的。你尾巴翹上天了嗎？）

。
蘭貝爾一邊反省，一邊將大劍收回劍鞘。
與八國統一戰爭相較，現代的魔術師大多數都能輕鬆地殺死，但如果說自己完全沒有鬆懈的話，蘭貝爾也無法否認。
。
在那個時代，本以為已經打倒的魔術師卻用意想不到的方式存活下來，釀成大量士兵白白戰死的局面，這是在戰爭中十分常見的場面。
在蘭貝爾看來，那個時代的魔術師早已不是人類。
。
