「放過我們一馬吧！求你們了！要是女兒有個三長兩短的話，我也沒有活下去的意思了！」

。
亞爾蒂的父親扣頭，向魔術師哀求。
而亞爾蒂似乎並不清楚自己的處境，但她從眼前的私兵們身上感覺到了敵意，害怕地摟著父親瑟瑟發抖。
。

「拉戈爾子爵也不是因為壞心眼才張貼出那樣的通告。這是為了拯救你們的規則。現在，對於居住於拉戈爾子爵領的指定八區的領民們來說，並不存在足以養育孩子成長的環境。在指定八區居住的八成孩子，在迎來十五歲之前就已經死於非命。而在指定八區中繳納了特別稅的領民，則會認定為持有養育孩子能力的家庭而免除義務……您並沒有納稅吧？您是打算繼續恣意妄為地讓這孩子死去呢，還是說托付給拉戈爾子爵大人，從而過上安穩的生活呢……」

。
魔術師一字一句地講述。
。

「這種事情誰會相信！亞爾提具體要去哪裡、要做什麼都不說清楚！」

。

「行行行，我明白了。畢竟我也不是惡鬼。和平至上。若是無法接受的話，那我也只能說服到您接受為止了。畢竟讓所有人心服口服，是我們從拉戈爾子爵大人那裡接受的命令」

。
魔術師逢場作戲，深思熟慮般地哈腰點頭。
。

「拿等價袋出來」

。

「是，弗貝因(フォルベイン）大人！」

。
魔術師對背後的私兵作出命令。
私兵稱魔術師為弗貝因，把一份麻袋交給了他。
。
(麻袋裡的是銀幣嗎……）

。
從袋口中能窺伺到銀色的光澤。
。
這些錢肯定不是白送的。
這裡似乎本來就是實行用這個叫等價袋的東西強制地和孩子交換的制度。
。

「你們是什麼意思？不是說，要說服我嗎……」

。
亞爾蒂的父親不知如何是好。
這是當然了。說是要用言語來說服自己，卻突然拿出裝著貨幣的麻袋，簡直是讓人摸不著頭腦。
。
弗貝因深深地頷首，伸手打斷了他的發言。
他又從自己的披風裡拿出三枚金幣，並放進麻袋裡。
。

「什……咦？」

。
亞爾蒂的父親臉上流露出為難的神色。
。

「這樣您就心滿意足了吧？這是您不眠不休地工作幾天的分量呢？」

。
弗貝因在假面背後哈哈大笑。
私兵們也在背後大笑。
。

「來啊來啊，很想要對吧？是錢哦！來啊！」

。
弗貝因將袋子故意地放在亞爾蒂父親的面前左右搖晃。
之後反而往袋子裡抓了一把，在周圍亂灑金幣和銀幣。
毫不留情地對著以頭搶地的男人的後背和頭部灑落貨幣。
。

「撿啊，來撿啊！哈哈哈哈哈！你們這些乞食的，無論怎麼裝模作樣，都抵擋不住金錢的誘惑吧？哈哈哈、哈哈哈哈哈！啊啊，每次看到這副場景都感覺很愉悅啊！」

。
亞爾蒂的父親一時靜止，最終，開始收集起散落在地的貨幣。
見此情形，魔術師和私兵們捧腹大笑。
。

「那麼，您的女兒我們就收下了哦。期待您再次將健康的孩子，獻給拉戈爾子爵大人」

。
弗貝因殷勤地說著，禮貌地鞠躬。
而亞爾蒂父親卻向弗貝因遞出捧著一堆貨幣的雙手。
。

「如果不是為了女兒而用的話，這些錢給我也沒有任何意義……請您、請你……高抬貴手吧。這就是我唯一的願望。在我看來……即使托付給你們，亞爾蒂也不會獲得幸福……」

。

「哈？你傻的嗎」

。
弗貝因情緒失控地扭過脖子，從極近的距離透過白假面怒視著亞爾蒂父親的臉。
。

「明明這邊也已經為了不讓多餘的傳聞擴散開來而做出了最大的讓步，別給我得寸進尺了！礙手礙腳的小鬼變成金幣哦？很棒吧。夠了，殺了男的把他女兒帶走吧」

。
與弗貝因宣告的同時，背後的三名私兵開始了行動。
。

「一開始這麼做不就好了嗎。在這種傢伙身上，只會浪費錢的啦」
「就是就是。不如這樣吧，就當做是已經花了錢，我們瞞著露露可大人私吞了吧」

。
這讓亞爾蒂的父親變得狼狽不堪。
亞爾蒂握著父親的手，用懇求般地眼神看著蘭貝爾。
。

「為了知曉事情的原委，我才選擇靜觀其變。不過，聽到這裡也已經足夠了」

。
蘭貝爾拔出大劍。
。

「無論怎麼粉飾話語，你們都只是些骯髒的奴隷商罷了。非斬不可」

。
其中一名私兵對蘭貝爾的話嗤之以鼻，將劍尖對准了蘭貝爾。
。

「真是不走運呢，搞錯時代的土掉渣的鎧甲男。最近上頭不希望泄露多餘的情報呢。你也順便下地獄去吧」

。
下一瞬間，私兵的鎧甲上綻放出一道縱向的閃光。
身體不斷湧出鮮血並飛向後方。
因背部撞在牆壁上的沖擊，從斬擊的縫隙中垂落出內臟。
。

「給我訂正。這是雷吉歐斯王國的守護騎士之證、傳統的魔將之鎧、雷吉歐尼克斯·全魔金裝甲。絕不是搞錯時代的土掉渣的鎧甲」

。
即使想讓他訂正也已經無力回天了。
私兵的雙眼空虛地望著天上，從嘴裡也流淌出了鮮血。
他明顯地已經斷氣了。
。
剩下的兩名私兵瞠目結舌。
他們知道是大鎧的男人揮舞大劍，將他們的同伴砍飛。
然而，他的劍技卻完全看不清。
盡管武器是一目了然地擁有非同尋常質量的大劍。
。

「三個人一起上吧，我來做你們的對手」

。
蘭貝爾向弗貝因斷言。
由於弗貝因戴著白色的面具而無法獲悉他的表情，但他的肩膀正在瑟瑟發抖。
明顯在目睹了蘭貝爾的劍技之後內心產生了動搖。
