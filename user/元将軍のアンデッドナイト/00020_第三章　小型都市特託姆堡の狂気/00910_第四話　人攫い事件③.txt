被行商人刚扎斯邀请乘上马车、到达拉戈尔子爵领·中央都市库兰的兰贝尔，马上依照刚扎斯的情报前往贫民窟。
如果领主的士兵的确是在蹂躏着贫民的話，这对身为原·王国顶点士兵之一角的兰贝尔来说是个不可忽视的问题。
如有必要，兰贝尔打算当天就闯入领主的房间并进行拷问，根据回答来决定他的下场。
。
在狭窄的小巷中，用木板补缀外墙的房屋络绎不绝。
背靠着污浊的墙壁，一名瘦弱的老人坐在地上。
。

「老人家，我想知道关于拉戈尔和他的私兵的事情」

。
兰贝尔一说話，老人便睁大眼睛看向兰贝尔。
。

「老老、老朽什么都不知道！所以，请放过我吧！」

。
老人用颤抖的声音说着，一边爬了起来。
接着便东倒西歪地从兰贝尔处逃离。
恐怕是在畏惧着说出多余的話後，被拉戈尔子爵报复的事情吧。
。
(看来，恐怖已经是根深蒂固的样子了。没必要用威逼的方式打听情报）

。
兰贝尔就此打住，寻找下一个线索。
然而，无论是遇到什么人，他们在被质问时的反应和方才的老人也是大同小异。
究竟是怎么了，当兰贝尔把手放在头盔上沉思时，某件房屋的门中露出一丝门缝，一名小女孩在门缝中窥伺着兰贝尔。
从身高来看，似乎连十岁都不到。
赤红色的头发乱糟糟地，身上穿着用破布东拼西凑的衣服。
。
兰贝尔看向她时，诚惶诚恐地窥伺着兰贝尔的孩子脸上绽放出笑容。
。

「是骑士大人！是骑士大人吗！您是来打败那些坏人的对吧？」

。
兰贝尔走进那名少女。
少女开门来到兰贝尔的面前。
。
从到达中央都市库兰的贫民窟以来，反响最好的就是这孩子。
也许能打听到什么吧，兰贝尔的心中充满了这样的期待。
。

「唔姆，没错。我就是雷吉欧斯王国四魔将之一，兰贝尔·德拉克罗瓦」

。

「四麻将……？虽然不是很懂，不过好厉害的样子！好帅的名字啊！」

。

「你也这么想吧」

。
兰贝尔深深地点了点头。
。

「兰贝尔大人，是来这裡打败小丑的吗？」

。

「小丑……？那是谁」

。

「爸爸说过的哦。自从小丑到访领主大人的公馆之後，这裡就开始变得奇怪了……所以，骑士大人……」

。
在少女说完之前，兰贝尔往背後打了一拳。
有人从兰贝尔的背後，以他的肩膀为目标挥下了木棒。
兰贝尔以拳头折断了木棒，紧接着放出的贯通掌掠过袭击者的脸颊，贯穿了墙壁。
袭击者发出短短的悲鸣後便跌倒在地上。
接着兰贝尔伸出手指，高高地举起笼手。
只是轻轻一碰就能轻易地粉碎人类的头骨，这从那面被漂亮地打出洞穴的墙壁上已经一目了然了。
。

「爸、爸爸……？」

。
少女向袭击者开口。
兰贝尔观察了袭击自己的男人，
那是一名长着乱糟糟的红髮的、不修边幅的男人。
他的眼皮底下，不知是因为肮脏还是因为皮肤已经发灰，使他看上去有点衰老，但年龄估计在三十岁前後。
。
袭击者绝望地跪在原地。
然後就像是生了闷火一样，一边俯视着地面一边小声嘀咕。
。

「亚尔蒂……都说过几次了，不要给陌生人开门……」

。
看来少女名为亚尔蒂，而袭击者是她的父亲。
。

「给我解释一下。根据你的回答而保留你的处分」

。
兰贝尔故意泄露出瘴气，向男人盘问。
先不说勇猛的战士，这份迫力是普通人绝对不可能承受得住的。
在泄露出瘴气的兰贝尔的压迫感面前说谎的人是不存在的。
。
男人很快伏倒在地，向兰贝尔低头。
。

「要、要杀要剐悉听尊便！但是、请您放过我女儿吧……！自从妻子去世之後，她便是我生存的唯一的意义！」

。

「…………」

。
兰贝尔俯视着男人，慢慢地放下手臂。
。

「濒临死亡，最先乞求的是女儿的性命吗。真是了不起的骨气。好好珍惜吧」

。
男人连发生了什么都不知道，胆战心惊地抬起头仰视着兰贝尔。
。
(不过，这份畏惧也太过异常了吧）

。
无论是老人，还是亚尔蒂的父亲，反应都过于剧烈。
看来贫民窟裡的人都受到过巨大的心灵创伤呢。
。
这时，从某处传来了金属摩擦的声音。
是铠甲的声音，兰贝尔立刻就察觉了。
兰贝尔望向声音的方向，发现了三名武装统一的铠剑士，并且在那先头还有一名戴着兜帽、戴着刻画了记号的白面罩的魔术师。
。

「什么嘛，不是还有小孩子吗」
「就算狩猎多少次，也会有残存下来的呢」
「就跟土蛙一样呢。马上又会增加的啦」

。
铠剑士们笑言。
在统一的武器上，刻入了同样的纹章。
。
(是拉戈尔子爵的私兵吗。然而，先头的魔术师又是什么人？无论是恶趣味的外套也好面罩也好，都不觉得是贵族私兵所有的物品）

。

「这可不行啊。为了复兴贫困的领地，在领地指定区域内生活的年幼的孩子们，必须经由我们的指引，实施最低限度的教育，并作为劳动力派遣到邻近的各地。为了领地也请大家忍痛割爱并协助我们」

。
面罩的魔术师说明，并故意地张开手臂，感叹地摇了摇头。
。
(原来如此，这就是领主的士兵拐卖人口事件的真面目吗。说得倒是挺好听，但不服从的話，领主就会对贫民窟增加奴隶税吧）

。
兰贝尔将意识转移到背负着的大剑。
准备好在对方出手的瞬间立即拔刀。
