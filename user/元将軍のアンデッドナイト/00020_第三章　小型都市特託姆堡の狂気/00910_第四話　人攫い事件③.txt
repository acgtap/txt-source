被行商人剛扎斯邀請乘上馬車、到達拉戈爾子爵領·中央都市庫蘭的蘭貝爾，馬上依照剛扎斯的情報前往貧民窟。
如果領主的士兵的確是在蹂躪著貧民的話，這對身為原·王國頂點士兵之一角的蘭貝爾來說是個不可忽視的問題。
如有必要，蘭貝爾打算當天就闖入領主的房間並進行拷問，根據回答來決定他的下場。
。
在狹窄的小巷中，用木板補綴外牆的房屋絡繹不絕。
背靠著污濁的牆壁，一名瘦弱的老人坐在地上。
。

「老人家，我想知道關於拉戈爾和他的私兵的事情」

。
蘭貝爾一說話，老人便睜大眼睛看向蘭貝爾。
。

「老老、老朽什麼都不知道！所以，請放過我吧！」

。
老人用顫抖的聲音說著，一邊爬了起來。
接著便東倒西歪地從蘭貝爾處逃離。
恐怕是在畏懼著說出多餘的話後，被拉戈爾子爵報復的事情吧。
。
(看來，恐怖已經是根深蒂固的樣子了。沒必要用威逼的方式打聽情報）

。
蘭貝爾就此打住，尋找下一個線索。
然而，無論是遇到什麼人，他們在被質問時的反應和方才的老人也是大同小異。
究竟是怎麼了，當蘭貝爾把手放在頭盔上沉思時，某件房屋的門中露出一絲門縫，一名小女孩在門縫中窺伺著蘭貝爾。
從身高來看，似乎連十歲都不到。
赤紅色的頭髮亂糟糟地，身上穿著用破布東拼西湊的衣服。
。
蘭貝爾看向她時，誠惶誠恐地窺伺著蘭貝爾的孩子臉上綻放出笑容。
。

「是騎士大人！是騎士大人嗎！您是來打敗那些壞人的對吧？」

。
蘭貝爾走進那名少女。
少女開門來到蘭貝爾的面前。
。
從到達中央都市庫蘭的貧民窟以來，反響最好的就是這孩子。
也許能打聽到什麼吧，蘭貝爾的心中充滿了這樣的期待。
。

「唔姆，沒錯。我就是雷吉歐斯王國四魔將之一，蘭貝爾·德拉克羅瓦」

。

「四麻將……？雖然不是很懂，不過好厲害的樣子！好帥的名字啊！」

。

「你也這麼想吧」

。
蘭貝爾深深地點了點頭。
。

「蘭貝爾大人，是來這裡打敗小丑的嗎？」

。

「小丑……？那是誰」

。

「爸爸說過的哦。自從小丑到訪領主大人的公館之後，這裡就開始變得奇怪了……所以，騎士大人……」

。
在少女說完之前，蘭貝爾往背後打了一拳。
有人從蘭貝爾的背後，以他的肩膀為目標揮下了木棒。
蘭貝爾以拳頭折斷了木棒，緊接著放出的貫通掌掠過襲擊者的臉頰，貫穿了牆壁。
襲擊者發出短短的悲鳴後便跌倒在地上。
接著蘭貝爾伸出手指，高高地舉起籠手。
只是輕輕一碰就能輕易地粉碎人類的頭骨，這從那面被漂亮地打出洞穴的牆壁上已經一目了然了。
。

「爸、爸爸……？」

。
少女向襲擊者開口。
蘭貝爾觀察了襲擊自己的男人，
那是一名長著亂糟糟的紅髮的、不修邊幅的男人。
他的眼皮底下，不知是因為骯髒還是因為皮膚已經发灰，使他看上去有點衰老，但年齡估計在三十歲前後。
。
襲擊者絕望地跪在原地。
然後就像是生了悶火一樣，一邊俯視著地面一邊小聲嘀咕。
。

「亞爾蒂……都說過幾次了，不要給陌生人開門……」

。
看來少女名為亞爾蒂，而襲擊者是她的父親。
。

「給我解釋一下。根據你的回答而保留你的處分」

。
蘭貝爾故意泄露出瘴氣，向男人盤問。
先不說勇猛的戰士，這份迫力是普通人絕對不可能承受得住的。
在泄露出瘴氣的蘭貝爾的壓迫感面前說謊的人是不存在的。
。
男人很快伏倒在地，向蘭貝爾低頭。
。

「要、要殺要剮悉聽尊便！但是、請您放過我女兒吧……！自從妻子去世之後，她便是我生存的唯一的意義！」

。

「…………」

。
蘭貝爾俯視著男人，慢慢地放下手臂。
。

「瀕臨死亡，最先乞求的是女兒的性命嗎。真是了不起的骨氣。好好珍惜吧」

。
男人連發生了什麼都不知道，膽顫心驚地抬起頭仰視著蘭貝爾。
。
(不過，這份畏懼也太過異常了吧）

。
無論是老人，還是亞爾蒂的父親，反應都過於劇烈。
看來貧民窟裡的人都受到過巨大的心靈創傷呢。
。
這時，從某處傳來了金屬摩擦的聲音。
是鎧甲的聲音，蘭貝爾立刻就察覺了。
蘭貝爾望向聲音的方向，發現了三名武裝統一的鎧劍士，並且在那先頭還有一名戴著兜帽、戴著刻畫了記號的白面罩的魔術師。
。

「什麼嘛，不是還有小孩子嗎」
「就算狩獵多少次，也會有殘存下來的呢」
「就跟土蛙一樣呢。馬上又會增加的啦」

。
鎧劍士們笑言。
在統一的武器上，刻入了同樣的紋章。
。
(是拉戈爾子爵的私兵嗎。然而，先頭的魔術師又是什麼人？無論是惡趣味的外套也好面罩也好，都不覺得是貴族私兵所有的物品）

。

「這可不行啊。為了復興貧困的領地，在領地指定區域內生活的年幼的孩子們，必須經由我們的指引，實施最低限度的教育，並作為勞動力派遣到鄰近的各地。為了領地也請大家忍痛割愛並協助我們」

。
面罩的魔術師說明，並故意地張開手臂，感嘆地搖了搖頭。
。
(原來如此，這就是領主的士兵拐賣人口事件的真面目嗎。說得倒是挺好聽，但不服從的話，領主就會對貧民窟增加奴隷稅吧）

。
蘭貝爾將意識轉移到背負著的大劍。
準備好在對方出手的瞬間立即拔刀。
