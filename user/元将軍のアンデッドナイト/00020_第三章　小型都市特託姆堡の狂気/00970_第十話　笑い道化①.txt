「……沒想到，我居然是被救的一方。不過預料之外的是，只是稍微地深入敵陣，居然就遇到行踪不明的高位鍊金術師呢。這個地方已經超乎了我的想象。這就是『吹笛之惡魔』嗎」

。
克羅伊茨一邊抱著昏迷的孩子，一邊深深地嘆了口氣。
浸泡在血泊中的基爾多姆的上半身映入他的眼簾。
。

「基爾多姆·基雷納斯……為何，像他這樣高位的鍊金術師，會加入『吹笛之惡魔』……。他理應被多數人所認同，過著不愁吃穿的自由生活才是」

。

「叫克羅伊茨的。你說過你是王國的兵團一員沒錯吧」

。
蘭貝爾在克羅伊茨背後向他開口。
。

「……你到底是什麼人。究竟有何目的？如此強大的劍士，我是沒理由不知道的。而且，那身鎧甲……居然能輕易地將基爾多姆的魔術抵消，難不成……」

。

「我的話怎樣都好。我們的目的差不了多遠，你也已經察覺到了吧。那樣就可以了。時間緊迫。是進攻亦或是防守，拉戈爾子爵應該會採取某種行動的。比起這些，我更需要情報。克羅伊茨，你到底在這裡調查什麼？」

。

「……秘密，其實也說不上。況且在表明了身份的如今，也沒必要隱瞞情報了」

。
克羅伊茨的視線落在自己懷中的孩子。
。

「拉戈爾子爵實在是可疑。原本只是來確認，他是否存在著與『吹笛之惡魔』有瓜葛的痕跡……但來到這裡才知道，由於領主的指示，大規模的兒童誘拐正進行得如日中天」

。
這件事蘭貝爾也親眼目睹了。
這是在貧民窟裡走兩步就能知道的情報。
。

「而且問題是……盡管實施了如此大規模的誘拐，卻完全沒有將孩子們帶出境外的痕跡。若是將數十、數百的孩子長距離運輸的話，就不得不做好充分的準備。當然，與此相關的痕跡也理應存在，然而卻沒有發現一絲類似的痕跡。話說回來，即使在這片領地販賣奴隷，買家也是過於稀少。從外面的領地迎來了大富豪吧，似乎也沒有這一回事。被誘拐的孩子究竟在何處，如今仍不清楚」

。

「……原來如此，的確很可疑呢」

。
領主誘拐孩子的原因，現在還不明確。
。
以基爾多姆為前鋒，領內已經混入了相當數量的『吹笛之惡魔』的人員，這是毫無疑問的。
想要孩子的是『吹笛之惡魔』，而拉戈爾子爵只是被『吹笛之惡魔』命令而行動，這個路線的可能性很大。
。
(收集孩子的反國家魔術組織嗎……。有些許，難以想象呢）

。
八國統一戰爭時代的雷吉歐斯王國的鍊金術師、賢者多米莉奧涅在蘭貝爾腦海中一閃而過。
她為了用屍體制作格雷姆，曾經毀滅了數個村落。
。

「總之……我先帶領部下，暫時返回王都，請求其他兵團的幫助。你的話……雖然身手不凡，但別太勉強了。畢竟在此地潛伏著什麼都不出奇……」

。

「守株待兔，只會放跑敵人。不如說，我更理解不了為什麼你還這麼從容不迫。縱容他們的話，領民們必然會持續飽受他們的折磨。敵人也不愚蠢。我認為，這個地方潛伏著『吹笛之惡魔』的頭目八賢者。打敗他們的機會僅限於現在」

。

「八、八賢者？他們並不是單槍匹馬就能解決的對手！他們就是活生生的災厄！你或許不知道都市巴萊拉的事件。將一個都市陷入瀕臨滅亡的那個事件，其實是八賢者之一的『屍之丑老』曼奇和他的部下們一手造成的慘劇！只靠一個偵察部隊是不可能抗衡的！」

。
雖說單槍匹馬討伐掉曼奇的就是蘭貝爾本人，但克羅伊茨並不知情。
都市巴萊拉中跨著無頭馬的不死騎士的英雄譚雖然存在，在外面的人似乎都不相信這個故事。
相傳是領主的私兵和冒險者們合力，最終付出巨大的犧牲才討伐掉曼奇。
。

「還以為是個有點骨氣的傢伙呢，真是遺憾」

。
對於蘭貝爾的話語，克羅伊茨悔恨地咬著嘴唇。
。

「勇氣和魯莽不同。一不小心刺激到八賢者的話，也不知道會發生什麼事。比起這些，確切地剝奪拉戈爾的統治權，對受害者施以救濟才更重要吧。不是嗎？」

。
蘭貝爾沒有回應，而是背對著克羅伊茨開始遠去。
蘭貝爾的態度，不用說是已經對他失去了興趣。
。

「等、等下，接下來你打算去哪！」

。

「收集情報。我想獲得被誘拐的孩子的所在地，以及來訪這個地方的八賢者的有關的情報」

。

「然而……私兵並不知道上頭的事情。雖然在別的地方捕獲了幾名『吹笛之惡魔』的手下……但他們口風太緊。關於孩子們的所在之處，我的部下們也正在調查。也許會抓住什麼線索。得到情報時再通知你也可以……」

。

「我可等不了這麼長的時間。而且我已經找到了最佳的人選。他是熟知『吹笛之惡魔』在這個領內的計劃……同時，也明哲保身，是個將自己的性命優先于機密的男人」

。

「怎麼可能有這麼湊巧的人選……」

。

「我說的是拉戈爾子爵。直接沖進公館問他不就好了」

。
克羅伊茨沉默片刻。
難以判斷蘭貝爾究竟是在開玩笑，還是認真的。
克羅伊茨觀察蘭貝爾周圍的氣氛半晌，然後終於察覺『看樣子是認真的』。

。

「不、不不不！不行，這怎麼說都太過無謀了！那可是敵人的根據地啊！？也不知道有多少『吹笛之惡魔』的人……不，豈止如此，還有和八賢者相遇的危險！剛才也說過了吧？八賢者只憑一人之力就足以毀滅一個都市啊！」

。
背對著克羅伊茨的吶喊，蘭貝爾又邁開了步伐。
。

「讓、讓平民冒險，本人、『不死鳥之瞳』的克羅伊茨怎能揚長而去！那、那麼，我也隨你同去！」

。
克羅伊茨一邊抱著孩子，一邊用另一只手拿起細劍……但下一瞬間卻松手了。
他的指尖仍在痙攣。
由於在和蘭貝爾的戰鬥中使出的劍技的副作用，對手腕造成了急劇的負荷。
。

『倍連穿』原本就是對手腕不太友好的劍技。
在『三倍連穿』的節眼上，手腕就已經痛苦不堪。
然而，認為中斷連擊就會被敵人殺死的克羅伊茨沒能在中途停下，接連使出『四倍連穿』、『五倍連穿』，最終到達了『七倍連穿』的境界。
。

「咕，要是和我部下的白魔術師匯合的話，這種程度的損傷就不值一提。所以，可以稍等片刻……」

。

「比起這個，你先去把這個的孩子送去他的家人那裡吧」

。

「誒？」

。
留下凌亂著的克羅伊茨和失去意識的孩子，蘭貝爾迅速地離開了現場。
克洛伊身為王國調查兵團『不死鳥之瞳』的第二部隊長，雖說自己也是那麼地我行我素，但從未想過居然會有被人拜托照看小孩的一天。
