『死之天使』的副團長特尼雷爾正在通過特托姆堡的地下研究設施的走廊。
兩名身穿禮服的美少年跟隨在其左右。
。
特尼雷爾的臉上已經失去了以往那從容不迫的笑容。
他怒髮衝冠，太陽穴一抽一搐地。
他咬牙切齒地，下唇滲著一滴滴的鮮血。
。
「……特尼雷爾大人，您現在正打算進入地下，換言之就是要改變計劃對吧？」
。
身穿執事服的美少年洛比克向特尼雷爾詢問。
。
特尼雷爾原本的計劃是盡早地離開特托姆堡，在拉戈爾子爵領全土大開殺戒，然後著手轉移的準備，在王國的主要戰力到達之前撤退。
是特尼雷爾自己對駐紮在特托姆堡的魔術師們這麼說的。
。
特尼雷爾就像是觸電一樣地哆嗦了一下肩膀，接著他停止了前進，一把抓住了洛比克的腦袋，伸長脖子並把臉湊了過去。
。
「閉嘴，洛比克。知道之後你又能怎樣？」
。
「……非常抱歉，特尼雷爾大人」
。
洛比克並沒有將恐懼表現在臉上，只是平淡地回答特尼雷爾。
特尼雷爾往下一看，發現洛比克的手正在瑟瑟發抖時，他那緊張的臉上也擠出了一絲微笑。
接著他慢慢地把臉湊近，舔舐了洛比克的臉頰。
。
「沒錯，就是這樣洛比克。不許把表情刻在臉上。你也知道，我最討厭的就是煩人的小鬼對吧？」
。
他滿意地笑了笑，恢復到原來的姿勢。
然後開始盤算今後的計劃。
。
(讓我想想……本來打算在血洗拉戈爾子爵領之後揚長而去的，算了，沒時間了。收到隸屬於哈巴內的狂戰士部隊被殲滅的情報的時候我還以為是無稽之談……然而在『死之天使』中，那些一人就相當於一支魔術部隊的上位魔術師當中，已經有好幾人失去了聯絡。特托姆堡內似乎正在發生著某種異常的事情。雖說調查兵團的先遣部隊之流不值一提……看來，是我低估了他們的實力。說起來，雖然我沒怎麼認真聽，八賢者之一『嬉笑道化』的露露可大人也說過自己曾經遭遇到怪物了。看來有必要再次向她打聽一下情報呢）
。
特尼雷爾察覺到已經有異物混進了特托姆堡內。
。
(盡管在演講上聲稱要毀滅拉戈爾子爵領……但總覺得，有種不好的預感。身為『死之天使』的團長，八賢者『真理的紡織者』大人竟然在這個時間點上不見踪影，真是麻煩透頂了）
。
在特尼雷爾看來，八賢者『真理的紡織者』其實就是個人格分裂者。
不僅自我中心、自由奔放、神出鬼沒，而且性情不定，連言行也不一致。
與其說是人類，不如說他是名為天災的災害現象更加合適。
。
特尼雷爾聳肩並搖了搖頭。
一想到要去說服那個在這種時候還這麼陰晴不定的『真理的紡織者』，特尼雷爾就覺得煩躁。
前一任副團長也曾說服過『真理的紡織者』，不但沒有成功，而且還惹怒了他，結果前任副團長也在那一天從特托姆堡中消失了。
。
(在地上隨便配置的都只是下位魔術師。而地下設施就靠上位魔術師來鞏固防守，誘敵深入，一網打盡。在這裡還能投入大量的實驗體。而且這裡還是狹長的走廊。不僅是夾攻的好地方，而且對付那些只能一直深入未知的敵陣的敵人也有著心理上的優勢。雖然不知道來者何人，但特只能竭盡全力去迎擊了。也許並不需要這麼警備……夷平，我要將這個特托姆堡夷為平地。我要讓你們在絕望和恐怖、以及無止境的後悔中痛苦地死去）
。
◆
。
弗蘭達爾與他的人間家具一同被蘭貝爾切斷，血和內臟散落一地。
蘭貝爾走近弗蘭達爾的屍體，又砍了一刀，將分成兩半的腦袋敲碎。
。
「呃、那個，非常感謝您救我們一命……噫！」
。
被蘭貝爾拯救後，亞爾安特抱起了艾麗澤，把她安頓在輪椅上。
接著他推著輪椅，從背後誠惶誠恐地走近恩人，卻恰好碰見了蘭貝爾在鞭屍的場景。
艾麗澤也睜開眼睛，不由自主地用手捂住了想要大叫的嘴巴。
。
「這是以防萬一。這些魔術師不會那麼簡單就死掉的。失去了下半身而且還被埋在瓦礫之下，重逢時卻以四肢健全的姿態出現的傢伙我也曾遇過」
。
「誒？啊、啊哈哈哈……那種像故事裡一樣的怪物怎麼可能……」
。
稍加思索後，以為蘭貝爾在開玩笑的亞爾安特尷尬地笑了。
。
失去了下半身的話，即使做好周全的準備也無力回天。
再加上被埋在瓦礫之下的話，不僅耽誤了救援，而且以人類的身體也承受不了瓦礫的重量。
倘若不是在非常特殊的情況下，很難得以救治。
頂多是成為知性薄弱的不死者再次蘇醒吧。
。
「當然，我當場就把她的頭蓋骨給打碎了」
。
蘭貝爾理所當然地回答。
亞爾安特那尷尬的笑容戛然而止。
。
「說起來，你不是小孩子呢。看來你不是這裡的實驗體。而且穿著的法袍也和他們的一樣……」
。
「不、不不，我我我，我是，那個……！」
。
亞爾安特臉色刷白，慌慌張張地擺著手。
這樣下去他就要給弗蘭達爾陪葬了。
。
看見亞爾安特那手舞足蹈的樣子，艾麗澤小聲地笑了。
由於弗蘭達爾的死亡，之前被弗蘭達爾逼入絕境而緊綳了神經的他們也終於放鬆了下來。
。
大概過了一分鐘後，艾麗澤才冷靜下來。
沐浴在蘭貝爾和亞爾安特兩人的視線之下，艾麗澤害羞地臉紅了。
。
「不、不好意思……因為平時都沒怎麼笑，所以就情不自禁地……。大、大哥哥是個很厲害的人哦。因為頭腦聰明所以才被那些可怕的人帶來這裡，所以……」
。
艾麗澤一邊低著頭來掩飾自己害羞的樣子，一邊袒護著亞爾安特。
。
「看那邊，是蘭貝爾大人！」
「太好了，終於發現蘭貝爾大人了！」
。

調查兵團『不死鳥之瞳』第二部隊的士兵們陸續從聲音傳來的方向出現。
蘭貝爾和他們一起行動，一邊打倒『吹笛之惡魔』的魔術師一邊前進，突然蘭貝爾那作為不死者的魔力感知能力對亞爾安特等人產生反應，察覺到異常的蘭貝爾於是只身一人前往了他們的方向。
。
「鎧之劍士喲，請不要單獨行動了啊！要是部下們的士氣下降了你怎麼負責！」
。
克羅伊茨在路途上也早已被蘭貝爾感染，他用一半玩笑一半認真的語氣責備了獨斷專行的蘭貝爾。
。
「……那本來是您的任務哦，克羅伊茨大人」
。
克羅伊茨的屬下弗雷站在他的身旁，無奈地嘆了口氣。
