全身被棱镜覆盖的怪异之龙潜入了那扭曲空间的阴影之中。
龙头完全被吞没的同时，黑色的阴影仿佛不曾存在过一样变得朦胧，逐渐消失。
。
「……嘿诶，竟然能完全防住镜界之妖精龙的火焰。真有趣」
。
银髮的女子睁开眼皮，一边舔舐着上唇一边凝视着兰贝尔。
。
「魔矿石……而且还是，世界上最优秀的重金属，魔金奥尔钢。连魔银都能熔断的妖精龙之炎都能抵挡，那麼那副铠甲中的魔金含有量必然不低。穿着那麼笨重的玩意还能走动什麼的，就像是行走的魔导兵器一样呢」
。
「竟、竟然是魔金奥尔钢！？」
。
对女子的发言感到惊讶的反而是克罗伊茨。
作为稀有金属的魔金，是一种只会在极富含魔力的场所的中央局部生成的金属。
只是，历史上幾乎没有发现一整块魔金的个例。
。
即便如此，传说只要挖出一克就等於一辈子都可以衣食无忧的魔金奥尔钢从八国统一战争之前，各国就已经开始竞相挖掘。
到了现在这个时代，各个地方的魔金都已经被采掘一空，有人说，在谁也无法踏足、现已成为了魔兽们的乐园的地底之下，仍沉眠着大量的魔金。
。
「使用大量魔金的铠甲什麼的，实在是让人无法想象……铠之剑士，阁下，到底是在何处得到……？」
。
「此乃吾忠义之证」
。
兰贝尔向银髮女子举起大剑。
。
「很抱歉剑士先生，我可是很厉害的哦？剑士先生看上去那麼有趣，我都有点想放你一马了呢……」
。
兰贝尔没有反应。
女子装模作样地叹了口气。
。
「亚琉斯，这是我的名字哦。虽然，更多人称呼我为『异界的具现者』、『镜龙之女王』呢。能防住那一击，就证明您的实力不容忽视呢。虽说规模不大，但我也曾攻陷过一个街道哦」
。
「即使对手是邪道，也必须报上自己的名号。兰贝尔·德拉克罗瓦。在我还是部队长的时候，曾用一支部队攻陷了一座城池。虽然那场战鬥并不轻鬆」
。
说着，兰贝尔便开始行动。
他径直地缩短与亚琉斯之间的距离。
。
「您意外地有幽默感呢。不过，如果您真的是传说中的大英雄，那就试试撑过这一击吧」
。
亚琉斯高举手臂。
四重魔法阵同时展开，使空间产生了歪曲。
从四处歪曲中，四条镜龙——镜界之妖精龙一齐探出了脑袋。
。
「镜界的精灵在这边的世界显现之时，只要支付相应的魔力，就能复制出相应数量的个体哦。严格来说，虽然看起来就像是同时存在一样，但这并不是正确的说法呢。应该叫镜像？不过，妖精龙的吐息、利爪，对我们来说绝对不是什麼幻觉哦」
。
四条妖精龙各自把头朝向兰贝尔，并同时张嘴。
真红的光芒在它们的喉咙深处若隐若现。
。
「再见了，大英雄。就算那副铠甲再厲害，被活埋的話也束手无策吧？」
。
四条龙一齐放出的业火往兰贝尔的方向袭来。
。
亚琉斯又舔了舔上唇。
。
妖精龙之炎拥有着将地面轻易击穿的威力。
即使兰贝尔的铠甲多麼强韧，站着的地方崩塌的話，他也只能无奈地掉到洞裡。
待他坠落到最深处之後，活埋也行，用其他魔术也行，总之想怎样都可以。
亚琉斯瞄准的就是这一点。
。
然而，兰贝尔一跃而起，在半空变换持剑的方式来操纵重心，见缝插针般地躲避火焰。
身穿巨大的全身铠却拥有着难以置信的灵巧性。
即使爆炸後的热浪近在咫尺，兰贝尔也不以为意，继续往前冲。
。
兰贝尔那毫无时间损失、已经达到艺术级别的一连串动作，让亚琉斯惊叹不已。
。
「不、不是吧……难道说，他真的是叛逆的大英雄……？」
。
本应必中的攻击却全部落空，这也许只是自己失误了吧，她不禁这麼想到。
。
「不，即使有多少能耐，你也只会被我打倒！是你逼我出全力的！」
。
亚琉斯高举双手。
与此同时，上空又一次出现扭曲。
。
但这次的扭曲和之前的规模不一样。
之前的扭曲直径最多也只有一米左右的长度，但这一次的直径长度却超过了五米。
接着，与直径相应的巨大化妖精龙的脑袋穿过次元之洞，在悬空中显现。
。
「镜界之龙的存在概念，和这边的世界不同。因此能够同时存在多体……而且，在我能够控制的範围内，连体积也能膨胀幾倍！」
。
亚琉斯双目充血，从鼻孔中流淌着鲜血。
脸上浮现着血管。
这就是急剧地消耗玛娜的弊端。
。
「不仅是规模。妖精龙的这一击的速度堪比落雷！能躲开的話就试试吧！」
。
波纹扩散至巨大化妖精龙的全身。
紧接着，妖精龙的口中散發出强烈的光芒。
真红的光束从巨大化妖精龙的口中发射，直迫兰贝尔。
。
「哈啊啊啊啊啊啊啊啊！」
。
兰贝尔以大剑一斩。
妖精龙之炎与兰贝尔的大剑交错，光芒四射。
。
最後只剩下被暴风摧残的大地，以及依然无伤的兰贝尔。
巨大化妖精龙的吐息，被兰贝尔的一闪分成两截，因失去了方向性而散射，进而流失了所有的威力。
。
「不、不是吧……竟然把那束光线、劈开了？」
。
亚琉斯愕然失色。
这不可能。
然而，这件不可能的事情就在眼前發生。
。
「……这次就先撤退好了。再会吧，自称兰贝尔的剑士。美人的邀请可是不能拒绝的哟」
。
亚琉斯身後的空间發生歪曲，从中出现的妖精龙把她的身体吞没在阴影中。
。
这就是亚琉斯最後的手段——强行用镜龙进行的空间转移。
进入妖精龙的口中，和妖精龙一起穿梭异界，转移到与这个世界完全不同的坐标处。
穿梭异界时必须用结界保护自己的肉身，且玛娜的消耗必然激增，但即使存在着这样的缺点，这份力量也是异常强大的。
亚琉斯性格慎重，只要有这个空间转移技能，就不可能给予她致命性的打击。
。
「哈啊啊啊啊啊啊啊！」
。
在咆哮的同时兰贝尔往地一踢并砍下大剑。
刚吞下亚琉斯的妖精龙的脸上，闪过了一道纵向的斩击。
体表上泛起波纹，妖精龙的头部被一刀两断。
。
脑袋被切成左右两半的妖精龙發出难以形容的惨叫，外形崩溃，犹如被背後的黑洞吸走般地逐渐消失。
最後，只留下连同妖精龙一同、被从头切到尾的亚琉斯的尸体。
死去的亚琉斯的脸上只有惊愕。
。
「抱歉呢，我性格比较急躁」
