全身被棱鏡覆蓋的怪異之龍潛入了那扭曲空間的陰影之中。
龍頭完全被吞沒的同時，黑色的陰影仿佛不曾存在過一樣變得朦朧，逐漸消失。
。
「……嘿誒，竟然能完全防住鏡界之妖精龍的火焰。真有趣」
。
銀髮的女子睜開眼皮，一邊舔舐著上唇一邊凝視著蘭貝爾。
。
「魔礦石……而且還是，世界上最優秀的重金屬，魔金奧爾鋼。連魔銀都能熔斷的妖精龍之炎都能抵擋，那麼那副鎧甲中的魔金含有量必然不低。穿著那麼笨重的玩意還能走動什麼的，就像是行走的魔導兵器一樣呢」
。
「竟、竟然是魔金奧爾鋼！？」
。
對女子的發言感到驚訝的反而是克羅伊茨。
作為稀有金屬的魔金，是一種只會在極富含魔力的場所的中央局部生成的金屬。
只是，歷史上幾乎沒有發現一整塊魔金的個例。
。
即便如此，傳說只要挖出一克就等於一輩子都可以衣食無憂的魔金奧爾鋼從八國統一戰爭之前，各國就已經開始競相挖掘。
到了現在這個時代，各個地方的魔金都已經被採掘一空，有人說，在誰也無法踏足、現已成為了魔獸們的樂園的地底之下，仍沉眠著大量的魔金。
。
「使用大量魔金的鎧甲什麼的，實在是讓人無法想象……鎧之劍士，閣下，到底是在何處得到……？」
。
「此乃吾忠義之證」
。
蘭貝爾向銀髮女子舉起大劍。
。
「很抱歉劍士先生，我可是很厲害的哦？劍士先生看上去那麼有趣，我都有點想放你一馬了呢……」
。
蘭貝爾沒有反應。
女子裝模作樣地嘆了口氣。
。
「亞琉斯，這是我的名字哦。雖然，更多人稱呼我為『異界的具現者』、『鏡龍之女王』呢。能防住那一擊，就證明您的實力不容忽視呢。雖說規模不大，但我也曾攻陷過一個街道哦」
。
「即使對手是邪道，也必須報上自己的名號。蘭貝爾·德拉克羅瓦。在我還是部隊長的時候，曾用一支部隊攻陷了一座城池。雖然那場戰鬥並不輕鬆」
。
說著，蘭貝爾便開始行動。
他徑直地縮短與亞琉斯之間的距離。
。
「您意外地有幽默感呢。不過，如果您真的是傳說中的大英雄，那就試試撐過這一擊吧」
。
亞琉斯高舉手臂。
四重魔法陣同時展開，使空間產生了歪曲。
從四處歪曲中，四條鏡龍——鏡界之妖精龍一齊探出了腦袋。
。
「鏡界的精靈在這邊的世界顯現之時，只要支付相應的魔力，就能複製出相應數量的個體哦。嚴格來說，雖然看起來就像是同時存在一樣，但這並不是正確的說法呢。應該叫鏡像？不過，妖精龍的吐息、利爪，對我們來說絕對不是什麼幻覺哦」
。
四條妖精龍各自把頭朝向蘭貝爾，並同時張嘴。
真紅的光芒在它們的喉嚨深處若隱若現。
。
「再見了，大英雄。就算那副鎧甲再厲害，被活埋的話也束手無策吧？」
。
四條龍一齊放出的業火往蘭貝爾的方向襲來。
。
亞琉斯又舔了舔上唇。
。
妖精龍之炎擁有著將地面輕易擊穿的威力。
即使蘭貝爾的鎧甲多麼強韌，站著的地方崩塌的話，他也只能無奈地掉到洞裡。
待他墜落到最深處之後，活埋也行，用其他魔術也行，總之想怎樣都可以。
亞琉斯瞄准的就是這一點。
。
然而，蘭貝爾一躍而起，在半空變換持劍的方式來操縱重心，見縫插針般地躲避火焰。
身穿巨大的全身鎧卻擁有著難以置信的靈巧性。
即使爆炸後的熱浪近在咫尺，蘭貝爾也不以為意，繼續往前沖。
。
蘭貝爾那毫無時間損失、已經達到藝術級別的一連串動作，讓亞琉斯驚嘆不已。
。
「不、不是吧……難道說，他真的是叛逆的大英雄……？」
。
本應必中的攻擊卻全部落空，這也許只是自己失誤了吧，她不禁這麼想到。
。
「不，即使有多少能耐，你也只會被我打倒！是你逼我出全力的！」
。
亞琉斯高舉雙手。
與此同時，上空又一次出現扭曲。
。
但這次的扭曲和之前的規模不一樣。
之前的扭曲直徑最多也只有一米左右的長度，但這一次的直徑長度卻超過了五米。
接著，與直徑相應的巨大化妖精龍的腦袋穿過次元之洞，在懸空中顯現。
。
「鏡界之龍的存在概念，和這邊的世界不同。因此能夠同時存在多體……而且，在我能夠控制的範圍內，連體積也能膨脹幾倍！」
。
亞琉斯雙目充血，從鼻孔中流淌著鮮血。
臉上浮現著血管。
這就是急劇地消耗瑪娜的弊端。
。
「不僅是規模。妖精龍的這一擊的速度堪比落雷！能躲開的話就試試吧！」
。
波紋擴散至巨大化妖精龍的全身。
緊接著，妖精龍的口中散發出強烈的光芒。
真紅的光束從巨大化妖精龍的口中發射，直迫蘭貝爾。
。
「哈啊啊啊啊啊啊啊啊！」
。
蘭貝爾以大劍一斬。
妖精龍之炎與蘭貝爾的大劍交錯，光芒四射。
。
最後只剩下被暴風摧殘的大地，以及依然無傷的蘭貝爾。
巨大化妖精龍的吐息，被蘭貝爾的一閃分成兩截，因失去了方向性而散射，進而流失了所有的威力。
。
「不、不是吧……竟然把那束光線、劈開了？」
。
亞琉斯愕然失色。
這不可能。
然而，這件不可能的事情就在眼前發生。
。
「……這次就先撤退好了。再會吧，自稱蘭貝爾的劍士。美人的邀請可是不能拒絕的喲」
。
亞琉斯身後的空間發生歪曲，從中出現的妖精龍把她的身體吞沒在陰影中。
。
這就是亞琉斯最後的手段——強行用鏡龍進行的空間轉移。
進入妖精龍的口中，和妖精龍一起穿梭異界，轉移到與這個世界完全不同的坐標處。
穿梭異界時必須用結界保護自己的肉身，且瑪娜的消耗必然激增，但即使存在著這樣的缺點，這份力量也是異常強大的。
亞琉斯性格慎重，只要有這個空間轉移技能，就不可能給予她致命性的打擊。
。
「哈啊啊啊啊啊啊啊！」
。
在咆哮的同時蘭貝爾往地一踢並砍下大劍。
剛吞下亞琉斯的妖精龍的臉上，閃過了一道縱向的斬擊。
體表上泛起波紋，妖精龍的頭部被一刀兩斷。
。
腦袋被切成左右兩半的妖精龍發出難以形容的慘叫，外形崩潰，猶如被背後的黑洞吸走般地逐漸消失。
最後，只留下連同妖精龍一同、被從頭切到尾的亞琉斯的屍體。
死去的亞琉斯的臉上只有驚愕。
。
「抱歉呢，我性格比較急躁」
