蘭貝爾率領著『不死鳥之瞳』的第二部隊，已經闖入了特托姆堡的內部。
阻擋在蘭貝爾他們的面前的，是『吹笛之惡魔』的魔術師團，人數約為二十。
。
『不死鳥之瞳』的大部分士兵都已經失去了坐騎。
有的在被敵人的魔術重創後落荒而逃，也有的為了保護主人而身亡命殞。
甚至可以稱之為死鬥。
然而奇跡的是『不死鳥之瞳』一方仍未損失一卒。
。
『不死鳥之瞳』那毫不迷茫的進擊，在正打算部署迎擊之陣的『吹笛之惡魔』魔術師看來簡直是不可思議。

『不死鳥之瞳』的第二部隊，包括隊長克羅伊茨在內共十一人。
算上蘭貝爾也只有十二人。
單是在場的魔術師的人數也有蘭貝爾他們的兩倍左右。
。
「這、這些傢伙是怎麼一回事……？為什麼，在這種情況下還能若無其事地來這裡……？」
。
魔術師們驚訝地眺望著蘭貝爾一行人。

『吹笛之惡魔』雖然是由走火入魔的魔術師組成，但在他們眼中，現在的蘭貝爾和他的同伴似乎都已經失去了冷靜。
。
「上吧，侍奉當今王家的人們啊！以劍展示你們的忠誠，將為國家帶來災難的敵人殲滅！」
。
蘭貝爾的吶喊在特托姆堡內回蕩不絕。
他完全支配了克羅伊茨的部隊，成為了名副其實的指揮官。
以蘭貝爾的話語為信號，劍士們開始放聲大吼並以劍指天，連克羅伊茨也在其中。
。
「真是些惡心的傢伙……別讓他們靠近，在我等的魔術範圍內收拾他們！首先是作為敵人士氣源泉的那個大鎧……」
。
一名『吹笛之惡魔』的魔術師向周圍發起號召。
魔術師們舉起錫杖，一齊向蘭貝爾的方向施放炎球和冰塊。
而蘭貝爾開始了沖刺並行如流水地反復揮舞大劍，將炎球吹滅，將冰塊敲碎。
無窮無盡的魔彈均被蘭貝爾的大劍一一擊落。
。
「喂、喂，根本攔不下來啊！」
「別停手，給我繼續打！被他接近的話就只有死路一條！」
。

以蘭貝爾為目標的魔彈愈發猛烈，如暴雨般傾瀉。
然而即便如此，也沒有一發能夠擊中蘭貝爾的身軀。
話雖如此，即使魔彈命中了蘭貝爾，恐怕也無法對魔金鎧造成一絲傷痕……。
。
「這次也是我來打頭陣，將敵人的陣型切斷。在陣型被打亂的時候，和上次一樣將敵人逐一擊破」
。
與此同時，蘭貝爾的速度瞬間提升至極限。
在急速靠近的魔人面前，『吹笛之惡魔』的魔術師們狼狽不堪。
蘭貝爾沒有錯失這個機會。
。
「哈啊啊啊啊啊！」
。
蘭貝爾雄叫並掄起大劍，突破了敵人單橫陣的中央部。
被卷入蘭貝爾的突進的魔術師被遠遠地吹飛。
。
然而他們並沒有發出尖叫。
正面承受了蘭貝爾突進的那三人早已粉身碎骨，在被吹飛的那一刻已經斷氣。
三具屍體難看地墜落，經重力加速後摔了在地上。
。
「噫、噫！」
。
對蘭貝爾感到畏懼的魔術師們自亂陣腳並開始四處逃竄。
。
「跟我一起上！將他們殺個片甲不留！」
。
遵從蘭貝爾的命令，『不死鳥之瞳』的士兵們向逃亡的魔術師們發起強襲。
若是正面交戰，『吹笛之惡魔』的魔術師們也許能利用人數的差距來取得勝利，然而在穿梭於戰場之中並釋放出不死者的瘴氣的蘭貝爾面前，他們的早已喪失了戰意。
。
從蘭貝爾手中逃竄出來的魔術師的頭頂上突然出現一個人影。
他是頭戴魔銀盔的部隊隊長克羅伊茨。
。
「雖然斬殺喪失戰意之徒並非我的本意，但我也決不能放虎歸山！」
。
克羅伊茨將男魔術師的腦袋踢飛，再度從空中對其他魔術師發起強襲。
克羅伊茨不給他們任何反擊的機會，在用細劍貫穿對方額頭的同時著陸，將細劍拔出並回旋一周，利用離心力刺穿了從背後襲來的敵人的胸膛。
克羅伊茨雖然被蘭貝爾順水推舟地奪走了指揮權，但與蘭貝爾的一戰以及在這絕境中的死鬥，令他的劍技更上一層樓。
。
以蘭貝爾作前鋒，擾亂敵人的陣型並挫敗敵人的戰意。
然後讓士氣高昂的『不死鳥之瞳』劍士們將四處逃竄的魔術師收拾乾淨。這一戰術在特托姆堡中完成並逐漸洗練。
。
不到眨眼之間魔術師們便四散而去，『不死鳥之瞳』的士兵們暫時迎來了休憩。
士兵們正互相分享在這次交戰後也奇跡般地生還的喜悅。
。
「大伙們，有人受傷了嗎！雖然不知道敵人下次會在什麼時候攻過來，但我會盡全力治好你們的！」
。
女劍士弗雷向士兵們呼喊。
受重傷的士兵搭著其他人人的肩膀，走向弗雷那邊。
蘭貝爾從不遠處觀察著他們的情況，忽然他發現克羅伊茨在往自己的方向靠近，便扭頭面對著他。
。
「有事么？」
。
「我原本，已經做好了在這個地方白白送死的覺悟……卻意外地吃得消呢。與『吹笛之惡魔』的魔術師們為敵，竟然能取得如此戰果，真是萬萬沒有想到。雖說，一大半的功勞都是閣下您的……」
。
克羅伊茨吐露了自己的心聲，並稱讚蘭貝爾的功績。
。
「對我來說，將他們殺光才是我想要的，但竟然放跑了四個人。下次的敵人，一定要給我全滅掉。向其他士兵傳令吧」
。
「啊、啊啊……我、我明白了……」
。
單憑一人便補足數量上的差距，且在不失一卒的情況下成功擊退敵人的蘭貝爾，卻並不滿意剛才的戰果。
。
「……鎧之劍士。您到底是何方神聖？實力如此高強的劍士，我竟從未聽聞，這怎麼想都太奇怪了。而且，您還擁有著鼓舞士氣、令己方發揮出正常以上實力的領袖魅力。比統領王家兵團部隊的我強大得多」
。
「我名為蘭貝爾。是雷吉歐斯王國的四魔將之一角，蘭貝爾·德拉克羅瓦」
。
經過數秒的沉默後，克羅伊茨露出苦笑。
。
「原來如此，八國統一戰爭的大英雄為對手，像我這種凡夫俗子怎麼說也沒有戰勝的情理。抱歉，我不應該打聽你的事情」
。
克羅伊茨以為蘭貝爾在開玩笑。
或許蘭貝爾自己也是抱著開玩笑的態度，才這麼輕易地報上名號。
。
「話說回來，鎧之劍士，蘭貝爾閣下……」
。
克羅伊茨開口的瞬間，他便感覺到蘭貝爾那強烈的殺氣。
克羅伊茨把話咽了下去，後退了幾步。
。
「啊，抱歉。有什麼地方冒犯到您的話……」
。
蘭貝爾徑直地往克羅伊茨的方向突進。
克羅伊茨沒能反應過來，被蘭貝爾伸出的手臂打到側腹，在地面上滾了約兩米遠的距離。
。
「咳哈！你、你幹什……」
。
克羅伊茨忍著側腹的疼痛抬起頭。
這時，一束大到能包住一個人的巨大火柱從天而降，直擊蘭貝爾。
蘭貝爾被火柱直擊，烈焰與暴風擴散，將地面掀起。
克羅伊茨屏住了呼吸。要不是被蘭貝爾保護，死的那個一定是自己。
。
「鎧、鎧之劍士！」
。
「首先解決掉一人。呀咧呀咧，真是麻煩，不就是王都兵團的一個部隊嘛，有必要讓我出手嗎。只不過是些雜魚被幹掉了而已嘛」
。
克羅伊茨面朝向射出火柱的方向並拔出細劍。
半空中，敞開著一個幽黑的空洞，就像是空間被其扭曲一樣。
此時，一頭猶如全身都被棱鏡覆蓋般地可怕的巨龍從黑洞中探頭。
如流水般的漣漪在棱鏡般的體表上蔓延。
。
一名慵懶的銀髮女人正站在在黑洞的正下方。
。
「鎧之劍士……這、都怪我……！」
。
克羅伊茨看向被煙霧埋沒的爆炸中心點，即是蘭貝爾剛才站著的地方。
。
「不要緊。比起這個，你平時應該更加小心附近的情況」
。
蘭貝爾寸步不離地聳立在推開克羅伊茨的地方，如仁王一般。
