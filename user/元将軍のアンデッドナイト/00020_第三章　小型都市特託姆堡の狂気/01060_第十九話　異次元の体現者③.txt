兰贝尔率领着『不死鸟之瞳』的第二部队，已经闯入了特托姆堡的内部。
阻挡在兰贝尔他们的面前的，是『吹笛之恶魔』的魔术师团，人数约为二十。
。
『不死鸟之瞳』的大部分士兵都已经失去了坐骑。
有的在被敌人的魔术重创后落荒而逃，也有的为了保护主人而身亡命殒。
甚至可以称之为死斗。
然而奇迹的是『不死鸟之瞳』一方仍未损失一卒。
。
『不死鸟之瞳』那毫不迷茫的进击，在正打算部署迎击之阵的『吹笛之恶魔』魔术师看来简直是不可思议。

『不死鸟之瞳』的第二部队，包括队长克罗伊茨在内共十一人。
算上兰贝尔也只有十二人。
单是在场的魔术师的人数也有兰贝尔他们的两倍左右。
。
「这、这些家伙是怎么一回事……？为什么，在这种情况下还能若无其事地来这里……？」
。
魔术师们惊讶地眺望着兰贝尔一行人。

『吹笛之恶魔』虽然是由走火入魔的魔术师组成，但在他们眼中，现在的兰贝尔和他的同伴似乎都已经失去了冷静。
。
「上吧，侍奉当今王家的人们啊！以剑展示你们的忠诚，将为国家带来灾难的敌人歼灭！」
。
兰贝尔的呐喊在特托姆堡内回荡不绝。
他完全支配了克罗伊茨的部队，成为了名副其实的指挥官。
以兰贝尔的話语为信号，剑士们开始放声大吼并以剑指天，连克罗伊茨也在其中。
。
「真是些恶心的家伙……别让他们靠近，在我等的魔术范围内收拾他们！首先是作为敌人士气源泉的那个大铠……」
。
一名『吹笛之恶魔』的魔术师向周围发起号召。
魔术师们举起锡杖，一齐向兰贝尔的方向施放炎球和冰块。
而兰贝尔开始了冲刺并行如流水地反复挥舞大剑，将炎球吹灭，将冰块敲碎。
无穷无尽的魔弹均被兰贝尔的大剑一一击落。
。
「喂、喂，根本拦不下来啊！」
「别停手，给我继续打！被他接近的話就只有死路一条！」
。

以兰贝尔为目标的魔弹愈发猛烈，如暴雨般倾泻。
然而即便如此，也没有一发能够击中兰贝尔的身躯。
話虽如此，即使魔弹命中了兰贝尔，恐怕也无法对魔金铠造成一丝伤痕……。
。
「这次也是我来打头阵，将敌人的阵型切断。在阵型被打乱的时候，和上次一样将敌人逐一击破」
。
与此同时，兰贝尔的速度瞬间提升至极限。
在急速靠近的魔人面前，『吹笛之恶魔』的魔术师们狼狈不堪。
兰贝尔没有错失这个机会。
。
「哈啊啊啊啊啊！」
。
兰贝尔雄叫并抡起大剑，突破了敌人单横阵的中央部。
被卷入兰贝尔的突进的魔术师被远远地吹飞。
。
然而他们并没有发出尖叫。
正面承受了兰贝尔突进的那三人早已粉身碎骨，在被吹飞的那一刻已经断气。
三具尸体难看地坠落，经重力加速后摔了在地上。
。
「噫、噫！」
。
对兰贝尔感到畏惧的魔术师们自乱阵脚并开始四处逃窜。
。
「跟我一起上！将他们杀个片甲不留！」
。
遵从兰贝尔的命令，『不死鸟之瞳』的士兵们向逃亡的魔术师们发起强袭。
若是正面交战，『吹笛之恶魔』的魔术师们也许能利用人数的差距来取得胜利，然而在穿梭于战场之中并释放出不死者的瘴气的兰贝尔面前，他们的早已丧失了战意。
。
从兰贝尔手中逃窜出来的魔术师的头顶上突然出现一个人影。
他是头戴魔银盔的部队队长克罗伊茨。
。
「虽然斩杀丧失战意之徒并非我的本意，但我也决不能放虎归山！」
。
克罗伊茨将男魔术师的脑袋踢飞，再度从空中对其他魔术师发起强袭。
克罗伊茨不给他们任何反击的机会，在用细剑贯穿对方额头的同时着陆，将细剑拔出并回旋一周，利用离心力刺穿了从背后袭来的敌人的胸膛。
克罗伊茨虽然被兰贝尔顺水推舟地夺走了指挥权，但与兰贝尔的一战以及在这绝境中的死斗，令他的剑技更上一层楼。
。
以兰贝尔作前锋，扰乱敌人的阵型并挫败敌人的战意。
然后让士气高昂的『不死鸟之瞳』剑士们将四处逃窜的魔术师收拾干净。这一战术在特托姆堡中完成并逐渐洗练。
。
不到眨眼之间魔术师们便四散而去，『不死鸟之瞳』的士兵们暂时迎来了休憩。
士兵们正互相分享在这次交战后也奇迹般地生还的喜悦。
。
「大伙们，有人受伤了吗！虽然不知道敌人下次会在什么时候攻过来，但我会尽全力治好你们的！」
。
女剑士弗雷向士兵们呼喊。
受重伤的士兵搭着其他人人的肩膀，走向弗雷那边。
兰贝尔从不远处观察着他们的情况，忽然他发现克罗伊茨在往自己的方向靠近，便扭头面对着他。
。
「有事么？」
。
「我原本，已经做好了在这个地方白白送死的觉悟……却意外地吃得消呢。与『吹笛之恶魔』的魔术师们为敌，竟然能取得如此战果，真是万万没有想到。虽说，一大半的功劳都是阁下您的……」
。
克罗伊茨吐露了自己的心声，并称赞兰贝尔的功绩。
。
「对我来说，将他们杀光才是我想要的，但竟然放跑了四个人。下次的敌人，一定要给我全灭掉。向其他士兵传令吧」
。
「啊、啊啊……我、我明白了……」
。
单凭一人便补足数量上的差距，且在不失一卒的情况下成功击退敌人的兰贝尔，却并不满意刚才的战果。
。
「……铠之剑士。您到底是何方神圣？实力如此高强的剑士，我竟从未听闻，这怎么想都太奇怪了。而且，您还拥有着鼓舞士气、令己方发挥出正常以上实力的领袖魅力。比统领王家兵团部队的我强大得多」
。
「我名为兰贝尔。是雷吉欧斯王国的四魔将之一角，兰贝尔·德拉克罗瓦」
。
经过数秒的沉默后，克罗伊茨露出苦笑。
。
「原来如此，八国统一战争的大英雄为对手，像我这种凡夫俗子怎么说也没有战胜的情理。抱歉，我不应该打听你的事情」
。
克罗伊茨以为兰贝尔在开玩笑。
或许兰贝尔自己也是抱着开玩笑的态度，才这么轻易地报上名号。
。
「話说回来，铠之剑士，兰贝尔阁下……」
。
克罗伊茨开口的瞬间，他便感觉到兰贝尔那强烈的杀气。
克罗伊茨把話咽了下去，后退了几步。
。
「啊，抱歉。有什么地方冒犯到您的話……」
。
兰贝尔径直地往克罗伊茨的方向突进。
克罗伊茨没能反应过来，被兰贝尔伸出的手臂打到侧腹，在地面上滚了约两米远的距离。
。
「咳哈！你、你干什……」
。
克罗伊茨忍着侧腹的疼痛抬起头。
这时，一束大到能包住一个人的巨大火柱从天而降，直击兰贝尔。
兰贝尔被火柱直击，烈焰与暴风扩散，将地面掀起。
克罗伊茨屏住了呼吸。要不是被兰贝尔保护，死的那个一定是自己。
。
「铠、铠之剑士！」
。
「首先解决掉一人。呀咧呀咧，真是麻烦，不就是王都兵团的一个部队嘛，有必要让我出手吗。只不过是些杂鱼被干掉了而已嘛」
。
克罗伊茨面朝向射出火柱的方向并拔出细剑。
半空中，敞开着一个幽黑的空洞，就像是空间被其扭曲一样。
此时，一头犹如全身都被棱镜覆盖般地可怕的巨龙从黑洞中探头。
如流水般的涟漪在棱镜般的体表上蔓延。
。
一名慵懒的银发女人正站在在黑洞的正下方。
。
「铠之剑士……这、都怪我……！」
。
克罗伊茨看向被烟雾埋没的爆炸中心点，即是兰贝尔刚才站着的地方。
。
「不要紧。比起这个，你平时应该更加小心附近的情况」
。
兰贝尔寸步不离地耸立在推开克罗伊茨的地方，如仁王一般。
