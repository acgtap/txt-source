小型都市特托姆堡與拉戈爾子爵領的礦山相鄰。
這個默默無聞地坐落在山中的都市，只是將八百人左右的人口硬塞進狹小的建築物中，嚴格來講還算不上都市，只是一個多少有些規模的設備齊全的村莊。
。
這個地方，原本是拉戈爾子爵為了將貧困的領民關在裡面、作為礦工度過一生而建立的都市，以掩人耳目的地下設施為主體，是個在王國的版圖上並不存在的都市。
因此被『吹笛之惡魔』相中。
。
由於特托姆堡的主要設施在地底，地上只布置著些零散而簡朴的矮層建築物。
並沒有從遠處眺望就能發現的高層建築物。
雖然都市中聳立著一座以偵查為目的而建起的簡朴的眺望塔，但仍保留著它老化後的模樣，即使不知道特托姆堡的人從遠處觀望，也不會去特別留意這座塔。
。
一名紅髮的青年，在質樸的特托姆堡大街上散步。
他披著薄薄的一層斗篷，脖子上還掛著懷錶。
。
他用手推著輪椅，輪椅上坐著一名聰明伶俐的少女。
猶如掛念著旁邊那個露出陰暗表情的沉默寡言的青年一樣，少女對他莞爾一笑。
。

「大哥哥，這裡的風景雖然很無聊，但比地下要好得多了呢。那個地方，在各種意義上都讓人心情陰郁呢」

。

「說的、是呢。艾麗澤醬」

。
青年用疲倦的聲音嘟囔。
艾麗澤是少女的名字。
。
露露可終歸只是『吹笛之惡魔』與拉戈爾子爵的中介者，真正支配特托姆堡的人，是『吹笛之惡魔』的八賢者之一、正體不明之鍊金術師『真理的紡織者』。

以他為首的組織——鍊金術師團『死之天使』，正在使用從拉戈爾子爵領中收集到的孩子進行魔術研究。
。
紅髮的青年名為亞爾安特，他雖然原本只是一名鍊金術師，但作為一名年輕的魔術師來說過於優秀，因此被『吹笛之惡魔』獵頭。
在親戚當中，心機叵測的叔父也對他說『你是被神選上了』並強行把他帶過去，當他察覺到目的地是當時眾人談虎色變的『吹笛之惡魔』相關的組織時，他便已經陷入了泥潭。
。
由於承受不住直接使用人體進行實驗的壓力，他頂多也只是幫忙構築魔法陣和設立假設，停留於只使用小型魔獸的生體實驗的階段。
連人體實驗場他都不敢去。
。
只是亞爾安特自己也被定期性地投下藥物，並且被刻入了謎一樣的魔法陣，他理解到自己也只不過是『死之天使』的實驗對象的其中之一罷了。
這種對待，對『死之天使』的下級團員來說並不稀奇。
。
由於價值觀不相符，亞爾安特被『死之天使』的團員所厭惡，然而卻微妙地被對自己抱著某種期待的副團長特尼雷爾喜歡上了。
曾經對亞爾安特出手的團員，都早已被特尼雷爾處置。
然而無論是哪一邊，都是亞爾安特所厭惡的對象。
。
從遠處觀察，『真理的紡織者』只是個被斗篷蓋著的凹凸不平的怪物，甚至連正常的對話也無法成立，實質上特尼雷爾才是這個組織的最高負責人。
。

『真理的紡織者』在腦海中記錄著各式各樣的魔法陣，擁有過目不忘的能力，因此沒有直接動手的必要。
而且，他還相當的神經質，絕不會在他人面前進行人體實驗，特尼雷爾曰。
總而言之，『真理的紡織者』到底是何物，至今無人知曉。
他也從未將自己的成果與他人共享過。
實驗前後的準備和處理似乎都是交給特尼雷爾來辦，總之，『真理的紡織者』是否擁有思考能力、是否採取有意義的行為，也許只有特尼雷爾和他本人才知道。
。
因此亞爾安特猜測那個凹凸不平的東西只是個魔導裝置，特尼雷爾才是真正八賢者『真理的紡織者』，只是不說出口而已。
。
對亞爾安特而言，『死之天使』的鍊金術師毫無例外、包括對其百依百順並逍遙自在地活著的自己都在厭惡的對象之內，但受到特尼雷爾寵愛也不失為一件壞事。
雖然在走出特托姆堡的瞬間就會被追殺，但像這樣將被當做實驗材料的孩子納入自己的管理之下的話，孩子們就可以得到在地上散步的權利。
雖然這只是毫無意義的自我滿足，但喜歡上了這段唯一可以走出外面的時間的孩子們也開始逐漸傾慕於他。
亞爾安特認為，散步是在這令人不適的人體實驗都市中留給他的唯一的慰藉，如果失去了這個支柱他早已發瘋了吧。
。

「大哥哥，我什麼時候會死呢？你不是知道的嗎？」

。
艾麗澤向亞爾安特開口。
他如鯁在喉。
特托姆堡中的生體實驗呈兩極分化，只有實驗至死亡為止，和觀察實驗過程到最後活生生地肢解兩種。
艾麗澤顯然是後者，由於被投用奇怪的魔術和藥品，她的雙腳已經癱瘓。
。

「……讓你為難了嗎？」

。

「沒事的。我是……絕對不會讓你死的」

。
亞爾安特無力地低語。
那是個不可能達成的約定。
。
特尼雷爾絕不會對研究表示妥協。
雖然他現在對亞爾安特抱有好意，但要是覺得自己礙事的話，立即被特尼雷爾處分是明擺著的事情。
即使抱著視死如歸的覺悟帶上艾麗澤逃走，也肯定會在這個都市的附近被抓捕。
。
艾麗澤也明白自己的處境。
即使自己的雙腳可以正常行走，逃出生天的可能性也沒有萬分之一。
。

「反正都要被殺的話，我希望被大哥哥殺死」

。
艾麗澤說完，看了眼亞爾安特的表情後，擠出搪塞般的笑容。
。

「對不起，忘了剛才的話吧，大哥哥」

。

「…………」

。
持續了短暫而又沉重的沉默之後，艾麗澤看著亞爾安特的臉，瞳孔略微放大。
。

「大哥哥，你的臉蛋」

。
艾麗澤戳了戳自己的臉頰。
亞爾安特雖然摸了摸自己的臉頰，但並沒有摸到什麼。
艾麗澤搖了搖頭後，露出了最近難得一見的、如孩子般純粹的笑容。
。

「我來幫你拿掉吧」

。
艾麗澤保持著端坐的姿勢，盡可能地伸長手臂。
亞爾安特雖然抱有疑問但也把臉湊了上去，此時艾麗澤突然探頭，吻了吻他的臉頰。
。

「艾麗澤……？」

。

「……雖然或許有些任性，但請別忘記我的事情哦，大哥哥」

。
對於少女那不遺餘力的笑容，亞爾安特卻無法回以微笑。
