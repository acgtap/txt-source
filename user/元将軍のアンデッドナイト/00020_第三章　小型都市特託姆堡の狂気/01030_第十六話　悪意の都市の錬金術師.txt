小型都市特托姆堡与拉戈尔子爵领的矿山相邻。
这个默默无闻地坐落在山中的都市，只是将八百人左右的人口硬塞进狭小的建筑物中，严格来讲还算不上都市，只是一个多少有些规模的设备齐全的村庄。
。
这个地方，原本是拉戈尔子爵为了将贫困的领民关在里面、作为矿工度过一生而建立的都市，以掩人耳目的地下设施为主体，是个在王国的版图上并不存在的都市。
因此被『吹笛之恶魔』相中。
。
由于特托姆堡的主要设施在地底，地上只布置着些零散而简朴的矮层建筑物。
并没有从远处眺望就能发现的高层建筑物。
虽然都市中耸立着一座以侦查为目的而建起的简朴的眺望塔，但仍保留着它老化后的模样，即使不知道特托姆堡的人从远处观望，也不会去特别留意这座塔。
。
一名红发的青年，在质朴的特托姆堡大街上散步。
他披着薄薄的一层斗篷，脖子上还挂着怀表。
。
他用手推着轮椅，轮椅上坐着一名聪明伶俐的少女。
犹如挂念着旁边那个露出阴暗表情的沉默寡言的青年一样，少女对他莞尔一笑。
。

「大哥哥，这里的风景虽然很无聊，但比地下要好得多了呢。那个地方，在各种意义上都让人心情阴郁呢」

。

「说的、是呢。艾丽泽酱」

。
青年用疲倦的声音嘟囔。
艾丽泽是少女的名字。
。
露露可终归只是『吹笛之恶魔』与拉戈尔子爵的中介者，真正支配特托姆堡的人，是『吹笛之恶魔』的八贤者之一、正体不明之炼金术师『真理的纺织者』。

以他为首的组织——炼金术师团『死之天使』，正在使用从拉戈尔子爵领中收集到的孩子进行魔术研究。
。
红发的青年名为亚尔安特，他虽然原本只是一名炼金术师，但作为一名年轻的魔术师来说过于优秀，因此被『吹笛之恶魔』猎头。
在亲戚当中，心机叵测的叔父也对他说『你是被神选上了』并强行把他带过去，当他察觉到目的地是当时众人谈虎色变的『吹笛之恶魔』相关的组织时，他便已经陷入了泥潭。
。
由于承受不住直接使用人体进行实验的压力，他顶多也只是帮忙构筑魔法阵和设立假设，停留于只使用小型魔兽的生体实验的阶段。
连人体实验场他都不敢去。
。
只是亚尔安特自己也被定期性地投下药物，并且被刻入了谜一样的魔法阵，他理解到自己也只不过是『死之天使』的实验对象的其中之一罢了。
这种对待，对『死之天使』的下级团员来说并不稀奇。
。
由于价值观不相符，亚尔安特被『死之天使』的团员所厌恶，然而却微妙地被对自己抱着某种期待的副团长特尼雷尔喜欢上了。
曾经对亚尔安特出手的团员，都早已被特尼雷尔处置。
然而无论是哪一边，都是亚尔安特所厌恶的对象。
。
从远处观察，『真理的纺织者』只是个被斗篷盖着的凹凸不平的怪物，甚至连正常的对話也无法成立，实质上特尼雷尔才是这个组织的最高负责人。
。

『真理的纺织者』在脑海中记录着各式各样的魔法阵，拥有过目不忘的能力，因此没有直接动手的必要。
而且，他还相当的神经质，绝不会在他人面前进行人体实验，特尼雷尔曰。
总而言之，『真理的纺织者』到底是何物，至今无人知晓。
他也从未将自己的成果与他人共享过。
实验前后的准备和处理似乎都是交给特尼雷尔来办，总之，『真理的纺织者』是否拥有思考能力、是否采取有意义的行为，也许只有特尼雷尔和他本人才知道。
。
因此亚尔安特猜测那个凹凸不平的东西只是个魔导装置，特尼雷尔才是真正八贤者『真理的纺织者』，只是不说出口而已。
。
对亚尔安特而言，『死之天使』的炼金术师毫无例外、包括对其百依百顺并逍遥自在地活着的自己都在厌恶的对象之内，但受到特尼雷尔宠爱也不失为一件坏事。
虽然在走出特托姆堡的瞬间就会被追杀，但像这样将被当做实验材料的孩子纳入自己的管理之下的話，孩子们就可以得到在地上散步的权利。
虽然这只是毫无意义的自我满足，但喜欢上了这段唯一可以走出外面的时间的孩子们也开始逐渐倾慕于他。
亚尔安特认为，散步是在这令人不适的人体实验都市中留给他的唯一的慰藉，如果失去了这个支柱他早已发疯了吧。
。

「大哥哥，我什么时候会死呢？你不是知道的吗？」

。
艾丽泽向亚尔安特开口。
他如鲠在喉。
特托姆堡中的生体实验呈两极分化，只有实验至死亡为止，和观察实验过程到最后活生生地肢解两种。
艾丽泽显然是后者，由于被投用奇怪的魔术和药品，她的双脚已经瘫痪。
。

「……让你为难了吗？」

。

「没事的。我是……绝对不会让你死的」

。
亚尔安特无力地低语。
那是个不可能达成的约定。
。
特尼雷尔绝不会对研究表示妥协。
虽然他现在对亚尔安特抱有好意，但要是觉得自己碍事的話，立即被特尼雷尔处分是明摆着的事情。
即使抱着视死如归的觉悟带上艾丽泽逃走，也肯定会在这个都市的附近被抓捕。
。
艾丽泽也明白自己的处境。
即使自己的双脚可以正常行走，逃出生天的可能性也没有万分之一。
。

「反正都要被杀的話，我希望被大哥哥杀死」

。
艾丽泽说完，看了眼亚尔安特的表情后，挤出搪塞般的笑容。
。

「对不起，忘了刚才的話吧，大哥哥」

。

「…………」

。
持续了短暂而又沉重的沉默之后，艾丽泽看着亚尔安特的脸，瞳孔略微放大。
。

「大哥哥，你的脸蛋」

。
艾丽泽戳了戳自己的脸颊。
亚尔安特虽然摸了摸自己的脸颊，但并没有摸到什么。
艾丽泽摇了摇头后，露出了最近难得一见的、如孩子般纯粹的笑容。
。

「我来帮你拿掉吧」

。
艾丽泽保持着端坐的姿势，尽可能地伸长手臂。
亚尔安特虽然抱有疑问但也把脸凑了上去，此时艾丽泽突然探头，吻了吻他的脸颊。
。

「艾丽泽……？」

。

「……虽然或许有些任性，但请别忘记我的事情哦，大哥哥」

。
对于少女那不遗余力的笑容，亚尔安特却无法回以微笑。
