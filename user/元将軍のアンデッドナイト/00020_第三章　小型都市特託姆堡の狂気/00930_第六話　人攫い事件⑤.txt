「……实在是，太精彩了。居然能够将那麼巨大的剑，轻而易举地挥舞。看来在铠甲裡面的，是显赫有名的剑士吧。我很清楚。您不只是在剑术方面，在人格方面肯定也十分卓越吧」

。
被白面具掩盖着脸的弗贝因，用徒有虚名的言语称讚了兰贝尔。
。

「所以，这麼做的話你还能怎麼办呢！？」

。
弗贝因将手上的锡杖对准了搂着父亲瑟瑟发抖的亚尔蒂。
。

「哈哈哈哈！太天真了！不许动哦。把那把剑放下，现在马上！」

。
弗贝因向兰贝尔命令。
兰贝尔，把剑丢到地上。
在面具的深处，弗贝因笑得更大声了。
。

「哈哈哈哈哈！这样形势就逆转了哇！」

。
松开手上的大剑、在弗贝因放鬆的一瞬间缩短了距离的兰贝尔，毫不留情地对弗贝因使出了一记回旋踢。
弗贝因的身体漂亮地折成两段飞了出去。
凝聚了兰贝尔力量的超重的铠甲，对弗贝因的腹部造成了极大的伤害。
。

「弗、弗贝因大人！」
「弗贝因大人被做掉了！那只荒唐的怪物到底是什麼！」

。
剩下两名私兵打算逃走。
然而，兰贝尔的速度是他们远远比不上的。
兰贝尔在眨眼间就追上了两人，用魔金手甲水平地对他们的脖子放出手刀。
两名私兵失去了意识，就这麼倒在地上。
通过压抑延髓、瞬发地阻断血液的流动，从而夺走对方的意识。
。

「嗯，还活着」

。
兰贝尔仿佛接受了这个结果般地独自一人颔首。
兰贝尔不擅长手下留情，过去有好幾次都是因为力度调整失败而将对方的颈骨直接折断。
。

「刚才真是危险。你们没事吧？」

。
兰贝尔向呆若木鸡的亚尔蒂和她的父亲询问。
男人呆滞地跪在地上，目瞪口呆，仍在思考着眼前的景象是梦境还是现实。
。
亚尔蒂也对眼前發生的凄惨的光景感到吃惊。
在幼小的少女面前，将人一分为二的场情景对她来说实在太过刺激。
。

「十、十分感激，骑士大人……」

。
纵使将眼前的惊悚的一幕保存在了记忆的一角，亚尔蒂也断断续续地向兰贝尔言谢。
亚尔蒂的父亲似乎也回过神来，表情恢復正常，向兰贝尔扣了好幾次响头。
。

「真的、真的是非常感谢！要是亚尔蒂被那些傢伙带走的話，都不知道会遭到什麼对待……！可是……这件事如果被子爵大人知道的話，骑士大人和附近的贫民们肯定会遭到復仇的吧……」

。
男人无力地沉吟。
接着他察觉到兰贝尔的视线，抬起头来并慌忙地左右摆手。
。

「不、不是！不是那样的！我并不没有为难骑士大人的意思！要是没有骑士大人的鼎力相助，我在刚才就已经失去我最重要的东西了！」

。
认为自己的发言也许会让人误解为有刁难含义的男人，拼命地在辩解。
兰贝尔无言地纹丝不动，毕竟完全看不出他的表情，无法从表情中得知他的感情，一接近便会感受到压迫感，只是站在旁边就已经自然而然地感到恐怖。
。

「没什麼，这句話虽然很容易让人误解，但我并没有在生气。而且……嗯？」

。
兰贝尔突然中断谈話，头盔一歪。
突如其来地感觉到了视线。
。
弗贝因抬起颤抖的手，用指尖将锡杖的前端指向了兰贝尔。
在他裂开的那一面的下方，已经被从他口中吐出的淤血玷污。
弗贝因在受到兰贝尔的猛踢後，还留了一口气。
。

「嗯，的确是有碾碎了内臟的感觉……居然还能动吗」

。

「哼、哼哼……侮辱我们、『吹笛之恶魔』的魔术师就是你的败因！土啊，粉碎敌人吧！」

。
他一边吐出着淤血一边行使魔术。
弗贝因近处的土块浮起形成子弹，往兰贝尔的上空飞去。
。
打偏了吗？并不是这样。
从附近一带中传来了嘎吱咯吱的可怕的响声。
下一瞬间，在背後的墙壁上出现了巨大的龟裂。
土之弹丸打穿的是古旧的房屋。
。

「唔姆……」

。
失去了支撑的房屋土崩瓦解，房屋的瓦砾朝着兰贝尔和亚尔蒂父女倾注。
古旧而不知何时会坍塌的危险建筑物，是处在这个贫民窟才能使用的攻击。
。
亚尔蒂的父亲抱紧她，宛如护盾般地蜷缩成一团。
。

「爸、爸爸……」

。
瓦砾无情地吞没了兰贝尔和父女。
最终轰鸣停息，濒死的弗贝因拖着痉挛的身体站了起身。
。

「哈、哈哈……活该。虽然断了气的子爵的私兵也被压扁了，嘛，就算失去了那种没用的废*物也没用太大问题吧。不过，内臟的受损太严重了。只能拜托露露可大人，去特托姆堡更换新的人工臟器了吗……」

。
弗贝因本来就因为疾病而曾经在死亡的边缘仿徨，也就是在那时将身体的一部分内臟和作为八贤者的实验品的结实的人工臟器替换了。
因此，即使受到了兰贝尔的一踢也只是濒临死亡而已。
。

「可是，这个男人到底是什麼人呢……即使是王家派来的调查兵，也太过引人注目了……。要是用这边的破屋就能杀掉他就好了呢……」

。
弗贝因往前面观察。
在瓦砾之山中，有一根布满尘埃的支柱。推开了残骸，大铠之男、兰贝尔正屹立着。
。
本应被瓦砾压扁的父女也无伤地站在兰贝尔前面。
。

「哈、哈！？」

。
兰贝尔纵然承受了落下的瓦砾，也若无其事地站立着。
以自己的巨躯守护了那对父女的事情也是一目了然。
落下的瓦砾对他来说不值一提。
他的铠甲也拥有着将其化为可能的硬度。
。
没有立即现身的原因，是为了尽可能地不让两父女受到伤害。
。

「不、不管怎麼说，这也太不可能了啊！」

。
弗贝因一边哭诉，一边背对着兰贝尔打算逃跑。
兰贝尔跳到他身後，在空中瞄准他的後颈放出一记手刀。
过度的杀伤力，让弗贝因的身体纵向地旋转，最後以额头撞地收场。
。

「你说了让我在意的話呢，『吹笛之恶魔』。因为你的缘故，好不容易抓住的俘虏也被瓦砾压扁了。把你知道的都给我一五一十地说出来」

。

「骑、骑士大人，那、那个人，已经……」

。

「唔姆……」

。
被亚尔蒂的父亲提醒，兰贝尔握着弗贝因的肩膀将他提起。
脑袋上下摇晃。颈骨已经折断了。
。
兰贝尔不太擅长手下留情。
何况是在空中放出的攻击，无疑是附带了铠甲的重量。
虽然考虑到这一点也已经留有餘力，然而并没有什麼意义。
。

「…………」

。
兰贝尔无言以对，慢慢地放下弗贝因的尸体，郑重地放在地上。
。

「骑、骑士大人，我……」

。
亚尔蒂的父亲开口。
看见兰贝尔消沉的样子，虽然心想不得不向他说些什麼，但又找不到合适的措辞。
原本，兰贝尔的存在就已经處於他们理解的範畴之外。
安慰的話语也不可能那麼容易地组织出来。
。

「……没事。下次的私兵肯定会生擒的」

。
兰贝尔一边说着，一边深深地颔首。
