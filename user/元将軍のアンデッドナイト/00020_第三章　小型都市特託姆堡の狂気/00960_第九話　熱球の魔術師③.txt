「雖然研究還未完成……但情況緊急，實在是迫不得已。熱球啊，寄宿於吾之手中吧！」

。
基爾多姆在手中生成了閃耀著白光的熱球。
基爾多姆後退，讓黃金之杖貼近熱球。
緊接著，紫電逐漸擴散並覆蓋了熱球。
。
電光相纏，具象成野獸的姿態。
白光的熱球仿佛心臟一般，寄宿於胸部的深處。
紫電之獸宛如發出咆哮一樣，仰天長嘯。
。

「那只雷獸……難道說，是活著的嗎？臨時生成的電氣之塊，居然是擬似生命體嗎？這種荒唐的魔術，怎麼可能存在……」

。
身處于蘭貝爾背後的克羅伊茨瞠目結舌。
。
(雖然在八國統一戰爭中已經有人用過了呢……按照克羅伊茨的說法，看來特斯拉格茲的功績已經被世人所忘記了呢）

。
于八國統一戰爭時期，蘭貝爾在與瑪邱拉斯王國的戰鬥中，有好幾次和以熱電離魔術生成的等離子獸戰鬥過的經驗。
。

「去吧，吾之獵犬廷達羅斯！追至地獄的深處，讓敵人的靈魂永遠不得安寧吧！」

。
基爾多姆揮舞金杖。
紫電之獵犬廷達羅斯在地面上俯沖，直奔蘭貝爾的所在之處。
廷達羅斯所經之處寸草不生，冒出濃濃黑煙。
。
蘭貝爾規避紫電之獵犬，往基爾多姆接近。
然而廷達羅斯迅速地轉身追趕蘭貝爾。
。

「無謂的抵抗。獵犬會追到你死為止。從等離子體的追踪者身上逃離的手段是不存在的！而且……再加上一只！」

。
基爾多姆生成了第二只廷達羅斯。
第二只獵犬在環顧四周後，也沖著蘭貝爾咬去。
。

「這樣就將軍了！」

。
進一步生成了第三只獵犬。
第三只獵犬以最短的路線接近蘭貝爾，成功與其他兩只獵犬匯合。
三只廷達羅斯一邊包圍著蘭貝爾一邊並行。
。
並行著的其中一只，張開大口並撲向蘭貝爾。
蘭貝爾預備用大劍反擊。
將飛撲而來的廷達羅斯的突擊閃避，用大劍砍向毫無防備的廷達羅斯的腹部。
廷達羅斯的身影只晃動了一瞬間。然而，很快又變回了原樣。
。

「沒用的。斬擊對電離氣體的廷達羅斯是無效的。相性太差了呢。即使你是多麼優秀的劍士，在熱電離魔術面前也束手無策。若不是魔術師的話，是不可能抵擋這一招的。鎧甲男啊，你就化為灰燼吧……」

。

「哈啊啊啊啊啊啊！」

。
蘭貝爾將下斬的大劍洶湧地往上挑起。
地面被割裂，塵土飛揚。
經歷了斬擊于暴風過後，大劍的一閃將地面分割成了地震過後般的裂縫。
。
而在地裂前方的廷達羅斯，以一分為二的狀態直立不動。
胸部深處的熱球也被斬裂，無法繼續運行魔術，並且熱量以分散的形式消散。
同時紫電之獸也無法維持存在而灰飛煙滅。
。
表情依然嚴肅的基爾多姆睜大了雙眼。
似乎是不能理解剛才發生了什麼事。
。

「真是的，無論在那個時代，對付這玩意就是費功夫」

。
第二只廷達羅斯也襲向蘭貝爾。
蘭貝爾握緊大劍，壓低重心並往地面一蹬。
廷達羅斯與蘭貝爾正面交鋒，決一勝負。
。
廷達羅斯的身體裝上了蘭貝爾魔金之鎧，身上纏繞的紫電被鎧甲彈開並綻裂開來。
因沖擊而被暴風吞沒的地面，被剜去一個圓形的大坑。
然而，屹立在中央的蘭貝爾卻相安無事。
。

「唔姆，這個辦法比較輕鬆」

。

「哈……哈？哈？」

。
基爾多姆那老練的賢者般的面孔，因眼前意味不明的光景而難堪地扭曲了。
作為能源體而幾乎不受物理性沖擊的廷達羅斯，被單純的一擊吹散。
。
即使攻擊火焰也不起作用，但用拳打出的風壓的確會對火焰的強弱產生影響。
雖說如此，將以高位魔術創造出來的廷達羅斯完全抹消什麼的，已經超越了基爾多姆理解的範疇。
。

「歸根結底，為何受到如此強大的沖擊，鎧甲卻能毫发無損！？為何能若無其事地，站在暴風的中心！？」

。

「原因很簡單。因為這副鎧甲，比暴風還要重」

。
蘭貝爾說著，再次舉起大劍並沖向基爾多姆。
追在蘭貝爾之後的第三只廷達羅斯並沒有太大的意義。
蘭貝爾的速度遠在紫電之獸之上。
而且即使紫電之獸能追上來，也能再次用體撞使其灰飛煙滅。
。

「本人基爾多姆……不可能在這種地方！吾、吾還得在那位大人的身邊，解開世界的真理啊！」

。

「要是連你這種傢伙都能做到的話，過去的人物……賢者多米莉奧涅那一類人早就實現了」

。

「劍士之流，居然說得好像知道一樣……！」

。
基爾多姆揭起黃金之杖。
巨大的熱球在錫杖當前出現。
從熱球中產生的紫電猶如將基爾多姆覆蓋一般，逐漸以半球的形狀展開。
超高密度的電離氣體化為了纏繞著紫光的障壁。
。
將所觸之物燃燒殆盡，是極其恐怖的破壞之屏障。
然而對基爾多姆來說，這是個可恨的判斷失誤。
簡而言之，他沒有想過之後的行動。
。
基爾多姆將所有殘余的瑪娜傾注，孤擲一注地展開這個絕對防御。
。
然而躲得過初一，躲不過十五。
最終瑪娜會耗盡，障壁隨之瓦解。
雖是最強的防護罩，但也需要相應的瑪娜。
消耗實在是太大。
。
(快、快思考、快思考啊基爾多姆。這一難，要如何躲過？這麼下去的話，必然會被斬殺。即便是饒命也應當嘗試一次嗎？）

。
再這麼無所作為下去，也只是用瑪娜來延長死期而已。
總之現在只能想出下一策。
。

「這傢伙應該對吾們的情報垂涎三尺才對。雖然吾絲毫沒有泄露情報的打算，但或許能爭取時間，找到一線生機……。而且吾的熱電離魔術，還留著最後的王牌。雖然吾也要對自己的安全賭一把……但只能出這一手了」

。
基爾多姆整理思緒，然後抬頭一看。
在破壞之屏障、超高密度的電離氣體之壁的上方忽然出現裂痕。
。

「嗯？」

。

「哈啊啊啊啊啊！」

。
伴隨著一聲怒吼，半圓的障壁以裂痕為起點潰散，揮舞著大劍的蘭貝爾從中出現。
基爾多姆瞠目結舌，發出無聲的慘叫。
。
破壞了基爾多姆的絕對防御的大劍，就這麼將他的身體一刀兩斷。
基爾多姆的身體被左右彈飛，而黃金之杖從所有者的手中脫離，插在地面上。
。
術者死亡的瞬間，最後的廷達羅斯的心臟部位的熱球也消失，維持不了身體而逐步瓦解。
。
蘭貝爾拔起貫穿了地面的大劍，望著基爾多姆的屍體。
。

「……原本，是打算先把防護罩打穿再說的呢。看來今天的狀態是真的很差」

。
再次失去了貴重的情報來源的蘭貝爾自言自語道。
