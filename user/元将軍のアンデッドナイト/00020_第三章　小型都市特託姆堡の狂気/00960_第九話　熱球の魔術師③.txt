「虽然研究还未完成……但情况紧急，实在是迫不得已。热球啊，寄宿於吾之手中吧！」

。
基尔多姆在手中生成了闪耀着白光的热球。
基尔多姆後退，让黄金之杖贴近热球。
紧接着，紫电逐渐扩散并覆盖了热球。
。
电光相缠，具象成野兽的姿态。
白光的热球仿佛心臟一般，寄宿於胸部的深处。
紫电之兽宛如发出咆哮一样，仰天长啸。
。

「那只雷兽……难道说，是活着的吗？临时生成的电气之块，居然是拟似生命体吗？这种荒唐的魔术，怎麼可能存在……」

。
身处于兰贝尔背後的克罗伊茨瞠目结舌。
。
(虽然在八国统一战争中已经有人用过了呢……按照克罗伊茨的说法，看来特斯拉格兹的功绩已经被世人所忘记了呢）

。
于八国统一战争时期，兰贝尔在与玛邱拉斯王国的战鬥中，有好幾次和以热电离魔术生成的等离子兽战鬥过的经验。
。

「去吧，吾之猎犬廷达罗斯！追至地狱的深处，让敌人的灵魂永远不得安宁吧！」

。
基尔多姆挥舞金杖。
紫电之猎犬廷达罗斯在地面上俯冲，直奔兰贝尔的所在之处。
廷达罗斯所经之处寸草不生，冒出浓浓黑烟。
。
兰贝尔规避紫电之猎犬，往基尔多姆接近。
然而廷达罗斯迅速地转身追赶兰贝尔。
。

「无谓的抵抗。猎犬会追到你死为止。从等离子体的追踪者身上逃离的手段是不存在的！而且……再加上一只！」

。
基尔多姆生成了第二只廷达罗斯。
第二只猎犬在环顾四周後，也冲着兰贝尔咬去。
。

「这样就将军了！」

。
进一步生成了第三只猎犬。
第三只猎犬以最短的路线接近兰贝尔，成功与其他两只猎犬汇合。
三只廷达罗斯一边包围着兰贝尔一边并行。
。
并行着的其中一只，张开大口并扑向兰贝尔。
兰贝尔预备用大剑反击。
将飞扑而来的廷达罗斯的突击闪避，用大剑砍向毫无防备的廷达罗斯的腹部。
廷达罗斯的身影只晃动了一瞬间。然而，很快又变回了原样。
。

「没用的。斩击对电离气体的廷达罗斯是无效的。相性太差了呢。即使你是多麼优秀的剑士，在热电离魔术面前也束手无策。若不是魔术师的話，是不可能抵挡这一招的。铠甲男啊，你就化为灰烬吧……」

。

「哈啊啊啊啊啊啊！」

。
兰贝尔将下斩的大剑汹涌地往上挑起。
地面被割裂，尘土飞扬。
经歷了斩击于暴风过後，大剑的一闪将地面分割成了地震过後般的裂缝。
。
而在地裂前方的廷达罗斯，以一分为二的状态直立不动。
胸部深处的热球也被斩裂，无法继续运行魔术，并且热量以分散的形式消散。
同时紫电之兽也无法维持存在而灰飞烟灭。
。
表情依然严肃的基尔多姆睁大了双眼。
似乎是不能理解刚才发生了什麼事。
。

「真是的，无论在那个时代，对付这玩意就是费功夫」

。
第二只廷达罗斯也袭向兰贝尔。
兰贝尔握紧大剑，压低重心并往地面一蹬。
廷达罗斯与兰贝尔正面交锋，决一胜负。
。
廷达罗斯的身体装上了兰贝尔魔金之铠，身上缠绕的紫电被铠甲弹开并绽裂开来。
因冲击而被暴风吞没的地面，被剜去一个圆形的大坑。
然而，屹立在中央的兰贝尔却相安无事。
。

「唔姆，这个办法比较轻鬆」

。

「哈……哈？哈？」

。
基尔多姆那老练的贤者般的面孔，因眼前意味不明的光景而难堪地扭曲了。
作为能源体而幾乎不受物理性冲击的廷达罗斯，被单纯的一击吹散。
。
即使攻击火焰也不起作用，但用拳打出的风压的确会对火焰的强弱产生影响。
虽说如此，将以高位魔术创造出来的廷达罗斯完全抹消什麼的，已经超越了基尔多姆理解的範畴。
。

「归根结底，为何受到如此强大的冲击，铠甲却能毫发无损！？为何能若无其事地，站在暴风的中心！？」

。

「原因很简单。因为这副铠甲，比暴风还要重」

。
兰贝尔说着，再次举起大剑并冲向基尔多姆。
追在兰贝尔之後的第三只廷达罗斯并没有太大的意义。
兰贝尔的速度远在紫电之兽之上。
而且即使紫电之兽能追上来，也能再次用体撞使其灰飞烟灭。
。

「本人基尔多姆……不可能在这种地方！吾、吾还得在那位大人的身边，解开世界的真理啊！」

。

「要是连你这种傢伙都能做到的話，过去的人物……贤者多米莉奥涅那一类人早就实现了」

。

「剑士之流，居然说得好像知道一样……！」

。
基尔多姆揭起黄金之杖。
巨大的热球在锡杖当前出现。
从热球中产生的紫电犹如将基尔多姆覆盖一般，逐渐以半球的形状展开。
超高密度的电离气体化为了缠绕着紫光的障壁。
。
将所触之物燃烧殆尽，是极其恐怖的破坏之屏障。
然而对基尔多姆来说，这是个可恨的判断失误。
简而言之，他没有想过之後的行动。
。
基尔多姆将所有残余的玛娜倾注，孤掷一注地展开这个绝对防御。
。
然而躲得过初一，躲不过十五。
最终玛娜会耗尽，障壁随之瓦解。
虽是最强的防护罩，但也需要相应的玛娜。
消耗实在是太大。
。
(快、快思考、快思考啊基尔多姆。这一难，要如何躲过？这麼下去的話，必然会被斩杀。即便是饶命也应当尝试一次吗？）

。
再这麼无所作为下去，也只是用玛娜来延长死期而已。
总之现在只能想出下一策。
。

「这傢伙应该对吾们的情报垂涎三尺才对。虽然吾丝毫没有泄露情报的打算，但或许能争取时间，找到一线生机……。而且吾的热电离魔术，还留着最後的王牌。虽然吾也要对自己的安全赌一把……但只能出这一手了」

。
基尔多姆整理思绪，然後抬头一看。
在破坏之屏障、超高密度的电离气体之壁的上方忽然出现裂痕。
。

「嗯？」

。

「哈啊啊啊啊啊！」

。
伴随着一声怒吼，半圆的障壁以裂痕为起点溃散，挥舞着大剑的兰贝尔从中出现。
基尔多姆瞠目结舌，发出无声的惨叫。
。
破坏了基尔多姆的绝对防御的大剑，就这麼将他的身体一刀两断。
基尔多姆的身体被左右弹飞，而黄金之杖从所有者的手中脱离，插在地面上。
。
术者死亡的瞬间，最後的廷达罗斯的心臟部位的热球也消失，维持不了身体而逐步瓦解。
。
兰贝尔拔起贯穿了地面的大剑，望着基尔多姆的尸体。
。

「……原本，是打算先把防护罩打穿再说的呢。看来今天的状态是真的很差」

。
再次失去了贵重的情报来源的兰贝尔自言自语道。
