兰贝尔正在突破拉戈尔子爵公馆的通道。
越是深入，敌人的警备越是强大，因此兰贝尔认为拉戈尔子爵肯定就在前面不远的地方。
。
即便是準备了隐藏的通道，兰贝尔也能够轻鬆地追上。
兰贝尔已经在贫民窟中得知，拉戈尔子爵是个大腹便便、连走两步都气喘吁吁的胖子。
况且他原本就因为嫌麻烦而基本上是足不出户的状态，也许是肥胖恶化，最近完全没有踏出过公馆半步。
。
(从公馆的外观上看……差不多到头了吧）

。
兰贝尔的回旋踢将大门轻易地粉碎。
在散落一地的木片那边，戴着假面的少女与身穿贵族服的面黄肌瘦的男人面面相觑。
男人的穿着虽然豪华，但脸上积聚着疲劳和焦躁，脸颊大大地凹陷进去，眼皮下的黑眼圈更是严重，在兰贝尔眼中这只不过是个寒酸的男人。
。
零零散散地，兰贝尔在途中见过的拉戈尔子爵的部下们在男人的周围站成一排。
。
假面的少女坐在椅子的靠背上，将双脚垂放在坐垫上，以倨傲无礼的坐姿俯视着瘦弱的男人。
私兵们发现兰贝尔后，便發出惊愕的声音。
。

「入、入侵者……！？」
「不可能，还以为下面發生了什麼事，没想到已经来到这裡了！情况竟然紧急到这个地步……」

。
假面的少女看了眼兰贝尔后耸了耸肩，啪塔啪塔地晃着腿。
。

「拉戈尔酱的私兵，根本派不上用场嘛！要不赶紧把这些没用的人一起送去特托姆堡吧？」

。

「饶、饶了我吧，露露可大人……！老、老朽，已经不能再为你们做事了！老、老朽，老朽的领地裡，居然發生这麼可怕的事……！老、老朽都不知情！毕竟，老朽……！老老、老朽绝对不会，泄露情报的……！所以、所以，请您离开这裡！求您了！」

。
瘦弱的男人敬畏地把手搭在头上，瑟瑟发抖。
。

「哈？侦查兵一来就想收手，就当什麼事都没發生过一样？我说啊，别把事情想得太美了哦？你是笨 蛋吧？在小看我们对吧？拉戈尔酱已经回不了头了哦，为什麼到现在还是不明白呢？」

。

「求、求……求您了！请离开这裡！请离开这裡！」

。
兰贝尔目睹了这一连串的对話，似乎掌握了事情的原委。
这弱不禁风的男人似乎是拉戈尔子爵。
虽然传闻中是个胖子，但据说他已经很久没有出过门了。
恐怕是在这期间，在和『吹笛之恶魔』的僵持中消磨了精神，结果就瘦成了这个样子。
。

「……嘛，总之，先解决掉入侵者再说吧。可以哦，这次就当做是特例，由我来处理入侵者吧。到时候你们一边看着一边思考吧。要是背叛了我们的話，到底会有什麼下场呢」

。
少女说完便高高跳起，落在兰贝尔的前方。
她以下蹲的姿势单手撑地，抬头仰视着兰贝尔。
。

「欢迎光临，入侵者先生，感谢您特意莅临这个寒酸又无聊的地方。話说回来，这裡又不是我家呢！啊哈！」

。

「你，是『吹笛之恶魔』的人吗……」

。

「啊哈哈哈哈，你只说中了一半哦？我呢，是『吹笛之恶魔』的八贤者之一，露露可·露露可。你可真是不走运啊，居然敢在我逗留的期间潜入这裡！」

。
假面的少女露露可，以单手支撑身体，往兰贝尔放出锐利的一踢。
兰贝尔则用大剑防御。
在一瞬内响起了三声锐利的金属音。
然而，兰贝尔却纹丝不动。
。

「啧！」

。
露露可用手持的真红金属杖叩向兰贝尔的大剑，同时往地面一踢，从兰贝尔的攻击範围中逃脱。
。

「以这麼娇小的身型，放出这样的踢击。这麼说，你改造过自己的身体对吧」

。

「…………」

。
露露可无言地盯着兰贝尔。
。
与其他『吹笛之恶魔』的人完全不一样，露露可其实是，王家鍊金术师团在过去暗中进行的生体实验的实验体。
因此她的身型虽然娇小，但身体能力却高得令人吃惊。
并且，作为身体被不断改造的结果，她的身体机能已经损坏，完全停止了成长。
明明已经不是少女的年龄，但却依然保持着幼小外貌的原因就如上述所说的那样。
。
而那鍊金术师团早已经由教会的异端审问会之手毁灭，他们的魔爪甚至触及到作为实验体、也可以说是被害者的露露可身上，但只有她一人想方设法地逃出生天，并且在『吹笛之恶魔』中不断积累功绩，最终成为了八贤者之一。
。

「啊哈，虽然比想象中的能幹得多，但靠这种程度就想赢过我们的話，那你就大错特错了」

。
说着，露露可举起真红之杖，在空中划圆。
。

「回应我的声音，从物界中现身吧，舞蹈断头台 骚灵剪刀」

。
魔法阵呈现。
两把刀刃上各自长出三只眼球，，巨大的剪刀浮在半空。
。

「召唤魔术，真是麻烦呢」

。

「啊哈，毕竟大哥哥又顽强又强大，所以我想稍微认真一点呢。虽说如此，要是比顽强的話我也不会输的哦！」

。

「和女孩子为敌没什麼心情呢……」

。
骚灵剪刀一边自转，一边以高速的动作在兰贝尔的周边移动。
自在地盘旋的骚灵剪刀，和露露可的超越人类领域的体术，受到这两者攻击的人还能相安无事的人并不多。
但露露可让骚灵剪刀慎重地在範围之外回旋。
在轻而易举地抵挡自己的三连踢的时点，露露可就已经对兰贝尔产生了不吉的预感。
。

「何时开始进攻好呢……一、二、三、四！」

。
露露可压低重心，在兰贝尔的周围左右移动，与骚灵剪刀同时进入了夹击兰贝尔的位置。
此刻，骚灵剪刀完全處於死角。
。
(得手了……！）

。
在她这麼以为的一刹那，骚灵剪刀就被大剑打落到地上并粉碎。
刀刃四分五裂，散落一地。骚灵剪刀变成光粒并逐渐消失。
。
(骗人的吧，明明是完全處於死角！？）

。
露露可心想情况不对，便往後一跳。
然而，迅速接近而来的兰贝尔对准露露可挥下大剑。
。
露露可接连往後退，贴着墙壁躲过兰贝尔的攻击。
。

「啊哈，哎呀，真是吓死我了呢。不过，我还是无伤……」

。
下一瞬间，在露露可的脸上出现纵向的伤痕。
露露可摸了摸脸上的伤口。
。
(要是再靠近一点，没有回避之术的話我就已经死了……在八贤者中特化了近战的我、露露可·露露可大人竟然……！）

。

「虽然不怎麼起劲，但如果你自称是八贤者的話，那就只能让你死在这裡了」
