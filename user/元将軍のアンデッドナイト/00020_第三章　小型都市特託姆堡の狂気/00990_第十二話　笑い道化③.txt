蘭貝爾正在突破拉戈爾子爵公館的通道。
越是深入，敵人的警備越是強大，因此蘭貝爾認為拉戈爾子爵肯定就在前面不遠的地方。
。
即便是準備了隱藏的通道，蘭貝爾也能夠輕鬆地追上。
蘭貝爾已經在貧民窟中得知，拉戈爾子爵是個大腹便便、連走兩步都氣喘吁吁的胖子。
況且他原本就因為嫌麻煩而基本上是足不出戶的狀態，也許是肥胖惡化，最近完全沒有踏出過公館半步。
。
(從公館的外觀上看……差不多到頭了吧）

。
蘭貝爾的回旋踢將大門輕易地粉碎。
在散落一地的木片那邊，戴著假面的少女與身穿貴族服的面黃肌瘦的男人面面相覷。
男人的穿著雖然豪華，但臉上積聚著疲勞和焦躁，臉頰大大地凹陷進去，眼皮下的黑眼圈更是嚴重，在蘭貝爾眼中這只不過是個寒酸的男人。
。
零零散散地，蘭貝爾在途中見過的拉戈爾子爵的部下們在男人的周圍站成一排。
。
假面的少女坐在椅子的靠背上，將雙腳垂放在坐墊上，以倨傲無禮的坐姿俯視著瘦弱的男人。
私兵們發現蘭貝爾後，便發出驚愕的聲音。
。

「入、入侵者……！？」
「不可能，還以為下面發生了什麼事，沒想到已經來到這裡了！情況竟然緊急到這個地步……」

。
假面的少女看了眼蘭貝爾後聳了聳肩，啪塔啪塔地晃著腿。
。

「拉戈爾醬的私兵，根本派不上用場嘛！要不趕緊把這些沒用的人一起送去特托姆堡吧？」

。

「饒、饒了我吧，露露可大人……！老、老朽，已經不能再為你們做事了！老、老朽，老朽的領地裡，居然發生這麼可怕的事……！老、老朽都不知情！畢竟，老朽……！老老、老朽絕對不會，泄露情報的……！所以、所以，請您離開這裡！求您了！」

。
瘦弱的男人敬畏地把手搭在頭上，瑟瑟發抖。
。

「哈？偵查兵一來就想收手，就當什麼事都沒發生過一樣？我說啊，別把事情想得太美了哦？你是笨 蛋吧？在小看我們對吧？拉戈爾醬已經回不了頭了哦，為什麼到現在還是不明白呢？」

。

「求、求……求您了！請離開這裡！請離開這裡！」

。
蘭貝爾目睹了這一連串的對話，似乎掌握了事情的原委。
這弱不禁風的男人似乎是拉戈爾子爵。
雖然傳聞中是個胖子，但據說他已經很久沒有出過門了。
恐怕是在這期間，在和『吹笛之惡魔』的僵持中消磨了精神，結果就瘦成了這個樣子。
。

「……嘛，總之，先解決掉入侵者再說吧。可以哦，這次就當做是特例，由我來處理入侵者吧。到時候你們一邊看著一邊思考吧。要是背叛了我們的話，到底會有什麼下場呢」

。
少女說完便高高跳起，落在蘭貝爾的前方。
她以下蹲的姿勢單手撐地，抬頭仰視著蘭貝爾。
。

「歡迎光臨，入侵者先生，感謝您特意蒞臨這個寒酸又無聊的地方。話說回來，這裡又不是我家呢！啊哈！」

。

「你，是『吹笛之惡魔』的人嗎……」

。

「啊哈哈哈哈，你只說中了一半哦？我呢，是『吹笛之惡魔』的八賢者之一，露露可·露露可。你可真是不走運啊，居然敢在我逗留的期間潛入這裡！」

。
假面的少女露露可，以單手支撐身體，往蘭貝爾放出銳利的一踢。
蘭貝爾則用大劍防御。
在一瞬內響起了三聲銳利的金屬音。
然而，蘭貝爾卻紋絲不動。
。

「嘖！」

。
露露可用手持的真紅金屬杖叩向蘭貝爾的大劍，同時往地面一踢，從蘭貝爾的攻擊範圍中逃脫。
。

「以這麼嬌小的身型，放出這樣的踢擊。這麼說，你改造過自己的身體對吧」

。

「…………」

。
露露可無言地盯著蘭貝爾。
。
與其他『吹笛之惡魔』的人完全不一樣，露露可其實是，王家鍊金術師團在過去暗中進行的生體實驗的實驗體。
因此她的身型雖然嬌小，但身體能力卻高得令人吃驚。
並且，作為身體被不斷改造的結果，她的身體機能已經損壞，完全停止了成長。
明明已經不是少女的年齡，但卻依然保持著幼小外貌的原因就如上述所說的那樣。
。
而那鍊金術師團早已經由教會的異端審問會之手毀滅，他們的魔爪甚至觸及到作為實驗體、也可以說是被害者的露露可身上，但只有她一人想方設法地逃出生天，並且在『吹笛之惡魔』中不斷積累功績，最終成為了八賢者之一。
。

「啊哈，雖然比想像中的能幹得多，但靠這種程度就想贏過我們的話，那你就大錯特錯了」

。
說著，露露可舉起真紅之杖，在空中划圓。
。

「回應我的聲音，從物界中現身吧，舞蹈斷頭台 騷靈剪刀」

。
魔法陣呈現。
兩把刀刃上各自長出三只眼球，，巨大的剪刀浮在半空。
。

「召喚魔術，真是麻煩呢」

。

「啊哈，畢竟大哥哥又頑強又強大，所以我想稍微認真一點呢。雖說如此，要是比頑強的話我也不會輸的哦！」

。

「和女孩子為敵沒什麼心情呢……」

。
騷靈剪刀一邊自轉，一邊以高速的動作在蘭貝爾的周邊移動。
自在地盤旋的騷靈剪刀，和露露可的超越人類領域的體術，受到這兩者攻擊的人還能相安無事的人並不多。
但露露可讓騷靈剪刀慎重地在範圍之外回旋。
在輕而易舉地抵擋自己的三連踢的時點，露露可就已經對蘭貝爾產生了不吉的預感。
。

「何時開始進攻好呢……一、二、三、四！」

。
露露可壓低重心，在蘭貝爾的周圍左右移動，與騷靈剪刀同時進入了夾擊蘭貝爾的位置。
此刻，騷靈剪刀完全處於死角。
。
(得手了……！）

。
在她這麼以為的一剎那，騷靈剪刀就被大劍打落到地上並粉碎。
刀刃四分五裂，散落一地。騷靈剪刀變成光粒並逐漸消失。
。
(騙人的吧，明明是完全處於死角！？）

。
露露可心想情況不對，便往後一跳。
然而，迅速接近而來的蘭貝爾對準露露可揮下大劍。
。
露露可接連往後退，貼著牆壁躲過蘭貝爾的攻擊。
。

「啊哈，哎呀，真是嚇死我了呢。不過，我還是無傷……」

。
下一瞬間，在露露可的臉上出現縱向的傷痕。
露露可摸了摸臉上的傷口。
。
(要是再靠近一點，沒有回避之術的話我就已經死了……在八賢者中特化了近戰的我、露露可·露露可大人竟然……！）

。

「雖然不怎麼起勁，但如果你自稱是八賢者的話，那就只能讓你死在這裡了」
