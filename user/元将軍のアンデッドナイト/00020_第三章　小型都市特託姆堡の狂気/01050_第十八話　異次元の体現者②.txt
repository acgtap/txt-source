由于特托姆堡是以在地下或洞窟建造的设施为主，地上的街道平时都处于空无一人的状态。
然而今天却有一个将近五十人的魔术师集团驻足在那荒废的街道上。
。
「看她那手忙脚乱的模样，真是的，露露可大人究竟在害怕些什么呢，嗯哼哼。她虽然让我们尽带上尽可能多的研究成果逃离这里，但我完全搞不懂她的意思。真是难以理解」
。
站在集团中心的是一名身穿黑色西装的纤细男子。
他就是小型都市特托姆堡的炼金术师团『死之天使』的副团长——特尼雷尔。
。
由于身为团长的八贤者『真理的纺织者』不愿与特尼雷尔以外的人接触，实际上特尼雷尔才是特托姆堡的支配者。
。
「王国兵团的侦察兵什么的，有什么可怕的呢。你们也是这么想的吧？我可爱的娃娃们」
。
说到这里，特尼雷尔看向站在他左侧的美少年执事。
在特尼雷尔的右侧也站着一名身穿同样服装的美少年执事。
两人的眼神已死，脸上并没有掺杂多余的感情。
。
这些少年都是特尼雷尔因兴趣而收集的奴隶。
特尼雷尔在从拉戈尔子爵领中掳走的非检测体的孩子当中，挑选出自己喜欢的少年，将其作为助手而放置在自己身旁。
当然，那些抵抗的、不长记性的、老是犯错的人，都会立即被送去实验室当实验材料。
。
特尼雷尔露出性质恶劣的笑容，同时抚摸并舔舐了少年的脸颊。
。
此时，一名身披法衣的『吹笛之恶魔』的魔术师往特尼雷尔的方向靠近。
。
「特尼雷尔大人，我有要事相报。我们在特托姆堡附近发现了十余名剑士」
。
接受报告的特尼雷尔的嘴唇弯成了新月的形状。
。
「区区十人！而且，想必都只是些兵团中的凡人吧。哈哈哈哈，哈哈哈哈哈！这个笑話真是好笑啊！啊哈哈哈哈！他们到底想到这里做什么？想自杀吗？那可得赶快送他们一程呢！」
。
说着说着，他不禁拍起手来并大笑。

『吹笛之恶魔』的其他魔术师们也迎合特尼雷尔放声大笑。
。
「诸君，给我听好了。要是一直待在这个地方的話，总有一天，国家的上级士兵和教会的异端审问会的人都必将蜂拥而来。我等『死之天使』的团长『真理的纺织者』大人也认为，现在『仍』不是和他们交手的时候。我等不得不离开特托姆堡这个地方。但是，如果我等只是一味地逃跑的話，只会助长了王家嚣张的气焰而已」
。
特尼雷尔张开双臂并开始演讲。
。
「让前往这里的王国兵团的人们，以最为凄惨的方式死去吧！不，不仅仅是这样！这样还不足够！我等要在这片领地筑起让王国全土震惊的凄惨地狱！」
。
他折起身子，仿佛是想抓住某物般地将双臂伸向天空。
。
「我等崇高之『吹笛之恶魔』决不允许背叛！然而拉戈尔子爵经已被毒虫吞食！与我等结下同盟的他，却无耻地背叛了我等！那么我等即使在这片领地大开杀戒，也不属为毫无秩序的暴行，此乃因果报应！趁审问会还未行动，吾等要对拉戈尔子爵领作出制裁，将其末路置于王国众目之下！让王国知道我等的实力！我等的撤退，绝不给予对方任何心理上的余裕！」
。
特尼雷尔也知晓，在拉戈尔子爵身上施下的咒术是种会被复杂而不稳定的感情所左右、即使在某一刻失控也不出奇的术式。
表面上对拉戈尔子爵施下咒术是为了埋下在他背叛之时的保险，但更深一层的目的是通过实验，来确认这个咒术在订立契约的场合是否能发挥出充分的效果。
。
然而，特尼雷尔想要的，只是找一个能在这片领地大开杀戒的理由罢了。
将国内的贵族作为合作者而拉拢入伙的『吹笛之恶魔』，决不能留下抛弃拉戈尔子爵这个累赘并大闹一场的证据。
其他合作人对『吹笛之恶魔』产生怀疑后背叛，并再一次依附到王家一方的可能性很高。
。
然而，若是在将拉戈尔子爵作为背叛者处置的情况下对他的领地降下制裁的話，反而会震慑到其他合作者。
。
「必须让王国和这个国家的子民知晓。我等『吹笛之恶魔』是多么优秀的存在！已逝的八贤者『尸之丑老』曼奇大人，只靠数人就对冒险者之都巴莱拉造成了毁灭性的灾害！然而，只是这样还不足够！原本的話，应当将巴莱拉的子民们一人不剩地变为不死者，曼奇大人也本应复归！那位大人失手了！我等将放弃这片领地并掀起第二波进攻，正式向王家宣战！」
。
魔术师们狂欢。
在场的人当中，一声不吭地站着的只有成为特尼雷尔的奴隶的美少年们。
从别处奔跑而来的第三名美少年执事，在特尼雷尔面前止步，目无表情地低下头。
。
「主人，亚尔安特大人说不定会趁这个机会逃脱。他与被检测体已经不见踪影」
。
「我深爱的洛比克哟。不必担心。我不是说过，你们不要干预亚尔安特君的事吗？」
。
「……可是，现在是紧急情况」
。
「唔姆，你说的也有道理。他是和小白鼠一起消失的吧。他自己大概也有逃走的打算。不过，我绝对不会让亚尔安特君逃掉的哦。况且如果被卷入交战的話，那也只会是死路一条。虽然他看上去像是很老实的样子，但在魔术方面他毫无疑问是个天才。先不说现在，将来的話，他的才能甚至能凌驾在我之上。正因为如此，我才把他带到这里来」
。
特尼雷尔嘴角上翘，露出令人反感的微笑。
。
「……亚尔安特大人逃不出去。虽然我不清楚这份自信是从哪里来，但主人这么主张的話，我也不多说什么」
。
「不过，对你们而言，这可真是稀奇呢。竟然对亚尔安特君的行踪关心到向我诘问的程度」
。
「我认为，不能让阻碍主人计划的人苟活下去。虽然我不太清楚详细的状况，但亚尔安特大人是您看中的人吧？」
。
特尼雷尔顶着那副恶心的笑容，弯下身子并窥视少年执事的瞳孔深处。
。
「不对吧，洛比克。虽说自己也是迫不得已而只能侍奉于我的囚犯之身，然而自以为与自己的处境相似的亚尔安特君却抛下了自己，在某个地方获得幸福什么的，你是在担心这件事吧？我说的没错吧？」
。
特尼雷尔窥视洛比克的神色，并用舌头从少年的眼角处舔到他下颚。
。
「我明白的哦，洛比克。你那丑陋卑鄙的，打从心底里希望他人遭受到不幸的嫉妒心。放心好了洛比克，无论是谁都逃不出我的手掌心」
。
即使在被特尼雷尔舔舐的时候，洛比克也一直面无表情地站着不动。
就连被说中内心痛处的惊讶也没有表现在脸上。又或许，这类感情早已被连根拔起般地剥夺了。
。
特尼雷尔直起身子并发出哄笑。
。
「那么，诸君！无论是谁也无法阻止我们的脚步！你们兵分多路，在这片土地上席起秩序的风暴！打起精神来，将前往这里的王国兵团的敌人，蹂躏到不成人样为止！」
。
「特尼雷尔大人！有、有急事相报！」
。
一名『吹笛之恶魔』的魔术师，惊慌失措地跑到特尼雷尔的面前。
。
「发生什么事？」
。
「哈巴内大人、哈巴内大人率领的狂战士部队……！」
。
「哈巴内阁下开始行动了吗……那样的話，王国兵团的人都已经变成尸体了呢。本想把他们生擒后再慢慢炮制，没想到还没开始就已经结束了呢。哈巴内阁下的行动太独断了」
。
特尼雷尔露出厌恶的表情。
。
哈巴内乃『吹笛之恶魔』的魔术师之一，在『死之天使』中，他进行着通过剥夺人类的意识来解放筋力的束缚，从而量产狂战士的研究。
。
他率领的狂战士部队，由拥有惊人力量和抗击打能力的剑士组成。
因为剑技拙劣以及智商上的欠缺而无法做到随机应变，但他们每一个都是单凭力气就能压制剑之达人的强者。
。
「那些狂战士，大概连人质是什么的智商都没有呢。顺带一提，哈巴内阁下有没有这个智商也是个问题。敌人一个都不能放生。真是的，计划都被搅乱了……」
。
特尼雷尔抱着脑袋，深深地叹了口气。
。
「不，是全灭了！哈巴内大人率领的狂战士部队，被王国兵团的一个部队全灭了！」
。
「你、你说什么！？这不可能！敌人不是只有十人左右吗！？」
。
「是、是的！但是，他们正以惊人之势攻了进来……狂战士部队在一瞬间就被……！他们已经闯入特托姆堡的内部了！」
