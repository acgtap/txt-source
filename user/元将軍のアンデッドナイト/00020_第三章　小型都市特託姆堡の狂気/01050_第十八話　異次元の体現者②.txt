由於特托姆堡是以在地下或洞窟建造的設施為主，地上的街道平時都處於空無一人的狀態。
然而今天卻有一個將近五十人的魔術師集團駐足在那荒廢的街道上。
。
「看她那手忙腳亂的模樣，真是的，露露可大人究竟在害怕些什麼呢，嗯哼哼。她雖然讓我們盡帶上盡可能多的研究成果逃離這裡，但我完全搞不懂她的意思。真是難以理解」
。
站在集團中心的是一名身穿黑色西裝的纖細男子。
他就是小型都市特托姆堡的鍊金術師團『死之天使』的副團長——特尼雷爾。
。
由於身為團長的八賢者『真理的紡織者』不願與特尼雷爾以外的人接觸，實際上特尼雷爾才是特托姆堡的支配者。
。
「王國兵團的偵察兵什麼的，有什麼可怕的呢。你們也是這麼想的吧？我可愛的娃娃們」
。
說到這裡，特尼雷爾看向站在他左側的美少年執事。
在特尼雷爾的右側也站著一名身穿同樣服裝的美少年執事。
兩人的眼神已死，臉上並沒有摻雜多余的感情。
。
這些少年都是特尼雷爾因興趣而收集的奴隷。
特尼雷爾在從拉戈爾子爵領中擄走的非檢測體的孩子當中，挑選出自己喜歡的少年，將其作為助手而放置在自己身旁。
當然，那些抵抗的、不長記性的、老是犯錯的人，都會立即被送去實驗室當實驗材料。
。
特尼雷爾露出性質惡劣的笑容，同時撫摸並舔舐了少年的臉頰。
。
此時，一名身披法衣的『吹笛之惡魔』的魔術師往特尼雷爾的方向靠近。
。
「特尼雷爾大人，我有要事相報。我們在特托姆堡附近发現了十餘名劍士」
。
接受報告的特尼雷爾的嘴唇彎成了新月的形狀。
。
「區區十人！而且，想必都只是些兵團中的凡人吧。哈哈哈哈，哈哈哈哈哈！這個笑話真是好笑啊！啊哈哈哈哈！他們到底想到這裡做什麼？想自殺嗎？那可得趕快送他們一程呢！」
。
說著說著，他不禁拍起手來並大笑。

『吹笛之惡魔』的其他魔術師們也迎合特尼雷爾放聲大笑。
。
「諸君，給我聽好了。要是一直待在這個地方的話，總有一天，國家的上級士兵和教會的異端審問會的人都必將蜂擁而來。我等『死之天使』的團長『真理的紡織者』大人也認為，現在『仍』不是和他們交手的時候。我等不得不離開特托姆堡這個地方。但是，如果我等只是一味地逃跑的話，只會助長了王家囂張的氣焰而已」
。
特尼雷爾張開雙臂並開始演講。
。
「讓前往這裡的王國兵團的人們，以最為淒慘的方式死去吧！不，不僅僅是這樣！這樣還不足夠！我等要在這片領地築起讓王國全土震驚的淒慘地獄！」
。
他折起身子，仿佛是想抓住某物般地將雙臂伸向天空。
。
「我等崇高之『吹笛之惡魔』決不允許背叛！然而拉戈爾子爵經已被毒虫吞食！與我等結下同盟的他，卻無恥地背叛了我等！那麼我等即使在這片領地大開殺戒，也不屬為毫無秩序的暴行，此乃因果報應！趁審問會還未行動，吾等要對拉戈爾子爵領作出制裁，將其末路置於王國眾目之下！讓王國知道我等的實力！我等的撤退，絕不給予對方任何心理上的餘裕！」
。
特尼雷爾也知曉，在拉戈爾子爵身上施下的咒術是種會被複雜而不穩定的感情所左右、即使在某一刻失控也不出奇的術式。
表面上對拉戈爾子爵施下咒術是為了埋下在他背叛之時的保險，但更深一層的目的是通過實驗，來確認這個咒術在訂立契約的場合是否能发揮出充分的效果。
。
然而，特尼雷爾想要的，只是找一個能在這片領地大開殺戒的理由罷了。
將國內的貴族作為合作者而拉攏入伙的『吹笛之惡魔』，決不能留下拋棄拉戈爾子爵這個累贅並大鬧一場的證據。
其他合作人對『吹笛之惡魔』產生懷疑後背叛，並再一次依附到王家一方的可能性很高。
。
然而，若是在將拉戈爾子爵作為背叛者處置的情況下對他的領地降下制裁的話，反而會震懾到其他合作者。
。
「必須讓王國和這個國家的子民知曉。我等『吹笛之惡魔』是多麼優秀的存在！已逝的八賢者『屍之丑老』曼奇大人，只靠數人就對冒險者之都巴萊拉造成了毀滅性的災害！然而，只是這樣還不足夠！原本的話，應當將巴萊拉的子民們一人不剩地變為不死者，曼奇大人也本應復歸！那位大人失手了！我等將放棄這片領地並掀起第二波進攻，正式向王家宣戰！」
。
魔術師們狂歡。
在場的人當中，一聲不吭地站著的只有成為特尼雷爾的奴隷的美少年們。
從別處奔跑而來的第三名美少年執事，在特尼雷爾面前止步，目無表情地低下頭。
。
「主人，亞爾安特大人說不定會趁這個機會逃脫。他與被檢測體已經不見踪影」
。
「我深愛的洛比克喲。不必擔心。我不是說過，你們不要干預亞爾安特君的事嗎？」
。
「……可是，現在是緊急情況」
。
「唔姆，你說的也有道理。他是和小白鼠一起消失的吧。他自己大概也有逃走的打算。不過，我絕對不會讓亞爾安特君逃掉的哦。況且如果被卷入交戰的話，那也只會是死路一條。雖然他看上去像是很老實的樣子，但在魔術方面他毫無疑問是個天才。先不說現在，將來的話，他的才能甚至能凌駕在我之上。正因為如此，我才把他帶到這裡來」
。
特尼雷爾嘴角上翹，露出令人反感的微笑。
。
「……亞爾安特大人逃不出去。雖然我不清楚這份自信是從哪裡來，但主人這麼主張的話，我也不多說什麼」
。
「不過，對你們而言，這可真是稀奇呢。竟然對亞爾安特君的行踪關心到向我詰問的程度」
。
「我認為，不能讓阻礙主人計划的人苟活下去。雖然我不太清楚詳細的狀況，但亞爾安特大人是您看中的人吧？」
。
特尼雷爾頂著那副惡心的笑容，彎下身子並窺視少年執事的瞳孔深處。
。
「不對吧，洛比克。雖說自己也是迫不得已而只能侍奉於我的囚犯之身，然而自以為與自己的處境相似的亞爾安特君卻拋下了自己，在某個地方獲得幸福什麼的，你是在擔心這件事吧？我說的沒錯吧？」
。
特尼雷爾窺視洛比克的神色，並用舌頭從少年的眼角處舔到他下顎。
。
「我明白的哦，洛比克。你那醜陋卑鄙的，打從心底裡希望他人遭受到不幸的嫉妒心。放心好了洛比克，無論是誰都逃不出我的手掌心」
。
即使在被特尼雷爾舔舐的時候，洛比克也一直面無表情地站著不動。
就連被說中內心痛處的驚訝也沒有表現在臉上。又或許，這類感情早已被連根拔起般地剝奪了。
。
特尼雷爾直起身子並发出哄笑。
。
「那麼，諸君！無論是誰也無法阻止我們的腳步！你們兵分多路，在這片土地上席起秩序的風暴！打起精神來，將前往這裡的王國兵團的敵人，蹂躪到不成人樣為止！」
。
「特尼雷爾大人！有、有急事相報！」
。
一名『吹笛之惡魔』的魔術師，驚慌失措地跑到特尼雷爾的面前。
。
「發生什麼事？」
。
「哈巴內大人、哈巴內大人率領的狂戰士部隊……！」
。
「哈巴內閣下開始行動了嗎……那樣的話，王國兵團的人都已經變成屍體了呢。本想把他們生擒後再慢慢炮制，沒想到還沒開始就已經結束了呢。哈巴內閣下的行動太獨斷了」
。
特尼雷爾露出厭惡的表情。
。
哈巴內乃『吹笛之惡魔』的魔術師之一，在『死之天使』中，他進行著通過剝奪人類的意識來解放筋力的束縛，從而量產狂戰士的研究。
。
他率領的狂戰士部隊，由擁有驚人力量和抗擊打能力的劍士組成。
因為劍技拙劣以及智商上的欠缺而無法做到隨機應變，但他們每一個都是單憑力氣就能壓制劍之達人的強者。
。
「那些狂戰士，大概連人質是什麼的智商都沒有呢。順帶一提，哈巴內閣下有沒有這個智商也是個問題。敵人一個都不能放生。真是的，計划都被攪亂了……」
。
特尼雷爾抱著腦袋，深深地嘆了口氣。
。
「不，是全滅了！哈巴內大人率領的狂戰士部隊，被王國兵團的一個部隊全滅了！」
。
「你、你說什麼！？這不可能！敵人不是只有十人左右嗎！？」
。
「是、是的！但是，他們正以驚人之勢攻了進來……狂戰士部隊在一瞬間就被……！他們已經闖入特托姆堡的內部了！」
