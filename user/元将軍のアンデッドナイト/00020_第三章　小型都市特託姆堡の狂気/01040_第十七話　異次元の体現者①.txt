王國調查兵團『不死鳥之瞳』的第二部隊長克羅伊茨騎著馬，率領著十名部下往位於拉戈爾子爵領內的礦山進發。
。
「克羅伊茨大人……您確定，真的要前去那座礦山嗎？」
。
克羅伊茨的部下、名為弗雷的黑短髮的女劍士驚訝地詢問。
。
「啊、啊啊，至少那個男人不是敵人。那個男人的確是保護了孩子。而且……他將自稱為『吹笛之惡魔』爪牙的魔術師一刀兩斷之事也是事實。情報的確屬實」
。
克羅伊茨含糊不清地回答。
從魔銀制頭盔中裸 露的嘴唇，被以能夠咬出牙印的力度緊咬著。
。
「那個、我想問的不是這個問題……克羅伊茨大人也是明白的吧？我已經做好了為國捐軀的打算。可是，這難道不是……傳說中單刀赴會的無謀之舉嗎……？」
。
「這並不是無謀之舉……哪怕是解決掉多一個敵人，就結果而言，也能使一百名平民獲救……沒錯，這絕不是白白送死。傳令的話，靠琉克一個人就足夠了」
。
克羅伊茨就像是在說服自己一樣地說著。
。
「克羅伊茨大人是認真的話，我也會陪伴您共赴黃泉。可是、可是……這真的不是讓我們去白白送死的吧？」
。
克羅伊茨並沒有向她作出回答。
。
「姆？還以為又遇到了哨兵，原來是你們嗎。能和你們匯合真是幸運呢」
。
出現在克羅伊茨他們面前的大鎧之男蘭貝爾的大劍上，又滴落著新鮮的血液。
。
。
。
——時間回溯至數刻前。
。
在和蘭貝爾的決鬥中讓手臂負傷的克羅伊茨，耿直地找出了被蘭貝爾(強行）託付的孩子的父母，然後拜託『不死鳥之瞳』的白魔術師即女劍士弗雷，讓她立刻治療手臂的傷勢。
只要仍逗留在這個地方，敵人會從何處來襲也是個未知數。
漫長而完整的治療什麼的，怎麼可能等得下去。
。
雖說魔術師和劍士兩者的強度無法相比較，但基爾多姆是一名顯赫有名的鍊金術師，他的等級明顯在克羅伊茨之上。
雖然克羅伊茨不認為這麼有名氣的人會陸續登場，但毋庸置疑的是，實力接近於他的佼佼者正潛伏在此地。
而且可以確定的是，潛伏在此地的八賢者恐怕是比基爾多姆還要強許多的存在。
。
「克羅伊茨大人……您手臂上的傷，究竟是怎樣造成的！？肌肉的斷裂實在是太過嚴重……難道是遭人毒打了嗎！」
。
弗雷一邊抬起克羅伊茨的手臂一邊大呼小叫。
。
「啊、啊啊，有點……亂來了呢。嘛，雖然最後贏的人是我，不對，應該是他……」
。
沒想到，不僅被毫無關係的劍士化解了所有的攻擊，而且還不知道收手，是自己對自己的手臂造成了傷害什麼的，克羅伊茨不敢對部下這麼說，只好含糊其辭。
。
「無論我們的判斷是否落空，這個地方都必然是『吹笛之惡魔』的根據地之一。但實話說，這次的對手單靠我們的部隊是絕對不可能應付得過來的。雖然本應讓『不死鳥之瞳』全軍出擊……但到了那時，敵人也已經完全撤退了吧」
。
「背叛者是拉戈爾子爵嗎……。他雖然是個臭名遠揚的男人，但我認為他並不是個能幹出這麼大逆不道的事情的料」
。
原本他們的任務就是調查拉戈爾子爵是否存在奇怪的舉動。
但令他們始料未及的是，現在居然變成了要與一大批『吹笛之惡魔』的魔術師戰鬥的狀況。
即使是給予了敵人撤離的時間，也必須得先折返一次才行。
。
「……只不過，在他們撤退之前，必然會先遇到一個男人」
。
在克羅伊茨的腦海裡，宣稱要獨自一人闖入拉戈爾子爵公館的、身披鎧甲的巨漢的身影一閃而過。
該不會是真的闖進去了吧，克羅伊茨不敢相信，但如果是那個男人的話，也許真的會這麼做吧。
不管怎麼說，只要他認真行動，事態都必然會朝著驚人的方向發展。
即使在掌握結果之後再行動也不遲，克羅伊茨這麼判斷。
。
「首先必須跟那個大鎧之男匯合呢。要手刃那些傢伙……也必須得借他之手。那個男人應該不會成為我們的累贅的」
。
為了在部下的面前保持威嚴，克羅伊茨並沒有把真相坦言。
作為守護王國的劍而日復一日地鍛鍊的他，居然輸給了一般人什麼的，怎麼可能說得出口呢。
這不僅羞恥，而且還關係到部下們的士氣。
克羅伊茨自身的自尊心也是個大問題。
。
「這世上居然還有能與克羅伊茨大人不相伯仲的劍士……不敢相信……！身為克羅伊茨大人的左右手，我也必須得不斷精進才行呢」
。
「嗯、啊啊，嘛，你努力吧」
。
相對於使命感旺盛的弗雷，克羅伊茨也只是語無倫次地應聲附和而已。
此時，一串笨重的腳步聲正逐漸往他們的方向靠近。
克羅伊茨和弗雷回頭，發現一名身披大鎧的男人正站在他們背後。
毫無疑問是幫助過克羅伊茨的男人。
。
「來、來者何人！」
。
察覺到蘭貝爾的非凡氣質的弗雷，把手放在劍鞘上並與蘭貝爾對峙，
。
「他、他就是我之前所說的合作者！弗雷，快、快把手從劍鞘上鬆開！」
。
要是得罪了他就完蛋了。
克羅伊茨趕緊命令弗雷，讓她訂正自己的行為。
。
「您就是那個人嗎。那可真是太失禮了……」
。
盡管對克羅伊茨的樣子抱有若干的疑問，弗雷也乖乖地退下了。
克羅伊茨輕輕低頭。
。
「剛才真是幫大忙了。請允許我再次道謝。不過，不管怎麼說，看來您是打消了直接闖入拉戈爾子爵的公館的念頭呢。我真誠地感到欣慰。您實在是過於無謀了。您心中的正義感，即使從我的立場來講也是值得稱讚的。然而，急著送死並不是什麼好事」
。
克羅伊茨與大鎧之男辭別之後，只不過經過了數刻的時間。
在這段時間之內是不可能向子爵公館發起襲擊的。正常來想的話。
。
相隔幾秒後，克羅伊茨再度開口。
。
「……鎧之劍士，您有無與我們共同作戰的打算？要是您加入的話，我們的第二部隊……」
。
「我失敗了。也可以說是敗北了。拉戈爾子爵因他們的咒術而死亡。八賢者也逃走了。這都是我的疏忽。即使只有八賢者也應該確實地殺死。想顧全兩者卻得不償失」
。
「……嗯？」
。
克羅伊茨的嘴角在無法理解的語言面前扭曲了。
。
「請。請等一下。照你這種說法，就像是襲擊已經結束、您凱旋而歸一樣，但這肯定是我們之間的話題不一樣對吧」
。
克羅伊茨向蘭貝爾伸手，打算打斷他的發言。
然而蘭貝爾並不在意克羅伊茨的話語，繼續開口。
。
「不過我得到了情報。那些傢伙的據點是位於礦山附近的特托姆堡。據說，有數十名左右的『吹笛之惡魔』的魔術師匿藏在那。現在馬上前去打擊的話，必定能大幅削減『吹笛之惡魔』的戰鬥力」
。
「居然有數十名！？」
。
「數量並不是問題。大部分也只有基爾多姆那種程度的實力而已吧」
。
「基爾多姆有數十人！？」
。
克羅伊茨開始想像，數十名表情猙獰的基爾多姆在他面前生成熱球，並操縱大量的等離子獸的情景。
冷靜一想的話，這確實是能輕易顛覆他曾經歷過的修羅場的地獄。
。
「雖然有些許麻煩，但也只能去做了。只靠我一人的話，也許會出現不少漏網之魚。能助我一臂之力嗎？」
。
已經是打算先折返一次的克羅伊茨驚愕地張大嘴巴，目瞪口呆地注視著蘭貝爾。
弗雷似乎也是跟不上理解的樣子，輪流地向克羅伊茨和大鎧之男送去尋求解釋般的視線。
