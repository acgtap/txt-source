王国调查兵团『不死鸟之瞳』的第二部队长克罗伊茨骑着马，率领着十名部下往位于拉戈尔子爵领内的矿山进发。
。
「克罗伊茨大人……您确定，真的要前去那座矿山吗？」
。
克罗伊茨的部下、名为弗雷的黑短发的女剑士惊讶地询问。
。
「啊、啊啊，至少那个男人不是敌人。那个男人的确是保护了孩子。而且……他将自称为『吹笛之恶魔』爪牙的魔术师一刀两断之事也是事实。情报的确属实」
。
克罗伊茨含糊不清地回答。
从魔银制头盔中裸 露的嘴唇，被以能够咬出牙印的力度紧咬着。
。
「那个、我想问的不是这个问题……克罗伊茨大人也是明白的吧？我已经做好了为国捐躯的打算。可是，这难道不是……传说中单刀赴会的无谋之举吗……？」
。
「这并不是无谋之举……哪怕是解决掉多一个敌人，就结果而言，也能使一百名平民获救……没错，这绝不是白白送死。传令的話，靠琉克一个人就足够了」
。
克罗伊茨就像是在说服自己一样地说着。
。
「克罗伊茨大人是认真的話，我也会陪伴您共赴黄泉。可是、可是……这真的不是让我们去白白送死的吧？」
。
克罗伊茨并没有向她作出回答。
。
「姆？还以为又遇到了哨兵，原来是你们吗。能和你们汇合真是幸运呢」
。
出现在克罗伊茨他们面前的大铠之男兰贝尔的大剑上，又滴落着新鲜的血液。
。
。
。
——时间回溯至数刻前。
。
在和兰贝尔的决斗中让手臂负伤的克罗伊茨，耿直地找出了被兰贝尔(强行）托付的孩子的父母，然后拜托『不死鸟之瞳』的白魔术师即女剑士弗雷，让她立刻治疗手臂的伤势。
只要仍逗留在这个地方，敌人会从何处来袭也是个未知数。
漫长而完整的治疗什么的，怎么可能等得下去。
。
虽说魔术师和剑士两者的强度无法相比较，但基尔多姆是一名显赫有名的炼金术师，他的等级明显在克罗伊茨之上。
虽然克罗伊茨不认为这么有名气的人会陆续登场，但毋庸置疑的是，实力接近于他的佼佼者正潜伏在此地。
而且可以确定的是，潜伏在此地的八贤者恐怕是比基尔多姆还要强许多的存在。
。
「克罗伊茨大人……您手臂上的伤，究竟是怎样造成的！？肌肉的断裂实在是太过严重……难道是遭人毒打了吗！」
。
弗雷一边抬起克罗伊茨的手臂一边大呼小叫。
。
「啊、啊啊，有点……乱来了呢。嘛，虽然最后赢的人是我，不对，应该是他……」
。
没想到，不仅被毫无关系的剑士化解了所有的攻击，而且还不知道收手，是自己对自己的手臂造成了伤害什么的，克罗伊茨不敢对部下这么说，只好含糊其辞。
。
「无论我们的判断是否落空，这个地方都必然是『吹笛之恶魔』的根据地之一。但实話说，这次的对手单靠我们的部队是绝对不可能应付得过来的。虽然本应让『不死鸟之瞳』全军出击……但到了那时，敌人也已经完全撤退了吧」
。
「背叛者是拉戈尔子爵吗……。他虽然是个臭名远扬的男人，但我认为他并不是个能干出这么大逆不道的事情的料」
。
原本他们的任务就是调查拉戈尔子爵是否存在奇怪的举动。
但令他们始料未及的是，现在居然变成了要与一大批『吹笛之恶魔』的魔术师战斗的状况。
即使是给予了敌人撤离的时间，也必须得先折返一次才行。
。
「……只不过，在他们撤退之前，必然会先遇到一个男人」
。
在克罗伊茨的脑海里，宣称要独自一人闯入拉戈尔子爵公馆的、身披铠甲的巨汉的身影一闪而过。
该不会是真的闯进去了吧，克罗伊茨不敢相信，但如果是那个男人的話，也许真的会这么做吧。
不管怎么说，只要他认真行动，事态都必然会朝着惊人的方向发展。
即使在掌握结果之后再行动也不迟，克罗伊茨这么判断。
。
「首先必须跟那个大铠之男汇合呢。要手刃那些家伙……也必须得借他之手。那个男人应该不会成为我们的累赘的」
。
为了在部下的面前保持威严，克罗伊茨并没有把真相坦言。
作为守护王国的剑而日复一日地锻炼的他，居然输给了一般人什么的，怎么可能说得出口呢。
这不仅羞耻，而且还关系到部下们的士气。
克罗伊茨自身的自尊心也是个大问题。
。
「这世上居然还有能与克罗伊茨大人不相伯仲的剑士……不敢相信……！身为克罗伊茨大人的左右手，我也必须得不断精进才行呢」
。
「嗯、啊啊，嘛，你努力吧」
。
相对于使命感旺盛的弗雷，克罗伊茨也只是语无伦次地应声附和而已。
此时，一串笨重的脚步声正逐渐往他们的方向靠近。
克罗伊茨和弗雷回头，发现一名身披大铠的男人正站在他们背后。
毫无疑问是帮助过克罗伊茨的男人。
。
「来、来者何人！」
。
察觉到兰贝尔的非凡气质的弗雷，把手放在剑鞘上并与兰贝尔对峙，
。
「他、他就是我之前所说的合作者！弗雷，快、快把手从剑鞘上松开！」
。
要是得罪了他就完蛋了。
克罗伊茨赶紧命令弗雷，让她订正自己的行为。
。
「您就是那个人吗。那可真是太失礼了……」
。
尽管对克罗伊茨的样子抱有若干的疑问，弗雷也乖乖地退下了。
克罗伊茨轻轻低头。
。
「刚才真是帮大忙了。请允许我再次道谢。不过，不管怎么说，看来您是打消了直接闯入拉戈尔子爵的公馆的念头呢。我真诚地感到欣慰。您实在是过于无谋了。您心中的正义感，即使从我的立场来讲也是值得称赞的。然而，急着送死并不是什么好事」
。
克罗伊茨与大铠之男辞别之后，只不过经过了数刻的时间。
在这段时间之内是不可能向子爵公馆发起袭击的。正常来想的話。
。
相隔几秒后，克罗伊茨再度开口。
。
「……铠之剑士，您有无与我们共同作战的打算？要是您加入的話，我们的第二部队……」
。
「我失败了。也可以说是败北了。拉戈尔子爵因他们的咒术而死亡。八贤者也逃走了。这都是我的疏忽。即使只有八贤者也应该确实地杀死。想顾全两者却得不偿失」
。
「……嗯？」
。
克罗伊茨的嘴角在无法理解的语言面前扭曲了。
。
「请。请等一下。照你这种说法，就像是袭击已经结束、您凯旋而归一样，但这肯定是我们之间的話题不一样对吧」
。
克罗伊茨向兰贝尔伸手，打算打断他的发言。
然而兰贝尔并不在意克罗伊茨的話语，继续开口。
。
「不过我得到了情报。那些家伙的据点是位于矿山附近的特托姆堡。据说，有数十名左右的『吹笛之恶魔』的魔术师匿藏在那。现在马上前去打击的話，必定能大幅削减『吹笛之恶魔』的战斗力」
。
「居然有数十名！？」
。
「数量并不是问题。大部分也只有基尔多姆那种程度的实力而已吧」
。
「基尔多姆有数十人！？」
。
克罗伊茨开始想象，数十名表情狰狞的基尔多姆在他面前生成热球，并操纵大量的等离子兽的情景。
冷静一想的話，这确实是能轻易颠覆他曾经历过的修罗场的地狱。
。
「虽然有些许麻烦，但也只能去做了。只靠我一人的話，也许会出现不少漏网之鱼。能助我一臂之力吗？」
。
已经是打算先折返一次的克罗伊茨惊愕地张大嘴巴，目瞪口呆地注视着兰贝尔。
弗雷似乎也是跟不上理解的样子，轮流地向克罗伊茨和大铠之男送去寻求解释般的视线。
