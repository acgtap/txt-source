「『吹笛之惡魔』的、狂戰士魂淡……。但是，我的部下和同胞們，是絕對不會放過你們的罪行的……」

。
男子咬牙切齒地怒視著蘭貝爾。
。

「都說過我不是他們的同伴了……嗯？」

。
蘭貝爾保持著持劍的姿勢往後觀察。
。
散發著紫紅的光芒，纏繞著紫電的光球正往蘭貝爾的後背迫近。
蘭貝爾縱身一躍而退避。被雷球炸裂的地面迸發出紫電。
。

「唔姆，居然被躲過去了。雖說是準備將打贏的那一方瞬殺的呢」

。
從陰影中出現的人是，戴著厚實白色手袋的、身著禮服的高大老人。
老人手上握著黃金光澤的特徵性的錫杖。
。
在其背後，站著四名身纏披風並戴著假面的劍士。
四名劍士的假面雖為純白色的樸素假面，但在表情上卻分別表現出喜怒哀樂四種感情。
。

「這幾天，聽說有個奇怪的傢伙在街上徘徊不定，應該是你們沒錯吧。吾是『吹笛之惡魔』的基爾多姆(ジルドーム）。大鎧的劍士啊，你就死在這裡吧」

。
四名劍士對基爾多姆的命令作出反應，拔出劍並分別擺出獨特的姿勢。
。
蘭貝爾才不過是第一天來到這個都市。
恐怕他們口中所說的奇怪的傢伙指的就是那個魔銀的劍士吧，蘭貝爾偷瞄了他一眼後，又將視線轉移到基爾多姆身上。
。

「聽說，那是一個長久以來在暗中活躍的組織，沒想到這麼輕易地就報上了名號呢」

。

「無妨。知曉『吹笛之惡魔』之大名的敵人，吾必然會殺之」

。
老人講手上的黃金之杖指向蘭貝爾。
。

「炎熱啊，寄宿於吾之手中吧！」

。
在老人的另一只手中，呈現出小型的火球。
從火球中迸發出來的紫電分成兩個球體並逐漸膨脹。
。
基爾多姆的魔術是以炎熱生成電離氣體，加以控制後便能隨意操縱。
這是老練的基爾多姆才可能使出的超精密高位魔術。
。

「那、那個熱球，還有基爾多姆這名字！？難道是貨真價實的基爾多姆·基雷納斯(ジルドーム・ジレイナス）嗎！」

。
慘敗給蘭貝爾的細劍使驚呼。
。

「你知道他？」

。
蘭貝爾轉過頭盔向他詢問。
。
對於蘭貝爾的提問，男子歪了歪嘴角。
雖然在先前的慘敗中蘇醒，但在『吹笛之惡魔』再次發起襲擊的情況下，就如蘭貝爾一開始所說的那樣，他的確不是敵人。
在此之上，已經沒有和他對峙的理由。
。

「……當然知道，基爾多姆是行踪不明的，國內最高級的鍊金術師啊。畢竟使用者甚少，實際上他是雷吉歐斯王國當中，操使失傳已久的熱電離魔術的三人之一。倒不如說，為什麼你會不知道？」

。
基爾多姆·基雷納斯。
原本他也是名德高望重的鍊金術師，但他領悟到，以過去的方法是不可能到達存在於魔術之盡頭的真理，被不擇手段地到達魔術深淵的八賢者之一、被稱為『真理的編織手』的鍊金術師所吸引，最終加入了『吹笛之惡魔』。

。

「只有、三人？特斯拉格茲(テスラゴズ）也沒能出人頭地呢」

。
特斯拉格茲是存在於八國統一戰爭時代，與雷吉歐斯王國戰鬥到底的瑪邱拉斯王國所屬的鍊金術師。
是個用一輩子的時間創造出熱電離魔術的理論和魔術式，直至運用於實戰之前不斷提升自己的技術，並且擁有掌握技術的出眾素養，就連細致的學習方法都進行了系統化的男人。
十名瑪邱拉斯王國的魔術師中就有一人會操縱這迷一般的魔術，瑪邱拉斯王國也因熱電離魔術的存在而大幅領先其他國家。
。

「對手太糟糕了，鎧男。作為後衛時便擁有無敵性能的熱電離魔術的使用者，再加上四名前衛。帶著孩子逃跑吧。一分鐘……不，我替你拖延三分鐘的時間」

。
男子站起身來，摩挲握著細劍的左手。
。

「不……」

。
在蘭貝爾打算開口之前，男子用手指著蘭貝爾，仿彿在用手制止他說話。
。

「我可不想欠你人情呢。但是，只有一件事請你幫我完成。請向我的部下傳達，暫時撤退，和其他部隊一起將拉戈爾子爵抓獲。報上『不死鳥之瞳』的第二部隊隊長、克羅伊茨(クロイツ ）之名的話，應該會有人相信的」

。
男子、克羅伊茨露出無畏的笑容，向蘭貝爾轉達。
。
由於『吹笛之惡魔』引發的使都市巴萊拉陷入將近毀滅狀態的事件，王家開始正式加強警備，出動王國監察兵團『不死鳥之瞳』的第二部隊，尋找擁有可疑傳聞的貴族。
。

「是嘛，那就拜託你了」

。
蘭貝爾一邊說著，一邊把牽腸掛肚的孩子交給了克羅伊茨。
克羅伊茨雖然楞了一下，但因為蘭貝爾催促般地向他招手，所以就不由自主地接過了孩子。
。

「不、不是這個意思，所以說讓我來殿後……」

。
蘭貝爾握緊手上的大劍並沖向基爾多姆。
四名假面劍士跳到蘭貝爾的面前。
猶如在大劍的攻擊範圍之外包圍著蘭貝爾的四人，分別用長短不一的劍瞄準了蘭貝爾。
與此同時，處於後方的基爾多姆也揮下了錫杖。
。

「化成焦炭吧！」

。
基爾多姆放出纏繞著紫電的赤色的雙球。
企圖在蘭貝爾只顧著和四名劍士纏鬥的時候，用最擅長的魔術解決掉他。
。
那四名劍士只是以藥物、洗腦魔術和生體魔術製造出來的狂戰士。
將他們的劍技一一化解，並且回避從遠處放出的等離子球，這近乎是不可能做到的事。
。
即使是從曾經與眾多劍士交過手的克羅伊茨的角度來看，那四名劍士也不是省油的燈。
雖然看上去是十分亂來的陣型，但一切的一切都有條有理。
看似毫無連攜的不規則動作，在陷入亂戰之際，用手下們準確地維持著能解決掉蘭貝爾的陣型。
。

「別犯傻！快逃啊！」

。
克羅伊茨往蘭貝爾大喊。
。
下一瞬間，斬向蘭貝爾的四名劍士，人首分離，朝著不同的方向飛去。
蘭貝爾將大劍高速地回轉了一圈。
那一記回旋斬的威力驚人。猶如將四面八方的一切裁斷般，如同暴風。
克羅伊茨將蘭貝爾的身姿幻視成龍卷，擦亮了雙眼。
。
將前衛悉數瞬殺的蘭貝爾，易如反掌地躲開了遲來的光球。
。
克羅伊茨和基爾多姆都睜大雙眼，注視著蘭貝爾。
。

「竟然這麼地、超乎常人！……通常魔術不起效的話，只能使用隱藏的王牌，魔犬廷達羅斯了！」

。
失去了前衛的基爾多姆顯露出苦澀的表情。
