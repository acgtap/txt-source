「『吹笛之恶魔』的、狂战士魂淡……。但是，我的部下和同胞们，是绝对不会放过你们的罪行的……」

。
男子咬牙切齿地怒视着兰贝尔。
。

「都说过我不是他们的同伴了……嗯？」

。
兰贝尔保持着持剑的姿势往後观察。
。
散发着紫红的光芒，缠绕着紫电的光球正往兰贝尔的後背迫近。
兰贝尔纵身一跃而退避。被雷球炸裂的地面迸发出紫电。
。

「唔姆，居然被躲过去了。虽说是準备将打赢的那一方瞬杀的呢」

。
从阴影中出现的人是，戴着厚实白色手袋的、身着礼服的高大老人。
老人手上握着黄金光泽的特徵性的锡杖。
。
在其背後，站着四名身缠披风并戴着假面的剑士。
四名剑士的假面虽为纯白色的樸素假面，但在表情上却分别表现出喜怒哀乐四种感情。
。

「这幾天，听说有个奇怪的傢伙在街上徘徊不定，应该是你们没错吧。吾是『吹笛之恶魔』的基尔多姆(ジルドーム）。大铠的剑士啊，你就死在这裡吧」

。
四名剑士对基尔多姆的命令作出反应，拔出剑并分别摆出独特的姿势。
。
兰贝尔才不过是第一天来到这个都市。
恐怕他们口中所说的奇怪的傢伙指的就是那个魔银的剑士吧，兰贝尔偷瞄了他一眼後，又将视线转移到基尔多姆身上。
。

「听说，那是一个长久以来在暗中活跃的组织，没想到这麼轻易地就报上了名号呢」

。

「无妨。知晓『吹笛之恶魔』之大名的敌人，吾必然会杀之」

。
老人讲手上的黄金之杖指向兰贝尔。
。

「炎热啊，寄宿於吾之手中吧！」

。
在老人的另一只手中，呈现出小型的火球。
从火球中迸发出来的紫电分成两个球体并逐渐膨胀。
。
基尔多姆的魔术是以炎热生成电离气体，加以控制後便能随意操纵。
这是老练的基尔多姆才可能使出的超精密高位魔术。
。

「那、那个热球，还有基尔多姆这名字！？难道是货真价实的基尔多姆·基雷纳斯(ジルドーム・ジレイナス）吗！」

。
惨败给兰贝尔的细剑使惊呼。
。

「你知道他？」

。
兰贝尔转过头盔向他询问。
。
对於兰贝尔的提问，男子歪了歪嘴角。
虽然在先前的惨败中苏醒，但在『吹笛之恶魔』再次发起袭击的情况下，就如兰贝尔一开始所说的那样，他的确不是敌人。
在此之上，已经没有和他对峙的理由。
。

「……当然知道，基尔多姆是行踪不明的，国内最高级的鍊金术师啊。毕竟使用者甚少，实际上他是雷吉欧斯王国当中，操使失传已久的热电离魔术的三人之一。倒不如说，为什麼你会不知道？」

。
基尔多姆·基雷纳斯。
原本他也是名德高望重的鍊金术师，但他领悟到，以过去的方法是不可能到达存在於魔术之尽头的真理，被不择手段地到达魔术深渊的八贤者之一、被称为『真理的编织手』的鍊金术师所吸引，最终加入了『吹笛之恶魔』。

。

「只有、三人？特斯拉格兹(テスラゴズ）也没能出人头地呢」

。
特斯拉格兹是存在於八国统一战争时代，与雷吉欧斯王国战鬥到底的玛邱拉斯王国所属的鍊金术师。
是个用一辈子的时间创造出热电离魔术的理论和魔术式，直至运用於实战之前不断提升自己的技术，并且拥有掌握技术的出众素养，就连细致的学习方法都进行了系统化的男人。
十名玛邱拉斯王国的魔术师中就有一人会操纵这迷一般的魔术，玛邱拉斯王国也因热电离魔术的存在而大幅领先其他国家。
。

「对手太糟糕了，铠男。作为後卫时便拥有无敌性能的热电离魔术的使用者，再加上四名前卫。带着孩子逃跑吧。一分钟……不，我替你拖延三分钟的时间」

。
男子站起身来，摩挲握着细剑的左手。
。

「不……」

。
在兰贝尔打算开口之前，男子用手指着兰贝尔，仿佛在用手制止他说話。
。

「我可不想欠你人情呢。但是，只有一件事请你帮我完成。请向我的部下传达，暂时撤退，和其他部队一起将拉戈尔子爵抓获。报上『不死鸟之瞳』的第二部队队长、克罗伊茨(クロイツ ）之名的話，应该会有人相信的」

。
男子、克罗伊茨露出无畏的笑容，向兰贝尔转达。
。
由於『吹笛之恶魔』引发的使都市巴莱拉陷入将近毁灭状态的事件，王家开始正式加强警备，出动王国监察兵团『不死鸟之瞳』的第二部队，寻找拥有可疑传闻的贵族。
。

「是嘛，那就拜托你了」

。
兰贝尔一边说着，一边把牵肠挂肚的孩子交给了克罗伊茨。
克罗伊茨虽然楞了一下，但因为兰贝尔催促般地向他招手，所以就不由自主地接过了孩子。
。

「不、不是这个意思，所以说让我来殿後……」

。
兰贝尔握紧手上的大剑并冲向基尔多姆。
四名假面剑士跳到兰贝尔的面前。
犹如在大剑的攻击範围之外包围着兰贝尔的四人，分别用长短不一的剑瞄准了兰贝尔。
与此同时，處於後方的基尔多姆也挥下了锡杖。
。

「化成焦炭吧！」

。
基尔多姆放出缠绕着紫电的赤色的双球。
企图在兰贝尔只顾着和四名剑士缠斗的时候，用最擅长的魔术解决掉他。
。
那四名剑士只是以药物、洗脑魔术和生体魔术制造出来的狂战士。
将他们的剑技一一化解，并且回避从远处放出的等离子球，这近乎是不可能做到的事。
。
即使是从曾经与众多剑士交过手的克罗伊茨的角度来看，那四名剑士也不是省油的灯。
虽然看上去是十分乱来的阵型，但一切的一切都有条有理。
看似毫无連携的不规则动作，在陷入乱战之际，用手下们準确地维持着能解决掉兰贝尔的阵型。
。

「别犯傻！快逃啊！」

。
克罗伊茨往兰贝尔大喊。
。
下一瞬间，斩向兰贝尔的四名剑士，人首分离，朝着不同的方向飞去。
兰贝尔将大剑高速地回转了一圈。
那一记回旋斩的威力惊人。犹如将四面八方的一切裁断般，如同暴风。
克罗伊茨将兰贝尔的身姿幻视成龙卷，擦亮了双眼。
。
将前卫悉数瞬杀的兰贝尔，易如反掌地躲开了迟来的光球。
。
克罗伊茨和基尔多姆都睁大双眼，注视着兰贝尔。
。

「竟然这麼地、超乎常人！……通常魔术不起效的話，只能使用隐藏的王牌，魔犬廷达罗斯了！」

。
失去了前卫的基尔多姆显露出苦涩的表情。
