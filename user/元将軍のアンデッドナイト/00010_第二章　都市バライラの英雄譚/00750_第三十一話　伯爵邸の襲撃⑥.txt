堕为不死者的优诺斯的凶刃乘上魔法之风，向罗宾汉袭来。
罗宾汉那作为弓手而锻炼到至高境界的动态视力捕捉到了优诺斯的不规则移动。
。

「啧！」

。
小刀从罗宾汉的指尖处投掷出去。
因为舞游剑必须用风魔术预先做出轨道，从中派生的动作模式已经固定。
对於只是一味地模仿生前的战鬥经验中的动作的不死族来说，这个特征十分显著。
。
罗宾汉已经从正面击败了优诺斯的舞游剑一次。
因此他才能够看穿舞游剑那複杂的动作，并向轨迹的前方投出小刀。
依从自己制造的风而移动的舞游剑并不能急停。
。
释放在无法防御的位置的小刀，优诺斯用双目捕捉到了。
但已经应付不过来了。
小刀扎在了优诺斯的太阳穴上。
优诺斯那血管暴露的空虚的白眼惊愕地睁大了。
。
在优诺斯停下动作的一瞬间，罗宾汉拉弓放箭。
右膝盖、左手、接着是右眼。
无论是生命力有多麼顽强的不死者，关节部位或感觉器官被破坏的話只会成为众矢之的。
。
本应是一击必中的机会。
但优诺斯立即翻过身子并用剑一挥。
。

「风啊，回弹吧！」

。
优诺斯制造的风扰乱了箭矢的轨道。
只有一支没有被错开的箭矢射中了优诺斯的腹部。
优诺斯用手拔出插在腹部的箭矢，用力握断并扔在地上。
。
见此情形，罗宾汉一边拉开距离一边咋舌。
。
(和之前的不死者相比，防守能力太强了……）

。
罗宾汉的箭矢即使被铠甲挡住，也至少能让对手伤筋断骨。
尽管腹部受到了伤害，优诺斯反而利用箭矢的力量，扭转身体并重整体势，作出对应追击的姿势。
。

「正因为是不死者所以才能够作出这样的动作吗。优诺斯的剑术和魔术，再加上不死者的顽强，真是太糟糕了。这恶劣的性格和人格一起消失才是唯一的救赎吧」

。
罗宾汉一边用右眼警戒优诺斯的动作，一边转动左眼确认屋子裡的情况。

『舞动之剑』的五名冒险者正和曼奇带来的另外两名不死者交战。
范多处在中心，大声地发出指示互相协作，保持对智力低下的不死者的优势。
但在罗宾汉看来，那两名不死者虽不及优诺斯，但也是用剑的高手。
虽然现在是用数量和連携的差距来压制，但考虑到不死者的不死性质和韧性的話，已经受伤出血、只是握着剑就感到疲倦的『舞动之剑』的冒险者们迟早也会支撑不住。
。
八贤者之一，奇怪的老人曼奇一动不动。
只是露出浅薄的坏笑，观察着战况。
在他左右还有两名不死者。
。
让那两名不死者也活动起来的話，无论何时都可以结束这场战鬥吧。
而他却没有这麼做，这就是曼奇把这场战鬥当做是让不死者和冒险者战鬥的遊戏的证明。
。
罗宾汉也察觉到了这件事。
可以的話，他打算拖延和优诺斯的打斗，对疏忽大意的曼奇给予致命性的打击。
。
但这近乎是不可能的。
毕竟在曼奇身旁不仅配置着两名不死者，而且用盖洛夫之书召唤的冥府精灵·杜拉罕也藏形匿影地守护着他。
比起那些不死者，曼奇才是真正的强敌。
。
杜拉罕于不可见状态中操使双剑的神速的一击，已经超越了人类能够处理的範畴。
况且曼奇的手上还有盖洛夫之书，倘若曼奇认真战鬥的話，也担心和杜拉罕同样等级的精灵增多。
在曼奇持有盖洛夫之书以上，他们就必败无疑。
。
(战力完全不够……。可怕的是，即便是有部下的辅助，用盖洛夫之书进行大规模的死操术和精灵召唤，从曼奇身上看不出一丁点因玛娜不足所造成的疲劳。若是常人的話，在玛娜枯竭时早就失去意识了。要杀死那傢伙的話，只能让他在还保留着手牌的情况下死去了……不过看来他是不会那麼轻易地给我们这种机会了）

。
罗宾汉早已断定，用正攻法是不可能攻破的。
然而，对曼奇制定策略，罗宾汉却连实行计划的犹豫的时间都没有。
。

「啊哈，啊哈哈哈哈！」

。
持剑的优诺斯笔直地冲向罗宾汉。
罗宾汉拉弓作出迎击。
在这刹那，优诺斯急剧提高了速度，完全回避了三连射。
。

「动、动作，比生前还要快了！？这不可能！」

。
罗宾汉接连放出的箭矢，都被优诺斯的剑逐一拦截。
用肩膀挡住牵制的箭矢，并活用不死族的不死性质强行缩短距离。
。

「哦呀，变得有趣起来了呢」

。
曼奇兴致勃勃地把意识转向优诺斯那边。
以曼奇的話语为开端，罗宾汉开始解析优诺斯的状态。
用双眼观察他双脚的动作、手臂的动作、面肌的动作，他立马就得出了结论。
。
(与其说是变快了，不如说只是单纯地提高了筋力吗。不对，恐怕只是一开始并没有发挥出全力而已。而现在却变得能熟练使用的話就是说……）

。

「好强、好强的力量！啊哈、啊哈哈哈哈哈！这样我、就是，都市巴莱拉，最强的冒险者了！」

。

「恢復了一点理智吗！」

。
这也不是不可能的事。
拥有强大玛娜的高位魔术师，或是持有敏锐的精神的剑士，苏醒成为不死者後，残留在尸体上的玛娜作为根基，有时会出现人格復甦的情况。
不过也并不是回到完整的状态，最多也只是机械性地重複生前的話语或思考。
优诺斯恰巧也是这样。
。

「拥有这个力量的話，拥有这个力量的話，我就天下无敌了！啊哈、啊哈哈哈！我、我从出生时就拥有剑才和智谋，注定要成为贵族……！在这之前，你太碍事了罗宾汉！！」

。

「蠢 货！你已经不是活人了啊！」

。
优诺斯生前的剑术和魔术的本领以及不死族的耐久，此时更是因为恢復了理智而完全地驾驭住了强大的尸体。
行动单调——不死族最大的弱点也克服了。
。
罗宾汉的三连射击，被优诺斯用最低限度的动作避开了致命伤。
即使打在身上，只要不是明显的缺损和关节的损伤，对不死化的优诺斯来说是没问题的。
。

「风啊，助我一臂之力吧」

。
优诺斯在绝佳的时机使用了风魔术。
优诺斯并没有浮在空中，而是低空滑行略过箭矢，在罗宾汉的面前着陆。
因为长距离的移动会成为罗宾汉动态视力的猎物，所以他毫不犹豫地选择短距离的高速移动，这是深思熟虑的理性的行动。
。

「失手了！」

。

「去死的人，是你啊！」

。
罗宾汉後跳回避优诺斯的连击。
然而，对於优诺斯穷追不捨的攻击，罗宾汉并不能再次拉开间隔。
最终，优诺斯的剑划过罗宾汉的脸颊，留下一条红色的细线。
。
罗宾汉往背後一跳，碰到了後面的墙壁。
。
(啧，没有后路了吗。太糟糕了）

。
优诺斯瞬间缩短了距离。
在罗宾汉做好觉悟的瞬间，伴随着一声巨响，窗户被打破了。
从外面闯入的人，是身着如同金属块的铠甲的彪形大汉。
大汉在罗宾汉的身旁着地。房间的地板被超重的铠甲碾压。
。
房间内的气氛为之一变。
发出攻势的优诺斯，也察觉到危险而马上退却。
其他不死者和冒险者也停止战鬥，和交战的对手拉远距离，转移到乱入的人物身上。
就连曼奇，也停下了脸色难看的笑容，露出阴险的表情。
。

「妨碍老朽的餘兴的人是谁？」

。

「餘兴，吗。你还想着继续狩猎吗」

。
大铠的男人，用巨大的剑叩向地面。
房间再次摇动，大大的裂纹陷在地上。
。

「从现在起，被狩猎的就是你」

。
在兰贝尔头盔的深处，空洞的两个眼窝紧盯着曼奇。
