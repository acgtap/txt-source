「蘭貝爾·德拉克羅瓦！？在敬愛蓋洛夫大人的老朽面前，偏要冒充那個蘭貝爾！罪該萬死！」

。
曼奇大叫道，用力握緊了蓋洛夫之書。
。

「不，老朽不會那麼容易讓你死去的！老朽要用蓋洛夫大人的魔術，把你變成只會感受到痛苦的肉塊，讓你苟延殘喘直至死去！只不過是打倒了兩只強化不死者而已，別太得意忘形了！」

。
蘭貝爾沖向曼奇。
曼奇用左手支撐著書，右手對准蘭貝爾。
。

「火啊 燒盡一切吧！」

。
曼奇以蓋洛夫之書為媒介發動了魔術。
從曼奇的右手中出現了超過兩米高的火焰，燒焦了天花板和地板。
曼奇的魔術比蘭貝爾的動作遲了一步，緊隨其後放出了火焰。
。

「喲吼吼吼！你的鎧甲會燒焦你的皮肉，成為陪伴你的棺材吧！」

。

「這個魔術的規模，是怎麼回事……！？」

。

『舞動之劍』的冒險者們目瞪口呆地嘟囔道。
曼奇的魔術和普通的魔術師的魔術簡直是不能相提並論。
發動需要的時間、規模和威力。
不僅如此，用於引發特異現象的瑪娜的總量，實在是太過龐大了。
。
在魔術中，無中生有必然是要消耗大量的瑪娜。
因此有不少使用土或風進行攻擊的魔術師。
即使是操縱火焰，最多也只是放出拳頭那麼大的火球而已。
若是普通的魔法師，生成那麼大量的火焰的話，應該很快就會因為瑪娜枯竭而不能行動，嚴重的話連命都不保。
。
曼奇以歷史上的怪物，鍊金術師蓋洛夫的魔導書為媒介從而操使魔術。
蓋洛夫之書因其擁有複雜的魔術式而將瑪娜的效率提升至極限，作為魔術的媒介已經是完成品。
這份力量固然是無以倫比的，但能生成這個規模的火焰的原因，還是因為曼奇那壓倒性的瑪娜總量。
。

「這可不是冒險者能應付的範圍……這個魔術，已經能匹敵五大宮廷魔術師了哦」

。
正在和強化不死者戰鬥並指揮同伴的范多，用餘光觀察曼奇和蘭貝爾的交戰後流下了一行冷汗。
。

「擁有這種程度的魔術本領，為何要加入禁忌魔術組織……！這樣下去可接近不了他啊！」

。
范多所說的話有三個錯誤。
第一是，曼奇擅長的到底只是死靈魔術，這只不過是逼不得已才拿出的手段罷了。
第二是，曼奇並不是作為魔術師而踏上邪道，只不過是從年幼時人格就已經扭曲的人獲得了強大的力量進而成為了強大的惡人而已。
而最後的一個錯誤是，對蘭貝爾來說，這種程度的火焰沒什麼大不了的。
。
盡管蘭貝爾在一紙之隔回避，但動作上還尚有餘裕。
蘭貝爾鋌而走險，用最短的路線拉近了和曼奇的距離。
。

「使用蓋洛夫之書，就這種程度嗎」

。
對於蘭貝爾的評價，曼奇那溝壑縱橫的臉上又增加了幾分皺紋，露出了憤怒的表情。
范多所畏懼的匹敵王國魔術師頂點的曼奇的魔術，對曾經承受過蓋洛夫釋放的魔術的蘭貝爾來說，這只是遠比想象中要遜色的伎倆罷了。
。
最終，蘭貝爾的大劍在能動的範圍內捕捉到曼奇。
兩人四目相接。
。

「那本書就讓我破壞它吧」

。

「**，去死的那個是你！」

。
在蘭貝爾眼前，身著漆黑之鎧的無頭劍士，用右劍高舉的姿勢突然出現。
是藏形匿影地守在曼奇身旁的、蓋洛夫契約下的冥府十三精靈之一杜拉罕。
出其不意揮下的刀刃，被蘭貝爾的大劍擋住了。
。

「蓋洛夫的十三精靈，第六之杜拉罕嗎。兩百年不見了呢」

。
蘭貝爾的劍和杜拉罕的劍交鋒，互相抵消。
杜拉罕用雙劍向蘭貝爾放出超越人類的劍技。
突破音速的連擊，都被蘭貝爾一一化解。
。
蘭貝爾轉為被動，在杜拉罕的劍技面前讓了將近三十手。
即使是制霸八國統一戰爭的雷吉歐斯王國四魔將之一角，蓋洛夫的十三精靈也不是那麼容易就能打敗的敵人。
蘭貝爾往地一踢，跳向後方。
。

「原來如此，在我沉睡的期間變強了嗎」

。

「…………」

。
杜拉罕無言地擺好姿勢。
。
杜拉罕左右的劍是不一樣的。
右手的劍，是用魔銀秘絲利爾打造而成，是質量和硬度平衡的薄刃劍。
是杜拉罕高速連擊的主力。
。
左手的劍全身均黑，若隱若現。
一眼就知道是異界的魔劍。
作為杜拉罕身體的一部分的這把劍，是穿過肉體，直擊並斬斷生命的被詛咒的魔劍。
蘭貝爾陷入防守也是因為這把魔劍。
擁有實體的魔銀劍，和穿過實體的防御不能的魔劍。
。
只是其中一把的話就不難對付。
然而，對於必須用不同方法對應的神速的雙劍連擊，蘭貝爾也陷入了苦戰。
。
曼奇在壓制都市巴萊拉的冒險者公會時，基本上都是用杜拉罕來碾壓。
杜拉罕這種程度的精靈劍士只要把咒劍一揮，無論是多麼強大的公會，也只會被一瞬間鎮壓下來。
。
對曼奇來說，魔術所造成的瑪娜的浪費是要盡量避免的。
況且正面交戰的話，以大型公會為對手，即使是曼奇也不可能毫髮無損地接連攻陷。
而且，杜拉罕那不對屍體造成損傷的劍，和操縱屍體的曼奇相性良好。
。
若是召喚第七以上的精靈，曼奇不僅需要布置大型的儀式，對自身也有危險。
到第十二以上的精靈，連召喚本身都是不可能的。
所以曼奇經常使役第六的杜拉罕。
無論遇到多麼強大的劍士，都能用其超凡的劍技瞬殺的杜拉罕，對曼奇來說就是最強之劍。
但另一方面，就是因為太強所以也太無聊了，通常都只是用來壓制大局或是像這次一樣作為最後的保險。
。
自從曼奇得到蓋洛夫之書以來，由於杜拉罕的力量，他從未遇到過生命攸關的情況。
但杜拉罕現在卻陷入僵局的情況，對曼奇造成了極大的壓力。
。

「你都在干些什麼！不要再玩耍了！不用生擒了，不用了！殺了他！快給我殺了他啊！」

。
對曼奇的命令作出反應，杜拉罕再次沖了出去。
超越常人能夠認知的非人的決鬥再次開始。
曼奇用他那痙攣的嘴角作出扭曲的笑容，向那場戰鬥伸出手腕。
。

「火啊 燒盡一切吧！」

。
作為精靈的杜拉罕，即使損失了身體的大部分，也只會在這個世界消失回歸異界而已。
因此曼奇才將杜拉罕連同蘭貝爾一齊燃盡。
。
兩名劍士被曼奇釋放的熊熊烈焰包圍。
。

「喲吼吼吼！幹掉了，幹掉了！終於殺死了！喲吼吼吼……」

。
曼奇放聲大笑。
從火焰中，附有瘴氣的黑色之劍被彈飛在外。
。

「喲呵……！？」

。
在下一瞬間，杜拉罕從火焰中被踢飛，摔在了牆上。
手中的劍消失不見。漆黑之鎧被一刀兩斷，或許是已經不能維持在這個世界的存在，身體開始變得透明。
。

「怎、怎麼會！？為什麼……」

。
從炎之柱中，出現了扛著大劍的蘭貝爾的身影。
。

「雖然不知道你是否有支援的打算，不過失去集中力的是你那邊的精靈呢」

。
被曼奇的火焰吸引注意力的杜拉罕，于一瞬間，用雙劍使出的精妙的連擊變得遲鈍。
這一瞬間足以分出勝負。
結果，在激戰中勝出的人是身纏不受半吊子魔術影響的純魔金之鎧、並且在精神面上也是更勝一籌的蘭貝爾。
