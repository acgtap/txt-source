「別想逃，把蓋洛夫之書交出來！」

。
蘭貝爾一邊騎著無頭的黑馬，一邊揮舞大劍，在荒廢的都市巴萊拉中突進。
目擊到異樣風貌的騎士的居民們，大呼小叫地到處亂竄。
。
蘭貝爾在追趕著的，是跨在擁有雙面的奇怪妖馬特洛伊上的八賢者曼奇。
雖然兩者都操縱著異形之馬，但相比于蘭貝爾的迫力，曼奇明顯處於劣勢。
曼奇自己也做好了被追上的話就會被殺死的覺悟。
。
雖說夢魘是匹有著驚人速度的好馬，不過背負著身穿魔金鎧的超重的蘭貝爾對他造成了莫大的負擔。
然而，即便是這樣它也著實地縮短著和曼奇的距離。
照這樣下去的話，不過多久應該就能追上曼奇了吧。
。

「你果然不是人類吧！可惡，為何、為何，要妨礙老朽！為何要追著老朽不放！」

。
曼奇一邊謾罵蘭貝爾這個不講道理的石頭腦袋，一邊斜倪著背後的蘭貝爾。
慌慌張張地翻開蓋洛夫之書。
。
(就連第六位的杜拉罕也敗在他手下。第五位的刻耳柏洛斯簡直不在話下！怎麼可能和那種怪物正面交戰！在儀式完成之前，不能再被那傢伙干擾了！）

。
經過伯爵府邸的一戰，曼奇已經深切地認識到蘭貝爾的可怕之處。
在此之上，已經完全沒有正面交鋒的打算。
。

「回應我的呼喚，從冥府中現身吧，黑之傀儡師莉莉絲！」

。
曼奇叫喚的同時，在他跨著的妖馬的尾上，出現了黑色的外套。
病態般的白色肌膚的少女，從外套中窺伺著周圍。
異界的少女，胸部以下空無一物，宛如固定在曼奇的背後般地漂浮在空中。
少女くすくす地竊笑，伸出細長的手指。
。
這是蓋洛夫的十三精靈的第四位，黑之傀儡師莉莉絲。
雖然本體並沒有戰鬥能力，但她能夠將視野內的屍體如同自己的傀儡一樣自由自在地操縱。
。
只是，這終究只是莉莉絲讓屍體活動起來，並不能再現屍體生前的戰鬥技術，無論是達人的屍體還是小孩的屍體，都沒有太大的區別。
雖然只要有屍體就能夠當場制造出大量的士兵，但到底也只是依賴數量的雜兵而已。
。
因此，需要使用到她的情況少之又少。
但在經歷過大規模戰鬥的現在的都市巴萊拉中，黑之傀儡師能发揮出壓倒性的力量。
隱藏在外套中的莉莉絲的眼瞳捕捉到蘭貝爾。她的嘴角微微上揚。
。

「莉莉絲、嗎……」

。
莉莉絲雙手高舉。
在她的十只手指尖處發出青白色的光芒，倒在路邊的屍體緩緩起身，一齊堵住了蘭貝爾的前進方向。
屍體的數目很快超過了二十。
女性、小孩、冒險者、老人。
雖是雜亂無章的構成，但經過莉莉絲之手，即使失去了手腳他們也一概成為了好戰的狂戰士。
。
不死者瞄准的並不是蘭貝爾。
而是蘭貝爾所騎的無頭馬，夢魘的四足。
即使不敵蘭貝爾，若是落馬的話對他本人和馬也會造成巨大的傷害吧。
不死者壓低姿勢等待時機，打算撲向夢魘的俊足。
。

「去吧！擋住他、擋住他！」

。
曼奇滿懷期待地叫到，但這完全沒有意義。
蘭貝爾揮舞的大劍將死者彈開，接著被無頭馬踢飛，強行開出一條道路。
只是稍微地拉開了一點距離，僅此而已。
。

「你，你到底是什麼人啊！」

。
復活的屍體的軍勢好幾次堵住了蘭貝爾的去路，蘭貝爾也好幾次輕而易舉地越過障礙。
。

「差不多該死心了吧。在此之上不要再褻瀆死者了」

。
蘭貝爾說著，突然有種奇怪的感覺。
周圍的屍體之中，無頭的屍體似乎在增多。
而且在追趕曼奇的同時數量逐漸增多。
這不是偶然。是因為某些目的，才製作出無頭的屍體。
也就是說，他在準備著什麼。
曾經和最惡的鍊金術師蓋洛夫戰鬥過的蘭貝爾知曉他的目的。
。

「那是，屍玩竜之宴……！」

。
蘭貝爾也產生了動搖。
走在前頭的曼奇緩緩回頭，在褪青的面容上浮露出醜惡的笑容。
。

「為什麼你會知道？算了，無所謂了。來到這裡的話，就已經阻止不了老朽了！老朽的莉莉絲，雖然不至於殺死你，但也足以阻止你前進」

。
曼奇移動的前方，有兩名身披長袍的可疑人物。
在他們周圍刻畫著直徑約十米的巨大魔法陣，而于刻在內側的圓陣中，堆積了大量的人頭。
。
對異界的住民的精靈來說，合適的時刻、場所、狀況和魔力場是必要的。
越是高位的精靈，條件難以滿足的傾向越強。
用魔術干涉，強行制造出合乎召喚精靈的條件的行為，被稱作儀式。
屍玩竜之宴也是儀式的一種，對於召喚以蓋洛夫之書結成契約的高位精靈是不可欠缺的。
。

「是你輸了，鎧之大男……不對，不死騎士啊。不過是個不死者，將老朽逼到這個地步真是值得稱讚。然而，老朽最初的目的就是屍玩竜之宴！這只不過是為了召喚出，用屍山堆積而成的詛咒之龍布局而已！而且這個布局現在已經完成了！喲呵呵呵！」

。
妖馬特洛伊停下了腳步。
兩名待機著的男人，對跨著特洛伊的曼奇投向視線。
。

「連莉莉絲也已經召喚出來，也就是說，曼奇大人要進行儀式了呢！」

。

「可是，背後的那個人是……」

。
兩名身披長袍的男人，看到沖向他們的蘭貝爾便不由自主地痙攣起來。
曼奇無視那兩人，抬頭看向在特洛伊的尾巴附近浮游著的女童、黑之傀儡師莉莉絲。
。

「莉莉絲啊，呼喚出屍玩竜吧！」

。
莉莉絲露出和年齡不相符的大人般的笑容，向魔法陣內堆積起來的人頭伸出手。
因莉莉絲的力量，作為屍體一部分的人頭開始作出規則的浮動。
與此同時，緊閉的雙目大大地睜開，暴露出焦點不合的眼瞳。
。
人頭在魔法陣上方漂浮，以規則的形狀設置在上空。
人頭的下顎顫動，發出咯噠咯噠的笑聲。
毫無疑問是極度惡趣味的景觀。
才剛升起的月亮，照映出怪異的世界。
魔法陣中，滴血的梟首在飛舞。
曼奇翻開蓋洛夫之書吼叫道。
。

「回應我的呼喚，從冥府中現身吧，屍玩の竜ラウンプゥプ！」

。
瑪娜之光聚集於魔法陣的中央，具象出膨脹而成的巨大輪廓。
被發出狂笑的無數的梟首包圍，高達接近兩米的巨大的人頭從中升起。
人頭的眼中沒有任何感情，連皮膚也有某種異質的感覺。
人頭的頭頂沒有頭髮，逐漸放大的眼瞳中有著嬰兒的人偶般的感覺。
。
在身體上雖然也有膚色，但構造上明顯和人類不一樣。
長長的胴體有著好幾處的關節，從側腹處延伸出如同多足類動物般的，大量的手腕。
異形的身體彎折，降落到地上。全長超過二十米。
冥府的精靈屍玩の竜ラウンプゥプ，大大地張開面無表情的大口，發出驚悚的哭叫聲。
。

「啊……啊啊啊啊啊啊啊啊啊！」

。
它啃食了被莉莉絲升到空中的一個人頭，又厭煩地將另一個人頭用手握碎。
。
即將接近魔法陣而急速移動的蘭貝爾，也領悟到已經阻止不了ラウンプゥプ的出現，讓夢魘的速度降了下來。
只是看著就能奪取人類的意識的異形怪物，扭轉那令人不快的身體而改變身體的方向。
。

「曼、曼奇大人！姑且！請讓它停下……」

。
兩名部下從唐突地開始動作的ラウンプゥプ身邊逃離，並向曼奇求救。
然而在曼奇的眼中並沒有他們的存在。
瘋狂的老人的雙眼已經陶醉于異形的怪物。
觀察那多節的胴體，不気味的巨頭，還有不快的動作，他發出了感嘆。
。

「哦哦哦哦……很好，非常好。無論觀察多少次，ラウンプゥプ果然很美麗，很壯觀啊！」

。
曼奇無視揚起悲鳴，尋求幫助的部下。
其中一人成為了ラウンプゥプ的墊板被壓扁，從口中吐出內臟而絕命。
第二人被無數的巨腕抓住並提起，碎屍萬段而死亡。
改變方向的ラウンプゥプ抬起巨大的上體，俯視跨著夢魘的蘭貝爾。
