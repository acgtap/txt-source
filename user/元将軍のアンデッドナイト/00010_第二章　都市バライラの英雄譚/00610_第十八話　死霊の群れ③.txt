湧出不死者的建築物，是大型冒險者公會『鬼蟻之軍勢』的據點。

『鬼蟻之軍勢』，是都市巴萊拉中的強有力的公會之一。
相比於其他公會的優勢，是其壓倒性的加盟冒險者數。
雖然個人的素質並不是很高，但總共八十名成員的大型公會，在雷吉歐斯王國全國中也是屈指可數的。
因而，在都市巴萊拉中有著不俗的影響力。
。
正是『鬼蟻之軍勢』的冒險者們被奪取了性命，被死操術操縱著身體，成為了不死者。
接連湧出的冒險者們，各自拿著自己的武器，不斷襲擊著路上的行人。
街道頓時陷入在大混亂之中。
。

「為、為什麼會這樣……！」
「快逃啊！怎麼看都不正常啊！」

。
被捉住的行人，身體被變成不死者的冒險者包圍，接著就被不死者用刀劍刺殺了。
也有失去了理性，用牙齒撕咬對方的不死者。
。
蘭貝爾的關注點，已經從羅賓漢轉移到了原『鬼蟻之軍勢』的不死者身上了。
蘭貝爾筆直地奔往『鬼蟻之軍勢』的據點。
。

「是、是我啊！埃德加先生，請您恢復理智吧！」

。
一名被不死者包圍的女冒險者，拼死地向一名男性哀求。
她用顫抖的雙手握著劍，但看見男性搖搖晃晃地靠近她時，她卻松開了手上的劍。
兩眼充血的男人，雙手粗魯地拿著兩把小刀，飛快地襲向女冒險者。
在女冒險者做好覺悟閉上雙眼的時候，從旁插入的蘭貝爾用籠手擋住了男人的小刀，接著順勢把男人彈開了。
男人撞在了其他不死者身上，在地面上翻滾。
。

「啊……」

。
這時全身脫力的冒險者，視線無力地落在支離玻碎的男人身上，然後用責難的眼神看向蘭貝爾。

「他已經死了，沒救了。趕緊離開吧。我可沒有確保每一個人的逃生路線的餘裕」

。
蘭貝爾無情地說道。
但那也是妥善的說法。
引起這次暴動的人，估計就在不遠的地方。
如果一個人一個人地進行救助的話，結果只會造成更多的死者。
。
咻地一聲，從上方飛來了箭矢。
箭矢精確地將三個不死者的膝蓋射穿從而奪走行走的機能，令它們摔倒。
出現在屋頂上的人，是拉弓的橙髮美青年羅賓漢。
。

「真是令人作嘔的做法」

。
羅賓漢厭倦地嘟噥道，迅速地準備好下一支箭矢。
幾乎同時射出的三支箭矢，又射穿了另外三個不死者的膝蓋。
。
停止成為死操術對象的不死者的行動的條件，根據術者的手法也有著很大的差異。
打碎頭部雖然不失為一種方法，即使奪走一只腳，屍體也會繼續爬行。
要確實地停止機能的話，奪走雙腳是最好的辦法。
姑且不論，從高處精確地狙擊到處活動的不死者的膝關節什麼的，也只有他這種技術高超的人才能做到了。
。

「……」

。
蘭貝爾和羅賓漢一瞬間對上了視線。
蘭貝爾輕輕點頭，接著立刻把意識轉移到不死冒險者身上。
那一瞬間他們便達成了停戰協議。
。
在蘭貝爾看來，羅賓漢的人格已經支離破碎了。
他在前几日也心血來潮地動身襲擊善良的冒險者。
因為把食人魔群招引到都市而被判為死罪，越獄後又對領主懷恨在心，真是令人驚訝。
說實話，他的人格已經不正常了。
。
但是，這次的敵人，規模實在是太大了。
不死冒險者的數目至少也有五十以上。
只是這個數量就已經遠超羅賓漢造成的災害了吧。
。
現在可不是和其他敵人纏斗的時候。
——現在，放他一馬吧。
這就是蘭貝爾得出的結論。
。
羅賓漢站在高處，優先地狙擊那些襲擊領民的不死者，使他們不能動彈。
他的行為對蘭貝爾來說也是有益的，
這樣一來，蘭貝爾就能毫無顧慮地尋找引起不死者騷動的主犯了。
。
不顧一切地打倒他的話，受害者則會增加。
只能放著他不管，等到騷動落幕之後再開始尋找羅賓漢了。
。

「哈啊啊！」

。
豪快地揮舞著的大劍，把不死者的身體一刀兩斷。
蘭貝爾的動作中沒有一絲躊躇。
即使原本是人類，但現在的蘭貝爾只不過是一具骸骨。
在戰場上不需要同情。操縱屍體進行戰鬥的魔術師，在八國統一戰爭時代俯首皆是。
猶豫只會成為弱點。
蘭貝爾曾經也斬下過部下的頭顱。
。

「咿！！救我！！快救我啊！！」

。
一名初老的老人，在路上連滾帶爬地逃走著。
在他手中不知道藏著什麼東西。
在那名男人進入蘭貝爾的視線時，蘭貝爾折起雙腿迅速地跳到老人的面前。
接著用大劍放出斜向上的袈裟斬。
。

「啊噶！怎、怎麼會……」

。
男人的身體被切成兩半，倒在地上。
被切開的黑水晶落在了地上。以此為信號，周圍的不死者從口中吐出混雜著綠色和紅色的液體，然後就像扯線人偶一樣倒下了。
被切開的黑水晶，宛如蒸发般地消失了。
。
死黑水晶……是有著以群為單位，支配不死者的力量的水晶。
死黑水晶的成分中，術者的瑪娜占據了一大部分。
在行使死操術之前先生成死黑水晶，用其中的瑪娜對不死者進行管理。
。
蘭貝爾那作為不死者的嗅覺，能夠分辨出生者。
感知所有生物自身所持有的根源上的能量，從中再判別對象是否存活。
因此自從蘭貝爾成為不死者之後，能夠感覺到瑪娜塊的存在。
。
察覺到那名形跡可疑的老人也是多虧了這個能力。
男人假裝成受害者，在死黑水晶的有效範圍內來回行走。
。

「……」

。
但是，這也太奇怪了。
蘭貝爾不認為，這名男人能夠憑一人之力殺死『鬼蟻之軍勢』的所有人。
如果是在襲擊時分成多個水晶，這個人只是管理著其中一個死黑水晶的話，那就說得通了。
就像是印證了蘭貝爾的想法一樣，從遠處也傳出了悲鳴。
而且還不是從同一個方向傳來。都市巴萊拉的各處同時地傳來了悲鳴。
。
分岔路的其中之一被箭矢射中。
。

「我去那邊。你去另一邊吧。怎樣？不是場壞交易吧」

。
從頭上傳來羅賓漢的聲音。
在災害範圍擴大的情況下，分散實力者的確能減輕受災。
而且對羅賓漢來說，兵分兩路的話，完事之後被當即處分的危險度也降低了。
完成自己的工作之後，就能輕輕鬆鬆地離開這裡了。
。

「……」

。
蘭貝爾無言地背對著羅賓漢奔跑。
羅賓漢跳下來的時候，已經看不見蘭貝爾的身影了。
。

「雖然也可以趁著騷動，趕緊地完成我的目的……嘛，這麼做那些傢伙也不會高興的吧」

。
羅賓漢 朝著蘭貝爾驅趕的方向，小聲地嘀咕道。
羅賓漢吹了吹口哨。
蒼色 的名馬、塞拉夫在空蕩蕩的街道上疾馳。
羅賓漢騎上塞拉夫，前往了蘭貝爾的反方向的街道。
