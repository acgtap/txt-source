在伯爵府邸三樓的辦公室窗戶旁，蒙德伯爵目瞪口呆地看著領地的光景。
從傳來在都市巴萊拉的中央部發起了謎一樣的暴動的情報，到格拉斯科率領私兵團對其進行鎮壓、調查，還沒有經過多少時間。
然而，現在的都市卻從四處各地冒出滾滾濃煙，橫屍遍野，一眼就能看出失去理性的冒險者們正在進行屠殺。
。

「原來是這樣，正體不明的暴動的真面目，竟然是死靈之群造成的……。這麼一來，幕後黑手肯定是，『吹笛之惡魔』……雖然有聽說過傳聞，但沒想到居然這麼快就開始行動了」

。
蒙德伯爵緊握的拳頭微微顫抖，俯視著化為地獄的自己的領地。
銀髮的文雅男子、『舞動之劍』的公會長優諾斯來到他的身旁，向下俯視街道。
他稍稍地睜開細長的眼睛，觀察狂暴的不死者，流露出十分不愉快的表情。
。
優諾斯打算通過這次危機，向蒙德伯爵出謀划策，以此提高蒙德伯爵對自己的信任。
比起蒙德伯爵的家臣們，在殘酷的冒險者世界中，把自身天生的聰慧和上進心作為武器而斬荊披棘的優諾斯擁有著更豐富的實戰經驗，而且頭腦也很靈活。
在危機中做出貢獻的，不是因身份或血緣關係而納入考慮範圍的家臣，而是優諾斯。
因此，隨意訪問蒙德伯爵的辦公室的特權也被伯爵允許了。
。
優諾斯最初也因為這個出乎意料的事態而暗自竊喜。
但『吹笛之惡魔』造成的損失遠遠超過了他的想象。
。
優諾斯至今為止的活躍都只是為了被冒險者之都巴萊拉看中。
只是，這樣下去都市內的冒險者人數也會減半的吧。
今後至少十年都會以都市復興作為主要工作，能否恢復到和以前同樣的繁榮也是個問題。
先不說這些，連這次暴動能不能鎮壓住也不得而知。
在這樣下去，這個都市連存在的價值都會蕩然無存。
。

「雖然已經向各冒險者公會派遣使者，號召他們協力鎮壓『吹笛之惡魔』的暴動……但受災的規模實在是太大了。即使是向附近的領地請求救援，也不知道何時才會到達。對方的行動，實在是太快了」

。

「是、這樣呢」

。
對於蒙德伯爵的哀嘆，優諾斯也只能消極地回應。
。

「我在考慮要不要也讓優諾斯殿下的『舞動之劍』，前去討伐不死者以及『吹笛之惡魔』的魔術師……優諾斯殿下，你怎麼想？」

。
蒙德伯爵話語的含義，是放棄伯爵府邸的防御。
原本，蒙德伯爵是因為畏懼未被處刑並化為狂人的羅賓漢的復仇，才雇傭了『舞動之劍』。

此時都市全境開始被『吹笛之惡魔』攻擊，這對蒙德伯爵來說是不幸中的大幸。
。
但事到如今，他認為繼續把一大勢力的『舞動之劍』作為伯爵府邸的護衛而束縛著的話，可能只是一步壞棋。
總之，受災規模比想象中要大得多。
。
被蒙德伯爵徵求意見時，優諾斯從窗戶觀望處於正下方的伯爵府邸的圍牆周圍。
在伯爵府邸周圍，『舞動之劍』的部下，和蒙德伯爵的一部分私兵，在和攻進來的不死者展開戰鬥。
。
戰鬥目前還算安穩。
先不說蒙德伯爵的私兵，持有洗練的戰鬥技能的『舞動之劍』冒險者們行動流暢，現階段似乎還未出現死傷者。
。
但留在此處，被攻陷也是時間的問題。
敵人不只是不死化的冒險者，還混入了『吹笛之惡魔』的魔術師。
。
頂尖冒險者公會『白銀的意志』已經全滅，這個情報已經被蒙德伯爵得知。
優諾斯聽聞，認為『白銀的意志』應該是不會輸給區區不死化的冒險者。
那麼，殲滅『白銀的意志』的人肯定是『吹笛之惡魔』的魔術師了。
雖然他的部下現在還有打飛不死者的餘裕，但倘若『吹笛之惡魔』直接攻進來的話，這裡也保不住了。
。

「……蒙德伯爵大人，放棄都市巴萊拉吧。直到其他都市為止，由我們『舞動之劍』來護衛」

。
對於優諾斯來說這是最好的辦法了。
不僅能夠從淪為危險地帶的都市巴萊拉中脫出，同時也能向蒙德伯爵賣人情。
。

「什！」

。
蒙德伯爵對優諾斯的話語感到驚訝。
。

「優諾斯殿下，你是說讓老朽放棄領地逃之夭夭嗎！？這麼做的話，這個領地就真的完了。要是領主也逃走的話，私兵和冒險者們都會紛紛放棄這個地方各走各路。這不是正如對方所願了嗎！」

。
領主放棄領地逃之夭夭什麼的，就像是在宣言接下來這個領地就要毀滅了一樣。
為了領地而捨身戰鬥的冒險者們也會丟掉手中的劍逃走的吧。
而且蒙德伯爵才剛剛向各冒險者公會派出號召迎戰的使者。
在這個時刻，只有自己臨陣逃脫什麼的，實在是言而無信的卑鄙的行為。
。

「捨棄尊嚴吧，蒙德伯爵大人。即使您在或不在，對領地的危機都沒有多大影響。被發覺的話，士氣的確是會下降的呢。結果也許會引起混亂，招惹到多餘的人呢。因此，只向一部分信賴的家臣傳話，說『蒙德伯爵還在伯爵府邸中』。然後為了安全起見……」

。

「……欺騙領民，只有老朽一人遠走高飛？沒想到，優諾斯殿下竟是如此不知廉恥之人！」

。
優諾斯陷入沉默。
。
優諾斯擅長讀出並誘導對方的思考。
但這只是優諾斯在出身地的治安惡劣的貧民窟成長時掌握的技術。
。
在優諾斯成長的街道，想要生存下去的話就必須犧牲周圍的人。
做不到這件事的人，早已經死了。
由於在這種環境下成長的緣故，優諾斯擁有著人在本質上就應為自我中心的合理主義者的思考方式。
在打算誘導對方思考的時候，這個傾向便暴露無遺。
因此優諾斯也一曾否定了蘭貝爾的主張。
。

「……十分抱歉。老朽似乎也稍微缺乏冷靜了。對優諾斯殿下發火了。不同的人，有不同的想法。老朽或許也覺得逃走是個好辦法。但是，老朽現在不想從這個都市中逃出去」

。

「……不，哪裡的話。說出過分的話是在下的錯。請原諒我的無禮」

。
優諾斯低下頭，接著便自覺地離開了辦公室。
在最後從門縫中看了一眼，看向疲憊不堪的蒙德伯爵。
。
走出辦公室的優諾斯，悄悄地嘖了嘖舌。
。
(蒙德伯爵還不能死。只有我一個人也好，也應該早早地離開這裡嗎？）

。
優諾斯這麼思考著走下樓梯時，從樓上傳來了爆炸的聲音。
雖然那並不是蒙德伯爵的辦公室，但似乎是某處受到了魔術的攻擊。
傭人們的悲鳴接連不斷地傳來。
優諾斯預想中的『吹笛之惡魔』的魔術師的襲擊，似乎提早一步開始了。
