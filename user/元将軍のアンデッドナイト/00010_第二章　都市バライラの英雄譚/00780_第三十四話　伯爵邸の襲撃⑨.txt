「這……這不可能，這絕對不可能！杜拉罕怎麼可能會在劍的較量上戰敗！絕對不可能！作為完美的精靈劍士的杜拉罕的劍速，被沉重的肉體束縛著的人類怎麼可能承受得住！」

。
曼奇一邊後退一邊說著。
後背觸碰到牆壁時，他不由得停下了腳步。
因恐懼而瑟瑟發抖的膝蓋跪了在地上。
。
接著，他抬頭一看。
曼奇的瞳孔中，映入了蘭貝爾的身影。
。

「很不巧，我早已不是人類了。不過，除了不會感到疲倦以外也並沒有太多的變化」

。
背對著火焰的大鎧的男人此刻即為惡鬼。
。

「回、回應我的呼喚，從冥府中顯現吧，虛空之塔——骸骨巴別塔……」

。
與曼奇吶喊的同時，蓋洛夫之書再次放射出瑪娜的光輝。
白色的三角錐在蘭貝爾的腳下出現。
蘭貝爾往後一跳，拉遠了距離。
。
接著，白色的三角錐繼續往上升，延伸出直徑三米左右的白色的圓柱並貫穿了天花板。
辦公室激烈地晃動。被貫穿的地板陷入龜裂，從被粉碎天花板中落下瓦礫。
。
並且將曼奇所在的另一側的牆壁完全地破壞了。
曼奇的人影也被白色的圓柱推開，從伯爵府邸中被彈飛。
。

「又被他逃掉了嗎……不過，現在也顧不上那麼多了。首先要解決掉眼前的問題呢」

。
圓柱中積聚起大量的骸骨並逐漸成形。
數個頭骨等間隔地並排著，用虛空無神的眼窩看向外邊。
。

「出現了嗎。蓋洛夫的十三精靈的第一精靈，骸骨巴別塔……」

。
虛空之塔·骸骨巴別塔和杜拉罕一樣，也是蓋洛夫契約下的冥府精靈之一。

蓋洛夫于八國統一戰爭時，活用了召喚骸骨巴別塔時的奇襲性質，反復地進行召喚並用作武器。
在蓋洛夫的十三精靈中排行第一，同時也是最無力的。
然而，這並不意味著其毫無用處。
蓋洛夫反而最喜歡使用既能成為劍也能成為盾的，使用方便的骸骨巴別塔。
。
蘭貝爾稍有警覺地架起大劍。
。

「……果然，對手是蓋洛夫的詛咒的話，是不會那麼簡單的呢。因為放出了炎柱，還以為骸骨巴別塔是否因為某種原因而不能使用了呢……。在範圍有限的屋內，和骸骨巴別塔戰鬥果然還是會有點辛苦呢」

。
骸骨巴別塔可以均分質量並化成複數體，複數體也可以被同時地召喚出來。
擅長心理戰的蓋洛夫，以此作為佯攻，並附加上召喚的時間差，從而進行高速的操縱。
蓋洛夫操縱骸骨巴別塔，將八國統一戰爭的有名的英雄玩弄在鼓掌之間。
。
就連蘭貝爾的直屬部下也葬身於蓋洛夫那操縱自如的骸骨巴別塔之下。
蘭貝爾自然而然地往握著大劍的手中注入了力量。
。
然而，畢竟蓋洛夫是在八國統一戰爭終盤的殘酷的時代中寥寥無几的上位魔術師，這是用他那異常的瑪娜才能實行的戰術。
即使曼奇想要模仿這個戰術進行連續的召喚，也只會白白浪費自身的瑪娜。
。
對提高緊張感的蘭貝爾視若無睹，骸骨巴別塔的身影變得淡薄，在打破牆壁後便消失了。
。

「……嗯？」

。
映入蘭貝爾的眼中只有緩慢的退場。
這明顯不是被魔術所操縱而回到異界，而是因為術者提供的瑪娜在中途切斷而導致的消滅，蘭貝爾是這麼猜測的。
。
而事實也正是如此。
曼奇將蓋洛夫主要用來攻擊的骸骨巴別塔，僅用於處在劣勢時的撤退的掩護。
。
蘭貝爾從牆壁的大洞中探出身體。
從蒙德伯爵府邸的三樓鳥瞰，都市巴萊拉鱗次櫛比的建築物看上去就像是縮小了一樣。
其中，蘭貝爾發現了誇著奇怪的馬，沾滿血跡的曼奇的身影。馬的頭部和通常的不一樣，並且在胸上還依附著人面。
馬頭本身也有三只眼睛，只是看著就覺得不舒服。
。
這是蓋洛夫的十三精靈的第三精靈，雙頭的靈馬特洛伊。
。

「不會讓你逃掉的！」

。
蘭貝爾打算從三樓跳下去，但聽到『舞動之劍』的冒險者的悲鳴後轉身一看。
在他身後，『舞動之劍』的冒險者和羅賓漢仍然在和強化不死者戰鬥。
化為戰場的辦公室，因骸骨巴別塔的緣故地板和天花板被大面積地損壞，而且曼奇放出的火焰也開始擴散，蔓延到整個洋館。
。
以不知疲倦、無視肉體損傷繼續戰鬥的強化不死者為對手，即使是一流的冒險者們也陷入了苦戰。
拖得越久，越對不死者有利。
。
(這樣下去的話支撐不了多久……但也不能白白地讓蓋洛夫之書逃掉。恐怕，曼奇在我出現之前都沒有使出真本事。此時爭取到了時間的他，必然是打算召喚出蓋洛夫之書中的上位精靈。若是召喚出第十三只精靈的話，失去了舊時的直屬部下的現在的我，並沒有能夠對抗的手段……）

。
回想起過去在戰場上的種種，即使是身經百戰的蘭貝爾也戰慄了。
蓋洛夫的第十三只精靈，就連作為契約主的蓋洛夫也無法控制，是宛如顯現出地獄般的怪物。
蘭貝爾即使和部下齊心協力共同作戰，也無法正面擊敗那只怪物。
。
不可以給予曼奇過多的時間。
但是，如果追上去的話，留在伯爵府邸裡的人都會被殺死。
。
其中一名冒險者向猶豫不決的蘭貝爾大喊。
。

「快去追上那個怪胎！這裡就交給我吧！」

。
那名身穿綠色的華麗貴族服裝的修長男子是羅賓漢。
從對手處移開了視線的羅賓漢，受到了化為不死者的優諾斯的襲擊。
。

「哈哈哈哈哈！羅賓漢，你也，去死吧！你也想在地獄，和以前的部下相見對吧？哈哈哈哈！」

。
浮在空中的優諾斯的臉頰、胸膛和大腿被箭矢刺穿。
優諾斯被箭矢對消了攻勢，伸長雙腳降落到地板上。
優諾斯伸長手臂延長了攻擊的範圍，用長劍的前端瞄准了羅賓漢。

還擊的劍掠過了羅賓漢的額頭。
完整的臉龐上被划過一條薄薄的血線。
。

「啊哈、啊哈哈哈！哈哈……」

。
一支箭矢，深深地扎在狂笑中的優諾斯的額頭上。
從零距離放出的箭矢的沖擊將優諾斯的脖子折斷，誇張地向後彎折。
優諾斯的身體翻到了。
。

「這些傢伙就由我來幹掉吧」

羅賓漢氣喘吁吁地向蘭貝爾說道。

蘭貝爾的巨軀和魔金奧爾鋼鎧的重量，從蒙德伯爵府邸的三樓的高度落下並加速。
