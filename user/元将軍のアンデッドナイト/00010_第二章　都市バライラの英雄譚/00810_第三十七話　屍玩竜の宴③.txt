「這傢伙果然很討厭呢」

。
蘭貝爾仰視抬起上半身的屍玩の竜ラウンプゥプ，喃喃自語。
令人反感的巨頭也俯視著他，不知所措地張著嘴。
合不上焦點的瞳孔咕嚕咕嚕地打轉。
。

「喲呵、喲吼吼吼！一切都結束了！老朽贏了！雖然不知道，身為不死者的你為何要死守這個地方，但ラウンプゥプ出現的現在，這個都市就注定要毀滅！真是遺憾呢！喲吼吼吼！」

。
隔在ラウンプゥプ的另一側，跨著特洛伊的曼奇嘲笑道。
從一開始，在都市中的虐殺和不死者的大量生產，都只不過是作為ラウンプゥプ的貢品，一口氣地破壞都市的準備而已。
ラウンプゥプ的規模，說實話，已經超越了用少人數就能對付的範疇。
。
雖然將召喚主的曼奇殺死的話ラウンプゥプ也應該會消失，但曼奇擁有名為妖馬特洛伊的俊足。
再加上黑之召喚師莉莉絲的存在。
莉莉絲已經開始聚集起在場的屍體。
越過無數的不死者和ラウンプゥプ，要擊殺騎著特洛伊的曼奇的確是很困難。
不先將作為攻擊和防守的核心、ラウンプゥプ攻破，就不可能到達曼奇的所在之處。
。

「啊啊啊啊啊啊啊啊啊！」

。
ラウンプゥプ發出奇怪的聲音，用那惡心的頭部迫近蘭貝爾。
在夢魘疾馳並回避的正後方的地面，被輕易地挖去了一塊。
蘭貝爾回首一看，只看見從口中流出泥沙，咯吱咯吱地彎起嘴角大笑的嬰兒的笑容。
。

「真不想和你這傢伙再次相遇！」

。
ラウンプゥプ追趕在背負著蘭貝爾而疾走的夢魘的後方。
用無數的關節比人類多一節的巨大手腕撲向它們。
在前方，被大量的不死者堵住了道路。
是莉莉絲聚集起來的不死者。
。
蘭貝爾的視野中，映現出一名眼神憂郁、黑眼圈濃厚的過於瘦弱的壯年男子的身影。
是用一人之力就操縱國家，並造成一千萬死者的鍊金術師蓋洛夫的虛影。
雖然他早已不在人世，但被蓋洛夫的三只精靈圍困住的蘭貝爾，產生了當時和他對峙一樣的感覺。
。
蘭貝爾回想起來在殺死蓋洛夫之前，第一次看到他的臉的時候的事情。
雖為不問敵我、大量殺人的戰犯，然而卻有著沒有任何意志的空虛的目光。
所以蘭貝爾最初並不能理解，為何這個人是蓋洛夫。
然而，當這個男人露出惡魔一樣的表情，同時周圍也變成地獄的時候，蘭貝爾完全理解到，這個人並不是影武者而是貨真價實的蓋洛夫。
。
就在一瞬間，蓋洛夫的幻影消逝了。
取而代之，從遠處傳來曼奇的哄笑。
。

「立場逆轉了呢，不死騎士！接受蓋洛夫大人恩惠的老朽！繼承蓋洛夫大人的遺志和力量的老朽是無敵的！代替蓋洛夫大人，由老朽支配西沃米莉亞大陸，將現世逆轉成冥府！」

。
曼奇對自己的魔力和蓋洛夫之書擁有絕對的自信。
。
他雖為『吹笛之惡魔』的八賢者之一，但對組織完全沒有忠誠心。
歸根絕地，曼奇因為他那奇怪的容貌和出身，從一開始就不對他人抱有留戀。
作為例外，曼奇自以為是地產生共鳴，將同樣作為死操術師而崇拜的蓋洛夫，不用說只是當做死者對待而抱有執著而已。
曼奇堅信著只要擁有蓋洛夫之書，自己就是八賢者中的頂點，就連自己遲早會成為組織的頭目的事情也考慮過。
。

「你，居然說自己繼承了蓋洛夫的意志？」

。
蘭貝爾舉起大劍。
夢魘向不死者之群突擊。
蘭貝爾一揮的同時，十只以上的不死者就失去了上半身，下半身也被夢魘踢飛。
不斷接近的ラウンプゥプ碾碎屍體，緊追不舍。
。

「別太高估自己了。只是被書的詛咒操縱，犯下殺人罪名的鼠輩，居然敢冒充蓋洛夫的後繼者」

。

「什！」

。
曼奇氣得滿臉通紅，張口結舌，在打算說出什麼之前就閉嘴了。
因為蘭貝爾放出的濃密的瘴氣壓制了曼奇。
感覺到從眼前的正體不明的不死騎士中放出的驚人威壓，曼奇的怒火被其熄滅。
嘴巴愣是張不開，依舊是合攏的狀態。身體不由自主地開始顫抖。
。

「並不是你在使用魔術和精靈。而是，你被魔術和精靈操縱著。只是誤以為自己得到龐大力量的凡人罷了」

。

「咕……嗚……」

。
曼奇雖然想要反駁，但一下子又說不出話來。
在身旁笑著的童女、莉莉絲的笑聲，在曼奇聽來就像是在嘲笑他自己一樣。
一頓沉默後，曼奇的硬直終於解開了。
剛才感覺到的恐懼只是心理作用，曼奇對自己這麼解釋道，為了讓自己放下心來，他對名為ラウンプゥプ下達命令。
。

「快、快點碾碎它們，ラウンプゥプ！」

。
ラウンプゥプ再次將頭部高高的抬起。
咕嚕咕嚕地轉動的雙眼，鎖定了蘭貝爾。
巨頭在一陣顫動後，張開大口，以蘭貝爾為目標直沖。
。
高速的頭部捕捉到蘭貝爾的位置。
身纏超重魔金鎧的蘭貝爾，被輕鬆地撞飛。
無頭馬夢魘也被拋到了空中，從腹部的一側處摔在地上。
。

「喲吼吼吼吼吼！不愧是ラウンプゥプ！快，趕緊解決掉它……」

。

「啊啊啊啊啊啊啊啊啊！」

。
ラウンプゥプ大大地張開嘴巴，岔開上半身並发出慘叫。
這惹人厭煩的叫聲，也被曼奇的命令抵消並讓其沉默了。
然後，曼奇注意到了。
橫跨ラウンプゥプ的左眼，有一處被無骨的刀刃挖去的，大大的傷口。
曼奇睜大了雙眼。
。

「ラウンプゥプ！？不、不可能！這可是最上位的精靈啊！？為什麼，為什麼會被一個人類造成這種傷害！？」

。
倒在地上的蘭貝爾站了起來。
。

「果然，在沒有部下的情況下，用普通的方法對付不了ラウンプゥプ呢。而且……夢魘也暫時跑不動了嗎。拉開距離等待時機也做不到了呢」

。
即使受到能夠掘起大地的ラウンプゥプ的一擊，魔金之鎧也沒有一點傷痕。
。

「你就在那裡等著吧。等我解決掉這傢伙，下一個就是你了」

。
曼奇又一次反駁不了蘭貝爾的发言。
蘭貝爾的話語並不是虛張聲勢，這在剛才和ラウンプゥプ的正面激突中經已得到證明。
。
雖說傷口不淺，只是被砍了一刀。ラウンプゥプ並不會那麼容易倒下。
優勢仍然在ラウンプゥプ這邊，這是毋庸置疑的。
然而曼奇認為，ラウンプゥプ這種程度的精靈應該是用於戰爭的兵器，並不是靠個人就能夠
對付的東西。
而將ラウンプゥプ視為絕對的這個想法，就被剛才的一刀輕易地粉碎了。
