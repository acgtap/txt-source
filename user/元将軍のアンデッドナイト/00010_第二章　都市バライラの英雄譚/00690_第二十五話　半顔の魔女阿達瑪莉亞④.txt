對於輕鬆掙脫拘束，並進入戰鬥姿態的蘭貝爾，阿達瑪莉亞用看怪物般的眼神看著他。
。
阿達瑪莉亞也是心里有數的。
只用一人之力擊潰他們主要戰力的男人，必然是怪物般的存在。
但是，阿達瑪莉亞認為只是這樣是不足以估計蘭貝爾的強大的。
站在面前的男人，是怪物之上的某種存在。
。

「已經撐不住了！至少也要守護住阿達瑪莉亞大人啊！」

。

「我們來爭取時間！阿達瑪莉亞大人，把這件事傳達給芒西大人……」

。
阿達瑪莉亞的兩名部下舉起杖，阻擋在蘭貝爾的面前。
。
阿達瑪莉亞便開始往蘭貝爾的反方向逃跑。
她已經徹底理解了
這次襲擊的成敗，完全取決於能否殺死眼前的這名身穿大鎧的劍士。
只要那名劍士還活著，隨著時間的推移，這邊的戰力只會被他削弱而已。
必須盡快地讓芒西處置他。
。
蘭貝爾無言地橫起大劍。
。

「不要再接近他，這已經是我們的極限了」

。
站在兩人背後的男人，向他們發出忠告。
被詢問的一方，一邊盯著蘭貝爾一邊首肯。
不知何時，斜舉起大劍的蘭貝爾迫近到他的眼前。
嘴上說著要勇敢地犧牲自我的的兩人，從喉嚨的深處揚起了悲鳴。
。
在蘭貝爾揮下的大劍面前，兩人的上半身同時飛了出去。
被打飛的兩人的上半身落在地上，從斷面處流淌出大量的鮮血。
阿達瑪莉亞的最後一名部下，用微微顫抖的杖尖指向蘭貝爾，一步一步地往後退。
。

「啊……啊……」

。

「怎麼？不攻擊我嗎？」

。
男人和蘭貝爾之間還有一段距離。
完全在魔術師的射擊範圍之內。
。
最好的時機，應該是在蘭貝爾斬殺前面兩人的那一瞬間。
男人卻輕易地放跑了這個機會。
不過事到如今，處於對魔術師有利的這個距離的話，應該還有一絲勝機。
。
男人用意識讓心臟的鼓動平靜下來。
屏住呼吸，讓混亂的精神稍微冷靜下來後，再次把杖尖指向蘭貝爾。
。

「嗚……嗚……嗚哇啊啊啊！」

。
男人把杖丟棄，往與阿達瑪莉亞不同的方向逃跑了。
。
(鎧劍士肯定會去追作為頭目的阿達瑪莉亞大人的……！）

。
那麼自己就能脫身了，這的確是卑鄙的想法。
但這麼做也肯定不會被責怪的。和阿達瑪莉亞大人一起逃走的話，兩人必定會被他一同斬殺。
反抗也是毫無意義的，剛才的兩人已經證明了這一點。
。

「真是沒用的傢伙。你的選擇只有死路一條」

。
粉碎那名男人的天真的想法的蘭貝爾，是某種意義上的傲慢。
。
通過聲音，察覺到鎧劍士不知為何地驅往自己逃跑的方向的男人發出了悲鳴。
蘭貝爾用單手輕鬆地操使大劍。
蘭貝爾伸長手臂，巨大的凶刃從男人的背部貫穿心臟，接著切斷骨肉和右肩，從右肩上拔出來。
踢開斷氣的男人，蘭貝爾迅速地轉變方向，接著追趕最後一人的阿達瑪莉亞。
。
阿達瑪莉亞在空曠的平地上摔了一跤。
還以為發生了什麼事，原來是自己的膝蓋已經在抽搐了。
由於恐懼，雙腳已經不聽使喚。
。

「哼、哼哼……居然能讓本人、感到恐懼……哼哼……。這樣子，不就像個小姑娘一樣了嗎」

。
阿達瑪莉亞小聲地自嘲。
死的覺悟已經做好了。也沒有對跌倒一事抱有遺憾。因為要從那裡逃出來，無論腳程有多快都是不可能的事情。
。
蘭貝爾舉起大劍，向阿達瑪莉亞接近。
阿達瑪莉亞讓兩膝貼著地面，接著把杖尖對准了蘭貝爾。
在下一瞬間，阿達瑪莉亞的視野上下顛倒了。
劇痛在手臂上游走，杖也飛走了。手臂被劍鞘掃蕩了，她遲一步反應過來。
之後，從手臂中傳出清脆的聲音。蘭貝爾的膝蓋壓在了阿達瑪莉亞的手臂的關節上。
手骨明顯地被碾碎了。
。
蘭貝爾的大劍架在阿達瑪莉亞的鼻子前面。
。

「芒西這個傢伙，就是你們的頭目吧。把這件事傳達給芒西大人……嗎？看來你們所說的虫子，並不能隨心所欲地操縱呢」

。

「……殺了我吧。不好意思，我可不是會吐露出情報的那種人哦」

。
阿達瑪莉亞這麼說著，展示自己右半邊臉那被剝掉皮肉的異形的容貌。
。

「這是在王國異端審問會被抓住的時候弄的哦。但是，我堅持著毫不知情的樣子……最終成功地逃出生天了！」

。
王國異端審問會，是以根絕那些以研究和使用禁忌魔術為目的的魔術組織或魔術師為目的而設立的禁忌魔術組織。
以惡制惡，貫徹這個矛盾的方針的組織，是由在王國內無人能夠企及的高位魔術師們集結而成的。
雖然審問官的情報被隱匿著，但據說有死刑犯整容成為了其中一員，經歷過因國王的指示而研究不老不死的時代，用冤罪燒毀整個領地之類的，是個有著各種黑色傳言的組織。
只是那個組織的活動多半不會出現在表面的社會，『吹笛之惡魔』也視其為敵。
。
一旦被抓住並判為冤罪，即使在之後證明了清白也絕不會被放生。
不過，阿達瑪莉亞在『吹笛之惡魔』中接受了充分的對拷問的訓練。
即使在反復拷問當中也一味地裝作被卷入的可憐人，最終讓異端審問官疏忽大意，從他們的魔爪中逃離了。
。
蘭貝爾，把頭盔摘了下來。
。

「誒……啊、啊……」

。
隱藏在莊嚴的頭盔內的災禍般的骷顱頭暴露無遺。
在空虛的眼窩深處，燃燒著象徵憎惡的赤紅的災禍的妖光。
那道光，從阿達瑪莉亞的上方俯視著她。
。

「從前我就不太擅長纖細的工作，拷問也不太拿手呢。這一方面，似乎是古力夫比較擅長呢」

。
阿達瑪莉亞完全不明白是怎麼一回事。
不死族竟保留著如此這般的意志，簡直是不敢相信。
。
只是，有一件事她還是明白的。
在暴露容貌之上，他並不打算使用說出情報就放你一條生路的天真的手段，而是打算用純粹的暴力和折磨讓自己吐露出情報才對。
