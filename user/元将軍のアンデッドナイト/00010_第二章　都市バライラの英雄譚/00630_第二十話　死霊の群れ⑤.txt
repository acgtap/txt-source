在都市巴萊拉的中央地區，變成了不死者的公會『白銀的意志』的冒險者們正在進行殺 戮。
曾經以白色為基調的他們的正裝，現在也被染上了泥土和鮮血的顏色，早已面目全非。
他們的膚色也變成了青白色，完全感覺不到體溫。
睜開的眼睛中沒有黑眼珠，完全地翻著白眼。
。
「嘰哈、嘰哈哈哈哈……哈哈哈哈！」
。
在屍山中喀噠喀噠地顫動下顎發出笑聲的人，是『白銀的意志』的女公會長米迦勒。
灰色而渾濁的眼瞳中，浮現出細細的血管。
脖子朝著奇怪的方向歪曲。握著劍的手，毫無生氣地垂落在地上。
已經沒有曾經被稱為劍舞的女神的樣子了。
。
但她的劍術卻絲毫不遜於生前，觀察她周圍的屍體就能明白了。
在屍體的胸 部上，都被刺出了赤紅的空洞。
。
「……沒想到，居然會發生這樣的慘劇」
。
格拉斯科拔出長劍後閉上雙眼，讓絮亂的呼吸鎮靜下來。
。
「跟我上！他們頂多也只不過是三十人！不用手下留情，打倒他們！看他們的樣子，已經沒救了！」
。
『白銀的意志』的不死冒險者數量是三十。
與此相對，在場的蒙德伯爵的私兵則有近百名。
即使『白銀的意志』的冒險者實力有多強，三倍的戰力差距也是很大的。
。
無差別地襲擊生者的『白銀的意志』的冒險者們，因格拉斯科率領的私兵團的登場，突然開始把他們認定為攻擊的對象。
交戰開始了。
。
站在格拉斯科面前的冒險者，是瘦長的雙劍使男人。
別稱二重刃的巴爾德特，在生前也是個馳名的劍士。
格拉斯科也認得他的臉。
。
巴爾德特的搖搖晃晃地走過來。
向右，向左。然後縮短和格拉斯科之間的距離，舉起雙刃向格拉斯科發起突襲。
格拉斯科確實地用劍防住了他的連擊，兩次攻擊都被他彈開了。
格拉斯科雖然在技巧上遠不及巴爾德特，但在體格上對付纖細的巴爾德特時的優勢很大。
。
巴爾德特流利地卸去反彈力，抑制住反作用力。
接著，在格拉斯科發起追擊之前，迅速地重整體勢。
。
「阿南達(Ananda），趁現在！」
。
「是！」
。
在格拉斯科斜後方準備著的阿南達跳出來，瞄準失去平衡的巴爾德特的側腹揮下劍。
巴爾德特雖然在一紙之隔扭轉手腕，用雙劍的其中之一承受了阿南達的攻擊，但因為腳的位置站的不太好，受到沖擊而導致身體傾斜。
。
「歐拉！」
。
此時格拉斯科放出了全力一擊。
巴爾德特為了防御而被砍斷左腕，接著就這麼承受了側腹上的沖擊。
手上傳來了骨頭碎裂的觸感。巴爾德特的左腕保持著握劍的姿勢落在地上，巴爾德特的身體則倒在不遠的地方。
。
「好、好嘞，用數量壓制他們……」
。
格拉斯科這麼說著的時候，忽然傳來了刺耳的笑聲。
。
「嘰哈，嘰哈哈哈哈」
。
是『白銀的意志』的公會長米迦勒。
在他們這群實力者之中，她的存在是最突出的。
米迦勒步履蹣跚地行走，但把阿南達鎖定為目標時，極端地挺直後背，筆直地走過來。
。
「喂阿南達，小心點！」
。
阿南達伸出手上的劍，打算牽制接近他的米迦勒。
牽制住，本應如此。但米迦勒華麗地穿過他的劍，用前端尖銳的單手劍筆直地刺穿阿南達的胸 部。
。
米迦勒的劍是以刺殺敵人為目的而特化的。
突刺是以點為範圍的攻擊，因此也是最難防御的，而是還是能夠最快使出的劍技。
米迦勒把突刺作為重點。
她那千錘百鍊的刺擊，就連看清她的初動都很困難。
。
「阿、阿南達……」
。
格拉斯科呆滯地說道。
這時，在米迦勒的背後，有個男人慢吞吞地站了起來。
。
是剛才本應受到了致命傷的雙劍的巴爾德特。
從受到格拉斯科一擊的腹部處不斷地流淌著鮮血，在那深處的內臟若隱若現。
但他卻毫不在乎地站了起來，仿彿沒有受到過傷害一樣。
因為變成了單手的狀態所以戰鬥力也應該是有所下降的，但拼命打倒的對手卻又爬了起來什麼的，就像是在做噩夢一樣。
。
狀況惡劣的不只是格拉斯科一人。
最初私兵團也是占據了優勢，但在無論受到多少傷害都不會倒下的不死士兵面前，私兵團的人數逐漸地在減少。
。
從格拉斯科的背後傳來了悲鳴。
他往後一瞥，保護著格拉斯科的正後方的部下，被其他的不死者刺穿了喉嚨，無力地倒在了地上。
明明在數量上是能夠取勝的，回過神時格拉斯科的三個方向已經被不死者包圍了。
。
「可惡，到此為止了嗎……真是做了和自己不相稱的事情啊。早知道就趕緊逃跑了」
。
格拉斯科深深地嘆了口氣，環顧四周。
格拉斯科到來時還沒有逃出去的領民們的身影，現在已經消失不見了。
。
「不過，為什麼呢。感覺還不賴，真是不可思議」
。
格拉斯科扛起大劍，向米迦勒突進。
米迦勒伸直了向前傾的後背，讓劍尖與格拉斯科重疊。
。
「屍體就給我乖乖地倒下吧！」
。
格拉斯科從間隔外掄起大劍。
對此，米迦勒則伸長了握劍的手。
。
若是同時地釋放劍技的話，先到達對方身體的劍技必然是突刺技。
但若是格拉斯科這種程度的劍士，米迦勒在觀察到劍開始揮動的那一瞬間，就能夠在劍完全揮下之前奪走對方的性命。
。
格拉斯科的身上，刺著米迦勒的劍尖。
格拉斯科的劍，卻還在米迦勒的正上方。
。
然而米迦勒這種以刺擊為核心的戰鬥方法，有一個很大的缺點。
即使搶先奪取了對方的性命，也不能抹消掉對方已經揮下的劍的余勢。
。
例如現在的情況，即使格拉斯科死亡，已經揮下的劍也會直擊米迦勒的頭部。
米迦勒換手持劍，斬擊並橫切了格拉斯科的腹部。
格拉斯科的劍落空，叩在了地上。接著就像是屈服於自己的劍的反沖一樣，格拉斯科雙膝著地。

「嗚……可、可惡……」
。
米迦勒沒有刺向心臟的部位，因此才避免了當場死亡。
但狀況毫無改變。不如說沒能和米迦勒同歸於盡，狀況明顯是惡化了。
。
單手的巴爾德特用劍砍向格拉斯科的後背。
格拉斯科用餘光看著巴爾德特的臉，做好覺悟並闔上了雙眼。
。
聽到了風被劍切裂的聲音。
聽到了骨肉被切斷的聲音。
格拉斯科以為那是自己的身體被砍的聲音，但他感覺不到疼痛。
睜開眼皮，發現自己還活著，於是便慢慢地回頭。
在巴爾德特的正中央出現紅色 的細線。
巴爾德特跪倒在地上，此時，左右兩邊身體往相對地方向倒下了。
。
在其身後，屹立著似曾相識的鎧甲男。
