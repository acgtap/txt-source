『放心吧小鬼，我們一定會替你報仇的』

。
身著鮮綠色耀眼衣服的青年，撫摸著一個哭泣著的孩子的腦袋，輕而易舉地接下了難題。
。

『你是認真的嗎公會長大人。失敗的話我們都會成為犯罪者的！最壞的結局，奧加之群會分散入侵都市裡的。即便是成功了，大概也會被蒙德伯爵盯上的吧』

。
衣著整潔的老劍士吃驚地說著，卻也站起來開始做好出發的準備。
。
老劍士其實是傳聞中都市巴萊拉歷代最強的冒險者公會，『蒼穹的大弓』的公會長副官。
他曾是侍奉於貴族的騎士。
雖然在隱居之後打算過上安穩的日子，但被青年的領袖氣質和強大的意志打動，作為一名冒險者而復歸本職。
現在多數是和『蒼穹的大弓』的主力冒險者一起勸告冒失的公會長，並擔任與貴族相關的委托人交涉的角色。
。

『沒問題。不管怎樣，我們已經是伯爵大人不可忽視的存在了。而且，正因為是其他人不敢接下的任務才有挑戰的價值。那才叫英雄對吧？如果討厭被卷入的話，離開這個公會也是可以的哦？』

。

『……您可真是的個無可救藥的人呢。的確，我也對那個判斷遲鈍的蒙德伯爵也不是沒有意見的。讓那些只是稍微发育過度的食人鬼和那個假裝安定主義的伯爵認清我們的實力吧。這次我也出陣吧』

。
得到公會長副官的許可，公會裡的冒險者們一同起立歡呼。
。

『不愧是羅賓漢大人！對孩子和有困難的人，還有像我這麼可愛的女孩子最溫柔了！日後肯定會成為美談的！』

。
公會長的青年，對飛抱過來的少女用最小限度的動作躲開了。
。

『別說那麼難聽的話啦，夏露娜』

。
名為夏露娜的少女是他其中一名部下。
在戰鬥中，用鈎爪進行近戰是她最為擅長的。
不僅是從死角處襲擊，連得意的飛撲都被公會長輕易地躲開雖然是有點可惜，但她也是一臉自豪的樣子。
這是因為對自己的頭目的崇拜。
。
——數日之後，『蒼穹的大弓』的冒險者全部被宣告判刑。
理由是，由於被功績迷惑，以自己的理由對領主通告的禁止狩獵的奧加之群出手並討伐失敗，還把奧加和奧加王招引到都市巴萊拉之中，造成了多數的死者。
◆

「真是辛苦你了，不僅把那些對我心有怨念的冒險者和淪為犯罪者而不能正常工作的冒險者聚集起來，還揭穿了妨礙我們的指揮者。沒想到，居然會這麼徹底地隱藏自己的身份呢。托你的福，我也殺了不少毫無關聯的人呢」

。
羅賓漢用充滿怨恨和狂氣的眼神看向優諾斯。
。

「為了達成目的而毫不吝嗇地殺 人、嗎。你或許和我很相似呢。英雄、羅賓漢」

。
優諾斯用挑撥的語氣說道。
。

「我不否認。我也、不覺得自己能死得光彩。你最好也做好覺悟吧」

。

「希望惡有惡報的人只不過是弱者。但事實上，相比起惡人，弱者只能選擇悲慘地死去，這就是世界的真理。祈禱惡有惡報的人，從價值觀的根本上，就應該認識到自己廉價的願望和毫無意義的南柯一夢，最後只能承認自己度過空虛的一生」

。
優諾斯歪著嘴角說道，橫著拿劍。
隔著劍刃的一面觀察羅賓漢的神情，優諾斯吭笑了。
。

「我出生在貧民窟。被我奪走麵包的孩子在幾天之後，就變成皮包骨的樣子，被野狗生吃掉了。但我還活的好好地。貧民窟外面雖然美麗，但在本質上一點都沒有改變！」

。
優諾斯縮小了和羅賓漢距離。
。
羅賓漢用手指夾著三支箭矢，以優諾斯為目標依次搭在弓弦上射出。
經由羅賓漢的神技，幾乎同時放出的三連射襲向優諾斯。
優諾斯避開了其中兩支，剩下的一支則用劍打落。
。
間不容发再次放出的四支箭矢，也沒能傷害到優諾斯一絲一毫。
箭矢在長劍的面前被斬斷落地。
終於，羅賓漢進入了優諾斯的劍的範圍內。
。

「『蒼穹的大弓』對我來說，也是那些被我奪走麵包的貧民窟的孩子。就連你也一樣」

。
優諾斯把劍尖向自身傾斜的長劍反握，向羅賓漢突刺。
。
羅賓漢別過身子躲開。
然後把手裡的弓松開，用空出來的手取出隱藏在衣服裡的小刀，行云流水地用手接住。
。
小刀的刀尖形狀奇特，刀身本身就向外彎曲，是有些與眾不同的小刀。
當然，小刀的長度遠不及優諾斯的長劍。
。
優諾斯確信了自己的勝利。
在輕鬆突破弓手的進攻範圍的時點上，優諾斯的勝利已經是板上釘釘的事情了。
何況一邊是熟練的劍士，另一邊則是自暴自棄地掏出小刀的弓手。
。
優諾斯站在自己的劍剛好能夠到的位置，活用細長的刀身发起單方面的進攻。
。
對此羅賓漢並不能用小刀進行防御。
只能一味地改變姿勢，躲開優諾斯的劍。
。

「那麼，這樣又如何」

。
優諾斯暫時地拉開距離並弓起後背。
。

「風啊，祝我一臂之力吧」

。
在優諾斯的劍尖顯現出魔法陣，優諾斯向地面一踢。
狂風成為了優諾斯的推進器。優諾斯在不規則的高速移動中伸出長劍，準確地瞄准了羅賓漢。
舞游劍的優諾斯的本領，是將移動和劍術用風魔術糅合在一起的、難以預測和回避的一擊。
。
在被風纏繞的飛行中，優諾斯確實地和羅賓漢對上了視線。
這不可能，優諾斯自言自語道。
。
對優諾斯來說，要完全掌握這一招需要好幾年的時間。
從高速移動的疾風的內側看清周圍的動作是必須的。說出來簡單但做起來很難。
然而，羅賓漢輕鬆地跟上了優諾斯那不規則的動作。這是他無論如何都接受不了的事情。
。
為了完全发揮出弓的優點，羅賓漢在小時候就徹底地鍛煉了動態視力。
優諾斯揮下的劍，在他看來也如同靜止一般。
即使是風魔術的高速移動也不是例外。
。

「嘖！」

。
優諾斯通過高速移動穿過羅賓漢的正旁邊，打算在兩人的身影交錯之時用劍一斬。
然而，卻變成優諾斯轉為防守的一側，扭轉手腕用劍防御羅賓漢放出的一擊的窘況。
。
在空中遭到預想不到的攻擊而轉為防守的優諾斯只好笨拙地著地了。
他慌張地往地面一蹬，拉開了和羅賓漢的距離。
。
(那種小刀居然能把我逼入困境……。看來他對弓以外的武器也很擅長呢。不過，就算他怎麼伸長手臂，拉開這麼大的距離也應該沒問題。羅賓漢已經把最擅長的弓扔到地上了吧。用不了弓……）

。
思考到這裡，優諾斯的腹部被箭矢貫穿了。
優諾斯踢向地面打算往後退的步調被打亂，就這麼摔倒在地。
。

「不、不可能……他不可能用弓射箭……」

。
優諾斯抬頭一看，羅賓漢手中依然拿著先前的小刀。
定睛一看，他察覺到奇怪的事情。
形狀奇特的小刀上，仔細一看的話似乎在內側有一條細絲。
那把小刀原來還能當弓用，優諾斯現在才发覺。
。

「太、大意了嗎。可惡……我、居然……」

。
優諾斯準備起身時，他的指甲被羅賓漢踩碎了。
優諾斯發出短短的悲鳴。
。

「優諾斯，你說善惡和死亡並沒有多少因果關系呢。雖然我不聽不懂你的哲學，但在這裡肯定是謬論」

。

「等、等一下羅賓漢！和我交涉吧！」

。

「要問為什麼的話，決定你生死的不是命運之神，而是我」

。
羅賓漢收起小刀，拿起落在地上的樸素的弓。
他拉起箭矢，把箭矢的尖端對准了優諾斯後背的，心臟的另一側。
。

「住手啊！我還、我還不能死！我擁有著巨大的才能！絕不只是這種都市的冒險者公會的公會長的程度……更大、更大的一番事業還在等著我來做！現在只不過是瓶頸罷了！我是不會在這種地方……！」

。
優諾斯在地上蠕動企圖逃跑。
羅賓漢放出箭矢。陽台上濺滿了鮮血。
