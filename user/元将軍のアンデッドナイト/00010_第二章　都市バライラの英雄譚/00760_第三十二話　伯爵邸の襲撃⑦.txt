曼奇将兰贝尔视为异物。
作为这个时代最高位的死操术师的曼奇，能感觉到兰贝尔那不为人类的气息的一鳞半爪。
进入了临战状态的兰贝尔的周围，被浓密的瘴气缠绕着。
这是促使一直以来都以轻松的心情进行单方面狩猎的曼奇进入警戒的充分的要素。
。
(这个威压，是瘴气吗？那样的話，是不死者……？不过，保有这种程度的自我意识的話，应该是最高位的巫妖吧。要是真有那种东西存在的話，『奇美拉之尾』应该会通知我才对……。但假如是人类的話，那就肯定不是一般人了。这种人物，居然在事前的调查中都没有发觉，真是可怕呢。难道说，他不是公会的冒险者吗？）

。
所谓的巫妖，是指死操术师为了得到永恒的生命，对自己施加诅咒从而获得不老不死的身体的魔物。
因为生命已经凋零，虽然在身体的机能、精神和玛娜上都容易出现错误，但比普通的不死者更容易保留自我意识。
把自身的玛娜固定在头骨中，得到史莱姆状的身体并存活了两百年的多米莉奥涅在广义上也算是巫妖。
但因为不存在只靠坚韧的意志和玛娜就能保留完整而稳定的自我意识的不死者的前例，所以并没有能够定义兰贝尔现在的状态的词汇。
。
曼奇注视着兰贝尔，把手用力一甩。
。

「不确定的要素要确实地消灭才行。还好老朽留了一手」

。
守在曼奇两边的不死者，举起手中的剑往兰贝尔突进。
。
(带来这里的，是老朽亲自使用盖洛夫之书这个最好的魔术媒介，直接嵌入魔术式的强化不死者……若是罗宾汉这种实力的話，花费一定的时间或许能打倒一只，但两只就对付不过来了！）

。
被灌入的玛娜摆脱了大脑对于保护自身的控制，用超越极限的速度奔跑着。
这两名不死者的身体能力明显凌驾于生前。
。
它们在兰贝尔的前方往地面一踢，左右分散。
一侧的不死者精准地狙击了兰贝尔的死角。
。
真是完美的动作。
曼奇嘴角放松，几乎发出了感叹声。
即使是对长期制造不死者的曼奇来说，速度这么快的不死者也不是那么常见的。
毕竟是使用冒险者之都的上位冒险者那日常锻炼而成的，千锤百炼的肉体的尸体所生成的东西。
。

「哟吼吼，哟吼吼吼！太棒了！老朽太感动了。感谢你们，冒险者们啊。太感谢了，为了成为老朽的尸体人偶而不断地锻炼身体，太感谢了……！这里，优秀尸体实在是太多了！」

。
慷慨陈词的曼奇，在丑陋的面容上露出恍惚的眼瞳，张开双臂大叫道。
罗宾汉一边牵制着优诺斯，一边也在视野的一角观察着兰贝尔和曼奇，他对那两名不死者的动作感到惊愕。
。
(原以为因为是优诺斯的尸体所以才这么强，原来不是！只要是最低限度锻炼过的尸体，那种水准的不死者当场就能做出来吗！虽然是用两面夹击来给予致命伤的战术，但用上这个练度的动作，再加上不死者的强度的話，已经不是人类能承受的程度了……！）

。
不死者不是只靠一刀两刀就能解决的。
要打倒它们的話，只能通过破坏身体的关节部位、用火烧，或者破坏脊髓和大脑的一部分。
即使被对方砍中也不会停止攻击，将回避和防御置之度外的不合常识的剑技，不可能防御得住。
而且现在是，从两个方向袭来的一流剑士的进攻。
。
以兰贝尔的颈部为目标，两把凶刃从不同的方向砍来。
。

「太美妙了！哟吼吼吼吼！」

。
与此同时，鲜血飞溅，不死者的尸体从不同的方向被砍飞了。
只留有胸部以上部分的不死者，各自撞在了墙壁上。
而留在兰贝尔附近的剩余的部分，怎么说也已经动不了吧。
。

「……哈？」

。
毕竟不是剑士，死操术师曼奇不仅看不清兰贝尔的神速之刃，连发生了什么事都不知道。
接近他的两名不死者的上半身，就这么消失了。
与其说是剑术，不如说用魔术来解释更容易让人接受。
。

「你是在试探我的剑吗。真是意味不明的家伙呢」

。

「你、你、你不是人类吗……不、不可能，这……老朽的，强化不死者……」

。

「这本书……是人皮做的？难道说，是盖洛夫那家伙的魔导书吗？」

。
兰贝尔把大剑上的血甩干净，向曼奇问道。
。

「只是一看，就知道是盖洛夫之书吗……？你、你果然是巫妖吧！」

。

「我，是巫妖？」

。

「老朽说错了吗？可是，并非死操术师的人，一眼就能判断出盖洛夫之书的实物什么的，怎么可能！」

。

「毕竟我亲眼目睹过实物了呢。在意想不到的地方，我终于捣毁了长年以来的心结。不过没想到，居然过了两百年还残留下来了呢」

。

「说什么、你在说什么，意味不明的事情……」

。

「处决掉盖洛夫的人，就是我。没能摧毁那家伙的魔导书，是我最后悔的事情」

。

「什、什么！」

。
只因一名炼金术师的暗中活动，阿茹古罗斯王国化为暴虐无道的野蛮国家，不断重复着无谋的战争和非人道的攻击。
而让其打上终止符的人，就是当时雷吉欧斯王国四魔将之一角的兰贝尔。
。
在二百年前，阿茹古罗斯宫殿的最上层。
被逼至绝境、敌我不分地让狂暴的精灵和死操术师堆积起大量尸体的盖洛夫，被兰贝尔正面击溃了。
然而，被世人畏惧的人类最恶的炼金术师盖洛夫并没有就这么结束。
盖洛夫在被兰贝尔斩杀的瞬间，把自己存在的证明的魔导书，转移到了处于远处的弟子的手上。
。
将自己的魔术，和与结下契约的精灵的联系留在人世，打算对世界造成极大的伤害。
作为造成将近一千万人死亡的史上最恶的战犯的老炼金术师最后的执念，兰贝尔结果还是没有赢过他。
至此，在之后的两百年中，盖洛夫的怨念不断变更持有者，堆积起了尸体之山。
。

「原来是这样，吾之失态，就在你手上吗」

。
因兰贝尔的愤怒和兴奋，愈发浓密的瘴气支配了四周。
。

「这不可能！你想说自己是，反逆的四魔将兰贝尔吗！不管怎么说，这都是不可能的！」

。

「诚然。我即是四魔将的末席，在历史的终幕刻上污名的，兰贝尔·德拉克罗瓦。那本被诅咒的魔导书，就让我来连同你一起斩断吧」

。
兰贝尔放出冷淡的話语，架起大剑。
曼奇睁开厚重而下垂的眼皮下的双眼，紧盯着兰贝尔。
