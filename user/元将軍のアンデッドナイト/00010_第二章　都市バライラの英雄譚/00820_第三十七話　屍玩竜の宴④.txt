ラウンプゥプ露出愤怒的表情俯视着兰贝尔。
由于大剑的一击，ラウンプゥプ其中一边的眼皮左右开裂，那一侧的眼球上也被划出一条惨不忍睹的痕迹，流出青黑色的犹如眼泪般的液体。
。
兰贝尔也举起大剑，紧盯ラウンプゥプ那骇人的脑袋。
。
趁着兰贝尔一动不动地与ラウンプゥプ对峙的时候，莉莉丝的不死军团包抄到兰贝尔的後方。
不死者向兰贝尔靠近，在挥剑的同时，ラウンプゥプ也行动了。
。

「啊啊啊啊——啊啊啊啊——！」

。
ラウンプゥプ将毛骨悚然的胴体的各处辗轧不断蠕动，打算用头部碾碎兰贝尔而向他迫近。
兰贝尔用最小的动作躲过那一击，对迫近而来的颜面放出一记剑击。
。
然而，ラウンプゥプ的脸上并没有新的伤痕。
ラウンプゥプ的体表硬得惊人。
对兰贝尔而言，若不是出尽全力的一击，恐怕都不能对它造成一点伤害。
并且ラウンプゥプ拥有的和巨躯不相符的灵巧，也没有给予兰贝尔放出强击的机会。
。
骇人的头部，用升上天一样的速度再次扬起。
在兰贝尔准备好迎接下一击之前，ラウンプゥプ的头部再次高速地砸向兰贝尔。
噩梦般的连续攻击。
这次也在咫尺之前被兰贝尔成功回避。
取而代之，被不幸卷入的不死者血肉横飞，身首分离。
过大的威力，在一瞬间将不死者切成碎片。
。
巨躯使出的二连击。
目睹了兰贝尔拼命应付的场景的曼奇，再次认识到ラウンプゥプ的强大力量，取回了刚才失去的余裕。
他那布满粘汗、痉挛不已的嘴角微微一翘，露出苦涩的笑容。
。
(竟然能够保持速度，并使出二连击吗！果然老朽的ラウンプゥプ是无敌的！虽然侧脸的确是被打伤，但那种东西只不过是侥幸罢了！只要ラウンプゥプ不断使出刚才的连击，那家伙就会逐渐失去平衡陷入苦战……能行，能干掉他！）

。
这时，曼奇预料之外的事情发生了。
迅速升起的ラウンプゥプ的脑袋，向兰贝尔放出了第三次的连击。
。

「哟呵……？」

。
大地被挖起，尘土飞扬。
因此曼奇并不知道，这次攻击有没有打中。
。
ラウンプゥプ的连击并没有停留在第三次。
第四次、第五次的猛攻的速度不降反升，对处于烟尘中的兰贝尔不断放出。
。

「哟吼吼吼……？」

。
越重的东西越快。只是这样就足以构成威胁。
重量加上速度的破坏冲击，连大地的形状都为之改变。
就连身在远处的曼奇都能感受到震动。
。
在视野恶劣中，拥有压倒性破坏力的ラウンプゥプ的五连击。
。
(即使是那个不死骑士，也不可能全身而退！）

。
第六次、第七次，接着是第八次的间不容发的猛攻。
纠缠着兰贝尔的不死者被卷进ラウンプゥプ的猛攻中，肉块从尘土中飞出。
。
在ラウンプゥプ可怕的八连击的面前，曼奇兴奋得往前接近。
。

「哟吼吼吼吼吼吼吼！？太棒了，做的真是太棒了ラウンプゥプ！刚才的动作是怎么办到的！这样的話那家伙就……！」

。
尘土消散。
在不死者的血肉堆积而成的山上，屹立着回避掉ラウンプゥプ连击的兰贝尔。
。
曼奇的笑容凝固了。
他脸上的肌肉痉挛，暴怒地大叫。
。

「为何……为何！这是为何！为何，目睹ラウンプゥプ的身姿後也不绝望！为何死咬着老朽不放！ラウンプゥプ可是最上位的精灵啊！」

。

「……真是危险，不过想躲开还是可以做到的。好不容易准备的马这么快就失去，果然还是很心疼。对这种拥有巨躯的对手，只能靠正面突破了」

。
与失去理智放声怒吼的曼奇相对，兰贝尔从容不迫地信口开河。
。

「你、你以为能赢吗！？这种说法，简直是……简直是……」

。
对兰贝尔的話语感到惊愕，曼奇在妖马特洛伊上失去平衡，在落马的前一瞬间勉强站稳。
他的呼吸因冲击和极度的兴奋而絮乱，由于过激的悸动而导致心脏的剧痛。
。
(这家伙，并不是那种投身于没有胜算的战斗中的人！就像是、就像是……拥有充分的胜算一样的说法！）

。
曼奇将ラウンプゥプ用作歼灭都市巴莱拉的手段。
然而，却被一名不死剑士所阻挠。
这不可能。ラウンプゥプ可不是只局限于这种程度的精灵。
而是能够毁灭一个都市的兵器。
。
(不管怎么说……也太不可能了吧！这可是杀死一千万人，和盖洛夫大人结下契约的高位精灵啊！为何连一名敌人都杀不死？）

。
在八连击结束之後，ラウンプゥプ终于产生了硬直。
兰贝尔没有错过这次机会。
他将大剑拔起，往地面一踢高高地跳起。大剑的剑锋，捕捉到了ラウンプゥプ的额头。
。
对ラウンプゥプ的额头造成巨大的刺伤，在额上插出了一条裂缝。
。

「啊啊、啊啊啊啊啊啊啊！」

。
ラウンプゥプ愤怒地放出头撞。
然而在那前方并没有兰贝尔的身影。
。

「这裡哦！」

。
兰贝尔在斩伤ラウンプゥプ额头的同时，移动到了ラウンプゥプ的头顶上。
大剑再次刺入ラウンプゥプ的头顶。
。
ラウンプゥプ发出奇怪的叫声并扭动身体，打算将兰贝尔抖落。
兰贝尔则在ラウンプゥプ那巨大的身躯上往返，反复地降落到地上并再次跳到ラウンプゥプ身上，为了不被ラウンプゥプ抓住而到处跑动。
。
从ラウンプゥプ身体裡生出的无数的手腕企图抓住兰贝尔而蠢蠢欲动。
然而在下一瞬间，这些手腕都被切断并打飞。
兰贝尔一边鲁莽地放出剑击一边到处乱窜，将ラウンプゥプ的身体逐步击破。
。

「啊啊啊啊啊、啊啊啊啊啊啊啊啊啊！」

。
ラウンプゥプ高高地跳起。
而兰贝尔也利用这个动作跳到更高的地方。
。

「『月羽』！」

。
在高空中抡起大剑的兰贝尔的身影，宛如一幅名画。
聚在一起的不死者，对处于高空的兰贝尔晃动高举的手臂。
。
兰贝尔将大剑一口气挥下，将剑尖挥向正下方。
身缠魔金铠的兰贝尔的全部重量，集中于剑尖之上。
积聚所有重力势能的一击，对准ラウンプゥプ身体中央的骨节眼释放。
以剑尖为中心产生的冲击波，如同排山倒海之势，将不死者之群吹飞。
。
ラウンプゥプ那蜈蚣状的身体也被一分为二。
从ラウンプゥプ处传来怪异的叫声。
失去了头部的後半的身体，并没有就此停止活动，而是在地上不断挣扎。
