ラウンプゥプ露出憤怒的表情俯視著蘭貝爾。
由於大劍的一擊，ラウンプゥプ其中一邊的眼皮左右開裂，那一側的眼球上也被劃出一條慘不忍睹的痕跡，流出青黑色的猶如眼淚般的液體。
。
蘭貝爾也舉起大劍，緊盯ラウンプゥプ那駭人的腦袋。
。
趁著蘭貝爾一動不動地與ラウンプゥプ對峙的時候，莉莉絲的不死軍團包抄到蘭貝爾的後方。
不死者向蘭貝爾靠近，在揮劍的同時，ラウンプゥプ也行動了。
。

「啊啊啊啊——啊啊啊啊——！」

。
ラウンプゥプ將毛骨悚然的胴體的各處輾軋不斷蠕動，打算用頭部碾碎蘭貝爾而向他迫近。
蘭貝爾用最小的動作躲過那一擊，對迫近而來的顏面放出一記劍擊。
。
然而，ラウンプゥプ的臉上並沒有新的傷痕。
ラウンプゥプ的體表硬得驚人。
對蘭貝爾而言，若不是出盡全力的一擊，恐怕都不能對它造成一點傷害。
並且ラウンプゥプ擁有的和巨軀不相符的靈巧，也沒有給予蘭貝爾放出強擊的機會。
。
駭人的頭部，用升上天一樣的速度再次揚起。
在蘭貝爾準備好迎接下一擊之前，ラウンプゥプ的頭部再次高速地砸向蘭貝爾。
噩夢般的連續攻擊。
這次也在咫尺之前被蘭貝爾成功回避。
取而代之，被不幸卷入的不死者血肉橫飛，身首分離。
過大的威力，在一瞬間將不死者切成碎片。
。
巨軀使出的二連擊。
目睹了蘭貝爾拼命應付的場景的曼奇，再次認識到ラウンプゥプ的強大力量，取回了剛才失去的餘裕。
他那布滿粘汗、痙攣不已的嘴角微微一翹，露出苦澀的笑容。
。
(竟然能夠保持速度，並使出二連擊嗎！果然老朽的ラウンプゥプ是無敵的！雖然側臉的確是被打傷，但那種東西只不過是僥倖罷了！只要ラウンプゥプ不斷使出剛才的連擊，那傢伙就會逐漸失去平衡陷入苦戰……能行，能幹掉他！）

。
這時，曼奇預料之外的事情發生了。
迅速升起的ラウンプゥプ的腦袋，向蘭貝爾放出了第三次的連擊。
。

「喲呵……？」

。
大地被挖起，塵土飛揚。
因此曼奇並不知道，這次攻擊有沒有打中。
。
ラウンプゥプ的連擊並沒有停留在第三次。
第四次、第五次的猛攻的速度不降反升，對處於煙塵中的蘭貝爾不斷放出。
。

「喲吼吼吼……？」

。
越重的東西越快。只是這樣就足以構成威脅。
重量加上速度的破壞沖擊，連大地的形狀都為之改變。
就連身在遠處的曼奇都能感受到震動。
。
在視野惡劣中，擁有壓倒性破壞力的ラウンプゥプ的五連擊。
。
(即使是那個不死騎士，也不可能全身而退！）

。
第六次、第七次，接著是第八次的間不容髮的猛攻。
糾纏著蘭貝爾的不死者被卷進ラウンプゥプ的猛攻中，肉塊從塵土中飛出。
。
在ラウンプゥプ可怕的八連擊的面前，曼奇興奮得往前接近。
。

「喲吼吼吼吼吼吼吼！？太棒了，做的真是太棒了ラウンプゥプ！剛才的動作是怎麼辦到的！這樣的話那傢伙就……！」

。
塵土消散。
在不死者的血肉堆積而成的山上，屹立著回避掉ラウンプゥプ連擊的蘭貝爾。
。
曼奇的笑容凝固了。
他臉上的肌肉痙攣，暴怒地大叫。
。

「為何……為何！這是為何！為何，目睹ラウンプゥプ的身姿後也不絕望！為何死咬著老朽不放！ラウンプゥプ可是最上位的精靈啊！」

。

「……真是危險，不過想躲開還是可以做到的。好不容易準備的馬這麼快就失去，果然還是很心疼。對這種擁有巨軀的對手，只能靠正面突破了」

。
與失去理智放聲怒吼的曼奇相對，蘭貝爾從容不迫地信口開河。
。

「你、你以為能贏嗎！？這種說法，簡直是……簡直是……」

。
對蘭貝爾的話語感到驚愕，曼奇在妖馬特洛伊上失去平衡，在落馬的前一瞬間勉強站穩。
他的呼吸因沖擊和極度的興奮而絮亂，由於過激的悸動而導致心臟的劇痛。
。
(這傢伙，並不是那種投身於沒有勝算的戰鬥中的人！就像是、就像是……擁有充分的勝算一樣的說法！）

。
曼奇將ラウンプゥプ用作殲滅都市巴萊拉的手段。
然而，卻被一名不死劍士所阻撓。
這不可能。ラウンプゥプ可不是只局限於這種程度的精靈。
而是能夠毀滅一個都市的兵器。
。
(不管怎麼說……也太不可能了吧！這可是殺死一千萬人，和蓋洛夫大人結下契約的高位精靈啊！為何連一名敵人都殺不死？）

。
在八連擊結束之後，ラウンプゥプ終於產生了硬直。
蘭貝爾沒有錯過這次機會。
他將大劍拔起，往地面一踢高高地跳起。大劍的劍鋒，捕捉到了ラウンプゥプ的額頭。
。
對ラウンプゥプ的額頭造成巨大的刺傷，在額上插出了一條裂縫。
。

「啊啊、啊啊啊啊啊啊啊！」

。
ラウンプゥプ憤怒地放出頭撞。
然而在那前方並沒有蘭貝爾的身影。
。

「這裡哦！」

。
蘭貝爾在斬傷ラウンプゥプ額頭的同時，移動到了ラウンプゥプ的頭頂上。
大劍再次刺入ラウンプゥプ的頭頂。
。
ラウンプゥプ發出奇怪的叫聲並扭動身體，打算將蘭貝爾抖落。
蘭貝爾則在ラウンプゥプ那巨大的身軀上往返，反復地降落到地上並再次跳到ラウンプゥプ身上，為了不被ラウンプゥプ抓住而到處跑動。
。
從ラウンプゥプ身體裡生出的無數的手腕企圖抓住蘭貝爾而蠢蠢欲動。
然而在下一瞬間，這些手腕都被切斷並打飛。
蘭貝爾一邊魯莽地放出劍擊一邊到處亂竄，將ラウンプゥプ的身體逐步擊破。
。

「啊啊啊啊啊、啊啊啊啊啊啊啊啊啊！」

。
ラウンプゥプ高高地跳起。
而蘭貝爾也利用這個動作跳到更高的地方。
。

「『月羽』！」

。
在高空中掄起大劍的蘭貝爾的身影，宛如一幅名畫。
聚在一起的不死者，對處於高空的蘭貝爾晃動高舉的手臂。
。
蘭貝爾將大劍一口氣揮下，將劍尖揮向正下方。
身纏魔金鎧的蘭貝爾的全部重量，集中於劍尖之上。
積聚所有重力勢能的一擊，對準ラウンプゥプ身體中央的骨節眼釋放。
以劍尖為中心產生的衝擊波，如同排山倒海之勢，將不死者之群吹飛。
。
ラウンプゥプ那蜈蚣狀的身體也被一分為二。
從ラウンプゥプ處傳來怪異的叫聲。
失去了頭部的後半的身體，並沒有就此停止活動，而是在地上不斷掙扎。
