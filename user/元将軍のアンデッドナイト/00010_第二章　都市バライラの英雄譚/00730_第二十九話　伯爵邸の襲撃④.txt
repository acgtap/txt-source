在蒙德伯爵府邸三樓辦公室，『舞動之劍』的五名冒險者正和不死兵交戰。
房間內一片混亂。
桌椅被翻倒，書物散落在地板上，屍體橫七豎八地倒在地上。
。
蒙德伯爵站在房間的最深處，被作為公會長輔佐的赤短髮大漢、范多所保護。
。

「嚕哦哦哦哦哦！」

。
范多左右揮斧。
接踵而來的三名化為不死者的冒險者，被分別重創了額頭、頭頂和側頭部，腦漿爆開而當場倒下。

『舞動之劍』是都市巴萊拉的首位冒險者公會的候補。
實力比那些冒險者要高好幾個段位。
而負責輔助優諾斯的范多，即使是以數個不死者為對手也勉強能做到。
。
然而，范多的動作在一瞬間遲鈍了。
他的目光，落在倒下的那三名不死者身上。
。
雙眼合不上焦點、無力地吐出舌頭的人是『赤鼠團』的貝利夏，面目全非、看不出原本容貌的人是『義勇之證』的福爾克，腦袋被漂亮地一分為二的人是福爾克的摯友馬爾西斯。
范多知道這三個人的長相和名字。
給人印象沉穩、身為常識人的范多，曾多次和其他公會進行交涉和情報交換。
因此范多交際廣泛，他也擅長記住別人的容貌。
但唯有現在，交際廣泛反而拖了范多的後退。
。

「有機可，乘！」

。
身上這裡那裡包著綳帶的高個子男人，把范多狠狠地踢上了天。
范多的巨軀在空中旋轉。他勉強地重整姿勢著地，但被踢中的部位已經開裂，流淌出大量的鮮血。
。

「失敬，鄙人的爪子太長了」

。
身長高得可怕的男人笑著說道。
不只是身長，他的手和腳一看就覺得很長，作為人類來說實在是太過怪異。
范多也判斷不出，自己的對手到底是人類，還是用屍體作成的不死者。
混在不死者的雜兵堆中，男人的動作實在是過於微妙。
。

「可惡的怪物……等優諾斯大人回來，你就死定了……」

。
范多氣喘吁吁地嘟囔道。
。

「才不是怪物……的確，鄙人的身體有七成是接縫上去的，但剩下的三成是原本的肉體！真是、太失禮了！這是對鄙人『接縫的布爾科尼』，最大的侮辱！」

。
自稱布爾科尼的男人，彎曲手臂抱著腦袋，一邊晃動身體一邊大喊大叫。
長袍脫落，露出被縫合過的傷痕累累的奇異的禿頭。
雙眼就像屍體那樣沉墜，和他那激動的動作形成鮮明的對比。
。
蒙德伯爵心情複雜地注視著他們的交戰。
優諾斯拋棄了都市巴萊拉，大概已經剛好逃出去了吧。
從情況來看，他對『舞動之劍』的部下什麼都沒說就顧著自己逃跑了，而讓他們知道這件事的時機也完全地錯過了。
況且在現在的情形，讓他們知道優諾斯的事情導致戰鬥的感覺變遲鈍的話，蒙德伯爵將必死無疑，在場的『舞動之劍』冒險者們也難逃一劫。
。

「曼奇大人也在趕往這邊。你們，全部人，都會經歷比死還可怕的事情啊」

。
布爾科尼手持射程廣闊的武器向范多毆打。
范多用斧刃承受，然後用力推回去。
斧頭傳來了切裂腐肉的感覺。
從布爾科尼的手臂中流出茶色的迷之液體，但還不至於被切斷。
。
范多的斧頭，比起銳利度更著重於用打擊力敲擊敵人，是和鈍器相近的武器。
然而即使用其直擊，布爾科尼的手臂也太堅硬了。
別說是切斷手臂，連打斷骨頭的感覺都沒有。
范多從手感中得出結論，布爾科尼的肉體是和橡皮相近的東西。
布爾科尼的全身，是以分散沖擊為特化的粗繩一樣的東西。
。
總而言之，對於不能切斷布爾科尼的肉身的范多來說，已經無計可施了。
貫穿腦殻，直接將腦袋敲碎的話大概是能殺死的吧，但范多不認為布爾科尼會露出破綻。
。
布爾科尼揮下長長的手臂。
雖然只是單純的動作，卻有著相當的射程。
手臂在揮擺時略微地伸長，掠過了本應處在範圍外的范多的下巴。
范多下巴的皮膚被輕易地剝落，要是被挖的再深一點的話，連骨頭都可能會被削掉，疼得估計站都站不起來。
。

「去死，吧！」

。
布爾科尼繼續向范多揮下雙臂。
即使他盡力用斧頭的側面錯開攻擊，但威力實在是太大了。
第一擊、第二擊、第三擊用同樣的姿勢來承受。然而與先前的攻擊的角度稍有偏離的第四擊，騙過范多的防守將斧頭打落了。
。

「優、優諾斯大人……我也，到此為止了……」

。
這時，布爾科尼突然翻了個跟斗，和范多拉開距離。
因為他聽到了箭矢飛快地射向自己的聲音。
。
翻轉一圈的布爾科尼，用自己長到令人髮指的指尖拔出箭矢。
同時也感到驚愕。
這支箭矢是瞄准了自己和范多交戰的間隙，預讀出相應的動作從而放出的攻擊。
這就是說，對手將布爾科尼迅速的動作完全地看穿了。
。

「……有，趣。居然能看清鄙人，布爾科尼的動作……！但是鄙人下一次，不會輸……」

。
在布爾科尼對射手產生興趣時，他的腹部傳來了劇痛。
箭矢其實有兩支。將一支箭隱藏在暗處而一同射出，刺穿了布爾科尼的腹部。
是常人不可能做到的速度和精度的連射。
箭矢穩穩地擊中了布爾科尼那怪異的身體的要害。
。

「殺、殺了你……」

。
因內臟損傷而痛苦地呻吟的布爾科尼，又被三支箭縱向地射穿了腦袋。
高個子的布爾科尼，像斷線的娃娃一樣跪倒在地上。
。

「你、你到底是……什麼人……」

。
視線的前方，羅賓漢悠然地站在辦公室的入口處。
對此而感到驚訝的不只是不死者，『舞動之劍』的冒險者和蒙德伯爵也感到十分驚訝。
。

「……你是來取老朽的首級的嗎。在被不死者殺死之前，用自己的雙手來解決，是這樣嗎」

。
蒙德伯爵咬牙切齒地盯著羅賓漢。
羅賓漢則沉默不語地回瞪蒙德伯爵。
。

「為什麼、為什麼你會在這裡！偏偏、偏偏選在這個時候！」

。
女冒險者塔爾米亞發出驚愕和厭惡的聲音。
塔爾米亞對小刀的運用突然變得遲鈍，被不死兵的劍彈走。
。

「糟……！」

。
這個瞬間，塔爾米亞面前的不死兵的頭頂被箭矢漂亮的穿透。
很明顯是來幫手的。
。

「為、為什麼……！」

。
那之後，接連射出的箭矢也正確地擊倒了幾乎毫無抵抗能力的不死兵。
在清理完不死兵後，羅賓漢踏進回到了平靜的辦公室。
。
在『舞動之劍』的冒險者們眼中，羅賓漢的行動的確是太矛盾了。
要問為什麼的話，羅賓漢曾經在『舞動之劍』執行任務時突然殺出，連他們的同伴也殺死了。
技術超一流，腦子裡面壞掉的英雄。是最惡的狂人。
但在現在，他卻對本應是明確的目標的領主和曾經襲擊過的對手伸出援手。
即使現在成為友方也不能讓人接受。
。
在羅賓漢看來，事到如今，也沒有殺死『舞動之劍』的冒險者的理由了。
羅賓漢為了讓優諾斯感受和自己同樣的絕望，即便是毫無關係的人，對於奪走作為優諾斯的部下的『舞動之劍』的冒險者的性命一事，他也無動於衷。
他就是這麼怨恨著從自己那裡奪走了一切的優諾斯。
。
將他的那些互相信賴的同伴們，一人不剩地侮辱、殺死、使他痛苦，哪怕只是將十分之一的痛苦返還給他也好。
然而，優諾斯完全沒有這樣的感情。
。
當他和優諾斯決一死戰時，明白到優諾斯只是將『舞動之劍』的部下們當作道具來使用後，這個時候就已經失去了對他們下手的理由了。
羅賓漢給予優諾斯的喪失感，只不過是那種失去了便利的道具程度的感覺而已。
。
雖說如此，隨意對他人的性命插手的羅賓漢，他也理解到自己已經淪落到了和優諾斯同樣的地位。
事到如今也沒有再次成為英雄的打算。
。

「聽說過這個傳聞吧？有一個鋤強扶弱，已經壞掉的沒落英雄」

。
羅賓漢嘴角一歪笑了笑，從懷裡掏出金幣用手指彈起。
。

「現在殺了你們也無補於事。要是沒有人存活下來的話，那麼誰來證明我是最強的事實。只限今天，讓我成為你們的同伴吧。很可靠對吧，伯爵大人？」

。

「……你這傢伙，竟敢在那個時候……！」

。

『舞動之劍』的其中一名冒險者，端著劍走向羅賓漢。
范多制止了他。
。

「……雖然他不能信任，也不可原諒。但在現在的狀況下和羅賓漢開戰的話，我們也只會全滅罷了。我明白你的心情，但還是先控制住吧。現在只能對那傢伙的心血來潮賭一把了」

。

「范多先生……可是……」

。

「現在也不見優諾斯大人出現。最壞的情況，你也是知道的。我知道你們各有各的想法……現在什麼都別說，聽我的吧」

。
聽到優諾斯的名字，羅賓漢的心中又激起了漣漪。
但他沒有表現在外，露出毫不知情的表情。
。

「喲吼吼吼……優諾斯，說的是哪個人呢？」

。
從門外傳來沙啞的聲音。
從門外出現的，是個皮膚蒼白、年齡奇高、身長腫瘤、過於醜陋的禿頭的奇怪老人。
。
在他背後，五名不死兵接踵而至。
其中有一名細眼的長髮的男人。
眼球被血染成真紅的顏色，背上被箭矢深深地刺穿的男人，是化成不死者的『舞動之劍』公會長優諾斯。
。
見此，『舞動之劍』的冒險者們和蒙德伯爵僵住了。
只有羅賓漢用冷淡的眼神看著變成不死者的優諾斯。
雖說是很倉促，但他對自己沒有將屍體徹底破壞而感到後悔。
。

「居然能打敗布爾科尼，看來這個叫作羅賓漢的也是個不錯的玩具呢。喲吼吼吼」

。
奇怪的老人、『吹笛之惡魔』的八賢者之一曼奇，看到伏在地上的部下的屍體後，喜不自禁地笑了。
