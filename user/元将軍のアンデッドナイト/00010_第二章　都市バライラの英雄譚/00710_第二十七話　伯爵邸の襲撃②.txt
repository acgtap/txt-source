從伯爵府邸中不斷傳出悲鳴和叫喊。
接著回響起刀刃交接的聲音和魔術攻擊的爆炸聲。
留在伯爵府邸裡的私兵和優諾斯部下的『舞動之劍』正在應戰『吹笛之惡魔』的襲擊，優諾斯推測到。
。
優諾斯無聊地發出鼻鳴，如同遠離發出聲音的方向一樣加快了步伐。
。

「開始了嗎。如果是我的話，逃出這裡簡直不能再容易了……」

。
這是一步登上冒險者公會頂點的絕好的機會。
但，如果領地自身已經衰退的話，那就毫無意義了。
捨棄『舞動之劍』的部下和放棄至今為止累計的功績，都是十分可惜的事情。
。
但那種程度的部下只要再征集一次就好了。
不如說『舞動之劍』的創立和運營使他獲得了運作冒險者公會的經驗，他也有著下次能做得更好的自信。
。
本來打算把『舞動之劍』的上位隊伍也一起帶走，但優諾斯不認為會有人聽從他那捨棄領地從別的地方重新開始的判斷。
優諾斯對待部下時，會徹底地隱藏自己冷酷而現實主義的那一面，假裝成與之正反面的人。
畢竟在用人時這麼做會更方便。
。
優諾斯掃興地在通道中奔跑時，忽然聽到了匆忙接近這邊的腳步聲。
雖然他想盡量地避開其他人，但這樣就不得不繞遠路了。在繞路避開這個人之後遇到其他人的機率也很高，也毫無意義。
優諾斯嘖了嘖舌，目光朝下看了看腰上的佩劍。
然後繼續向前走，再次開始奔跑。
。
從通道的另一邊出現的是蒙德伯爵的私兵。
。

「你是、『舞動之劍』的公會長優諾斯殿下嗎。可是，敵人的奇襲似乎是在上層……為什麼會在這裡？」

。
私兵停下腳步，用詫異的眼神看向優諾斯。
但優諾斯沒有停下腳步。
。

「啊啊，因為我也很害怕，感覺差不多是時候離開這個地方了呢。沒有打招呼就先行離開這一事，請你代我向伯爵大人道歉」

。
優諾斯面無表情地與呆若木雞的私兵擦身而過。
。

「你、你在說什麼！你以為是為了什麼，才把你召過來的……！」

。

「不巧，我接受的委托是處理掉那发狂的英雄大人呢。我可不會去跟反王國組織作對。那是你們的工作吧」

。

「你、你這個膽小鬼！這樣你也配叫做『舞游劍的優諾斯』嗎！」

。
私兵朝著優諾斯大吼。
優諾斯沒有作出回應。
只是心想，不是『舞動之劍』或『吹笛之惡魔』的魔術師真是太好了。
。
如果是伯爵的私兵的話，畢竟不是和『舞動之劍』的部下那麼親近，在這種緊急的時候是不會糾纏不清的吧。
要是遇到敵對的魔術師，無論優諾斯態度如何，交戰都是不可避免的。
。

「就因為敵人太可怕所以就逃跑了？看來，你不是那種驚世駭俗的厲害的傢伙呢！為什麼你到現在還擔任著『舞動之劍』的頭目，簡直就是不可思議！雜魚！別再讓我看見你！」

。
對於私兵的謾罵，優諾斯也只是當做耳邊風。
只是無動於衷地在通道中奔走。
對於那名私兵，優諾斯沒有任何興趣。
。
走了幾步，感覺到奇妙氣息的優諾斯停下腳步回頭觀望。
在私兵的前進方向，即是優諾斯過來的方向的前方，站著一名身纏黑色斗篷的男人。
。
長長的衣袖覆蓋著身體，完全看不見雙手。
但從衣袖的末端處露出了細長的雙劍的劍身。
。
在斗篷深處的男人的臉上沒有眼皮。只有宛如被刀具亂切般地，千瘡百孔的皮肉勉強地粘在眼睛上。
真紅而充血的雙目一直睜得大大的。
然後，也沒有鼻子。在異形般容貌的中心，只殘留著兩個空洞。
。

「你、你這傢伙是、黑鬼！原來是『吹笛之惡魔』的一員嗎！」

。
黑鬼是最近突然出現在都市巴萊拉的過路殺人魔。
據目擊者所說的異樣的容貌，稱之為恐懼的黑鬼。
因為发現了追尋黑鬼的一流冒險者被碎屍萬段的屍體，他也被冒險者們深切地戒備著。
。
黑鬼掄起雙劍，向私兵迫近。
。
私兵雖然擋住了三下攻擊，但黑鬼的雙劍實在是太快了。
不僅如此，因為黑鬼的長袍連手腕和劍柄一起隱藏了起來，劍的軌跡也變得難以預測。
私兵的劍剛被打落，黑鬼的另一把劍就迅速地刺向腹部。
。

「噶，哈……」

。
私兵膝蓋一軟，跪倒在地。
配合他的動作，黑鬼把劍抽出，然後第一次打開了話匣子。
。

「還活著嗎？放心吧，那裡是最長也最痛苦的，並且是必死無疑的位置」

。
私兵想要自行了斷，匍匐在地打算拾起落在地上的劍。
但黑頭巾的男人無情地踐踏私兵的手掌，並把他的劍踢到了遠處。
私兵陷入了絕望，伏倒在血泊之中。
黑巾男笑容扭曲，滿足地看著私兵的表情，接著又向優諾斯襲來。
。

「還以為『吹笛之惡魔』只有魔術師……原來如此，是咒劍士嗎」

。

「正是！」

。
咒劍士即為接受強化身體能力的魔術的劍士。
但因為強行增強身體能力的話會對身體產生極大的副作用，是活不了多久的。
。
魔術本身就十分復雜，再加上提高身體能力的魔術自身的風險。
並且對其他生體系禁忌魔術的轉用的危險度極高，能容許咒劍士的國家少之又少，研究當然也並沒有推進。
。

「吾肉體動作的精密度，是經芒西大人的魔術調整的！早已超越常人的極限！」

。
黑鬼得意地說著，向優諾斯舉起劍。
與之相對，優諾斯也把劍拔出。
細長的劍描繪出圓陣，確實地彈開了黑鬼的劍。
黑鬼反復不斷地左右舞劍，而優諾斯則是毫不費力地進行反擊。
優諾斯看穿黑鬼給劍注入力量的瞬間，在他還未发揮出完全的臂力的絕好的時機挫敗了他。
。

「哦？看來不只是條雜魚呢！有趣！」

。
優諾斯的劍尖發出光芒，在虛空中描繪出魔法陣。
。

「風啊，祝我一臂之力吧！」

。
在優諾斯的周圍卷起狂風，纏繞著他浮在空中。
。

「魔劍嗎，真是無聊的把戲……」

。
黑鬼和優諾斯拉開了距離。
被風包圍的優諾斯用不能預測的軌跡在空中飛舞，縱身略過黑鬼的旁邊。
兩者交錯時響起金屬碰撞的聲音。
。

「嘖……」

。
黑鬼那異形的容貌皺起眉頭，露出不快的表情。
兩劍相交但卻沒有完全防住，黑鬼的手腕被優諾斯深深地切裂了。
。
這是配合了魔術的無規則高速移動的難以躲避的劍技。
這就是他被稱作舞游劍的優諾斯的原因，也是『舞動之劍』這個公會名的來由。
。
但黑鬼的痛覺在成為咒劍士之時，就被芒西刻意地弱化了。
在命若懸絲時還能夠繼續戰鬥，就像是狂戰士一樣。
黑鬼用流血過激的手腕再次橫起劍，緊盯著優諾斯。
。
對此，優諾斯在空中翻轉的同時舉起劍。
。

「風啊 斬斷敵人吧」

。
優諾斯一邊降落，一邊對黑鬼降下劍。
從劍尖閃耀的光描繪出魔法陣，向黑鬼釋放出風之刃。
。

「什！」

。
原本，劍士能使用魔法這件事本身就很稀奇。在極近距離的戰鬥之中，集中精力並行使魔術是極為困難的事情。
鍛煉和實踐，並且在劍和魔術兩方面的造詣都是必要的。
。
黑鬼往後一跳，用劍彈開優諾斯的風刃。
。

「該死，淨會用些小把戲……」

。
黑鬼又一次握好劍，在捕捉到本應在正前方的優諾斯之際，優諾斯便於黑鬼的面前落下並做好攻擊的態勢。
慌張的黑鬼打算把劍用力一揮進行牽制並拉遠距離，但反應過於遲鈍。
。
優諾斯用劍橫切。
黑鬼的腹部被切裂，內臟從中灑落。
緊接著他毫不迷茫的換手持劍，從黑鬼的眼珠處貫穿，然後抽離。
黑鬼斷氣了。
。

「你、你這麼厲害，為什麼……要逃……？」

。
倒在血泊中的私兵嘟囔道。
。

「因為這場戰鬥對我沒有好處」

。
優諾斯冷淡地說著，接著轉過身來。
盯著二樓的陽台。
以優諾斯的風魔術，只是二樓的高度的話應該能無傷降落吧。
優諾斯這麼判斷道，便向陽台移動。
他從圍牆上再次俯視這個都市時，不由自主地浮現出吃驚的表情。
。

「真是不走運。我居然在這麼不中用的都市呆了這麼多年」

。

「對於你不走運這件事我表示同意」

。
朝聲音的方向一看。
優諾斯立即拔劍後退。
他用劍把襲來的箭矢全數擊落。
。

「姑且是想再確認一下都市的狀況才來這裡一趟的，但沒想到居然會先遇見你呢。原以為你是那種被逼到絕境之後，不把敵人殺光就絕不會善罷甘休的人呢……看來，你也沒有多少時間思考了呢？」

。
顯眼的鮮綠色貴族服飾，和橙色的卷髮。
正是戰神羅賓漢。
。

「……預見到蒙德伯爵大人的危機，於是就率先來報仇嗎。這只是遷怒于人而已，不過事到如今我也沒有制止你的意義。你走吧，趕緊的……」

。

「要抓住你的狐狸尾巴可是花了我一番功夫呢，舞游劍的優諾斯。已經理解了吧？我襲擊你的部下，並且現在對你拉開弓弦的理由。即便如此，你還想抵賴嗎？」

。
羅賓漢向優諾斯架著弓。
。
優諾斯的撲克臉崩潰，細長的雙目興奮地睜開。
然後露出挑撥般地淺淺的微笑。
。

「這樣的話，那才叫做遷怒于人吧。前往討伐被禁止討伐的魔物。在那時不應做好被他人妨礙的覺悟嗎？」
