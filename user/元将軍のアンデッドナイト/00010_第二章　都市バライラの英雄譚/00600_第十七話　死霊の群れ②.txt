在都市巴萊拉貧民窟的某個魔法道具雜貨店的地下，蘭貝爾在與十二名魔術師對峙著。
。

「真是遺憾呢。你發現了我們的裡公會『暗夜的時計塔』的老巢，然後打算趁我們不注意潛入進來……對吧？你錯了，是我，把你引誘到這裡來的。聽說，有個像狗一樣嗅探著我們行踪的傢伙呢」

。
在十二名魔術師中年紀最大的，四十歲左右的男人向蘭貝爾這麼說道。
雕刻在臉上的紋身，和渾濁的三白眼是他的特徵。
從嘴裡露出的舌尖上，連著金***墜。
他的名字是尼德瓦爾。是有著毒蛇的尼德瓦爾的別名的魔術師。
他正是是『暗夜的時計塔』的公會長，且被雷吉歐斯王國指名通緝著。
。

「有個女人向你哭訴了對吧？戀人被『暗夜的時計塔』殺死了，請幫我報仇什麼的。真是遺憾呢，那個啊，其實是我的女人喲。是個不錯的美人對吧？面白いくらいコロっと引っ掛かってくれたぜ(看不懂）」

。
尼德瓦爾說著，其他的魔術師們放聲大笑。
。

「這就叫瓮中之鱉啊鎧甲混球」
「不愧是尼德瓦爾大人！知道我們被跟踪之後，立馬就作出攻勢了」
「饒命給我們看看啊？有趣的話，尼德瓦爾大人有可能會放你一條生路的哦？」

。
過了許久，大失所望的蘭貝爾嘀咕道。
。

「姑且還是過來一看……果然，只是些小人物而已嗎。浪費時間了」

。
蘭貝爾泄氣地卸下肩膀，用手扶著頭盔陷入了低沉，以他為目標，組成圓陣包圍著蘭貝爾的魔術師們，各自拿起杖開始了詠唱。
。

「別殺了他哦。必須打聽出是誰命令他來找我們的！」

。
那番話宛如信號一般，炎之魔彈一齊向蘭貝爾放出。
下一瞬間，本應低頭的蘭貝爾的身影消失，出現在別的地方。
。

「啊……？」

。
對不規則放出的炎之魔彈，蘭貝爾宛如沒有穿上那笨重的鎧甲般輕盈地躲開了。
和他的神速相比的話，炎之魔彈就如同於靜止一樣。
消失、出現、消失、出現。明明還停留在視野之內，但尼德瓦爾他們的動態視力完全適應不了他的動作。
在組成圓陣的的一名魔術師眼前，蘭貝爾突然出現。
。

「啊……」

。
在魔術師察覺到被蘭貝爾接近之後，大劍已經砍下了魔術師的頭顱，被踢飛的肢體撞在了後方的牆壁上。
接著蘭貝爾橫跳，砍下了隔壁的魔術師的頭顱。
結束之後，又往下一個魔術師的所在之處飛去。
。

「別、別過來！別過來啊！」

。
其中一名魔術師，把杖對準蘭貝爾。
用顫抖的雙手握著杖，打算發射魔彈，然而還是敗給了從正面突擊而來的蘭貝爾的壓力，無意識地鬆手了。
結果，毫不抵抗地被蘭貝爾刎下了頭顱。
。
蘭貝爾繞著圓陣不斷跳躍，把包圍他的魔術師一個接一個地砍殺。
最後的一人便是尼德瓦爾。
。

「別、別太囂張了啊！出來吧毒蛇們！」

。
三條有著赤紅色外表的蛇從尼德瓦爾的杖中釋放，筆直地襲向蘭貝爾。
。
那是用瑪娜作成的迫真的蛇，直至瑪娜用盡為止，三條蛇作出如同有意識的真蛇一樣的動作。
因而在空中不規則地蠕動，即便是躲開也會追咬不放。
雖然只有三條蛇，但要完全躲開從近距離釋放的三條猛蛇是十分困難的。
並且，蛇牙上還隱藏著只是輕輕一碰就能奪走對手意識的猛毒。
若觸碰到鱗片的話，碰到鱗片的地方就會立即**，從而失去戰鬥能力。
。
是十分難纏的飛行道具。
同時也是成為尼德瓦爾的代名詞的魔法。
。
當然，對蘭貝爾來說毫無意義。
蘭貝爾用大劍畫出圓形，彈開了三條猛毒蛇的蛇頭。
倘若是放著不管，三條毒蛇也會被蘭貝爾的魔金鎧反彈消失，但蘭貝爾深愛著只授予給四魔將的這個鎧甲，不想被污穢的蛇毒玷污。
。

「等、等一下！WO、WO、我的背後，可是有『殺戮曲馬団』撐著的哦？要是殺了我的話，你可是會被追殺一輩子的哦！變成那樣的話，你這一生就只能在他們的陰霾之下苟且偷生了。永遠都無法安眠的……」

。

「不巧，我不需要睡眠」

。
況且蘭貝爾最近已經殺死了『殺戮曲馬団』的其中一名副團長。
要是被盯上了的話，那應該在那個時候早就被盯上了。
事到如今才屈服於這樣的威脅，簡直是毫無意義。
。
蘭貝爾放出的突刺，貫穿了尼德瓦爾的胸膛，把他釘在了牆上。
尼德瓦爾的身體停留在空中，手腳耷拉地垂下。蘭貝爾拔出大劍，尼德瓦爾便倒在地上。
。
蘭貝爾俯視著尼德瓦爾的屍體，搖了搖頭。
。

「……落空了呢」

。
蘭貝爾單獨地對『吹笛之悪魔』進行調查。
蘭貝爾認為，若『吹笛之悪魔』打算對近處的都市巴萊拉發動襲擊的話，那麼他們可能早已潛入都市巴萊拉之內。
不能先發制人嗎？蘭貝爾心想，於是他收集情報，尋找聚集著形跡可疑的魔術師的地方。
。
蘭貝爾把大劍收回劍鞘中，離開了『暗夜的時計塔』的據點。
接著繼續在貧民窟探索，突然和某個身穿髒亂衣服的乞丐對上了眼神。
雖然是在貧民窟很常見的風貌，但蘭貝爾感覺到了一瞬間的殺氣。
蘭貝爾停下腳步時，乞丐也站了起來，往蘭貝爾的反方向逃跑。
。
看到他的跑步的樣子，蘭貝爾確信了。
偽裝成乞丐的人，是戰神羅賓漢。
隱藏著正體，在都市巴萊拉散步的時候遭遇到蘭貝爾，明白到自己的變裝騙不了蘭貝爾便撒腿就跑。
。
蘭貝爾也迅速地追在羅賓漢之後。
。

「抱歉，借我用用啦」

。
羅賓漢踢飛了牽著馬走的男人，輕盈地乘上他的馬，對馬的側腹用力一踢便沖了出去。
正好在場的人們，慌張地讓出道路以免被卷入。
。
(總算是甩掉他了。我還不能就這麼死去呢……。不過，在街道上騎馬太引人注目了。得趕緊在某個地方下來……）

。
讓馬減速的時候，從背後傳來悲鳴聲和歡呼聲。
如同馬蹄般巨大的腳步聲傳入耳中。
感覺到騰騰殺氣的羅賓漢，不敢相信地回過頭。
。
羅賓漢回頭一看。
在視線的另一邊，穿著大鎧的男人，用不劣於馬的速度往這邊奔跑。
。

「……開玩笑的吧」

。
羅賓漢一邊提高馬的速度，一邊扔掉偽裝用的破布，把隱藏著的弓拿在手上並回過半身。
然後對緊追其後的蘭貝爾射出箭矢。
然而蘭貝爾毫不猶豫地斬落了箭矢。
完全看不到他減速的樣子。
。
羅賓漢臉色發青了。
怎麼說也不可能。太荒唐了。
。

「可惡……如果是塞拉弗，肯定能逃出去的！」

。
即使穿過了貧民窟，蘭貝爾也沒有減速。
就像是沒有疲勞感一樣地飛奔著。
從這裡開始的路人會變得更多。。
。
沒有辦法的羅賓漢，只能在馬背上跳起，在空中轉了一圈之後著地。
打算進入狹窄的小道甩開蘭貝爾。
之後，羅賓漢向近處的大型建築物旁邊跑去，卻突然停下腳步。
。

「……？」

。
蘭貝爾對羅賓漢的行動感到驚訝。
要是停下腳步的話，被自己追上的可能性就變得更大。
冒著暴露真身的風險，當機立斷地全力逃跑，有著毫不猶豫地搶走行人的馬的行動力的羅賓漢，為什麼會突然地停下來。
。
羅賓漢眼前的建築物的門，從裡面被踢開了。
從裡面不斷湧出了充滿殺氣的冒險者們。
。
冒險者們翻著白眼，從口中滴落著唾液。
共同點是他們的行動都十分生硬，似乎有著奇怪的傾向。
他們的表情顯得毫無生氣。
。
(這個情況……動作精度的惡劣，是大規模的死操術嗎！？）

。
街道上似乎沒有在使用大型死操術的人。
蘭貝爾察覺到，『吹笛之悪魔』開始行動了。
