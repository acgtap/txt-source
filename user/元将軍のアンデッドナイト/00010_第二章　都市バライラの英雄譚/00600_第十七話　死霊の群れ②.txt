在都市巴莱拉贫民窟的某个魔法道具杂货店的地下，兰贝尔在与十二名魔术师对峙着。
。

「真是遗憾呢。你发现了我们的里公会『暗夜的时计塔』的老巢，然后打算趁我们不注意潜入进来……对吧？你错了，是我，把你引诱到这里来的。听说，有个像狗一样嗅探着我们行踪的家伙呢」

。
在十二名魔术师中年纪最大的，四十岁左右的男人向兰贝尔这么说道。
雕刻在脸上的纹身，和浑浊的三白眼是他的特征。
从嘴里露出的舌尖上，连着金***坠。
他的名字是尼德瓦尔。是有着毒蛇的尼德瓦尔的别名的魔术师。
他正是是『暗夜的时计塔』的公会长，且被雷吉欧斯王国指名通缉着。
。

「有个女人向你哭诉了对吧？恋人被『暗夜的时计塔』杀死了，请帮我报仇什么的。真是遗憾呢，那个啊，其实是我的女人哟。是个不错的美人对吧？面白いくらいコロっと引っ挂かってくれたぜ(看不懂）」

。
尼德瓦尔说着，其他的魔术师们放声大笑。
。

「这就叫瓮中之鳖啊铠甲混球」
「不愧是尼德瓦尔大人！知道我们被跟踪之后，立马就作出攻势了」
「饶命给我们看看啊？有趣的話，尼德瓦尔大人有可能会放你一条生路的哦？」

。
过了许久，大失所望的兰贝尔嘀咕道。
。

「姑且还是过来一看……果然，只是些小人物而已吗。浪费时间了」

。
兰贝尔泄气地卸下肩膀，用手扶着头盔陷入了低沉，以他为目标，组成圆阵包围着兰贝尔的魔术师们，各自拿起杖开始了咏唱。
。

「别杀了他哦。必须打听出是谁命令他来找我们的！」

。
那番話宛如信号一般，炎之魔弹一齐向兰贝尔放出。
下一瞬间，本应低头的兰贝尔的身影消失，出现在别的地方。
。

「啊……？」

。
对不规则放出的炎之魔弹，兰贝尔宛如没有穿上那笨重的铠甲般轻盈地躲开了。
和他的神速相比的話，炎之魔弹就如同于静止一样。
消失、出现、消失、出现。明明还停留在视野之内，但尼德瓦尔他们的动态视力完全适应不了他的动作。
在组成圆阵的的一名魔术师眼前，兰贝尔突然出现。
。

「啊……」

。
在魔术师察觉到被兰贝尔接近之后，大剑已经砍下了魔术师的头颅，被踢飞的肢体撞在了后方的墙壁上。
接着兰贝尔横跳，砍下了隔壁的魔术师的头颅。
结束之后，又往下一个魔术师的所在之处飞去。
。

「别、别过来！别过来啊！」

。
其中一名魔术师，把杖对准兰贝尔。
用颤抖的双手握着杖，打算发射魔弹，然而还是败给了从正面突击而来的兰贝尔的压力，无意识地松手了。
结果，毫不抵抗地被兰贝尔刎下了头颅。
。
兰贝尔绕着圆阵不断跳跃，把包围他的魔术师一个接一个地砍杀。
最后的一人便是尼德瓦尔。
。

「别、别太嚣张了啊！出来吧毒蛇们！」

。
三条有着赤红色外表的蛇从尼德瓦尔的杖中释放，笔直地袭向兰贝尔。
。
那是用玛娜作成的迫真的蛇，直至玛娜用尽为止，三条蛇作出如同有意识的真蛇一样的动作。
因而在空中不规则地蠕动，即便是躲开也会追咬不放。
虽然只有三条蛇，但要完全躲开从近距离释放的三条猛蛇是十分困难的。
并且，蛇牙上还隐藏着只是轻轻一碰就能夺走对手意识的猛毒。
若触碰到鳞片的話，碰到鳞片的地方就会立即**，从而失去战斗能力。
。
是十分难缠的飞行道具。
同时也是成为尼德瓦尔的代名词的魔法。
。
当然，对兰贝尔来说毫无意义。
兰贝尔用大剑画出圆形，弹开了三条猛毒蛇的蛇头。
倘若是放着不管，三条毒蛇也会被兰贝尔的魔金铠反弹消失，但兰贝尔深爱着只授予给四魔将的这个铠甲，不想被污秽的蛇毒玷污。
。

「等、等一下！WO、WO、我的背后，可是有『杀戮曲马団』撑着的哦？要是杀了我的話，你可是会被追杀一辈子的哦！变成那样的話，你这一生就只能在他们的阴霾之下苟且偷生了。永远都无法安眠的……」

。

「不巧，我不需要睡眠」

。
况且兰贝尔最近已经杀死了『杀戮曲马団』的其中一名副团长。
要是被盯上了的話，那应该在那个时候早就被盯上了。
事到如今才屈服于这样的威胁，简直是毫无意义。
。
兰贝尔放出的突刺，贯穿了尼德瓦尔的胸膛，把他钉在了墙上。
尼德瓦尔的身体停留在空中，手脚耷拉地垂下。兰贝尔拔出大剑，尼德瓦尔便倒在地上。
。
兰贝尔俯视着尼德瓦尔的尸体，摇了摇头。
。

「……落空了呢」

。
兰贝尔单独地对『吹笛之悪魔』进行调查。
兰贝尔认为，若『吹笛之悪魔』打算对近处的都市巴莱拉发动袭击的話，那么他们可能早已潜入都市巴莱拉之内。
不能先发制人吗？兰贝尔心想，于是他收集情报，寻找聚集着形迹可疑的魔术师的地方。
。
兰贝尔把大剑收回剑鞘中，离开了『暗夜的时计塔』的据点。
接着继续在贫民窟探索，突然和某个身穿脏乱衣服的乞丐对上了眼神。
虽然是在贫民窟很常见的风貌，但兰贝尔感觉到了一瞬间的杀气。
兰贝尔停下脚步时，乞丐也站了起来，往兰贝尔的反方向逃跑。
。
看到他的跑步的样子，兰贝尔确信了。
伪装成乞丐的人，是战神罗宾汉。
隐藏着正体，在都市巴莱拉散步的时候遭遇到兰贝尔，明白到自己的变装骗不了兰贝尔便撒腿就跑。
。
兰贝尔也迅速地追在罗宾汉之后。
。

「抱歉，借我用用啦」

。
罗宾汉踢飞了牵着马走的男人，轻盈地乘上他的马，对马的侧腹用力一踢便冲了出去。
正好在场的人们，慌张地让出道路以免被卷入。
。
(总算是甩掉他了。我还不能就这么死去呢……。不过，在街道上骑马太引人注目了。得赶紧在某个地方下来……）

。
让马减速的时候，从背后传来悲鸣声和欢呼声。
如同马蹄般巨大的脚步声传入耳中。
感觉到腾腾杀气的罗宾汉，不敢相信地回过头。
。
罗宾汉回头一看。
在视线的另一边，穿着大铠的男人，用不劣于马的速度往这边奔跑。
。

「……开玩笑的吧」

。
罗宾汉一边提高马的速度，一边扔掉伪装用的破布，把隐藏着的弓拿在手上并回过半身。
然后对紧追其后的兰贝尔射出箭矢。
然而兰贝尔毫不犹豫地斩落了箭矢。
完全看不到他减速的样子。
。
罗宾汉脸色发青了。
怎么说也不可能。太荒唐了。
。

「可恶……如果是塞拉弗，肯定能逃出去的！」

。
即使穿过了贫民窟，兰贝尔也没有减速。
就像是没有疲劳感一样地飞奔着。
从这里开始的路人会变得更多。。
。
没有办法的罗宾汉，只能在马背上跳起，在空中转了一圈之后着地。
打算进入狭窄的小道甩开兰贝尔。
之后，罗宾汉向近处的大型建筑物旁边跑去，却突然停下脚步。
。

「……？」

。
兰贝尔对罗宾汉的行动感到惊讶。
要是停下脚步的話，被自己追上的可能性就变得更大。
冒着暴露真身的风险，当机立断地全力逃跑，有着毫不犹豫地抢走行人的马的行动力的罗宾汉，为什么会突然地停下来。
。
罗宾汉眼前的建筑物的门，从里面被踢开了。
从里面不断涌出了充满杀气的冒险者们。
。
冒险者们翻着白眼，从口中滴落着唾液。
共同点是他们的行动都十分生硬，似乎有着奇怪的倾向。
他们的表情显得毫无生气。
。
(这个情况……动作精度的恶劣，是大规模的死操术吗！？）

。
街道上似乎没有在使用大型死操术的人。
兰贝尔察觉到，『吹笛之悪魔』开始行动了。
