
———再怎麼超乎意料也要有限度


。

说不定早已经注意到了，『中村・信一』这个男人并没有将这座学园视为有半点学习价值的地方。
即便跟其他学生同样是出自於“想知道自己所不知道的事情”为出發点，但他从起初开始就并非是想要“学习”，而是为了“调查”而来到这裡。
原本他就不怎麼知道如今世界的常识，根本没有餘力享受学校生活。
搞不好会因为常识的不同而引**乱，他总是会如此胆颤心惊。
。
虽说如此，要是因为惧怕而什麼也不做就没有来到这裡的意义了。
首先想要调查就需要瞒过监视的耳目。
虽然摆脱他们的手段多得是，但考虑到後果就不能这样做。
难得对方过分低估了这边的能力。
要是再增加引人注意的要素并非是上上策。
。
还是先安分一阵子，等待监视鬆懈以後才是聪明的选择。
为此他想要见识一下学生们的平均实力。
既然这座学园拥有“战鬥训练”，那想要实现这点是很简单的。
毕竟在歷史课之後便正是这样的课程。
所以趁着休息的时间来到了运动场，随後因为眼前的光景而哑口无言。
。
「什麼鬼，那是？」
。
运动场只是有名无实，其实是在观众席围绕下，圆形的宽敞闘技场。
难以称得上是运动场或者体育馆，根本就是圆顶比赛场馆。
总觉得又是将这裡是学校的认知都全盘摧毁掉，他决定将其抛之脑後，不再多做思考。
。
然而在那边进行的战鬥，令他感觉到强烈的违和感。
武装不同是理所当然的，毕竟剑与魔法的世界法兰迪亚跟科学技术连次元都可穿越的伽雷斯特是不可能会相同的。
虽说即便是这样，在使用者依然是人类的情况下武器还是有相似之处。
。
然而唯独身上所穿戴的颜色各异的全身铠甲就不同。
正所谓是钢之铠甲。它并非是通过将素材加工为铠甲的形态供人员穿戴後从而具备本身应有的防护作用，而是藉由以覆盖人体为前提进而自动展开的机器铠甲来实现这一点。
应该是有某种机关吧，根據人员的不同，有的是全身武装，有的只是武装手脚部分，有的甚至只露出脸部而已。
随後从那覆盖全身的动力装置中展开出武装。
。
究竟是什麼样的技术呢。
。
小型的机器部件逐渐组合变形，大小与重量也随之發生变化，形成剑、大锤、大砲。
。
「喂喂」
。
对於眼前有许多仿彿就像是以前看到的变身超人一般的情景，他感到头疼起来。
虽说这也谈得上是男孩子的梦想，不过这种光景的衝击性过高，以至於不会有这种想法。
。
「呜哇，果然上级生的模拟战好厲害啊！」
。
「我也想快点使用那种东西！」
。
「前辈！加油！」
。
更重要的是对此进行讚扬以及加油声令他觉得头晕。
这座第一运动场的场地是十分宽敞的，在休息时间时便会向外开放。
在足以匹敌东京体育场的场地裡，挥舞着巨大武器的学生们以两人到四人为一组，各自在战鬥範围内的圆圈裡进行战鬥。
其中有三个人在最为宽敞，最为显眼的中央圆形战鬥区域中进行战鬥。
。
「嘿，给我更努力一点啦，贵族大人！！」
。
黑髮男人仅在手脚上展开绘有白线黑线设计的装甲。
他双手握着粗长的剑柄，挥舞着饰有多数装饰的护手上所延伸出的能量状剑刃。
那把有着自身的身高两倍长的大剑将金黄色和藏青色的铠甲砍开。
骤然间，光是餘波就导致场地的地面上迸裂出巨大的伤痕。
。
「咕」
「啧」
。
除了脸部以外全身都由装甲所覆盖的男女都为了应付此招而利用腿部引擎而远远拉开距离。
隐约能窥视到的髮色是鲜艳的色彩且没有违和感，应该是伽雷斯特人吧。
金黄色铠甲的是女学生，而藏青色铠甲的是男学生，不过双方的髮色是截然相反。
他们在拉开距离後，手上便出现幾个砲管并且释放出光芒。
。
「你可别以为能避开！」
「接招吧！！」
。
砲管在两人的手腕上以其为中心进行回转，开始释放出砲击。
（飒:不知为何联想到洛克人）
释放出的砲击并非是物理性的砲弹，而是光块。从颜色来看应该是梵托。
在转换为破坏性能量的砲击之雨中，白色的学生傲然屹立，无畏地笑着。
。
「啊哈哈哈，这种小儿科对我有用吗！！」
。
他既没有避开也没有採取防御的姿势，仅仅是站着承受着攻击。
随着响起的高亢笑声，砲击之雨岂止是没有给他造成伤害，甚至连冲击力都没有。
没有命中的砲击在地面上炸出不小的凹坑，由此可见威力非同小可。
白色装甲以及他本身的防御值相加，实在是太高了。
。
「可惡，明明都提高威力了！」
。
「居然连新型都没能起到作用！」
。
「两打一也还是这种惨样吗！
　区区亚丽丝迪露的跟班还想打赢我，真是不知天高地厚！」
。
男子的表情上露出真心的嘲笑，举起大剑宛如抗在肩膀上。
相对的，金黄色和藏青色的学生则是消除掉砲管，跟对手一样取出武器形成剑刃。
从梵托中抽出的能量都是金色光辉这点便可以证明。
不过在大小和长度的方面上，两人都逊色於白色的学生。
。
「这种玩具能够承受住我的攻击吗！！」
。
 犹如疾驰一般，在腿部装甲迸發出火焰的他飞奔而出。
并且就这样毫不对準，顺势地全力地挥下巨大的剑刃。
。
「啊啊啊！？」
「呀啊啊啊！？」
。
两人将大剑交叉起来的防御姿势犹如纸张一般惨遭溃散，剑刃在两人的之间砍下。
这道攻击岂止是没有直接命中，就连掠过对手都没有。不过它的餘波则是导致他们俩身上的装甲产生裂痕，并且还振飞出圆圈区域。
应该是通知模拟战结束吧，在此时响起一阵宛如警报器的声音。
从状况来看，完全就是双人组的败北。
。
「啧，杂鱼就别大放厥词了。
　既然要说得那麼了不起，至少也要抗住我一击啊」
。
白色的学生并没有看向遭受振飞的衝击而一动不动的对手，而是消除大剑像是完全没有兴趣一般走开了。
。
「还是老样子真是无双呢，真宫寺那傢伙」
。
「真宫寺前辈，好棒！」
。
「即便是哈泽和卡露联合也打不赢吗……」
。
「这样一来，除了帕蒂艾露以外就全灭了」
。
在观众席上观望战鬥的观众们半是讚扬，半是失望地说着。
而在二楼最前方的席位旁边楼梯下看着的信一，则是边听着他们所说边严肃地蹙起眉头，仿彿在观察着整个场地。
。
「呐，你觉得我在这时候应该要露出什麼样的表情？」
。
「啾、啾、啾？」
。
虽然他向头上的搭档说着，不过她像是感到迷惑般歪着头而已。
信一也感到抱歉地想着自己要是處於相反的立场上也应该会露出相似的反应吧。
。
小看他们了。
。
他无法否认自己因为通晓其他的异世界知识而骄傲自满。
所以自以为已经知道对待异文化的方式。
在这之前他只是觉得其中还有地球文化混杂在其中，所以觉得很难应对而已。
然而实际上并非是如此，并非是这种程度的事。
进行武装，拿起武器攻击对手，将其打倒。
在战鬥中要做的事情，无论是伽雷斯特还是法兰迪亚都的确是一样的。
不过其中的某些方面，实在是、没错，未免也太───
。
「这场幼稚的战鬥是什麼鬼」
。
───根本是小孩子打架的水平。
撇除外行人、学生、异文化这些方面後，就是这样的评价。
跟事先得到并已“体验的情报”完全不同。
如果要纠正中央展开的战鬥，首先是动作幅度过大。
只为了避开大剑而使用装甲的加速功能这点也是浪费。
考虑到那攻击的衝击波，拉开的距离只要一半就行，如果可以立刻反击就更不用说了。
然而他们无谓地拉开距离导致反击都无法实现，并且在之後發射的砲击大多都是射偏的。
每一發砲击中所感觉到的能量是无可挑剔的，然而裡面毫无一物，伤害薄弱。
话是这麼说，那位白色的男学生所採取的以站立姿势依靠着防御力也是问题。
这次的砲击单单只是能量而已所以还好。可如果是携带其他效果的砲击呢。他简直就像是没有考虑过会受到致命性异常状态的可能性。
在最後的攻防中两人从正面等待着显然是强力一击的到来，这是下下策。
当时应该是避开攻击并且对他的破绽进行攻击才是常规的战鬥方式吧。
而那位选择这种敷衍攻击来作为最终一击的白色学生也是理解不能。
强大的加速以及没有活用这点而挥下的大剑是完全对不上应当进行攻击的时机。
。
正如字面意思上全靠蛮力。
。
如果能活用自身强大的力量值，应该还是能成为名留歷史的战士吧。
然而那只是在糟蹋自己的力量，无谓地浪费体力而已。
所利用的科技如此之高，可所作所为就跟单纯的互殴别无二致。
简单来说，这种幼稚的战鬥方式就是给出强力一击的人获胜。
根據状况，这样做是可以起到优势，可从周围的反应来看，这似乎才是普通的。
世界不同甚至都能产生这麼大的区别吗，信一不禁远望着天空。
。
不过其实信一也不是瞧不起或者想批评伽雷斯特的战鬥方式。
世界是不同的，假如说这种方式也能成立的话，他根本没有资格提出意见。
实际上他们的威力十分优秀，在对人以外应该是出色的“兵器”。
只不过跟自己的相性实在太好了，所以他只是觉得“根本不会输”这点太棘手了。
。
「………虽然到现在才想到，没有在别人面前进行战鬥真的是做对了……」
。
回想起来到这裡之前，那些日子中所引發的骚乱後，他耸着肩膀。
与其说是感到安心，他更是後悔当初应该更慎重地隐瞒起来。
现在想起来已经展现给某些人看到了，总有种危险的感觉。
他反省着自己比起想定中还要如同呆在那边时所行动一般的举动。
哈啊，他微微叹了口气摇了摇头，为了上课而正打算离开观众席时。
。
「喂，你─────刚刚说什麼了？」
。
突然间，有股蕴含怒意的声音从背後那边的场地上传来───
