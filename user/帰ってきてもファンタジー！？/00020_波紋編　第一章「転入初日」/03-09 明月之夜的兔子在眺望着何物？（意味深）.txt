弗蘭克也注意到這是自己的責任，以惶恐地態度向她道歉，對此也許是沒有意料到，沒辦法順利接受道歉的她顯得驚慌失措。
。
「啊……那個、我並不是想要讓……哥、不對老師道歉的……」
。
當中莫名有著某種無法令人聯想到是兄妹的距離感，以及有所顧慮。信一則是感到心酸似的定睛注視著他們。
。
「……話說回來，接下來的課程怎麼辦？」
。
不過他立刻將表露在外的感情掩飾起來，以認真的神色詢問道。
雖然從這說話內容很難斷定究竟是不懂察言觀色呢，還是硬是無視。
。
「雖然說得有理，不過這可不是把亞麗逼走的人該說的話吧？」
。
言外之意就是在指責他是否知道這件事，不過信一擺出一副事不關己的表情。
。
「…………哈啊，弗蘭克老師，還有什麼要教的嗎？」
。
「…………是呢，剩下的只有試用技能而已」
。
雖說如此，的確也不能就這樣直接中斷課程。
恢復成認真神色的芙莉蕾看著她。
。
「那麼露歐娜，作為向特別科學生的請求，接下來的課程輔導就拜託你了。
　我必須得回到那些傢伙了……」
。
她注視著不知發生什麼事的１－Ｄ班的學生們。
不過從他們眼神中顯露出不悅的神色這點來看，應該是察覺到由於信一的緣故導致課程中止，自己所憧憬的亞麗絲迪露也回去了。
她有必要回去一趟，將他們意識拉回到課程之上。不過，
。
「好過分，芙多納老師！居然要我被信一推倒！？」
。
狐狸娘將剛才的行為作為把柄，像是保護自己一般抱住自己。
又開始了，芙莉蕾如此呆然著，不過當事人比起她還要更快進行否定。
。
「放心吧，我可沒有半點念頭想要推倒你……不如說是不想？」
。
「身為女孩子，這句話總覺得好令人恥辱！？」
。
「………小孩子嗎」
。
「你說什麼了嗎，信一！？」
。
「好了啦，關於技能的課程快點教我吧，課程時間沒剩多少了」
。
「對我的態度一下子就隨意起來了！？」
。
隨後原本在今天應該是初次見面的兩人進行著猶如相聲般的對話後，邊聊著不著邊際的事情，邊開始進行技能相關的課程。
見狀，芙莉蕾也立刻放下緊緊握住的拳頭。
。
「怎麼了，芙莉蕾老師？」
。
「哥、弗蘭克老師有沒有覺得剛才我揍中村的時候，他的動作有古怪的地方？」
。
「誒？」
。
被詢問的他對於剛才的認知僅僅是被揍的學生而已。
因此面對“同事”的詢問，他只是一副感到疑惑不解的表情。
。
「我感覺那傢伙像是硬要壓抑住自己反射性想要施展的動作」
。
「怎麼可能！為什麼要做出這麼麻煩的事情……」
。
「如果不這樣做……難得通過捉弄帕蒂艾露，借此當做沒有發生過的企圖就完全白費了」
。
「啊！？」
。
她這麼一說以後，他才注意到自己遺忘掉剛才的事情。
在那之後所發生的超乎尋常的事情將之前衝擊性的光景都給全數覆蓋了。
。
「……你、究竟是什麼人？」
。
她定睛注視著安分地接受狐狸娘授課的少年。
那眼神已經不是看著棘手的問題兒童，而是其他。
自從轉學會面以來僅僅只是幾小時，正如他所說，光憑這點時間根本不可能完全了解對方。
不過他實在是完全沒有上進心，以及說過不想引人注目。
再怎麼說是公開交流之前的人物，對於能力值低下這件事也沒有半點自卑。
以及他那身能夠反射性將刀刃揮出的嫻熟動作。
