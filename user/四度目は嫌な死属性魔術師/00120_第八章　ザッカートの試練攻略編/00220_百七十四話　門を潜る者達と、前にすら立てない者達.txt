创立了冒险者公会的法马斯・哥德。他也推广了魔物加工法，从其传说中意外得知了魔物加工法其实是アーク等四人的功绩真相。
。
只是，关於《堕落的勇者》的寨卡特的传说、传承或其功绩幾乎没有留下。为了成为贝武夫的後继者而挑战寨卡特，为了这目的而挑战《寨卡特的考验》的海恩兹们，最重要的是对他详细调查。

「万事俱备……但这是怎麼一回事啊」
。
在炎热的夏天太阳下，海恩兹一边说着一边叹气。视线中看著能够探知《寨卡特的考验》下一个出现地点的魔术道具《考验的探索者》。
。
那是依然指向山谷的墙壁。
。
大约两个月前，在初夏的时候就一直没有变化。
。
「不管怎样，都只能等待了吧。小心技术不要变迟钝啊。」
。
在海恩兹背後，格闘士的杰尼弗发出了声音。
。
「不要说因为等累了就停止等待啊？」
。
「当然了」
。
一边这样说着一边回头看，在杰尼弗的视线看见了像露营村的视野。
。
「我等了那麼久，若你说不等了要回去的話，我会把你打飞的。」
。
「不要用打的，用踢的可不可以？但是那个《考验的探索者》是不是坏了啊？」
。
「啊，这两个月我查了好幾次。都是正常运作着」
。
「那麼今後也是和昨天一样，只等着《寨卡特的考验》出现吧。」
。
这样说着，精灵的女神官，戴安娜擦着洗完的头髮出现了。

「夏天最好的享受就是沐浴，海恩兹也流汗了吧？现在使用的是蒂蕾沙和塞伦，下一个到你吧。」
。
「……已经开了沐场了吗？若有多一个月时间的話便可以开拓魔境了，真正的建设工程马上就要动工了」
。
在海恩兹视线前面的野营村，最初是等待着《寨卡特的考验》海恩兹们的小规模野营地。
。
但是，在等待了两个月的时间内，支援者们为了支持海恩兹们而停留，为了S级冒险者的名誉贵族们和行商商人们也来了。
。
那个规模已经超过了小村庄的规模。但这裡基本上是危险的魔境。
。
「那也不错啊。本来这裡并不是那麼大的魔境，也没有迷宫」
。
「因为杰尼弗们为了打发时间，魔物也变少了」
。
一直等待着很闲的海恩兹们幾次回到了镇上度过，但长时间离开是不行的，但在这个状态下不能接受其他委托或是潜入迷宫。
。
因此，为了消磨时间就狩猎了这个魔境的魔物。结果就淨化了这个魔境，这个魔境最高阶位的魔物只有5级，所以根本不能威胁到海恩兹们，所以魔物瞬间减少了很多数量。
。
所以，临时露营村在魔境中很和平。尽管如此，数日一次也有哥布林等智力低的魔物袭击过来，但被海恩兹们的支持者和原阿尔达过激派的犯罪奴隷们驱除了。

「法森公爵已经準备了开拓计划，或许已经进入準备阶段了吧」
。
「之后从《寨卡特的考验》中出来的話，也许不是魔境，而是完全的村庄了」
。
「哈哈，这件事……」
。
「即使不说也知道。……虽然不知道现在哪裡有《寨卡特的考验》，但可能还有我们以外的挑战者。而且最起码也和我们一样能在内部生存一个月以上时间吗？」
。
《寨卡特的考验》在内部不存在挑战者的話，会在一个月後【转移】到另一个地方。
。
反过来说，只要有人在内部，有挑战者存在的話，就会一直留在那个地方。
。
本应《寨卡特的考验》会【转移】到山谷的墙壁上，但两个月以上也不出现的話应该是谁在挑战中了。
。
「除了我们以外，还有不懂珍惜生命的傢伙啊。听说海恩兹们警告後，其他的挑战者幾乎都没有了。」
。
「到底是谁呢。如果是B级以上的冒险者的話，大概会有传闻的吧」
。
「戴安娜，B级的話是不够的。能存活一个月以上的情况下，应该是A级或以上的。没错吗？海恩兹。」
。
「……我觉得应该是。之後我们攻略的話，最少也要花一个月以上的时间。」
。
若果能在两个月以上，近三个月时间在《寨卡特的考验》裡生存下去的話……现在也许还在继续攻略中。
。
如果花这样的时间一直存活下去的話，那大概是到达了自己返回的第四十二层雪山的阶层了吧。这样想的話，海恩兹便浮现出複杂的表情。
。
「……我当然想自己攻略《寨卡特的考验》。但是，又不想现在在哪裡挑战考验的挑战者失败……不想他们死呢。」
。
若在海恩兹们面前出现了《寨卡特的考验》的話，这意味着在裡面的挑战者已经不在了，很高概率已经死亡。

「不要在意啊，那些傢伙也已经觉悟了吧。而且你们都能生还，不一定会死吧」
。
「那也是啊，对了……向阿尔达祈祷吧。希望他们生还。」
。
「是啊。阿尔达一定也希望这一点吧」
。
这样的三人祈祷着在某个地方的挑战者们的生还……他们做梦也没有想到被祈祷的阿尔达和作为挑战者的班达鲁们也不会高兴。
。
同一时间，班达鲁们已经攻略了雪山阶层，更越过了下面的冰原阶层，到达了底层。
。
今天他们还继续等待着《寨卡特的考验》。
