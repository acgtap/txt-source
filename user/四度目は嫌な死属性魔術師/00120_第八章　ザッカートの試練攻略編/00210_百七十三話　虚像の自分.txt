《迷宫的邪神》格法多加恩坐在《寨卡特的考验》的最深处，只是静静地注视着班达鲁的行动。
。
在自己创造出来的迷宫中的他，没注意到罗德科尔特想舍弃这个世界，亦没注意到已经失败了。
。
他的身体微微地颤抖着。
。
『我所配置的所有考验都是以设想过的不同方法攻略的。生成的魔物被蹂躏，配置的陷阱全部被践踏……！』
。
虽然班达鲁已经很自重了，但从格法多加恩的角度来看卻很糟。
。
随着来自边界山脉内挑战者的活跃，上层的谜题在很早阶段已经有完美回答的挑战者。
。
在攻略完中层的严峻环境後进入的阶层，是和平的沙滩阶层，但设置了定时毁灭，但从几十年前就开始出现了一定数量能攻略的人。
。
但是，有挑战者帶着上层迷宫的骨壁到下层，代替魔物破坏自然，以这样的方法来攻略的人是从来不存在的。
。
应该是严格的中层，但一天比一天更进步地攻略。即使是边界山脉内部的挑战者，每年都会出现几个脱出者，但他们一个脱出者也没有。
。
使之成为可能的是，对于格法多加恩来说，是未知的技能和职业的效果，或是【魔王的碎片】，以及那辆马车的不死族吧。
。
适应严格的环境，不仅仅是利亚魔物的素材，还以【魔王的碎片】生成的素材来利用的惊人行动力。意味着有超越常识的力量。

比起这些，还有看起来没有界限的强大战斗力。
。
只要不把这些全部封印起来，就不可能阻碍他们攻略。所以，班达鲁作为考验的对象是不合适的。
。
就如同在陆地上安装的陷阱想要捕获大海中游泳的鲨鱼。因为太过不同而无可奈何。
。
虽然他承认了这一点，但格法多加恩却在冲击的感动下大声喊叫。
。
『好厉害！那才是超越设想的行动，是连大神都玩弄的寨卡特！』
。
如果他有肉体的話，脸颊会潮红，眼睛会流出喜悦的眼泪。格拉德加恩就是那么为班达鲁的行为而感动。
。
格法多加恩从一开始就没有期待着完美攻略《寨卡特的考验》各阶层的挑战者。
。
只要超越这些考验的人，就是适合当寨卡特的後继者。所以以那样的想法创造了迷宫。所以，即使不是模范答案，如果能走到最裡面的話，就会被认为是寨卡特的後继者。
。
对自称《法命神》的阿尔达、和与阿尔达的众神相关的掌权者。首先要告诉他们真相，促使他们提心吊胆。不那样做的話，作为魔王军的一员，被给予机会的勇者寨卡特使徒的我就没资格当了。
。
……格法多加恩最不能接受的是，如果要伤害到比他自己更贵重的寨卡特遗产的話，就只能抹杀掉。

但是，自从创造了这个《寨卡特的考验》之後，已经有一百年了。很多挑战者在上层败退，边界山脉内的挑战者也没有攻略到超过下层的人。

约百年前从维达那裡收到神谕，格法多加恩认为是『寨卡特的後继者出现』的预言，但也有很多人认为这是错误的。

格法多加恩觉得有希望的有两组人。第一组是一群四人，他们在向阿达尔祈祷之後尝试攻略，是叫《五色之刃》的队伍。
。
最初只觉得向阿尔达献上祈祷，是个误会了的挑战者。但是，在某种意义上来说，虽然没通过考验，但有只凭力量来攻略的氛势。这是意想不到的。
。
优秀的边界山脉的挑战者，也有很多人使用头脑。但在错误的方向上继续进行挑战的人只有他们。在边界山脉外的阿尔达信徒的挑战者中，是有最深的攻略度，也是最值得惊叹的。
。
那个战斗能力……是的，只有战斗能力。
。
但是，结果还是从中层返回，在雪山的阶层中一人丧命撤退了。但是，因为他们是第一个攻略至中层的边界山脉外的挑战者，所以记忆犹新。
。
但是格法多加恩对第二组的班达鲁一行人的期待比《五色之刃》更强烈。

『和他们相比，他们只不过是个石头。啊啊……寨卡特啊』
。
对于格法德加恩来说，班达鲁操纵不死族，还可以自由地使用【魔王的碎片】都不是问题。

说不定他就是寨卡特。仅仅如此想，一切事物就会变得美好。

现在马上就想加入他们。虽然是这么想的，但也是过急了。「做好的东西必须慎重地尝试」。既然制作了这个《寨卡特的考验》，就必须慎重地给挑战者尝试。

但是，班达鲁们马上就要到下层了。在设置了寨卡特们被神赋予的能力、思想、形态的考验阶层。

 在那之前设置的补给所对他们来说可能没什么意义，但如果是寨卡特的接班人的話一定会发挥什么作用吧。

『哦，把全部的实力给我这个使徒看看吧……』

 可能是认为下阶层的入口，班达鲁手交叉地呻吟着。

「嗯，这裡的信息应该是《补给所》吧？」

「应该是那个……但感觉不太像呢？」

拿着从挑战者那裡听到的信息文件的伊丽斯，眼前的情景和《补给所》这句話完全不搭，她也很疑惑地侧着脖子和尾巴。

「好像没有店主的廉价出售物品的市場啊」

爱莲奥诺拉和金巴利这样评价说着，补给品真的很令人失望。

在《补给所》一看，像没有店主的市场一样，那裡陈列着一堆被认为是二手货有着伤痕的武具和魔术道具。

萨姆粗略地看了这些，说了。

『好像是这个迷宫死了的挑战者们所持有的装备和物资和遗失物』

贝尔蒙德从排列的剑中，拿起一把来看看。

看起来厚看的剑，但可能是用柔软性优秀的大马士革钢制作的。或重量和外表一样，但有特殊功能的魔剑。

……但那些剑很多是从中间被切断的，所以不能发挥本来的功能。而且，因为连剑尖都看不到，所以即使是班达鲁也不能修理。

虽说是挑战失败了，但是为了挑战作为危险的难关而广为人所知的《寨卡特的考验》，战者们所佩戴的装备，很多都是珍贵的材料兼用优秀的技术加工了的高级魔术道具。

食物都是普通的保存食物。所以带着新鲜食物的班达鲁们没有必要。

在爱莲眼中看来是这样，但在班达鲁眼裡就不一样了。

在班达鲁的話语中，大家不由得从补给品的东西移离了一步。不，作为不死者的萨利亚和利塔，还有艾拉是满不在乎的。

「没问题，没长霉菌」

在下层，是与上层相同的谜题和考验但更严酷。是以召唤勇者的神们给与寨卡特们力量形成的伟业，并以思想的型式给与考验的阶层。

那个难度极高，突破中层的猛者们也在这个下层接连脱落是至今为止的常事。

特别是在补给所的深处，越过门後的第一个考验是因其困难而闻名的。

在镜子构成的房间裡，班达鲁与自己的虚像相对。

「初次见面，我」

『这是我第一次来，我』

最初的考验，那是与自己的对决。

「所谓的人生就是与自己的战斗」，是希尔维罗所说「人心中有天使的自己和恶魔的自己」，是参考了这句話的考验。
。
挑战者在与虚像的自己进行对話，然後必须在实战中战胜他。

因为虚像是复制挑战者的记忆和人格的存在，所以在问答中会毫不留情地挖出了伤疤，并且每次挑战者受到精神上的伤害时，虚像的战斗力便会上升。

『我不是应该更早认真的进行魔眼的对策吗？我不是忘了谷巴蒙《破坏的魔眼》的事吧。』

「嗯，在爱莲奥诺拉的《魅的魔眼》和移植到贝尔蒙德的《石化的魔眼》中要加以抵抗的話有点难。所以研究了可以用【异形精神】和【状态异常耐性】打消。
如果是我的話我应该知道吧」

『哎呀，骗不到我啊。我就是我』

好像和以前的熟人加深交流一样，用平静理性的语气进行对話，班达鲁和虚像的班达鲁，一直以这种形式继续对話。

在这个考验中，很多情况下会发展到激烈的口角，如果不是这样的話，就会开始否定虚像，相反的話就会流着眼泪希望虚像接受自己的原谅，平时无论怎样冷静都会被隐藏在内心深处的伤疤刺激……。

『没有什么不完备的，但那个海恩兹呢？阿尔达没有承认死者苏生。他认为是扰乱秩序啊。』

。
「是啊。在地球上也有这样的言论。既然阿尔达的教义不允许，他们也不会容忍我，而且他们是我必须去杀的。如果可以的話，就用暗杀或阴谋达成目标。」

「已经到时间了吗？我想是否能听到以前的挑战者所接受的禅问答性问题」

『没有必要吧。不管说什么我都不会动摇。』

虚像像是放弃了任务。因为知道那是没有意义的事。

即使质问他杀人理由的是非，也会回答「要看情况和对方是谁」，即使问正义是什么，也只回答「不确定的东西」。而且，即使虚像问什么爭议性的问题，也只会说「也有这样的情况」。

对于复仇的是非，即使是虚像也肯定的。为了消灭从自己夺走幸福的人，使被再次剥夺的可能性変成零。是为了幸福的人生所需要的生产性行为。

町田亚乱和岛田泉们其他的转生者把看作问题的死者不死化的事，对班达鲁来说并不是奇怪的事。但因为死者是班达鲁的伙伴，只是增加了那些伙伴而已。

而且因为是信仰对象的维达们保证没有错，所以不是很在意。
。
『不过，我要问最後一次。死，你怎想？』

。
但是，虚像的质问也有价值。对使用相同属性的自己提出了问题。这是与另一个自己的问答，所以能很好的认清问题所在。
。
对于这一点，班达鲁在考虑了一下之後回答了。

「从生者到死者的变化。不可逆的事被认为是正确的，但我是司掌死的，所以应该颠覆死的现象。
也就是说，我喜欢那样做」

对班达鲁来说，死绝对不是神圣的。只是一种现象。
。
因此，让达露西亚复活的事没有任何疑问。

。
那样说着，虚像的班达鲁从镜子裡无声无息地溜走。那个样子果然和班达鲁一样。

『质问了也不会动摇，是我不能复制的记忆太多了吗……老实说，我现在是雜鱼程度的实力。』

「死属性魔术和【魔王的碎片】不能使用吗？」

。
『不行。说多一点的話【装群术】和【死灵魔术】都是不能用的。能复制的，只有照在镜子裡的我，不能复制伙伴……例如顺服的魔物，在这个迷宫中不能被视为挑战者的存在是无法复制的』

由格法多加恩的力量创造出来的万能虚像，能进行的复制也存在界限。

当然，通常没有超过界限的事都能做到……可惜班达鲁是规格外。
。
『这样说吧，我已经完了。请为妈妈的复活加油』

。
这样说了的虚像，班达鲁伸出了钩爪，但在碰到虚像时停止了动作。
。
「我怎么也没有干劲。果然是因为我自己吧？一般来说，见到自己就会互相憎恨呢。」

『说不定是意外的自恋。但是怎么办呢，不管我的我，只要我存在，我也不能前进的。』

。
「很为难啊，我像这样攻击我，心情会不好的」

。
虽然是暂时的虚像，但班达鲁还是不想攻击虚像而前进。

「果然是我自己吧……我是我。我是有我的记忆和人格的我」

。
『我也有我的记忆和人格。但是我是我的虚像』

。
「如果有记忆和人格的話，可以说是我。因为姿态也是一样的。虚实的差异是有意义的吗？」

。
『真实是虚像，虚像是真实的吗？不可能。我就是我，因为我是我』

。
「『我就是我——』」

。
「是我」

那样的说話结束时，班达鲁只餘一个人。然後无声无息的出现了一道门，那是去下一个阶层的楼梯。

　寄生虫を寄生させてヴァンダルーに装備される事で、試練をスルーしたイリスが浮かない顔で呟く。真面目な性格の彼女は、不正を行ったような後ろめたさを覚えているのだろう。

「仕方ないだろう。自分との問答と戦いなんて、乗り越えられる保証が無いぞ」
　イリスと同じように試練をスルーしたバスディアは気にした様子も無かった。因みに、試験をスルーしたのは彼女達だけでは無くほぼ全員である。
　補給場を通り過ぎる際、ヴァンダルーは寄生虫が寄生できないレギオンを除いた全員を体内に装備し、ゴースト達だけを連れて進んだからだ。

　あの試練は、鏡に映る実態を持った存在しか対象に成らない。よって、ヴァンダルーの体内に装備されている者達は試練の対象に成らないのだ。
「……バスディアもあっさりクリアしそうだと思うよ」
「そうでもないぞ、これでも悩んだり不安になったりしている。ジャダルは寂しがっていないだろうかとか」
「それは……どうなのだろうか？」

　虚像の自分自身と子育てに付いて問答をした後、戦う。勇者の後継者を決める試練として、それはいいのだろうか？

「だが危険なのは本当だ。試練の間は挑戦者ごとに個別の空間に隔離されるから、誰も助けには行けない。我もダンジョン攻略の最中でなければ、自分自身との戦いに興味があったが……」
　戦闘狂の気があるヴィガロも、安全策を取って試験をスルーしていた。

『俺達は最初から対象じゃないから、気楽なものだけどな』
「ギシャア」
　巨人種ゾンビのボークスや百足の魔物のピートがそう言い合う。この虚像の試練の対象は、ヴィダの新種族を含む人間と境界山脈内部で国を治めているノーブルオークやハイコボルト等、一部の魔物だけだった。
　それ以外の魔物やアンデッド等は鏡に映っても対象に成らない。

　勿論テイマーの挑戦者がテイムした魔物を連れ込む事は今迄に何度もあったのだが、このダンジョンはザッカートの後継者を見つけるための試練である。そのため、この虚像の試練の対象にはならないのだと推測されている。

『……終わった』
　そこに、試練を終えたレギオンが転がって現れた。何時に無くテンションが低いが、激戦を潜り抜けて来たようには見えない。

「お疲れ様です。何かありましたか？」
『試練は、思っていた以上につまらなかった』
『見た目はそっくりなのが出て来たけど、意味のある言葉を言わなかったんだよ。ジャック達あんなに変じゃないよ』
『失礼な話さ。酔っ払ったプルートーだってあんなに変じゃなかったのにねぇ』
『あなたが飲み物にアルコールを混ぜたのよね、バーバヤガー……！』
『イザナミ、昔の事を掘り起こすんじゃないよっ！』

　どうやら、現れたレギオンの虚像はバグを起こしてしまったようだ。
　一つの身体に幾つもの魂が融合しているレギオンのような存在に、人間用の試練が対応できるはずもない。ヴァンダルーの時も、正常に働いているとは言えない状態であったのだし。

『まあ、簡単に倒せたからいいけど』
「そうですか、グファドガーンにはあった時にもっと汎用性を持たせた方が良いと言いましょうか。じゃあ、先に進みましょう」
　こうして下層での最初の試練をヴァンダルー達は突破したのだった。
