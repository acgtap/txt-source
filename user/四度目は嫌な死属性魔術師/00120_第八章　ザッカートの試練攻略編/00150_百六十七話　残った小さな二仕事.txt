在豪华的、但满是血腥味的房间裡，班达鲁正【飞行】中考虑现在继续要做什么。
。
这个房间的主人是阿莱斯家族，是一个很粗暴的集团，亦是最近突然暂露头角的武斗派黑手党的基地。
。
而且现在也是血腥味的产生源。
。
「王，被告知的家伙都收拾好了」
。
「没有被告知的家伙，大家都没有处理。」
。
「我们很了不起吧！？」
。
「是的，谢谢大家。活着的人，让他们選擇之後要被杀掉还是消除记忆吧。」
。
布拉格为首的黑色哥布林等人，口头上报告着战果。他们把今天晚上在这座建筑裡的阿莱斯家族的头目和干部，以及他们的护卫们全都处理掉了。
。
那是用几次刺腹，用斧头打碎头，简单地用棒棍多次打击头部，像是犯罪组织间的争斗中被杀害一样的手段。
。
「总之，把这个帕尔巴克伯爵家徽章的戒指放在桌上……然後你出去」
。
在头目的桌子上放了戴了徽章戒指的托马斯右手，然後班达鲁用【装群术】拿出了装备在体内的男人。
。
『……』
。
看到脸上有刀疤的男人的脸时，在場的多尔顿惊讶地发出了声音。
。
「那家伙是『八裂』波拉姆？是僱佣兵的杀手，到现在为止杀了几十人的家伙。是不分生死的奖金榜首，在冒险者公会的委托墻上看过几次他的通缉令。他在这个城市裡吗？」

「是的。他是我知道的杀手中最知名的，真是个会做这种事的家伙啊。」
。
为了说服雷夠斯顿伯爵家而多次进入这个米尔格盾国王都的班达鲁，得到了当时聚集的灵中杀害自己的罪犯信息。
。
但是，因为在雷夠斯顿伯爵家的迁移结束前不能引起骚动，所以要处理的只有个人行动的杀手和犯罪者。
。
然後，把杀手和犯罪者装备著准备以後的用途，所以现在带着他来了。
。
「就在这裡，用阿莱斯家族的保镖用的剑再杀一次」
。
班达鲁刺了波拉姆几剑後，在杂乱的房间裡血腥味更浓了。
。
「这样的話，托马斯·帕尔帕克委托的暗杀者波拉姆们暗杀阿莱斯家族和被委托的托马斯因某种原因而被杀了。
其他地方也要重新杀了伪装战斗过，除了波拉姆以外的暗杀者还有几个，之後带走会被怀疑要背叛头目的干部的尸体，那就没问题了吧。伪装工作完成。」
。
「等一下。大概对你来说不是那么费工夫，但为什么要做这样的伪装工作呢？
这是为谁做的伪装工作啊？」
。
多尔顿问道正在选择帶回去的尸体的班达鲁。他好像不能理解做这一系列伪装工作的目的。
。
托马斯·帕尔巴克军务卿的身影消失了。在宅邸裡的骑士们都晕倒了，他们因毒物而引起记忆障碍，不记得犯人的事。房间裡留下了大量的血迹和军务卿的爱剑，而且他最後把遗嘱交给了照顾他的管家老爷爷。

阿德帝国皇帝马其克萨尔和原种吸血鬼比尔凱恩认为犯人只能是班达鲁吧。
。
虽然没有班达鲁是犯人的证据，但相反，这正是确实的证据。
。
只有他能做到这一点，完全无声无息地做到这一点。
。
虽然说想混肴视听，但班达鲁现在正进行的伪装工作，从多尔顿来看，只是一个大规模粗糙和显眼的伪装。
。
确实，阿莱斯家族是武斗派，亦很粗暴，是用毒辣的手法崭露头角的新兴犯罪组织。但是，归根到底只是一个犯罪组织而已。背後没有大人物贵族和其他国家的谍报机关，原种吸血鬼等什么的更是不可能看上他们。
。
那个阿莱斯家族没有直接的利害关系也没有怨恨，为什么要实行国家大人物贵族的当主本人的杀害呢？
。
如果失败的話，将以国家反叛罪全部斩首。即使成功，也没能得到什么。而且，如果事後事情暴露的話，还是国家反叛罪。
。
而且，即使是作为执行犯的《八裂》波拉姆和其他的杀手，从多尔顿来看，亦没有能杀军务卿的实力。虽然不知道他们的实力，但认为他们没能把在帕尔巴克伯爵家的骑士十几人一瞬间夺取意识的实力。
。
他们的名字是知道的，但从多尔顿来看只是二流……杀人的对象是商人等的普通人，以及D级冒险者而已。
。
他们很难被想象能杀现任军务卿的犯人。
。
在那样的伪装工作中，马其克萨尔和比尔凱恩不会认为不是班达鲁干的。
。
「虽然不好听，但这样的話皇帝和原种吸血鬼是不会被欺骗的。我觉得能被欺骗的只是不知道你的存在的普通军人和没有和他们相关的人员而已」
。
「那样的話已有足够的意义了。这是对什么都不知道的人的伪装工作」

「原来这是你的目的吗！？不，你考虑太周详了！？什么都不知道的家伙在这件事上几乎没有注意到你的可能性。即使有关系者发现了真相你也不在乎！？」
。
班达鲁的根据地是边界山脉的对面。万一优秀的忠臣和忠实的骑士们，还有托马斯的妻子们发现真正的犯人，也不能跨越山脈去追捕对手。
。
虽然舆论可能会推动到举军的讨伐，但跨越山脉的安全隧道已经被塞住了。要有能再举军讨伐的军队可能需要十年以上的时间……在托马斯不在的现在，可能需要更长的时间。
。
暗黑哥布林们不理解他们为何要讨论这些不必要的事情。
。
「我是为了库尔特们而做的，是因为雷夠斯顿伯爵家的缘故」
。
「库尔特？……雷夠斯顿伯爵家？为什么会从你的口中说出这些名字？」
。
「S级冒险者的你都不知道吗？」
。
「……你，S级冒险者队伍也不可能什么都知道吧！？还有，事到如今你为什么还会说出令人难以理解的話！？」
。
S级冒险者队伍的《暴虐之岚》并不拥有自己的信息网，而且对已经沦落的雷夠斯顿伯爵家也没有兴趣和警戒的必要。因此，没注意到雷夠斯顿伯爵家奇怪的行动。
。
「在进入安全的建筑物前不能进行秘密的详细的解说。哎呀，简单说明的話，我是想猎头雷夠斯顿伯爵家……不，不是物理上取了头的意思，是让他们迁移到我们国家而已。」

粗略地说明的班达鲁。伪装工作的解释也同样简单地说明。
。
「这是为了留在这个国家的雷夠斯顿伯爵家有缘的人们的作业。」
。
有雷夠斯顿家名的人们，以及与之相近的人，希望迁移的佣人和骑士，其家族全体已经迁移到塔洛斯海姆了。但是，与雷夠斯顿伯爵家有关系的人们还有很多留在这个国家。
。
现家主阿萨德和前家主塞尔的妻子的老家，和他们已出嫁贵族家的姐姐和妹妹们。骑士妻子的老家和佣人的亲戚。这是一个很庞大的数量。
。
「在雷夠斯顿伯爵家消失的同一天晚上，托马斯·帕克佩克伯爵消失了。这样的話，什么都不知道的人会怀疑雷夠斯顿伯爵家吧？」
。
被问到的多尔顿点头表示「确实」。
。
「由于前军务卿和现军务卿，前者是在国家失态后被下令辞职，後者卻被重用。即使被怀疑也不奇怪」
。
实际上，托马斯的立场并不是很被重用的，但从旁人看的話是不会知道的。
。
「对真实证据感兴趣的人们，会默认那个吧？」
。
「也就是说，国家的重要人物消失了，像谜一样的消失，米尔格盾国也不会说出真相，若随便泄露你的秘密的話，可能会陷入国内的恐慌，考虑到这一点是正好的安排吧。」
。
「不，会引致国内恐慌也说得太过分了吧！现任军务卿也不会发起无谋的第二次远征吧」
。
「……啊，我明白了，但为什么把雷夠斯顿家和伪装的工作联系在一起呢？难道说，为了雷夠斯顿伯爵家的亲属不被逮捕处罚的事吗？」
。
「是的。顺便详细说一下吧。之後的工作都是为了杀这家伙的善後吧」
。
因为，雷夠斯顿伯爵家的亲戚们都没注意到班达鲁的存在。若報仇後不处理好的話，库尔特们会感到难过吧。

在谈判迁移同一个时期，从灵们那裡听到阿莱斯家族和『八裂』们的所作所为所以有了杀意。
。
所以，顺便去做了这样拙劣的伪装工作。
。
是等如挥手一般同样程度的轻松。
。
当然，马其克萨尔和比尔凱恩等也会像多尔顿所说那样知道真相，但也不会在表面上表现出来吧。这是把怀疑雷夠斯顿伯爵家的角色转换到阿莱斯家族和其杀手们的作业。
。
当然，雷夠斯顿伯爵家的身影消失後，也有人会把事件联想起来。但是伯爵家的亲戚，贵族、骑士、御用商人等社会立场很强。不会因为只是怀疑便做出无理被批判的事。
。
也会有鳮旦里挑骨头的人们，但那是没有办法的事。
。
因为不能把一切都做得很完美。
。
「……嗯，应该这么说吧。你是普通地做到无法相信的事的存在，但和你相同的存在我也想到有谁和你相同。」
。
多尔顿在班达鲁的行动中，想起了施耐德。即使是普通人做不了或不敢做的事，他也会轻松地做到。
。
例如在大街上暴打杀害了孩子奴隶的贵族。他也不是个普通的家伙。
。
不过，根据班达鲁禅述事情的真相後，似乎更是高度计划性的挖墙脚的行动……现场的多尔顿惊讶的不得了。
。
「嗯，做得不好吗？」
「不要这样说。很好，而且相当明快」
。

一边那么说一边拍拍班达鲁的头表示钦佩，他没有认为班达鲁做的事是正义。因为他也做过复仇的行为。
。
而且，多尔顿也是在暗地里进行了维达的新种族保护的施耐德同伴，所以他隐瞒了自己是暗黑精灵，和他一起进行了冒险者的活动。他们在阿德帝国和米尔格盾国都是犯罪者。是被標示为恶的人们。
。
他和施耐德也有怨恨过别人，但也不随便憎恨别人。而且经常先守护着自己的正义。
。
从那个多尔顿来看，没想到班达鲁做的事特别有问题。

随便做的结果，灭了一个犯罪组织，好像制造了几十个死人……但他们是一邦犯罪者，就算被抓了也只能堕入成为犯罪奴隶的结果。就结果而言，应该认为因为杀了他们而保护了更多的生命。
。
「但是，做这个伪装工作从表面上来看，那个军务卿便是国贼，他也不能保有社会的声誉，但继续伯爵家是他孩子内的某人吧，那没所谓吗？」
。
「啊，没所谓。不管他们怎样都可以」
。
阿莱斯家族和无关系的托马斯扯上了关系，感觉到了那句「我也不会理会」的本意是，冷漠。
。
他们在托马斯死後，无论受到什么样白眼的对待都无所谓。虽然他是国贼的家族，但似乎未至要被排除。又或可能作为有贡献的伯爵家的后人而被同情，托马斯的孩子也从遗嘱中继承了伯爵家，可能会说「他是个优秀的父亲」吧，但班达鲁无论他们将来是怎样都没兴趣。喜欢怎样就怎样吧。
。
所以班达鲁自己什么都不做。
。
「哈哈！御子真的很平静啊！复仇後我还怕你成了复仇鬼，真是太过虑了。
对了，我还有話想说。哪裡有静下来慢慢谈的地方？」
。
「那么，我们到都外面去吧。对了，多尔顿先生，最後一份工作都完了。顺便说一下，我肚子饿了，可以喝一杯吗？」
。
「啊，没关系。我也渴了，我也喝一杯吧。」
。
「你也喝吗！？」
。
布拉格听到後感到吃惊，但多尔顿不知道他们为什么这么吃惊，也没有深思熟虑地回答了。

「哦，我乡里的暗黑精灵都很能喝啊。其中，我是被人叫作呑兵卫的男人啊！」

「那，那么……」
。
「外面的黑精灵，好厉害啊……」

不知为何暗黑哥布林们都对多尔顿感到敬佩，因为这些人都是比较下位的种族吗？一边这样想着一边跟在班达鲁後面。
。
之後在小巷子裡，从灵听到的犯罪者中，杀了比较殘暴的两人，班达鲁吸了一个人的血，然後再把另一个人交给多尔顿。

「我先问一下，你懂喝吗？」班达鲁听说多尔顿想喝所以问了。

多尔顿是以暗黑精灵们的名誉发誓的酒豪。当然，也不会喝血，所以会说「错了，我误以为是酒」这是理所当然的。

这个夜晚，来自米尔格盾国的雷夠斯顿伯爵家的每个人和骑士们的身影消失了，然後托马斯也失踪了。
。
之後有关的调查中，从阿莱斯家族的据点的头目和干部们的尸体一起被发现了，也确认了刻有帕尔巴克伯爵家纹的戒指的托马斯的右手，犯人是与阿莱斯家族一起被雇佣的杀手干的而结束调查。
。
由此，米尔格盾国军的重建将会更加迟缓。
。
托马斯他最近反复自问的问题，在哪裡自己错了？那个回答应该是「自己比哥哥更优秀。」，只因为如此的理由而被邪神派的吸血鬼邀请並吸引加入，而且谋杀了哥哥。
。
虽然年轻时有点优秀，但十年或二十年後也不见得是这样。而且，即使不继承帕尔巴克伯爵家的家督，也能成为哥哥的家臣，支撑伯爵家和国家兴起的道路，还可以在其他的贵族家就职而拔杆兴起的路。
。
另外，即使托马斯不成为伯爵家的家主，国家也不会灭亡，也不会世界毁灭。
。
因为优秀的自己必须继承伯爵家，这个信念加上野心深值在内心，所以便向原种吸血鬼们谋求便利。
。
那正是托马斯所犯下的最大的错误，但他自己卻没有这样的自觉。最後的结果便是这样了。

班达鲁和多尔顿对彼此间的情况进行了粗略的交流。
。
班达鲁惊讶于边界山脉的对面的状况以及认知自己的存在。但最吃惊的是雷吉恩。
。
班达鲁他们不认知《暴虐之岚》的其他成员有邪神的转生体，原种吸血鬼，还有他们比想象中保护了更多的维达新种族。
。
特别惊讶的是被选为扎克塔的勇者，与十万年前的伯德木一起在阿尔达侧战斗著的英雄神法马斯·哥德其实现在是站在维达的一侧。
。
「我讨厌被打一百次啊。嗯，我也不打算自愿被他打」，班达鲁听了多尔顿这样转述法马斯的说話，然後多尔顿继续说道。
。
「啊，考虑以後要做的事吧。之後，我们可以暂时移动，但暂时不能去那裡。也不能用雷吉恩的【转移】，因为太明目张胆地当天往返会引人怀疑的。」
。
S级冒险者队伍《暴虐之岚》，在阿德帝国皇帝马其克塞尔和邪神派的原种***尔凯恩的角度来看，是被认定为维达信徒和维护维达新种族的団体。
。
因此，通常的监视和通过无法想象的魔术道具等特殊手段进行了监视。
。
所以，如果暴露了雷吉恩的存在和能在边界山脉内外轻易【转移】的話，事情便会变得很糟糕。
。
『面对那个剑士裡尔特，只有我们就可以了，即使没能打倒，也能用【转移】逃走』
。
「……打败《光速剑》的是雷吉恩小姐吗？因为光看外表不能想像。那家伙也太可怜了。
但是，能妨害空间属性魔术【转移】的方法有很多。如石化、时间停止等封住行动的方法也很多。所以不要疏忽大意了。」
。
如果封住了雷吉恩，班达鲁就没法用【转移】逃走了。如果要分开战斗的話，他的敌人无论用什么手段都要把雷吉恩先排除。

「确实，要更加注意」
。
雷吉恩拥有不寻常的再生能力和生命力，两断後也能马上恢复，他们不存在能成为内脏和骨骼等要害部位。因为有作为人格之一的艾斯基德【僵尸统率者】的力量。
。
某种意义上，是比班达鲁更难杀的存在。
。
但是使之不能行动的方法就只是不杀便能解决。对于有经验与各种各样的人战斗的多尔顿的指摘，雷吉恩也认同这一点。
。
『原来如此，确实应该考虑在魔术和状态异常中被封住的危险。从成为这个身体之後，危机感变得薄弱了是不能否定的。
但是，希望你不要叫我做小姐』
。
『因为还有阎王、幽灵、杰克等人在呢。』
。
「……啊，那怎么称呼这家伙啊？」
「直接称呼他的名字便好了」
「啊，雷吉恩虽然只有肉的简单结构，但内心其实是很複习的。我知道了，以後小心点」
。

对雷吉恩说後，多尔顿突然拉紧了脸向班达鲁道歉。
。

好感度直接飙升。

班达鲁杀了托马斯·帕尔帕克後约过了一个月左右。在米尔格盾国还是全国大骚动吧。但是，马其克萨尔也在含涩中。

托马斯是以祖国独立为目标，此次可能是与邪神派的原种吸血鬼相关的人干的吧，不想弄脏自己的手而排除了他是好的，但也为帝国之盾的米尔格盾国军队的延迟重组而痛心。

信奉《悦命的邪神》西里酥卡卡的原种吸血鬼比尔凱恩，因为失去了有力的交易对象，但组织本身就等于半毁的状态，所以实际上也没什么损失。

《寨卡特的考验》在边界山脉外是不同的，在这裡基本上没有死者。但若挑战者太弱的話，可能会不小心就死了」

在山脉外流传著『攻略迷宫的話会成为贝武夫的後继者』这样的谣言。不过，在迷宫内有描述勇者和众神等的石碑，但还会被误解真不可思议。」

在大陆南部中的强者为了挑战试炼而聚集在一起，在大会上也有多次各国的国王进行对战的事。

高等哥布林国和高等精灵国，还有贵族奧克帝国的候选人，很多因为布基尼引起的内乱而被杀。

【灵斗士】和其他的导士等的职业都令他犹豫不决，但因为要挑战能让达露西亚复活的迷宫，所以决定选择比较有用的职业。
