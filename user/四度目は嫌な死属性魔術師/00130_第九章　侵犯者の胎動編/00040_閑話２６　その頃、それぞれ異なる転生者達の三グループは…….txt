因为是在没有肉体的状态，即使使用错了也不会致死，而且自己也被监视着，问题应该不会发生吧。她就是这麼想的……。

罗德科尔特在关注其他事情的时候，会对她们的监视变得松懈……或许。

首先对三个人而言，決定隠瞒在一瞬间看到「像野兽眼睛一样什麼的」，接下来行动之前，都不可以让罗德科尔德知道这件事。

在班达鲁的实验用迷宫度过的时候，阿沙兹三人从旧斯库拉自治区的边界回到了索伦公爵领的都城。

然後住到裡面的其中的一个旅馆後，三人聚集在房间内坐下来。
。
「到这裡来了，没有被尾随吧。有没有啊，天道？」
。
「嗯。应该有。」

【千里眼】的天道，注意到了尾随自己的存在。因此，他们回到了一个热闹的城市。
。
即使错了也不考慮摆脱随行者或者是击退他之类的事。因为从经验中知道，与《origin》时不同，在这个世界上，有很多无法摆脱的存在。

当然，在《origin》中，阿沙兹们也不是无敌的。但是，对方只是用近代兵器武装起来的军队，或者像棋子一样使用武装集团的大富豪和贪污的政治家，但并不是个人的强度。

但是在这个《拉姆达》中，在阿沙兹的感觉上，只有原始的武装都有压倒性力量的个人存在。

那个《谜之猎头魔》也是其中一人。
。
「《千里眼》也没能直接确认的傢伙，而且我什麼都没注意到。探测热源也没反应。这个世界满是怪物。」

「阿沙兹，在我的热源探知中也探测不到的魔物，一定是他的不死者。」

「包含这一点在内，都是全都是怪物。」

阿沙兹这样说着，叹了口气。自从转生到这个世界後，他多次经歷了失败。

知识了武技和无属性魔术的存在，但卻没有实战经验所以常常会疏忽大意。

但是，以失败为食粮，阿沙兹比前世更加谨慎。没有发展到被班达鲁碎魂，也许也是因为这个原因。
。
觉得阿沙兹必须停止班达鲁的理由，是因为他看到被粉碎了灵魂的【死镰】。

因为会对轮回转生系统带来很大的坏影响，而且想避免碎魂的杀人行为。

不是杀人，而是消灭灵魂。犯了什麼罪才要处以这样的大刑？有理由允许吗？他就是不认同。

为什麼不直接告诉他，而是说「停止使用死属性魔术」？那是因为即使班达鲁同意了，他也会用除粉碎以外的方法来捕捉灵魂。

不破坏灵魂就直接封印。或只破坏记忆、人格等灵魂的一部分。又或分割後限制一部分功能。

灵魂实际是怎样的构造，一部分被破坏也没关係吗？阿沙兹不知道。

罗德科尔特解释说，转生者的灵魂不会在地上徘徊，会迅速回到他的神域。

但是，从亚乱们那裡收到了『不要相信罗德科尔德』的消息後，计划就被打乱了。

能碎魂的班达鲁明显与死属性魔术相关联。因此，如果不能使用死属性魔术的話，班达鲁就无法粉碎灵魂。

「但是【比尔塞斯】……想问一下鲛岛的安危。」

「没办法。我们不可能知道萨尔亚的事情。而且，如果说出了他的名字的話，可能会被发现他是转生者，反而会使他更危险。」

萨尔亚•雷夠斯顿是【比尔塞斯】，亚乱们得知他现在和这个世界的家人一起生活着。

但是，如果阿沙兹知道了他在那裡，就不知道他会否为了萨尔亚的安全而採取什麼样的行动。

「不管怎麼样都不会想那样的事情吧。若和他一起战鬥的話，之後我们会很危险的。」

和对抗班达鲁的穆拉姆们共同战鬥，不可能。和加奈子一样，都是不能相信的。

「本来，我就不想和班达鲁战鬥。我只是想停止他的错误。

「不使用罗德科尔特的力量的方法啊……你真的有这样的方法吗？我觉得没有这样的方法。」

「《origin》封印魔法的魔法道具是使其不能使用魔法。类似的东西在这个世界上也有，但是对那傢伙来说是绝对行不通的。因为他的魔力是规格外啊。」

所以，苏菲仍很怀疑。
。

对吃惊的泰达的问题，阿沙兹回答了。

在《origin》中唯一拥有死属性的班达鲁死後，各国也在背後继续进行死属性的研究。

就是这样有用的死属性。实际上，班达鲁使用死属性的力量，便成为了一国的支配者。

因此，若果阿沙兹把可以封印死属性的技术推广到各国，认为应该不会在这个世界发生死属性的研究競争。

「……确实，若六道那傢伙转生来的时候也许会有帮助。虽然不认为那傢伙能否完成死属性的研究」

在《origin》中还活着的百人众的幹部，在村上背後操线並继续研究死属性魔术的男子，苏菲点了点头。

「是啊……我不知道是否真的能做到，但还是要试试看吧」

并不是忘记了能像马奧那样从这个大陆逃出去，便能完全断绝和班达鲁关係的选择。但是，若选择了这條路泰达仍然害怕着班达鲁的力量。
。
即使从巴恩加大陆离开，也不能逃避吗？只要在这个世界生存下去，就不得不担心他会不会改变初衷来杀自己呢。
。
对於记住这种恐怖感和不安的泰达来说，阿沙兹的提案就等同於光明。
。
「好，既然两人都接受了，那就用这个方案吧。虽然这麼说，但还没有线索……嗯，要寻找遗迹吗？……也许不能花那麼多时间」

这样点头的阿沙兹还有很讨厌的预感。自称为一国的统治者的班达鲁若继续扩大势力的話，总有一天会和阿德帝国、奧尔巴选王国发生真正的战争。而且是会卷入大陆全境的大战争。
。
（为了使其不这样做，就一定要发现能与死属性对抗的什麼方法。那傢伙如果知道其他国家拥有对死属性的对抗方法，说不定会变得慎重。
若他会从索伦公爵领撤退，在山脉中呆着的話应该会保持和平）
。
实际上，《法命神》阿尔达的势力已经开始行动了，即使班达鲁老实地在边界山脉内部呆着，如果预测没错的話，也会发展成席卷全世界的战乱……但阿沙兹仍不知道那件事。

在班达鲁在海边寻找船的时候，《origin》的雨宫宽人在自己的家裡度过了休息日。
。
与《第八引导》的战鬥，之後发生的国防部大事件之後，过了一年以上，世界形势发生了很大的变化。
。
在《第八引导》的据点中派遣特殊部队的国家和《百人众》的关係变得不好，表面上是修復了，但实际上关係还很差。
。
在合众国总统亲自辞职後的大选。政权交替，至今国防部仍在重建中。
。
在联合国，全加盟国缔结禁止死属性研究的条约的声音高涨了。宽人也打算为了缔结条约而行动，但由於某个人物的反对而无法实现，条约最终也被毁掉了。
。
「妈妈，叔叔还没来吗？」
「呵呵，博很喜欢大叔呢」
。

她与雨宫宽人结婚，从成濑成美成为了雨宫成美的她，对长子博笑了一笑。
。
「不那麼着急他也会马上来的。那傢伙今天应该是休班的」
。
一边苦笑一边对兒子说的宽人，但对会来的那句話有点忧心。
。
「爸爸，好久没见到叔叔了。爸爸也不是这样想的吗？」
「是啊。爸爸也很久没见他了。」
。

为了不表现出动摇的表情，马上改口了，不过成美似乎已经发现了。
。
那时候，内线电話告诉客人到了。
。
好久没来过雨宫家的叔叔……【阿瓦隆】六道圣，嘴角上露出了平静的笑容。
。
「好久不见，这样说真奇怪。但因为有很多工作，亦要在很多的媒体上露面。」
「……是啊。」
「但是，希望你能明白。反对那个死属性禁止研究条约，是我考虑後的结果」
。

反对条约的某个人物，是现在《百人众》的幹部们中也是代表性立场的六道本人。
。
在三波浅黄和圆藤硬弥去世的现在，对最信赖他的宽人来说是一种冲击。但是现在他也明白了。

「只在联合国的加盟国禁止的話也没有意义。不仅如此，在非加盟国和大规模的犯罪组织进行死属性的研究的时候，为了準备对抗手段也不赞成禁止全部的条约。是这样吧？」
。
「而且，我们始终只是一个组织的人。不是政治家，也不是政客。如果在联合国表决中被否决的話，我们不赞成也不会有其他结果的。
他们是不在意我们两人的。」
。
宽人和成美，对两人的話六道说「能理解我真是太好了」，笑了。
。
但事实是不一样的。六道圣自己秘密地进行死属性研究，由於条约会妨碍他所以才反对。
。
他从各国的要人、富豪以及帶领幾个转生者他们，被有实力的国家禁止进行死属性的研究是很不方便的。
。
如果条约被缔结的話，对死属性进行研究的试药、灵药、魔术道具和分析设备的交易和进出口的监视会变得严格。若计划暴露了的話，到现在为止的辛苦都会泡湯。
。
（三波、圆藤，还有天道都消失了，真是幸运啊。托您们的福，我能很容易地做我想做的事。果然，我被命运所喜爱）
。
那个浅黄转生到了异世界，硬弥在神域至今仍不由得痛苦地注视着自己，六道确信自己是特别的。
。
「这样难熬的話题就到此为止吧。成美，虽然晚了，但这是庆祝孩子的礼物。」
。
收到漂亮包装的盒子，成美微笑着说「谢谢」。宽人像原来的朋友一样接触六道。
。
「叔叔，我的份是！？」
「喂，博！现在不是说失礼的話的时候」
「啊，博的份就在这裡」
。

六道拿出一个小一点的袋子，博收到它後说「谢谢！」就跑出去了。成美说「等一下，等我道谢了後才打开袋子！」一边斥责一边追上博也。
。
「对不起，他是一名粗鲁的男孩」
「没什麼，有精神便是好事。但是名字一样呢」
「……啊，好像是这样。我与自己相似的名字的他、天宫博人没见过面，所以不太清楚」
「其实我也不记得了他的事。」
。

宽人没有见过面……正确的是，在不死化前没能见到天宫博人。

六道也是，一边想着用语言回应着记不住的天宫博人，实际上，欺骗宽人们是为了维持现在的关係，那样的演戏也很辛苦。
。
（如果只交给人型的【変型】，真正的我会忘记雨宫的『善良挚友』的演技方法。）
。
然後，作为演剧的继续，「这样说的話」改变了話题。
。
「听说出生的第二个孩子是女兒。但名字是冥……果然？」
「啊，很难取名嘛，所以我把它改成日文名字」
「是吗……在《第八引导》中，特别是布鲁托和巴巴卡亚的粉丝还有很多。你最好小心点比较好。

对了，已经接受了属性诊断了吗？」
。
在科学和魔术共存的《origin》中，知道自己的孩子属性的适性是很重要的。因此，在孩子的时候检查魔力并诊断素质，在发达的国家中是常识。
。
「不，因为还没到一岁，所以还没去检查。太幼小的話，因为魔力很弱，检查机器也检查不出来的情况很普遍。」
。
「你这样说也没错，在这个国家……检查是要满三岁吗？」
。
（因为对布鲁托的名字太在意了吧，也没可能会有死属性的素质吧。我太在意了）
。
六道这样考虑的时候，放在别的房间的婴儿床上躺着的冥，眼睑突然打开了。
。
「……」
。
露出来的眼睛，但包含应该是白色的部分，全部都是漆黑的。但是，一眨眼间又変回了普通的眼睛。
。
然後就这样闭上双眼，再次开始睡觉。
。
这小小的存在，有着异样的现象，连神都不察觉。
