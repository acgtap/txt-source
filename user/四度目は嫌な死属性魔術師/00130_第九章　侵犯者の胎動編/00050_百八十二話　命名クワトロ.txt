說著的《幽靈》……在《origin》生活的時候，包括加奈子的轉生者們都沒意識到他的存在，在他的提問中，加奈子回答說「是的，就是這樣」。

因此，制作者的寨卡特也不能斷言，有與原創相同程度的性能。

被這樣說的大古們互相看了看後，就無奈地回答了。

「在我們中擅長槍的是【超感覺】和【奧丁】，還有後來加入的村上也很擅長。還有【永恆之槍】的海藤加塔納。

『ウ゛ォォォォォォ……』
。
船上發出轟鳴聲，遇難船變成了幽靈船。吧嗒吧嗒啪嗒啪嗒地揚起破了的帆，黏著藻類和藤壺的槳敲打著海面。
。
『是大船啊，陛下！』
。
「哇，很像腿多的生物。像皮特一樣」
。
「哦，外面世界的船不僅大，還有這麼多的槳啊。」
。
利維亞公主和帕碧娜從薩姆的貨架上眺望著完成的幽靈船，鬼人國的公主奧迪華卡歡呼著。
。
帕碧娜比起看船，更有興趣看從船上溢出的無數的蟹、蝦、和類似皮特的瀨尿蝦等東西。
。
『BOSS，是加萊賽戰船，是帆船和魔導船混合一起的船啊。』
。
和帕碧娜們不同，關於大型船有一定程度了解的原帝國軍人金伯利，滿是對完成的船感到不安的表情。
。
順便一提，魔導船不僅造船時需要木匠，也需要鍊金術師，是作為魔術道具而建造的船。主要擁有它的是王侯貴族和有巨額財富的商人，還有為了在魔海化的海洋航行的冒險者。

恐怕是想嘗試從海路進入邊界山脈內側時遇難了。
。
『但是，我們也從海路進入魔海。那樣的話，沒關係嗎？』
。
「嗯，大概。船身或許會變得不穩定，但移動是沒問題的。若發生故障時就再改造一下吧」
。
如果真正發生問題而沉沒的話，乘著薩姆逃到天空就行了，所以沒問題吧。
。
「但是班達魯，船上有四個船長開始爭論了。那是你做的不死者嗎？」
。
「不，應該是自然不死化的各自的船的船長吧。因為我選擇了有很多怨念的船的零件。所以有很多很好的不死者，但也要招募船員了。」
。
「打掃的時候需要人手啊」
。
對說自己是船長的海賊船、軍船、商船和魔導船的船長進行的爭吵後，四個人也採用了。
。
在那之後，提拔了徘徊在周圍的不死者船員，看從《暴虐之嵐》得到的攻略式海圖。
。
「首先【維持鮮度】之後【除臭】和【殺菌】……貼上的貝殻類和藻類，太浪費了，之後吃掉吧」
。
雖然是不死者，卻與海洋生物共存的他們，今後將以清淨的身體重過人生。
。
「我想說現在先向魔大陸出發吧。途中可能會沖刺幾個魔海，但總會有辦法的。
靠你做先鋒了，骨人，去吧。」
。
『請交給我吧，我的主啊。來吧，終於到我們活躍的時候了！』
。
用幾根繩子和幽靈船連接起來的……曾經支配著塔洛斯海姆南大沼澤地的龍僵屍裡約，牠緊貼著回應。

『ＧＵＲＯＯＯＯ……』
。
於是，里約從鰐魚相似的頭部發出低沉的鳴叫，把尾巴和四肢動起來，開始牽動幽靈船。
。
比沼澤地更深，雖然對有著波浪的大海有些困惑，但前進卻很有力。即使是岩石的石礁，也一邊粉碎一邊前進。
。
「幽靈船本身還只是差不多階位3，但里約現在是階位9。而且還坐著骨人。如果是普通魔物的話就會讓路吧」
。
「但是，班達魯，船的名字想成怎樣了？」
。
「嗯……我想起名誇特羅號」
。
被帕碧娜問到的班達魯，暫時考慮了之後用西班牙語說了四個詞。由來自不必說，是因為用四艘不同的船的零件制造的幽靈船。
。
雖然有些像西班牙語食物的比薩的名字，但是沒關係吧。
。
『誇特羅號……好像是老板所愛的奇怪名字』
。
『金伯利先生，其他船的名字會是什麼？』
。
『大概是很娘的名字呢，像《武士鯨魚號》這樣的形容詞和形容海中生物的名字很多呢』
。
海是《水與知識的女神》佩利亞和其從屬神所司的，而且是魚和海生哺乳類獸王的區域，所以很多人在船上都知道女神和生物的名字。
。
『嘛，不會因為不同名字而受到懲罰吧。瑪莉號和伊利號啦，相似名字的船都堆積如山了，這個名字不是更好嗎？』
。
「啊，是啊。那麼請多多關照」
。
在班達魯說話的時候，被命名為誇特羅號的幽靈船，發出如同摩擦般喜悅的聲音。除了撇除是幽靈船以外，也是世界上規模最大的大型船誕生。
。
「那麼，和船員一起打掃船吧。能吃的東西要回收，之後再在船上燒烤吧」
。
在船內棲息著的海洋生物和垃圾還堆積如山。雖然已經用了一次【殺菌】，但放任不管的話也會馬上長出霉菌的。

「目標，世界上最乾淨的幽靈船」
。
「確實，如果不乾淨的話，今晚睡覺的地方也會很為難啊」
。
「小不點，你可以晚上回塔洛斯海姆啊，所以沒關係。但如果要住在這裡的話，也有薩姆啊」
。
「啊，是啊。」
。
立刻用【幽體離散】的分身去打掃了、船員們馬上幫助班達魯，接著從薩姆貨台上出來的帕碧娜和奧迪華卡出拿著甲板擦幫忙了。
。
因為是在魔海航行，所以也需要有警戒員，所以那是奧爾比亞和金伯利負責的。
。
就這樣，航海第一天，親切的船員們提供了海螺、牡蠣。也從魔物得到了甲殻類和魚等食物，大家都很開心。
。
但是在航海的第七天，天空突然陰暗起來。
。
「這就是《暴虐之嵐》施耐德們聯絡信息中的日蝕啊」
。
『陛下，是不吉利的前兆啊，可能會發生什麼事！』
。
利維亞公主望著缺黑的太陽，露出不安的表情。對於把《太陽的巨人》塔洛斯作為父母的巨人種來說，日蝕是會發生不吉利的事情的前兆。
。
『那不是只是以前的塔洛斯海姆的故事嗎？』
。
但是，就像是奧爾比亞回答的那樣，對新的塔洛斯海姆來說並不是太在意。
。
『啊，是這樣啊。是陛下的話大概沒問題』
。
但是，被說的班達魯不是望向天空，而是向大海注視著。
。
「有什麼東西……爬上來」
。
一邊這樣說著一邊伸長【魔王的觸角】，在頂端使【魔王的眼球】發動。那個渾濁的眼睛，充滿著青白色不祥的光輝。
