因此，制作者的寨卡特也不能断言，有与原创相同程度的性能。

在受到神秘的魔法少女的人物支持的同时，但加奈子并不是制作炸弹，而是进行烟花的制作。

黑色炎剂所需要的金属材料要分配合适的剂量，金属燃烧反应的火焰的颜色等都与『地球』和『origin』不同，困惑的同时也继续以烟花的形式进行了尝试。

在大古朝着的视线前方，雷吉恩保持沉默，但默默地發出了声响。

【魔像创成】技能，使各自的零件融合，再使其不死化。

。
顺便一提，魔导船不仅造船时需要木匠，也需要鍊金术师，是作为魔术道具而建造的船。主要拥有它的是王侯贵族和有巨额财富的商人，还有为了在魔海化的海洋航行的冒险者。

恐怕是想尝试从海路进入边界山脉内侧时遇难了。
。
『但是，我们也从海路进入魔海。那样的話，没关係吗？』
。
「嗯，大概。船身或许会变得不稳定，但移动是没问题的。若發生故障时就再改造一下吧」
。
如果真正發生问题而沉没的話，乘着萨姆逃到天空就行了，所以没问题吧。
。
「但是班达鲁，船上有四个船长开始争论了。那是你做的不死者吗？」
。
「不，应该是自然不死化的各自的船的船长吧。因为我选择了有很多怨念的船的零件。所以有很多很好的不死者，但也要招募船员了。」
。
「打扫的时候需要人手啊」
。
对说自己是船长的海贼船、军船、商船和魔导船的船长进行的争吵後，四个人也採用了。
。
在那之後，提拔了徘徊在周围的不死者船员，看从《暴虐之岚》得到的攻略式海图。
。
「首先【维持鲜度】之後【除臭】和【杀菌】……贴上的贝壳类和藻类，太浪费了，之後吃掉吧」
。
虽然是不死者，却与海洋生物共存的他们，今後将以清淨的身体重过人生。
。
「我想说现在先向魔大陆出發吧。途中可能会冲刺幾个魔海，但总会有办法的。
靠你做先锋了，骨人，去吧。」
。
『请交给我吧，我的主啊。来吧，终於到我们活跃的时候了！』
。
用幾根绳子和幽灵船连接起来的……曾经支配着塔洛斯海姆南大沼泽地的龙僵尸裡约，牠紧贴着回应。

『ＧＵＲＯＯＯＯ……』
。
於是，里约从鳄鱼相似的头部發出低沉的鳴叫，把尾巴和四肢动起来，开始牵动幽灵船。
。
比沼泽地更深，虽然对有着波浪的大海有些困惑，但前进却很有力。即使是岩石的石礁，也一边粉碎一边前进。
。
「幽灵船本身还只是差不多阶位3，但里约现在是阶位9。而且还坐着骨人。如果是普通魔物的話就会让路吧」
。
「但是，班达鲁，船的名字想成怎样了？」
。
「嗯……我想起名夸特罗号」
。
被帕碧娜问到的班达鲁，暂时考虑了之後用西班牙语说了四个词。由来自不必说，是因为用四艘不同的船的零件制造的幽灵船。
。
虽然有些像西班牙语食物的比萨的名字，但是没关係吧。
。
『夸特罗号……好像是老板所爱的奇怪名字』
。
『金伯利先生，其他船的名字会是什麼？』
。
『大概是很娘的名字呢，像《武士鲸鱼号》这样的形容词和形容海中生物的名字很多呢』
。
海是《水与知识的女神》佩利亚和其从属神所司的，而且是鱼和海生哺乳类兽王的区域，所以很多人在船上都知道女神和生物的名字。
。
『嘛，不会因为不同名字而受到惩罚吧。玛莉号和伊利号啦，相似名字的船都堆积如山了，这个名字不是更好吗？』
。
「啊，是啊。那麼请多多关照」
。
在班达鲁说話的时候，被命名为夸特罗号的幽灵船，發出如同摩擦般喜悦的声音。除了撇除是幽灵船以外，也是世界上规模最大的大型船诞生。
。
「那麼，和船员一起打扫船吧。能吃的东西要回收，之後再在船上烧烤吧」
。
在船内栖息着的海洋生物和垃圾还堆积如山。虽然已经用了一次【杀菌】，但放任不管的話也会马上长出霉菌的。

「目标，世界上最乾净的幽灵船」
。
「确实，如果不乾净的話，今晚睡觉的地方也会很为难啊」
。
「小不点，你可以晚上回塔洛斯海姆啊，所以没关係。但如果要住在这裡的話，也有萨姆啊」
。
「啊，是啊。」
。
立刻用【幽体离散】的分身去打扫了、船员们马上帮助班达鲁，接着从萨姆货台上出來的帕碧娜和奧迪华卡出拿着甲板擦帮忙了。
。
因为是在魔海航行，所以也需要有警戒员，所以那是奥尔比亚和金伯利负责的。
。
就这样，航海第一天，亲切的船员们提供了海螺、牡蛎。也从魔物得到了甲壳类和鱼等食物，大家都很开心。
。
但是在航海的第七天，天空突然阴暗起来。
。
「这就是《暴虐之岚》施耐德们联络信息中的日蝕啊」
。
『陛下，是不吉利的前兆啊，可能会發生什麼事！』
。
利维亚公主望着缺黑的太阳，露出不安的表情。对於把《太阳的巨人》塔洛斯作为父母的巨人种来说，日蝕是会發生不吉利的事情的前兆。
。
『那不是只是以前的塔洛斯海姆的故事吗？』
。
但是，就像是奥尔比亚回答的那样，对新的塔洛斯海姆来说並不是太在意。
。
『啊，是这样啊。是陛下的話大概没问题』
。
但是，被说的班达鲁不是望向天空，而是向大海注视着。
。
「有什麼东西……爬上来」
。
一边这样说着一边伸长【魔王的触角】，在顶端使【魔王的眼球】發动。那个浑浊的眼睛，充满着青白色不祥的光辉。
