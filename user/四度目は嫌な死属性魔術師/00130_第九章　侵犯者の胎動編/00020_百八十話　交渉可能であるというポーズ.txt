『阿沙兹君……【退魔师】，这样说明後，知道班达鲁不是全世界的敌人吧，你明白了吗？但如果你不相信，那也没什麼关係。』

被这样说的阿沙兹，老实说是半信半疑。如果可以的話，想问一下以什麼样的形式能和众神见面，然後具体地说了什麼……。
。
「我们相信！所以，我想再请你重新帮我们流亡」
。
「不要说流亡，这是移民。那有一个建议，可以问一下吗？」
。
加奈子们见到一点空隙便插話进来，而且改变了話题。为了应对她们，连班达鲁也没有多余的餘地。
。
「知道了，先相信吧。但是，即使是神也会有不同的判断。如果继续使用对世界来说是异物的死属性魔力的話，以後可能会发生不可挽回的事情。
想问问……班达鲁，你会不会想在建立不死者的王国後考虑征服世界之类的事？」
。
「啊，那当然没那回事」
。
班达鲁是这样回答的，但他在考虑为什麼阿沙兹想抹杀死亡属性的事。
。
然後推测的结果是，他的常识和宗教观不能承认不死者的存在，所以把死属性的厌恶联繫在一起，而且因为厌恶，所以认为死属性是大量杀戮武器一样危险的东西。
。
（如果是这样的話，太麻烦了。上半的推测是毫无办法的故事，下半的推测也是错的。因为，如果是我的魔力的話，即使使用的魔术是死属性以外的属性，危险度也不会改变）
。
现在班达鲁的魔力超过了五十亿。如果是这样的話，用火属性就可以在地上创造一个小太阳，用水属性的話，可以将南国変成冰原，如果用风属性的話，可以用雷填满天空，如果用地属性的話，也可以引起地壳变动。即使是光和生命属性，危险度也是一样的。
。
并且即使是无属性魔术也能穿透空间。
。
也就是说，阿沙兹说的《危险》，即使班达鲁不使用死属性魔术也不会消失。
。
（即使说明了也不会理解，所以不做吧。之後还有不死者的交配实验，所以就保持沉默吧）
。
「说起来，若要说停止使用死属性魔术。说服的不是我个人，而是要向因为没死屬性而破产的我的国家的人民们说服。」
。
塔洛斯海姆的基础设施和各种产业都依靠班达鲁的死属性魔术。

守护塔洛斯海姆的城墙和建筑物，以及在工厂裡移动的全部都是魔像。在城墙上戒备外敌的十字弓也是不死者。
。
仓库的食物不腐烂，能维持新鲜度，制造各种各样的发酵食品，为了维持卫生状态而制作的消毒液，代替路灯的鬼火，都是以死属性魔力运作的魔术道具。
。
如果那些都不动的話，国家运营会造成很大的障碍。特别是在古尔的情况下，如果使生殖能力和胎儿的生存率提高的魔术道具停止了功能，维持种族的事也会变得很难。
。
如果以阿沙兹的要求中包含的现有不死者也要排除的話，就连人口都会急剧减少。
。
即使被要求这样的事也不可能得到回应，如果是无论如何也要被逼迫的話，只能发展到战争和杀戮。
。
「说起来，要停止死属性，具体是如何？」
。
如果只是不使用魔术就很简单，但是属性的适性和生理一样。对於班达鲁来说，阿沙兹的要求等於永远不会出汗。实现是不可能的。
。
「有幾个方法。首先，伪的死亡……变成假死状态，回到罗德科尔特的神域进行其他属性的素质——」
「啊，除了涉及罗德科尔特的方法以外」
「……果然不行吗」
。

阿沙兹从罗德科尔特那裡听到了幾个方法，说明要从班达鲁里封住死属性的方法，但是被班达鲁拒绝了。
。
弄错别人的名字。对犯了这样错误的罗德科尔特，班达鲁当然不可能信任。本来要警惕杀人的神是理所当然的。
。
「那就……现在没办法了」
。
在这个世界上转生的阿沙兹，打算说服他再相信罗德科尔特。但是，从亚乱们听到加奈子的信息时就被教过了。『绝对不要相信罗德科尔特』。
。
因为罗德科尔特过于警戒班达鲁，所以想把包括《拉姆达》和《origin》在内的多个世界，从运行轮回转生的系统中分离出来的事。
。
由於其详细情况与轮回转生有关，所以亚乱们已经传达给转生到《拉姆达》的阿沙兹。
。
但是，到那时为止说好的约定就是罗德科尔特提出的，但现在卻说他是无法信任。
。
那个亚乱们的忠告，以及从见沼瞳那的信息。从这两个地方收到信息开始，阿沙兹就失去了强迫自己的想法。

（正如泰达所说，回到原点比较好。再调查一下这个世界的事，寻找不借助罗德科尔特力量而能封印死属性的方法，然後再重新说服不就能得到好的答复了吗。）
。
但是，抹杀死属性的存在和说服班达鲁的目的并没有改变。
。
那样的阿沙兹看不到班达鲁的内心，虽然他很爽快地退下了，但仍是什么話也没有说。
。
「但是，我给你忠告。我们《百人众》就不用说了。布鲁托她们《第八引导》也是被她们背叛的。最後也一定会背叛你的。」
。
「啊！？这次是对这边的事出口术吗？？」
。
「没想到因为自己不顺利而想妨碍其他人啊！」
。
突然被指向矛头的加奈子们反射性地回話。
。
『别想拉别人的脚，太难看了』
『作为大人怎能做这样的事呢』
「是啊。」
。

然後，布鲁托和见沼瞳，还有班达鲁也支援加奈子的说話。
。
难道没想到被背叛的雷吉恩们会说出那样的話吗？阿沙兹就不用说，加奈子也吓得停住了。
。
「等一下，为什麼你们站到她那边了？被他们背叛了，见沼瞳也被利用了，难度班达鲁这样都不在乎？」
。
苏菲代替嘴动不了的阿沙兹，雷吉恩等人依次回答。
。
『确实，我们《第八引导》被她们背叛了……但没什麼怨恨呢』
『因为我本来就知道他们本来就会背叛我们！结果只不过就是那样而已！』
『互相利用而已，因为能引诱你们出來然後杀掉。所以还要感谢他们呢。』
『虽然很不甘心，但却不觉得怨恨。那是我们死後的感想。』
『哈哈哈，确实是的。』
。

「诶？我很喜欢不被怨恨，但是太过意外了，你也很困惑吧？」
。
加奈子高兴地这样说雷吉恩们对她们没有怨恨。
。
『虽然被人怀疑也很为难……因为在前世的我们最终的目的是『有意义的说再见』。为了尽量不要留下遗憾，把舞台整理好，尽自己喜欢的胡闹，然後向世界说再见。所以我对她们的帮助没有怨恨』
。
理解作为巴巴亚卡声音的说明，梅丽莎表示「原来如此」点头。
。
但是，如果是这样的話，为什麼对《百人众》也怨恨呢？这样的疑问浮现在泰达脸上，瞳注意到了。

『因为她……【伊芙利特】、【千里眼】，利用我的未来视【凝视者】的力量在《百人众》裡也是一样的。在那之後，我吸毒令至自己很虚弱。
他们带着这样的我从病房裡带出来，也接受了布鲁托的治疗，也见到了杰克』
。
『所以在小瞳和杰克眼裡，村上和加奈子是丘比特啊。啊，但是要杀班达鲁的話就不允许了。』
。
雷吉恩对加奈子的好感度微妙的高，好像和这样的瞳和杰克的心理有关係。
。
顺便说一下，班达鲁的态度也是受到了雷吉恩的影响。阿沙兹在《地球》的时候就很不擅长应对这样的人。
。
『而且，杀人的理由不一样。这些傢伙是从欲望中利用我们来杀人的，但是你们是正义的吧？
如果是欲望的話，利害关係会一致的，但是你的正义与我们的正义没有一致吧』
。
用巴巴亚卡的声音那样说明，阿沙兹无奈地吐了一口气。
。
「……如果可以的話，我不想和你们撕杀。我知道了，不要再提出来了。
再见，班达鲁……」
。
这样说後，阿沙兹就转身离去了。
。
「【人偶】想杀你的时候，我只是被他们利用了。不知道能不能相信我……。
关于阿沙兹，我试着说服他，所以现在就这样吧。」
。
另一方面，泰达这样说後，苏菲就一直无言地跟着走了。
。
从黑暗的山路走远的三人虽然没有注意到，但在那个背後的斯莱格他们正在跟踪。
。
然後，过了一段时间，班达鲁嘟囔着。
。
「那个傢伙，说了『再见』吧？……但是现在结束了也不能收拾他们啊。」
。
『如果做了，就只会促进从现在开始转生来的《百人众》的危机感，进而团结起来吗？』
。
有能力的转生们，一个人或幾个人的話都没有问题，但若是有被训练有素的複数转生者缠着的話就非常麻烦了。即使现在打倒一个阿沙兹，剩下的数十人会为了避免和班达鲁硬碰硬而团结起来。
。
如果可以的話，保持在《origin》分裂的状态下是最方便的。
。
「是啊……那麼，我要说回移居的話题了」
。
同意阎魔的話的班达鲁将视线转向加奈子的时候，她锐气地举起了手。
。
「为了这件事，我一定要提出建议！但我也不知道为什麼！」
「什麼？」
「让我们成为维达的新种族、例如吸血鬼便可以了！」
。

突如其来的加奈子的提案，让班达鲁眼前一亮。

「没听过啊，加奈子！？」
「喂，这是怎麼说的！？」
。

但梅丽莎和大古两人不知道该提案是什麼，所以慌慌张张地追问加奈子。
。
「我没说过呢。只是，如果能接受这个提案的話，我们就很容易被相信了。虽然我忘了详细的理由」
。
其实也没有那麼坏的对两人说了不知所以的说明。
。
「吓了一跳。其实，我们要向你们提出同样的建议，并以此作为移民的条件。虽然不是吸血鬼也没关係」
。
『说話得很顺利就得救了』
。
听了班达鲁他们的说話，梅丽莎和大古再次感到惊讶。
