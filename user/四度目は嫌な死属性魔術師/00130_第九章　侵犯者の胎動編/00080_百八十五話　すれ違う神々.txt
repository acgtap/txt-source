出现了的女巨人将脚边逃跑的魔物的火炎树踢散，朝着夸特罗号一直前进。

在凛然的美貌中，从全身上下飘荡着神的气息。如果不注意她走路时能摇动的大地的巨大影响力……不，就算这样都能评为女神。
。
『我的名字是《月之巨人》迪安娜。《巨人神》芝诺的孩子之一。你们是维达的御子、新的勇者和他的同伴吗？』
。
站在火炎树森林边缘的巨人，迪安娜这麼问。虽然是以一种平静的语调，但奧迪华克、加奈子她们和夸特罗号等都被吓到了。

『你普通说話我也可以听到的。不可以从那样的船上大声叫喊。看，危险，请你停下来』

注意班达鲁不要掉入毒海，清了喉咙咳嗽了一下的迪安娜再次庄严地说。

『你们的事是从维达的忠实使徒施耐德他们那裡听到的。我们衷心欢迎你们的到访』

装备了裡约和夸特罗号的班达鲁，除了和雷吉恩一起回到了塔洛斯海姆的加奈子她们以外的同伴一起在迪安娜的双手中。

『如果让你们走路的話，可能要花幾个月时间』

比《暗海的邪神》古特索索特也不逊色的巨体迪安娜，每次走路时树和岩石都会破碎，魔物逃窜。即使脚底是一个怎样环境的魔境，她的脚步也没混乱。

「这不是单纯的距离问题。」

『嗯。从那傢伙……从施耐德他们那裡听到的，你们称这裡为魔大陆，与人类居住的大陆相比，是残酷的环境的魔境乱立的地方，出现的魔物只有在迷宫中出现的强而有力的个体。
但因为这裡是大陆的边缘，所以还好，只有普通的阶位10的魔物群。』
。
「阶位10是普通……群……」
。
在边界山脉内部，在迷宫外也有阶位10的魔物，如在山脉上飞行的飓风龙等，但只不过是有限的个体。

她的样子让迪安娜苦笑。

『大部分的原因是《解放的恶神》拉波比法多。那个恶神，促使魔物的阶位提升，利用牠们与其他邪神恶神霸权争夺，而且我们也来了这个地方的缘故，所以这个大陆变得比其他地方更危险了。
但是，现在那傢伙已经不再存在，高位的魔物数量正在逐渐减少。不到一百年应该就能平静下来……但在刚才的日蝕後，又再変得更加荒芜了』
。
因为趕走了在这个大陸统领其他邪神恶神的拉波比法多，所以比以前更安全。
。
『但是，这样的结果给你们带来负担。从那條废龙听了大概的情况……我想扎塔克那裡也有話要说，而且我也要道歉。对不起』

。
当然全部都是拉波比法多的责任，但被允许逃亡的迪安娜也感到了自责。

「废龙？……不要介意，有那句道歉的話就足够了。比起那个，为何在日蝕後会变得荒芜呢？」

『阿尔达动起来了……准确的说，因太阳阴沉而引起的《日蝕》，被光和生命属性管理的魔王军残党的封印松弛了』
。
看来除了《暗海的邪神》，也有其他復活的邪神恶神。
。
『巴恩加亚大陆因为在他膝下，所以《日蝕》时他也应该注意到吧……不是有其他因素转移了他的注意力的話，就是故意的。使众神施于的封印减弱了。
在与魔王军的战鬥中封印他们的时候，有多个大神的神威。但现在进行世界管理的只有阿尔达一柱』

因为那个阿尔达为了引发《日蝕》，所以从世界的管理中拔手，连封印都鬆散了。当然，现在不是所有邪神恶神的封印都解开了。

但是，在这个魔大陆和其周边海域，魔王格德拉尼斯当时『对能力低下的部下被封印不理会』的封印，之後在迪安娜她们与拉波比法多的战鬥中而被减弱的封印漏洞有好幾个。

因为它们一齐解封，所以好像很辛苦。
。
『因为日蝕的关係，所以我很忙。但很多都不是那麼强的神格，所以没什麼大事……迎接你们的任务，本来是白天和晚上的两个交替制，但是为了封印，在白天那个追著逃跑的魔龍恶神，所以消耗了体力睡着了，所以等到了晚上我才能来。
虽然不知道为什麼引起《日蝕》，但肯定是阿尔达惹出來的麻烦。』

「恐怕这就是阿尔达的目的之一吧。」

到那时一直沉默地听迪安娜的話的格法多加恩突然开口了。

「虽然除了边界山脉内部以外的巴恩加亚大陆也会直接受到伤害，但另一方面，你们也为了应对解除了封印的邪神恶神的处理而消耗。
无论人类有多少受害，阿尔达神支援的英雄们也会应对，如果同时进行信仰收集和英雄强化的話，收支就会平衡。所以结果好像没有想象中那麼坏」
。
说了这样推测的格法多加恩，迪安娜情不自禁地瞪圆了眼反问。

『你……难道是格法多加恩吗？』
「是的，你没注意到吗？」
。

看来在可怜的美少女精灵中寄宿的格法多加恩，迪安娜至今都没注意到。
。
「如果这样的話，作为我应该值得高兴。因为这个附体的完成度是连神的眼睛也能欺骗。
我会按照需要美少女的寨卡特的遗志去当的」

『……不，我只发现与一个精灵少女相似的怪东西……说起来虽然不好听，但从我看来，你们也有很多怪人』

半高贵奥克的帕碧娜，有着奇妙高阶位的鬼魂们，明明是不死者却有着理性的骨人，比谁都没有气息存在感的班达鲁。还有到刚才为止都闭嘴的格拉德加恩的存在被埋没在这样群体的角落裡。

『比什麼的存在的你们什麼都好……不，这是怎麼意思呢？』

为了掩盖困惑和惊讶，班达鲁这样说着是为了阻止他们的困惑和惊讶。除了她以外，现在的格法多加恩的身影会令很多众神惊讶，所以想省去反复说明的工夫。

「比起那个，你和《太阳的巨人》塔洛斯有什麼关係吗？在塔洛斯海姆裡留下的记录没有你的名字，所以我们的知识不知道你的存在。」

虽然也有错开話题的意思，但在听到《月之巨人》迪安娜的尊称时就开始关注着这件事。

包括我在内是神代时代创造的巨人，是巨人的始祖，成为大神的《巨人神》芝诺之子是塔洛斯。芝诺从太阳那裡得到了温暖，从月亮上拿着光，然後给予了我们双胞胎』

回答试图改变話题的班达鲁的提问时，她向班达鲁背後的利维亚公主微笑着。
。
『所以，我是所有巨人种的婶婶。在十万年前的战鬥中，我也和塔洛斯一起战鬥了……你们都因为我们力量不足而劳苦了』

成为撒旦染血幽灵的利维亚公主慌慌张张地现身，缠着黑色和橙色的火焰，深深地低头。

『不，不，说得太过了！但是……如果有一天能来塔洛斯海姆的話，我会很高兴的。』
。
『啊，一定。你变成了幽灵後，好像更像我的哥哥』
。
对像服装一样缠着火焰的利维亚公主，迪安娜微笑着说：『对你，和你的同胞赐与祝福』。

迪安娜带着班达鲁他们来到的地方，乍一看是一个只有一块可怕的岩石和熔岩池蔓延的死亡大地。
。
但是，大气中好像没有火山性气体等有毒物质。当然很热，但还是能忍受。
。
如果用吸收热能【鬼火】的話，就只是觉得温暖的程度。
。
「大概周围的熔岩不是火山活动，而是被扎塔克影响而产生的东西吧」
。
格法多加恩的推测，但班达鲁卻有点疑问。
。
「原来如此……这就是大神的力量吗？但是，应该并非仅仅如此吧」
。
「是这样的。这裡是半神域化。十万年前，扎塔克与邪神恶神融合，变成了不同的存在，所以无法回到原来的神域。所以现在使这个魔大陆的一部分神域化吧。……虽然看起来很勉强」

格法多加恩推测，因为全大陆都是因魔力而被污染的魔大陆，所以扎塔克才把自身三分之二和邪神恶神融合，正因为是扎塔克才可以做到的苦肉之策。
。
「呵呵，这是历史性的大发现」
。
汗水浸湿全身的鲁奇里亚诺，为了不失去意识，所以一边写着笔记一边这样说。
。
因为在这裡聚集了很多不寻常的存在，所以因为过于紧张而无法止住汗水。
。
像龙人女性一样的龙，《山妃龙神》提雅瑪特。有两个头的巨大的鸟，《鸟兽王》拉法兹。有似巨人巨体，披着毛皮的鬼人族始祖，相反，与人种相同的小体格卻散发着惊人存在感的魔人族始祖。
。
其他还有幾柱神，以及其眷属聚集在一起。

其中特别的是，全身散发着淡淡光芒的冒险者风格的男人。像与岩山融为一体的姿态坐着，一副严肃的表情，闭上双眼的男神。
。
……有另一种意义上不同的存在，比平常更大的飞龙一直在冒险者风格的男人旁边颤抖，那是什麼呢？
。
虽然一行人很在意那与现场不同感觉的飞龙，但那时，右半身像被黑沙铁一样的东西覆盖着，左半身露出了像黑霧一样的东西的男神……《火焰和破坏的战神》扎塔克突然睁着眼睛，打开了长牙的嘴。
。
『■■■■■■■■■～っ！』
。
就像暴风雨一样激烈，伴随着物理压力的叫喊声，班达鲁也变得僵硬了。
。
只是，完全不知道叫的意思。
。
「真对不起！？」
。
『老板，或许应该先跪着，献上祈祷，或给供品之类。』
。
「班达鲁，为什麼被人生气了？」
。
被喊声的震撼力压下，鲁奇里亚诺反射性的停止了笔记而道歉，金伯利惊慌失措，帕碧娜感到困惑。
。
『小主人，怎麼办？』
。
『是否要供品古特索索特的鱼生！ ？』
。
「冷静下来，萨利亚已经分体了，虽然大家都很惊讶。但他没有生气啊」
。
慌慌张张地让大家平静下来的班达鲁，其实他也很困惑。但习惯了异形邪神恶神的他，没有感觉到扎塔克在生气。
。
只是觉得他的声音很大而已。

但是，不知道叫喊声的意思。所以魔人族的始祖拍了拍很紧张的冒险者风格男人的肩膀。

『啊，啊。父亲……扎塔克说：「你终於来了，有勇者之魂碎片的维达之子啊」。并不是生气』
。
看来，他……英雄神似乎能明白扎塔克叫声的意思。

扎塔克再次叫喊，这次马上由法马斯翻译。看来扎塔克与《魔尘的邪神》和《邪闇的恶神》融合後，似乎无法像以前那样传达意思，所以现在是需要其他的神翻译的状态。

『欢迎……虽然这麼说，但是在这个只有岩石和熔岩的荒芜的地方，疼！？现在翻译的是扎塔克的話啊！』
。
『虽然知道这一点，但通过贵先生的嘴听到就会反射性地动手。对正在说話的父亲，还有客人对不起啊。』
。
在翻译途中打了法马斯上魔人族始祖，道歉後就从原来的位置退到後面去了。周围的众神也注视着魔人族的始祖，但没能看到有谁同情法马斯的样子。
。
从这件事中也可以看出，领导和他的关係并不是和解，只是处于休战的状态。
。
当然，如果考虑到从大约10万年前开始的维达派和阿尔达势力的战鬥和争执的話，也许能休战就已经是大让步了。

对於魔人族的始祖和其他众神来说，法马斯伤害了自己的父母兄弟，是杀了很多他们的孩子的敌人主战力之一。但因为他主要与扎塔克战鬥，所以因他的手受害的人比剩下的两个战斗系勇者少得多。
。
但是，如果他一直把自己选为勇者的扎塔克当作伙伴一起战鬥的話，就不会参加那场敌对的战鬥。这样想的話，就不能忘记过去的恩怨了。
。
凝视着这般态度的众神的扎塔克的嘴角仿佛浮现了苦笑般的笑容，也许是错觉吧。
。
『■■■■■■～！■■■、■■、■■■■！』
。
『虽然我想和你一起再次与阿尔达一战，但遗憾的是我们现在暂时不能动。在前些时间的日蝕後，魔王军残党的封印有一部分减弱了，如果放任他们在这个魔大陆的話，魔物就会持续增加，魔境会无止境地扩展到海里，如果快的話，可能会在幾年後到达巴恩加大陆。
扎塔克，保持一点节奏吧。如果一个词内包含太多意思的話，我就来不及翻译了！』
。
看来扎塔克的叫喊中包含了压缩了密度的信息，所以法马斯翻译得很辛苦。
。
「原来如此，我明白了。可能是因为我的原因，因为我破坏了封印维达的阿尔达的神威。」
。
班达鲁把这信息传达给了众神。
。
『母上復活……不，是找回了意识吗？？』
『总有一天我能再感受她的爱，这一天能实现吗？』
『クエェェェ！』
。

维达直系之子的魔人和鬼人的始祖在惊愕和喜悦中打颤，提雅玛特陶醉得尾巴像小狗般搖罢着，並张开翅膀发出尖叫。
。
扎塔克也仰天长啸……不，发出了高兴的笑声。
。
『给我带来了超出预期的喜讯。对我的妹妹、孩子们的母亲的解放表示感谢……别说那麼快，我翻译不了。
……虽然我没有值得接受的价值，但我也要感谢你』
。
他立起肩膀，折了腰，深深地低头了。
。
「不，我很乐意接受它。」
。
相对的，班达鲁也要在众神前礼貌。法马斯对他的应对感到惊讶，不由得抬起头来。
。
但是，班达鲁的兴趣还是在那头翼龙上。其他的神都很高兴，不知为何只有牠是死鱼的眼睛，而且流着泪说『已经不行了……结束了……』嘟囔着。心裡想，牠是翼龙的上位种吗？
。
（难道不是翼龙吗？有点像夏咻《蜥蜴人的管理者》）
。
在塔洛斯海姆的南面，现在是蜥蜴人和其变异种的鳄鱼人，还有斯库拉族的主要居住地的大沼泽地。曾经是boss角色的牠，但现在卻像被抛弃的小狗般眼睛地哭。
。
班达鲁终於想起来了。
。
这样想来，今年还没去露过面呢，新种大米的成长和摩羯羊农场的產量也增加了吧，下次一定要记得去大沼泽地拜访。
。
只是觉得那样。
。

「班达鲁啊，扎塔克说：『我们会越来越不能行动。如果阿尔达是像十万年前一样进攻的話怎麼办』。」
。
最後，格法多加恩代替翻译不来的法马斯。
。
「要说越来越不能行动的理由是如果自己有很大的行动的話，会被阿尔达察觉到，会很容易地发展成神和神的战鬥，就是这样的理由。」
。
「也有人认为这是与关於世界管理的问题。即使能战胜阿尔达，但之後如果不能维持属性管理的話，世界就会荒废，最坏的情况下会灭亡」
。
格法多加恩没有翻译扎塔克的話，就向班达鲁说明神所抱持的忧虑情况。
。
这个《拉姆达》的世界，由神来管理地水火风，时间和空间，以及光和生命的八个属性。如果那个管理发生故障的話，世界便会混乱，最终会崩溃。
。
现在，《法命神》阿尔达本来是光属性，但也兼任维达的生命属性管理，而被魔王毁灭了的《风与艺术之神》希薩利翁被英雄神納恩卢德纳取代了其风属性的管理，代替扎塔克的英雄神法马斯接管了火属性的管理，一边接受其他神们的辅佐一边进行着管理。
。
