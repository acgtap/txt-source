



轮回转生神罗德科尔特，听到《法命神》阿尔达的消息後充满了沉重的叹息。
。
『比起当初的预定，转生者的存在早就暴露了吗？』
。
罗德科尔特不懂得發展……为了使他的轮回转生系统的《拉姆达》人口增加，把一百个转生者送进了《拉姆达》。
。
现在在《拉姆达》世界的主神《法命神》阿尔达不能拒绝，是擅自的做法。
。
当然，知道阿尔达为顶点的众神对於异世界的知识和技术感到反感。尽管如此，坚信只要实行了的話，他们也不能怎麼样。
。
在《地球》转生到《origin》积累经验的转生者他们当然会在《拉姆达》中活用异世界的知识和技术。农业、渔业、烹饪、服饰、经济……应该会在各种各样的领域中發挥力量。
。
但是，罗德科尔特推测，如果不是很不自然地急剧增长，也不是大规模的話，阿尔达他们一定不会注意到。
。
转生者他们在《拉姆达》被父母命名，作为人种、精灵、矮人而诞生。这样的转生者会进行發明，从阿尔达他们看来，不会知道是《拉姆达》的人们独自發明的？还是异世界的知识和技术？
。
当然，炎剂和蒸汽机等，明显不是《拉姆达》的人们突然想到的不自然技术，如果转生者他们再现的話就会被神发觉了，但罗德科尔特推测，这是转生者转生很後之後才会發生的事。
。
黑色炎剂等的制法，即使用了『地球』和『origin』的制法也可能在《拉姆达》没有效果。因此，需要长期的试验错误，而且黑色炎剂获得的爆破力很可能比普通冒险者的武技和魔术还差。
。
即使做枪和爆弹，也必须把零件全部做出来。因此，许多转生者估计都会失去制作的意慾。
。
即使是蒸汽机，在發明的过程中也要有相应的资金和设备。要充分準备那个，最早也要成人後幾年。
。
然後，在那个时候，一百零一个的转生者各自都拥有着社会地位，巩固了实力，应该是无法轻易排除的存在。
。
然後即使被阿尔达发现，也已经晚了。即使想让神来煽动信徒，也会被教皇所阻挠。如果随便行动的話，不仅仅是转生者，与转生们有点关係的人都会无差别地被杀戮，所以教皇很有可能会站出来。
。
如果發生这样的事，就可以把所有的一切都全盘托出，令其事後同意就可以解決了。

当然，《拉姆达》中有危险的魔物，还有已经存在的狂热集团。但是，为了使转生者他们对抗那样的威胁，在『origin』已经积累了足够经验。
。
如果使用了给予的作弊能力和魔术的才能，就可以發挥出《拉姆达》世界的B级冒险者的力量。
。
虽然有一个叫做【物体创造】力量的贝克茉秋的转生者，但只要注意不要给他直接制造枪和炎剂便行。
。
然後经过了一百年，《拉姆达》世界的人口增加後，就能妨碍维达的新种族……总而言之在城市裡生活的兽人、巨人种、暗黑精灵等，魔人族和吸血鬼等都会被消灭而减少数量，对罗德科尔特来说是很好的结果。
。
『但实际上，从班达鲁不规则的诞生开始，设想的转生者《origin》死亡後的计划已经大不相同了。而且班达鲁拉拢了以成人身体转生的转生者中的三人，然後更被阿尔达发觉了。
但还有很多转生者还未在《拉姆达》转生呢。』
。
这样下去的話，在罗德科尔特力量的基础，属於他的轮回转生系统的《拉姆达》世界人口不会再增加，而由班达鲁所支配的帝国，将来《拉姆达》是由不死者和魔物，还有维达新种族所支配。
。
『不，比起那个，阿尔达的来信怎回復？大约每三天来一通啊！？』
。
『有的没的乱在一起……难道，鲛岛和田中……抛弃萨尔亚和吉克，浅黄都放弃了他们？』
。
『如果是村上他们的話，放弃了也没关係』
。
亚乱和泉，硬弥也对从一定时间發送来的阿尔达的信息感到压力。
。
『不如我作为使者去说明情况吧？』
。
『驳回。你到了《拉姆达》之後就会那样逃走吧，打算和班达鲁或阿沙兹直接联繫吧？』
。
『……啊，暴露了吗。』
。
事实上是有可能的，无论如何，亚乱在罗德科尔特的监察中无法到神域外，所以要试着做些什麼可能的事。
。
像是故意一样咂嘴给对方看的亚乱俯瞰着罗德科尔特，他也『确实，派遣使者的事也必须考虑吗？』在心中嘟哝着。
。
罗德科尔特进行的是，以《地球》来比喻的話，是违反规定的职务，是不正当的，在管辖範围单方面的侵害。当然『这也是为了世界的』这样的借口是行不通的。

在罗德科尔特發送转生者之前，《拉姆达》确实是从长期来看，可能会越来越衰退，但看不到马上会破灭。
。
但现在，阿尔达害怕的有一百人，可以制造出影响这个世界的毒害的炎剂和蒸汽机。
。
所以，在罗德科尔特的回復中，说实話坦白一切，还是直接道歉，或者是说谎都会觉得被糊弄。
。
但是，说谎不太有意义。可能会找与罗德科尔特有合作关系的穆拉姆集团谈一谈，但若找与信赖之间没有关係的阿沙兹和马奥的話，还有脱离穆拉姆的薰•德島，应该会直接说出自己所知道的。
。
在阿德帝国方面，诞生了能接受神谕的少年教皇，他在奥尔巴选王国方面也有其动向。现在，能介入到信徒们之中，所以能把握在场所有转生者的所在并不是很难的事。
。
『那麼，把到现在为止我的事和班达鲁是什麼人，把所有的事实告诉阿尔达他们，并请求他们打倒班达鲁。』
。
所以，罗德科尔特決定说实話，把所有的东西都说出来。但坚决不道歉。
。
『……你认为这样的事情行得通吗？如果惹众神發怒的話，你也自身难保啊。』
。
『圆藤硬弥，你有点是对的。但是，也错了很多。就算阿尔达他们怎麼生气，对我什麼都做不到，那是有理由的。』
。
罗德科尔特是拉姆达的人和动物的轮回转生神。把他从《拉姆达》中排除的話，会直接关系到轮回转生不能正确运行。
。
即使阿尔达们想要代替罗德科尔特，但罗德科尔特在《拉姆达》诞生之前，就已经开始了轮回转生。所以阿尔达势力的神是没有能代理他的工作经验。
。
《拉姆达》的人们，若没有灵魂，就会产生怨灵侵占肉体的孩子，陷入混乱之中，相反，魔王式和维达式轮回转生系统中转生的魔物和维达新种族会越来越繁荣。
。
对阿尔达来说是绝对不能接受的事态。
。
说明事件之後，罗德科尔特是这样总结的。
。
『阿尔达他们害怕我从《拉姆达》中脱离系统。所以，如果我说明事情，应该不会责难我。我与维达、利库伦、祖鲁旺不同，他们应该没有忘记，因为我不是《拉姆达》世界的神，所以逃走是可能的。
……对了，也可以告诉他们，我不是维达的伙伴。如果知道我不是敌对的大神方的話，那麼阿尔达会积极的和我的合作吧』
。
听到这件事的泉，不由得叹了口气。
。
『也就是说想隐瞒了。你已经被认定是《拉姆达》的神了，不管怎麼想也不能逃避《拉姆达》的事』
。
罗德科尔特的阴谋，只有在自己成为《拉姆达》之神之前的状态才通用。
。
他不能从《拉姆达》中逃脱……如果知道绝对不能从轮回转生系统中脱离《拉姆达》的話，阿尔达他们会对罗德科尔特作出惩罚吧。

为了完成轮回转生系统的管理运行，除此之外的思考力和行动力，将《法之桩》进行封杀，这是当然的，然後在这期间，自己和从属神偷走有关轮回转生的技巧，剥夺罗德科尔特的权能的事都可以。
。
但是，阿尔达不知道罗德科尔特已经成为了《拉姆达》的神。罗德科尔特的存在的重要性是众所周知的，在维达派基地的边界山脉内，阿尔达势力的神的眼睛没有观察到。
。
维达派……班达鲁只是把自己唾弃的神的存在推广开来的，因为不知道对罗德科尔特的影响，所以也不会进行密告。
。
『話也说得太多了。但是，为了慎重起见，让我止住你们的口吧』
。
於是，把泉他们的嘴以作为主神的权能封住，所以并不能说出真話，然後罗德科尔特从三人面前消失了，他自己走向了从不踏足的神域深處。
。
默默地送别那个的泉他们，从背後感受到了一丝的感觉，但对罗德科尔特的不满集中意识，所以没在意到。
。
在《百人众》中，她们接受应对审问的训练，所以很容易有意识地马上忘记小事情。
。
『即使重新开始，也不能单纯的解决吧。对於百人的转生者，我自己都没有办法令到他们不转生到《拉姆达》，所以没有办法。……希望转生者他们不要做炎剂和蒸汽机，亦發誓今後也不会做这样的事。』
。
神对其他的神进行的誓言，等於诅咒的束缚力。作为神的身份本身就是罗德科尔特压倒性地超过阿尔达，但是在《法命神》阿尔达的权能领域。也不能简单地违抗吧。
。
但是那样的話也没关係。如果把班达鲁放任，自己就会危险了，然後事态也会恶化。
。
对《拉姆达》是否会發展，对现在的罗德科尔特来说只不过是一个微不足道的问题。
。
『问题是我是否被要求交出……恐怕阿尔达也会犹豫吧』
。
难处理的问题封印在自己这裡，可以成为打倒班达鲁的王牌。
。
魔王格德拉尼斯的《理性》、《感情》、《本能》、《记忆》、《力》五个被撕裂的灵魂。
。
那是由贝武夫等三个勇士打倒後，被其他的阿尔达勢力的神分拆的。
。
当时的罗德科尔特，因为是危险的东西，所以很想拒绝，但『魔王復活後不仅仅是这个世界，更会走向其他的异世界……所以你不得不协助，说不定会向你掌管轮回转生的其他世界伸出侵略之手。』，所以就答应了。
。
从那之後，过了10万年以上，施於了过剩的封印，在除了罗德科尔特以外不能接近的神域深處保存下来。

罗德科尔特会不会把它当成了对抗班达鲁的王牌？当然，不会让格德拉尼斯復活，以作要挟把班达鲁打倒……若復活格德拉尼斯的灵魂，可能自己也会被毁灭。
。
无论作为神的地位怎麼高，罗德科尔特终究只是轮回转生之神，不擅长战鬥。对格德拉尼斯来说，罗德科尔特只是一个巨大的图案，是一个动作迟钝的木偶。
。
『在灵魂中，如果将《记忆》和《力》埋在一个波长相合的转生者中……但是，不符合穆拉姆、安德森和阿基拉。若以班达鲁的例子来看，若与魔王格德拉尼斯的魔力相近的話，应该可以处理』
。
也就是说，拥有死属性的魔力的存在，相反，如果能对抗死属性的魔力的力量的存在的話，也可以处理魔王灵魂的碎片。《理性》、《感情》，以及《本能》也很危险，但如果不包含格德拉尼斯的意志的《记忆》和《力》的話，应该可以与使用与魔王肉体的碎片一样。
。
但是，穆拉姆他们三人的组合中，哪一个都没有适性。即使给予了也会马上失控吧。
。
如果是【退魔师】的阿沙兹，他能把拥有属性的魔力打消，就有可能了……哎呀，不行吧。虽然比穆拉斯能忍受更长时间，但最终还是会失控吧。
。
『决定把它交给村上吗？但不能承受碎片力量的灵魂……给予无法处理的力量也没有用处。』
。
罗德科尔特将视线转向魔王灵魂的碎片，是一个扭曲的球体。是一种汇集了破碎的碎片，强行粘在一起的东西。
。
那有大小两个，在罗德科尔特面前浮着。
。
『可能会有一些前主人的灵魂碎片混在一起，如果他的自我强度夠强的話，就可以抑制住了。而且到底用不用都是他的责任』
。
然後拿了球体的罗德科尔特，为了回应阿尔达的回復，再次回到了神域的中央。

--------

《时间与术之魔神》的利库伦以老人和青年和少年的三个姿态在创建的神域中，向自己的信徒们《看》。
。
那个样子很奇怪，老人很为难的露出脸，青年苦笑着，少年高兴地笑着。
。
『信徒们的样子，很奇怪呢？』
。
在那裡有着四个头部的狮子的神，《空间和创造之神》祖鲁旺出现了这样询问。然後，利库伦鄒著眉头回答。
。
『没有什麼稀奇的事。我的信徒们很好地听我的話』
。
『那麼，那些信徒能了解我们的立场，与阿尔达神殿保持距离，但和维达新种族是真正合作吗？』
。
阿尔达势力的神正在活跃中，想急造英雄们的事，当然意识到了。这是为了打倒正在恢復气势的维达派和领导的班达鲁。
。
因此，他们对自己的信徒们讲解了要与阿尔达神殿保持距离，再努力与维达派合作。
。
能接受神谕的信徒数量很少，但是这样的人大多不是指导其他的信徒的人，所以应该不可能实现。
。
『除了雷吉恩之外，还有十多个人能接收到，我会一直关注着收到神托的信徒们的动向……如果把他们的回答翻译成语言的話，就是『不能立即回答。请让我讨论一下』的意思。』
。
『……那回復应该不是很好吧？』
。
『是那样的，兄弟』
。
接受了利库伦神托的人们，即使有程度的差别也很惊讶，总之先闭嘴吟味着神托的意思，然後收集信息和分析而行动起来。
。
自己接受的神托是正确的吗……或怀疑神的判断是不正确的，然後怀疑是否要服从。
。
怀疑这个神的話的信徒们在沉默中很可爱，但又很夸张。
。
『我不是善，不是恶。是掌管时间和术的大神。所以，供奉我的人都是探究者，亦是学生。
对他们来说，我是一个引导和先發者，同时也是研究对象。必须那样看。观察研究对象的行为，若有疑虑，就要将其进行调查，那样人才能了解神』
。
『看起来很开心……你是不是有点受虐狂的兴趣呢？不过，你的信徒大概是魔术师公会的研究者、学者、隐遁的贤者，和维达的信徒合作也不会有很大的力量』
。
对神的腔调哀叹的祖鲁旺，利库伦问了他。

『那麼，你说的可爱的信徒们是怎样的呢？』
。
『你想问吗？那听就好了，在我的信徒们裡，也给所有能接受除去雷吉恩外，能接受神托的人投下了神托。但之後还没有看到所以不知道！』
。
『……就是放下神托後置之不理吧』
。
反过来，利库伦也叹气了，祖鲁旺用四个头笑着『哈哈哈』。
。
『神不知在什麼时候能照顾到他们的感觉，那些可爱的人，也不可能会迫不及待地信仰我。我的話简直是参考意见的程度。
之後要如何行动，各有不同。如果落魄的話会好好守护的，但选择的责任首先要由本人来承担。因为人不是人偶。』
。
原本擅长诡计的神……并不担心利库伦的正确与错误，其作用是摧毁现有过去的概念和传统，并带来混乱，所以他对自己的信徒们也有距离，是自己对信徒们拉开了距离。虽然信徒们很可爱，但知道自己的性质上，若太过与信徒的距离过短的話会扰乱人们的日常生活。
。
『平常不是这样吗？你的信徒们也不会成为多数派，即使不马上和维达派合流也不会有太多影响。……把信徒送进去的事，罗德科尔特的动向是？』
。
『我已经知道了。看来，最近好像要把所有的事都说出来，谋求取得合作。我们是维达伙伴的事。似乎也打算当作伴手礼。』
。
在前一段时间，试着把空间弯曲，看看罗德科尔特的神域。这个尝试是以失败为前提实行的。实际上，因为不能隐瞒，所以被罗德科尔特的御使们发现了。
。
但是，罗德科尔特在自己的御使中也没有人望，她们什麼都没有说，就一直协助着。
。
如果罗德科尔特注意到周围的人，可能还会注意到了祖鲁旺的窥视……但他在《拉姆达》诞生的遥远以前，是一个不属於任何世界的神。对这样的他来说要注意到周围的人，等於是难题。
。
