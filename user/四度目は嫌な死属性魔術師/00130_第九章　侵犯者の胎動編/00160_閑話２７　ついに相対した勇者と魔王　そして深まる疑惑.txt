這樣下去就會死了，所以ザレス直接派遣了分靈和阿蒂法特，雖然他有素質，但實力還是平均的衛兵。實力上是會輸給D級冒險者的。因此，無法忍受降臨的ザレス的分靈，之後全身肌肉疼痛，背負了十幾個的疲勞兼骨折。
。
當然，使用不可能的ザレス本體的力量超過了原本極限。
。
雖然意識幾天後就恢復了，但是比其他的英雄候選人遲了。
。
『但是總比失去是比較好的結果吧。因為幾乎沒有時間找其他的候選人。
邊界山脈內部的樣子沒有變化嗎？』
。
『我是那樣聽聞的。但是，維達的復活果然是事實。』
。
邊界山脈內部有結界，包括阿爾達的眾神只能從遠處眺望。如果是神的眼睛的話，即使從雲上也能看到地上，但因為結界的原因，幾乎看不到。
。
……特別是在塔洛斯海姆的情況下，仔細觀察長時間的話，會有変得軟弱的御使，甚至連英靈都會產生精神異常，連看到什麼都不記得了。
。
但是，邊界山脈外部的動作可以詳細了解。
。
『獲得維達幫助的人們，在奧爾巴選王國和大陸外的維達信徒中有幾人。其中也有人說受到了神托。雖然有幾個是騙人的，但不是全部。』
。
『果然，那時御身所流的血是懲罰維達的神威被破壞的影響嗎？』
。
『雖然這是很難承認的事，但解放後也很順利……好像在順利地恢復著』
。
貝武夫和阿爾達自己也受了傷，在虛弱的基礎上施予了幾發《法之柱》的神威。但是，人類社會的維達信徒陷入了少數派。所以，若神威被打破，維達被解放了，她也不會在一百年內給予別人加護。
。
除非模仿轉生前的靈魂，用自己的血肉來給予加護。

即使有幾十年，也只能對少數人給予加護。首先將自己的健在傳達給信徒們，再次使信仰變得鞏固，所以阿爾達預計維達不會勉強自己給予加護。
。
『恐怕是班達魯吧。那傢伙把邊界山脈內的維達新種族集中起來，再創造不死者和魔物信仰維達，作為其力量來源。』
。
『榮耀的大神的一柱……現在卻和忌辰的邪神惡神一樣。果然那傢伙是瘋了吧？』
。
裘拉托斯咪著眼睛，因為這樣的問題誰都不知道。他說的話……想起了維達所選的勇者寨卡特的事，然後對阿爾達搖頭說道：『不知道』。
。
『正如你所記錄的那樣，那個人最初都有著奇特的想法，但並不邪惡。寧可保持同伴之間的和睦……但從什麼時候開始就和貝武夫他們發生衝突，但那個行動不太有效。
維達也許在他喪生時就瘋了，想起來也許是從那之前就被寨卡特迷惑了。』
。
如果是這樣的話，那便是個可怕的事。雖然不知道當時人是否有自覺，但在裘拉托斯的記錄中，他沒有秘密並巧妙地把包括維達的許多神靈和邪神惡神迷惑。
。
『主啊，在邊界山脈內有關於活動的報告。』
。
那時，出現了的御使行了一禮後開始發表報告。
。
『邊界山脈內的塔洛斯海姆里看到沒見過的魔物。而且，好像有什麼很重要的慶祝活動。除了警戒周圍的異樣魔物群外，維達的新種族和凶惡的不死者和魔物也引起了騷動，而且向天空拋出什麼東西。』
。
雖說是被結界阻撓的原因，但在不確定的報告中，阿爾達放下內心的不滿回到了現在。
。
『是不是軍隊出陣的儀式？』
。
『雖然不能確認，但也很可怕。但沒有像軍隊一樣的編成。』
。
『裘拉托斯，維達的祭日是今天嗎？』
。
『……不，應該沒什麼特別的祭日。但是，因為我沒有在邊界山脈內的維達新種族逃跑後的記錄，所以有可能是新定的節日。』
。
看來是這樣，阿爾達的意見似乎也能理解。對於眾神來說，祭祀是一件很重要的事。維持每天沒有變化的信仰是不可或缺的。
。
但是，突然想到了一件事。
。
『……扔了什麼東西嗎？關於那件事，請詳細說明一下』
。
在《拉姆達》的祭典中舉行的演出，大多是用魔術把花瓣隨風飄散，或者用戴著顏色的光點綴建築物的牆壁和夜空。但它是一種新的制作方式，可以發動神奇的魔術，向天空發出巨大的噪音。但是，御使對那種普通的事情，用了《什麼》來形容。
。
被這樣要求的御使，似乎也不能理解。
。
『那就是……他們投的是和人頭差不多大小的球體，打上後在空中爆炸，然後在夜空中發出聲音和稀有的火焰。因為是在結界外，所以不知道是什麼樣的東西。
但是，因為有人確認了以前也有幾位人類測試爆炸球體的威力，不是嗎？』
。
『看來作為兵器制造的魔術道具，現在是作為祭祀的演出而使用，阿爾達。』
。
這樣說的裘拉托斯已經整理好了，但是對他的話，阿爾達的舊記憶被刺激了。

『難道……煙花。你說是炎劑嗎？』
。
不知不覺中，阿爾達這樣嘟囔著，裘拉托斯愕然了。
。
『煙花？主啊，所謂的煙花――』
。
『不要把現在所聽到的事說出來！喂！快點去通告！』
。
擋住了想要回話的御使們，裘拉托斯用力地想停止說話，阿爾達嚴令。御使慌慌張張地說『御意！』就消失了。
。
確認了這一點之後，裘拉托斯問了阿爾達。
。
『我的主阿爾達啊，隨便說出禁忌就不像你了。難道是因為班達魯制造了炎劑嗎？……是勇者貝武夫所說，那是創造了世界上最忌諱的炎劑嗎？』
。
在阿爾達和貝武夫的教導中，大部分的異世界技術都被否定。這是為了保護世界獨有的文明發展的規則，其中也有為了保護世界本身而被指定為禁忌的技術。
。
在貝武夫的世界，破壞了森林，污染了大氣的蒸汽機，亦引發了許多的戰爭，這是污染大氣和海洋的化石燃料的研究。而且，還制造了無數的兵器，剝奪了數不清的生命的最忌諱的發明、炎劑。
。
這與勇者寨卡特想再現的核不同，只要知道制作方法，即使沒有勇者的特別力量，也可以量產，所以被視為危險的技術。
。
『但是，怎麼說班達魯也是在這個世界誕生的存在。從一開始創造炎劑是不可能的嗎？不，或許是從《寨卡特的考驗》中……可能是保護寨卡特遺產的格法多加恩保管的吧。那傢伙是看著它創造出來的』
。
裘拉托斯是這樣敘述的，但阿爾達意識到了可怕的可能性。
。
沒有預兆的行動和想法，還有特殊能力，而且有【導士】和異世界的技術。從這個組合中可以推測出的答案只有一個。
。
『難道，班達魯是從異世界傳送來的嗎？』
。
這樣想的話，如果班達魯和貝武夫是一樣的話，他的異常性也可以理解。
。
『不可能，阿爾達啊！那傢伙被記錄在這個世界上作為丹皮爾出生了！只要不是轉生……不，真的是那樣，從異世界召喚勇者的事，暴露出焦躁的羅德科爾特，對我們這樣的暴行怕我們拒絕……！』
。
雖然動搖了，但阿爾達和裘拉托斯也察覺到，只有羅德科爾特可以做到這一點。
。
確實，羅德科爾特反對勇者的召喚。但是，其理由是『召喚了自己的輪回轉生系統沒有管轄的世界的居民』，所以並不是責備異世界的居民的召喚本身。
。
自己的輪回轉生系統管轄範圍內的異世界的靈魂，到這個《拉姆達》世界裡時，恐怕是在維持前世的記憶和知識的狀態下轉生的，對羅德科爾特來說，是值得忌諱的事。
。
『我真沒想到……但這件事只是我和你之間知道。所以不要吵鬧，不要有任何行動。』
。
『遵旨。我的主』
。
還不過是推測，沒有什麼證據。但是，如果這個推測真的發生了的話，全世界的所有孩子都無法相信。
。
『羅德科爾特將轉生者送進來了嗎？還有，那傢伙是想用班達魯作什麼企圖呢？還是他其實不能控制班達魯嗎？
首先必須要確認。』
