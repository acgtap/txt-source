『获得维达帮助的人们，在奥尔巴选王国和大陆外的维达信徒中有幾人。其中也有人说受到了神托。虽然有幾个是骗人的，但不是全部。』
。
『果然，那时御身所流的血是惩罚维达的神威被破坏的影响吗？』
。
『虽然这是很难承认的事，但解放後也很顺利……好象在顺利地恢復着』
。
贝武夫和阿尔达自己也受了伤，在虚弱的基础上施予了幾發《法之柱》的神威。但是，人类社会的维达信徒陷入了少数派。所以，若神威被打破，维达被解放了，她也不会在一百年内给予别人加护。
。
除非模仿转生前的灵魂，用自己的血肉来给予加护。

即使有幾十年，也只能对少数人给予加护。首先将自己的健在传达给信徒们，再次使信仰变得巩固，所以阿尔达预计维达不会勉强自己给予加护。
。
『恐怕是班达鲁吧。那傢伙把边界山脉内的维达新种族集中起来，再创造不死者和魔物信仰维达，作为其力量来源。』
。
『荣耀的大神的一柱……现在卻和忌辰的邪神恶神一样。果然那傢伙是疯了吧？』
。
裘拉托斯咪着眼睛，因为这样的问题谁都不知道。他说的話……想起了维达所选的勇者寨卡特的事，然後对阿尔达摇头说道：『不知道』。
。
『正如你所记录的那样，那个人最初都有着奇特的想法，但并不邪恶。宁可保持同伴之间的和睦……但从什麼时候开始就和贝武夫他们發生衝突，但那个行动不太有效。
维达也许在他丧生时就疯了，想起来也许是从那之前就被寨卡特迷惑了。』
。
如果是这样的話，那便是个可怕的事。虽然不知道当时人是否有自觉，但在裘拉托斯的记录中，他没有秘密并巧妙地把包括维达的许多神灵和邪神恶神迷惑。
。
『主啊，在边界山脉内有关於活动的报告。』
。
那时，出现了的御使行了一礼後开始发表报告。
。
『边界山脉内的塔洛斯海姆里看到没见过的魔物。而且，好像有什麼很重要的庆祝活动。除了警戒周围的异样魔物群外，维达的新种族和凶恶的不死者和魔物也引起了骚动，而且向天空抛出什麼东西。』
。
虽说是被结界阻挠的原因，但在不确定的报告中，阿尔达放下内心的不满回到了现在。
。
『是不是军队出阵的仪式？』
。
『虽然不能确认，但也很可怕。但没有像军队一样的编成。』
。
『裘拉托斯，维达的祭日是今天吗？』
。
『……不，应该没什麼特别的祭日。但是，因为我没有在边界山脉内的维达新种族逃跑後的记录，所以有可能是新定的节日。』
。
看来是这样，阿尔达的意见似乎也能理解。对於众神来说，祭祀是一件很重要的事。维持每天没有变化的信仰是不可或缺的。
。
但是，突然想到了一件事。
。
『……扔了什麼东西吗？关於那件事，请详细说明一下』
。
在《拉姆达》的祭典中举行的演出，大多是用魔术把花瓣随风飘散，或者用戴着颜色的光点缀建筑物的墙壁和夜空。但它是一种新的制作方式，可以發动神奇的魔术，向天空發出巨大的噪音。但是，御使对那种普通的事情，用了《什麼》来形容。
。
被这样要求的御使，似乎也不能理解。
。
『那就是……他们投的是和人头差不多大小的球体，打上後在空中爆炸，然後在夜空中發出声音和稀有的火焰。因为是在结界外，所以不知道是什麼样的东西。
但是，因为有人确认了以前也有幾位人类测试爆炸球体的威力，不是吗？』
。
『看来作为兵器制造的魔术道具，现在是作为祭祀的演出而使用，阿尔达。』
。
这样说的裘拉托斯已经整理好了，但是对他的話，阿尔达的旧记忆被刺激了。

『难道……烟花。你说是炎剂吗？』
。
不知不觉中，阿尔达这样嘟囔着，裘拉托斯愕然了。
。
『烟花？主啊，所谓的烟花――』
。
『不要把现在所听到的事说出来！喂！快点去通告！』
。
挡住了想要回話的御使们，裘拉托斯用力地想停止说話，阿尔达严令。御使慌慌张张地说『御意！』就消失了。
。
确认了这一点之後，裘拉托斯问了阿尔达。
。
『我的主阿尔达啊，随便说出禁忌就不像你了。难道是因为班达鲁制造了炎剂吗？……是勇者贝武夫所说，那是创造了世界上最忌讳的炎剂吗？』
。
在阿尔达和贝武夫的教导中，大部分的異世界技术都被否定。这是为了保护世界独有的文明發展的规则，其中也有为了保护世界本身而被指定为禁忌的技术。
。
在贝武夫的世界，破坏了森林，污染了大气的蒸汽机，亦引發了许多的战争，这是污染大气和海洋的化石燃料的研究。而且，还制造了无数的兵器，剥夺了数不清的生命的最忌讳的发明、炎剂。
。
这与勇者寨卡特想再现的核不同，只要知道制作方法，即使没有勇者的特别力量，也可以量产，所以被视为危险的技术。
。
『但是，怎麼说班达鲁也是在这个世界诞生的存在。从一开始创造炎剂是不可能的吗？不，或许是从《寨卡特的考验》中……可能是保护寨卡特遗产的格法多加恩保管的吧。那傢伙是看着它创造出来的』
。
裘拉托斯是这样叙述的，但阿尔达意识到了可怕的可能性。
。
没有预兆的行动和想法，还有特殊能力，而且有【导士】和异世界的技术。从这个组合中可以推测出的答案只有一个。
。
『难道，班达鲁是从异世界传送来的吗？』
。
这样想的話，如果班达鲁和贝武夫是一样的話，他的异常性也可以理解。
。
『不可能，阿尔达啊！那傢伙被记录在这个世界上作为丹皮尔出生了！只要不是转生……不，真的是那样，从异世界召唤勇者的事，暴露出焦躁的罗德科尔特，对我们这样的暴行怕我们拒绝……！』
。
虽然动摇了，但阿尔达和裘拉托斯也察觉到，只有罗德科尔特可以做到这一点。
。
确实，罗德科尔特反对勇者的召唤。但是，其理由是『召唤了自己的轮回转生系统没有管辖的世界的居民』，所以并不是责备异世界的居民的召唤本身。
。
自己的轮回转生系统管辖範围内的异世界的灵魂，到这个《拉姆达》世界裡时，恐怕是在维持前世的记忆和知识的状态下转生的，对罗德科尔特来说，是值得忌讳的事。
。
『我真没想到……但这件事只是我和你之间知道。所以不要吵闹，不要有任何行动。』
。
『遵旨。我的主』
。
还不过是推测，没有什麼证据。但是，如果这个推测真的發生了的話，全世界的所有孩子都无法相信。
。
『罗德科尔特将转生者送进来了吗？还有，那傢伙是想用班达鲁作什麼企图呢？还是他其实不能控制班达鲁吗？
首先必须要确认。』
