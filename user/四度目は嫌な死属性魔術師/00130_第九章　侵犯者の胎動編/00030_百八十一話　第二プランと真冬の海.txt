因此，加奈子使用了由羅德科爾特給予的能力【維納斯】。

【維納斯】的能力，其他的轉生者都是以為是使他人入迷的能力，實際上是可以複製自己和他人的記憶和感情，亦能包含自己的對象上貼上複製的能力。

例如，事先準備好能力，在與某個特定的人物直接見面時，使其潛伏的能力發動。

加奈子在神域期間，發現羅德科爾特的注意力從自己脫離時，把能力施給自己。和班達魯直接見面時，對「維達的新種族化」的思想會活化【維納斯】的能力。

「原來如此……即使迷宮也只是背景而已，這也是其中的一環。」
。
「不，那是為了慎重起見而建的，那是我不會告訴你我在哪裡的轉移物。」
。
「哦，我明白了。」
。
「那麼請重新振作，請確認狀態。羅德科爾特的加護和幸運消失了嗎？」
。
羅德科爾特的加護和幸運，是班達魯以外的轉生者們在《origin》的時候開始得到的。
。
但是，【羅德科爾特的加護】並沒有顯示在雷吉恩的見沼瞳狀態上。
。
從那裡開始，班達魯等人想起了假設：「如果從羅德科爾特的影響下脫離的話，就不會顯示加護和幸運了吧」。
。
所以，試著先嘗試一個星期左右。
。
「【狀態】……啊，消失了」
「我也是。順便說一下，【目標雷達】也消失了。但是【宙斯盾】完好無缺」
「我也還剩下【維納斯】呢。因為很方便所以很好」
。

三人的加護和幸運都消失了，順便說一下，【目標雷達】這一億以上的死屬性魔力感知也消失了。

「這樣我們就可以移民了……啊……也許是我的錯覺，你看起來很耀眼……」
「是的，太好了。不是只有我一個人這樣感覺到」
「大古，梅麗莎，這是一時的症狀。不久就會習慣了」
。

作為引導的結果，他們似乎感受到班達魯的魅力。
。
「到現在為止沒有表情所以沒注意到……真是個美少年啊」
。
「……在你面前的是在《地球》中的同學，是和你同一年齡的人。請冷靜下來，原偶像」
。
在加奈子被血迷的時候，家裡的門和激烈的聲音一起被踢破了。
。
實際上，這個家的草原是在班達魯的地下工作室創造的實驗用迷宮的內部。可以更安全地作為居住空間和娛樂設施來活用，是為了在指定階層不產生魔物的實驗創造的東西。
。
附近有古爾們進行訓練所使用的迷宮入口，入口看起來像是偽裝往下層的樓梯。
。
是的，加奈子已經移居到塔洛斯海姆了。

---

班達魯引導的效果，已經從羅德科爾特分離出來的加奈子，之後再次提供了其他轉生者的信息後，為了成為塔洛斯海姆的國民，正在被真正的指導著……接受了法律、不死者和其他種族的交流方法等的講座後，在接受維達新種族化的儀式前，被分配到雷吉恩的部下。
。
「因為也有監視的意思，所以我明白了……但到底你們是怎樣的狀態呢？」
。
還是一如既往地把兜帽長胞穿上的雷吉恩，大古覺得很奇怪。
。
「依然很奇怪，一次聽到多個聲音……可以確認嗎？」
。
『是啊，從現在開始你就是我們的部下了。』
『我們一直這個樣子真是太拘束了！』
『模仿有骨架的人的動作，對現在的我來說是不太純熟。』
。

『那麼，從現在開始看看我們本來的形狀吧。想想在《origin》的恐怖電影和遊戲中出現的怪物，很有意思的。但如果不能忍受的話就放棄吧』
。
『為了慎重起見，注視著班達魯後背吧。放心，我準備了各種藥品。』
。
每次被雷吉恩的人格發出聲音的時候，加奈子的臉色就漸漸變差了。特別是突然回頭看的梅麗莎，發現了真相的她被班達魯看到了，她的面色已經變得蒼白如灰。
。
班達魯必須要保護的對像就像雷吉恩，所以從現在開始就必須習慣。
。
『最後要說的話，就算是悲鳴而害怕，我們也不會在意』
。
「不，我覺得我能忍住的 ――」
。
大古抽搐的悲鳴聲Miji Miji尖叫同時擴大，掩蓋了其他的聲音。
。
但意外的是，加奈子立刻從沖擊中恢復，梅麗莎和大古由班達魯的【精神侵蝕】技能恢復，似乎不需要班達魯分泌的藥劑。

就這樣暫時花了一段時間的班達魯，但事態卻有各種各樣的發展。
。
他們的引導後不久，班達魯從定期來訪的《暴虐之嵐》的約會場所留下的留言，首次知道阿德帝國的政變。
。
《法命神》阿爾達在歷史上也是如此，積極地干涉人類社會。而且，在時間點上只能認為那是為了對付班達魯他們的行動。
。
但只是讓皇帝退位，在其他表面上還沒有出現什麼奇怪的行動。
。
另一方面，班達魯現在在做什麼的話，就在剛迎來新年的冬天海邊尋找能渡海的船。
。
「我為了阿爾達做了好事吧。魔王軍殘黨的惡神和原種吸血鬼都被我消滅了，他們猜不到是我這個大冷門干的吧？」
。
「但是，班達魯被阿爾達教的偉大人們誇獎也不會高興吧？」
。
「啊，也是這樣」
。
換算成人種的話是年齡七歲的帕碧娜，和十歲半左右的班達魯相比，即使無視身體的大小，也看似比班達魯更年長。
。
帕碧娜和班達魯，從深淵噩夢馬的薩姆貨台上露出了臉，看著外面的景色。
。
『哎呀，海即使在冬天也很溫暖』
「因為邊界山脈內部的海，是在巴恩加亞大陸的南邊」
『陛下，那艘船不是很好嗎？船頭的裝飾很帥哦』
「……利維亞公主，可惜那艘船只有前半。」

在南面藍色的大海裡，受到白浪沖擊的岩礁、有一艘附有藤壺和貝類的沉船上，有『アン゛アーー！』大喊大叫的不死者船員們。

「嗯，但是無論是哪艘遇難船也是大同小異的狀態。但如果有尺寸合適的遇難船的後半的話，就用不死化緊貼吧」

《暴虐之嵐》在留言中留下了去魔大陸的海圖，所以班達魯他們正在尋找適合的船。
