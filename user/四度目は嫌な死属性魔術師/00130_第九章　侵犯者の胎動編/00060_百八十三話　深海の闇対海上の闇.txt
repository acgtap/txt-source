在《拉姆達》世界各地的各種各樣的人們都看到了日蝕。
。
眾神關注的是在阿德帝國內阿爾達勢力將增加對人類社會的影響力。
。
『阿爾達是打算把世界卷回神治時代嗎？』，祖魯旺感嘆道，進一步的連鎖反應可能會增加，維達也深深地嘆了口氣。
。
然後，羅德科爾特知道了阿爾達勢力正式的動作起來，但還猶豫著要不要和他們合作。
。
『如果與阿爾達勢力合作的話，即使班達魯和邊界山脈內的維達新種族被清除，我也會繼續被認為是『拉姆達之神』。所以不能隨便插手。
如果潛藏在選王國的穆拉姆能成功的話……雖說是這樣，但已經不能支援他們了。』
。
被《雷云之神》フィトゥン改造的哈迪姆，與羅德科爾特精神上的聯繫斷絕了。雖然可以看到記錄，但若想降下神托是無法的。
。
『不可能有神能看到我送過去的轉生者……。
但是，費圖恩，在阿爾達陣營的戰神中也是一個強大的神。將其成為附體，不是御使和分靈的狀態，我是不是沒有必要重新給予加護啊。』
。
作為神的地位，羅德科爾特是壓倒性的高，但輪回轉生之神和戰神的加護相比，後者更適合戰鬥。
。
雖然很在意給予的加護、幸運、目標雷達等現在會變成什麼樣，但比起強行介入，這樣放任不管更好。
。
『那麼，要進行支援穆拉姆們嗎？對阿沙茲的支援我沒必要出口，亞亂們會自行進行吧……但這是心理作用嗎？』
。
突然對自己的神域感到不協調的羅德科爾特，但沒有入侵者的跡象，亦沒有任何人接觸了輪回轉生系統。
。
但是還是讓人覺得不舒服，所以就把注意力轉向了《origin》。
。
『六道聖。好像是為了繼續研究不存在的事情而繼續研究死屬性，但在現在的情況下，肯定是徒勞的。果然，我還是不能阻止他。

《origin》的人們啊，無論你們做什麼都是徒勞的，不能從經驗和教訓中意識到死屬性永遠是無法獲得的嗎？而且以雨宮寬人作為踏腳石真的合適嗎？』

。。。。。。

阿德帝國的現皇帝馬其克薩爾，「果然發生了啊。」對自己的發言感到沮喪，並嘆氣了。
。
從宮殿中俯視的日蝕，如果在神的眼睛裡他是一種威脅的話，那就沒辦法了。
。
「到現在為止一直在利用信仰。雖然現在立場顛倒了，但我也知道不要抱怨……」
。
神是為人而存在的，人不必為神而存在。這是馬其克薩爾對神的認識。
。
而且，他的希望是阿德帝國的生存和繁榮。既然自稱是皇帝，那就是理所當然的了。但是，如果神殿的力量增加的話，那對他來說就危險了。
。
在神殿裡宣揚的教誨和國家的利害不一定一致。
。
馬其克薩爾也不特別喜歡制造邪惡，若善行能為這個國家發展得更好，他願意這樣做。誠實，真誠地，沒有謊言和惡習，只揮刀消滅邪惡，馬其克薩爾也是想這樣做的。
。
但是現實並不是這樣的。是由無數的陰謀構成的，別人的失敗會被譴責，或是被秘密利用，顛倒黑白，或黑白兩面都存在。
。
阿爾達教要消滅維達的新種族，但其實要抑制至不根除的程度。若只強制回收民和貴族所擁有的奴隷處死的話，只不過是令維達新種族稍為減少而已，但其結果是使經濟疲憊和受到擁有奴隷的中上流階層人仕反感而已。
。
如果反覆向各地的維達新種族隱居處遠征的話，就會浪費金錢和將兵。確實，如果遠征成功的話，可能會暫時得到奴隷和財寶，但維達新種族中隱居的地方都是一些不適合人族治理的土地，所以應該優先取得優良的領地。
。
位於索倫領的斯庫拉自治區就是一個好例子。雖然不是不能住人，但為了得到充分的領地收入，需要大規模的開拓，也需要先住的維達新種族們的智慧和技術。
。
把那樣的土地當作領地，對將兵他們來說就等同於刑罰吧。當然，馬其克薩爾的帝國對增加這樣的邊疆直轄地也不會高興。
。
因此，維達的新種族，可以拒絕當帝國的附屬國，亦不需要消耗國力去消滅他們，作為代替，雖然他們不受到國民歡迎，但作為勞動力保留下來是很好的。
。
對這個日蝕的預言的次代教皇艾麗克會考慮那個嗎？
。
「《零劍》的卡馬恩，你覺得余能保持帝位到那時候？」
。
馬其克薩爾這樣問道，但在除了皇帝以外沒有其他人的房間裡，在沒有前兆的情況下，壯年的男人出現了。他是直屬皇帝的秘密部隊，《邪碎十五劍》的司令塔《零劍》卡馬恩。
。
現在因為年紀大而從現役退出，但他自己也是原《邪碎十五劍》的正規成員。
。
「相當悲觀啊」
。
「當然，對方是被神選出來的聖人。不能以皇帝的權威壓倒。如果聽到被天光包圍的神的聲音，無論多麼堅定的人都會瘋狂跳舞的」
。
人，無論是誰都想認為自己是最特別的。自己也擁有傳說中英雄所擁有的獨特技能。誰都有過這樣的想象吧。神的加護和聲音，就是那麼甜蜜的誘惑。

然後按照神的聲音來保證「正確」，相反，就成了「惡」的存在。
。
「在維持治世的基礎上，阿爾達的教義很方便所以就利用了。如果知道現在是這樣的話，余在繼承帝位的時候，就應該把國教換成祖魯旺、利庫倫或是博坦都可以。」

「開玩笑。博提還說得過去，若所有的國民都是祖魯旺和利庫倫的信徒就太瘋狂了。不到十年帝國就會崩潰的」
。
卡馬恩一邊這樣說著一邊苦笑。順便一提，佩利亞為什麼沒參加候選國教是因為作為附屬國之一的海國海卡拉哈德是佩利亞的聖地。
。
「當然是開玩笑。那麼，一邊表示恭順邊同時繼續進行抵抗吧……但不管怎樣掙扎最多都只能撐個五年吧。如果運氣不好的話，半年就可以收拾包袱了。幸好把我的孩子藏在施耐德他們一黨裡，即使最壞的情況下也會留下血脈。」
。
如果留下血脈的話，即使可能性接近零，也許會有帝國再興起的可能性吧。
。
「如果前景那麼悲觀的話，不如托施耐德向邊界山脈內的班達魯尋求保護吧？」
。
「卡馬恩，你真是開玩笑。余會馬上被殺，然後就結束了。」
。
「他可能會考慮活用陛下，認為有利用價值呢？」
。
「不，對他們來說是多餘的……被帝國追趕乞求的皇帝沒有價值」
。
施耐德方面，起初就是潛藏的敵人，皇帝的血脈亦早已在他們那裡了……以吉克命名的男孩，皇帝的血脈。如果用心利用的話，就應該排除掉馬其克薩爾，讓年幼的吉克成為傀儡就是最好的吧。
。
……但以他的脾氣，即使不想這些事也會殺掉馬其克薩爾。
。
即使是班達魯，也不會對失去權威的皇帝感到興趣。從綁架前馬梅米公爵作為人質但不利用的觀點來看，還是會馬上處理掉馬其克薩爾的。

「不，如果是小皇子和公主們的話，他們可能會意外的接受吧。如果不是我的直系，嗯，應該想得更深入……討論一下吧」
。
馬其克薩爾是因為壽命比人種長的混血精靈，所以到現在為止繼承人是最低限度的。因為即使現在做了孩子，也會根據出生的種族比他早點死去。
。
直系孩子的數量減少，所以除了吉克外，還只剩下還是少年的小皇子一人而已。
。
如果能留下其親屬的話，即使他一個人也能倖存下來成為幕後對抗的勢力。
。
馬其克薩爾這樣點頭時，房間的門被敲了。是宰相的信息，由於日蝕發生的事必須緊急召開會議。
。
「知道了，馬上就去」
。
在馬其克薩爾這樣回答之前，卡馬恩就消失了。

。。。。。。。

得知發生了日蝕的比爾凱恩，為了抑制焦躁咬了手指。
。
「拙劣……阿爾達開始行動是很拙劣的……！」
。
他是一位供奉著十萬年巴恩加亞大陸黑暗的《悅命的邪神》西里酥卡卡的三個原種吸血鬼之一，但不是與《法命神》阿爾達有往來。
。
他是操縱和誘惑人們，在人們背後潛伏著。
。
神治時代結束後，與人治時代開始一起制作人類國家的複雜階級社會，比爾凱恩他們暗藏著黑暗的身影在人類社會中。
。
而且，比爾凱恩他們都很適應人類社會。當然，也有過多次陷入危機的事情。但都彼此一起克服了……是逃走了。
。
若果這幾年不陷入這樣的危機，他們的黑暗地位也很泰然。
。
派系的三個原種吸血鬼的關係是鬆散的，伙伴們成為了應該踢掉的競爭對手。這與班達魯的出現重疊，所以在不到十年的時間裡，蒂尼西亞，谷巴蒙就被毀滅了。
。
「在這個時機阿爾達才行動！？別開玩笑啦！打算倒行神治時代，也不能在地上降臨啊！」
。
焦躁和悔恨變成了憤怒，把自己的手指和爪咬成千塊，混血的咆哮響起。
。
发一次脾氣的話，會因為不停地破壞全部的部下和房子，所以他周圍已經沒有一個部下了。
。
空氣在咆哮上顫動，木料和地板都碎掉了掉落到房子的一樓。在無人的宅邸響起了轟鳴聲中，掉落到一樓的比爾凱恩就這樣用手臂打碎了牆壁。
。

「……不，即使罵也不會有什麼改變。」
。
因為破壞了牆壁，所以直接看到了日蝕的比爾凱恩恢復了冷靜。即使是陰沉黑暗的他，也因為久違地注視著太陽而平息了瘋狂。
。
「阿爾達不把我們放在眼裏……所以，才不會到現在才想清理我們。我們從十萬年前就開始供奉《悅命的邪神》，也增加了吸血鬼都沒理會過。
所以，阿爾達行動的原因還是班達魯啊。」
。
急切行動的阿爾達的理由，只想到這一點。班達魯只順手就把自己的組織和一族的大部分消滅了，但還有數名心腹的人存活。
。
反正阿爾達會開始行動吧，是不完全的神治時代，恐怕維持不了一千年吧。在這期間，在某個地方睡覺，等阿爾達失去力量，從地上離開時再次在人類社會背後控制吧。
。
雖然神可以直接治好人間，但畢竟人的本性是不會改變的。善良、純粹的人，只有由惡構成的惡人，除此之外的大多數都是兩者都有的。
。
利用能利用的就可以了。增加大大減少了的一族，能再次創造組織。
。
「不……如果班達魯贏了，等待千年後的我就真是失敗了。即使不是那樣，在沉睡期間被阿爾達找到的話就沒命了。
果然，不可以當洞穴裡的熊嗎？莫爾托爾！莫爾托爾在嗎？？」
。
「哦，我回來了！」
。
好像是在房子外避難的比爾凱恩的四人心腹之一，由比爾凱恩出生的貴族種吸血鬼跑到了主人身邊。
。
「如果要吃飯的話，馬上就準備好了……」
。
莫爾托爾向比平時更早平息心情的比爾凱恩問道，他用溫和的微笑回應「不，喉嚨不干」。
。
「那個，你抓住了與班達魯接觸的人們的動向吧？」
。
他被比爾凱恩分配到索倫公爵領的舊斯庫拉自治區。因此，知道了加奈子和阿沙茲他們與班達魯接觸。
。
「是的。加奈子和其他兩人與班達魯一起消失了，而阿沙茲和其他兩名現在在索倫公爵領上尋找與【魔王的碎片】封印相關的文獻和物品」
。
如果捕捉到一次信息的話，只要追尋在各公會的邪神派吸血鬼的信息網，就可以馬上調查到名字。
。
即使在組織的重組還沒完全的現在，也有可能辦到。

「【魔王的碎片】封印的文獻啊，總之是找有關的物品，但真是相當危險的模仿啊。異世界來的人。」
。
「啊！？那個人類是從異世界來的嗎？？確實，我覺得這是一個聽不慣的名字……」
。
吃驚的莫爾托爾這樣說後，比爾凱恩就開口了。
。
「恐怕是這樣吧。貝武夫他們最初也是像他們一樣有著奇妙的名字。那是……是鈴木正平嗎？
而且，他們的出身地到底是不明白的吧？」
。
「是的，目前正在調查」
。
莫爾托爾命令周圍的人進行對阿沙茲他們登記的冒險者公會周圍的城鎮和村子，但誰也不知道他們的事。父母兄弟，當然也不能找到青梅竹馬和朋友任何一個熟人。
。
那太奇怪了。
。
對冒險者公會的註冊，在哪裡的支部都可以。而且，冒險者公會的支部，在城市裡一定會有。
。
所以，立志成為冒險者的人，應該在離開出生地最近的冒險者支部進行註冊。因為去一個很遠的支部，會浪費路銀，冒著被山賊襲擊的危險性，沒什麼好處。
。
「但是，如果說那些傢伙突然出現在異世界的話，那就可以理解了。原來如此……但是，加奈子和同伴的梅麗莎是精靈，另一個的大古是人種，而且都是一般的名字？」
。
「是的。你也知道，如果他們和班達魯一起消失的話，我就追不上了。已經不能再追踪他們的身影。
比起那個，他們相繼與班達魯接觸，果然他也是從異世界來的嗎？不，在異世界裡應該只有人種。那為什麼……？」
。
雖然班達魯被懷疑是從異世界來的存在的比爾凱恩，但他也沒有意識到，有神將記憶和人格轉生到這個世界上。雖然知道輪回轉生概念的常識，但也不知道能這樣把前世的記憶和人格轉生到現世界來。
。
「嗯，也要調查他們才知道他們會是哪一個神召喚的。再加上各種猜測。所以調查的進展情況是？」
。
被問到的莫爾托爾震動著肩膀強行平伏，然後把聲音擠出來。
。
「那就是……他們的一個人，泰達奇妙地敏銳，無法接近。我打算通過觀察與看起來像是支援者的角式與其接觸……」
。
「原來如此。也許是從神賜予的力量吧……那麼，用平常不同的方法試探一下吧。從遠處控制的老鼠、鳥、蝙蝠等使魔開始……不，乾脆把它們放在他們的地下室裡使役吧。
或者，在被知道的情況下反覆探索，觀察那個叫泰達的人種。反覆幾次的話，就知道那個泰達為什麼會注意到這邊的人了。」
。
「嗯，主人說的什是。」
。
沒有失望的樣子的比爾凱恩，莫爾托爾鬆了一口氣，但還是低頭著。但是，當抬起頭的他看到的是長著鈎爪的手指。

「接下來的報告，要回應我的期待。好嗎？」
。
比爾凱恩用五根手指重覆著像折斷什麼的動作一邊這樣說，莫爾托爾以半悲鳴說道「御意！」，就逃跑似的離開了。
。
「如果那個叫阿沙茲的人是與班達魯敵對的話，那傢伙也會成為談判的材料。關於【魔王的碎片】的調查，如果是為了對抗班達魯而進行的手段，那就是最理想的」
。
比爾凱恩賭上了這一剎那的神治時代，並賭上阿爾達會勝利。在此基礎上，想和阿爾達建立休戰協定，直到班達魯死去。然後，將再次君臨社會的黑暗面為目標。
。
即使暗黑精靈是母親，丹皮爾也不會長生不老吧。雖然能活幾千年，但還是活不了一萬年吧。
。
如果是沒有班達魯的邊界山脈內的塔洛斯海姆就沒什麼大威脅了，所以也應該會有自己重臨的時機。那樣考慮的比爾凱恩，還不知道班達魯能使用【年輕化】的術。

。。。。。。。

被封印在巴恩加亞大陸南部深海的《暗海的邪神》古特索索特發出了瘋狂的聲音。
