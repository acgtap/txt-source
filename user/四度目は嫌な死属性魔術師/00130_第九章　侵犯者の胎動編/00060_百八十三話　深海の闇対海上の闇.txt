在《拉姆达》世界各地的各种各样的人们都看到了日蝕。
。
众神关注的是在阿德帝国内阿尔达势力将增加对人类社会的影响力。
。
『阿尔达是打算把世界卷回神治时代吗？』，祖鲁旺感叹道，进一步的连锁反应可能会增加，维达也深深地叹了口气。
。
然後，罗德科尔特知道了阿尔达势力正式的动作起来，但还犹豫着要不要和他们合作。
。
『如果与阿尔达势力合作的話，即使班达鲁和边界山脉内的维达新种族被清除，我也会继续被认为是『拉姆达之神』。所以不能随便插手。
如果潜藏在选王国的穆拉姆能成功的話……虽说是这样，但已经不能支援他们了。』
。
被《雷云之神》フィトゥン改造的哈迪姆，与罗德科尔特精神上的联繫断绝了。虽然可以看到记录，但若想降下神托是无法的。
。
『不可能有神能看到我送过去的转生者……。
但是，费图恩，在阿尔达阵营的战神中也是一个强大的神。将其成为附体，不是御使和分灵的状态，我是不是没有必要重新给予加护啊。』
。
作为神的地位，罗德科尔特是压倒性的高，但轮回转生之神和战神的加护相比，後者更适合战鬥。
。
虽然很在意给予的加护、幸运、目标雷达等现在会变成什麼样，但比起强行介入，这样放任不管更好。
。
『那麼，要进行支援穆拉姆们吗？对阿沙兹的支援我没必要出口，亚乱们会自行进行吧……但这是心理作用吗？』
。
突然对自己的神域感到不协调的罗德科尔特，但没有入侵者的迹象，亦没有任何人接触了轮回转生系统。
。
但是还是让人觉得不舒服，所以就把注意力转向了《origin》。
。
『六道圣。好像是为了继续研究不存在的事情而继续研究死属性，但在现在的情况下，肯定是徒劳的。果然，我还是不能阻止他。

《origin》的人们啊，无论你们做什麼都是徒劳的，不能从经验和教训中意识到死属性永远是无法获得的吗？而且以雨宫宽人作为踏腳石真的合适吗？』

。。。。。。

阿德帝国的现皇帝马其克萨尔，「果然發生了啊。」对自己的發言感到沮丧，并叹气了。
。
从宫殿中俯视的日蝕，如果在神的眼睛裡他是一种威胁的話，那就没办法了。
。
「到现在为止一直在利用信仰。虽然现在立场颠倒了，但我也知道不要抱怨……」
。
神是为人而存在的，人不必为神而存在。这是马其克萨尔对神的认识。
。
而且，他的希望是阿德帝国的生存和繁荣。既然自称是皇帝，那就是理所当然的了。但是，如果神殿的力量增加的話，那对他来说就危险了。
。
在神殿裡宣扬的教诲和国家的利害不一定一致。
。
马其克萨尔也不特别喜欢制造邪恶，若善行能为这个国家發展得更好，他愿意这样做。诚实，真诚地，没有谎言和恶习，只挥刀消灭邪恶，马其克萨尔也是想这样做的。
。
但是现实并不是这样的。是由无数的阴谋构成的，别人的失败会被谴责，或是被秘密利用，颠倒黑白，或黑白两面都存在。
。
阿尔达教要消灭维达的新种族，但其实要抑制至不根除的程度。若只强制回收民和贵族所拥有的奴隷处死的話，只不过是令维达新种族稍为减少而已，但其结果是使经济疲惫和受到拥有奴隷的中上流階層人仕反感而已。
。
如果反复向各地的维达新种族隐居处远征的話，就会浪费金钱和将兵。确实，如果远征成功的話，可能会暂时得到奴隷和财宝，但维达新种族中隐居的地方都是一些不适合人族治理的土地，所以应该優先取得優良的领地。
。
位於索伦领的斯库拉自治区就是一个好例子。虽然不是不能住人，但为了得到充分的领地收入，需要大规模的开拓，也需要先住的维达新种族们的智慧和技术。
。
把那样的土地当作领地，对将兵他们来说就等同於刑罚吧。当然，马其克萨尔的帝国对增加这样的边疆直辖地也不会高兴。
。
因此，维达的新种族，可以拒绝当帝国的附属国，亦不需要消耗国力去消灭他们，作为代替，虽然他们不受到国民欢迎，但作为劳动力保留下来是很好的。
。
对这个日蝕的预言的次代教皇艾丽克会考虑那个吗？
。
「《零剑》的卡马恩，你觉得余能保持帝位到那时候？」
。
马其克萨尔这样问道，但在除了皇帝以外没有其他人的房间裡，在没有前兆的情况下，壮年的男人出现了。他是直属皇帝的秘密部队，《邪碎十五剑》的司令塔《零剑》卡马恩。
。
现在因为年纪大而从现役退出，但他自己也是原《邪碎十五剑》的正规成员。
。
「相当悲观啊」
。
「当然，对方是被神选出来的圣人。不能以皇帝的权威压倒。如果听到被天光包围的神的声音，无论多麼坚定的人都会疯狂跳舞的」
。
人，无论是谁都想认为自己是最特别的。自己也拥有传说中英雄所拥有的独特技能。谁都有过这样的想象吧。神的加护和声音，就是那麼甜蜜的诱惑。

然後按照神的声音来保证「正确」，相反，就成了「恶」的存在。
。
「在维持治世的基础上，阿尔达的教义很方便所以就利用了。如果知道现在是这样的話，余在继承帝位的时候，就应该把国教换成祖鲁旺、利库伦或是博坦都可以。」

「开玩笑。博提还说得过去，若所有的国民都是祖鲁旺和利库伦的信徒就太疯狂了。不到十年帝国就会崩溃的」
。
卡马恩一边这样说着一边苦笑。顺便一提，佩利亚为什麼没参加候选国教是因为作为附属国之一的海国海卡拉哈德是佩利亚的圣地。
。
「当然是开玩笑。那麼，一边表示恭顺边同时继续进行抵抗吧……但不管怎样挣扎最多都只能撑个五年吧。如果运气不好的話，半年就可以收拾包袱了。幸好把我的孩子藏在施耐德他们一党里，即使最坏的情况下也会留下血脈。」
。
如果留下血脈的話，即使可能性接近零，也许会有帝国再兴起的可能性吧。
。
「如果前景那麼悲观的話，不如托施耐德向边界山脉内的班达鲁寻求保护吧？」
。
「卡马恩，你真是开玩笑。余会马上被杀，然後就结束了。」
。
「他可能会考虑活用陛下，认为有利用价值呢？」
。
「不，对他们来说是多余的……被帝国追赶乞求的皇帝没有价值」
。
施耐德方面，起初就是潜藏的敌人，皇帝的血脈亦早已在他们那裡了……以吉克命名的男孩，皇帝的血脈。如果用心利用的話，就应该排除掉马其克萨尔，让年幼的吉克成为傀儡就是最好的吧。
。
……但以他的脾气，即使不想这些事也会杀掉马其克萨尔。
。
即使是班达鲁，也不会对失去权威的皇帝感到兴趣。从绑架前马梅米公爵作为人质但不利用的观点来看，还是会马上处理掉马其克萨尔的。

「不，如果是小皇子和公主们的話，他们可能会意外的接受吧。如果不是我的直系，嗯，应该想得更深入……讨论一下吧」
。
马其克萨尔是因为寿命比人种长的混血精灵，所以到现在为止继承人是最低限度的。因为即使现在做了孩子，也会根據出生的种族比他早点死去。
。
直系孩子的数量减少，所以除了吉克外，还只剩下还是少年的小皇子一人而已。
。
如果能留下其亲屬的話，即使他一个人也能倖存下来成为幕後对抗的勢力。
。
马其克萨尔这样点头时，房间的门被敲了。是宰相的信息，由於日蝕發生的事必须紧急召开会议。
。
「知道了，马上就去」
。
在马其克萨尔这样回答之前，卡马恩就消失了。

。。。。。。。

得知發生了日蝕的比尔凱恩，为了抑制焦躁咬了手指。
。
「拙劣……阿尔达开始行动是很拙劣的……！」
。
他是一位供奉着十万年巴恩加亚大陆黑暗的《悦命的邪神》西里酥卡卡的三个原种吸血鬼之一，但不是与《法命神》阿尔达有往来。
。
他是操纵和诱惑人们，在人们背後潜伏着。
。
神治时代结束後，与人治时代开始一起制作人类国家的複杂阶级社会，比尔凱恩他们暗藏着黑暗的身影在人类社会中。
。
而且，比尔凱恩他们都很适应人类社会。当然，也有过多次陷入危机的事情。但都彼此一起克服了……是逃走了。
。
若果这幾年不陷入这样的危机，他们的黑暗地位也很泰然。
。
派系的三个原种吸血鬼的关係是鬆散的，伙伴们成为了应该踢掉的競争对手。这与班达鲁的出现重叠，所以在不到十年的时间裡，蒂尼西亚，谷巴蒙就被毁灭了。
。
「在这个时机阿尔达才行动！？别开玩笑啦！打算倒行神治时代，也不能在地上降临啊！」
。
焦躁和悔恨变成了愤怒，把自己的手指和爪咬成千块，混血的咆哮响起。
。
发一次脾气的話，会因为不停地破坏全部的部下和房子，所以他周围已经没有一个部下了。
。
空气在咆哮上颤动，木料和地板都碎掉了掉落到房子的一楼。在无人的宅邸响起了轰鸣声中，掉落到一楼的比尔凱恩就这样用手臂打碎了墙壁。
。

「……不，即使骂也不会有什麼改变。」
。
因为破坏了墙壁，所以直接看到了日蝕的比尔凱恩恢復了冷静。即使是阴沉黑暗的他，也因为久违地注视着太阳而平息了疯狂。
。
「阿尔达不把我们放在眼裏……所以，才不会到现在才想清理我们。我们从十万年前就开始供奉《悦命的邪神》，也增加了吸血鬼都没理会过。
所以，阿尔达行动的原因还是班达鲁啊。」
。
急切行动的阿尔达的理由，只想到这一点。班达鲁只顺手就把自己的组织和一族的大部分消灭了，但还有数名心腹的人存活。
。
反正阿尔达会开始行动吧，是不完全的神治时代，恐怕维持不了一千年吧。在这期间，在某个地方睡觉，等阿尔达失去力量，从地上离开时再次在人类社会背後控制吧。
。
虽然神可以直接治好人间，但毕竟人的本性是不会改变的。善良、纯粹的人，只有由恶构成的恶人，除此之外的大多数都是两者都有的。
。
利用能利用的就可以了。增加大大减少了的一族，能再次创造组织。
。
「不……如果班达鲁赢了，等待千年後的我就真是失败了。即使不是那样，在沉睡期间被阿尔达找到的話就没命了。
果然，不可以当洞穴裡的熊吗？莫尔托尔！莫尔托尔在吗？？」
。
「哦，我回来了！」
。
好像是在房子外避难的比尔凱恩的四人心腹之一，由比尔凱恩出生的贵族种吸血鬼跑到了主人身边。
。
「如果要吃饭的話，马上就準备好了……」
。
莫尔托尔向比平时更早平息心情的比尔凱恩问道，他用温和的微笑回应「不，喉咙不干」。
。
「那个，你抓住了与班达鲁接触的人们的动向吧？」
。
他被比尔凱恩分配到索伦公爵领的旧斯库拉自治区。因此，知道了加奈子和阿沙兹他们与班达鲁接触。
。
「是的。加奈子和其他两人与班达鲁一起消失了，而阿沙兹和其他两名现在在索伦公爵领上寻找与【魔王的碎片】封印相关的文献和物品」
。
如果捕捉到一次信息的話，只要追寻在各公会的邪神派吸血鬼的信息网，就可以马上调查到名字。
。
即使在组织的重组还没完全的现在，也有可能办到。

「【魔王的碎片】封印的文献啊，总之是找有关的物品，但真是相当危险的模仿啊。异世界来的人。」
。
「啊！？那个人类是从异世界来的吗？？确实，我觉得这是一个听不惯的名字……」
。
吃惊的莫尔托尔这样说後，比尔凱恩就开口了。
。
「恐怕是这样吧。贝武夫他们最初也是像他们一样有着奇妙的名字。那是……是铃木正平吗？
而且，他们的出身地到底是不明白的吧？」
。
「是的，目前正在调查」
。
莫尔托尔命令周围的人进行对阿沙兹他们登记的冒险者公会周围的城镇和村子，但谁也不知道他们的事。父母兄弟，当然也不能找到青梅竹马和朋友任何一个熟人。
。
那太奇怪了。
。
对冒险者公会的注册，在哪裡的支部都可以。而且，冒险者公会的支部，在城市裡一定会有。
。
所以，立志成为冒险者的人，应该在离开出生地最近的冒险者支部进行注册。因为去一个很远的支部，会浪费路银，冒着被山贼袭击的危险性，没什麼好处。
。
「但是，如果说那些傢伙突然出现在异世界的話，那就可以理解了。原来如此……但是，加奈子和同伴的梅丽莎是精灵，另一个的大古是人种，而且都是一般的名字？」
。
「是的。你也知道，如果他们和班达鲁一起消失的話，我就追不上了。已经不能再追踪他们的身影。
比起那个，他们相继与班达鲁接触，果然他也是从异世界来的吗？不，在异世界裡应该只有人种。那为什麼……？」
。
虽然班达鲁被怀疑是从异世界来的存在的比尔凱恩，但他也没有意识到，有神将记忆和人格转生到这个世界上。虽然知道轮回转生概念的常识，但也不知道能这样把前世的记忆和人格转生到现世界来。
。
「嗯，也要调查他们才知道他们会是哪一个神召唤的。再加上各种猜测。所以调查的进展情况是？」
。
被问到的莫尔托尔震动着肩膀强行平伏，然後把声音挤出来。
。
「那就是……他们的一个人，泰达奇妙地敏锐，无法接近。我打算通过观察与看起来像是支援者的角式与其接触……」
。
「原来如此。也许是从神赐予的力量吧……那麼，用平常不同的方法试探一下吧。从远处控制的老鼠、鸟、蝙蝠等使魔开始……不，乾脆把它们放在他们的地下室裡使役吧。
或者，在被知道的情况下反覆探索，观察那个叫泰达的人种。反复幾次的話，就知道那个泰达为什麼会注意到这边的人了。」
。
「嗯，主人说的什是。」
。
没有失望的样子的比尔凱恩，莫尔托尔松了一口气，但还是低头着。但是，当抬起头的他看到的是长着钩爪的手指。

「接下来的报告，要回应我的期待。好吗？」
。
比尔凱恩用五根手指重覆着像折断什麼的动作一边这样说，莫尔托尔以半悲鳴说道「御意！」，就逃跑似的离开了。
。
「如果那个叫阿沙兹的人是与班达鲁敌对的話，那傢伙也会成为谈判的材料。关於【魔王的碎片】的调查，如果是为了对抗班达鲁而进行的手段，那就是最理想的」
。
比尔凱恩赌上了这一刹那的神治时代，并赌上阿尔达会胜利。在此基础上，想和阿尔达建立休战协定，直到班达鲁死去。然後，将再次君临社会的黑暗面为目标。
。
即使暗黑精灵是母亲，丹皮尔也不会长生不老吧。虽然能活幾千年，但还是活不了一万年吧。
。
如果是没有班达鲁的边界山脈内的塔洛斯海姆就没什麼大威胁了，所以也应该会有自己重临的时机。那样考虑的比尔凱恩，还不知道班达鲁能使用【年轻化】的术。

。。。。。。。

被封印在巴恩加亚大陆南部深海的《暗海的邪神》古特索索特發出了疯狂的声音。
。
『愚蠢的众神！虽然不知道为什麼会發生日蝕，但我要感谢你！』
。
这种姿态，在邪神恶神中可以说是比较简单的。在鱼头上长着拔出牙齿的嘴，被鳞覆盖的四肢长着的尾巴。虽然细部不同，但基本上和半鱼人差不多。
。
只不过大小是超过城池，能抓住大型帆船的巨型。
。
『托日蝕的福，大海变成了无星的黑暗！没有被神、御使、英灵的监视，在封印变弱的现在，没有任何东西能束缚我！集合吧，我的眷属们！』
。
古特索索特的呼喊响彻了冰冷的深海，从黑暗中渐渐出现了他的眷属。
。
「Gyububu！」
「Gyoppabugyuge」
。

这是被称为神的古特索索特的咒语，一个个骑着像鲨鱼和深海鱼的魔物半人鱼出现了。
。
古特索索特是创造出半人鱼的邪神的一柱，也特别与《海之神》特里斯坦激烈競争的邪神。
。

重获自由的古特索索特首先想把仇恨发泄出来，首先是美人鱼，那是特里斯坦的孩子。
