「而且，原来的我们也不是贤者，亦不懂執政。」
。
「到现在为止一次也没有王的存在。」
。
在这个宴会中聚集的众人都是「领袖角式」，长或町长等职务的人是从一开始就不存在的。
。
「我们是按照众神的方针运营城市的。当然并不是所有的指示，但与整体城市相关的事情都会实行。」
。
而且这些城市是超越边界山脉内部国家的宗教国家……不，甚至可以说是神治国家。
。
在边界山脉内部的各国会任命王和女王，以国家的守护神为首治理国家。
。
但是守护这个城市的守护神他们，至今仍在地上继续存在着。这是因为扎塔克周围变成了他的神域。因此，只要能忍耐酷热的話，谁都可以去拜访神並直接语言交流。
。
因此，没有必要任命王等代表者，对於镇上的人们来说，为政者和神是同一存在的状态。
。
「因为神说没关係，所以这样就好了。」
。
是这麼说的。
。
「被那些神信赖的师傅，已经是值得信任的存在了。」
。
『恭喜，小主人，国土和国民增加了！』
。
『能很好地交流呢，少爷』
。
对这样说的鲁奇里阿诺和祝福的萨姆他们。实际上，班达鲁也没有拒绝的理由。
。
本来预定了各种各样的援助，但是从塔洛斯海姆离开後，使用迷宫的【转移】和雷吉恩和格法多加恩的能力，就可以瞬间移动。
。
而且这些城市并没有严重的问题和负债。亦同样是维达新种族的国家，初时对骨人他们等不死者很是惊讶，但很快就习惯了。



「那麼，把这些城市的居民作为我的国家的国民来迎接。彼此彼此，请多关照」
。
就这样，塔洛斯海姆的国土和国民增加了。

---

然後在回到回去塔洛斯海姆的幾天前，班达鲁的身影再次出现在扎塔克他们的众神前。
。
『果然小妾还是不能给予加护。即使是这样，但小妾在龙神中，也是最顶级的。』
。
『要是提雅玛特也不行的話，我也不行的了。最差的考虑是，如果不是大神等级就不行』
。
想要加护的提雅玛特和和迪安娜不可能，所以很遗憾地离开了。
。
『■■■■■■！』
。
取而代之的扎塔克發出声音，挥舞着手臂。
。
《获得了【扎塔克的加护】！》
。
脑内广播传来了，告知接受了扎塔克的加护。
。
『我的加护你能接受到吗？』
。
然後，到现在为止，只认知到尖叫或呻吟声的扎塔克的说話，现在就能理解了。虽然还夹杂着像噪音一样的东西，很难完全听懂，但还可以理解语言的意思。
。
『我～扎塔克～』
。
「啊，不，不。加护接受了。後面的話也大致明白了」
。
『哦，真的吗？？至今为止，有好些人都接受了扎塔克的加护。但没人能听懂他的说話呢。』
。
对惊讶的众神，法马斯说『那麼，现在是时候说了。』。
。
『我想你们也从施耐德那听到，但再次拜托了。对不起，我想你打我，直到我们和阿尔达的战爭结束。』
。
因为没有翻译的需要，但大致理解上是这样的，扎塔克是这麼说的。
。
「如果想战鬥的話就打他一百次，他不能拒绝」
。
『告诉我为什麼啊！我还不想消失』
。
生前是勇者的英雄神法马斯是比古特索索索特更高等的神。但也没有信心不会被毁灭。
。
因为只有防御而不能回避，如果继续接受班达鲁认真的攻击的話就危险了。因为他如果认真的話，就可以用【死炮】和【虚炮】进行一百次攻击。魔力反动使肉体没有恢復的时间。而且这个条件没有设置时间限制。
。
即使忍耐了而避免消失，也会受到严重的创伤。
。
因此，向班达鲁低头请求，但班达鲁摇了摇头。
。
「不，我不想打你。当然，前提是你能和阿尔达势力战鬥」
。
班达鲁没有想过攻击法马斯。



『什麼？那样好吗，我生前对你们……不相信寨卡特他们，结果他们被杀了。而且在十万年前的战鬥中，因为我造成巨大的损失。如果我加入那场战鬥的話，现在维达派的状态会比现在更好。
对於那些认为我是好人的人们，只因为我创造了冒险者公会而已，对维达派来说，亦对你个人来说，也不是很好的组织。不打我的理由应该是厌烦吧。』
。
听到法马斯告白的幾位众神点头，扎塔克脸上皱着眉。然後班达鲁也点头表示「嗯，是啊」。
。
「我只是带着灵魂的碎片创造出来的灵魂，所以没有当时的记忆。所以亦没有直接的怨恨。
关於冒险者公会，只能说你是很久以前创立者。你死後幾千，幾万年後的组织你没有责任啊。」
。
创立者中包括英雄神法马斯，但并没有指挥冒险者公会。各支部都会装饰着他的小神像，以及描绘圣印的挂毯等，而且也不是所有的职员都是信徒。
。
因此，把到现在为止经歷过的事的责任全责备法马斯並不合适。
。
「尽管如此，但我还是不想打一次。因为从多尔顿那听到。」
。
『你为什麼觉得是那样？』
。
「不，看你现在周围人对你的待遇……一点」
。
是的，从班达鲁来到魔大陆之後看到的现状，感到同情和怜悯。
。
注意到魔人族的始祖……对法马斯很是不高兴。
。
「我也没有特别想为你辩护。若果边界山脉南部的神们，塞尔克斯、穆布布佩斯、佐佐岡提，维达想我打你的話，我也不会阻止。」
。
班达鲁很同情法马斯，亦有怜悯的意思。怎麼想他也不是太坏，所以应该原谅他。
。
但是，如果对他有直接仇恨的神来復仇的話，就没有什麼理由去劝阻的了。
。
『是吗……已经太足够了，谢谢。而且这只不过是自我满足了吧……马马虎虎的程度』
。
法马斯说後，又再次向班达鲁低下了头。但是他真正低头的对像是班达鲁的战友他们。
。
魔人的始祖和鬼人的始祖用苦涩的表情凝视着那个背影，突然叹了口气。
。
『鬼人的話……虽然阿尔达现在行动了，但以前的遗恨就能忘记吗？』
。
『魔人的話也是这麼想。和勇者殿一样，还没法为他辩护』
。
这是自法马斯来到魔大陆後一直憎恶他的两柱神，到何时才能真的不去憎恨呢？所以他因为持续的憎恨而遭到了骂声。
。
很难忘记而原谅，法马斯在十万年前夺走了太多的东西。但是，若是持续憎恨的心态，那现在对待的态度实在是太温和了。
。
由於这个原因，所以仍是三心两意的心态，未贯彻任何一端的想法，亦未有相对的行动。



『对不起……』
。
向背对着的魔人和鬼人的始祖道歉的法马斯。班达鲁虽然还不明白才刚刚见面的众神心中的想法，但总觉得这样的发展很好，因为事情似乎以某种方向发展。
。
就这样，班达鲁他们留下了留学的奧迪华卡，暂时回到了塔洛斯海姆。



•技能解说：魔王
。
【魔王融合】的上位技能。吸收一定数量以上的【魔王的碎片】，並反复发动使其觉醒的技能。
。
主要的效果是将【魔王的碎片】能以与自己的肉体相同的感觉来发动。像在眼睑前被挥手的話，会反射性地闭上眼睑，能像这样反射性地发动【魔王的碎片】。
。
虽然应该有其他效果，但现阶段不明。
。
另外，其副作用，【魔王的碎片】的效果也会波及到技能持有者的肉体。
。
这个技能是【魔王侵蚀度】水平达到上限也都不能觉醒的。
