「現在這個城市是陛下的領地，我們是陛下的居民。陛下是我們的皇帝」
。
「請就這樣」
。
「……不，我們再等一會兒吧，好嗎？不需要那麼簡單地決定吧？聽聽街上其他人的意見，討論各種各樣的事情，至少要視察塔洛斯海姆吧？」
。
雖然班達魯慌慌張張地制止了角色們，但他們回答道：「不，沒有人會感到不滿吧」。
。
這個城市是魔大陸唯一住人的社區。因此，「別國」的存在從十萬年前起一次也沒存在過。
。
在城市裡有魔龍人、鬼龍人、古爾等多個種族，魔龍人和鬼龍人都是擁有提雅瑪特基因的兄弟姊妹，古爾是只有女性的存在，所以也與其他種族混血而產生了新的種族。



因此，種族之間的對立也並不是沒有，但是很少。
。
而且在街外的百年前……對維達的新種族他們來說，因為有《解放之惡神》拉波比法多率領的高等級魔物群，所以有著共同的強大外敵。因此他們基本上都是團結起來戰鬥而活下去。
。
「而且，原來的我們也不是賢者，亦不懂執政。」
。
「到現在為止一次也沒有王的存在。」
。
在這個宴會中聚集的眾人都是「領袖角式」，長或町長等職務的人是從一開始就不存在的。
。
「我們是按照眾神的方針運營城市的。當然並不是所有的指示，但與整體城市相關的事情都會實行。」
。
而且這些城市是超越邊界山脈內部國家的宗教國家……不，甚至可以說是神治國家。
。
在邊界山脈內部的各國會任命王和女王，以國家的守護神為首治理國家。
。
但是守護這個城市的守護神他們，至今仍在地上繼續存在著。這是因為扎塔克周圍變成了他的神域。因此，只要能忍耐酷熱的話，誰都可以去拜訪神並直接語言交流。
。
因此，沒有必要任命王等代表者，對於鎮上的人們來說，為政者和神是同一存在的狀態。
。
「因為神說沒關係，所以這樣就好了。」
。
是這麼說的。
。
「被那些神信賴的師傅，已經是值得信任的存在了。」
。
『恭喜，小主人，國土和國民增加了！』
。
『能很好地交流呢，少爺』
。
對這樣說的魯奇里阿諾和祝福的薩姆他們。實際上，班達魯也沒有拒絕的理由。
。
本來預定了各種各樣的援助，但是從塔洛斯海姆離開後，使用迷宮的【轉移】和雷吉恩和格法多加恩的能力，就可以瞬間移動。
。
而且這些城市並沒有嚴重的問題和負債。亦同樣是維達新種族的國家，初時對骨人他們等不死者很是驚訝，但很快就習慣了。



「那麼，把這些城市的居民作為我的國家的國民來迎接。彼此彼此，請多關照」
。
就這樣，塔洛斯海姆的國土和國民增加了。

---

然後在回到回去塔洛斯海姆的幾天前，班達魯的身影再次出現在扎塔克他們的眾神前。
。
『果然小妾還是不能給予加護。即使是這樣，但小妾在龍神中，也是最頂級的。』
。
『要是提雅瑪特也不行的話，我也不行的了。最差的考慮是，如果不是大神等級就不行』
。
想要加護的提雅瑪特和和迪安娜不可能，所以很遺憾地離開了。
。
『■■■■■■！』
。
取而代之的扎塔克發出聲音，揮舞著手臂。
。
《獲得了【扎塔克的加護】！》
。
腦內廣播傳來了，告知接受了扎塔克的加護。
。
『我的加護你能接受到嗎？』
。
然後，到現在為止，只認知到尖叫或呻吟聲的扎塔克的說話，現在就能理解了。雖然還夾雜著像噪音一樣的東西，很難完全聽懂，但還可以理解語言的意思。
。
『我～扎塔克～』
。
「啊，不，不。加護接受了。後面的話也大致明白了」
。
『哦，真的嗎？？至今為止，有好些人都接受了扎塔克的加護。但沒人能聽懂他的說話呢。』
。
對驚訝的眾神，法馬斯說『那麼，現在是時候說了。』。
。
『我想你們也從施耐德那聽到，但再次拜托了。對不起，我想你打我，直到我們和阿爾達的戰爭結束。』
。
因為沒有翻譯的需要，但大致理解上是這樣的，扎塔克是這麼說的。
。
「如果想戰鬥的話就打他一百次，他不能拒絕」
。
『告訴我為什麼啊！我還不想消失』
。
生前是勇者的英雄神法馬斯是比古特索索索特更高等的神。但也沒有信心不會被毀滅。
。
因為只有防御而不能回避，如果繼續接受班達魯認真的攻擊的話就危險了。因為他如果認真的話，就可以用【死炮】和【虛炮】進行一百次攻擊。魔力反動使肉體沒有恢復的時間。而且這個條件沒有設置時間限制。
。
即使忍耐了而避免消失，也會受到嚴重的創傷。
。
因此，向班達魯低頭請求，但班達魯搖了搖頭。
。
「不，我不想打你。當然，前提是你能和阿爾達勢力戰鬥」
。
班達魯沒有想過攻擊法馬斯。



『什麼？那樣好嗎，我生前對你們……不相信寨卡特他們，結果他們被殺了。而且在十萬年前的戰鬥中，因為我造成巨大的損失。如果我加入那場戰鬥的話，現在維達派的狀態會比現在更好。
對於那些認為我是好人的人們，只因為我創造了冒險者公會而已，對維達派來說，亦對你個人來說，也不是很好的組織。不打我的理由應該是厭煩吧。』
。
聽到法馬斯告白的幾位眾神點頭，扎塔克臉上皺著眉。然後班達魯也點頭表示「嗯，是啊」。
。
「我只是帶著靈魂的碎片創造出來的靈魂，所以沒有當時的記憶。所以亦沒有直接的怨恨。
關於冒險者公會，只能說你是很久以前創立者。你死後幾千，幾萬年後的組織你沒有責任啊。」
。
創立者中包括英雄神法馬斯，但並沒有指揮冒險者公會。各支部都會裝飾著他的小神像，以及描繪聖印的掛毯等，而且也不是所有的職員都是信徒。
。
因此，把到現在為止經歷過的事的責任全責備法馬斯並不合適。
。
「盡管如此，但我還是不想打一次。因為從多爾頓那聽到。」
。
『你為什麼覺得是那樣？』
。
「不，看你現在周圍人對你的待遇……一點」
。
是的，從班達魯來到魔大陸之後看到的現狀，感到同情和憐憫。
。
注意到魔人族的始祖……對法馬斯很是不高興。
。
「我也沒有特別想為你辯護。若果邊界山脈南部的神們，塞爾克斯、穆布布佩斯、佐佐岡提，維達想我打你的話，我也不會阻止。」
。
班達魯很同情法馬斯，亦有憐憫的意思。怎麼想他也不是太壞，所以應該原諒他。
。
但是，如果對他有直接仇恨的神來復仇的話，就沒有什麼理由去勸阻的了。
。
『是嗎……已經太足夠了，謝謝。而且這只不過是自我滿足了吧……馬馬虎虎的程度』
。
法馬斯說後，又再次向班達魯低下了頭。但是他真正低頭的對像是班達魯的戰友他們。
。
魔人的始祖和鬼人的始祖用苦澀的表情凝視著那個背影，突然嘆了口氣。
。
『鬼人的話……雖然阿爾達現在行動了，但以前的遺恨就能忘記嗎？』
。
『魔人的話也是這麼想。和勇者殿一樣，還沒法為他辯護』
。
這是自法馬斯來到魔大陸後一直憎惡他的兩柱神，到何時才能真的不去憎恨呢？所以他因為持續的憎恨而遭到了罵聲。
。
很難忘記而原諒，法馬斯在十萬年前奪走了太多的東西。但是，若是持續憎恨的心態，那現在對待的態度實在是太溫和了。
。
由於這個原因，所以仍是三心兩意的心態，未貫徹任何一端的想法，亦未有相對的行動。



『對不起……』
。
向背對著的魔人和鬼人的始祖道歉的法馬斯。班達魯雖然還不明白才剛剛見面的眾神心中的想法，但總覺得這樣的發展很好，因為事情似乎以某種方向發展。
。
就這樣，班達魯他們留下了留學的奧迪華卡，暫時回到了塔洛斯海姆。



•技能解說：魔王
。
【魔王融合】的上位技能。吸收一定數量以上的【魔王的碎片】，並反覆發動使其覺醒的技能。
。
主要的效果是將【魔王的碎片】能以與自己的肉體相同的感覺來發動。像在眼瞼前被揮手的話，會反射性地閉上眼瞼，能像這樣反射性地發動【魔王的碎片】。
。
雖然應該有其他效果，但現階段不明。
。
另外，其副作用，【魔王的碎片】的效果也會波及到技能持有者的肉體。
。
這個技能是【魔王侵蝕度】水平達到上限也都不能覺醒的。
