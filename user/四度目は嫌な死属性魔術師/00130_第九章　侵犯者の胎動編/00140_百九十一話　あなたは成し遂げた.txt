

S级冒险者队伍《暴虐之岚》的成员，其真实身份是原种吸血鬼（现在是深渊种原种）的佐尔科多裡奥，吉克自己和母亲的瑞吉儿还不知道，实际上的父亲是阿德帝国现皇帝马其克萨尔。
。
在维达信徒的隐身之处出生後度过，今後将在塔洛斯海姆成长。
。
对於複杂的身世，吉克少年自身是普通的人种。但因为混血了马其克萨尔的血，所以也有精灵的血统，耳朵有点尖。但意识到但不关注就不再在意了。
。
出生时也没有什麼受神关注的蹟像，如流星群染上夜空。他也没有独特技能，要像普通人那麼辛苦生活但亦受到旁人的祝福。在瑞吉儿和索德来看，虽然很聪明，但也不是让人吃惊的天才孩子。
。
（嗯……爸爸？）
。
这样的他从睡梦中醒来。自己在坚强又温柔的手臂中，所以感到平静。就这样想再次回到梦裡时，在薄薄的眼睑的间隙裡，看到不习惯的东西。
。
（谁……白的？）
。
睡眼惺忪看到一个白色人物。他被夾在中间，旁边有两个不自然的白脸少年，左右两个都是半透明，發出淡光的少年。
。
然後朦胧的视野开始変得清晰的瞬间，吉克的意识震惊了。
。
「嗯……什麼！？」
。
（班达鲁！？天宫博人，是不死者的班达鲁！？）
。
是的，吉克是被罗德科尔特送到这世界的转生者。
。
他在《地球》的名字是田中仁。是与当时的班达鲁在不同的班级的学生，幾乎没有任何认识。而在《origin》中，是一位被赋予了【韋驮天】能力的百人众一员。然後，【永恒之槍】的海藤加塔纳被【変形】狮子院真理杀害後發生的混乱，与【佩尔塞斯】和【乌尔德】同时期死了。
。
然後和其他两人一样拒绝了罗德科尔特的委托，转生到《拉姆达》……但首先，受到委托时，知道了关於班达鲁的消息。
。
与当时所看到的样子相比成长了少许，但在眼前只有他就没有其他了。
。
（啊，要被杀了！？）
。
反射性地吉克就是这麼想，想从索德的手臂中仰反，尽量想离开班达鲁。

「哦，好的好的。不需要吃惊啊」
。
「没关係啊，因为有姐姐们在」
。
但是幸运的是，佐德和梅尔汀一边这样说着，一边远离班达鲁。
。
（为什麼，是怎麼回事呢？我怎麼了……哦！？把嘴闭上，我！）
。
陷入了恐慌的吉克，突然发现了愚蠢的事，齿间颤抖，慌忙闭上了嘴。
。
然後吉克被带到瑞吉儿等待的地方，在此期间，至今为止的记忆和【韦驮天】的田中仁的记忆整合，自己在什麼状况开始有所体察……心情变得悲哀了。
。
（我能活下去吗？）
。
然後就这样睡着了。
。
因为与班达鲁的接触，比本来的年龄更快地恢復了记忆，和受到精神上的冲击，幼小的身体无法忍受。出现了和萨尔亚•雷夠斯顿一样的症状。
。
恐怕在醒来的时候也会忘记了前世的记忆吧。
。
「嗯，好像睡着了。……好像奇妙地被创伤了，希望他能喜欢班达鲁就好了。」
。
「但这国家有好多他的石像和小东西呢。」
。
在这个新市街建造的维达神殿裡也放着一个个小小的班达鲁像。此外，在维达神殿外面的摊档也有销售小型尺寸的班达鲁像。
。
主要是有好幾次停止建造巨大的班达鲁像，『如果是小像就没关係了！？』，维达神殿的现神殿长努阿扎所以就用核桃木雕刻了这样的东西。
。
「最重要的是，今後都会有好幾次和本人见面。如果能解开误解就好了」
。
索德说出担心的说話，但吉克已经失去了意识，因此能看到转生者们记录的亚乱们没能听到。

---

『又是这个模式啊！』亚乱感到焦躁向罗德科尔特怒吼，又要放弃他了，而且又不能和他联络。
。
【韦驮天】田中仁转生到这个世界时，是【永恒之枪】的海藤加塔纳刚战败时，应该是没有问题的。
。
当时是维达新种族的班达鲁，转生到和他不相容的阿德帝国皇帝马其克萨尔的兒子。
。
在他回復记忆时，阿德帝国和塔洛斯海姆应该是开战前时期的状态……说不定已经进入了真正的战争状态。所以，就算拒绝了委托的【韦驮天】也不能逃避，期待着一定会合作。
。
但是發生了意外的事态。马其克萨尔篡改了孕育着田中仁灵魂的婴儿的母亲的记忆，并将其送入隐藏维达新种族的《暴虐之岚》。但是，即使如此，罗德科尔特虽然不希望他会协助自己，但是如果他将来接近班达鲁，那就很容易收集到信息。
。
在《暴虐之岚》中，有邪神和原种吸血鬼的事令人惊讶，但如果转生者本人不犯错误的話，可能不会被发现。
。
……直到班达鲁越过系统的篱笆，从罗德科尔特的系统直接向维达系统引导人们的灵魂为止。
。
这样一来，就没办法了。
。
罗德科尔特可以给转生们带来《命运》和《幸运》。但是，那不是说「有意图，或者在理想的方向上容易进行」的意思。不能给予绝对的命运和幸运等不存在的东西。
。
特别是在像罗德科尔特轮回转生系统相类似的灵魂系统存在的《拉姆达》更是。
。
因此，对《暴虐的岚》保护的吉克无可奈何。《命运》多少妨碍了，但也许只是杯水车薪。因为班达鲁所拥有的命运是一个规格外的实力者。
。
或许【韦驮天】的田中仁从前世就有着不允许不死者存在的强烈宗教狂热者，又或者对罗德科尔特抱有坚固信赖的話也可另当别论……但现在不可能期望了。
。
所以罗德科尔特对【韦驮天】田中仁……吉克，和以前的【佩尔塞斯】鲛岛悠里……萨尔亚放弃了。
。
当然，没有对阿沙兹、加奈子，和其他转生者，包括村上他们说过吉克是【韦驮天】的事。
。
若想救出但失败令其去世的話会很麻烦，吉克的记忆和能力会恢復还是很久以後的事，所以对班达鲁也构不成威胁。
。
就这样，班达鲁就这样不经意地接受了另一个转生者了。

---

另一方面，那时的班达鲁、施耐德、莉莎娜、多尔顿和格法多加恩正在关注【魔王的碎片】的封印。
。
【魔王的碎片】暴走失控的哥布林，施耐德手下留情地进行了猛烈攻击削弱了它後，用薄带状的山铜，像木乃伊那样包着，那个宿主被封印，现在应该还活着。
。
实际上，施耐德背着搬运时也有时会有挣扎、呻吟……但现在没有了。
。
「难道，饿死了吗？」
。
「这麼说来，刚到塔洛斯海姆时就很安静了。封印後已经过了一个星期……」
。
多尔顿和莉莎娜看著很是老实的【魔王的碎片】，惊讶地凝视着。但是，施耐德摇了搖头。
。
「每一部分都有不同的差异，不能一概而论，但被劫持的宿主能不吃，不睡，直到宿主不行之前，才会寄生在新的宿主上。
根據记录，如果宿主不负大伤的話，即使最短也不会比一个月短。所以不是因为饿或衰弱而死去吧」
。
「封印的时候，我觉得宿主的哥布林吐血了」
。
「打到无法动弹，对於宿主来说是小伤吗？」
。
这样回答的莉莎娜，格法多加恩提出了一个疑问。也许因为受到亲人以外的人都吐槽的影响，施耐德把视线转向了班达鲁，他回答说『他还活着哦』。
。
『心臟因为碎片暴走而导致肉体结构变化而没有动，但至少【生命感知】有反应』
。
「是吧！？都是因为我手下留情啊！很难啊，因为减弱它都需要花些时间，但封印它是用另一种方法啊！」
。
愰然大悟的施耐德，其实宿主是處於相当衰弱的状态……大概还剩下一天壽命的事实，班达鲁没说出來。
。
『解开了』
。
然後，封印解除了。一溜溜的山铜带子解开了，奇怪身姿的哥布林出现。
。
皮肤的颜色和头部的形状，细细的手脚等和普通的哥布林没有不同。但是身体……确切地说，肋骨浮现在异常巨大的胸部，而且有多根管子，令人毛骨悚然的声音从管子里發出，从那裡听到了微弱的呼吸声音。

在有着令人毛骨悚然的管的巨人种胸部上，紧贴着小小的哥布林的头部。
。
被释放的【魔王的碎片】懒洋洋地站起来，身体不平衡，但表现出无限的喜悦。
。
『……本体……与…与……我……合……合流……』
。
班达鲁举起手示意【魔王的碎片】不要靠近，並试图发问。
。
『能把那个宿主放走，然後与我合流吗？』
。
与之相对，【魔王的碎片】停止了活动，从管子發出了幾次口哨般的呼吸声後回答。
。
『我、知識、没有。我、記憶、没有』
。
【魔王的碎片】是被切开无数的魔王格拉尼斯的肉片復活後各自变化的东西。这是不能以生命体形容的存在。有的只是想聚集在一起復活的本能。
。
在保持宿主活着的同时，可能很难找到吸收碎片的方法。这是班达鲁的想法，但【魔王的碎片】继续说。
。
『我的意思，不是本体的意思。本体的意思是我的意思』
。
『原来如此。也就是说，用我的意志来操纵便行了吧。那麼，就试着试试吧』
。
直接接触碎片，班达鲁尝试用【群体操作】技能来操作。尽量不要伤害宿主的身体，以恢復原状的形象使其分开。
。
於是【魔王的碎片】全身微微发抖，从管子吐出了黑色的液体。
。
黑色的液体变成一块，变成原形不明的肉片，跳跃似的跳到班达鲁处，發出刺耳的一声後，就融为一体了。
。
《【魔王的肺】融合了！》
《【魔王】、【群体操作】、【群体思操作】技能水平上升了！》
。

看来，碎片似乎不是浮着的气管，而是肺。
。
「……没关系吗？看到对健康不好的颜色的肉片，發出很大响声後从手上溜进去啊。」
。
在确认情况的时候，施耐德露出担心的表情，似乎他看到了。再看向他们，多尔顿和莉莎娜也有相似的表情。
。
吸收【魔王的碎片】对班达鲁来说已经是习以为常，但对施耐德他们来说是第一次看到令人毛骨悚然的现象，所以也没有办法。
。
当班达鲁第一次吸收【魔王的碎片】时，「快吐出來」爱莲奧诺拉也试图让他呕吐。

『我没关係。但是，宿主的结果还是没救出来』
。
听到班达鲁的话才注意到，施耐德他们看着宿主的哥布林，微微地反复发抖……继续抽搐。但是，它的间隔也渐渐変长，然後就变成不动了。
。
「失败了吗？」
「不是失败」
。
多尔顿看著死去的哥布林碎碎念，格法多加恩马上反驳。
。
「原本这个哥布林的身体也变得太奇怪了。没有碎片就不能维持生命。胸部应该没有除了肺部以外的内臟。
如果只是四肢和皮肤变异的话……如果有心臟的话，都可以暂时活下去吧」
。
「原来如此，怪不得说听不到心音。那麼是这傢伙运气不好啊。」
。
多尔顿理解後，就拔出了刀……脸上露出为难的样子。
。
「那麼，这个尸体怎麼办？如果要处理好，可以用魔术燒掉。」
。
看来，就像对待普通的哥布林一样，想把讨伐证明的耳朵切掉，但在边界山脉内是没有冒险者公会的。
。
『首先，会作为贵重的标本来购买。无法回復变异部分的意义上，我失败了，所以必须研究并将其用於未来。
啊，钱是这个国家的露娜货币好吗？因为不能和阿德帝国做交易，所以就使用自己的金属，出国的时候记得要全额寄存啊。』
。
「不，本来是带着礼物来的，所以钱就不需要了……自己的货币啊，还开发自己的金属吗？很厲害啊。
但是，如果能操作到其他宿主的碎片的话，那【魔王的碎片】的敌人对你来说就没威胁了吧？」
。
施耐德是这样说的，但是班达鲁回答说『我觉得不是这样。』
。
『刚才能很好地去做是因为宿主的意志被碎片吞没了，没有了存在。如果宿主的意识健在，想控制碎片，不可能操作吧。但可能多少会受到影响。』
。
像《解放的恶神》拉波比法多，或原种吸血鬼蒂尼西亚和谷巴蒙等，【魔王侵食度】水平不到10的对方是很难操纵的。班达鲁是这样推测的。
。
若能精益求精，想到方法加快侵蚀的步伐，使宿主的意识失去，就能勾起碎片对主体的渴望。
。
「是吗？侵蚀中途不太好办。」
「……不，我认为这是一种另类的攻击方法。」
。
因为控制碎片很困难，所以多少会受到外界的影响。从敌人看来，是相当有效的干扰。
。
虽然莉莎娜是这样想的，但是搬运哥布林尸体的班达鲁和施耐德，想着还是不太有效。

---

就这样，与施耐德他们汇合後，班达鲁他们也一直活跃着。
。
在特定的阶层中，一只魔物也不会出现，在海、高原和草原的阶层可以度假、农业和渔业的迷宫也完成了。除了魔物不出现的阶层以外，魔物出现率会增加，比通常的迷宫还会频繁地进行与魔物间的战鬥，这是被探索者公会的委托的搜索者们，和有效利用班达鲁的魔物们的目标前进。
。
「ギュオオオオオオオオオオオ♪」
。
特别是被赋予了广大的饵食场的巨大暴食蠕虫，据说是非常活跃的。
。
顺带一提，在《寨卡特的考验》中的恶魔们也在班达鲁创造的迷宫中度过。
。
魔大陆的开拓也在进行着。《剑王》沃尔克斯，最近得到《最强的木人》之名後，然後获得了《元祖木人》这别名的《冰神枪》米亥尔，获得了《死斧王》别名的维格罗他们的活跃，魔物的讨伐也在进行着。
。
《魔人王》哥德文的魔人族始祖，鬼人国的王天玛向鬼人的始祖打招呼，龙人国的使者被提雅玛特对自己国家的守护龙他们斥责……『关於那件事为什麼比其他国家知道得晚呢？』……奧迪华卡在留学时顺利地提升了等级並进行了转职。
。
只是在魔大陆上好幾个迷宫多数是C级，A级迷宫可以确认的只有三个。这是因为拉波比法多和《狂灾的恶神》达尔格索特在一百年前，以魔大陆为主支配着魔王军残党，为了增加自己的信徒和棋子，在迷宫中频繁地吐出了魔物，所以使迷宫不太能成长。
。
对於沃尔克斯和哥德文等一部分人来说，刺激是不够的……但能确认的C级迷宫就有三百个以上，与人类社会相比，是一个意想不到的人外魔境。
。
而且，D级以下的迷宫只存在数个。所以在地上蔓延的魔境中栖息的魔物最弱也有阶位4。
。
魔王军残党，不干涉栖息在地上魔境的魔物，只有弱弱的D级的迷宫，被放置了的结果到向C级成长了。
。
若在普通的冒险者公会的话，在新人长大前便会死在困难的环境中，所以世世代代维持环境的战士团的魔龙人和鬼龙人，古尔亚马逊他们是多麼强大，而且还持续的努力中。

因为还没有确认过的地区也有很多，所以魔大陆像边界山脉内部那样，使危险度下降是最優先的。
。
施耐德他们的《暴虐之岚》在参拜《维达的睡所》後，粗略地看了沼泽地的泥汤温泉，也看了塔洛斯海姆和边界山脉内的国家，然後再次回到塔洛斯海姆，和班达鲁进行了这样的对话。
。
「有女神在，感觉在边界山脉内安全生活是易事……但能把自己出生的土地上抛弃的傢伙们不会很多，这也是理所当然的。在长老连中也有不相信你存在的傢伙。
所以要说服他们。」
。
