## preface

```
主角醒來便是在異世界某迷宮的祭壇中，（被某異端審問團斷罪後執行了天罰？）開始在迷宮中漫無目的四處碰壁，想逃，路卻只有來時的單行道，也不知是哪個混球的設計，深陷絕望之時終於看到了希望的曙光，找到了人類的氣息，充滿著驚訝與喜悅，無奈命運的捉弄，深深地感受到了來自世界的惡意，遭受了背叛他，命運又該何去何從………………
劍與魔法的王道幻想，淚與汗水交織的冒險，編織出追夢的詩篇。
```
