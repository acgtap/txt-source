「莉米絲。不用這樣。比起加上大人這種像是外人稱呼的方式，還是被人以名字稱呼比較高興」
。
　認真的表情。由於他五官極為端正，表情認真起來以後就是一幅畫。
　如果在將來藤堂的活躍會記載在聖典之上，那估計會被稱之為絕世美青年。雖然我覺得說是絕世有些過頭。
　盡管我對於自己的容貌有自信，然而假設有十個女人，應該有八個會選擇藤堂吧。畢竟我眼神實在是太凶悍了。我已經預定在將來跟教會還有克里歐投訴。
。
　彼此互望的藤堂和莉米絲。
　我也不是為了成為陪襯才加入這隊伍的。
　我刻意無視無法插手的氛圍，拍了幾下藤堂的肩膀。
。
「那就請多多指教咯！藤堂！」
。
「噫！？」
。
　我明明沒那麼用力，可藤堂大聲地發出參加，肩膀猛然抖了下。
。
「你做什麼！」
。
　莉米絲從藤堂身上移開目光，用力拍起桌子。
　她瞪視這邊時，目光與以往的注視我的視線不同，那是生氣的眼神。
。
「在講禮儀之前，你對勇者大人的敬意根本就不夠！」
。

。
「嚯。你是說身為亞茲・古利德忠實下僕的我，對於聖勇者沒有抱有敬意？」
。
「嗚……這、這――」
。
　怎麼可能。我可是比起誰都還要敬愛勇者。
　如果不是這樣，即便是上司的命令，在這種狀況中我早就放棄任務了。
。
　搬出神的名號，莉米絲都沒辦法說得很肯定。不愧是不諧世事的大小姐，應該是受過良好的教育吧。
　像瞧不起神之類的事情，她一定是想都沒想過。明明我每天至少就有一次以上叫它去死。
。
　……不過要說我對於藤堂是否帶有敬意……應該是沒有吧……。
。
「……莉米絲，不要管他」
。
　亞莉亞在一瞬間露出責備的眼神，不過立刻面向藤堂那邊。她甚至都不望向我。
。
「藤堂殿下，總之所有人都到齊了，差不多也該出發了吧。據說即便是乘坐馬車趕路，也要花上五、六小時才能抵達維魯森林周邊的村子。如果在途中遭受魔物襲擊應該還會更花時間。如果再不快點，太陽都會下山」
。
「是啊……說的也是……。沒有時間再玩了」
。
　維魯森林的周邊街道還是經過某種程度的整備，然而這不代表完全可以避免魔物襲擊。只不過是比其他道路還要安全得多而已，該出現的時候就會出現。嘛，不過由於是在王都附近，幾乎沒有可能會出現盜賊一類的人就是了。
　我斬釘截鐵地宣言偷偷設定好的目標。
。
「總而言之，兩周之內將藤堂的等級提升到三十」
。
「……哈！？兩周……之內！？再怎麼樣也太――」
。
　亞莉亞驚訝得說不完話，而藤堂和莉米絲似乎不怎麼明白的樣子。
。
　一天一級。這種目標如果讓傭兵他們聽到應該會嘲笑說真是不怕死。
　然而，我可以使用輔助魔法和回復魔法。讓藤堂殺死所有的魔物也並非不可能。。
　嘛，關於亞莉亞和莉米絲在那之後就讓她們慢慢升級就好了。
。
「太亂來了！不對，最好不要這樣做！即便很有可能被魔族感知到……有必要這麼趕嗎！？」
。
「有」
。
　兩周之內升級到三十，我就可以得到獎勵。其金額是五百萬盧克斯。這幾乎相當於中層市民兩三年的薪水。
　我無視表情凶險提出抗議的亞莉亞，面向當事人的勇者。
。
「話是這麼說，這的確是相當亂來。如果藤堂覺得不行那就――」
。
「就這樣做」
。
　藤堂的表情略帶蒼白，然而是即答。
　害怕……應該不是這回事。在他那睜大的眼睛裡，其中央閃爍著漆黑的彩虹。
　那是決心亦或是覺悟。
。
　回想起他昨晚的樣子，我蹙起眉頭。
　那麼，我該怎麼跟上司克里歐報告呢……。
。
「變強，我會變強。變強到可以守護莉米絲，可以守護亞莉亞」
。
「是嗎」
。
　宣言倒是挺好的，幹勁也有。剩下的就只是調整一下，避免那會成為有勇無謀。
　總之我決定現在將一切的煩惱拋開，用僧侶特有的詞句來蒙混過關。
。
「願亞茲・古利德的加護賦予您」
