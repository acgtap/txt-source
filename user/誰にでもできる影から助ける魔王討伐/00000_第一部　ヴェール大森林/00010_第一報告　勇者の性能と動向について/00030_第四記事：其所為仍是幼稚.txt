　格拉斯蘭德・維多（Grassland・Wind）。

（颯:意思為草原之風）
。
　這是派遣亞莉亞來到藤堂身邊時從本家帶來的寶具之一。
。
　亞莉亞・利扎斯的本家——利扎斯家是代代培育優秀騎士出身的歷史悠久的名門武家。是這世界上存在的兩大主要劍術派系之一，普拉米亞流正統劍術的宗家。
　其當代家主諾德・利扎斯是統合王國騎士團與軍事的劍武院的最高幹部之一。在今世與魔族戰火連連的時代中，他是被譽為劍王的最著名英雄之一。
。
　其本家代代侍奉于盧克斯，深受其重用。在他們的寶物庫中魔劍與聖劍一類的武器不用說，還有能在戰場上能夠發揮出無比絕倫的寶物。
　據說諾頓・利扎斯年紀將近五十，不過似乎是相當溺愛女兒樣子。什麼叫這瘋丫頭令人為難！
。
　亞莉亞為了移動而取出的，正所謂是在文物都沒聽說過的傳說級寶具。
。
　它的大小是手掌能夠容納的程度。是用某種顏色為白，材質不明且質感平滑的物質做成馬車的模型。
。
　寶具幾乎都是用上一切魔術、物理性質的珍貴技術，基本上是無法量產的。
　格拉斯蘭德・維多也據說是原本妖精族為了移動而僅製作幾個的物品。盡管不知道究竟是有何緣故才傳到人類手中，不過一旦損壞應該就沒辦法再得到了吧。
。
　亞莉亞自信滿滿地將其取出來，然後撓撓臉頰。
。
「話是這麼說，我也不知道要怎麼使用……」
。
　喂，這根本是暴殄天物吧。
　我臉頰痙攣地告訴她。
。
「……要放在地上注入魔力」
。
　寶具幾乎都是物質化的魔術法則。
　物體本身刻印的魔術是非活性，平時雖然不具備任何效果，然而可以通過注入魔力將其中的奇跡顯現出來。
。
「嘿~我看看……」
。
　藤堂興趣盎然地抓住，將其放在地面上，以不習慣的動作注入魔力。
。
　人潮眾多的道路吵雜起來。
　模型馬車散發強光，悄然無聲地巨大化。藤堂慌張地後退幾步。
。
　僅僅幾秒鐘，在那邊佇立的就像是貴族會使用大型馬車。
。
「啊……誒誒誒！？這、這是什麼！？」
。
「格拉斯蘭德・維多……傳承中登場的魔法馬車。還真存在呢……」
。
　與瞪大眼睛啞口無言的勇者相反，莉米絲雖說有些驚訝，不過意外的冷靜。畢竟是魔術師家門出身，應該比較習慣這種奇異的魔導具吧。
　而將其帶來的當事人亞莉亞則是略帶驕傲地挺起胸部。
。
　窺視下內部，裡面的空間即便坐上四個人並放好行李依然還是綽綽有餘。
　顯然比表面上看上去還要寬敞，應該是施加上空間擴展的魔術吧。覆蓋車廂的純白色車篷頗厚，似乎能夠安定地抵擋風雨。
。
　然而這輛馬車與一般的車篷馬車最大差異，應該就是拉動馬車的白馬。
　這匹白馬具有比起普通馬匹大上半倍的體型，並且具備六條腿。質感與模型時一樣全都呈現平滑型，顯然不是生命體。
。
　馬匹簡短地低聲嘶鳴，毫無光芒的雙眸望向這邊。
。
　馬型的魔導人偶，不知疲憊無需飲食。我像是敲門般敲敲它的身體，只感覺到“叩叩”的堅硬觸感。它的力量應該不會弱小，威風到似乎連小型魔物都可以無視，直接踩死。既然是魔導具，根本不用懷疑它的是否順從。
　作為拉動馬車的存在，應該沒有比起它更好的吧。
。
　面對這樣一種異樣的魄力，莉米絲微微倒吸一口氣。
。
　明明魔王討伐之旅一旦失敗就會遺失，劍王也真是下了血本。
　縱使是劍武院最高幹部，這應該也不是靠金錢能夠得到的。
。
「這樣就不需要抽時間給馬休息了」
。
「好、好凶狠的馬……這世界的馬該不會都是這樣子的吧？」
。
　復活過來的藤堂流著冷汗，戰戰赫赫地望著馬匹的眼睛。
　怎麼可能。那明顯就不是生物的眼睛。
。
「不……那只是馬形的魔導人偶（魔像）。嘛，比起馬匹還要方便。既不需要吃飯也不用排泄。更不會受傷也不會疲勞。記得即便是損壞了，在某種程度上也能自我修復才對」
。
　只不過由於不是用於戰鬥，因此不是那麼耐抗。如果受到襲擊，估計得邊保護它邊戰鬥。
　嘛，只要縮小，就那尺寸也可以放進口袋隨身攜帶，所以也不難。
。
「是、是嗎……真是準備周到呢。魔像、魔像嗎……」
。
　藤堂眨眨眼，用手掌撫摸馬匹的身體。
。
「藤堂的世界沒有魔像嗎？」
。
「……並沒有實際存在過。不過在故事中倒有登場」
。
「那就跟這邊一樣了。格拉斯蘭德・維多是出現在神話中的寶具。鮮少可以見到。真沒想到利扎斯居然會有……」
。
「……雖然我不是這個意思就是了……」
。
　我的同事中有個寶具廚。下次就跟那傢伙炫耀吧。倘若事情順利，說不定還能讓他請我吃頓飯。
　在這時，莉米絲蹙起眉頭嘀咕道。
。
「原動力是魔力……我記得這是靠消耗車夫的魔力來運作的吧？」
。
　奇跡無法自然產生。
　變大的時候是使用魔力了，不過在行駛的期間應該也需要使用魔力吧。寶具的特徵是不用消耗那麼大量的魔力才對。

無論是因加護而具有高魔力的藤堂，還是莉米絲都可以來駕駛。而亞莉亞是劍士，魔力不是那麼高，不過還是稍微可以讓馬車動起來吧。
。
　魔術與神聖術的力量源泉是截然不同。
　我雖然具有高神力，可反過來並不具備多少魔力。
　由於還要使用通信的魔導具，所以我想盡可能節約，不過――。
。
　在這時，我注意到一個問題。
　我環顧這三人，高舉起手。
。
「這裡面有做過馬車車夫經驗的人，舉起手」

「……」

「……」

「……」
。
　對於面面相覷，沉默不語的三人，我不知道是第幾次深深地嘆了口氣。

。

　國王已提前將通行許可證交給了我們。
　在盧克斯東邊大門優先處理手續後，以藤堂為前頭的勇者隊伍首次踏出人類圈外。
。
　王都由巨大厚重的城牆所圍繞。雖說王都內部沒有魔物，可接下來就不一樣了。
。
　盡管街道經過整修，而騎士團以及將討伐魔物作為本行的傭兵都會定期性除掉周邊的魔物，不過在日常上受到魔物襲擊的可能性會增高。
　幸好這附近出現的魔物並不是那麼強。即便受到襲擊應該也不會輸，總之提高警惕沒有壞處。
。
　結果用消去法馬車的駕駛席――是由我來坐。我用力拉住銀色的繮繩。
。
　魔力通過掌心傳達給魔像，馬車緩慢地開始行駛。
　我是在幾年前學會馬車的駕駛方法。馬車的速度很快就達到跟一般的馬車同等的水平，初春涼爽的微風吹拂過臉頰，流逝在後頭馬。
。
　駕駛座有兩個人的空位，可我旁邊沒有半個人。
　他們三人都坐在厚長的蓬簾內部的車廂。也許是因為旅行剛開始，能聽見他們那平靜且充滿希望的談話。
。
　馬形魔像與真正的馬匹不同，十分的順從。光是輕輕拉住繮繩，它就如我所想的動起。
　既然是這樣，即便是外行人也能做到簡單的駕駛吧。
　如果可以我還真希望有人在旁邊看看駕駛方法，不過總覺得有點心累，我決定什麼都不說。即便是我也是會覺得疲勞的。
。
　王都的周邊全是毫無高地起伏的平原。
　東部則是延伸出一片草原，沒有任何的遮掩，能夠看到遠方的地平線。盡管有魔物的身影，不過都是在道路外頭，應該不會有問題。
。
　這可稱得上是與『草原之風』這輛馬車的名字相對應的第一步旅途。
。
　也許是懸架奏效，馬車的搖晃幅度比起以往乘坐的還要微小，舒服多了。
　打倒魔王以後，這輛馬車能不能送給我呢……應該行不通吧。如果是藤堂開口估計還有可能，可我又不是勇者，只是僧侶……。
。
　正當我發著呆想著這些事情，逃避著是否能打倒魔王的不安時，突然望見在道路外面有野獸正在接近。
。
　綠色毛皮的中型狼，它們是以幾十只為一個群體，利用銳利的獠牙與爪子襲來，名為草原狼的獸型魔物。
　具備一定的知性，懂得在襲擊獵物之前派出手下進行偵查。當那只魔物認為我們容易解決，本隊應該就會襲擊而來吧。
。
　每一只雖然不是很強，不過由於數量較多，屬於這附近絕不能大意的魔物。
　與維魯森林的魔物相比，存在力較少，必須要靠數量才能積累。在這裡對付它們也沒有好處。
　藤堂要是看見說不定會說出想練習或是想戰鬥之類的，真麻煩。
。
　前方幾十米處，草原狼露出狡猾的目光偷窺著這邊。我將殺意釋放到極限，全都集中在魔物身上。
　狼嚇得身體一抖，發出慘叫後逃之夭夭。
。
　草原狼的推薦討伐等級是在十級到二十級。
　應該是本能上察覺到彼此之間的存在力差距，沒有人類聰明的它們不會違抗本能襲擊而來。
。
「嗯？發生什麼事了？」
。
「沒事……什麼事也沒有」
。
　聽到背後通過厚長的蓬簾，顯得模糊不清的聲音，我毫無幹勁地回答道。

　駕駛馬車過了一小時。
　這期間雖然意圖襲擊的魔物比想像中還要多，不過只要釋放殺氣就立刻撤退了。沒有半只襲擊過來。
　直到剛才我都沒去想，這魔物的遭遇率――活性化估計是受到魔王進攻的影響。
。
　正當太陽高掛天際，我開始想讓誰坐在這邊學習馬車駕駛的時候，背後的蓬簾突然敞開。
。
「嗯？怎麼了？」
。
「嗚嗚……」
。
　出來的是藤堂。
　毫無血色的蒼白姿容，以及陰沉沉的眼神。
　這表情並非是昨天顯現的瘋狂狀態，單純只是身體不舒服的樣子。
。
　他用右手死命按住嘴巴，抬起頭用從未有過的求助眼神望著我。
。
「咕、我、我想呼吸外面……的空氣……」
。
　對於他死命擠出的話，我就察覺到一切了。
　我第一次因勇者的表情而驚呆到望著他。
。
　這傢伙……難不成這種程度的搖晃都會暈車嗎。
。
　在勇者的背後，亞莉亞和莉米絲都用擔心的臉色看著。這兩人完全沒有暈車的樣子。
。
　這也是當然。畢竟這輛馬車還算是相當舒服的。
。
　即便是身為貴人女兒的莉米絲與亞莉亞，應該也沒坐過這麼好的馬車吧。
。
　藤堂如今似乎都要嘔吐出來的樣子。不過也許是自尊心勝過痛苦，他用力按住嘴巴的同時以痛苦的語調喊道。
。
「別、用、這種、眼神看我！」
。

。
「就我起初的檢查，馬車的通氣性是完美的。跟外面的空氣應該沒什麼差別才對」
。
「別……別說了，外――」
。
　藤堂仍不服輸，我用食指抵住他流著冷汗的額頭。
。
「『六級異常狀態回復神法』」
。
　真沒想到我第一次對友方使用的神聖術居然會是治好暈車。即便是秩序神亞茲・古利德也無法預料得到。
　指尖前端閃爍著淡綠的光芒，隨後逐漸擴散到藤堂全身。
。
　神聖術之中存在等級劃分。
　當然，等級越高的，其難度以及消耗的神力也就越多。可如果是治好暈車的程度，利用最低級的『六級』就足夠了。
。
　效果發動僅是一瞬間。
　藤堂維持著按住嘴巴的姿勢，驚訝得睜大眼睛。他的樣子隱約挺滑稽的。
。
　我冷淡地詢問他。
。
「身體狀況如何？」
。
「啊……啊啊……」
。
　藤堂輕輕地把手從嘴邊放下。
　蒼白的面貌逐漸回復血氣，臉色比起剛好還要好很多。
　體力雖然多多少少有些損失，不過要痊癒也只是時間問題。
。
「你……你做什麼了……？」
。
「我是治療師」
。
　現在可沒有時間讓他休息，也沒有時間跟他爭論。
　接下來還需要行駛幾小時。如果每次都要讓一小時就能暈車的人休息一陣子，太陽都下山了。
　藤堂一時之間不知所措，不過什麼都沒說，又回到車廂裡。
。
　我直直地望著那邊一陣子，確認到他沒有出來以後，又面向前方。
。
「……真是的，前途多難啊」
。
　這就是所謂的文化差異嗎。藤堂應該是不習慣馬車，按照這樣子估計連騎馬都不行。
。
　等級提高以後可以避免暈車？
　升級以後整體能力都會提升，我想大概是不會再暈車，然而我所知道的知識裡並不知道這答案。
。
　結果直到抵達維魯森林最近的村子之前，勇者有三次從蓬簾里探出頭來。
