３８．我想成為縱使被盯著看也能好好說出NO的人
一一一一一一一一一一

　離開百貨商店，我走在街道上。
。
「嗯~……」
。
　邊走的同時，回想著剛才學生們的談話。
　『早熟』所帶來的增加ＳＰ和ＪＰ的效果。
　雖然跟其他人比較之後才明白，不過這技能比我想像中還要更作弊。
。
「雖說我能得到這技能只是單純運氣好，不過這樣一來我就更想要技能的說明啊……」
。
　即便獲得技能和職業，人物狀態面板上也不會出現說明。
　因此只能實際獲得並嘗試一下，從名字來推測個大概。
。
「果然技能說明是需要『鑑定』的吧」
。
　『鑑定』。
　解析技能的技能。
　我重新認知到它的必要性。
。
「雖然不知道獲得條件……不過感覺像是『觀察』的上級技能」
。
　『觀察』是我得到『密探』職業的時候獲得的技能。
　現在的級別是４。
　如果提升到ＬＶ１０，很有可能會得到『鑑定』不是嗎？
　雖然終究只是我的推測，但值得嘗試一下。
。
「感知系技能，再加上『觀察』的級別也都要提升啊……」
。
　以往只是滿不在乎地就放任不管了，不過說到底『早熟』是『固有技能』。
　從這點來看就跟其他技能不同。
　有必要了解清楚這個技能的事情。
　看來需要做的事情增加了。
。
「くぅーん？」
。
　莫莫向我詢問道“沒事吧?”。
　哎呀，完全熱衷於思考事情當中了。
。
「沒事喔，莫莫，你不用擔心啦」
。
　我蹲下來摸著莫莫的頭。
。
「……汪嗚」
。
　類似於“那就好”的回應。
　我不禁苦笑起來。
　莫莫真是愛操心，真可愛。
。
「嗯?那是……？」
。
　突然間，我望過去看到那裡有史萊姆。
　就是建設在牆邊的垃圾場。
　有幾隻史萊姆就混在垃圾堆中。
。
　連這種地方都有這些傢伙嗎。
　由於『危機感知』和『敵意感知』沒有反應，我都直接遺漏它們了。
　它們並沒有做什麼事情，僅僅是在原地動彈著而已。
　啊，不對，仔細一看垃圾正慢慢地被溶解。
　這些傢伙什麼都吃嗎。
　在幻想世界的小說裡，史萊姆一般都是雜食，看來現實中的史萊姆似乎也不是例外。
。
　嗯~怎麼辦。
　要狩獵這些傢伙應該很費時間吧……。
　反過來回報很少。
。
「嗯？」
。
　仔細一看，史萊姆裡面有一只顏色不同的史萊姆。
　其他的史萊姆都是藍色的，但那傢伙是紅色的。
　該不會是史萊姆的上級種？
　我就這樣繼續眺望著，而紅色的史萊姆也沒有在做什麼，僅僅是在原地蠕動著而已。
。
「真是不可思議的生物啊……」
。
　某種意義上是謎團最深的魔物了。
　嘛，總之就放任不管好了。
　反正經驗值也少，不管它也沒有危害。
。
「啊，對了」
。
　稍微有點好奇心，我從道具包裡面取出『垃圾』。
　是吃完飯以後的便當塑料盒。
　我把塑料盒丟給史萊姆。

『……』
。
　史萊姆並沒有反應―――不，有反應了。
　它們將塑料盒覆蓋住並將其溶解。
　……雖然溶解非常慢就是了。
。
「嘿ー……」
。
　我仔細地端詳它們的樣子。
　差不多經過十分鐘，史萊姆吃完塑料盒了。
　它們抖動著身體，又繼續到垃圾場吃起垃圾。
。
「這些傢伙真能吃啊」
。
　有種在看著大型微生物的感覺。
　嗯？怎麼了？
　顏色不同的紅色史萊姆緩慢地往這邊過來了。
　『敵意感知』和『危機感知』都沒有反應。
　姑且提高警惕離開距離。
　莫莫也準備好隨時進入戰鬥態勢。
。
≪紅色史萊姆注視著您，似乎是想要成為伙伴。請問需要加入為伙伴嗎？≫
。
　腦海裡響起天之聲。
。
「誒？」
。
　一瞬間我以為是聽錯了。
。
≪紅色史萊姆注視著您，似乎是想要成為伙伴。請問需要加入為伙伴嗎？≫
。

　天之聲再次在腦海中響起。
　並不是我聽錯了。
。
　真的假的，魔物也有這種事情發生嗎。
　話說回來，果然史萊姆也具備『意識』啊。
　史萊姆抖動著身體注視著這邊……在我看起來是這樣。
　該不會認為是我給它誘餌的吧？
。
「嗯~……」
。
　可是啊……。
　老實說很微妙。
　雖然網絡小說裡史萊姆成為同伴是王道劇情，可是在現實中就有點……。
　看著佇立在垃圾場的紅色史萊姆，我思考著。
　雖然是無害，但也沒有什麼特別的好處啊……。
　不如說考慮到我技能的特性，有它在反而會礙事吧？
　『潛伏』和『氣息消除』的效果只針對我。而躲在『影子』離的莫莫雖然是例外，但也適用於這點。
　而且它走得很慢，還非得背著它走動不可，戰鬥中似乎也是派不上用場。
　有它也是正如字面意思上的包裹。
。
　我在腦海裡選擇NO。
。
≪申請已經拒絕了≫
。
『……』
。
　是我的心理作用嗎，感覺紅色史萊姆似乎有點悲傷似的抖了抖。
　大概是我的錯覺，我很想這麼認為。
　我的錯覺，絕對
。
「……我、我們走吧，莫莫。去尋找其他的魔物吧」
。
「……汪」
。
　嗯，沒有做錯。
　我應該沒有做錯。
　再次邁開步伐後，肚子響了起來。
。
「……說起來，現在差不多到中午了嗎」
。
　看著手表上的時間，已經過了十二點了。
。
「總而言之，先休息一下吧，莫莫」
。
「汪」
。
　尋找似乎可以休息的建築物。
　……那邊就好吧。
　非常近的一座高層公寓。
　背對著抖動中的史萊姆，我和莫莫往那邊走去。


。

　另一方面，這時候在某座公寓屋頂上―――
。
「……果然在可以看見的範圍內，魔物變得越來越少了呢」
。
　通過照准器看著城市風景的同時，『她』嘀咕道。
　魔物出現的第三天。
　她從來沒有從這座公寓踏出一步，一直都在這裡反覆進行狙擊。
　她的努力有了回報，等級提升了相當多，點數也得到了許多。
。
　然而差不多也該到了極限。
　射擊範圍內出現的魔物數量變少了。
。
「差不多該離開了吧……」
。
　考慮到自己的職業和技能，還是不要離開這裡比較好。
　然而即便呆在這裡，目前也已經要到達極限了。
。
「使用點數強化技能和數值，然後找到目標建築物，接著繼續蹲在那邊就好了嗎……」
。
　只要找到符合職業條件的公寓或者高層大廈，然後將那裡作為新的據點就好了。
。
「可以的話，還真想要幫手啊。……話說回來，那個男人和汪醬還活著嗎」
。
　她想起昨天來到這裡的男性和柴犬。
　擁有著有趣的技能。
　雖然到最後她都沒有上去搭話，他們直接就站起來離開了。
。
「……嘛，畢竟是人渣，我覺得大概還活著就是了」
。
　不對，與其說是人渣，不如該說是陰險狡詐吧。
　從超市的戰鬥方式以及在這裡看到他的反應，她是這麼覺得。
　為什麼會這麼覺得。
　那是因為自己也是這樣。
　不想死，想要活著死。
　所以才會打倒魔物，提升等級，謀劃出掌握到安全的方法。
　話又說回來，她又如此想著。
。
「那只汪醬好可愛啊……」
。
　緊緊陪伴在那男人的身邊，晃著尾巴的樣子。
　那是多麼的可愛。
　她很想摸摸它，很想受到治癒。
　她並非貓黨，而是狗黨。
　特別是柴犬就是她的最喜。
　如果不是這座公寓禁止飼養寵物，她自己也想養一只這麼可愛的汪醬。
　她恍惚著，臉頰都泛起紅暈。
　仿彿就像是戀愛中的少女。
。
「……真想再見它一面啊，那只汪醬」
。
　在她的腦海裡，男人的容貌早已經曖昧不清，而莫莫的樣子則是美化了兩百倍左右。
　怕生的人，基本上都不會記住別人的長相。
。
「―――『地圖』」
。
　如此嘀咕後，她的人物狀態面板發生變化。
　變成顯示成立體的城市地圖。
。
「……嗯，這裡就好」
。
　離百貨商店有點距離的某座高層公寓。
　如果是這裡就可以符合自己職業的條件。
　雖然有必要再湊齊家具和食物，嘛，這些就先不管吧。
。
「那麼就出發吧」
。
　她把手法已經完全熟練的長槍背負在肩膀上。
　如此這般，她一之瀨奈津為尋求新的狩獵場而行動。
　其心裡的某處在祈禱著與那只柴犬的再次相遇。
