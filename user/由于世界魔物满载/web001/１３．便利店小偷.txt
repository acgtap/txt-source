１３．便利店小偷

「好，我們進去吧……」
。
「汪」
。
　確認到周圍沒有魔物的身影，我和莫莫走進了便利店。
　店裡沒有燈光。
　地面滿是玻璃碎片以及商品，感覺與其像是魔物幹的好事，更像是別人趁火打劫一樣。
。
「意外還留有很多東西啊ー……」
。
　也許是沒有拿乾淨吧。店裡還有許多食物和飲料。
　我將這些東西都全部收納到道具包裡。
。
　不過真不愧是這麼多的數量。
　可能要花很多時間才能全部收納完畢。
　我也在店裡走動的同時，收納著物品。
　於是發現一個在意的東西。
。
「……嗯？」
。
　我伸出手撿起地面上的『那個』。
　那是一個藍色的小石頭。
。
「這……該不會是魔石吧？」
。
　無論是顏色，還是形狀都跟我以往打倒的哥布林的魔石很相似。
　收納進去之後，正如我所意料，顯示的是『哥布林的魔石（極小）×１個』。
。
「這裡有發生過戰鬥嗎……？」
。
　可如果是這樣為什麼魔石會落在這裡啊？
。
「……因為慌慌張張逃跑了？」
。
　比如說，在那人趁火打劫的時候偶然遇到了哥布林，於是勉強打倒它以後就逃跑了。這樣就沒有閑工夫去確認這個魔石了吧。之後如果也包括聽到那聲通知在內，我不認為那人會繼續呆在這裡。
。
「嘛，我就心懷感激地收下了」
。
　我將魔石遞給莫莫。
　莫莫很高興似的吃著魔石。
　最終找到的魔石只有那一個。

。

　接下來到便利店的內部。
。
「嘿~便利店內部是這樣子的啊」
。
　我從店裡的工作人員出入口走進去，環視著裡面。
　我還是第一次看到便利店內部的樣子，原來是這種感覺。
　喔，從塑料瓶的縫隙間可以看得到店裡。
　總覺有股新鮮感，這種事真是有點興奮不已。
。
　幸好這邊整整齊齊。
　幾乎東西都沒有被人碰過，維持著原樣。
　對我來說也是可喜可賀的事情。
。
「好多箱子啊」
。
　咖啡、茶、酒等等都放在紙箱裡，而食物、特別是便當都幾乎沒有放在裡面。
　嘛，畢竟是便利店。保質期之類的應該很嚴格，為了能夠迅速處理掉，所以庫存不會弄太多了吧？
　喔，發售前的雜誌也有。
　我就心懷感激地收下了。
。

「莫莫，讓你久等了……啊咧？」
。
　莫莫沒有回應。
　我想著到底是怎麼了，環顧著店裡。於是莫莫嘴裡叼著什麼往這邊過來了。
　它嘴裡緊緊地咬著小袋包裝的狗糧。
。
「……你想要嗎?」
。
「汪」
。
　似乎是想要。
　不過它回答我的瞬間，狗糧掉在地面上了。
　於是莫莫連忙又再叼起來。
。
「……莫莫，你想要這個嗎？」

「汪」
。
　撲通。
　又掉下來了
　莫莫慌張地咬起來。
　好可愛。
。
　雖然我還想再問一遍，看看它慌張的樣子。不過再繼續欺負它也太可怜了。
　我從莫莫那裡拿走小袋裝的狗糧，收納進道具包裡。

「くぅーん？」
。
　誒~要拿走嗎？不能吃嗎？莫莫如此訴說著。
。
「不行，你剛剛不是吃了很多了嗎。之後會好好喂你吃的。忍耐一下吧」
。
「……わふん」
。
　有點沒有自信的回應。
　……不行嗎？不能給我嗎？
　就算你用這麼閃閃發光的眼神看著我，不行就是不行！
　我狠下心，承受著莫莫的眼神。
　最終莫莫終於放棄了。
　呼，哎呀，真是艱難的戰鬥……。

。
　之後過了幾分鐘。
。
「呼ー…這樣就大概差不多了吧……」
。
　收納工作終於結束了。
　結果店裡的庫存都全部裝進道具包裡了。
　目錄也是密密麻麻的。
　光憑這些應該食物吃個半年吧。
　終究是建立在我一個人吃的前提之下。
。
　那麼，接下來該怎麼辦？
　看了下手表，時間已經是下午三點了。
　吃點心的時間了。不，不對吧。
。
　在日落之前我得想辦法確保安全的地方。
　如果有夜行性的魔物就麻煩了。
　今天就結束搜索，姑且回到基地吧？
　再糟糕也能和莫莫互相把風，各自讓身體休息一下。
。
「嗯？」
。
　當我想著這些事情的時候，突然看到遠方有什麼奇怪的東西。
　遠處的某座建築物的屋頂上面，有條白色的布料在隨風飄蕩。
。
「那是什麼？」
。
　記得那邊應該是有大型購物中心吧。
　我偶爾在休息日也會過去一趟。
。
　從收納的目錄中取出『望遠鏡』。
　利用它放大看著白色的布料。
　雖然當中的字體不是很全面，不過寫著SOS。
。
「莫非是生存者？」
。
　怎麼辦？要去看一下嗎？
　可是那邊總覺得傳來了一股討厭的感覺啊……。
