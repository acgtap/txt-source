```
　　生于贫穷村落的少女——雪拉，在袭击她的JF军士兵身后，看见了「似乎很好吃」的死神。死神的镰刀还没挥下，雪拉就将死神吃个精光。于是，她成为王国军的士兵。手握巨大的镰刀，为了打倒可恨的JF军而战。
　　为了得到美味的食物而战——

死神を食べたらどうなるんだろう。
私には良く分からない。分かるのはたった一つ。
お腹がすいた。
```
